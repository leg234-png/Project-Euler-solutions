# Nombres square root smooth

## Description du Problème
Un entier positif est appelé square root smooth si tous ses facteurs premiers sont strictement inférieurs à sa racine carrée.

En incluant le nombre 1, il y a 29 nombres square root smooth ne dépassant pas 100.

**Objectif** : Combien y a-t-il de nombres square root smooth ne dépassant pas 10000000000 ?

---

## Exemples
### Exemple 1
**Entrée** : `limite = 100`
**Sortie** : `29`
**Explication** : Il y a 29 nombres square root smooth ≤ 100.

---

## Approche de Résolution

### Stratégie
1. x n'est pas square-root-smooth si et seulement si x = py où p est un nombre premier et y ≤ p.
2. Raison : x = py ≤ p² => √x ≤ p.
3. Trouver tous les nombres non-square-root-smooth x ≤ n.
4. Pour un y donné, alors p = x/y, donc il y a π(n/y) nombres premiers possibles.
5. Cependant, il y a π(y-1) nombres premiers qui ne sont pas autorisés.
6. Donc le nombre de nombres non-square-root-smooth x ≤ n = ∑_{y=1}^{√n} (π(n/y) - π(y-1)).
7. Le nombre de nombres square-root-smooth = n - (nombre de non-square-root-smooth).

### Observations
- On peut utiliser la fonction de comptage des nombres premiers π(x).
- Il existe des méthodes efficaces pour calculer π(x) pour de grandes valeurs.
- La formule permet de calculer sans itérer sur tous les nombres.

### Complexité
- **Temporelle** : O(√n × log log n) avec un crible pour π(x).
- **Spatiale** : O(√n) pour le crible.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long prime_pi(long long n) {
    if (n < 2) return 0;
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (long long i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (long long j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    long long count = 0;
    for (long long i = 2; i <= n; i++) {
        if (is_prime[i]) count++;
    }
    return count;
}

long long compute(long long n) {
    long long sqrt_n = sqrt(n);
    long long non_smooth = 0;
    
    for (long long y = 1; y <= sqrt_n; y++) {
        long long pi_n_over_y = prime_pi(n / y);
        long long pi_y_minus_1 = prime_pi(y - 1);
        non_smooth += (pi_n_over_y - pi_y_minus_1);
    }
    
    return n - non_smooth;
}

int main() {
    cout << compute(10000000000LL) << endl;
    return 0;
}
```
