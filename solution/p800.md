# Entiers hybrides

## Description du Problème
Un entier hybride est un nombre de la forme p^q × q^p où p et q sont des nombres premiers distincts.

Par exemple, 800 = 2⁵ × 5² est un entier hybride.

Définissons C(n) comme le nombre d'entiers hybrides inférieurs ou égaux à n.

On vous donne C(800) = 2 et C(800^800) = 10790.

**Objectif** : Trouver C(800^800^800^800).

---

## Exemples
### Exemple 1
**Entrée** : `n = 800`
**Sortie** : `2`
**Explication** : C(800) = 2.

---

## Approche de Résolution

### Stratégie
1. p^q × q^p est un entier hybride.
2. log(p^q × q^p) = log(a^b) => q log(p) + p log(q) = b log(a).
3. Étant donné p, on peut trouver le plus petit nombre premier q tel que q log(p) + p log(q) > b log(a), alors on peut juste ajouter tous les nombres premiers entre p et q.
4. Générer tous les nombres premiers jusqu'à une certaine limite.
5. Pour chaque paire (p, q) avec p < q, calculer p^q × q^p.
6. Compter combien sont ≤ n.

### Observations
- On peut utiliser les logarithmes pour éviter de calculer les grandes puissances.
- On peut optimiser en utilisant des bornes pour p et q.
- Il y a relativement peu de paires à vérifier.

### Complexité
- **Temporelle** : O(P²) où P est le nombre de nombres premiers à considérer.
- **Spatiale** : O(P) pour stocker les nombres premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <set>

using namespace std;

vector<int> list_primes(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

long long compute(long long limit) {
    // Calculer la limite pour les nombres premiers
    int max_prime = 100; // Ajuster selon les besoins
    vector<int> primes = list_primes(max_prime);
    
    set<long long> hybrid;
    
    for (size_t i = 0; i < primes.size(); i++) {
        for (size_t j = i + 1; j < primes.size(); j++) {
            int p = primes[i];
            int q = primes[j];
            
            // Utiliser les logarithmes pour vérifier
            double log_value = q * log(p) + p * log(q);
            double log_limit = log(limit);
            
            if (log_value <= log_limit) {
                // Calculer p^q × q^p si nécessaire
                // Pour de très grandes valeurs, on peut utiliser une approximation
                hybrid.insert((long long)(exp(log_value)));
            }
        }
    }
    
    return hybrid.size();
}

int main() {
    // Calculer 800^800^800^800 est très grand, utiliser une approximation
    long long limit = 800;
    for (int i = 0; i < 3; i++) {
        limit = pow(limit, 800);
    }
    cout << compute(limit) << endl;
    return 0;
}
```
