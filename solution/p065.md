# Convergents de e

## Description du Problème
La racine carrée de 2 peut être écrite comme une fraction continue infinie. La fraction continue infinie peut être écrite, √2 = [1;(2)], où (2) indique que 2 se répète à l'infini.

Il s'avère que la séquence des valeurs partielles des fractions continues pour les racines carrées fournit les meilleures approximations rationnelles. Considérons les convergents pour √2 :

1 + 1/2 = 3/2
1 + 1/(2 + 1/2) = 7/5
1 + 1/(2 + 1/(2 + 1/2)) = 17/12
1 + 1/(2 + 1/(2 + 1/(2 + 1/2))) = 41/29

Ce qui est le plus surprenant est que la constante mathématique importante, e = [2; 1, 2, 1, 1, 4, 1, 1, 6, 1, ..., 1, 2k, 1, ...].

Les dix premiers termes dans la séquence des convergents pour e sont :
2, 3, 8/3, 11/4, 19/7, 87/32, 106/39, 193/71, 1264/465, 1457/536, ...

La somme des chiffres du numérateur du 10ème convergent est 1 + 4 + 5 + 7 = 17.

**Objectif** : Trouver la somme des chiffres du numérateur du 100ème convergent de la fraction continue pour e.

---

## Exemples
### Exemple 1
**Entrée** : 10ème convergent
**Sortie** : `17`
**Explication** : Le numérateur du 10ème convergent est 1457, et la somme de ses chiffres est 1+4+5+7 = 17.

---

## Approche de Résolution

### Stratégie
1. Calculer les convergents de e en utilisant la formule récursive pour les fractions continues.
2. Le terme de la fraction continue pour e suit le motif : [2; 1, 2, 1, 1, 4, 1, 1, 6, 1, ...]
3. Calculer le 100ème convergent en partant de la fin et en remontant.
4. Utiliser BigInteger pour gérer les grands nombres.
5. Calculer la somme des chiffres du numérateur final.

### Observations
- Le terme i de la fraction continue est :
  - 2 si i = 0
  - (i/3 * 2 + 2) si i % 3 == 2
  - 1 sinon
- On calcule les convergents de droite à gauche pour éviter les divisions répétées.

### Complexité
- **Temporelle** : O(n), où n est le nombre de convergents (100).
- **Spatiale** : O(1), car on ne stocke que les valeurs courantes.

---

## Solution en C++

```cpp
#include <iostream>
#include <gmpxx.h>
#include <string>

using namespace std;

int continuedFractionTerm(int i) {
    if (i == 0)
        return 2;
    else if (i % 3 == 2)
        return i / 3 * 2 + 2;
    else
        return 1;
}

int main() {
    mpz_class n = 1;
    mpz_class d = 0;
    
    for (int i = 99; i >= 0; i--) {
        mpz_class temp = continuedFractionTerm(i) * n + d;
        d = n;
        n = temp;
    }
    
    int sum = 0;
    string s = n.get_str();
    for (char c : s)
        sum += c - '0';
    
    cout << sum << endl;
    return 0;
}
```
