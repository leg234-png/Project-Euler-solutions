# Somme de Puissances Propres

## Description du Problème
La série, 1¹ + 2² + 3³ + ... + 10¹⁰ = 10405071317.

**Objectif** : Trouver les dix derniers chiffres de la série, 1¹ + 2² + 3³ + ... + 1000¹⁰⁰⁰.

---

## Exemples
### Exemple 1
**Entrée** : Limite = 10
**Sortie** : `0405071317`
**Explication** : Les 10 derniers chiffres de 1¹ + 2² + ... + 10¹⁰.

---

## Approche de Résolution

### Stratégie
1. Pour chaque i de 1 à 1000 :
   - Calculer iⁱ mod 10¹⁰.
   - Ajouter à la somme.
2. Retourner la somme mod 10¹⁰.

### Observations
- On n'a besoin que des 10 derniers chiffres, donc on peut travailler modulo 10¹⁰.
- Pour calculer iⁱ mod 10¹⁰, on peut utiliser l'exponentiation modulaire.
- On peut utiliser `BigInteger` ou implémenter l'exponentiation modulaire manuellement.

### Complexité
- **Temporelle** : O(n × log n), où `n` = 1000.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>

long long modPow(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        base = (base * base) % mod;
        exp /= 2;
    }
    
    return result;
}

int main() {
    const long long modulus = 10000000000LL;  // 10^10
    long long sum = 0;
    
    for (int i = 1; i <= 1000; ++i) {
        sum = (sum + modPow(i, i, modulus)) % modulus;
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
