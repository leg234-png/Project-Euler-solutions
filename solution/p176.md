# Triangles rectangles qui partagent une cathète

## Description du Problème
Il existe exactement 47547 triangles rectangles à côtés entiers différents pour lesquels un triangle rectangle existe avec une des cathètes égale à 12.

**Objectif** : Trouver le plus petit entier qui peut être la longueur d'une cathète d'exactement 47547 triangles rectangles à côtés entiers différents.

---

## Exemples
### Exemple 1
**Entrée** : `k = 1`
**Sortie** : `Plus petit entier`
**Explication** : Le plus petit entier qui peut être une cathète d'exactement 1 triangle rectangle.

---

## Approche de Résolution

### Stratégie
1. Utiliser la formule pour le nombre de triangles rectangles avec une cathète donnée.
2. Si 2k+1 = p₁ × p₂ × ... × pₙ, alors le nombre de triangles est (2e₁-1)(2e₂+1)...
3. Factoriser 2×47547+1 = 95095 = 5 × 7 × 11 × 13 × 19.
4. Utiliser la formule pour trouver n = 2¹⁰ × 3⁶ × 5⁵ × 7³ × 11².

### Observations
- Pour une cathète de longueur n, le nombre de triangles rectangles est déterminé par la factorisation de 2k+1.
- La formule utilise les exposants des facteurs premiers dans la factorisation.
- On peut utiliser une recherche pour trouver le plus petit n avec exactement k triangles.

### Complexité
- **Temporelle** : O(√n) pour factoriser et calculer le nombre de triangles.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

int main() {
    // 2*47547 + 1 = 95095 = 5 * 7 * 11 * 13 * 19
    // Utilisant la formule (2e_1 - 1)(2e_2 + 1)...
    // n = 2^10 * 3^6 * 5^5 * 7^3 * 11^2
    long long result = 1;
    result *= pow(2, 10);
    result *= pow(3, 6);
    result *= pow(5, 5);
    result *= pow(7, 3);
    result *= pow(11, 2);
    
    cout << result << endl;
    return 0;
}
```
