# Entiers alexandrins

## Description du Problème
Un entier positif A est appelé un entier alexandrin s'il existe des entiers p, q, r tels que :

1/A = 1/p + 1/q + 1/r.

Par exemple, 630 est un entier alexandrin (p=5, q=−7, r=−18).

En fait, 630 est le 6ème entier alexandrin, les 6 premiers entiers alexandrins étant : 6, 42, 120, 156, 420 et 630.

**Objectif** : Trouver le 150000ème entier alexandrin.

---

## Exemples
### Exemple 1
**Entrée** : `n = 6`
**Sortie** : `630`
**Explication** : 630 est le 6ème entier alexandrin.

---

## Approche de Résolution

### Stratégie
1. Utiliser la formule A = p(p + k)((p² + 1)/k + p) où k|p² + 1 et p est un entier positif.
2. Pour chaque p, trouver tous les diviseurs k de p² + 1.
3. Générer tous les entiers alexandrins et les trier.
4. Retourner le 150000ème.

### Observations
- On n'a besoin de boucler que sur la première moitié des diviseurs.
- Si p² + 1 = ab, alors A₁ = p(p + a)(b + p) et A₂ = p(p + b)(a + p) donnent le même résultat.
- On peut optimiser en ne générant que les valeurs uniques.

### Complexité
- **Temporelle** : O(P × D) où P est le nombre de valeurs de p à tester et D est le nombre moyen de diviseurs.
- **Spatiale** : O(N) pour stocker les entiers alexandrins.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <algorithm>
#include <cmath>

using namespace std;

vector<long long> divisors_of(long long x) {
    vector<long long> divisors;
    for (long long i = 1; i * i <= x; i++) {
        if (x % i == 0) {
            divisors.push_back(i);
            if (i * i != x) {
                divisors.push_back(x / i);
            }
        }
    }
    return divisors;
}

long long compute() {
    vector<long long> alexandrian_integers;
    long long p = 1;
    
    while (alexandrian_integers.size() < 500000) {
        long long p2_plus_1 = p * p + 1;
        vector<long long> divisors = divisors_of(p2_plus_1);
        
        for (long long k : divisors) {
            long long A = p * (p + k) * ((p2_plus_1 / k) + p);
            alexandrian_integers.push_back(A);
        }
        p++;
    }
    
    sort(alexandrian_integers.begin(), alexandrian_integers.end());
    return alexandrian_integers[149999];
}

int main() {
    cout << compute() << endl;
    return 0;
}
```
