# Comptage de chiffres

## Description du Problème
Soit f(n,d) le nombre total de chiffres d qui ont été écrits après que le nombre n ait été écrit.

Les deux premières solutions de l'équation f(n,1)=n sont n=0 et n=1. La solution suivante est n=199981.

De la même manière, la fonction f(n,d) donne le nombre total de chiffres d qui ont été écrits après que le nombre n ait été écrit.

En fait, pour chaque chiffre d≠0, 0 est la première solution de l'équation f(n,d)=n.

**Objectif** : Soit s(d) la somme de toutes les solutions pour lesquelles f(n,d)=n. Trouver ∑s(d) pour d=1 à 9.

---

## Exemples
### Exemple 1
**Entrée** : `d = 1`
**Sortie** : `Somme des solutions`
**Explication** : Les solutions de f(n,1)=n incluent 0, 1, 199981, etc.

---

## Approche de Résolution

### Stratégie
1. Utiliser une recherche binaire non bornée pour trouver toutes les solutions.
2. Pour chaque position k dans n, calculer le nombre de fois que le chiffre d apparaît.
3. Utiliser la formule de comptage basée sur les puissances de 10.
4. Trouver tous les points fixes où f(n,d) = n.

### Observations
- La fonction f(n,d) compte le nombre total de chiffres d dans tous les nombres de 1 à n.
- On cherche les points fixes où f(n,d) = n.
- Il existe une périodicité dans les solutions qui permet d'optimiser la recherche.

### Complexité
- **Temporelle** : O(log n) pour chaque solution trouvée, où n est la valeur de la solution.
- **Spatiale** : O(1) pour la recherche binaire.

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <string>
#include <algorithm>

using namespace std;

long long c(long long x, int k, int d) {
    string x_str = to_string(x);
    reverse(x_str.begin(), x_str.end());
    x_str += "0";
    
    long long Y = (x / pow(10, k)) * pow(10, k - 1);
    int digit = x_str[k] - '0';
    
    if (d > 0) {
        if (digit > d) {
            return Y + pow(10, k - 1);
        } else if (digit == d) {
            return Y + (x % (long long)pow(10, k - 1)) + 1;
        } else {
            return Y;
        }
    } else if (d == 0) {
        if (digit > d) {
            return Y;
        } else if (digit == d) {
            return Y - pow(10, k - 1) + (x % (long long)pow(10, k - 1)) + 1;
        }
    }
    return 0;
}

long long f(long long n, int d) {
    long long total = 0;
    int len = to_string(n).length();
    for (int k = 1; k <= len; k++) {
        total += c(n, k, d);
    }
    return total;
}

long long s(int d) {
    long long x = 2, p = 1;
    vector<long long> values;
    long long count = 0;
    
    while (true) {
        long long upper = x + p;
        if (upper > (long long)d * 10000000000LL) {
            break;
        }
        
        long long v = f(x, d);
        
        if (p == 1 && f(upper, d) == upper) {
            count++;
            values.push_back(upper);
        }
        
        if (v > x + 1) {
            x = v - 1;
            p = 1;
        } else if (v < x) {
            x = upper;
            p *= 2;
        } else {
            x++;
            p = 1;
        }
    }
    
    long long sum = 0;
    for (long long val : values) {
        sum += val;
    }
    return sum;
}

int main() {
    long long total = 0;
    for (int d = 1; d <= 9; d++) {
        total += s(d);
    }
    cout << total << endl;
    return 0;
}
```
