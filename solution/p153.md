# Investigation des entiers de Gauss

## Description du Problème
Si on multiplie le numérateur et le dénominateur par le conjugué complexe de 1+2i : 1−2i, on obtient que 1+2i est un diviseur de 5.

Notez que 1+i n'est pas un diviseur de 5 car 5/(1+i) = 5/2 - 5/2i.

**Objectif** : Soit S(N) la somme de tous les diviseurs de Gauss de tous les entiers positifs n ≤ N. Trouver S(10⁸).

---

## Exemples
### Exemple 1
**Entrée** : `N = 5`
**Sortie** : `Somme des diviseurs de Gauss`
**Explication** : Pour n = 5, les diviseurs de Gauss incluent 1+2i, 1-2i, etc.

---

## Approche de Résolution

### Stratégie
1. Si z = a + bi divise d et g = gcd(a, b), alors g(a₁² + b₁²) doit diviser d.
2. Pour b = 0, on a le cas de base : somme de tous les diviseurs.
3. Pour a + bi avec gcd(a, b) = 1, on obtient 4 diviseurs différents : (a + bi, a - bi, b + ai, b - ai).
4. Stocker les contributions dans un dictionnaire et itérer sur tous les multiples.

### Observations
- Les entiers de Gauss sont de la forme a + bi où a et b sont des entiers.
- Un entier de Gauss z divise un entier d si d/z est aussi un entier de Gauss.
- On doit considérer tous les diviseurs de Gauss pour chaque entier n ≤ N.

### Complexité
- **Temporelle** : O(N × √N) pour énumérer tous les paires (a, b) et leurs multiples.
- **Spatiale** : O(√N) pour stocker le dictionnaire des contributions.

---

## Solution en C++

```cpp
#include <iostream>
#include <map>
#include <cmath>
#include <algorithm>

using namespace std;

long long S(long long N) {
    map<long long, long long> d;
    d[1] = 1;
    d[2] = 2;
    
    long long sqrtN = (long long)sqrt(N);
    for (long long a = 1; a <= sqrtN; a++) {
        for (long long b = a + 1; b <= sqrtN && a*a + b*b <= N; b++) {
            if (__gcd(a, b) == 1) {
                long long v = a*a + b*b;
                d[v] += 2*a + 2*b;
            }
        }
    }
    
    long long total = 0;
    for (auto& [v, contrib] : d) {
        for (long long g = 1; g * v <= N; g++) {
            total += contrib * g * (N / (g * v));
        }
    }
    
    return total;
}

int main() {
    cout << S(100000000) << endl;
    return 0;
}
```
