# Fenêtre dans une matrice

## Description du Problème
Soit A(k, n) le nombre total de matrices 2×n telles que la somme des entrées dans chaque fenêtre 2×k est k.

On vous donne que A(3, 9) = 560 et A(4, 20) = 1060870.

**Objectif** : Trouver A(10⁸, 10¹⁶). Donner votre réponse modulo 1000000007.

---

## Exemples
### Exemple 1
**Entrée** : `k = 3, n = 9`
**Sortie** : `560`
**Explication** : A(3, 9) = 560.

---

## Approche de Résolution

### Stratégie
1. Pour une matrice 2×n, chaque fenêtre 2×k doit avoir une somme de k.
2. Cela signifie que chaque fenêtre contient exactement k entrées de 1.
3. Utiliser la programmation dynamique pour compter les matrices valides.
4. Observer les patterns dans les valeurs de A(k, n).
5. Utiliser des formules combinatoires pour calculer efficacement.

### Observations
- Il y a des relations de récurrence pour A(k, n).
- On peut utiliser des techniques de comptage combinatoire.
- Pour de grandes valeurs, on doit optimiser considérablement.

### Complexité
- **Temporelle** : O(n/k × k²) avec optimisation.
- **Spatiale** : O(k) pour la DP.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long mod = 1000000007;

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long mod_inverse(long long a, long long mod) {
    return mod_pow(a, mod - 2, mod);
}

long long A(long long k, long long n, long long mod) {
    long long total = 0;
    long long f0 = 1;
    long long power = mod_pow(2, n / k, mod);
    
    for (long long a = 0; a <= n / k; a++) {
        total = (total + f0 * mod_pow(power, k - 2 * a, mod)) % mod;
        f0 = (f0 * ((k - 2 * a) * (k - 2 * a - 1) % mod)) % mod;
        f0 = (f0 * mod_inverse((a + 1) * (a + 1), mod)) % mod;
    }
    
    return total;
}

int main() {
    cout << A(100000000, 10000000000000000LL, mod) << endl;
    return 0;
}
```
