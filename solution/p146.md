# Investigating a Prime Pattern

## Description du Problème
Le plus petit entier positif n pour lequel les nombres n² + 1, n² + 3, n² + 7, n² + 9, n² + 13 et n² + 27 sont des nombres premiers consécutifs est 10. La somme de tous ces entiers n en dessous d'un million est 1242490.

**Objectif** : Quelle est la somme de tous ces entiers n en dessous de 150 millions ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : `true`
**Explication** : 10² + 1 = 101, 10² + 3 = 103, 10² + 7 = 107, 10² + 9 = 109, 10² + 13 = 113, 10² + 27 = 127 sont tous premiers et consécutifs.

---

## Approche de Résolution

### Stratégie
1. Exclure 90% des candidats en observant que n doit être divisible par 10.
2. Pour chaque n = 0 mod 10, générer l'ensemble {n² + 1, n² + 3, n² + 7, n² + 9, n² + 13, n² + 27}.
3. Vérifier que chaque nombre dans l'ensemble est premier.
4. Vérifier qu'aucun nombre entre n² + 1 et n² + 27 (sauf ceux dans l'ensemble) n'est premier.
5. Si toutes les conditions sont satisfaites, ajouter n à la somme.

### Observations
- n doit être pair (sinon n² + 1 est pair).
- n doit être divisible par 5 (sinon un des nombres est divisible par 5).
- Donc n = 0 mod 10.

### Complexité
- **Temporelle** : O(n × nombre_de_nombres_premiers), où n = 150 × 10⁶ / 10.
- **Spatiale** : O(√n) pour stocker les nombres premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>

std::vector<int> primes;

bool isPrime(long long n) {
    int end = sqrt(n);
    for (int p : primes) {
        if (p > end) break;
        if (n != p && n % p == 0) return false;
    }
    return true;
}

bool hasConsecutivePrimes(int n) {
    long long n2 = (long long)n * n;
    long long increments[] = {1, 3, 7, 9, 13, 27};
    
    // Vérifier que chaque nombre dans l'ensemble est premier
    for (long long inc : increments) {
        if (!isPrime(n2 + inc)) return false;
    }
    
    // Vérifier qu'aucun autre nombre n'est premier
    for (int i = 1; i < 27; i++) {
        bool inSet = false;
        for (long long inc : increments) {
            if (inc == i) {
                inSet = true;
                break;
            }
        }
        if (!inSet && isPrime(n2 + i)) return false;
    }
    
    return true;
}

int main() {
    const int LIMIT = 150000000;
    // Générer les nombres premiers jusqu'à √(LIMIT² + 27)
    // primes = listPrimes(...);
    
    long long sum = 0;
    for (int n = 0; n < LIMIT; n += 10) {
        if (hasConsecutivePrimes(n)) {
            sum += n;
        }
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
