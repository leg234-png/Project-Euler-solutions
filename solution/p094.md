# Almost Equilateral Triangles

## Description du Problème
Il est facilement prouvé qu'aucun triangle équilatéral n'existe avec des côtés de longueur entière et une aire entière. Cependant, le triangle presque équilatéral 5-5-6 a une aire de 12 unités carrées.

Nous définirons un triangle presque équilatéral comme un triangle pour lequel deux côtés sont égaux et le troisième diffère d'au plus une unité.

**Objectif** : Trouver la somme des périmètres de tous les triangles presque équilatéraux avec des longueurs de côtés entières et une aire entière et dont les périmètres ne dépassent pas un milliard (1000000000).

---

## Exemples
### Exemple 1
**Entrée** : `périmètre ≤ 20`
**Sortie** : `16`
**Explication** : Le triangle 5-5-6 a un périmètre de 16 et une aire entière de 12.

---

## Approche de Résolution

### Stratégie
1. Utiliser le théorème de Pythagore pour générer des triplets primitifs.
2. Pour chaque triplet primitif (a, b, c), vérifier si 2a = c ± 1.
3. Si c'est le cas, former un triangle presque équilatéral avec périmètre 3c ± 1.
4. Sommer tous les périmètres valides qui ne dépassent pas 10⁹.

### Observations
- Un triangle presque équilatéral peut être divisé en deux triangles rectangles.
- Si a = (c ± 1) / 2 et a² + b² = c², alors le triangle a une aire entière.
- On utilise la génération de triplets pythagoriciens primitifs avec s et t impairs et copremiers.

### Complexité
- **Temporelle** : O(√n), où n=10⁹, car on génère des triplets jusqu'à ce que le périmètre dépasse la limite.
- **Spatiale** : O(1), car on utilise seulement des variables pour accumuler la somme.

---

## Solution en C++

```cpp
#include <iostream>
#include <algorithm>

long long gcd(long long a, long long b) {
    while (b != 0) {
        long long temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

int main() {
    const long long LIMIT = 1000000000;
    long long sum = 0;
    
    // Générer des triplets pythagoriciens primitifs
    // s > t > 0 sont des entiers impairs copremiers
    for (long long s = 1; s * s <= (LIMIT + 1) / 3; s += 2) {
        for (long long t = s - 2; t > 0; t -= 2) {
            if (gcd(s, t) == 1) {
                long long a = s * t;
                long long b = (s * s - t * t) / 2;
                long long c = (s * s + t * t) / 2;
                
                if (a * 2 == c - 1) {
                    long long p = c * 3 - 1;
                    if (p <= LIMIT) sum += p;
                }
                if (a * 2 == c + 1) {
                    long long p = c * 3 + 1;
                    if (p <= LIMIT) sum += p;
                }
                if (b * 2 == c - 1) {
                    long long p = c * 3 - 1;
                    if (p <= LIMIT) sum += p;
                }
                if (b * 2 == c + 1) {
                    long long p = c * 3 + 1;
                    if (p <= LIMIT) sum += p;
                }
            }
        }
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
