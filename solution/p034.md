# Factorielle des Chiffres

## Description du Problème
145 est un nombre curieux, car 1! + 4! + 5! = 1 + 24 + 120 = 145.

**Objectif** : Trouver la somme de tous les nombres qui sont égaux à la somme des factorielles de leurs chiffres.

**NOTE** : Comme 1! = 1 et 2! = 2 ne sont pas des sommes, ils ne sont pas inclus.

---

## Exemples
### Exemple 1
**Entrée** : Nombres curieux
**Sortie** : `40730`
**Explication** : La somme de tous les nombres égaux à la somme des factorielles de leurs chiffres.

---

## Approche de Résolution

### Stratégie
1. Précalculer les factorielles de 0 à 9.
2. Pour chaque nombre i de 3 à une limite supérieure :
   - Calculer la somme des factorielles de ses chiffres.
   - Si cette somme est égale à i, ajouter i à la somme totale.
3. Retourner la somme totale.

### Observations
- Si un nombre a au moins n ≥ 8 chiffres, alors même si chaque chiffre est 9, n × 9! est toujours inférieur au nombre (qui est au moins 10ⁿ).
- Une limite sûre est 7 × 9! = 2540160, mais on peut utiliser 10 000 000 pour être sûr.
- On peut précalculer les factorielles pour éviter de les recalculer.

### Complexité
- **Temporelle** : O(n × d), où `n` est la limite et `d` est le nombre de chiffres.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>

int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

int factorialDigitSum(int n) {
    int fact[] = {1, 1, 2, 6, 24, 120, 720, 5040, 40320, 362880};
    int sum = 0;
    while (n > 0) {
        sum += fact[n % 10];
        n /= 10;
    }
    return sum;
}

int main() {
    int sum = 0;
    for (int i = 3; i < 10000000; ++i) {
        if (i == factorialDigitSum(i)) {
            sum += i;
        }
    }
    std::cout << sum << std::endl;
    return 0;
}
```
