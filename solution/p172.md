# Investigating numbers with few repeated digits

## Description du Problème
Combien de nombres de 18 chiffres (sans zéros en tête) peuvent être formés de telle sorte qu'aucun chiffre n'apparaisse plus de 3 fois ?

---

## Exemples
### Exemple 1
**Entrée** : `length = 2, maxCount = 3`
**Sortie** : Nombre de nombres de 2 chiffres où chaque chiffre apparaît au plus 3 fois
**Explication** : Tous les nombres de 2 chiffres (10-99) satisfont cette condition.

---

## Approche de Résolution

### Stratégie
1. Partitionner le problème par les fréquences de répétition des chiffres.
2. Pour chaque partition (façon d'exprimer 18 comme somme de 10 termes, chacun entre 0 et 3) :
   - Compter le nombre de façons d'assigner les 10 chiffres aux fréquences (coefficient multinomial).
   - Compter le nombre de permutations avec ces fréquences (coefficient multinomial).
3. Multiplier par 9/10 pour exclure les nombres commençant par 0.
4. Utiliser la récursion pour générer toutes les partitions possibles.

### Observations
- Une partition est une séquence non-croissante de 10 termes qui somment à 18.
- Chaque terme doit être entre 0 et 3.
- On utilise des coefficients multinomiaux pour compter les arrangements.

### Complexité
- **Temporelle** : O(P × 10!), où P est le nombre de partitions.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

const int LENGTH = 18;
const int MAX_COUNT = 3;
const int BASE = 10;

cpp_int factorial(int n) {
    cpp_int result = 1;
    for (int i = 2; i <= n; i++)
        result *= i;
    return result;
}

cpp_int partitionAndCount(int sum, int max, vector<int>& terms) {
    if (terms.size() == BASE) {
        if (sum == 0)
            return countWays(terms);
        else
            return 0;
    } else {
        cpp_int result = 0;
        for (int i = min(max, sum); i >= 0; i--) {
            terms.push_back(i);
            result += partitionAndCount(sum - i, i, terms);
            terms.pop_back();
        }
        return result;
    }
}

cpp_int countWays(const vector<int>& freqs) {
    vector<int> histogram(MAX_COUNT + 1, 0);
    for (int x : freqs)
        histogram[x]++;
    
    cpp_int ways = factorial(BASE);
    for (int x : histogram)
        ways /= factorial(x);
    
    ways *= factorial(LENGTH);
    for (int x : freqs)
        ways /= factorial(x);
    
    return ways;
}

int main() {
    vector<int> terms;
    cpp_int ways = partitionAndCount(LENGTH, MAX_COUNT, terms);
    
    ways = ways * (BASE - 1) / BASE;
    
    cout << ways << endl;
    return 0;
}
```
