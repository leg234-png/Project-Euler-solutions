# Plus Grand Produit dans une Série

## Description du Problème
Les quatre chiffres adjacents dans le nombre de 1000 chiffres qui ont le plus grand produit sont 9 × 9 × 8 × 9 = 5832.

**Objectif** : Trouver les treize chiffres adjacents dans le nombre de 1000 chiffres qui ont le plus grand produit. Quelle est la valeur de ce produit ?

---

## Exemples
### Exemple 1
**Entrée** : Nombre de 1000 chiffres, k = 4
**Sortie** : `5832`
**Explication** : Les quatre chiffres adjacents 9, 9, 8, 9 ont le produit 9×9×8×9 = 5832.

---

## Approche de Résolution

### Stratégie
1. Parcourir toutes les sous-chaînes de 13 chiffres consécutifs.
2. Pour chaque sous-chaîne, calculer le produit des chiffres.
3. Garder trace du produit maximum.
4. Retourner le maximum.

### Observations
- Le nombre de 1000 chiffres est donné comme une chaîne de caractères.
- On convertit chaque caractère en chiffre en soustrayant '0'.
- Le produit maximum peut être très grand (9¹³), donc on utilise `long long`.
- On peut éviter de recalculer le produit en utilisant une fenêtre glissante, mais pour 13 chiffres, le calcul direct est suffisant.

### Complexité
- **Temporelle** : O(n × k), où `n` = 1000 (longueur du nombre) et `k` = 13 (nombre de chiffres adjacents).
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>

int main() {
    const std::string number = 
        "73167176531330624919225119674426574742355349194934"
        "96983520312774506326239578318016984801869478851843"
        "85861560789112949495459501737958331952853208805511"
        "12540698747158523863050715693290963295227443043557"
        "66896648950445244523161731856403098711121722383113"
        "62229893423380308135336276614282806444486645238749"
        "30358907296290491560440772390713810515859307960866"
        "70172427121883998797908792274921901699720888093776"
        "65727333001053367881220235421809751254540594752243"
        "52584907711670556013604839586446706324415722155397"
        "53697817977846174064955149290862569321978468622482"
        "83972241375657056057490261407972968652414535100474"
        "82166370484403199890008895243450658541227588666881"
        "16427171479924442928230863465674813919123162824586"
        "17866458359124566529476545682848912883142607690042"
        "24219022671055626321111109370544217506941658960408"
        "07198403850962455444362981230987879927244284909188"
        "84580156166097919133875499200524063689912560717606"
        "05886116467109405077541002256983155200055935729725"
        "71636269561882670428252483600823257530420752963450";
    
    const int adjacent = 13;
    long long maxProd = -1;
    
    for (int i = 0; i + adjacent <= number.length(); ++i) {
        long long prod = 1;
        for (int j = 0; j < adjacent; ++j) {
            prod *= (number[i + j] - '0');
        }
        if (prod > maxProd) {
            maxProd = prod;
        }
    }
    
    std::cout << maxProd << std::endl;
    return 0;
}
```
