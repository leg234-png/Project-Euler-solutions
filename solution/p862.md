# Permutation de chiffres plus grande

## Description du Problème
Soit T(n) le nombre de permutations des chiffres de n qui sont strictement supérieures à n.

Par exemple, T(2302) = 4 car les permutations 3022, 3202, 3220, 3202 sont toutes supérieures à 2302.

Définissons aussi S(k) comme la somme de T(n) pour tous les nombres n à k chiffres. On vous donne S(3) = 1701.

**Objectif** : Trouver S(12). Donner votre réponse modulo 1123455689.

---

## Exemples
### Exemple 1
**Entrée** : `n = 2302`
**Sortie** : `4`
**Explication** : T(2302) = 4.

## Approche de Résolution

### Stratégie
1. Il y a (n+9)C9 chaînes croissantes (nombres avec chiffres non décroissants).
2. Penser aux étoiles et barres : placer 9 barres parmi n chiffres.
3. Pour chaque possibilité, compter le nombre de façons de permuter les chiffres.
4. Pour chaque possibilité x de longueur n, soit d_i = nombre de chiffres i dans x.
5. Compter le nombre de façons de permuter tous les chiffres sauf 0 : (d₁+...+d₉)!/∏d_i!.
6. Placer les 0 entre eux en excluant le début : C(n-1, d₀).
7. Total = (n-d₀)! × (n-1)! / ((n-1-d₀)! × ∏d_i!).

### Observations
- On peut utiliser les combinaisons pour compter efficacement.
- Il faut gérer les permutations avec répétitions.
- On peut optimiser en utilisant des factorielles précalculées.

### Complexité
- **Temporelle** : O((n+9)C9 × n) pour toutes les possibilités.
- **Spatiale** : O(n) pour les factorielles.
