# Semiprimes

## Description du Problème
Un nombre semi-premier est un nombre qui est le produit de exactement deux nombres premiers (pas nécessairement distincts).

Par exemple, `4 = 2 × 2`, `6 = 2 × 3`, `9 = 3 × 3`, `10 = 2 × 5`, `14 = 2 × 7`, `15 = 3 × 5`, `21 = 3 × 7`, `22 = 2 × 11`, `25 = 5 × 5`, `26 = 2 × 13` sont les 10 premiers nombres semi-premiers.

**Objectif** : Combien de nombres semi-premiers sont inférieurs à 10^8 ?

---

## Exemples
### Exemple 1
**Entrée** : `limit = 30`
**Sortie** : `10`
**Explication** : Les nombres semi-premiers < 30 sont : 4, 6, 9, 10, 14, 15, 21, 22, 25, 26.

---

## Approche de Résolution

### Stratégie
1. Générer tous les nombres premiers jusqu'à `LIMIT/2`.
2. Pour chaque nombre premier `p` :
   - Trouver tous les nombres premiers `q` tels que `q >= p` et `p × q <= LIMIT`.
   - Compter ces paires `(p, q)`.
3. Utiliser la recherche binaire pour trouver efficacement la borne supérieure.

### Observations
- On peut s'arrêter quand `p² > LIMIT`.
- Pour chaque `p`, on cherche tous les `q >= p` tels que `q <= LIMIT/p`.
- On utilise la recherche binaire pour trouver l'index de `LIMIT/p` dans la liste des nombres premiers.

### Complexité
- **Temporelle : O(π(√LIMIT) × log π(LIMIT))**, où π est la fonction de comptage des nombres premiers.
- **Spatiale : O(π(LIMIT/2))**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>

using namespace std;

const int LIMIT = 100000000 - 1;

vector<int> listPrimes(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i)
                isPrime[j] = false;
        }
    }
    
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i])
            primes.push_back(i);
    }
    return primes;
}

int main() {
    int count = 0;
    vector<int> primes = listPrimes(LIMIT / 2);
    int sqrtLimit = sqrt(LIMIT);
    
    for (int i = 0; i < primes.size() && primes[i] <= sqrtLimit; i++) {
        int maxQ = LIMIT / primes[i];
        auto it = upper_bound(primes.begin(), primes.end(), maxQ);
        int end = distance(primes.begin(), it);
        count += end - i;
    }
    
    cout << count << endl;
    return 0;
}
```
