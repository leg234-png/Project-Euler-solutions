# Période d'expansion décimale réciproque

## Description du Problème
Une unité de fraction 1/n a une expansion décimale récurrente si n et 10 sont copremiers. Si n et 10 ne sont pas copremiers, l'expansion décimale se termine.

Soit L(n) la longueur de la période de l'expansion décimale récurrente de 1/n. Par exemple, L(7) = 6 car 1/7 = 0.142857142857...

**Objectif** : Trouver ΣL(n) pour 3 <= n <= 10⁸.

---

## Exemples
### Exemple 1
**Entrée** : `n = 7`
**Sortie** : `6`
**Explication** : 1/7 = 0.142857142857... a une période de longueur 6.

---

## Approche de Résolution

### Stratégie
1. Pour calculer l'expansion décimale de 1/n, utiliser l'algorithme de division longue.
2. L'état x_i = 10^i mod n. L'expansion décimale cycle si et seulement si les états sont les mêmes à deux index.
3. Si n est copremier avec 10, alors par le théorème d'Euler, 10^totient(n) = 1 mod n.
4. La période est le plus petit entier positif k tel que 10^k = 1 mod n, et k divise totient(n).
5. Si n n'est pas copremier avec 10, factoriser les facteurs 2 et 5, puis appliquer la même logique sur la partie restante.
6. Pour chaque puissance de nombre premier p^k (p != 2, 5), calculer sa période.
7. Utiliser le PPCM pour combiner les périodes des différentes puissances de nombres premiers.

### Observations
- On précalcule les puissances de nombres premiers et leurs totients.
- On calcule les périodes pour chaque puissance de nombre premier.
- On combine les périodes en utilisant le PPCM.

### Complexité
- **Temporelle** : O(N log N), où N = 10⁸.
- **Spatiale** : O(N), pour les tableaux de périodes et de facteurs premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>

using namespace std;

const int LIMIT = 100000000;

vector<int> listPrimes(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

long long powMod(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        base = (base * base) % mod;
        exp /= 2;
    }
    return result;
}

int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

int lcm(int x, int y) {
    return x / gcd(x, y) * y;
}

    vector<int> primes = listPrimes(LIMIT);
```
