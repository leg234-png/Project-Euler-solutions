# Prime Subset Sums

## Description du Problème
Soit `S = {2, 3, 5, ..., 4999}` l'ensemble de tous les nombres premiers inférieurs à 5000.

**Objectif** : Trouver la somme de tous les nombres entiers inférieurs à 10^16 qui peuvent être exprimés comme la somme d'un sous-ensemble non vide de `S`, où la somme elle-même est un nombre premier.

---

## Exemples
### Exemple 1
**Entrée** : `S = {2, 3, 5}`
**Sortie** : Somme des sommes premières
**Explication** : Les sous-ensembles possibles : {2}=2, {3}=3, {5}=5, {2,3}=5, {2,5}=7, {3,5}=8, {2,3,5}=10. Les sommes premières sont 2, 3, 5, 7. Somme = 17.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique pour compter le nombre de façons d'obtenir chaque somme.
2. `numSubsets[i]` : nombre de façons d'obtenir la somme `i` en utilisant les nombres premiers vus jusqu'à présent.
3. Pour chaque nombre premier `p` :
   - Pour chaque somme `j` de `maxSum` à `p` (en ordre décroissant) :
     - `numSubsets[j] += numSubsets[j - p]`.
4. Après avoir traité tous les nombres premiers, additionner `numSubsets[i]` pour tous les `i` premiers.

### Observations
- On doit itérer en ordre décroissant pour éviter de compter deux fois.
- On utilise l'arithmétique modulaire pour gérer les grands nombres.

### Complexité
- **Temporelle : O(π(LIMIT) × S)**, où S est la somme maximale possible.
- **Spatiale : O(S)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

const int LIMIT = 5000;
const long long MODULUS = 10000000000000000LL;

vector<bool> listPrimality(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i)
                isPrime[j] = false;
        }
    }
    return isPrime;
}

int main() {
    vector<bool> isPrime = listPrimality(LIMIT * LIMIT / 2);
    vector<long long> numSubsets(LIMIT * LIMIT / 2, 0);
    numSubsets[0] = 1;
    int maxSum = 0;
    
    for (int i = 0; i < LIMIT; i++) {
        if (!isPrime[i])
            continue;
        maxSum += i;
        for (int j = maxSum; j >= i; j--) {
            long long temp = numSubsets[j] + numSubsets[j - i];
            if (temp < MODULUS)
                numSubsets[j] = temp;
            else
                numSubsets[j] = temp - MODULUS;
        }
    }
    
    long long sum = 0;
    for (int i = 0; i < numSubsets.size(); i++) {
        if (isPrime[i])
            sum = (sum + numSubsets[i]) % MODULUS;
    }
    
    cout << sum << endl;
    return 0;
}
```
