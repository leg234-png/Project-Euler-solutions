# Maximum de la fonction totient

## Description du Problème
La fonction totient d'Euler, φ(n) [parfois appelée la fonction phi], est définie comme le nombre d'entiers positifs n'excédant pas n qui sont relativement premiers à n. Par exemple, comme 1, 2, 4, 5, 7, et 8, sont tous inférieurs ou égaux à neuf et relativement premiers à neuf, φ(9) = 6.

| n | Relativement Premier | φ(n) | n/φ(n) |
|---|---------------------|------|--------|
| 2 | 1                   | 1    | 2      |
| 3 | 1,2                 | 2    | 1.5    |
| 4 | 1,3                 | 2    | 2      |
| 5 | 1,2,3,4             | 4    | 1.25   |
| 6 | 1,5                 | 2    | 3      |
| 7 | 1,2,3,4,5,6         | 6    | 1.1666... |
| 8 | 1,3,5,7             | 4    | 2      |
| 9 | 1,2,4,5,7,8         | 6    | 1.5    |
| 10 | 1,3,7,9            | 4    | 2.5    |

On peut voir que n = 6 produit un maximum n/φ(n) pour n ≤ 10.

**Objectif** : Trouver la valeur de n ≤ 1 000 000 pour laquelle n/φ(n) est un maximum.

---

## Exemples
### Exemple 1
**Entrée** : n ≤ 10
**Sortie** : `6`
**Explication** : Pour n = 6, n/φ(n) = 6/2 = 3, qui est le maximum pour n ≤ 10.

---

## Approche de Résolution

### Stratégie
1. Pré-calculer tous les totients jusqu'à 1 000 000 en utilisant un crible.
2. Pour chaque n, calculer le ratio n/φ(n).
3. Garder la valeur de n qui donne le ratio maximum.
4. Retourner cette valeur.

### Observations
- Le ratio n/φ(n) est maximisé quand n a beaucoup de facteurs premiers petits.
- En fait, n/φ(n) = ∏(p/(p-1)) pour tous les facteurs premiers p de n.
- Pour maximiser ce produit, on veut que n soit le produit de tous les petits nombres premiers jusqu'à ce que le produit dépasse la limite.
- Cependant, on peut simplement itérer et comparer les ratios.

### Complexité
- **Temporelle** : O(n log log n), pour le crible d'Eratosthène modifié pour calculer les totients.
- **Spatiale** : O(n), pour stocker les totients.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

vector<int> listTotients(int n) {
    vector<int> totients(n + 1);
    for (int i = 0; i <= n; i++)
        totients[i] = i;
    
    for (int i = 2; i <= n; i++) {
        if (totients[i] == i) {  // i est premier
            for (int j = i; j <= n; j += i)
                totients[j] -= totients[j] / i;
        }
    }
    return totients;
}

int main() {
    const int LIMIT = 1000000;
    vector<int> totients = listTotients(LIMIT);
    
    int maxNumer = 0;
    int maxDenom = 1;
    
    for (int n = 1; n < (int)totients.size(); n++) {
        if ((long long)n * maxDenom > (long long)maxNumer * totients[n]) {
            maxNumer = n;
            maxDenom = totients[n];
        }
    }
    
    cout << maxNumer << endl;
    return 0;
}
```
