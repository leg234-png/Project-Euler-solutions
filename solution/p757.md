# Nombres furtifs

## Description du Problème
Un nombre positif N est appelé furtif s'il existe des entiers positifs a, b, c, d tels que :
- N = ab = cd
- a + b = c + d + 1

Par exemple, 36 = 4×9 = 6×6 et 4+9 = 6+6+1, donc 36 est furtif.

On vous donne aussi qu'il y a 2851 nombres furtifs ne dépassant pas 10⁶.

**Objectif** : Combien y a-t-il de nombres furtifs qui ne dépassent pas 10¹⁴ ?

---

## Exemples
### Exemple 1
**Entrée** : `N = 36`
**Sortie** : `36 est furtif`
**Explication** : 36 = 4×9 = 6×6 et 4+9 = 6+6+1.

---

## Approche de Résolution

### Stratégie
1. N = ab = cd, a + b = c + d + 1.
2. Soit a < b, c, d, x = c - a, y = d - a.
3. xy = N - a(c+d) + a² = N - a(a+b-1) + a² = N - ab + a = a.
4. (x+1)(y+1) = xy + x + y + 1 = a + c - a + d - a + 1 = c + d + 1 - a = b.
5. x(y+1) = xy + x = c.
6. y(x+1) = d.
7. Donc N = x(x+1)y(y+1). Ce sont des nombres biproniques.
8. Pour un x donné, y peut prendre des valeurs y² + y < 10¹⁴/(x(x+1)).
9. Générer tous les nombres biproniques ≤ 10¹⁴.

### Observations
- Les nombres furtifs sont exactement les nombres biproniques.
- On peut générer efficacement tous les nombres biproniques.
- Il y a une borne supérieure claire pour x et y.

### Complexité
- **Temporelle** : O(√N × log N) pour générer tous les nombres biproniques.
- **Spatiale** : O(√N) pour stocker les nombres uniques.

---

## Solution en C++

```cpp
#include <iostream>
#include <set>
#include <cmath>

using namespace std;

long long compute(long long limit) {
    set<long long> stealthy;
    
    for (long long x = 1; x <= sqrt(limit); x++) {
        long long max_y = (sqrt(4.0 * limit / (x * (x + 1)) + 1) - 1) / 2;
        
        for (long long y = x; y <= max_y; y++) {
            long long n = x * (x + 1) * y * (y + 1);
            if (n <= limit) {
                stealthy.insert(n);
            } else {
                break;
            }
        }
    }
    
    return stealthy.size();
}

int main() {
    long long limit = 1;
    for (int i = 0; i < 14; i++) {
        limit *= 10;
    }
    cout << compute(limit) << endl;
    return 0;
}
```
