# Compter les Dimanches

## Description du Problème
Vous recevez les informations suivantes :

- Le 1er janvier 1900 était un lundi.
- Trente jours ont septembre, avril, juin et novembre.
- Tous les autres ont trente-et-un, sauf février seul, qui a vingt-huit, qu'il pleuve ou qu'il vente.
- Et les années bissextiles, vingt-neuf.
- Une année bissextile se produit sur toute année divisible par 4, mais pas sur un siècle sauf s'il est divisible par 400.

**Objectif** : Combien de dimanches sont tombés le premier du mois pendant le vingtième siècle (1er janvier 1901 au 31 décembre 2000) ?

---

## Exemples
### Exemple 1
**Entrée** : Période 1901-2000
**Sortie** : `171`
**Explication** : Le nombre de dimanches tombant le 1er du mois.

---

## Approche de Résolution

### Stratégie
1. Utiliser la congruence de Zeller pour calculer le jour de la semaine d'une date donnée.
2. Pour chaque mois de 1901 à 2000, vérifier si le 1er du mois est un dimanche.
3. Compter le nombre de dimanches trouvés.

### Observations
- La congruence de Zeller permet de calculer le jour de la semaine pour une date donnée.
- Formule : h = (q + ⌊(13(m+1))/5⌋ + K + ⌊K/4⌋ + ⌊J/4⌋ - 2J) mod 7
  - h : jour de la semaine (0 = samedi, 1 = dimanche, ..., 6 = vendredi)
  - q : jour du mois
  - m : mois (3 = mars, 4 = avril, ..., 14 = février)
  - K : année du siècle (année mod 100)
  - J : siècle (année / 100)
- Pour février, on utilise m = 14 et on décrémente l'année.

### Complexité
- **Temporelle** : O(n × 12), où `n` = 100 (nombre d'années).
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>

int dayOfWeek(int year, int month, int day) {
    if (month < 3) {
        month += 12;
        year--;
    }
    int k = year % 100;
    int j = year / 100;
    int h = (day + (13 * (month + 1)) / 5 + k + k / 4 + j / 4 - 2 * j) % 7;
    return (h + 5) % 7;  // Convertir : 0 = dimanche, 1 = lundi, ..., 6 = samedi
}

int main() {
    int count = 0;
    for (int y = 1901; y <= 2000; ++y) {
        for (int m = 1; m <= 12; ++m) {
            if (dayOfWeek(y, m, 1) == 0) {  // Dimanche
                count++;
            }
        }
    }
    std::cout << count << std::endl;
    return 0;
}
```
