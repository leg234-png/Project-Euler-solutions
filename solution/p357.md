# Nombres générateurs de nombres premiers

## Description du Problème
Considérons les diviseurs de 30 : 1, 2, 3, 5, 6, 10, 15, 30. On peut voir que pour chaque diviseur d de 30, d + 30/d est premier.

**Objectif** : Trouver la somme de tous les entiers positifs n <= 10⁸ tels que pour chaque diviseur d de n, d + n/d est premier.

---

## Exemples
### Exemple 1
**Entrée** : `n = 30`
**Sortie** : `30` (inclus)
**Explication** : Pour chaque diviseur d de 30, d + 30/d est premier :
- 1 + 30/1 = 31 (premier)
- 2 + 30/2 = 17 (premier)
- 3 + 30/3 = 13 (premier)
- 5 + 30/5 = 11 (premier)
- 6 + 30/6 = 11 (premier)
- 10 + 30/10 = 13 (premier)
- 15 + 30/15 = 17 (premier)
- 30 + 30/30 = 31 (premier)

---

## Approche de Résolution

### Stratégie
1. Précalculer un tableau de primalité pour [1, 10⁸ + 1].
2. Pour chaque entier n de 0 à 10⁸ :
   - Vérifier d'abord si n + 1 est premier (optimisation).
   - Si oui, vérifier tous les diviseurs d dans [1, √n] :
     - Si d divise n, vérifier si d + n/d est premier.
     - Si non, n n'est pas générateur de nombres premiers.
3. Si tous les diviseurs satisfont la condition, ajouter n à la somme.

### Observations
- La valeur maximale de f(d) = d + n/d pour un diviseur d de n est n + 1 (atteinte aux extrémités d = {1, n}).
- On n'a besoin de vérifier que les diviseurs dans [1, √n] car f(d) = f(n/d).
- On peut optimiser en vérifiant d'abord si n + 1 est premier.

### Complexité
- **Temporelle** : O(N × log N), où N = 10⁸, pour le crible et la vérification.
- **Spatiale** : O(N), pour le tableau de primalité.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

const int LIMIT = 100000000;

vector<bool> listPrimality(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    return isPrime;
}

int sqrt(int n) {
    return (int)std::sqrt(n);
}

bool isPrimeGenerating(int n, const vector<bool>& isPrime) {
    for (int i = 1, end = sqrt(n); i <= end; i++) {
        if (n % i == 0 && !isPrime[i + n / i]) {
            return false;
        }
    }
    return true;
}

int main() {
    vector<bool> isPrime = listPrimality(LIMIT + 1);
    long long sum = 0;
    
    for (int n = 0; n <= LIMIT; n++) {
        if (isPrime[n + 1] && isPrimeGenerating(n, isPrime)) {
            sum += n;
        }
    }
    
    cout << sum << endl;
    return 0;
}
```
