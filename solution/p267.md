# Billionaire

## Description du Problème
Vous commencez avec un capital de 1 £. Vous pouvez parier une fraction `f` de votre capital à chaque lancer de pièce. Si vous gagnez, votre capital est multiplié par `(1 + 2f)`. Si vous perdez, votre capital est multiplié par `(1 - f)`.

Vous jouez 1000 lancers de pièce.

**Objectif** : Quelle est la probabilité maximale de finir avec au moins 1 000 000 000 £ ? Donner la réponse arrondie à 12 décimales.

---

## Exemples
### Exemple 1
**Entrée** : `trials = 10, target = 100`
**Sortie** : Probabilité maximale
**Explication** : On cherche la fraction optimale à parier.

---

## Approche de Résolution

### Stratégie
1. Pour une fraction `f` et `w` victoires sur `n` lancers :
   - Capital final = `(1 + 2f)^w × (1 - f)^(n - w)`.
2. Utiliser un échantillonnage heuristique pour trouver la fraction optimale :
   - Niveau 1 : tester `f = 1/2`.
   - Niveau 2 : tester `f = 1/4, 3/4`.
   - Niveau k : tester `f = (2i+1)/2^k` pour `i = 0, 1, ..., 2^(k-1)-1`.
3. Pour chaque `f`, trouver le nombre minimum de pertes `i` tel que le capital final < 10^9.
4. La probabilité = somme des coefficients binomiaux `C(n, k)` pour `k < i` divisée par `2^n`.

### Observations
- Le résultat est indépendant de l'ordre des victoires et défaites.
- On utilise l'échantillonnage binaire pour trouver la fraction optimale.
- La probabilité est calculée avec des coefficients binomiaux.

### Complexité
- **Temporelle : O(levels × 2^levels × n)`.
- **Spatiale : O(1)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <cmath>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

const int TRIALS = 1000;

int calcBillionaireProbability(double betProportion) {
    double initCapital = 1.0;
    int i;
    for (i = 0; i <= TRIALS; i++) {
        double finalCapital = initCapital * pow(1 - betProportion, i) * pow(1 + betProportion * 2, TRIALS - i);
        if (finalCapital < 1e9)
            break;
    }
    return i;
}

cpp_int binomial(int n, int k) {
    if (k > n - k) k = n - k;
    cpp_int result = 1;
    for (int i = 0; i < k; i++)
        result = result * (n - i) / (i + 1);
    return result;
}

int main() {
    int maxIndex = -1;
    int prevChangeLevel = 1;
    
    for (int level = 1; level - prevChangeLevel <= 8; level++) {
        double scaler = pow(2, -level);
        for (int i = 1, end = 1 << level; i < end; i += 2) {
            int index = calcBillionaireProbability(i * scaler);
            if (index > maxIndex) {
                maxIndex = index;
                prevChangeLevel = level;
            }
        }
    }
    
    cpp_int binomialSum = 0;
    for (int i = 0; i < maxIndex; i++)
        binomialSum += binomial(TRIALS, i);
    
    cpp_int denominator = cpp_int(1) << TRIALS;
    double prob = (double)binomialSum / (double)denominator;
    
    cout << fixed << setprecision(12) << prob << endl;
    return 0;
}
```
