# Milliardaire

## Description du Problème
Votre retour est le double de votre mise pour les têtes et vous perdez votre mise pour les queues.

Par exemple, si f = 1/4, pour le premier lancer vous misez £0.25, et si les têtes apparaissent vous gagnez £0.5 et avez donc £1.5. Vous misez ensuite £0.375 et si le second lancer est queue, vous avez £1.125.

**Objectif** : En choisissant f pour maximiser vos chances d'avoir au moins £1,000,000,000 après 1000 lancers, quelle est la chance que vous deveniez milliardaire ?

Tous les calculs sont supposés être exacts (pas d'arrondi), mais donnez votre réponse arrondie à 12 chiffres après la virgule sous la forme 0.abcdefghijkl.

---

## Exemples
### Exemple 1
**Entrée** : `f = 1/4, 2 lancers`
**Sortie** : `Probabilité`
**Explication** : La probabilité d'avoir au moins £1,000,000,000 après 2 lancers.

---

## Approche de Résolution

### Stratégie
1. Noter que l'ordre des gains et pertes n'a pas d'importance.
2. Si on obtient h têtes, les gains totaux sont (1 + 2f)^h × (1 - f)^(1000-h).
3. Trouver f qui maximise les chances d'atteindre un milliard.
4. Trouver le nombre minimum de têtes h nécessaire pour atteindre un milliard avec le f optimal.
5. Calculer la probabilité d'obtenir au moins h têtes sur 1000 lancers.

### Observations
- La fonction find_f() confirme que f ≈ 0.146883922440941... prendrait 431.26 têtes pour atteindre un milliard.
- On a besoin d'au moins 432 têtes pour atteindre un milliard.
- La solution est la somme de h = 432 à 1000 de C(1000, h) / 2^1000.

### Complexité
- **Temporelle** : O(1000) pour calculer les combinaisons.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <cmath>

using namespace std;

double winnings(int heads, double f) {
    return pow(1 + 2*f, heads) * pow(1 - f, 1000 - heads);
}

pair<double, int> find_f() {
    double f = 0.001;
    double goal = 1e9;
    double step = 0.001;
    double best_f = 0;
    int corresponding_x = 1000;
    
    while (f < 0.5) {
        int x = 1;
        while (winnings(x, f) < goal) {
            x++;
        }
        if (x < corresponding_x) {
            best_f = f;
            corresponding_x = x;
        }
        f += step;
    }
    return {best_f, corresponding_x};
}

double n_choose_r(int n, int r) {
    if (r > n) return 0;
    double result = 1;
    for (int i = 0; i < r; i++) {
        result = result * (n - i) / (i + 1);
    }
    return result;
}

double compute() {
    auto [best_f, min_heads] = find_f();
    int required_heads = min_heads;
    
    double total_prob = 0;
    double prob_per_flip = 1.0 / pow(2, 1000);
    
    for (int h = required_heads; h <= 1000; h++) {
        total_prob += n_choose_r(1000, h) * prob_per_flip;
    }
    
    return total_prob;
}

int main() {
    cout << fixed << setprecision(12) << compute() << endl;
    return 0;
}
```
