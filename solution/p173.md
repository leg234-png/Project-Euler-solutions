# Using up to one million tiles how many different "hollow" square laminae can be formed?

## Description du Problème
On peut former des lamelles carrées creuses en plaçant des carreaux autour d'un carré central. Par exemple, en utilisant exactement 32 carreaux, on peut former une lamelle carrée creuse de 6×6 avec un trou central de 2×2.

**Objectif** : Combien de lamelles carrées creuses distinctes peuvent être formées en utilisant jusqu'à un million de carreaux ?

---

## Exemples
### Exemple 1
**Entrée** : `tiles = 32`
**Sortie** : Nombre de lamelles possibles
**Explication** : Une lamelle 6×6 avec un trou 2×2 utilise 36 - 4 = 32 carreaux.

---

## Approche de Résolution

### Stratégie
1. Pour chaque taille de carré extérieur `n` (à partir de 3) :
   - Pour chaque taille de trou intérieur `k` (de `n-2` à 1, en décrémentant de 2 pour garder la parité) :
     - Nombre de carreaux = `n² - k²`.
     - Si ce nombre dépasse `TILES`, arrêter.
     - Sinon, incrémenter le compteur.
2. Le trou intérieur doit avoir la même parité que le carré extérieur pour que la lamelle soit symétrique.

### Observations
- Le nombre de carreaux = `n² - k²` où `n` est la taille extérieure et `k` est la taille du trou.
- `k` doit être inférieur à `n` d'au moins 2.
- `k` et `n` doivent avoir la même parité.

### Complexité
- **Temporelle : O(√TILES × √TILES) = O(TILES)**.
- **Spatiale : O(1)**.

---

## Solution en C++

```cpp
#include <iostream>

using namespace std;

const int TILES = 1000000;

int main() {
    int count = 0;
    for (int n = 3; n <= TILES / 4 + 1; n++) {  // Outer square length
        for (int k = n - 2; k >= 1; k -= 2) {   // Inner square length
            long long tiles = (long long)n * n - (long long)k * k;
            if (tiles > TILES)
                break;
            count++;
        }
    }
    cout << count << endl;
    return 0;
}
```
