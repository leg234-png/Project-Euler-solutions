# Une séquence de Collatz modifiée

## Description du Problème
Étant donné un entier, nous pouvons lister la séquence d'étapes.

Par exemple, si a₁=231, alors la séquence {aₙ}={231,77,51,17,11,7,10,14,9,3,1} correspond aux étapes "DdDddUUdDD".

Bien sûr, il y a d'autres séquences qui commencent par cette même séquence "DdDddUUdDD....".

**Objectif** : Étant donné une séquence d'étapes, trouver le plus petit entier positif qui commence cette séquence.

---

## Exemples
### Exemple 1
**Entrée** : `seq = "DdDddUUdDD"`
**Sortie** : `231`
**Explication** : 231 est le plus petit entier qui commence cette séquence.

---

## Approche de Résolution

### Stratégie
1. Les règles sont :
   - a_{n+1} = a_n/3 si a_n ≡ 0 mod 3 (D)
   - a_{n+1} = (4a_n + 2)/3 si a_n ≡ 1 mod 3 (U)
   - a_{n+1} = (2a_n - 1)/3 si a_n ≡ 2 mod 3 (d)
2. Travailler à rebours depuis la fin de la séquence.
3. Pour chaque étape, déterminer les valeurs possibles de a_n en fonction de a_{n+1}.
4. Trouver le plus petit entier positif qui satisfait toutes les contraintes.

### Observations
- On peut résoudre récursivement en partant de la fin.
- Pour chaque étape, il y a généralement 3 valeurs possibles pour a_n.
- On doit vérifier que chaque valeur est un entier positif.

### Complexité
- **Temporelle** : O(3^L) dans le pire cas, où L est la longueur de la séquence.
- **Spatiale** : O(L) pour la récursion.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <climits>

using namespace std;

long long seq_finder(long long num, const string& mask, int count) {
    if (mask.empty()) {
        return count > 0 ? num : LLONG_MAX;
    }
    
    char first = mask[0];
    string remaining = mask.substr(1);
    
    if (first == 'D') {
        if (num % 3 == 0) {
            return seq_finder(num / 3, remaining, count + 1);
        } else {
            return LLONG_MAX;
        }
    } else if (first == 'U') {
        if (num % 3 == 1) {
            return seq_finder((4 * num + 2) / 3, remaining, count + 1);
        } else {
            return LLONG_MAX;
        }
    } else if (first == 'd') {
        if (num % 3 == 2) {
            return seq_finder((2 * num - 1) / 3, remaining, count + 1);
        } else {
            return LLONG_MAX;
        }
    }
    
    return LLONG_MAX;
}

long long compute(const string& seq) {
    long long limit = 1;
    for (int i = 0; i < seq.length(); i++) {
        limit *= 3;
    }
    
    long long min_start = LLONG_MAX;
    
    for (long long x = limit; x < limit + 3; x++) {
        char first_move = seq[0];
        long long start = 0;
        
        if (first_move == 'D' && x % 3 == 0) {
            start = x;
        } else if (first_move == 'U' && x % 3 == 1) {
            start = x;
        } else if (first_move == 'd' && x % 3 == 2) {
            start = x;
        }
        
        if (start > 0) {
            long long result = seq_finder(start, seq, 0);
            if (result != LLONG_MAX) {
                min_start = min(min_start, result);
            }
        }
    }
    
    return min_start;
}

int main() {
    string seq = "UDDDUdddDDUDDddDdDdDDUDDdUUDd";
    cout << compute(seq) << endl;
    return 0;
}
```
