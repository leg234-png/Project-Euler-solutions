# Rassemblement des haricots

## Description du Problème
Avec 5 bols, il faut à Peter 15 coups pour revenir à la situation initiale.

Soit M(x) représenter le nombre de coups requis pour revenir à la situation initiale, en commençant avec x bols. Ainsi, M(5)=15. On peut aussi vérifier que M(100)=10920.

**Objectif** : Trouver ∑ₖ₌₀¹⁰¹⁸ M(2ᵏ+1). Donner votre réponse modulo 7⁹.

---

## Exemples
### Exemple 1
**Entrée** : `x = 5`
**Sortie** : `15`
**Explication** : M(5) = 15.

---

## Approche de Résolution

### Stratégie
1. Observer les premières valeurs de M(n) pour détecter un pattern.
2. Observer que M(2ᵏ+1) suit un pattern spécifique.
3. Les 10 premiers termes de M(2ᵏ+1) sont : [2, 5, 15, 53, 207, 845, 3495, 14453, 59487, 243485].
4. Soustraire 2^(k+1) : [0, 1, 7, 37, 175, 781, 3367, 14197, 58975, 242461] = 4ᵏ - 3ᵏ (séquence OEIS A005061).
5. Donc M(2ᵏ+1) = 2^(k+1) - 3ᵏ + 4ᵏ.
6. Sommer cette formule pour k de 0 à 10¹⁸.

### Observations
- Il y a un pattern clair avec les puissances de 2.
- La formule M(2ᵏ+1) = 2^(k+1) - 3ᵏ + 4ᵏ peut être prouvée.
- On peut utiliser l'exponentiation modulaire rapide pour calculer les grandes puissances.

### Complexité
- **Temporelle** : O(log k) pour chaque terme, où k = 10¹⁸.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long geometric_sum(long long r, long long n, long long mod) {
    if (r == 1) {
        return n % mod;
    }
    // S = (r^(n+1) - 1) / (r - 1)
    long long numerator = (mod_pow(r, n + 1, mod) - 1 + mod) % mod;
    long long denominator = (r - 1) % mod;
    // Calculer l'inverse modulaire de denominator
    long long inv = mod_pow(denominator, mod - 2, mod);
    return (numerator * inv) % mod;
}

long long compute(long long limit, long long mod) {
    // ∑(2^(k+1) - 3^k + 4^k) pour k de 0 à limit
    // = 2∑2^k - ∑3^k + ∑4^k
    // = 2(2^(limit+1) - 1) - (3^(limit+1) - 1)/2 + (4^(limit+1) - 1)/3
    
    long long sum_2 = geometric_sum(2, limit, mod);
    long long sum_3 = geometric_sum(3, limit, mod);
    long long sum_4 = geometric_sum(4, limit, mod);
    
    long long total = (2 * sum_2 - sum_3 + sum_4 + mod) % mod;
    return total;
}

int main() {
    long long mod = 7 * 7 * 7 * 7 * 7 * 7 * 7 * 7 * 7; // 7^9
    cout << compute(1000000000000000000LL, mod) << endl;
    return 0;
}
```
