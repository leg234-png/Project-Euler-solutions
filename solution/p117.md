# Red, Green, and Blue Tiles

## Description du Problème
En utilisant une combinaison de carreaux gris carrés et de carreaux oblongs choisis parmi : carreaux rouges (mesurant deux unités), carreaux verts (mesurant trois unités) et carreaux bleus (mesurant quatre unités), il est possible de carreler une rangée mesurant cinq unités de longueur en exactement quinze façons différentes.

**Objectif** : Combien de façons peut-on carreler une rangée mesurant cinquante unités de longueur ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 5`
**Sortie** : `15`
**Explication** : Il y a 15 façons de carreler une rangée de 5 unités.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique.
2. ways[n] = nombre de façons de carreler une rangée de n unités.
3. Cas de base : ways[0] = 1.
4. Pour n ≥ 1 :
   - ways[n] = ways[n-1] + ways[n-2] + ways[n-3] + ways[n-4]
   - Correspondant à : carré gris (1), rouge (2), vert (3), bleu (4)

### Observations
- Contrairement au problème 116, les couleurs peuvent être mélangées.
- On peut utiliser n'importe quelle combinaison de carreaux.

### Complexité
- **Temporelle** : O(n), où n=50.
- **Spatiale** : O(n).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

const int LENGTH = 50;

int main() {
    std::vector<long long> ways(LENGTH + 1);
    ways[0] = 1;
    
    for (int n = 1; n <= LENGTH; n++) {
        for (int k = 1; k <= 4 && k <= n; k++) {
            ways[n] += ways[n - k];
        }
    }
    
    std::cout << ways[LENGTH] << std::endl;
    return 0;
}
```
