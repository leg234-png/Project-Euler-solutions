# Jeu de glissement

## Description du Problème
Soit S(m,n) représenter le nombre minimum de coups pour terminer le jeu sur une grille m par n. Par exemple, on peut vérifier que S(5,4)=25.

Il y a exactement 5482 grilles pour lesquelles S(m,n)=p², où p<100 est premier.

**Objectif** : Combien de grilles y a-t-il telles que S(m,n)=p², où p<10⁶ est premier ?

---

## Exemples
### Exemple 1
**Entrée** : `m=5, n=4`
**Sortie** : `25`
**Explication** : S(5,4) = 25.

---

## Approche de Résolution

### Stratégie
1. Observer les motifs récursifs dans les valeurs de S(m,n).
2. Les patterns émergent :
   - S(n, n) = 8 + S(n-1, n-1), S(2,2) = 5
   - S(m, n+1) = 6 + S(m, n) si n > m + 1
   - S(m, n+1) = 4 + S(m, n) si n = m + 1
3. Combiner ces formules :
   - S(m, n) = 8*m - 11 si m = n
   - S(m, n) = 6*n + 2*m - 13 si n > m
4. Pour S(m,n) = p², on a p² = 6n + 2m - 13.
5. Résoudre pour m et n en fonction de p.

### Observations
- Le problème a une structure récursive car avec des coups optimaux on peut réduire la grille.
- Les formules permettent de calculer S(m,n) directement sans simulation.
- On doit trouver tous les p premiers < 10⁶ et vérifier quels (m,n) donnent S(m,n) = p².

### Complexité
- **Temporelle** : O(P) où P est le nombre de nombres premiers < 10⁶.
- **Spatiale** : O(√limit) pour le crible de nombres premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<bool> prime_sieve(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    return is_prime;
}

long long compute(int limit) {
    vector<bool> is_prime = prime_sieve(limit);
    long long count = 0;
    
    for (int p = 2; p < limit; p++) {
        if (!is_prime[p]) continue;
        
        long long p_sq = (long long)p * p;
        
        // Pour n > m : p² = 6n + 2m - 13
        // => 2m = p² - 6n + 13
        // => m = (p² - 6n + 13) / 2
        // On doit avoir m > 0 et n > m
        
        for (long long n = 1; n < p_sq; n++) {
            long long numerator = p_sq - 6 * n + 13;
            if (numerator <= 0) break;
            if (numerator % 2 != 0) continue;
            
            long long m = numerator / 2;
            if (m > 0 && n > m) {
                count++;
            }
        }
        
        // Pour m = n : p² = 8m - 11
        // => m = (p² + 11) / 8
        long long m_eq = (p_sq + 11);
        if (m_eq % 8 == 0) {
            m_eq /= 8;
            if (m_eq > 0) {
                count++;
            }
        }
    }
    
    return count;
}

int main() {
    cout << compute(1000000) << endl;
    return 0;
}
```
