# Plus grand nombre premier

## Description du Problème
Considérons la séquence n² + k² avec n ≥ 1.

Si on écrit les premiers termes de cette séquence, on obtient 4, 7, 12, 19, 28, 39, 52, 67, 84, 103, 124, 147, 172, 199, 228, 259, 292, 327, 364, ....

On voit que les termes pour n=6 et n=7 (39 et 52) sont tous deux divisibles par 13. En fait, 13 est le plus grand nombre premier qui divise deux termes successifs de cette séquence.

Soit P(k) le plus grand nombre premier qui divise deux termes successifs de la séquence n² + k².

**Objectif** : Trouver les 18 derniers chiffres de ∑_{k=1}^{10000000} P(k).

---

## Exemples
### Exemple 1
**Entrée** : `k = 3`
**Sortie** : `13`
**Explication** : P(3) = 13.

---

## Approche de Résolution

### Stratégie
1. Soit q le plus grand nombre premier qui divise n² + k² et (n+1)² + k².
2. q|n² + k² et q|(n+1)² + k² => q|2n+1.
3. q|4(n²+k²) = (2n+1)(2n-1) + 4k² + 1 => q|4k² + 1.
4. Donc q sera le plus grand facteur premier de 4k² + 1.
5. Utiliser un algorithme de type "sieve" pour trouver rapidement le plus grand facteur premier de 4k² + 1.

### Observations
- Le problème se réduit à trouver le plus grand facteur premier de 4k² + 1.
- On peut utiliser un crible modifié pour calculer efficacement.
- Pour chaque k, on doit trouver le plus grand facteur premier.

### Complexité
- **Temporelle** : O(K × log K) où K = 10⁷.
- **Spatiale** : O(K) pour le crible.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long compute(long long limit, long long mod) {
    vector<long long> largest_prime(limit + 1, 0);
    
    for (long long k = 1; k <= limit; k++) {
        long long num = 4 * k * k + 1;
        long long max_prime = 0;
        
        for (long long d = 2; d * d <= num; d++) {
            while (num % d == 0) {
                max_prime = max(max_prime, d);
                num /= d;
            }
        }
        if (num > 1) {
            max_prime = max(max_prime, num);
        }
        
        largest_prime[k] = max_prime;
    }
    
    long long total = 0;
    for (long long k = 1; k <= limit; k++) {
        total = (total + largest_prime[k]) % mod;
    }
    
    return total;
}

int main() {
    long long mod = 1000000000000000000LL; // 10^18
    cout << compute(10000000, mod) << endl;
    return 0;
}
```
