# Somme Maximale de Chemin I

## Description du Problème
En commençant au sommet du triangle ci-dessous et en se déplaçant vers les nombres adjacents de la ligne inférieure, la somme maximale du haut vers le bas est 23.

```
   3
  7 4
 2 4 6
8 5 9 3
```

C'est-à-dire, 3 + 7 + 4 + 9 = 23.

**Objectif** : Trouver la somme maximale du haut vers le bas du triangle donné.

---

## Exemples
### Exemple 1
**Entrée** : Triangle de 4 lignes
**Sortie** : `23`
**Explication** : Le chemin 3 → 7 → 4 → 9 donne la somme maximale de 23.

---

## Approche de Résolution

### Stratégie
Utiliser la programmation dynamique en partant du bas du triangle :
1. Pour chaque cellule, calculer la somme maximale en partant de cette cellule.
2. Si on est à la base, la somme maximale est la valeur de la cellule.
3. Sinon, la somme maximale est la valeur de la cellule plus le maximum entre la cellule en bas à gauche et la cellule en bas à droite.
4. Traiter de bas en haut pour que les valeurs dépendantes soient calculées avant d'être utilisées.

### Observations
- On peut modifier le triangle original en place pour économiser de l'espace.
- On parcourt de la ligne avant-dernière vers le haut.
- Pour chaque cellule à la ligne i, on ajoute le maximum entre triangle[i+1][j] et triangle[i+1][j+1].

### Complexité
- **Temporelle** : O(n²), où `n` est le nombre de lignes du triangle.
- **Spatiale** : O(n²), pour stocker le triangle.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<std::vector<int>> triangle = {
        {75},
        {95, 64},
        {17, 47, 82},
        {18, 35, 87, 10},
        {20,  4, 82, 47, 65},
        {19,  1, 23, 75,  3, 34},
        {88,  2, 77, 73,  7, 63, 67},
        {99, 65,  4, 28,  6, 16, 70, 92},
        {41, 41, 26, 56, 83, 40, 80, 70, 33},
        {41, 48, 72, 33, 47, 32, 37, 16, 94, 29},
        {53, 71, 44, 65, 25, 43, 91, 52, 97, 51, 14},
        {70, 11, 33, 28, 77, 73, 17, 78, 39, 68, 17, 57},
        {91, 71, 52, 38, 17, 14, 91, 43, 58, 50, 27, 29, 48},
        {63, 66,  4, 68, 89, 53, 67, 30, 73, 16, 69, 87, 40, 31},
        { 4, 62, 98, 27, 23,  9, 70, 98, 73, 93, 38, 53, 60,  4, 23}
    };
    
    for (int i = triangle.size() - 2; i >= 0; --i) {
        for (int j = 0; j < triangle[i].size(); ++j) {
            triangle[i][j] += std::max(triangle[i + 1][j], triangle[i + 1][j + 1]);
        }
    }
    
    std::cout << triangle[0][0] << std::endl;
    return 0;
}
```
