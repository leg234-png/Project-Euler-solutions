# Duodigits

## Description du Problème
Un nombre duodigit est un nombre positif qui n'utilise pas plus de deux chiffres distincts. Par exemple, 12, 110 et 33333 sont des nombres duodigits, tandis que 102 n'en est pas un.

Soit d(n) le plus petit nombre duodigit multiple de n.

Par exemple, d(12) = 12, d(102) = 112122.

Soit D(k) = ∑_{n=1}^k d(n). On vous donne D(110) = 11047, D(150) = 53312 et D(500) = 29570988.

**Objectif** : Trouver D(50000). Donner votre réponse en notation scientifique arrondie à 13 chiffres significatifs (12 après la virgule décimale).

---

## Exemples
### Exemple 1
**Entrée** : `n = 12`
**Sortie** : `12`
**Explication** : d(12) = 12.

### Exemple 2
**Entrée** : `n = 102`
**Sortie** : `112122`
**Explication** : d(102) = 112122.

---

## Approche de Résolution

### Stratégie
1. Supposons qu'on ait un nombre n.
2. Soit A = {1, 11, ..., 11...11 (n fois), 11...111 (n+1 fois)}.
3. Pour x dans A, calculer x (mod n).
4. Comme A a n+1 éléments, il y aura deux éléments dans A tels que x₁ = x₂ (mod n) (x₁ > x₂).
5. => x₁ - x₂ = 0 (mod n) et alors x₁ - x₂ = 1111...000 et est divisible par n.
6. C'est-à-dire que x₁ - x₂ est un multiple duodigit de n consistant uniquement de 1 et de 0.
7. Générer tous les nombres duodigits possibles et trouver le plus petit multiple de n.

### Observations
- Le principe des tiroirs garantit l'existence d'un multiple duodigit.
- On peut générer efficacement les nombres duodigits.
- On peut utiliser une recherche BFS pour trouver le plus petit multiple.

### Complexité
- **Temporelle** : O(10^d × n) où d est le nombre de chiffres du résultat.
- **Spatiale** : O(10^d) pour stocker les candidats.

---

## Solution en C++

```cpp
#include <iostream>
#include <queue>
#include <set>
#include <string>
#include <cmath>

using namespace std;

long long d(long long n) {
    queue<long long> q;
    set<long long> seen;
    
    // Générer tous les nombres duodigits possibles
    for (int d1 = 0; d1 <= 9; d1++) {
        for (int d2 = d1; d2 <= 9; d2++) {
            if (d1 == 0 && d2 == 0) continue;
            q.push(d1);
            if (d1 != d2) q.push(d2);
        }
    }
    
    while (!q.empty()) {
        long long curr = q.front();
        q.pop();
        
        if (curr % n == 0 && curr > 0) {
            return curr;
        }
        
        if (curr > 1e15) continue; // Limite pour éviter l'overflow
        
        // Ajouter les prochains nombres duodigits
        string s = to_string(curr);
        set<char> digits(s.begin(), s.end());
        
        for (char c : digits) {
            long long next = curr * 10 + (c - '0');
            if (seen.find(next) == seen.end()) {
                seen.insert(next);
                q.push(next);
            }
        }
    }
    
    return 0;
}

double D(int k) {
    long long total = 0;
    for (int n = 1; n <= k; n++) {
        total += d(n);
    }
    return (double)total;
}

int main() {
    double result = D(50000);
    cout << scientific << setprecision(12) << result << endl;
    return 0;
}
```
