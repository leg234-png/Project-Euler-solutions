# Searching for a Maximum-Sum Subsequence

## Description du Problème
En regardant le tableau ci-dessous, on peut voir que le maximum de la somme de n'importe quel sous-tableau horizontal ou vertical adjacent est 16 (= 8 + 7 + 1).

Cependant, si l'on permet la recherche dans les trois directions (horizontale, verticale et diagonale), la somme maximale est 24 (= 8 + 7 + 4 + 5).

En utilisant le générateur de nombres aléatoires "Lagged Fibonacci Generator" décrit ci-dessous, créer un tableau de 2000 × 2000, puis trouver la somme maximale de n'importe quel sous-tableau (contigu) dans n'importe quelle direction (horizontale, verticale, diagonale ou anti-diagonale).

**Objectif** : Trouver la somme maximale de n'importe quel sous-tableau contigu dans toutes les directions.

---

## Exemples
### Exemple 1
**Entrée** : Tableau 3×3
**Sortie** : `24`
**Explication** : La somme maximale est 24 dans une direction diagonale.

---

## Approche de Résolution

### Stratégie
1. Générer le tableau 2000×2000 en utilisant le générateur Lagged Fibonacci.
2. Pour chaque ligne, colonne, diagonale et anti-diagonale, trouver la somme maximale de sous-tableau contigu.
3. Utiliser l'algorithme de Kadane pour trouver la somme maximale de sous-tableau.
4. Retourner le maximum parmi toutes les directions.

### Observations
- L'algorithme de Kadane trouve efficacement la somme maximale de sous-tableau en O(n).
- On doit scanner dans toutes les directions : horizontale, verticale, diagonale, anti-diagonale.

### Complexité
- **Temporelle** : O(n²), où n = 2000.
- **Spatiale** : O(n²) pour stocker le tableau.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

class LfgRandom {
private:
    std::vector<int> history;
    int index;
    int k;
    
public:
    LfgRandom() : k(1), history(55), index(0) {}
    
    int next() {
        int result;
        if (k <= 55) {
            result = (int)((100003LL - 200003LL*k + 300007LL*k*k*k) % 1000000) - 500000;
            k++;
        } else {
            result = (history[(index - 24 + 55) % 55] + 
                      history[(index - 55 + 55) % 55] + 1000000) % 1000000 - 500000;
        }
        history[index] = result;
        index = (index + 1) % 55;
        return result;
    }
};

int getMaxSubstringSum(const std::vector<std::vector<int>>& grid, 
                       int x, int y, int dx, int dy) {
    int maxSum = 0;
    int cur = 0;
    int size = grid.size();
    
    while (x >= 0 && x < size && y >= 0 && y < size) {
        cur = std::max(cur + grid[y][x], 0);
        maxSum = std::max(cur, maxSum);
        x += dx;
        y += dy;
    }
    return maxSum;
}

int main() {
    const int SIZE = 2000;
    std::vector<std::vector<int>> grid(SIZE, std::vector<int>(SIZE));
    
    LfgRandom rand;
    for (int y = 0; y < SIZE; y++) {
        for (int x = 0; x < SIZE; x++) {
            grid[y][x] = rand.next();
        }
    }
    
    int maxSum = 0;
    for (int i = 0; i < SIZE; i++) {
        maxSum = std::max(getMaxSubstringSum(grid, 0, i, 1, 0), maxSum);
        maxSum = std::max(getMaxSubstringSum(grid, i, 0, 0, 1), maxSum);
        maxSum = std::max(getMaxSubstringSum(grid, 0, i, 1, 1), maxSum);
        maxSum = std::max(getMaxSubstringSum(grid, i, 0, 1, 1), maxSum);
        maxSum = std::max(getMaxSubstringSum(grid, i, 0, -1, 1), maxSum);
        maxSum = std::max(getMaxSubstringSum(grid, SIZE - 1, i, -1, 1), maxSum);
    }
    
    std::cout << maxSum << std::endl;
    return 0;
}
```
