# Nombres Pentagonaux

## Description du Problème
Les nombres pentagonaux sont générés par la formule, Pₙ = n(3n−1)/2. Les dix premiers nombres pentagonaux sont :

1, 5, 12, 22, 35, 51, 70, 92, 117, 145, ...

On peut voir que P₄ + P₇ = 22 + 70 = 92 = P₈. Cependant, leur différence, 70 − 22 = 48, n'est pas pentagonale.

**Objectif** : Trouver la paire de nombres pentagonaux, Pⱼ et Pₖ, pour laquelle leur somme et leur différence sont pentagonales et D = |Pₖ − Pⱼ| est minimisée ; quelle est la valeur de D ?

---

## Exemples
### Exemple 1
**Entrée** : Nombres pentagonaux
**Sortie** : `5482660`
**Explication** : La différence minimale D entre deux nombres pentagonaux avec la propriété.

---

## Approche de Résolution

### Stratégie
1. Pour chaque indice i croissant :
   - Calculer Pᵢ.
   - Pour chaque indice j < i :
     - Calculer Pⱼ.
     - Calculer la différence D = Pᵢ − Pⱼ.
     - Vérifier si Pᵢ + Pⱼ et D sont tous deux pentagonaux.
     - Si oui et si D est plus petit que le minimum actuel, mettre à jour.
   - Si la différence entre Pᵢ et Pᵢ₋₁ est ≥ D minimum trouvé, arrêter.
2. Retourner D minimum.

### Observations
- Un nombre y est pentagonal si y = n(3n-1)/2 pour un n entier.
- Résolvant pour n : n = (√(24y + 1) + 1) / 6.
- y est pentagonal si √(24y + 1) + 1 est divisible par 6.
- On peut arrêter tôt quand la différence entre nombres pentagonaux consécutifs dépasse D.

### Complexité
- **Temporelle** : O(n²), où `n` est le nombre de nombres pentagonaux testés.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

long long pentagonalNumber(int n) {
    return (long long)n * (3 * n - 1) / 2;
}

bool isPentagonalNumber(long long y) {
    if (y <= 0) return false;
    
    long long temp = y * 24 + 1;
    long long sqrtTemp = sqrt(temp);
    
    if (sqrtTemp * sqrtTemp == temp) {
        return (sqrtTemp + 1) % 6 == 0;
    }
    return false;
}

int main() {
    long long minD = -1;
    
    for (int i = 2; ; ++i) {
        long long pentI = pentagonalNumber(i);
        
        if (minD != -1 && pentI - pentagonalNumber(i - 1) >= minD) {
            break;
        }
        
        for (int j = i - 1; j >= 1; --j) {
            long long pentJ = pentagonalNumber(j);
            long long diff = pentI - pentJ;
            
            if (minD != -1 && diff >= minD) {
                break;
            }
            
            if (isPentagonalNumber(pentI + pentJ) && isPentagonalNumber(diff)) {
                if (minD == -1 || diff < minD) {
                    minD = diff;
                }
            }
        }
    }
    
    std::cout << minD << std::endl;
    return 0;
}
```
