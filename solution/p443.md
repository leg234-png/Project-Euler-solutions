# Séquence GCD

## Description du Problème
Soit g(4)=13, g(5)=14, g(6)=16, g(7)=17, g(8)=18, g(9)=27, g(10)=28, g(11)=29, g(12)=30, g(13)=31, g(14)=32, g(15)=33, g(16)=34, g(17)=51, g(18)=54, g(19)=55, g(20)=60, ...

On vous donne que g(1000)=2524 et g(1000000)=2624152.

**Objectif** : Trouver g(10¹⁵).

---

## Exemples
### Exemple 1
**Entrée** : `n = 1000`
**Sortie** : `2524`
**Explication** : g(1000) = 2524.

---

## Approche de Résolution

### Stratégie
1. Si gcd(n, g(n-1)) = 1, alors g(n) = g(n-1) + 1.
2. Observer les patterns : on commence à 4, s'arrête à 8, saute à 9, s'arrête à 16, saute, etc.
3. Les sauts se produisent quand gcd(n, g(n-1)) > 1.
4. Analyser les différences entre les sauts pour trouver un pattern.
5. Utiliser ce pattern pour calculer efficacement g(10¹⁵) sans itérer sur tous les nombres.

### Observations
- Les sauts se produisent à des nombres premiers de la forme 2g(n-1) - 1.
- Après un saut, on continue avec g(n) = g(n-1) + 1 jusqu'au prochain saut.
- On peut utiliser cette propriété pour sauter de grands intervalles.

### Complexité
- **Temporelle** : O(J) où J est le nombre de sauts jusqu'à 10¹⁵.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long gcd(long long a, long long b) {
    while (b != 0) {
        long long temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

bool is_prime(long long n) {
    if (n <= 1) return false;
    if (n <= 3) return true;
    if (n % 2 == 0) return false;
    for (long long i = 3; i * i <= n; i += 2) {
        if (n % i == 0) return false;
    }
    return true;
}

long long compute(long long limit) {
    long long g = 13; // g(4) = 13
    long long n = 4;
    
    while (n < limit) {
        if (gcd(n + 1, g) == 1) {
            g++;
            n++;
        } else {
            // Saut possible
            long long candidate = 2 * g - 1;
            if (is_prime(candidate) && candidate > n) {
                // Grand saut
                long long jump = candidate - n;
                if (n + jump <= limit) {
                    n += jump;
                    g = candidate;
                } else {
                    // Continuer normalement
                    n++;
                    g++;
                }
            } else {
                n++;
                g++;
            }
        }
    }
    
    return g;
}

int main() {
    cout << compute(1000000000000000LL) << endl;
    return 0;
}
```
