# Somme Consécutive de Nombres Premiers

## Description du Problème
Le nombre premier 41 peut être écrit comme la somme de six nombres premiers consécutifs :

41 = 2 + 3 + 5 + 7 + 11 + 13

C'est la plus longue somme de nombres premiers consécutifs qui donne un nombre premier inférieur à cent.

La plus longue somme de nombres premiers consécutifs qui donne un nombre premier inférieur à mille contient 21 termes et est égale à 953.

**Objectif** : Quel nombre premier, inférieur à un million, peut être écrit comme la somme du plus grand nombre de nombres premiers consécutifs ?

---

## Exemples
### Exemple 1
**Entrée** : Limite = 100
**Sortie** : `41`
**Explication** : 41 est la plus longue somme de nombres premiers consécutifs (6 termes) qui donne un nombre premier < 100.

---

## Approche de Résolution

### Stratégie
1. Utiliser le crible d'Ératosthène pour trouver tous les nombres premiers jusqu'à 1 000 000.
2. Pour chaque position de départ i :
   - Calculer la somme cumulative des nombres premiers à partir de i.
   - Pour chaque position de fin j ≥ i :
     - Si la somme est ≤ 1 000 000 et est première :
       - Si la longueur (j - i + 1) est plus grande que le maximum actuel, mettre à jour.
3. Retourner le nombre premier avec la plus longue séquence.

### Observations
- On peut utiliser une somme cumulative pour éviter de recalculer.
- On peut arrêter quand la somme dépasse 1 000 000.
- On doit garder trace à la fois de la longueur de la séquence et du nombre premier résultant.

### Complexité
- **Temporelle** : O(n²), où `n` est le nombre de nombres premiers jusqu'à 1 000 000.
- **Spatiale** : O(n), pour stocker les nombres premiers et le crible.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

std::vector<bool> sieve(int n) {
    std::vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    int sqrtN = sqrt(n);
    for (int i = 2; i <= sqrtN; ++i) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    return isPrime;
}

std::vector<int> listPrimes(int n, const std::vector<bool>& isPrime) {
    std::vector<int> primes;
    for (int i = 2; i <= n; ++i) {
        if (isPrime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

int main() {
    const int limit = 1000000;
    std::vector<bool> isPrime = sieve(limit);
    std::vector<int> primes = listPrimes(limit, isPrime);
    
    long long maxSum = 0;
    int maxRun = -1;
    
    for (size_t i = 0; i < primes.size(); ++i) {
        long long sum = 0;
        for (size_t j = i; j < primes.size(); ++j) {
            sum += primes[j];
            if (sum > limit) {
                break;
            } else if ((int)(j - i) > maxRun && isPrime[sum]) {
                maxSum = sum;
                maxRun = j - i;
            }
        }
    }
    
    std::cout << maxSum << std::endl;
    return 0;
}
```
