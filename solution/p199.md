# Empaquetage itératif de cercles

## Description du Problème
À chaque itération, un cercle de taille maximale est placé dans chaque espace, ce qui crée plus d'espaces pour l'itération suivante. Après 3 itérations (illustré), il y a 108 espaces et la fraction de l'aire qui n'est pas couverte par les cercles est 0.06790342, arrondie à huit décimales.

**Objectif** : Quelle fraction de l'aire n'est pas couverte par les cercles après 10 itérations ? Donner votre réponse arrondie à huit décimales en utilisant le format x.xxxxxxxx.

---

## Exemples
### Exemple 1
**Entrée** : `iterations = 3`
**Sortie** : `0.06790342`
**Explication** : Après 3 itérations, la fraction non couverte est 0.06790342.

---

## Approche de Résolution

### Stratégie
1. Utiliser le théorème de Descartes pour trouver la courbure du quatrième cercle.
2. Étant donné trois cercles avec courbures k₁, k₂, k₃, on peut trouver la 4ème courbure : k₄ = k₁ + k₂ + k₃ ± 2√(k₁k₂ + k₁k₃ + k₂k₃).
3. Commencer avec 3 cercles de rayon 1, puis itérer récursivement.
4. Calculer l'aire couverte à chaque itération et soustraire de l'aire totale.

### Observations
- Le théorème de Descartes permet de calculer récursivement les cercles dans les espaces.
- On commence avec 3 cercles de rayon 1, puis on trouve le cercle extérieur (courbure négative) et les cercles intérieurs (courbure positive).
- L'aire d'un cercle de rayon r est πr².

### Complexité
- **Temporelle** : O(3^depth) pour la récursion, où depth est le nombre d'itérations.
- **Spatiale** : O(depth) pour la pile de récursion.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

const double PI = 3.14159265358979323846;

double circle_area(double r) {
    return PI * r * r;
}

double find_curvature(double k1, double k2, double k3) {
    double c = k1 + k2 + k3;
    double t = 2 * sqrt(k1*k2 + k1*k3 + k2*k3);
    return c + t; // Pour le cercle intérieur
}

double find_area_covered(double k1, double k2, double k3, int depth) {
    double k4 = find_curvature(k1, k2, k3);
    double r = 1.0 / k4;
    double accumulated_area = circle_area(r);
    
    if (depth == 1) {
        return accumulated_area;
    }
    
    accumulated_area += find_area_covered(k1, k2, k4, depth - 1);
    accumulated_area += find_area_covered(k2, k3, k4, depth - 1);
    accumulated_area += find_area_covered(k1, k3, k4, depth - 1);
    
    return accumulated_area;
}

double compute(int depth_limit) {
    double main_curv = 3 - 2*sqrt(3);
    double main_area = circle_area(-1.0/main_curv);
    
    double k = 1.0;
    double total_area = find_area_covered(k, k, k, depth_limit) * 3;
    total_area += find_area_covered(k, k, main_curv, depth_limit) * 3;
    
    return 1.0 - (total_area / main_area);
}

int main() {
    double result = compute(10);
    printf("%.8f\n", result);
    return 0;
}
```
