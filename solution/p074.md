# Chaînes de factorielles de chiffres

## Description du Problème
Le nombre 145 est bien connu pour la propriété que la somme de la factorielle de ses chiffres est égale à 145 :
1! + 4! + 5! = 1 + 24 + 120 = 145

Peut-être moins connu est 169, car il produit la plus longue chaîne de nombres qui revient à 169 ; il s'avère qu'il n'y a que trois de ces boucles :
169 → 363601 → 1454 → 169
871 → 45361 → 871
872 → 45362 → 872

Il n'est pas difficile de prouver que TOUT nombre de départ finira par se retrouver dans une boucle. Par exemple :
69 → 363600 → 1454 → 169 → 363601 (→ 1454)
78 → 45360 → 871 → 45361 (→ 871)
540 → 145 (→ 145)

En commençant par 69, on produit une chaîne de cinq termes non répétitifs, mais la plus longue chaîne non répétitive qui commence avec un nombre inférieur à un million est de soixante termes.

**Objectif** : Combien de chaînes, avec un nombre de départ inférieur à un million, contiennent exactement soixante termes non répétitifs ?

---

## Exemples
### Exemple 1
**Entrée** : 69
**Sortie** : `5`
**Explication** : 69 → 363600 → 1454 → 169 → 363601 → 1454 (boucle), donc 5 termes non répétitifs.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre de 0 à 999 999, calculer la longueur de la chaîne jusqu'à détecter une répétition.
2. Utiliser un HashSet pour détecter les cycles.
3. La fonction factorialize calcule la somme des factorielles des chiffres d'un nombre.
4. Compter combien de nombres ont exactement 60 termes non répétitifs.

### Observations
- Les factorielles de 0 à 9 sont : 1, 1, 2, 6, 24, 120, 720, 5040, 40320, 362880.
- On peut utiliser la mémorisation pour éviter de recalculer les longueurs de chaînes.
- Les boucles connues sont : 169 → 363601 → 1454 → 169, 871 → 45361 → 871, 872 → 45362 → 872.

### Complexité
- **Temporelle** : O(N × L), où N est la limite (1 000 000) et L est la longueur moyenne de chaîne.
- **Spatiale** : O(L), pour stocker les nombres vus dans une chaîne.

---

## Solution en C++

```cpp
#include <iostream>
#include <unordered_set>

using namespace std;

const int LIMIT = 1000000;
const int FACTORIAL[] = {1, 1, 2, 6, 24, 120, 720, 5040, 40320, 362880};

int factorialize(int n) {
    int sum = 0;
    while (n != 0) {
        sum += FACTORIAL[n % 10];
        n /= 10;
    }
    return sum;
}

int getChainLength(int n) {
    unordered_set<int> seen;
    while (true) {
        if (seen.find(n) != seen.end())
            return seen.size();
        seen.insert(n);
        n = factorialize(n);
    }
}

int main() {
    int count = 0;
    for (int i = 0; i < LIMIT; i++) {
        if (getChainLength(i) == 60)
            count++;
    }
    cout << count << endl;
    return 0;
}
```
