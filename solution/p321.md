# Échange de compteurs

## Description du Problème
Soit M(n) représenter le nombre minimum de coups/actions pour inverser complètement les positions des compteurs colorés ; c'est-à-dire, déplacer tous les compteurs rouges vers la droite et tous les compteurs bleus vers la gauche.

On peut vérifier que M(3)=15, qui est aussi un nombre triangulaire.

Si on crée une séquence basée sur les valeurs de n pour lesquelles M(n) est un nombre triangulaire, alors les cinq premiers termes seraient : 1, 3, 10, 22, et 63, et leur somme serait 99.

**Objectif** : Trouver la somme des quarante premiers termes de cette séquence.

---

## Exemples
### Exemple 1
**Entrée** : `n = 3`
**Sortie** : `15` (nombre triangulaire)
**Explication** : M(3) = 15, qui est un nombre triangulaire.

---

## Approche de Résolution

### Stratégie
1. Observer que M(1) = 3, M(2) = 8, M(3) = 15.
2. La séquence OEIS A005563 donne M(n) = n² + 2n = (n + 1)² - 1.
3. On veut M(n) = t(t + 1)/2, donc 2n² + 4n = t² + t.
4. Résoudre l'équation diophantienne 2n² + 4n - t² - t = 0.
5. Utiliser la méthode de résolution d'équations diophantiennes pour trouver toutes les solutions.
6. Les solutions suivent une récurrence linéaire.

### Observations
- M(n) = n² + 2n pour cette configuration.
- On doit résoudre une équation quadratique en n et t.
- Les solutions peuvent être générées récursivement à partir de solutions initiales.

### Complexité
- **Temporelle** : O(k) où k est le nombre de solutions à générer.
- **Spatiale** : O(k) pour stocker les solutions.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <algorithm>

using namespace std;

vector<long long> equation_solver(int x, long long startx, long long starty, 
                                  long long a, long long b, long long c,
                                  long long d, long long e, long long f) {
    long long x1 = startx;
    long long y1 = starty;
    vector<long long> solutions;
    
    while (solutions.size() < x) {
        long long xn = a * x1 + b * y1 + c;
        long long yn = d * x1 + e * y1 + f;
        if (xn > 0) {
            solutions.push_back(xn);
        }
        x1 = xn;
        y1 = yn;
    }
    return solutions;
}

long long compute(int limit) {
    vector<long long> solutions1 = equation_solver(40, 0, 0, 3, 2, 3, 4, 3, 5);
    vector<long long> solutions2 = equation_solver(40, 0, -1, 3, 2, 3, 4, 3, 5);
    
    set<long long> all_solutions;
    for (long long s : solutions1) {
        all_solutions.insert(s);
    }
    for (long long s : solutions2) {
        all_solutions.insert(s);
    }
    
    vector<long long> sorted_solutions(all_solutions.begin(), all_solutions.end());
    sort(sorted_solutions.begin(), sorted_solutions.end());
    
    long long total = 0;
    for (int i = 0; i < limit && i < sorted_solutions.size(); i++) {
        total += sorted_solutions[i];
    }
    
    return total;
}

int main() {
    cout << compute(40) << endl;
    return 0;
}
```
