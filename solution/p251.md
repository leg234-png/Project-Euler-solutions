# Triplets de Cardano

## Description du Problème
Un triplet d'entiers positifs (a,b,c) est appelé un triplet de Cardano si les racines de l'équation x³ - ax² + bx - c = 0 sont toutes réelles.

Par exemple, (2,1,5) est un triplet de Cardano.

Il existe 149 triplets de Cardano pour lesquels a+b+c ≤ 1000.

**Objectif** : Trouver combien de triplets de Cardano existent tels que a+b+c ≤ 110000000.

---

## Exemples
### Exemple 1
**Entrée** : `limit = 1000`
**Sortie** : `149`
**Explication** : Il y a 149 triplets de Cardano avec a+b+c ≤ 1000.

---

## Approche de Résolution

### Stratégie
1. Simplifier en utilisant l'algèbre : c = (a + 1)²(8a - 1)/(27b²).
2. Noter que b > 0 (si b = 0, alors a = 1/8 qui n'est pas un entier).
3. On a besoin que b² | (a + 1)²(8a - 1)/27, donc (a + 1)²(8a - 1) ≡ 0 (mod 27).
4. Les solutions sont a = 3k + 2.
5. Pour chaque k, générer (a + 1)²(8a - 1) et trouver tous ses diviseurs b².
6. Vérifier que a + b + c ≤ limite.

### Observations
- La condition modulaire réduit considérablement l'espace de recherche.
- On peut factoriser (a + 1)²(8a - 1) pour trouver tous les diviseurs b² possibles.
- On doit vérifier que c est un entier positif.

### Complexité
- **Temporelle** : O(limit × D) où D est le nombre moyen de diviseurs.
- **Spatiale** : O(√limit) pour stocker les facteurs premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <map>
#include <cmath>

using namespace std;

map<long long, int> prime_factors(long long n) {
    map<long long, int> factors;
    long long d = 2;
    while (n > 1) {
        while (n % d == 0) {
            factors[d]++;
            n /= d;
        }
        d++;
        if (d * d > n) {
            if (n > 1) {
                factors[n]++;
            }
            break;
        }
    }
    return factors;
}

vector<long long> get_square_divisors(long long n) {
    map<long long, int> factors = prime_factors(n);
    vector<long long> divisors = {1};
    
    for (auto& [p, exp] : factors) {
        int size = divisors.size();
        long long power = 1;
        for (int i = 0; i <= exp; i += 2) {
            for (int j = 0; j < size; j++) {
                divisors.push_back(divisors[j] * power);
            }
            power *= p * p;
        }
    }
    
    return divisors;
}

long long compute(long long limit) {
    long long count = 0;
    
    for (long long k = 0; ; k++) {
        long long a = 3 * k + 2;
        if (a > limit) break;
        
        long long expr = (a + 1) * (a + 1) * (8 * a - 1);
        if (expr % 27 != 0) continue;
        
        long long value = expr / 27;
        vector<long long> square_divisors = get_square_divisors(value);
        
        for (long long b_sq : square_divisors) {
            long long b = sqrt(b_sq);
            if (b * b != b_sq) continue;
            
            if (value % b_sq != 0) continue;
            long long c = value / b_sq;
            
            if (a + b + c <= limit && a > 0 && b > 0 && c > 0) {
                count++;
            }
        }
    }
    
    return count;
}

int main() {
    cout << compute(110000000) << endl;
    return 0;
}
```
