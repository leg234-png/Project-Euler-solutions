# Darts

## Description du Problème
Dans le jeu de fléchettes, un joueur lance trois fléchettes sur une cible divisée en vingt sections de taille égale numérotées de un à vingt.

Le score d'une fléchette est déterminé par le numéro de la région dans laquelle la fléchette atterrit. Une fléchette atterrissant à l'extérieur de l'anneau extérieur rouge/vert marque zéro. Les régions noires et crème à l'intérieur de cet anneau représentent des scores simples. Cependant, l'anneau extérieur rouge/vert et l'anneau du milieu marquent des scores doubles et triples respectivement.

Au centre de la cible se trouvent deux cercles concentriques appelés la région du taureau, ou œil de bœuf. Le taureau extérieur vaut 25 points et le taureau intérieur est un double, valant 50 points.

Il existe de nombreuses variations de règles, mais dans le jeu le plus populaire, les joueurs commenceront avec un score de 301 ou 501 et le premier joueur à réduire son total courant à zéro est un gagnant. Cependant, il est normal de jouer un système "doubles out", ce qui signifie que le joueur doit atterrir sur un double (y compris le double œil de bœuf au centre de la cible) sur sa dernière fléchette pour gagner ; toute autre fléchette qui réduirait son total courant à un ou moins signifie que le score pour cet ensemble de trois fléchettes est "bust".

Lorsqu'un joueur est capable de terminer sur son score actuel, cela s'appelle un "checkout" et le checkout le plus élevé est 170 : T20 T20 D25 (deux triples 20 et double taureau).

Il y a exactement onze façons distinctes de faire un checkout sur un score de 6.

**Objectif** : Combien de façons distinctes un joueur peut-il faire un checkout avec un score inférieur à 100 ?

---

## Exemples
### Exemple 1
**Entrée** : `score = 6`
**Sortie** : `11`
**Explication** : Il y a 11 façons distinctes de faire un checkout sur un score de 6.

---

## Approche de Résolution

### Stratégie
1. Générer toutes les valeurs de points possibles (singles, doubles, triples, taureaux).
2. Pour chaque score de 1 à 99, compter les façons de faire un checkout.
3. Un checkout nécessite que la dernière fléchette soit un double (ou double taureau).
4. Utiliser la programmation dynamique avec mémorisation pour compter les façons.

### Observations
- Les points possibles incluent : 1-20 (singles, doubles, triples), 25, 50.
- La dernière fléchette doit être un double.
- On doit éviter de compter les combinaisons équivalentes (ordre des deux premières fléchettes).

### Complexité
- **Temporelle** : O(score × nombre_de_points × nombre_de_fléchettes).
- **Spatiale** : O(score × nombre_de_points × nombre_de_fléchettes) pour la mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cstring>

int main() {
    // Générer les points possibles
    std::vector<int> points;
    for (int i = 1; i <= 20; i++) {
        for (int j = 1; j <= 3; j++) {
            points.push_back(i * j);
        }
    }
    points.push_back(25);
    points.push_back(50);
    
    // Points doubles pour le checkout
    std::vector<int> doublePoints;
    for (int i = 1; i <= 20; i++) {
        doublePoints.push_back(i * 2);
    }
    doublePoints.push_back(25 * 2);
    
    // DP : ways[throws][total][maxIndex]
    int ways[3][101][points.size()];
    memset(ways, -1, sizeof(ways));
    
    // Fonction récursive pour compter les façons
    auto countWays = [&](int throwz, int total, int maxIndex) -> int {
        if (ways[throwz][total][maxIndex] != -1) {
            return ways[throwz][total][maxIndex];
        }
        
        int result;
        if (throwz == 0) {
            result = (total == 0) ? 1 : 0;
        } else {
            result = 0;
            if (maxIndex > 0) {
                result += countWays(throwz, total, maxIndex - 1);
            }
            if (points[maxIndex] <= total) {
                result += countWays(throwz - 1, total - points[maxIndex], maxIndex);
            }
        }
        
        ways[throwz][total][maxIndex] = result;
        return result;
    };
    
    int checkouts = 0;
    for (int remainingPoints = 1; remainingPoints < 100; remainingPoints++) {
        for (int throwz = 0; throwz <= 2; throwz++) {
            for (int p : doublePoints) {
                if (p <= remainingPoints) {
                    checkouts += countWays(throwz, remainingPoints - p, points.size() - 1);
                }
            }
        }
    }
    
    std::cout << checkouts << std::endl;
    return 0;
}
```
