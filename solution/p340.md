# Fonction folle

## Description du Problème
Soit F(0) = 1 et F(n) = F(n-a) + F(n-b) + F(n-c) pour n > 0, où a, b, c sont des entiers positifs.

Aussi, soit S(a,b,c) = ∑ₙ₌₀ᵇ F(n).

Par exemple, si a=50, b=2000 et c=40, alors F(0)=3240 et F(2000)=2040.

Aussi, S(50,2000,40)=5204240.

**Objectif** : Trouver les 9 derniers chiffres de S(21⁷, 7²¹, 12⁷).

---

## Exemples
### Exemple 1
**Entrée** : `a=50, b=2000, c=40`
**Sortie** : `5204240`
**Explication** : S(50,2000,40) = 5204240.

---

## Approche de Résolution

### Stratégie
1. Simplifier F(n) en analysant les différents cas selon la valeur de n.
2. Si n est dans la plage (b-a, b], alors a + n > b, donc F(a + n) = a + n - c.
3. Si n est dans (b - (k+1)a, b - ka], alors F(n) = 4(k+1)a + n - (3k+4)c.
4. Utiliser cette formule pour calculer efficacement S(a,b,c).
5. Sommer sur tous les k de 0 à b/a.

### Observations
- La récursion directe ne fonctionnera pas pour de grandes valeurs.
- On peut simplifier F(n) en fonction de la plage dans laquelle n se trouve.
- La formule finale permet de calculer S(a,b,c) sans récursion.

### Complexité
- **Temporelle** : O(b/a) pour itérer sur les différentes plages.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long S(long long a, long long b, long long c, long long mod) {
    long long total = 0;
    long long k_max = b / a;
    
    for (long long k = 0; k <= k_max; k++) {
        long long range_start = max(0LL, b - (k + 1) * a);
        long long range_end = b - k * a;
        
        if (range_start < range_end) {
            long long count = range_end - range_start;
            long long sum_n = (range_start + range_end - 1) * count / 2;
            
            long long term1 = count * (4 * (k + 1) * a - (3 * k + 4) * c) % mod;
            long long term2 = sum_n % mod;
            
            total = (total + term1 + term2) % mod;
        }
    }
    
    return (total + mod) % mod;
}

int main() {
    long long a = mod_pow(21, 7, 1000000000);
    long long b = mod_pow(7, 21, 1000000000);
    long long c = mod_pow(12, 7, 1000000000);
    long long mod = 1000000000;
    
    cout << S(a, b, c, mod) << endl;
    return 0;
}
```
