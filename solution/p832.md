# Séquence Mex

## Description du Problème
Soit a et b deux entiers positifs. On écrit a, b et a⊕b sur un papier, où ⊕ est l'opération XOR.

Après le premier tour, {1, 2, 3} sera écrit sur le papier. Au deuxième tour, a=4 et parce que (4⊕5), (4⊕6) et (4⊕7) sont tous déjà écrits, b doit être 8.

Après n tours, il y aura 3n nombres sur le papier. Leur somme est notée M(n).

Par exemple, M(10) = 642 et M(1000) = 5432148.

**Objectif** : Trouver M(10⁹).

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : `642`
**Explication** : M(10) = 642.

---

## Approche de Résolution

### Stratégie
1. Observer les patterns : M(∑_{i=0}^n 4^i) = M(1 + 4^1 + ... + 4^n) = ∑_{j=1}^{4^(n+1)-1} j = (4^(n+1)-1)×4^(n+1)/2.
2. Les patterns montrent que les nombres sont organisés en blocs de 4.
3. Utiliser les patterns pour calculer M(n) efficacement.
4. Pour n = 4^k, il y a des formules directes.

### Observations
- Il y a des patterns clairs liés aux puissances de 4.
- On peut calculer M(n) récursivement en utilisant ces patterns.
- La formule permet de calculer sans générer tous les nombres.

### Complexité
- **Temporelle** : O(log n) pour le calcul récursif.
- **Spatiale** : O(log n) pour la mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <unordered_map>
#include <cmath>

using namespace std;

unordered_map<long long, long long> memo;

long long M(long long n) {
    if (n == 0) return 0;
    if (memo.find(n) != memo.end()) {
        return memo[n];
    }
    
    // Trouver la plus grande puissance de 4 ≤ n
    long long k = 0;
    long long four_power = 1;
    while (four_power * 4 <= n) {
        four_power *= 4;
        k++;
    }
    
    long long sum_powers = (four_power - 1) / 3;
    long long remaining = n - sum_powers;
    
    long long result = (four_power - 1) * four_power / 2;
    result += M(remaining);
    
    memo[n] = result;
    return result;
}

int main() {
    long long n = 1;
    for (int i = 0; i < 9; i++) {
        n *= 10;
    }
    cout << M(n) << endl;
    return 0;
}
```
