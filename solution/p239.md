# Vingt-deux nombres premiers insensés

## Description du Problème
Quelle est la probabilité que nous ayons un dérangement partiel tel que exactement 22 disques de nombres premiers soient trouvés loin de leurs positions naturelles ?

(N'importe quel nombre de disques non premiers peuvent également être trouvés dans ou hors de leurs positions naturelles.)

**Objectif** : Donner votre réponse arrondie à 12 décimales après la virgule sous la forme 0.abcdefghijkl.

---

## Exemples
### Exemple 1
**Entrée** : `25 nombres premiers, 3 choisis`
**Sortie** : `Probabilité`
**Explication** : La probabilité qu'exactement 22 nombres premiers restants soient dans des positions incorrectes.

---

## Approche de Résolution

### Stratégie
1. Choisir d'abord 3 nombres premiers parmi 25 : C(25,3).
2. Il reste 97 nombres, dont 22 sont premiers.
3. Trouver le nombre de dérangements tels que les 22 nombres premiers restants ne vont pas dans leur position.
4. Simplifier visuellement en disant que tous les 25 nombres premiers sont ordonnés dans les 25 premières places.
5. Utiliser une approche récursive avec mémorisation pour calculer les dérangements.

### Observations
- On peut choisir un choix libre, ce qui libère un nombre premier qui devient un choix libre.
- On peut choisir un nombre premier, ce qui fixe une deuxième place.
- À la fin, multiplier par le nombre de façons de choisir les choix libres.

### Complexité
- **Temporelle** : O(fixed × choice) pour le calcul récursif avec mémorisation.
- **Spatiale** : O(fixed × choice) pour la table de mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <unordered_map>
#include <cmath>

using namespace std;

unordered_map<string, long long> memo;

long long n_choose_r(long long n, long long r) {
    if (r > n) return 0;
    long long result = 1;
    for (long long i = 0; i < r; i++) {
        result = result * (n - i) / (i + 1);
    }
    return result;
}

long long derangements(long long fixed_left, long long choice) {
    if (fixed_left == 0) {
        // Calculer le factoriel de choice
        long long fact = 1;
        for (long long i = 1; i <= choice; i++) {
            fact *= i;
        }
        return fact;
    }
    
    string key = to_string(fixed_left) + "," + to_string(choice);
    if (memo.find(key) != memo.end()) {
        return memo[key];
    }
    
    long long total = 0;
    
    // A) Choisir un choix libre
    if (choice > 0) {
        total += choice * derangements(fixed_left, choice - 1);
    }
    
    // B) Choisir un nombre premier
    if (fixed_left > 0) {
        total += fixed_left * derangements(fixed_left - 1, choice + 1);
    }
    
    memo[key] = total;
    return total;
}

double compute() {
    long long ways_to_choose_3_primes = n_choose_r(25, 3);
    long long derangements_count = derangements(22, 75);
    long long total_arrangements = ways_to_choose_3_primes * derangements_count;
    
    long long total_possibilities = 1;
    for (long long i = 1; i <= 100; i++) {
        total_possibilities *= i;
    }
    
    return (double)total_arrangements / total_possibilities;
}

int main() {
    cout << fixed << setprecision(12) << compute() << endl;
    return 0;
}
```
