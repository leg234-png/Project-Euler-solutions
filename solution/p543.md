# Nombres somme-premiers

## Description du Problème
Soit P(i,k) = 1 si i peut être écrit comme somme de k nombres premiers (peuvent être répétés), sinon P(i,k) = 0.

Soit S(n) la somme de tous les P(i,k) sur 1 ≤ i, k ≤ n.

Par exemple, S(10) = 20, S(100) = 2402, et S(1000) = 248838.

Soit F(k) le k-ième nombre de Fibonacci (avec F(0)=0 et F(1)=1).

**Objectif** : Trouver la somme de tous les S(F(k)) sur 3 ≤ k ≤ 44.

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : `20`
**Explication** : S(10) = 20.

---

## Approche de Résolution

### Stratégie
1. Créer une matrice telle que matrix[i][k] = P(i, k).
2. P(i, 1) = 1 si i est premier, 0 sinon.
3. P(i, k) = 1 si P(i-p, k-1) = 1 pour un nombre premier p ≤ i.
4. Utiliser la programmation dynamique pour remplir la matrice.
5. Pour n ≥ 6, P(n, 3) = 1. Pour n ≥ 8, P(n, 4) = 1.
6. En général, pour n ≥ 2x, x ≥ 3, P(n, x) = 1.
7. Calculer S(n) en sommant tous les P(i, k).

### Observations
- On peut optimiser en utilisant les patterns observés.
- Pour de grandes valeurs, on peut utiliser des approximations.
- La matrice complète n'est pas nécessaire pour tous les cas.

### Complexité
- **Temporelle** : O(n² × π(n)) où π(n) est le nombre de nombres premiers ≤ n.
- **Spatiale** : O(n²) pour la matrice.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<bool> prime_sieve(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    return is_prime;
}

long long compute_S(int n) {
    vector<bool> is_prime = prime_sieve(n);
    vector<vector<bool>> P(n + 1, vector<bool>(n + 1, false));
    
    // P(i, 1) = 1 si i est premier
    for (int i = 2; i <= n; i++) {
        P[i][1] = is_prime[i];
    }
    
    // Calculer P(i, k) pour k > 1
    for (int k = 2; k <= n; k++) {
        for (int i = 2; i <= n; i++) {
            for (int p = 2; p <= i; p++) {
                if (is_prime[p] && i - p >= 0) {
                    P[i][k] = P[i][k] || P[i - p][k - 1];
                }
            }
        }
    }
    
    long long total = 0;
    for (int i = 1; i <= n; i++) {
        for (int k = 1; k <= n; k++) {
            if (P[i][k]) total++;
        }
    }
    return total;
}

long long fibonacci(int n) {
    if (n == 0) return 0;
    if (n == 1) return 1;
    long long a = 0, b = 1;
    for (int i = 2; i <= n; i++) {
        long long temp = a + b;
        a = b;
        b = temp;
    }
    return b;
}

int main() {
    long long total = 0;
    for (int k = 3; k <= 44; k++) {
        long long F_k = fibonacci(k);
        total += compute_S(F_k);
    }
    cout << total << endl;
    return 0;
}
```
