# Équations de partitions entières

## Description du Problème
Pour tout m≥1, soit P(m) la proportion de telles partitions qui sont parfaites avec k≤m.

Ainsi P(6)=1/2.

Dans le tableau suivant sont listées certaines valeurs de P(m) :
P(5)=1/1, P(10)=1/2, P(15)=2/3, P(20)=1/2, P(25)=1/2, P(30)=2/5, ..., P(180)=1/4, P(185)=3/13.

**Objectif** : Trouver le plus petit m pour lequel P(m) < 1/12345.

---

## Exemples
### Exemple 1
**Entrée** : `m = 6`
**Sortie** : `1/2`
**Explication** : P(6) = 1/2.

---

## Approche de Résolution

### Stratégie
1. Soit x = 2^t, alors on a l'équation 2^(2t) - 2^t - k = 0, qui est une équation quadratique.
2. On a 2^t = x = (1 + √(1 + 4k)) / 2, permettant de résoudre pour t = log₂(1 + √(1 + 4k)) - 1.
3. Mais on a besoin que x soit un entier, ce qui implique que √(1 + 4k) doit être un entier, donc 1 + 4k doit être un carré.
4. Boucler à travers les carrés et tester si a² - 1 est divisible par 4. Si oui, on a trouvé une valeur valide de k, puis tester si t est un entier.

### Observations
- Une partition est parfaite si elle satisfait certaines conditions.
- On peut résoudre analytiquement pour trouver les valeurs de k valides.
- On doit vérifier que t est un entier pour que la partition soit parfaite.

### Complexité
- **Temporelle** : O(√m) pour boucler jusqu'à trouver la bonne proportion.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long compute(double frac) {
    long long perf_count = 0;
    long long total_count = 0;
    
    long long a = 2;
    while (true) {
        long long v = a * a - 1;
        if (v % 4 == 0) {
            long long k = v / 4;
            double t = log2(1 + a) - 1;
            if (t == (long long)t) {
                perf_count++;
            }
            total_count++;
            if ((double)perf_count / total_count < frac) {
                return k;
            }
        }
        a++;
    }
}

int main() {
    cout << compute(1.0 / 12345) << endl;
    return 0;
}
```
