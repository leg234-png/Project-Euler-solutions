# Squarefree Binomial Coefficients

## Description du Problème
Les coefficients binomiaux C(n, k) peuvent être arrangés en forme de triangle de Pascal, où chaque nombre est la somme des deux nombres directement au-dessus.

**Objectif** : Trouver la somme de tous les nombres distincts dans les 51 premières lignes du triangle de Pascal qui sont "squarefree" (sans facteur carré).

Un nombre est squarefree s'il n'est divisible par aucun carré de nombre premier.

---

## Exemples
### Exemple 1
**Entrée** : Triangle de Pascal jusqu'à la ligne 5
**Sortie** : Somme des nombres squarefree distincts
**Explication** : Par exemple, 6 = 2×3 est squarefree, mais 4 = 2² n'est pas squarefree.

---

## Approche de Résolution

### Stratégie
1. Collecter tous les nombres distincts dans les 51 premières lignes du triangle de Pascal.
2. Pour chaque nombre, vérifier s'il est squarefree :
   - Générer tous les nombres premiers jusqu'à √max.
   - Pour chaque nombre premier p, vérifier si p² divise le nombre.
   - Si aucun p² ne divise le nombre, il est squarefree.
3. Additionner tous les nombres squarefree distincts.

### Observations
- Un nombre est squarefree s'il n'est divisible par aucun carré de nombre premier.
- On peut optimiser en pré-calculant les carrés des nombres premiers.
- Les coefficients binomiaux peuvent être très grands, donc on utilise BigInteger.

### Complexité
- **Temporelle : O(N × P)**, où N est le nombre de nombres distincts et P est le nombre de nombres premiers ≤ √max.
- **Spatiale : O(N + P)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <set>
#include <vector>
#include <cmath>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

cpp_int binomial(int n, int k) {
    if (k > n - k) k = n - k;
    cpp_int result = 1;
    for (int i = 0; i < k; i++) {
        result = result * (n - i) / (i + 1);
    }
    return result;
}

vector<int> listPrimes(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i)
                isPrime[j] = false;
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i])
            primes.push_back(i);
    }
    return primes;
}

bool isSquarefree(long long n, const vector<long long>& primesSquared) {
    for (long long p2 : primesSquared) {
        if (p2 > n) break;
        if (n % p2 == 0)
            return false;
    }
    return true;
}

int main() {
    set<long long> numbers;
    long long max = 0;
    
    for (int n = 0; n <= 50; n++) {
        for (int k = 0; k <= n; k++) {
            cpp_int x = binomial(n, k);
            if (x > numeric_limits<long long>::max())
                throw runtime_error("Number too large");
            long long val = x.convert_to<long long>();
            numbers.insert(val);
            max = max(max, val);
        }
    }
    
    vector<int> primes = listPrimes(sqrt(max));
    vector<long long> primesSquared;
    for (int p : primes)
        primesSquared.push_back((long long)p * p);
    
    long long sum = 0;
    for (long long n : numbers) {
        if (isSquarefree(n, primesSquared))
            sum += n;
    }
    
    cout << sum << endl;
    return 0;
}
```
