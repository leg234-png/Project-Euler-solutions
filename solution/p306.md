# Jeu de bande de papier

## Description du Problème
n=2 : Un seul coup valide, après lequel le second joueur perd.
n=3 : Deux coups valides, mais les deux laissent une situation où le second joueur perd.
n=4 : Trois coups valides pour le premier joueur, qui peut gagner en peignant les deux carrés du milieu.
n=5 : Quatre coups valides pour le premier joueur, mais peu importe ce que le joueur fait, le second joueur (bleu) gagne.

**Objectif** : Donc, pour 1≤n≤5, il y a 3 valeurs de n pour lesquelles le premier joueur peut forcer une victoire. Pour 1≤n≤10⁶, combien de valeurs de n le premier joueur peut-il forcer une victoire ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 4`
**Sortie** : `true` (premier joueur gagne)
**Explication** : Le premier joueur peut gagner en peignant les deux carrés du milieu.

---

## Approche de Résolution

### Stratégie
1. Utiliser le théorème de Sprague-Grundy pour résoudre ce jeu impartial.
2. Calculer la fonction Grundy pour chaque position.
3. Une position est gagnante si sa valeur Grundy est non nulle.
4. Pour n > 14, on observe que a(n) = a(n-5) + 34 (séquence OEIS A215721).
5. Utiliser cette périodicité pour calculer efficacement les valeurs.

### Observations
- Le théorème de Sprague-Grundy permet de résoudre les jeux impartiaux.
- La fonction mex (minimum excludant) est utilisée pour calculer les valeurs Grundy.
- Il y a une périodicité dans les positions perdantes qui permet d'optimiser.

### Complexité
- **Temporelle** : O(n²) pour calculer les valeurs Grundy jusqu'à n=14, puis O(1) par valeur avec la périodicité.
- **Spatiale** : O(n) pour stocker les valeurs Grundy.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>

using namespace std;

int mex(const set<int>& alist) {
    int mex_val = 0;
    while (alist.find(mex_val) != alist.end()) {
        mex_val++;
    }
    return mex_val;
}

long long compute(long long limit) {
    vector<int> array(limit + 1, 0);
    array[2] = 1;
    
    vector<int> losses = {0, 1};
    int x = 3;
    
    while (losses.size() != 14 && x < limit) {
        set<int> mex_list;
        for (int y = 0; y < x / 2; y++) {
            int t;
            if (y == 0) {
                t = array[x - 2];
            } else {
                t = array[y] ^ array[x - y - 2];
            }
            mex_list.insert(t);
        }
        array[x] = mex(mex_list);
        if (array[x] == 0) {
            losses.push_back(x);
        }
        x++;
    }
    
    if (losses.size() == 14) {
        long long count = 0;
        for (int i = 1; i <= min(limit, (long long)x); i++) {
            if (array[i] != 0) {
                count++;
            }
        }
        
        // Utiliser la périodicité pour n > 14
        long long period = 5;
        long long increment = 34;
        for (long long n = x; n <= limit; n++) {
            long long pos = (n - 14) % period;
            if (pos == 0 || pos == 2 || pos == 3) {
                count++;
            }
        }
        
        return count;
    }
    
    long long count = 0;
    for (int i = 1; i <= limit; i++) {
        if (array[i] != 0) {
            count++;
        }
    }
    return count;
}

int main() {
    cout << compute(1000000) << endl;
    return 0;
}
```
