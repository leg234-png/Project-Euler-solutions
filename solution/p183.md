# Produit maximum de parties

## Description du Problème
Il s'avère que le maximum pour N=11 est trouvé en divisant onze en quatre parties égales, ce qui mène à Pmax=(11/4)⁴ ; c'est-à-dire, M(11)=14641/256=57.19140625, qui est un nombre décimal terminant.

Cependant, pour N=8, le maximum est atteint en le divisant en trois parties égales, donc M(8)=512/27, qui est un nombre décimal non terminant.

Soit D(N)=N si M(N) est un nombre décimal non terminant et D(N)=−N si M(N) est un nombre décimal terminant.

**Objectif** : Trouver ∑ₙ₌₅¹⁰⁰⁰⁰ D(N).

---

## Exemples
### Exemple 1
**Entrée** : `N = 8`
**Sortie** : `8` (non terminant)
**Explication** : M(8) = 512/27 est non terminant, donc D(8) = 8.

### Exemple 2
**Entrée** : `N = 11`
**Sortie** : `-11` (terminant)
**Explication** : M(11) = 14641/256 est terminant, donc D(11) = -11.

---

## Approche de Résolution

### Stratégie
1. Pour chaque N, calculer X = round(N/e) pour obtenir le nombre optimal de parties.
2. Calculer M(N) = (N/X)^X.
3. Vérifier si M(N) est un nombre décimal terminant en vérifiant si N/X peut être exprimé comme k/(2^c × 5^d).
4. Diviser X par gcd(N, X) et continuer à diviser par 2 et 5. Si on obtient 1, c'est terminant.

### Observations
- Le maximum est atteint lorsque les parties sont aussi égales que possible.
- Un nombre décimal est terminant si et seulement si son dénominateur (sous forme réduite) n'a que des facteurs 2 et 5.
- On peut utiliser round(N/e) pour approximer le nombre optimal de parties.

### Complexité
- **Temporelle** : O(N) pour itérer sur tous les N de 5 à 10000.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>
#include <algorithm>

using namespace std;

int compute(int limit) {
    int total = 0;
    for (int N = 5; N <= limit; N++) {
        int X = round((double)N / M_E);
        int den = X / __gcd(N, X);
        
        while (den % 2 == 0) {
            den /= 2;
        }
        while (den % 5 == 0) {
            den /= 5;
        }
        
        if (den == 1) {
            total -= N;
        } else {
            total += N;
        }
    }
    return total;
}

int main() {
    cout << compute(10000) << endl;
    return 0;
}
```
