# Primes with Runs

## Description du Problème
En considérant les nombres premiers à 4 chiffres contenant des chiffres répétés, il est clair qu'ils ne peuvent pas tous être identiques : 1111 est divisible par 11, 2222 est divisible par 22, etc. Mais il y a neuf nombres premiers à 4 chiffres contenant trois uns :
1117, 1151, 1171, 1181, 1511, 1811, 2111, 4111, 8111.

Nous dirons que M(n,d) représente le nombre maximum de chiffres répétés pour un nombre premier à n chiffres où d est le chiffre répété, N(n,d) représente le nombre de tels nombres premiers, et S(n,d) représente la somme de ces nombres premiers.

Ainsi M(4,1)=3 est le nombre maximum de chiffres répétés pour un nombre premier à 4 chiffres où un est le chiffre répété, il y a N(4,1)=9 tels nombres premiers, et la somme de ces nombres premiers est S(4,1)=22275. Il s'avère que pour d=0, il n'est possible d'avoir que M(4,0)=2 chiffres répétés, mais il y a N(4,0)=13 tels cas.

Pour d=0 à 9, la somme de tous les S(4,d) est 273700.

**Objectif** : Trouver la somme de tous les S(10,d).

---

## Exemples
### Exemple 1
**Entrée** : `n = 4, d = 1`
**Sortie** : `22275`
**Explication** : S(4,1) = 22275, la somme des 9 nombres premiers à 4 chiffres avec trois uns.

---

## Approche de Résolution

### Stratégie
1. Pour chaque chiffre d de 0 à 9, trouver le nombre maximum de répétitions M(10,d).
2. Pour chaque nombre de répétitions de M(10,d) jusqu'à 0, essayer toutes les façons de remplir les positions restantes.
3. Générer toutes les permutations des chiffres et vérifier si le nombre est premier.
4. Sommer tous les nombres premiers trouvés pour obtenir S(10,d).

### Observations
- On commence par le nombre maximum de répétitions et on diminue jusqu'à trouver des nombres premiers.
- On doit éviter les zéros en tête.
- On utilise un crible de primalité pour vérifier efficacement.

### Complexité
- **Temporelle** : O(10 × 9^(10-rep) × 10! × log n) pour tester toutes les combinaisons.
- **Spatiale** : O(√n) pour le crible de primalité.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>

const int DIGITS = 10;

std::vector<int> primes;

bool isPrime(long n) {
    for (int p : primes) {
        if (n % p == 0) return false;
    }
    return true;
}

long toInteger(const std::vector<int>& digits) {
    long result = 0;
    for (int x : digits) {
        result = result * 10 + x;
    }
    return result;
}

int main() {
    // Générer les nombres premiers jusqu'à √(10^DIGITS)
    int limit = sqrt(pow(10, DIGITS));
    // ... générer primes ...
    
    long total = 0;
    for (int digit = 0; digit < 10; digit++) {
        for (int rep = DIGITS; rep >= 0; rep--) {
            long sum = 0;
            long count = pow(9, DIGITS - rep);
            
            for (long i = 0; i < count; i++) {
                std::vector<int> digits(DIGITS);
                std::fill(digits.begin(), digits.begin() + rep, digit);
                
                long temp = i;
                for (int j = 0; j < DIGITS - rep; j++) {
                    int d = temp % 9;
                    if (d >= digit) d++;
                    digits[DIGITS - 1 - j] = d;
                    temp /= 9;
                }
                
                std::sort(digits.begin(), digits.end());
                do {
                    if (digits[0] > 0) {
                        long num = toInteger(digits);
                        if (isPrime(num)) {
                            sum += num;
                        }
                    }
                } while (std::next_permutation(digits.begin(), digits.end()));
            }
            
            if (sum > 0) {
                total += sum;
                break;
            }
        }
    }
    
    std::cout << total << std::endl;
    return 0;
}
```
