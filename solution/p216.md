# Investigating the primality of numbers of the form 2n²-1

## Description du Problème
Considérons les nombres `t(n) = 2n² - 1` pour `n > 1`.

**Objectif** : Combien de nombres `t(n)` sont premiers pour `2 ≤ n ≤ 50 000 000` ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 2`
**Sortie** : `t(2) = 7` (premier)
**Explication** : `t(2) = 2×2² - 1 = 7`, qui est premier.

---

## Approche de Résolution

### Stratégie
1. Générer la séquence `t(n) = 2n² - 1` pour tous les `n` de 2 à la limite.
2. Utiliser un crible modifié :
   - Pour chaque `i` en ordre croissant :
     - Si `t(i)` n'a pas été modifié, alors `t(i)` est premier.
     - Si `t(i) > 1`, alors `t(i)` est un facteur premier `p`.
     - Pour tous les `j` de la forme `kp ± i` (où `j > i`), diviser tous les facteurs de `p` de `t(j)`.
3. Le lemme clé : si `p` divise `t(n)`, alors `p` divise aussi `t(kp ± n)` pour tous les `k` valides.

### Observations
- Aucun `t(n)` n'est divisible par 2, 3 ou 5.
- Si `p` divise `t(n)`, alors `p` divise `t(j)` si et seulement si `j ≡ ±n (mod p)`.
- On peut utiliser un crible similaire au crible d'Ératosthène.

### Complexité
- **Temporelle : O(n log n)**.
- **Spatiale : O(n)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

const int LIMIT = 50000000;

int main() {
    vector<long long> sequence(LIMIT + 1);
    sequence[0] = sequence[1] = -1;
    for (int i = 2; i <= LIMIT; i++)
        sequence[i] = 2LL * i * i - 1;
    
    int count = 0;
    for (int i = 2; i <= LIMIT; i++) {
        long long term = sequence[i];
        if (term == 2LL * i * i - 1)
            count++;
        
        if (1 < term && term <= LIMIT * 2LL) {
            int p = (int)term;
            // j = kp + i
            for (int j = i + p; j <= LIMIT; j += p) {
                while (sequence[j] % p == 0)
                    sequence[j] /= p;
            }
            // j = kp - i
            int start = i + (p - i) * 2 % p;
            for (int j = start; j <= LIMIT; j += p) {
                while (sequence[j] % p == 0)
                    sequence[j] /= p;
            }
        }
    }
    
    cout << count << endl;
    return 0;
}
```
