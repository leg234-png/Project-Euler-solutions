# Tribonacci non-divisors

## Description du Problème
La suite de Tribonacci est définie par :
- `T₁ = T₂ = T₃ = 1`
- `Tₙ = Tₙ₋₁ + Tₙ₋₂ + Tₙ₋₃` pour `n > 3`

Les premiers termes sont : 1, 1, 1, 3, 5, 9, 17, 31, 57, 105, 193, ...

**Objectif** : Trouver le 124ème nombre impair qui ne divise aucun terme de la suite de Tribonacci.

---

## Exemples
### Exemple 1
**Entrée** : `n = 1`
**Sortie** : Premier nombre impair qui ne divise aucun terme
**Explication** : On cherche les nombres impairs qui ne divisent aucun terme de la suite.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre impair `i`, vérifier s'il divise un terme de la suite de Tribonacci.
2. Utiliser l'algorithme de détection de cycle de Floyd :
   - La suite modulo `i` est périodique.
   - Si un terme est 0 modulo `i`, alors `i` divise un terme.
   - Si on détecte un cycle sans avoir trouvé 0, alors `i` ne divise aucun terme.
3. Compter les nombres impairs qui ne divisent aucun terme jusqu'à en trouver 124.

### Observations
- La suite modulo `m` est périodique (au plus `m³` états possibles).
- On peut utiliser la détection de cycle pour éviter de calculer tous les termes.

### Complexité
- **Temporelle : O(m³)` pour chaque `m` testé.
- **Spatiale : O(1)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

void tribonacci(vector<int>& state, int mod) {
    int a = state[0];
    int b = state[1];
    int c = state[2];
    state[0] = b;
    state[1] = c;
    state[2] = (a + b + c) % mod;
}

bool hasTribonacciMultiple(int modulus) {
    vector<int> slow = {1, 1, 1};
    vector<int> fast = {1, 1, 1};
    bool head = true;
    
    while (true) {
        if (slow[0] % modulus == 0)
            return true;
        if (!head && slow == fast)
            return false;
        tribonacci(slow, modulus);
        tribonacci(fast, modulus);
        tribonacci(fast, modulus);
        head = false;
    }
}

int main() {
    const int INDEX = 124;
    int count = 0;
    for (int i = 1; ; i += 2) {
        if (!hasTribonacciMultiple(i)) {
            count++;
            if (count == INDEX) {
                cout << i << endl;
                break;
            }
        }
    }
    return 0;
}
```
