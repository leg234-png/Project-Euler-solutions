# (prime-k) factorielle

## Description du Problème
Pour un nombre premier p, on définit S(p) = (Σ((p-k)!) mod p pour 1 <= k <= 5.

Par exemple, si p = 7, (7-1)! + (7-2)! + (7-3)! + (7-4)! + (7-5)! = 6! + 5! + 4! + 3! + 2! = 720 + 120 + 24 + 6 + 2 = 872.

Comme 872 mod 7 = 4, S(7) = 4.

**Objectif** : Trouver ΣS(p) pour 5 <= p <= 10⁸.

---

## Exemples
### Exemple 1
**Entrée** : `p = 7`
**Sortie** : `4`
**Explication** : S(7) = (6! + 5! + 4! + 3! + 2!) mod 7 = 872 mod 7 = 4.

---

## Approche de Résolution

### Stratégie
1. Utiliser le théorème de Wilson : (p-1)! ≡ -1 (mod p) pour p premier.
2. Simplifier l'expression :
   - (p-5)! + (p-4)! + (p-3)! + (p-2)! + (p-1)!
   - = (p-5)! × (1 + (p-4) + (p-4)(p-3) + (p-4)(p-3)(p-2) + (p-4)(p-3)(p-2)(p-1))
   - = (p-5)! × (1 + (-4) + (-4)(-3) + (-4)(-3)(-2) + (-4)(-3)(-2)(-1))
   - = (p-5)! × (1 - 4 + 12 - 24 + 24)
   - = (p-5)! × 9
   - = (p-1)! / ((p-1)(p-2)(p-3)(p-4)) × 9
   - = (p-1)! / 24 × 9
   - = (p-1)! × 3 / 8
   - = -1 × 3 / 8 (par le théorème de Wilson)
   - = -3/8 mod p
   - = (p - 3) × (8^(-1) mod p) mod p
3. Pour chaque nombre premier p >= 5 :
   - Calculer l'inverse modulaire de 8 modulo p.
   - Calculer S(p) = (p - 3) × (8^(-1) mod p) mod p.
4. Sommer tous les S(p).

### Observations
- On utilise le théorème de Wilson pour simplifier l'expression.
- L'inverse modulaire peut être calculé efficacement avec l'algorithme d'Euclide étendu.
- Tous les calculs sont effectués modulo p.

### Complexité
- **Temporelle** : O(P × log p), où P est le nombre de nombres premiers jusqu'à 10⁸.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

const int LIMIT = 100000000;

vector<bool> listPrimality(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    return isPrime;
}

// Calcule l'inverse modulaire de a modulo m (algorithme d'Euclide étendu)
long long modInverse(long long a, long long m) {
    long long m0 = m;
    long long y = 0, x = 1;
    
    if (m == 1) {
        return 0;
    }
    
    while (a > 1) {
        long long q = a / m;
        long long t = m;
        
        m = a % m;
        a = t;
        t = y;
        
        y = x - q * y;
        x = t;
    }
    
    if (x < 0) {
        x += m0;
    }
    
    return x;
}

int s(int p) {
    long long inv8 = modInverse(8 % p, p);
    return (int)((long long)(p - 3) * inv8 % p);
}

int main() {
    vector<bool> isPrime = listPrimality(LIMIT);
    long long sum = 0;
    
    for (int i = 5; i < isPrime.size(); i++) {
        if (isPrime[i]) {
            sum += s(i);
        }
    }
    
    cout << sum << endl;
    return 0;
}
```
