# Digit Power Sum

## Description du Problème
Le nombre 512 est intéressant car il est égal à la somme de ses chiffres élevée à une certaine puissance : 5+1+2=8, et 8³=512. Un autre exemple d'un nombre avec cette propriété est 614656=28⁴.

Nous définirons a_n comme étant le n-ième terme de cette séquence et insistons sur le fait qu'un nombre doit contenir au moins deux chiffres pour avoir une somme.

On vous donne que a₂=512 et a₁₀=614656.

**Objectif** : Trouver a₃₀.

---

## Exemples
### Exemple 1
**Entrée** : `n = 2`
**Sortie** : `512`
**Explication** : 5+1+2=8, 8³=512.

### Exemple 2
**Entrée** : `n = 10`
**Sortie** : `614656`
**Explication** : 6+1+4+6+5+6=28, 28⁴=614656.

---

## Approche de Résolution

### Stratégie
1. Générer tous les candidats de la forme n^k où n ≥ 2, k ≥ 2.
2. Pour chaque candidat, vérifier si la somme de ses chiffres élevée à une puissance égale le nombre.
3. Trier tous les candidats valides.
4. Retourner le 30ème élément.

### Observations
- On génère les puissances n^k pour différentes valeurs de n et k.
- On vérifie si digitSum(n^k)^k = n^k.
- On doit limiter la recherche pour éviter les nombres trop grands.

### Complexité
- **Temporelle** : O(k × n × log(n^k)) pour générer et vérifier les candidats.
- **Spatiale** : O(nombre_de_candidats) pour stocker et trier.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <gmpxx.h>

int digitSum(const mpz_class& x) {
    std::string s = x.get_str();
    int sum = 0;
    for (char c : s) {
        sum += c - '0';
    }
    return sum;
}

bool isDigitSumPower(const mpz_class& x) {
    int ds = digitSum(x);
    if (ds == 1) return false;
    
    mpz_class base = ds;
    mpz_class pow = base;
    while (pow < x) {
        pow *= base;
    }
    return pow == x;
}

int main() {
    const int INDEX = 30;
    std::set<mpz_class> candidates;
    
    mpz_class limit = 1;
    limit <<= 64;  // Limite initiale
    
    while (candidates.size() < INDEX) {
        candidates.clear();
        for (int k = 2; mpz_class(1) << k < limit; k++) {
            for (int n = 2; ; n++) {
                mpz_class pow = mpz_class(n).pow(k);
                if (pow >= limit && pow.get_str().length() * 9 < n) {
                    break;
                }
                if (pow >= 10 && isDigitSumPower(pow)) {
                    candidates.insert(pow);
                }
            }
        }
        if (candidates.size() < INDEX) {
            limit <<= 8;
        }
    }
    
    std::vector<mpz_class> sorted(candidates.begin(), candidates.end());
    std::cout << sorted[INDEX - 1] << std::endl;
    return 0;
}
```
