# Prime Square Remainders

## Description du Problème
Soit pₙ le n-ième nombre premier : 2, 3, 5, 7, 11, …, et soit r le reste lorsque (pₙ-1)ⁿ + (pₙ+1)ⁿ est divisé par pₙ².

Par exemple, lorsque n=3, p₃=5, et 4³ + 6³ = 280 ≡ 5 mod 25.

La plus petite valeur de n pour laquelle le reste dépasse d'abord 10⁹ est 7037.

**Objectif** : Trouver la plus petite valeur de n pour laquelle le reste dépasse d'abord 10¹⁰.

---

## Exemples
### Exemple 1
**Entrée** : `n = 3`
**Sortie** : `5`
**Explication** : (5-1)³ + (5+1)³ = 64 + 216 = 280 ≡ 5 mod 25.

---

## Approche de Résolution

### Stratégie
1. Utiliser le résultat du problème 120 : (a-1)ⁿ + (a+1)ⁿ mod a² = 2an si n est impair.
2. Pour n impair, le reste est 2an (sans modulo car a > 2n pour n ≥ 5).
3. Parcourir les nombres premiers impairs (n impair) et vérifier quand 2npₙ > 10¹⁰.
4. Retourner le premier n qui satisfait la condition.

### Observations
- On peut ignorer les n pairs car le reste serait 2, qui est trop petit.
- Pour n ≥ 5, on a pₙ > 2n, donc 2npₙ > 4n², ce qui dépasse rapidement 10¹⁰.

### Complexité
- **Temporelle** : O(n log log n) pour générer les nombres premiers, puis O(n) pour parcourir.
- **Spatiale** : O(n) pour stocker les nombres premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

std::vector<int> listPrimes(int limit) {
    std::vector<bool> isPrime(limit + 1, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i * i <= limit; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= limit; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    std::vector<int> primes;
    for (int i = 2; i <= limit; i++) {
        if (isPrime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

int main() {
    const long long THRESHOLD = 10000000000LL;
    std::vector<int> primes = listPrimes(1000000);
    
    for (int n = 5; n <= primes.size(); n += 2) {
        long long rem = (long long)n * primes[n - 1] * 2;
        if (rem > THRESHOLD) {
            std::cout << n << std::endl;
            return 0;
        }
    }
    return 0;
}
```
