# Triplets premiers

## Description du Problème
Chaque entier positif a jusqu'à huit voisins dans le triangle.

Un ensemble de trois nombres premiers est appelé un triplet premier si l'un des trois nombres premiers a les deux autres comme voisins dans le triangle.

Par exemple, dans la deuxième rangée, les nombres premiers 2 et 3 sont des éléments d'un triplet premier.

Si la rangée 8 est considérée, elle contient deux nombres premiers qui sont des éléments d'un triplet premier, c'est-à-dire 29 et 31.

**Objectif** : Si la rangée 9 est considérée, elle contient seulement un nombre premier qui est un élément d'un triplet premier : 37. Trouver la somme de tous les nombres premiers dans la rangée n qui sont des éléments d'un triplet premier, pour n = 5678027 et 7208785.

---

## Exemples
### Exemple 1
**Entrée** : `n = 8`
**Sortie** : `60` (29 + 31)
**Explication** : Les nombres premiers 29 et 31 dans la rangée 8 sont des éléments de triplets premiers.

---

## Approche de Résolution

### Stratégie
1. Générer le triangle et identifier les positions des nombres premiers.
2. Pour chaque nombre premier, vérifier s'il a deux voisins qui sont aussi premiers.
3. Utiliser un crible segmenté pour générer les nombres premiers dans la plage nécessaire.
4. Calculer les indices des voisins dans le triangle et vérifier leur primalité.

### Observations
- Le triangle a une structure spécifique où chaque nombre a jusqu'à 8 voisins.
- On doit générer les nombres premiers dans une grande plage pour les grandes rangées.
- Un crible segmenté est nécessaire pour gérer les grandes valeurs.

### Complexité
- **Temporelle** : O(n log log n) pour le crible, où n est la valeur maximale dans la rangée.
- **Spatiale** : O(√n) pour le crible segmenté.
