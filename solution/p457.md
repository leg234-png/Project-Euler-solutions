# Un polynôme modulo le carré d'un nombre premier

## Description du Problème
Soit f(n) = n² - 3n - 1.

Soit R(p) le plus petit entier positif n tel que f(n) mod p² = 0 si un tel entier n existe, sinon R(p) = 0.

**Objectif** : Soit SR(L) = ∑R(p) pour tous les nombres premiers ne dépassant pas L. Trouver SR(10⁷).

---

## Exemples
### Exemple 1
**Entrée** : `p = nombre premier`
**Sortie** : `R(p)`
**Explication** : Le plus petit n tel que f(n) mod p² = 0.

---

## Approche de Résolution

### Stratégie
1. Si on a un tel nombre, alors n² - 3n - 1 = kp² => n² - 3n - (1 + kp²) = 0.
2. n = (3 ± √(13 + 4kp²)) / 2.
3. n doit être un entier, donc 13 + 4kp² = q².
4. Noter que 13 + 4kp² est impair => q² est impair => q est impair, soit q = 2x + 1.
5. On obtient q² = 13 (mod p²).
6. Utiliser le critère d'Euler pour vérifier si 13 est un résidu quadratique modulo p.
7. Utiliser l'algorithme de Tonelli-Shanks pour trouver q tel que q² = 13 mod p.
8. Utiliser le lemme de Hensel pour élever à p².

### Observations
- On doit résoudre une équation quadratique modulo p².
- Le lemme de Hensel permet d'élever une solution de p à p².
- On doit vérifier la condition de résidu quadratique d'abord.

### Complexité
- **Temporelle** : O(P × log² p) où P est le nombre de nombres premiers.
- **Spatiale** : O(1).
