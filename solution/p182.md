# RSA encryption

## Description du Problème
L'algorithme RSA implique le chiffrement et le déchiffrement de messages. Pour un message `m` et une clé publique `(n, e)`, le message chiffré est `c = m^e mod n`.

Un message `m` est dit "non dissimulé" si `m^e ≡ m (mod n)`.

**Objectif** : Pour `n = p × q` où `p = 1009` et `q = 3643`, trouver la somme de toutes les valeurs de `e` (1 < e < φ(n)) pour lesquelles le nombre de messages non dissimulés est minimal.

---

## Exemples
### Exemple 1
**Entrée** : `p = 3, q = 5, e = 3`
**Sortie** : Nombre de messages non dissimulés
**Explication** : Un message `m` est non dissimulé si `m^3 ≡ m (mod 15)`.

---

## Approche de Résolution

### Stratégie
1. Pour chaque `e` tel que `gcd(e, p-1) = 1` et `gcd(e, q-1) = 1` :
   - Calculer le nombre de messages non dissimulés modulo `p` : `countUnconcealed(p, e)`.
   - Calculer le nombre de messages non dissimulés modulo `q` : `countUnconcealed(q, e)`.
2. Par le théorème des restes chinois, le nombre total de messages non dissimulés = `countUnconcealed(p, e) × countUnconcealed(q, e)`.
3. Trouver le minimum de ce produit.
4. Additionner tous les `e` qui atteignent ce minimum.

### Observations
- Un message `m` est non dissimulé si `m^e ≡ m (mod p)` et `m^e ≡ m (mod q)`.
- On peut utiliser le théorème des restes chinois pour combiner les résultats.

### Complexité
- **Temporelle : O(φ(n) × (p + q))**, où φ(n) = (p-1)(q-1).
- **Spatiale : O(p + q)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <climits>

using namespace std;

const int P = 1009;
const int Q = 3643;
const int TOTIENT = (P - 1) * (Q - 1);

int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

long long powMod(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp % 2 == 1)
            result = (result * base) % mod;
        base = (base * base) % mod;
        exp /= 2;
    }
    return result;
}

int countUnconcealed(int modulus, int e) {
    int count = 0;
    for (int m = 0; m < modulus; m++) {
        if (powMod(m, e, modulus) == m)
            count++;
    }
    return count;
}

vector<int> countAllUnconcealed(int prime) {
    vector<int> numUnconcealed(prime - 1, INT_MAX);
    for (int e = 0; e < prime - 1; e++) {
        if (gcd(e, prime - 1) == 1)
            numUnconcealed[e] = countUnconcealed(prime, e);
    }
    return numUnconcealed;
}

int main() {
    vector<int> numUnconcealedP = countAllUnconcealed(P);
    vector<int> numUnconcealedQ = countAllUnconcealed(Q);
    
    int minUnconcealedP = *min_element(numUnconcealedP.begin(), numUnconcealedP.end());
    int minUnconcealedQ = *min_element(numUnconcealedQ.begin(), numUnconcealedQ.end());
    
    long long sum = 0;
    for (int e = 0; e < TOTIENT; e++) {
        if (numUnconcealedP[e % (P - 1)] == minUnconcealedP && 
            numUnconcealedQ[e % (Q - 1)] == minUnconcealedQ)
            sum += e;
    }
    
    cout << sum << endl;
    return 0;
}
```
