# XOR-premiers

## Description du Problème
Un XOR-premier est un entier n supérieur à 1 qui n'est pas un produit XOR de deux entiers supérieurs à 1.

Par exemple, 9 n'est pas un XOR-premier car 9 = 3⊗3.

De même, 5 = 3⊗3 n'est pas un XOR-premier.

Les premiers XOR-premiers sont 2, 3, 7, 11, 13, ... et le 10-ième XOR-premier est 41.

**Objectif** : Trouver le 5000000-ième XOR-premier.

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : `41`
**Explication** : Le 10-ième XOR-premier est 41.

---

## Approche de Résolution

### Stratégie
1. Les XOR-premiers sont les polynômes irréductibles sur GF(2).
2. Utiliser un crible pour trouver tous les XOR-premiers.
3. Pour chaque nombre, vérifier s'il peut être factorisé comme produit XOR.
4. Si non, c'est un XOR-premier.
5. Collecter jusqu'à obtenir le 5000000-ième.

### Observations
- Les XOR-premiers correspondent aux polynômes irréductibles sur GF(2).
- On peut utiliser des techniques de théorie des nombres finis.
- Il existe des algorithmes efficaces pour générer les polynômes irréductibles.

### Complexité
- **Temporelle** : O(N²) pour vérifier tous les nombres.
- **Spatiale** : O(N) pour stocker les XOR-premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>

using namespace std;

long long xor_product(long long x, long long y) {
    long long prod = 0;
    while (y != 0) {
        if (y % 2 == 1) {
            prod ^= x;
        }
        x <<= 1;
        y >>= 1;
    }
    return prod;
}

bool is_xor_prime(long long n) {
    for (long long x = 2; x * x <= n; x++) {
        for (long long y = x; y * y <= n; y++) {
            if (xor_product(x, y) == n) {
                return false;
            }
        }
    }
    return true;
}

long long compute(int target) {
    vector<long long> xor_primes;
    long long n = 2;
    
    while (xor_primes.size() < target) {
        if (is_xor_prime(n)) {
            xor_primes.push_back(n);
        }
        n++;
    }
    
    return xor_primes[target - 1];
}

int main() {
    cout << compute(5000000) << endl;
    return 0;
}
```
