# Generalised Hamming Numbers

## Description du Problème
Un nombre de Hamming de type n est un nombre positif qui n'a pas de facteur premier supérieur à n.

Ainsi, les nombres de Hamming de type 5 sont 1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15, ...

**Objectif** : Combien de nombres de Hamming de type 100 sont inférieurs à 10^9 ?

---

## Exemples
### Exemple 1
**Entrée** : `type = 5, limit = 20`
**Sortie** : `13`
**Explication** : Les nombres de Hamming de type 5 < 20 sont : 1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15, 16, 18.

---

## Approche de Résolution

### Stratégie
1. Générer tous les nombres premiers ≤ 100.
2. Utiliser une récursion pour générer tous les nombres de Hamming :
   - `count(primeIndex, product)` : compte les nombres de Hamming en multipliant `product` par les nombres premiers restants.
   - Pour chaque nombre premier `p` à partir de `primeIndex` :
     - Multiplier `product` par `p` jusqu'à ce que `product > LIMIT`.
     - Récursivement compter avec le nombre premier suivant.
3. Le cas de base : si tous les nombres premiers ont été utilisés, retourner 1 si `product ≤ LIMIT`, sinon 0.

### Observations
- Un nombre de Hamming de type n est un produit de puissances de nombres premiers ≤ n.
- On peut générer tous ces nombres récursivement en multipliant par les nombres premiers disponibles.

### Complexité
- **Temporelle : O(2^π(n))**, où π(n) est le nombre de nombres premiers ≤ n.
- **Spatiale : O(π(n))** pour la pile de récursion.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

const long long LIMIT = 1000000000;

vector<int> listPrimes(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i)
                isPrime[j] = false;
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i])
            primes.push_back(i);
    }
    return primes;
}

int count(const vector<int>& primes, int primeIndex, long long product) {
    if (primeIndex == primes.size())
        return product <= LIMIT ? 1 : 0;
    else {
        int result = 0;
        while (product <= LIMIT) {
            result += count(primes, primeIndex + 1, product);
            product *= primes[primeIndex];
        }
        return result;
    }
}

int main() {
    vector<int> primes = listPrimes(100);
    cout << count(primes, 0, 1) << endl;
    return 0;
}
```
