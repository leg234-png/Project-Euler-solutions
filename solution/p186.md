# Connectedness of a network

## Description du Problème
Un générateur pseudo-aléatoire (Lagged Fibonacci Generator) génère des appels téléphoniques. Chaque appel connecte deux numéros de téléphone.

**Objectif** : Combien d'appels doivent être générés avant que 99% des numéros de téléphone (990 000 sur 1 000 000) soient connectés au numéro 524287 ?

---

## Exemples
### Exemple 1
**Entrée** : Réseau avec quelques connexions
**Sortie** : Nombre d'appels nécessaires
**Explication** : On utilise une structure Union-Find pour suivre les composantes connectées.

---

## Approche de Résolution

### Stratégie
1. Utiliser une structure Union-Find (Disjoint Set) pour gérer les connexions.
2. Générer des appels en utilisant le générateur LFG :
   - Pour les 55 premiers appels : `result = (100003 - 200003*k + 300007*k*k*k) % 1000000`.
   - Pour les appels suivants : `result = (history[24] + history[55]) % 1000000`.
3. Pour chaque appel entre `caller` et `callee` :
   - Si `caller != callee`, unir les deux composantes.
   - Vérifier la taille de la composante contenant 524287.
   - Si la taille ≥ 990000, arrêter.
4. Retourner le nombre d'appels générés.

### Observations
- Le générateur LFG utilise un historique circulaire de 55 éléments.
- On utilise l'union par rang et la compression de chemin pour optimiser Union-Find.

### Complexité
- **Temporelle : O(n × α(n))**, où n est le nombre d'appels et α est la fonction d'Ackermann inverse.
- **Spatiale : O(n)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

class DisjointSet {
private:
    struct Node {
        Node* parent;
        int rank;
        int size;
        
        Node() : parent(this), rank(0), size(1) {}
    };
    
    vector<Node*> nodes;
    
    Node* find(Node* node) {
        if (node->parent != node)
            node->parent = find(node->parent);
        return node->parent;
    }
    
public:
    DisjointSet(int size) {
        nodes.resize(size);
        for (int i = 0; i < size; i++)
            nodes[i] = new Node();
    }
    
    void unionSets(int i, int j) {
        Node* x = find(nodes[i]);
        Node* y = find(nodes[j]);
        if (x == y) return;
        
        if (x->rank == y->rank)
            x->rank++;
        else if (x->rank < y->rank)
            swap(x, y);
        
        y->parent = x;
        x->size += y->size;
        y->size = 0;
    }
    
    int size(int i) {
        return find(nodes[i])->size;
    }
};

class LfgRandom {
private:
    int k;
    vector<int> history;
    int index;
    
public:
    LfgRandom() : k(1), history(55, 0), index(0) {}
    
    int next() {
        int result;
        if (k <= 55) {
            result = (int)((100003LL - 200003LL*k + 300007LL*k*k*k) % 1000000);
            k++;
        } else {
            int idx24 = (index - 24 + 55) % 55;
            int idx55 = (index - 55 + 55) % 55;
            result = (history[idx24] + history[idx55]) % 1000000;
        }
        
        history[index] = result;
        index = (index + 1) % 55;
        return result;
    }
};

int main() {
    DisjointSet ds(1000000);
    LfgRandom rand;
    int i = 0;
    
    while (ds.size(524287) < 990000) {
        int caller = rand.next();
        int callee = rand.next();
        if (caller != callee) {
            ds.unionSets(caller, callee);
            i++;
        }
    }
    
    cout << i << endl;
    return 0;
}
```
