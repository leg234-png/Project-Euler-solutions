# Un nombre de Stoneham

## Description du Problème
Soit A = ∑_{i=1}^∞ 1/(3^i × 10^(3^i)).

Définissons A(n) comme les 10 chiffres décimaux à partir du n-ième chiffre.

Par exemple, A(100) = 4938271604 et A(10⁸) = 2584642393.

**Objectif** : Trouver A(10¹⁶).

---

## Exemples
### Exemple 1
**Entrée** : `n = 100`
**Sortie** : `4938271604`
**Explication** : A(100) = 4938271604.

---

## Approche de Résolution

### Stratégie
1. A = ∑_{i=1}^∞ 1/(3^i × 10^(3^i)).
2. A(n) = 10 chiffres après la n-ième position.
3. Si 3^k > n => 3^k × 10^(3^k) > n × 10^n, donc on n'a plus besoin de sommer les termes après cela.
4. On a S = ∑_{i=1}^k 1/(3^i × 10^(3^i)).
5. On cherche les 10 chiffres après la n-ième position.
6. On peut multiplier le tout par 10^(n-1).
7. Donc S₁ = ∑_{i=1}^k 10^(n - 3^i - 1)/3^i.
8. On veut la partie fractionnaire de ce qui reste.
9. Mais 10^(n - 3^i - 1) est énorme, donc on calcule 10^(n - 3^i - 1) mod 3^i.
10. On peut faire cela car r = 10^(n - 3^i - 1)/3^i - ⌊10^(n - 3^i - 1)/3^i⌋ = (10^(n - 3^i - 1) mod 3^i) / 3^i.

### Observations
- On peut utiliser l'exponentiation modulaire pour calculer les grandes puissances.
- On n'a besoin que de la partie fractionnaire.
- Le calcul modulaire permet d'éviter les grands nombres.

### Complexité
- **Temporelle** : O(log n × log 3^i) pour chaque terme.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <cmath>
#include <iomanip>

using namespace std;

string compute(long long n) {
    int limit = (int)log(n) / log(3);
    double total = 0.0;
    
    for (int i = 1; i <= limit; i++) {
        long long div = pow(3, i);
        long long exp = n - div - 1;
        if (exp < 0) continue;
        
        long long temp = 1;
        long long base = 10;
        long long exp_temp = exp;
        while (exp_temp > 0) {
            if (exp_temp % 2 == 1) {
                temp = (temp * base) % div;
            }
            base = (base * base) % div;
            exp_temp /= 2;
        }
        
        total += (double)temp / div;
    }
    
    double fractional = total - floor(total);
    string result = to_string(fractional);
    return result.substr(2, 10);
}

int main() {
    long long n = 1;
    for (int i = 0; i < 16; i++) {
        n *= 10;
    }
    cout << compute(n) << endl;
    return 0;
}
```
