# Différence d'exposant

## Description du Problème
Soit v_p(n) le plus grand entier r tel que p^r divise n.

Soit D(n, m) = ∑_{p premier} |v_p(n) - v_p(m)|.

Soit S(N) = ∑_{1 ≤ n, m ≤ N} D(n, m).

On vous donne que S(10) = 210 et S(10²) = 37018.

**Objectif** : Trouver S(10¹²). Donner votre réponse modulo 1000000007.

---

## Exemples
### Exemple 1
**Entrée** : `N = 10`
**Sortie** : `210`
**Explication** : S(10) = 210.

---

## Approche de Résolution

### Stratégie
1. S(N) = ∑_{1 ≤ n, m ≤ N} D(n, m) = ∑_{p ≤ N} ∑_{1 ≤ n, m ≤ N} |v_p(n) - v_p(m)|.
2. Pour chaque nombre premier p, calculer la contribution séparément.
3. Pour p, trouver combien de nombres ont v_p(n) = k pour chaque k.
4. floor(N/p^r) - floor(N/p^(r+1)) donne le nombre de n avec v_p(n) = r.
5. Calculer ∑_{1 ≤ n, m ≤ N} |v_p(n) - v_p(m)| en utilisant ces comptages.
6. Sommer sur tous les nombres premiers.

### Observations
- On peut traiter chaque nombre premier indépendamment.
- Le calcul de |v_p(n) - v_p(m)| peut être simplifié.
- On peut utiliser des formules pour calculer efficacement les sommes.

### Complexité
- **Temporelle** : O(π(N) × log N) où π(N) est le nombre de nombres premiers ≤ N.
- **Spatiale** : O(log N) pour stocker les comptages.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long mod = 1000000007;

vector<int> prime_sieve(long long n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (long long i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (long long j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (long long i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

long long contribution(long long N, int p) {
    vector<long long> counts;
    long long power = 1;
    int exp = 0;
    
    while (power <= N) {
        long long count = N / power - N / (power * p);
        counts.push_back(count);
        power *= p;
        exp++;
    }
    
    long long total = 0;
    for (size_t i = 0; i < counts.size(); i++) {
        for (size_t j = 0; j < counts.size(); j++) {
            long long diff = abs((long long)i - (long long)j);
            total = (total + counts[i] * counts[j] % mod * diff) % mod;
        }
    }
    
    return total;
}

long long compute(long long N) {
    vector<int> primes = prime_sieve(N);
    long long total = 0;
    
    for (int p : primes) {
        total = (total + contribution(N, p)) % mod;
    }
    
    return total;
}

int main() {
    long long N = 1;
    for (int i = 0; i < 12; i++) {
        N *= 10;
    }
    cout << compute(N) << endl;
    return 0;
}
```
