# Nombre Triangulaire Très Divisible

## Description du Problème
La séquence des nombres triangulaires est générée en ajoutant les nombres naturels. Ainsi, le 7ème nombre triangulaire serait 1 + 2 + 3 + 4 + 5 + 6 + 7 = 28. Les dix premiers termes seraient :

1, 3, 6, 10, 15, 21, 28, 36, 45, 55, ...

Listons les facteurs des sept premiers nombres triangulaires :

1 : 1
3 : 1, 3
6 : 1, 2, 3, 6
10 : 1, 2, 5, 10
15 : 1, 3, 5, 15
21 : 1, 3, 7, 21
28 : 1, 2, 4, 7, 14, 28

On peut voir que 28 est le premier nombre triangulaire à avoir plus de cinq diviseurs.

**Objectif** : Quelle est la valeur du premier nombre triangulaire à avoir plus de cinq cents diviseurs ?

---

## Exemples
### Exemple 1
**Entrée** : Nombre minimum de diviseurs = 5
**Sortie** : `28`
**Explication** : 28 est le premier nombre triangulaire avec plus de 5 diviseurs (6 diviseurs).

---

## Approche de Résolution

### Stratégie
1. Générer les nombres triangulaires un par un : T(n) = 1 + 2 + ... + n = n(n+1)/2
2. Pour chaque nombre triangulaire, compter le nombre de diviseurs.
3. Arrêter dès qu'on trouve un nombre avec plus de 500 diviseurs.

### Observations
- Pour compter les diviseurs efficacement :
  - Parcourir de 1 à √n
  - Si i divise n, alors n/i divise aussi n
  - Compter 2 diviseurs pour chaque paire (i, n/i)
  - Si n est un carré parfait, compter le diviseur √n une seule fois
- On peut optimiser en utilisant la factorisation en nombres premiers, mais pour ce problème, la méthode directe est suffisante.

### Complexité
- **Temporelle** : O(n × √T(n)), où `n` est le nombre triangulaire recherché et `T(n)` sa valeur.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

int countDivisors(int n) {
    int count = 0;
    int sqrtN = sqrt(n);
    
    for (int i = 1; i < sqrtN; ++i) {
        if (n % i == 0) {
            count += 2;  // i et n/i sont des diviseurs
        }
    }
    
    if (sqrtN * sqrtN == n) {
        count++;  // Carré parfait
    }
    
    return count;
}

int main() {
    int triangle = 0;
    int i = 1;
    
    while (true) {
        triangle += i;
        if (countDivisors(triangle) > 500) {
            std::cout << triangle << std::endl;
            break;
        }
        ++i;
    }
    
    return 0;
}
```
