# Cribbage

## Description du Problème
Dans le jeu de cribbage, un joueur marque des points en comptant différentes combinaisons dans sa main :
- Paires. Une paire est deux cartes du même rang. Chaque paire vaut 2 points.
- Runs. Une run est un ensemble d'au moins 3 cartes dont les rangs sont consécutifs, par exemple 9, 10, Valet. Notez que l'As n'est jamais haut, donc Dame, Roi, As n'est pas une run valide. Le nombre de points pour chaque run est la taille de la run. Toutes les runs localement maximales sont comptées.
- Fifteens. Un fifteen est une combinaison de cartes qui a une valeur totale de 15. Chaque fifteen vaut 2 points.

**Objectif** : Calculer le score total pour certaines configurations de cartes.

**Explication** : Le score calculé selon les règles du cribbage.

## Approche de Résolution

### Stratégie
1. Pour chaque main de cartes, identifier toutes les paires.
2. Identifier toutes les runs (séquences consécutives de 3+ cartes).
3. Identifier toutes les combinaisons qui somment à 15.
4. Calculer le score total en sommant tous les points.

### Observations
- Les runs doivent être localement maximales.
- Les fifteens peuvent être formés avec n'importe quelle combinaison de cartes.
- L'As a une valeur de 1, pas 11.

### Complexité
- **Temporelle** : O(2^n) pour générer toutes les combinaisons.
- **Spatiale** : O(n) pour stocker les cartes.
