# Torricelli Triangles

## Description du Problème
Soit ABC un triangle dont tous les angles intérieurs sont inférieurs à 120 degrés. Soit X un point quelconque à l'intérieur du triangle et soit XA = p, XC = q et XB = r.

Fermat a mis au défi Torricelli de trouver la position de X telle que p + q + r soit minimisée.

Torricelli a pu prouver que si des triangles équilatéraux AOB, BNC et AMC sont construits sur chaque côté du triangle ABC, les cercles circonscrits de AOB, BNC et AMC se croiseront en un seul point, T, à l'intérieur du triangle. De plus, il a prouvé que T, appelé le point de Torricelli/Fermat, minimise p + q + r. Encore plus remarquable, on peut montrer que lorsque la somme est minimisée, AN = BM = CO = p + q + r et que AN, BM et CO se croisent également en T.

Si la somme est minimisée et que a, b, c, p, q et r sont tous des entiers positifs, nous appellerons le triangle ABC un triangle de Torricelli. Par exemple, a = 399, b = 455, c = 511 est un exemple de triangle de Torricelli, avec p + q + r = 784.

**Objectif** : Trouver la somme de toutes les valeurs distinctes de p + q + r ≤ 120000 pour les triangles de Torricelli.

---

## Exemples
### Exemple 1
**Entrée** : `a = 399, b = 455, c = 511`
**Sortie** : `784`
**Explication** : p + q + r = 784 pour ce triangle de Torricelli.

---

## Approche de Résolution

### Stratégie
1. Utiliser les propriétés géométriques du point de Torricelli.
2. Pour un triangle avec des côtés entiers, le point de Torricelli peut être calculé.
3. Vérifier si p, q, r sont entiers et p + q + r ≤ 120000.
4. Compter toutes les valeurs distinctes.

### Observations
- Le problème nécessite une compréhension approfondie de la géométrie.
- On peut utiliser des formules trigonométriques ou des méthodes numériques.

### Complexité
- **Temporelle** : O(n³) pour tester toutes les combinaisons de a, b, c.
- **Spatiale** : O(n) pour stocker les valeurs distinctes.
