# Modified Fibonacci Golden Nuggets

## Description du Problème
Considérons la série polynomiale infinie A_G(x) = x G₁ + x² G₂ + x³ G₃ + ⋯, où Gₖ est le k-ième terme de la relation de récurrence du second ordre Gₖ = G_{k-1} + G_{k-2}, G₁ = 1 et G₂ = 4 ; c'est-à-dire, 1, 4, 5, 9, 14, 23, ….

Pour ce problème, nous nous intéresserons aux valeurs de x pour lesquelles A_G(x) est un entier positif.

Les valeurs correspondantes de x pour les cinq premiers nombres naturels sont montrées ci-dessous.

Nous appellerons A_G(x) un "golden nugget" si x est rationnel, car ils deviennent de plus en plus rares ; par exemple, le 20ème golden nugget est 211345365.

**Objectif** : Trouver la somme des trente premiers golden nuggets.

---

## Exemples
### Exemple 1
**Entrée** : `x = 2/5`
**Sortie** : `2`
**Explication** : A_G(2/5) = 2.

---

## Approche de Résolution

### Stratégie
1. Similaire au problème 137, mais avec une séquence modifiée Gₖ.
2. La série A_G(x) peut être résolue analytiquement.
3. Les golden nuggets suivent une séquence récurrente basée sur la séquence G.
4. Calculer les 30 premiers golden nuggets et les sommer.

### Observations
- Les golden nuggets sont liés à la séquence G modifiée.
- On peut dériver une formule récurrente pour les golden nuggets.

### Complexité
- **Temporelle** : O(n), où n = 30.
- **Spatiale** : O(1).

---
