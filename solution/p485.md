# Nombre maximum de diviseurs

## Description du Problème
Soit d(n) le nombre de diviseurs de n.

Soit M(n,k) le maximum de d(j) pour n ≤ j ≤ n + k - 1.

Soit S(u,k) = ∑M(n,k) pour 1 ≤ n ≤ u - k + 1.

On vous donne que S(1000,10) = 17176.

**Objectif** : Trouver S(100000000, 100000).

---

## Exemples
### Exemple 1
**Entrée** : `u = 1000, k = 10`
**Sortie** : `17176`
**Explication** : S(1000,10) = 17176.

---

## Approche de Résolution

### Stratégie
1. Précalculer le plus petit facteur premier (SPF) pour chaque nombre jusqu'à u+k.
2. Utiliser le SPF pour obtenir rapidement la factorisation en nombres premiers.
3. Calculer d(n) pour tous les n en utilisant la factorisation.
4. Créer un tableau contenant tous les d(n).
5. Résoudre le problème de "fenêtre glissante" pour trouver M(n,k).
6. Comparer le nouvel élément de manière un peu brute force.

### Observations
- Le SPF permet de factoriser rapidement chaque nombre.
- Le problème de fenêtre glissante peut être optimisé avec une structure de données appropriée.
- On peut utiliser une deque ou un segment tree pour maintenir le maximum dans une fenêtre.

### Complexité
- **Temporelle** : O(u × log u) pour calculer tous les d(n), plus O(u × k) pour la fenêtre glissante.
- **Spatiale** : O(u) pour stocker d(n) et le SPF.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <deque>

using namespace std;

vector<int> spf_sieve(int N) {
    vector<int> spf(N + 1);
    for (int i = 0; i <= N; i++) {
        spf[i] = i;
    }
    
    for (int i = 2; i * i <= N; i++) {
        if (spf[i] == i) {
            for (int j = i * i; j <= N; j += i) {
                if (spf[j] == j) {
                    spf[j] = i;
                }
            }
        }
    }
    return spf;
}

int d(int n, const vector<int>& spf) {
    int total = 1;
    while (n != 1) {
        int p = spf[n];
        int count = 0;
        while (n % p == 0) {
            count++;
            n /= p;
        }
        total *= (count + 1);
    }
    return total;
}

long long compute(int u, int k) {
    vector<int> spf = spf_sieve(u + k);
    vector<int> divisors(u + k + 1);
    
    for (int i = 1; i <= u + k; i++) {
        divisors[i] = d(i, spf);
    }
    
    long long total = 0;
    deque<int> dq;
    
    // Initialiser la première fenêtre
    for (int i = 1; i < k; i++) {
        while (!dq.empty() && divisors[dq.back()] <= divisors[i]) {
            dq.pop_back();
        }
        dq.push_back(i);
    }
    
    for (int n = 1; n <= u - k + 1; n++) {
        // Ajouter le nouvel élément
        int new_idx = n + k - 1;
        while (!dq.empty() && divisors[dq.back()] <= divisors[new_idx]) {
            dq.pop_back();
        }
        dq.push_back(new_idx);
        
        // Retirer les éléments hors de la fenêtre
        while (!dq.empty() && dq.front() < n) {
            dq.pop_front();
        }
        
        total += divisors[dq.front()];
    }
    
    return total;
}

int main() {
    cout << compute(100000000, 100000) << endl;
    return 0;
}
```
