# Cercles tangents

## Description du Problème
Soit S(n) = ∑r_A + r_B + r_C, pour 0 < r_A ≤ r_B ≤ n où r_A, r_B et r_C sont des entiers.

La seule solution pour 0 < r_A ≤ r_B ≤ 5 est r_A = 4, r_B = 4 et r_C = 1, donc S(5) = 4+4+1 = 9.

On vous donne aussi S(100) = 3072.

**Objectif** : Trouver S(10⁹).

---

## Exemples
### Exemple 1
**Entrée** : `n = 5`
**Sortie** : `9`
**Explication** : S(5) = 9.

---

## Approche de Résolution

### Stratégie
1. On sait que 1/√c = 1/√b + 1/√a => √c = √a√b/(√a + √b).
2. Donc c = ab/(a+b+2√(ab)).
3. Soit g = gcd(a,b), alors a = ga₁, b = gb₁ où gcd(a₁, b₁) = 1.
4. Cela signifie c = a₁b₁g/(a₁+b₁+2√(a₁b₁)).
5. Et on sait que a₁b₁ doit être un entier => a₁b₁ = z², donc a₁ = x², b₁ = y².
6. Finalement : c = x²y²g/(x² + y² + 2xy) = (xy)²g/(x+y)².
7. Comme gcd(x, y) = 1, on conclut que g = n(x+y)².
8. a = n(x+y)²x², b = n(x+y)²y², c = nx²y².
9. Générer tous les (x,y) copremiers tels que x < y et y⁴ < limit, et trouver tous les multiples.

### Observations
- La condition de tangence donne une relation spécifique entre les rayons.
- On peut paramétrer les solutions en utilisant des triplets (x, y, n).
- La somme des multiples suit des formules de nombres triangulaires.

### Complexité
- **Temporelle** : O(√limit) pour générer tous les (x,y) copremiers.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long T(long long n) {
    return n * (n + 1) / 2;
}

long long compute(long long limit) {
    long long total = 0;
    long long sqrt_limit = sqrt(limit);
    
    for (long long y = 2; y <= sqrt_limit; y++) {
        for (long long x = 1; x < y; x++) {
            if (__gcd(x, y) == 1) {
                long long y4 = y * y * y * y;
                if (y4 >= limit) break;
                
                // n(x+y)²y² ≤ limit => n ≤ limit/((x+y)²y²)
                long long max_n = limit / ((x + y) * (x + y) * y * y);
                
                if (max_n > 0) {
                    // Somme de r_A + r_B + r_C pour tous les n
                    // r_A = n(x+y)²x², r_B = n(x+y)²y², r_C = nx²y²
                    // Somme = n(x+y)²(x² + y²) + nx²y² = n((x+y)²(x²+y²) + x²y²)
                    long long coeff = (x + y) * (x + y) * (x * x + y * y) + x * x * y * y;
                    total += T(max_n) * coeff;
                }
            }
        }
    }
    
    return total;
}

int main() {
    cout << compute(1000000000) << endl;
    return 0;
}
```
