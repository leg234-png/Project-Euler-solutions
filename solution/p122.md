# Efficient Exponentiation

## Description du Problème
La façon la plus naïve de calculer n¹⁵ nécessite quatorze multiplications :
n × n × ⋯ × n = n¹⁵.

Mais en utilisant une méthode "binaire", vous pouvez le calculer en six multiplications :
n × n = n²
n² × n² = n⁴
n⁴ × n⁴ = n⁸
n⁸ × n⁴ = n¹²
n¹² × n² = n¹⁴
n¹⁴ × n = n¹⁵

Cependant, il est encore possible de le calculer en seulement cinq multiplications :
n × n = n²
n² × n = n³
n³ × n³ = n⁶
n⁶ × n⁶ = n¹²
n¹² × n³ = n¹⁵

Nous définirons m(k) comme étant le nombre minimum de multiplications pour calculer nᵏ ; par exemple m(15)=5.

**Objectif** : Trouver ∑ₖ₌₁²⁰⁰ m(k).

---

## Exemples
### Exemple 1
**Entrée** : `k = 15`
**Sortie** : `5`
**Explication** : m(15) = 5 multiplications.

---

## Approche de Résolution

### Stratégie
1. Utiliser une recherche en profondeur limitée pour trouver les chaînes d'addition minimales.
2. Une chaîne d'addition est une séquence où chaque élément est la somme de deux éléments précédents.
3. Pour chaque k, trouver la chaîne d'addition la plus courte qui produit k.
4. Le nombre d'opérations est la longueur de la chaîne moins 1.

### Observations
- On peut utiliser une recherche progressive en profondeur (depth-limited search).
- On ne considère que les chaînes strictement croissantes.
- On utilise un backtracking pour explorer toutes les possibilités.

### Complexité
- **Temporelle** : Exponentielle dans le pire cas, mais gérable avec des optimisations.
- **Spatiale** : O(k) pour stocker la chaîne.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <climits>

const int LIMIT = 200;

std::vector<int> minOperations(LIMIT + 1, -1);
int numUnknown = LIMIT - 1;

void exploreChains(std::vector<int>& chain, int maxOps) {
    if (chain.size() > maxOps || numUnknown == 0) {
        return;
    }
    
    int max = chain.back();
    for (int i = chain.size() - 1; i >= 0; i--) {
        for (int j = i; j >= 0; j--) {
            int x = chain[i] + chain[j];
            if (x <= max) break;
            if (x <= LIMIT) {
                chain.push_back(x);
                if (minOperations[x] == -1) {
                    minOperations[x] = chain.size() - 1;
                    numUnknown--;
                }
                exploreChains(chain, maxOps);
                chain.pop_back();
            }
        }
    }
}

int main() {
    minOperations[0] = 0;
    minOperations[1] = 0;
    
    for (int ops = 1; numUnknown > 0; ops++) {
        std::vector<int> chain = {1};
        exploreChains(chain, ops);
    }
    
    int sum = 0;
    for (int x : minOperations) {
        sum += x;
    }
    std::cout << sum << std::endl;
    return 0;
}
```
