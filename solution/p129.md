# Repunit Divisibility

## Description du Problème
Un nombre composé entièrement de uns est appelé un repunit. Nous définirons R(k) comme étant un repunit de longueur k ; par exemple, R(6) = 111111.

Étant donné que n est un entier positif et gcd(n, 10) = 1, on peut montrer qu'il existe toujours une valeur, k, pour laquelle R(k) est divisible par n, et soit A(n) la plus petite telle valeur de k ; par exemple, A(7) = 6 et A(41) = 5.

La plus petite valeur de n pour laquelle A(n) dépasse d'abord dix est 17.

**Objectif** : Trouver la plus petite valeur de n pour laquelle A(n) dépasse d'abord un million.

---

## Exemples
### Exemple 1
**Entrée** : `n = 7`
**Sortie** : `6`
**Explication** : A(7) = 6, car R(6) = 111111 est divisible par 7.

### Exemple 2
**Entrée** : `n = 17`
**Sortie** : `> 10`
**Explication** : A(17) > 10, donc 17 est la plus petite valeur avec A(n) > 10.

---

## Approche de Résolution

### Stratégie
1. Parcourir n à partir de 10⁶ + 1.
2. Pour chaque n, calculer A(n) en trouvant le plus petit k tel que R(k) ≡ 0 mod n.
3. R(k) mod n peut être calculé récursivement : R(k) = (R(k-1) × 10 + 1) mod n.
4. Arrêter quand on trouve n tel que A(n) > 10⁶.

### Observations
- Par le principe des tiroirs, A(n) ≤ n pour n copremier avec 10.
- Donc on doit chercher n > 10⁶.
- On peut calculer R(k) mod n efficacement sans calculer R(k) entier.

### Complexité
- **Temporelle** : O(n × A(n)), où n est la valeur recherchée.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>

int findLeastDivisibleRepunit(int n) {
    if (n % 2 == 0 || n % 5 == 0) {
        return 0;  // Pas de solution
    }
    
    int sum = 1;  // Égal à R(k) mod n
    int pow = 1;  // Égal à 10^k mod n
    int k = 1;
    
    while (sum % n != 0) {
        k++;
        pow = (pow * 10) % n;
        sum = (sum + pow) % n;
    }
    
    return k;
}

int main() {
    const int LIMIT = 1000000;
    
    for (int n = LIMIT + 1; ; n++) {
        if (findLeastDivisibleRepunit(n) > LIMIT) {
            std::cout << n << std::endl;
            return 0;
        }
    }
    return 0;
}
```
