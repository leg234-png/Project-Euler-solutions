# Consecutive positive divisors

## Description du Problème
Trouver le nombre d'entiers `n`, `1 < n < 10^7`, pour lesquels `n` et `n+1` ont le même nombre de diviseurs positifs.

Par exemple, 14 a les diviseurs 1, 2, 7, 14 et 15 a les diviseurs 1, 3, 5, 15, donc 14 et 15 ont tous les deux 4 diviseurs.

---

## Exemples
### Exemple 1
**Entrée** : `n = 14`
**Sortie** : `14 et 15 ont tous les deux 4 diviseurs`
**Explication** : 14 a 4 diviseurs (1, 2, 7, 14) et 15 a 4 diviseurs (1, 3, 5, 15).

---

## Approche de Résolution

### Stratégie
1. Utiliser un crible modifié pour calculer le nombre de diviseurs de chaque nombre.
2. Initialiser `numDivisors[i] = 2` pour tous les `i` (chaque nombre a au moins 1 et lui-même).
3. Pour chaque `i` de 2 à `LIMIT` :
   - Pour chaque multiple `j = 2i, 3i, 4i, ...` jusqu'à `LIMIT` :
     - Incrémenter `numDivisors[j]` (car `i` est un diviseur de `j`).
4. Compter les paires consécutives `(i, i+1)` où `numDivisors[i] == numDivisors[i+1]`.

### Observations
- On peut calculer le nombre de diviseurs efficacement avec un crible.
- Pour chaque nombre `i`, on parcourt ses multiples et on incrémente leur compteur de diviseurs.

### Complexité
- **Temporelle : O(n log n)**, où n = 10^7 (crible).
- **Spatiale : O(n)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

const int LIMIT = 10000000;

int main() {
    vector<int> numDivisors(LIMIT + 1, 2);
    
    for (int i = 2; i < numDivisors.size(); i++) {
        for (int j = i * 2; j < numDivisors.size(); j += i)
            numDivisors[j]++;
    }
    
    int count = 0;
    for (int i = 2; i < numDivisors.size() - 1; i++) {
        if (numDivisors[i] == numDivisors[i + 1])
            count++;
    }
    
    cout << count << endl;
    return 0;
}
```
