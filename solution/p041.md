# Nombre Premier Pandigital

## Description du Problème
On dira qu'un nombre à n chiffres est pandigital s'il utilise tous les chiffres de 1 à n exactement une fois. Par exemple, 2143 est un pandigital à 4 chiffres et est aussi premier.

**Objectif** : Quel est le plus grand nombre premier pandigital à n chiffres qui existe ?

---

## Exemples
### Exemple 1
**Entrée** : Nombres pandigitaux premiers
**Sortie** : `7652413`
**Explication** : Le plus grand nombre premier pandigital.

---

## Approche de Résolution

### Stratégie
1. Commencer par n = 9 et descendre jusqu'à n = 1.
2. Pour chaque n :
   - Générer toutes les permutations des chiffres 1 à n.
   - Tester chaque permutation pour voir si elle est première.
   - Retourner le premier nombre premier trouvé (le plus grand pour ce n).
3. Si aucun n'est trouvé pour n, passer à n-1.

### Observations
- Pour n = 9, la somme des chiffres est 45, qui est divisible par 3, donc aucun nombre pandigital 1-9 n'est premier.
- Pour n = 8, la somme est 36, aussi divisible par 3.
- Pour n = 7, la somme est 28, donc il peut y avoir des nombres premiers.
- On peut utiliser l'algorithme de permutation suivante pour générer toutes les permutations.

### Complexité
- **Temporelle** : O(n! × √m), où `n` est le nombre de chiffres et `m` est la valeur du nombre.
- **Spatiale** : O(n), pour stocker les chiffres.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>

bool isPrime(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    
    int sqrtN = sqrt(n);
    for (int i = 3; i <= sqrtN; i += 2) {
        if (n % i == 0) {
            return false;
        }
    }
    return true;
}

int toInteger(const std::vector<int>& digits) {
    int result = 0;
    for (int d : digits) {
        result = result * 10 + d;
    }
    return result;
}

bool nextPermutation(std::vector<int>& arr) {
    int n = arr.size();
    int i = n - 2;
    
    while (i >= 0 && arr[i] >= arr[i + 1]) {
        i--;
    }
    
    if (i < 0) {
        return false;
    }
    
    int j = n - 1;
    while (arr[j] <= arr[i]) {
        j--;
    }
    
    std::swap(arr[i], arr[j]);
    std::reverse(arr.begin() + i + 1, arr.end());
    
    return true;
}

int main() {
    for (int n = 9; n >= 1; --n) {
        std::vector<int> digits;
        for (int i = 1; i <= n; ++i) {
            digits.push_back(i);
        }
        
        int result = -1;
        do {
            int num = toInteger(digits);
            if (isPrime(num)) {
                result = num;
            }
        } while (nextPermutation(digits));
        
        if (result != -1) {
            std::cout << result << std::endl;
            return 0;
        }
    }
    
    return 0;
}
```
