# Chiffres triés

## Description du Problème
Soit f(d) le nombre formé en triant les chiffres de d en ordre croissant, en supprimant les zéros en tête.

Par exemple, f(4321) = 1234, f(5500) = 55, f(9900) = 99, f(321) = 123.

Soit S(n) la somme de f(d) pour tous les entiers positifs d de n chiffres ou moins. On vous donne S(1) = 45 et S(5) = 1543545675.

**Objectif** : Trouver S(18). Donner votre réponse modulo 1123455689.

---

## Exemples
### Exemple 1
**Entrée** : `n = 1`
**Sortie** : `45`
**Explication** : S(1) = 45.

---

## Approche de Résolution

### Stratégie
1. Il y a (n+9)C9 chaînes croissantes (nombres avec chiffres non décroissants).
2. Penser aux étoiles et barres : placer 9 barres parmi n chiffres.
3. Pour chaque combinaison, compter combien il y en a en utilisant n!/∏(d_i!) où d_i est le nombre de fois que le chiffre i apparaît.
4. Par exemple, 00001 apparaît 5!/4! = 5 fois : 1, 10, 100, 1000, 10000.
5. Utiliser itertools pour générer toutes les combinaisons avec répétition.

### Observations
- On peut utiliser les combinaisons avec répétition pour générer tous les nombres possibles.
- Les factorielles peuvent être précalculées.
- On doit gérer l'arithmétique modulaire.

### Complexité
- **Temporelle** : O((n+9)C9 × n) pour toutes les combinaisons.
- **Spatiale** : O(n) pour les factorielles.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
#include <cmath>

using namespace std;

long long mod = 1123455689;

long long factorial_mod(int n, long long mod) {
    long long result = 1;
    for (int i = 2; i <= n; i++) {
        result = (result * i) % mod;
    }
    return result;
}

long long combs(string x) {
    int n = x.length();
    long long total = factorial_mod(n, mod);
    
    vector<int> count(10, 0);
    for (char c : x) {
        count[c - '0']++;
    }
    
    for (int cnt : count) {
        total = (total * mod_inverse(factorial_mod(cnt, mod), mod)) % mod;
    }
    
    return (total * stoll(x)) % mod;
}

long long S(int n, long long mod) {
    long long total = 0;
    
    // Générer toutes les combinaisons avec répétition de longueur n
    function<void(string, int, int)> generate = [&](string current, int pos, int last_digit) {
        if (pos == n) {
            total = (total + combs(current)) % mod;
            return;
        }
        
        for (int d = last_digit; d <= 9; d++) {
            generate(current + to_string(d), pos + 1, d);
        }
    };
    
    generate("", 0, 0);
    
    return total;
}

int main() {
    cout << S(18, mod) << endl;
    return 0;
}
```
