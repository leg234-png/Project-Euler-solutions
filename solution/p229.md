# Quatre représentations utilisant des carrés

## Description du Problème
Nous nous intéressons aux nombres n qui admettent des représentations de tous les quatre types suivants :

n = a₁² + b₁²
n = a₂² + 2b₂²
n = a₃² + 3b₃²
n = a₇² + 7b₇²

où les aₖ et bₖ sont des entiers positifs.

Il y a 75373 tels nombres qui ne dépassent pas 10⁷.

**Objectif** : Combien y a-t-il de tels nombres qui ne dépassent pas 2×10⁹ ?

---

## Exemples
### Exemple 1
**Entrée** : `limit = 10⁷`
**Sortie** : `75373`
**Explication** : Il y a 75373 nombres qui satisfont les quatre conditions.

---

## Approche de Résolution

### Stratégie
1. Générer tous les nombres de la forme a² + 7b² jusqu'à la limite.
2. Filtrer ceux qui peuvent aussi s'écrire a² + 3b².
3. Filtrer ceux qui peuvent aussi s'écrire a² + 2b².
4. Filtrer ceux qui peuvent aussi s'écrire a² + b².
5. Compter les nombres qui satisfont les quatre conditions.

### Observations
- On peut utiliser des ensembles pour stocker les nombres de chaque forme.
- On filtre progressivement en commençant par la forme la plus restrictive.
- Pour gérer de grandes limites, on peut traiter par segments.

### Complexité
- **Temporelle** : O(limit × √limit) pour générer tous les nombres de chaque forme.
- **Spatiale** : O(limit) pour stocker les ensembles.

---

## Solution en C++

```cpp
#include <iostream>
#include <set>
#include <cmath>

using namespace std;

long long compute(long long limit) {
    set<long long> sqs7, sqs3, sqs2, sqs1;
    
    // Générer a² + 7b²
    for (long long a = 1; a * a <= limit; a++) {
        for (long long b = 1; a*a + 7*b*b <= limit; b++) {
            sqs7.insert(a*a + 7*b*b);
        }
    }
    
    // Filtrer a² + 3b²
    long long maxsqs = *sqs7.rbegin();
    for (long long a = 1; a * a <= maxsqs; a++) {
        for (long long b = 1; a*a + 3*b*b <= maxsqs; b++) {
            long long v = a*a + 3*b*b;
            if (sqs7.find(v) != sqs7.end()) {
                sqs3.insert(v);
            }
        }
    }
    
    // Filtrer a² + 2b²
    maxsqs = *sqs3.rbegin();
    for (long long a = 1; a * a <= maxsqs; a++) {
        for (long long b = 1; a*a + 2*b*b <= maxsqs; b++) {
            long long v = a*a + 2*b*b;
            if (sqs3.find(v) != sqs3.end()) {
                sqs2.insert(v);
            }
        }
    }
    
    // Filtrer a² + b²
    maxsqs = *sqs2.rbegin();
    for (long long a = 1; a * a <= maxsqs; a++) {
        for (long long b = 1; a*a + b*b <= maxsqs; b++) {
            long long v = a*a + b*b;
            if (sqs2.find(v) != sqs2.end()) {
                sqs1.insert(v);
            }
        }
    }
    
    return sqs1.size();
}

int main() {
    cout << compute(2000000000) << endl;
    return 0;
}
```
