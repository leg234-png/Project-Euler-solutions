# Nombres cycliques

## Description du Problème
Un nombre cyclique avec n chiffres a une propriété très intéressante :
- Il est de la forme 1/p où p est un nombre premier.
- Sa représentation décimale répète une séquence de n-1 chiffres.
- Multiplier ce nombre cyclique par 1, 2, 3, ..., n produit des permutations cycliques des chiffres.

Le prochain nombre cyclique est 0588235294117647 avec 16 chiffres.

**Objectif** : Trouver la somme des chiffres du nombre cyclique pour lequel la partie entière de 1/p commence par 00000000137... et se termine par ...56789 (c'est-à-dire les 5 derniers chiffres avant la répétition sont 56789).

---

## Exemples
### Exemple 1
**Entrée** : `Nombre cyclique`
**Sortie** : `Somme des chiffres`
**Explication** : La somme des chiffres du nombre cyclique recherché.

---

## Approche de Résolution

### Stratégie
1. Les nombres premiers génèrent ces cycles. Le nombre doit être entre 724600000 et 730000000.
2. Utiliser le petit théorème de Fermat pour vérifier rapidement si un nombre est premier.
3. Vérifier si les 5 derniers chiffres sont 56789.
4. Vérifier que la période de 1/p est p-1 (ce qui garantit que c'est un nombre cyclique).
5. Calculer la somme des chiffres du cycle.

### Observations
- Un nombre cyclique a une période de longueur p-1.
- On peut utiliser la division longue pour trouver la période.
- Le nombre premier doit être dans une plage spécifique.

### Complexité
- **Temporelle** : O(P × log p) où P est le nombre de candidats à tester.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>
#include <string>

using namespace std;

bool is_prime(long long x) {
    if (x <= 1) return false;
    if (x <= 3) return true;
    if (x % 2 == 0) return false;
    for (long long i = 3; i * i <= x; i += 2) {
        if (x % i == 0) return false;
    }
    return true;
}

int period_length(long long p) {
    // Trouver la période de 1/p
    long long remainder = 1;
    int count = 0;
    vector<long long> seen(p, -1);
    
    while (remainder != 0 && seen[remainder] == -1) {
        seen[remainder] = count;
        remainder = (remainder * 10) % p;
        count++;
    }
    
    return count;
}

long long compute() {
    for (long long p = 724600000; p <= 730000000; p++) {
        if (p % 100000 == 56789 && is_prime(p)) {
            if (period_length(p) == p - 1) {
                // Calculer la somme des chiffres
                long long sum = 0;
                long long remainder = 1;
                for (int i = 0; i < p - 1; i++) {
                    remainder *= 10;
                    sum += remainder / p;
                    remainder %= p;
                }
                return sum;
            }
        }
    }
    return 0;
}

int main() {
    cout << compute() << endl;
    return 0;
}
```
