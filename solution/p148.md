# Exploring Pascal's Triangle

## Description du Problème
Nous pouvons facilement vérifier qu'aucune des entrées dans les sept premières lignes du triangle de Pascal n'est divisible par 7 :

```
     1
    1 1
   1 2 1
  1 3 3 1
 1 4 6 4 1
1 5 10 10 5 1
1 6 15 20 15 6 1
```

Cependant, si nous vérifions les cent premières lignes, nous trouverons que seulement 2361 des 5050 entrées ne sont pas divisibles par 7.

**Objectif** : Trouver le nombre d'entrées qui ne sont pas divisibles par 7 dans le premier milliard (10⁹) de lignes du triangle de Pascal.

---

## Exemples
### Exemple 1
**Entrée** : `n = 7`
**Sortie** : `28`
**Explication** : Toutes les 28 entrées dans les 7 premières lignes ne sont pas divisibles par 7.

---

## Approche de Résolution

### Stratégie
1. Utiliser le théorème de Lucas pour compter les coefficients binomiaux non divisibles par 7.
2. Pour chaque ligne n, compter combien de C(n, k) ne sont pas divisibles par 7.
3. Utiliser la représentation en base 7 de n et k.
4. C(n, k) n'est pas divisible par 7 si et seulement si chaque chiffre de k est ≤ le chiffre correspondant de n en base 7.

### Observations
- Le théorème de Lucas permet de calculer efficacement C(n, k) mod 7.
- On peut utiliser la représentation en base 7 pour compter.

### Complexité
- **Temporelle** : O(n log₇ n), où n = 10⁹.
- **Spatiale** : O(log n).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

long long countNonDivisible(int n, int p) {
    // Convertir n en base p
    std::vector<int> digits;
    int temp = n;
    while (temp > 0) {
        digits.push_back(temp % p);
        temp /= p;
    }
    
    // Pour chaque ligne, compter les entrées non divisibles
    long long count = 0;
    for (int i = 0; i <= n; i++) {
        // Vérifier si C(n, i) n'est pas divisible par p
        // en utilisant le théorème de Lucas
        bool divisible = false;
        int j = i;
        for (int d : digits) {
            if (j % p > d) {
                divisible = true;
                break;
            }
            j /= p;
        }
        if (!divisible) count++;
    }
    return count;
}

int main() {
    const int LIMIT = 1000000000;
    const int P = 7;
    
    long long total = 0;
    for (int n = 0; n < LIMIT; n++) {
        total += countNonDivisible(n, P);
    }
    
    std::cout << total << std::endl;
    return 0;
}
```
