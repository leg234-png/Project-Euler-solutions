# Graphe de totient

## Description du Problème
Soit t(n) le nombre de triangles dans le graphe de totient pour n.

Soit T(N) = ∑_{n=1}^N t(n). On vous donne T(10) = 26 et T(10²) = 5282.

**Objectif** : Trouver T(10¹²). Donner votre réponse modulo 715827883.

---

## Exemples
### Exemple 1
**Entrée** : `N = 10`
**Sortie** : `26`
**Explication** : T(10) = 26.

---

## Approche de Résolution

### Stratégie
1. Le graphe de totient pour n a des sommets 1, 2, ..., n.
2. Il y a une arête entre i et j si gcd(i, j) = 1.
3. Un triangle est un ensemble de 3 sommets tous connectés entre eux.
4. Pour chaque n, compter le nombre de triangles dans le graphe.
5. Utiliser des techniques de comptage combinatoire pour calculer efficacement.

### Observations
- Un triangle (i, j, k) nécessite gcd(i, j) = gcd(j, k) = gcd(i, k) = 1.
- On peut utiliser la fonction de Möbius pour compter.
- Pour de très grandes valeurs, on doit utiliser des techniques avancées.

### Complexité
- **Temporelle** : O(N³) dans le pire cas pour compter tous les triangles.
- **Spatiale** : O(N) pour stocker les informations du graphe.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

long long mod = 715827883;

int count_triangles(int n) {
    int count = 0;
    
    for (int i = 1; i <= n; i++) {
        for (int j = i + 1; j <= n; j++) {
            if (__gcd(i, j) != 1) continue;
            for (int k = j + 1; k <= n; k++) {
                if (__gcd(i, k) == 1 && __gcd(j, k) == 1) {
                    count++;
                }
            }
        }
    }
    
    return count;
}

long long compute(long long N, long long mod) {
    long long total = 0;
    
    for (int n = 1; n <= N; n++) {
        total = (total + count_triangles(n)) % mod;
    }
    
    return total;
}

int main() {
    long long N = 1;
    for (int i = 0; i < 12; i++) {
        N *= 10;
    }
    cout << compute(N, mod) << endl;
    return 0;
}
```
