# Tri-coloration d'une grille triangulaire

## Description du Problème
Considérons la configuration suivante de 16 triangles disposés en rangées :

Chaque triangle peut être coloré en rouge, vert ou bleu. Deux triangles sont considérés comme voisins s'ils partagent un côté. Notez : s'ils ne partagent qu'un sommet, ils ne sont pas voisins.

Une coloration est valide si aucun triangle n'a les trois voisins de la même couleur que lui.

**Objectif** : Une coloration C′ obtenue d'une coloration C par rotation ou réflexion est considérée comme distincte de C sauf si les deux sont identiques. Combien de colorations valides distinctes y a-t-il pour la configuration ci-dessus ?

---

## Exemples
### Exemple 1
**Entrée** : Grille triangulaire avec 3 rangées
**Sortie** : `Nombre de colorations valides`
**Explication** : Le nombre de façons de colorer la grille avec les contraintes données.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique avec mémorisation.
2. Pour chaque rangée, générer toutes les chaînes de couleurs valides.
3. Pour chaque masque de la rangée précédente, compter le nombre de façons de colorer la rangée actuelle.
4. Itérer sur toutes les rangées et sommer les résultats.

### Observations
- Les triangles dans une rangée impaire ont des contraintes différentes de ceux dans une rangée paire.
- On peut utiliser un masque pour représenter la coloration de la rangée précédente.
- La mémorisation est essentielle pour éviter les recalculs.

### Complexité
- **Temporelle** : O(rows × 3^(2*rows)) dans le pire cas, mais optimisé par la mémorisation.
- **Spatiale** : O(rows × 3^(2*rows)) pour la table de mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <unordered_map>
#include <string>
#include <vector>

using namespace std;

unordered_map<string, long long> memo;

vector<string> gen_string(const string& mask, int pos, const string& prev) {
    vector<string> strings;
    
    if (pos == mask.length() + 2) {
        return {prev};
    }
    
    if (pos == 0) {
        for (char x : {'R', 'G', 'B'}) {
            auto result = gen_string(mask, pos + 1, prev + x);
            strings.insert(strings.end(), result.begin(), result.end());
        }
    } else {
        if (pos % 2 == 1) {
            for (char x : {'R', 'G', 'B'}) {
                if (x != mask[pos - 1] && x != prev.back()) {
                    auto result = gen_string(mask, pos + 1, prev + x);
                    strings.insert(strings.end(), result.begin(), result.end());
                }
            }
        } else {
            for (char x : {'R', 'G', 'B'}) {
                if (x != prev.back()) {
                    auto result = gen_string(mask, pos + 1, prev + x);
                    strings.insert(strings.end(), result.begin(), result.end());
                }
            }
        }
    }
    return strings;
}

long long compute(int row, const string& mask) {
    if (row == 1) {
        return gen_string(mask, 0, "").size();
    }
    
    string key = to_string(row) + "_" + mask;
    if (memo.find(key) != memo.end()) {
        return memo[key];
    }
    
    long long total = 0;
    auto strings = gen_string(mask, 0, "");
    for (const string& s : strings) {
        total += compute(row - 1, s);
    }
    
    memo[key] = total;
    return total;
}

int main() {
    int rows = 8;
    string initial_mask = string(rows * 2, 'X');
    cout << compute(rows, initial_mask) << endl;
    return 0;
}
```
