# Carrés sous une hyperbole

## Description du Problème
Le diagramme montre certains de ces carrés étiquetés par nombre.

S₂ a un carré à sa gauche et aucun en dessous, donc l'index de S₂ est (1,0).

On peut voir que l'index de S₃₂ est (1,1) comme l'index de S₅₀.

50 est le plus grand n pour lequel l'index de Sₙ est (1,1).

**Objectif** : Trouver le plus grand n pour lequel l'index de Sₙ est (3,3).

---

## Exemples
### Exemple 1
**Entrée** : `index = (1,1)`
**Sortie** : `50`
**Explication** : 50 est le plus grand n pour lequel l'index est (1,1).

---

## Approche de Résolution

### Stratégie
1. Récursivement trouver les carrés jusqu'à une certaine profondeur, similaire au problème 199.
2. Le point en bas à gauche d'un carré est (a, b) et sa largeur est c.
3. Cela signifie que (a+c, b+c) doit se trouver sur y=1/x, donc b+c = 1/(a+c) => ab + (a+b)c + c² = 1.
4. Résoudre c² + c(a+b) + (ab - 1) = 0 => c = (-(a+b) ± √((a-b)² + 4))/2.
5. Trier les carrés par aire et trouver celui avec l'index (3,3).

### Observations
- Les carrés sont générés récursivement en divisant l'espace sous la courbe.
- Chaque carré crée deux nouveaux carrés (à droite et en haut).
- On doit suivre le nombre de carrés à gauche et en dessous pour chaque carré.

### Complexité
- **Temporelle** : O(2^depth) pour générer tous les carrés jusqu'à une profondeur donnée.
- **Spatiale** : O(2^depth) pour stocker tous les carrés.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>

using namespace std;

struct Square {
    double area;
    int left;
    int under;
};

double find_square_width(double a, double b) {
    double cons = -(a + b) / 2.0;
    double sqrt_val = sqrt((a - b) * (a - b) + 4) / 2.0;
    return cons + sqrt_val;
}

vector<Square> find_squares(double p1, double p2, int left, int under, int depth) {
    double w = find_square_width(p1, p2);
    vector<Square> squares;
    squares.push_back({w * w, left, under});
    
    if (depth == 1) {
        return squares;
    }
    
    auto right_squares = find_squares(p1 + w, p2, left + 1, under, depth - 1);
    auto up_squares = find_squares(p1, p2 + w, left, under + 1, depth - 1);
    
    squares.insert(squares.end(), right_squares.begin(), right_squares.end());
    squares.insert(squares.end(), up_squares.begin(), up_squares.end());
    
    return squares;
}

long long compute(int depth_limit, int target_left, int target_under) {
    vector<Square> squares = find_squares(1.0, 0.0, 0, 0, depth_limit);
    
    sort(squares.begin(), squares.end(), [](const Square& a, const Square& b) {
        return a.area > b.area;
    });
    
    long long max_index = 0;
    for (size_t i = 0; i < squares.size(); i++) {
        if (squares[i].left == target_left && squares[i].under == target_under) {
            max_index = max(max_index, (long long)(i + 1));
        }
    }
    
    return max_index;
}

int main() {
    cout << compute(50, 3, 3) << endl;
    return 0;
}
```
