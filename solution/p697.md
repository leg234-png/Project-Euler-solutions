# Séquence en décroissance aléatoire

## Description du Problème
Soit X₀ = c et, pour n > 0, Xₙ = Uₙ × X_{n-1} où Uₙ est un nombre réel choisi au hasard entre zéro et un, uniformément, et indépendamment de tous les choix précédents (Uₘ)_{m<n}.

Si on désire qu'il y ait précisément une probabilité de 25% que X₁₀₀ < 1, alors on peut arranger cela en fixant c tel que log₁₀ c ≈ 46.27.

**Objectif** : Supposons maintenant que c est fixé à une valeur différente, de sorte qu'il y ait précisément une probabilité de 25% que X₁₀₀₀₀₀₀₀ < 1. Trouver log₁₀ c et donner votre réponse arrondie à deux places après la virgule décimale.

---

## Exemples
### Exemple 1
**Entrée** : `n = 100, probabilité = 25%`
**Sortie** : `46.27`
**Explication** : log₁₀ c ≈ 46.27.

---

## Approche de Résolution

### Stratégie
1. On cherche P(X_n < 1) = 0.25.
2. P(U_n × U_{n-1} × ... × U₁ × c < 1) = 0.25.
3. P(U_n × U_{n-1} × ... × U₁ < 1/c) = 0.25.
4. P(-ln(U_n) - ln(U_{n-1}) - ... - ln(U₁) > -ln(1/c)) = 0.25.
5. La somme de n variables exponentielles est une distribution gamma Γ(n, 1).
6. P(Γ(n, 1) > ln(c)) = 0.25.
7. Trouver ln(c) en utilisant la fonction quantile de la distribution gamma.
8. log₁₀ c = ln(c) / ln(10).

### Observations
- On peut utiliser les fonctions de distribution gamma.
- Le problème nécessite le calcul de quantiles de la distribution gamma.
- On peut utiliser des bibliothèques mathématiques ou des approximations.

### Complexité
- **Temporelle** : O(1) pour le calcul du quantile.
- **Spatiale** : O(1).
