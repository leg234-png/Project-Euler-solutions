# Large Non-Mersenne Prime

## Description du Problème
Le premier nombre premier connu pour dépasser un million de chiffres a été découvert en 1999, et est un nombre premier de Mersenne de la forme 2^6972593 - 1 ; il contient exactement 2 098 960 chiffres. Par la suite, d'autres nombres premiers de Mersenne, de la forme 2^p - 1, ont été trouvés qui contiennent plus de chiffres.

Cependant, en 2004, il a été trouvé un nombre premier non-Mersenne massif qui contient 2 357 207 chiffres : 28433 × 2^7830457 + 1.

**Objectif** : Trouver les dix derniers chiffres de ce nombre premier.

---

## Exemples
### Exemple 1
**Entrée** : `28433 × 2^7830457 + 1`
**Sortie** : `8739992577` (les 10 derniers chiffres)
**Explication** : On calcule le nombre et on prend les 10 derniers chiffres.

---

## Approche de Résolution

### Stratégie
1. Calculer 2^7830457 mod 10^10 en utilisant l'exponentiation modulaire rapide.
2. Multiplier le résultat par 28433 mod 10^10.
3. Ajouter 1 mod 10^10.
4. Retourner les 10 derniers chiffres.

### Observations
- On utilise l'exponentiation modulaire pour éviter de calculer le nombre entier massif.
- On travaille modulo 10^10 pour obtenir seulement les 10 derniers chiffres.
- L'exponentiation modulaire utilise la méthode "exponentiation par carrés" : a^b mod m peut être calculé efficacement.

### Complexité
- **Temporelle** : O(log n), où n=7830457, car l'exponentiation modulaire nécessite O(log n) multiplications.
- **Spatiale** : O(1), car on travaille avec des nombres de taille fixe (10^10).

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <iomanip>

long long modPow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

int main() {
    const long long MOD = 10000000000LL; // 10^10
    long long n = modPow(2, 7830457, MOD);
    n = (n * 28433) % MOD;
    n = (n + 1) % MOD;
    
    std::cout << std::setfill('0') << std::setw(10) << n << std::endl;
    return 0;
}
```
