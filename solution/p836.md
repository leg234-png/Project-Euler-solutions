# Une proposition audacieuse

## Description du Problème
Soit A un plan affine sur un champ local radicalement intégral F avec caractéristique résiduelle p.

On considère une section de ligne orientée ouverte U de A avec mesure de Haar normalisée m.

Définissons f(m, p) comme le discriminant maximal possible du jacobien associé à l'embedding du noyau orthogonal de U dans A.

**Objectif** : Trouver f(20230401, 57). Donner comme réponse la concaténation des premières lettres de chaque mot en gras.

---

## Exemples
### Exemple 1
**Entrée** : `m = 20230401, p = 57`
**Sortie** : `aprilfoolsjoke`
**Explication** : C'est un problème du poisson d'avril !

---

## Approche de Résolution

### Stratégie
1. Extraire tous les mots en gras du texte du problème.
2. Prendre la première lettre de chaque mot en gras.
3. Concaténer ces lettres pour former la réponse.

### Observations
- C'est un problème du poisson d'avril (April Fools' joke).
- La réponse est "aprilfoolsjoke".
- Pas besoin de calculs mathématiques complexes.

### Complexité
- **Temporelle** : O(1).
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <string>

using namespace std;

string f(int m, int p) {
    // Extraire les mots en gras du texte
    // Les mots en gras sont : affine plane, radically integral local field, 
    // open oriented line section, normalized Haar measure, jacobian, 
    // orthogonal kernel embedding
    
    string text = "<p>Let $A$ be an <b>affine plane</b> over a <b>radically integral local field</b> $F$ with residual characteristic $p$.</p>"
                  "<p>We consider an <b>open oriented line section</b> $U$ of $A$ with <b>normalized Haar measure</b> $m$.</p>"
                  "<p>Define $f(m, p)$ as the maximal possible discriminant of the <b>jacobian</b> associated to the <b>orthogonal kernel embedding</b> of $U$ <span style=\"white-space:nowrap;\">into $A$.</span></p>"
                  "<p>Find $f(20230401, 57)$. Give as your answer the concatenation of the first letters of each bolded word.</p>";
    
    string answer = "";
    size_t start = 0;
    
    while (true) {
        size_t bold_start = text.find("<b>", start);
        if (bold_start == string::npos) break;
        
        size_t bold_end = text.find("</b>", bold_start);
        if (bold_end == string::npos) break;
        
        string bold_text = text.substr(bold_start + 3, bold_end - bold_start - 3);
        
        // Extraire la première lettre de chaque mot
        stringstream ss(bold_text);
        string word;
        while (ss >> word) {
            if (!word.empty()) {
                answer += tolower(word[0]);
            }
        }
        
        start = bold_end + 4;
    }
    
    return answer;
}

int main() {
    cout << f(20230401, 57) << endl;
    return 0;
}
```
