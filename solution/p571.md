# Nombres super-pandigitaux

## Description du Problème
Un nombre n est appelé k-super-pandigital s'il est pandigital dans toutes les bases de 2 à k (inclus).

Par exemple, 978 = 1111010010₂ = 1100020₃ = 33102₄ = 12403₅ est le plus petit nombre 5-super-pandigital.

De même, 1093265784 est le plus petit nombre 10-super-pandigital.

La somme des 10 plus petits nombres 10-super-pandigitaux est 20319792309.

**Objectif** : Quelle est la somme des 10 plus petits nombres 12-super-pandigitaux ?

---

## Exemples
### Exemple 1
**Entrée** : `k = 5`
**Sortie** : `978`
**Explication** : 978 est le plus petit nombre 5-super-pandigital.

---

## Approche de Résolution

### Stratégie
1. Générer tous les nombres en ordre lexicographique.
2. Pour chaque nombre, vérifier s'il est pandigital dans toutes les bases de 2 à k.
3. Un nombre est pandigital en base b s'il contient tous les chiffres de 0 à b-1.
4. Convertir le nombre en base b et vérifier.
5. Collecter les 10 premiers nombres qui satisfont la condition.
6. Sommer ces nombres.

### Observations
- On peut générer les nombres efficacement en ordre lexicographique.
- La vérification pandigital peut être optimisée.
- On peut utiliser des techniques de backtracking pour générer les candidats.

### Complexité
- **Temporelle** : O(N × k × log n) où N est le nombre de candidats à tester.
- **Spatiale** : O(k) pour stocker les chiffres dans chaque base.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <algorithm>

using namespace std;

vector<int> to_base(long long n, int base) {
    vector<int> digits;
    if (n == 0) {
        digits.push_back(0);
        return digits;
    }
    while (n > 0) {
        digits.push_back(n % base);
        n /= base;
    }
    reverse(digits.begin(), digits.end());
    return digits;
}

bool is_pandigital(long long n, int base) {
    vector<int> digits = to_base(n, base);
    set<int> seen;
    for (int d : digits) {
        if (d >= base) return false;
        seen.insert(d);
    }
    return seen.size() == base;
}

bool is_super_pandigital(long long n, int k) {
    for (int base = 2; base <= k; base++) {
        if (!is_pandigital(n, base)) {
            return false;
        }
    }
    return true;
}

long long compute(int k, int count) {
    vector<long long> super_pandigital;
    long long n = 1;
    
    while (super_pandigital.size() < count) {
        if (is_super_pandigital(n, k)) {
            super_pandigital.push_back(n);
        }
        n++;
    }
    
    long long total = 0;
    for (long long num : super_pandigital) {
        total += num;
    }
    return total;
}

int main() {
    cout << compute(12, 10) << endl;
    return 0;
}
```
