# Bitwise-OR operations on random integers

## Description du Problème
Soit `y₀, y₁, y₂, ...` une suite de nombres entiers aléatoires non signés de 32 bits.

Soit `x₀ = 0` et `xᵢ = xᵢ₋₁ | yᵢ₋₁` pour `i ≥ 1`.

**Objectif** : Trouver l'espérance du nombre d'étapes nécessaires pour que `xᵢ = 2³² - 1` (tous les bits sont à 1). Donner la réponse arrondie à 10 décimales.

---

## Exemples
### Exemple 1
**Entrée** : `n = 1` bit
**Sortie** : `2.0`
**Explication** : Pour 1 bit, l'espérance est 2 (distribution géométrique avec p=0.5).

---

## Approche de Résolution

### Stratégie
1. Définir `E[Xₙ]` comme l'espérance du nombre d'étapes pour que `n` bits soient tous à 1.
2. `E[X₀] = 0`.
3. Pour `E[Xₙ]` :
   - Dans une étape, `k` bits (0 ≤ k ≤ n) sont mis à 1 avec probabilité `C(n,k) / 2ⁿ`.
   - L'espérance restante est `E[Xₙ₋ₖ]`.
   - `E[Xₙ] = Σ (C(n,k) × (1 + E[Xₙ₋ₖ])) / 2ⁿ` pour k de 0 à n.
4. Simplifier : `E[Xₙ] = (2ⁿ + Σ C(n,k) × E[Xₙ₋ₖ] pour k < n) / (2ⁿ - 1)`.
5. Calculer récursivement `E[X₃₂]`.

### Observations
- On peut résoudre le système d'équations linéaires récursivement.
- On utilise des fractions exactes pour éviter les erreurs d'arrondi.

### Complexité
- **Temporelle : O(n² × log n)`.
- **Spatiale : O(n)`.

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <vector>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

struct Fraction {
    cpp_int num, den;
    
    Fraction(cpp_int n = 0, cpp_int d = 1) : num(n), den(d) {
        cpp_int g = gcd(n, d);
        num /= g;
        den /= g;
    }
    
    Fraction operator+(const Fraction& other) const {
        return Fraction(num * other.den + other.num * den, den * other.den);
    }
    
    Fraction operator*(const Fraction& other) const {
        return Fraction(num * other.num, den * other.den);
    }
    
    double toDouble() const {
        return num.convert_to<double>() / den.convert_to<double>();
    }
};

cpp_int binomial(int n, int k) {
    if (k > n - k) k = n - k;
    cpp_int result = 1;
    for (int i = 0; i < k; i++)
        result = result * (n - i) / (i + 1);
    return result;
}

int main() {
    const int SIZE = 32;
    vector<Fraction> expect(SIZE + 1);
    expect[0] = Fraction(0, 1);
    
    for (int n = 1; n <= SIZE; n++) {
        Fraction sum(0, 1);
        for (int k = 0; k < n; k++) {
            cpp_int binom = binomial(n, k);
            Fraction term(expect[k].num * binom, expect[k].den);
            sum = sum + term;
        }
        cpp_int twoPowN = cpp_int(1) << n;
        Fraction temp = sum + Fraction(twoPowN, 1);
        expect[n] = Fraction(temp.num, temp.den * (twoPowN - 1));
    }
    
    cout << fixed << setprecision(10) << expect[SIZE].toDouble() << endl;
    return 0;
}
```
