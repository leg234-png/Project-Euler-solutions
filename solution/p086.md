# Route cuboïde

## Description du Problème
Une araignée, S, se trouve dans un coin d'une pièce cuboïde, mesurant 6 par 5 par 3, et une mouche, F, se trouve dans le coin opposé. En voyageant sur les surfaces de la pièce, la distance "ligne droite" la plus courte de S à F est 10 et le chemin est montré sur le diagramme.

Cependant, il y a jusqu'à trois candidats de "plus court" chemin pour n'importe quel cuboïde donné et la route la plus courte n'a pas toujours une longueur entière.

Il peut être montré qu'il y a exactement 2060 cuboïdes distincts, en ignorant les rotations, avec des dimensions entières, jusqu'à une taille maximale de M par M par M, pour lesquels la route la plus courte a une longueur entière quand M=100. C'est la plus petite valeur de M pour laquelle le nombre de solutions dépasse d'abord deux mille ; le nombre de solutions quand M=99 est 1975.

**Objectif** : Trouver la plus petite valeur de M telle que le nombre de solutions dépasse d'abord un million.

---

## Exemples
### Exemple 1
**Entrée** : M = 100
**Sortie** : `2060`
**Explication** : Il y a 2060 cuboïdes distincts avec M=100 pour lesquels la route la plus courte a une longueur entière.

---

## Approche de Résolution

### Stratégie
1. Pour un cuboïde de dimensions x×y×z, les trois chemins possibles sont :
   - √((x+y)² + z²)
   - √((y+z)² + x²)
   - √((z+x)² + y²)
2. Le plus court chemin est le minimum de ces trois.
3. Générer tous les triplets pythagoriciens primitifs.
4. Pour chaque triplet (a, b, c), trouver toutes les façons de diviser a en x et y tels que x+y=a, et z=b.
5. Compter les solutions cumulatives jusqu'à dépasser 1 000 000.

### Observations
- On utilise le théorème des triplets pythagoriciens pour générer les triplets.
- Pour chaque triplet, on divise a en x et y de toutes les façons possibles.
- On maintient un compteur cumulatif des solutions.

### Complexité
- **Temporelle** : O(M²), où M est la valeur recherchée.
- **Spatiale** : O(M), pour stocker les solutions.

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <algorithm>
#include <cmath>

using namespace std;

int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

struct Triple {
    int a, b, c;
    Triple(int a, int b, int c) : a(a), b(b), c(c) {}
    bool operator<(const Triple& other) const {
        if (a != other.a) return a < other.a;
        if (b != other.b) return b < other.b;
        return c < other.c;
    }
};

int main() {
    vector<set<vector<int>>> solutions;
    vector<int> cumulativeSolutions;
    cumulativeSolutions.push_back(0);
    
    int limit = 1;
    while (true) {
        while ((int)solutions.size() < limit)
            solutions.push_back(set<vector<int>>());
        
        // Générer les triplets pythagoriciens
        for (int s = 3; ; s += 2) {
            bool outerBreak = false;
            for (int t = s - 2; t > 0; t -= 2) {
                if (s * s / 2 >= limit * 3) {
                    outerBreak = true;
                    break;
                }
                if (gcd(s, t) == 1) {
                    for (int k = 1; ; k++) {
                        int a = s * t * k;
                        int b = (s * s - t * t) / 2 * k;
                        int c = (s * s + t * t) / 2 * k;
                        if (a >= limit && b >= limit)
                            break;
                        
                        // Trouver les divisions
                        int z = b;
                        for (int x = 1; x < a; x++) {
                            int y = a - x;
                            if (y < x) break;
                            int minPath = min({(x+y)*(x+y) + z*z, (y+z)*(y+z) + x*x, (z+x)*(z+x) + y*y});
                            if (minPath == c * c) {
                                int maxDim = max({x, y, z});
                                if (maxDim < limit) {
                                    vector<int> soln = {x, y, z};
                                    sort(soln.begin(), soln.end());
                                    solutions[maxDim].insert(soln);
                                }
                            }
                        }
                        
                        z = a;
                        for (int x = 1; x < b; x++) {
                            int y = b - x;
                            if (y < x) break;
                            int minPath = min({(x+y)*(x+y) + z*z, (y+z)*(y+z) + x*x, (z+x)*(z+x) + y*y});
                            if (minPath == c * c) {
                                int maxDim = max({x, y, z});
                                if (maxDim < limit) {
                                    vector<int> soln = {x, y, z};
                                    sort(soln.begin(), soln.end());
                                    solutions[maxDim].insert(soln);
                                }
                            }
                        }
                    }
                }
            }
            if (outerBreak) break;
        }
        
        for (int i = cumulativeSolutions.size(); i < limit; i++) {
            int sum = cumulativeSolutions[i - 1] + solutions[i].size();
            cumulativeSolutions.push_back(sum);
            if (sum > 1000000) {
                cout << i << endl;
                return 0;
            }
        }
        
        limit *= 2;
    }
    
    return 1;
}
```
