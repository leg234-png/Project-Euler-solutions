# Représentation de Zeckendorf

## Description du Problème
Tout entier positif peut être écrit de manière unique comme une somme de nombres de Fibonacci non consécutifs où aucun nombre de Fibonacci n'est utilisé plus d'une fois.

Une telle somme est appelée la représentation de Zeckendorf du nombre.

Pour tout entier n>0, soit z(n) le nombre de termes dans la représentation de Zeckendorf de n.

Ainsi, z(5)=1, z(14)=2, z(100)=3, etc.

Aussi, pour 0<n<10⁶, ∑z(n)=7894453.

**Objectif** : Trouver ∑z(n) pour 0<n<10¹⁷.

---

## Exemples
### Exemple 1
**Entrée** : `n = 5`
**Sortie** : `1`
**Explication** : z(5) = 1 car 5 = 5 (un seul terme de Fibonacci).

### Exemple 2
**Entrée** : `n = 14`
**Sortie** : `2`
**Explication** : z(14) = 2 car 14 = 13 + 1 (deux termes).

---

## Approche de Résolution

### Stratégie
1. La représentation de Zeckendorf peut être trouvée en utilisant un algorithme glouton.
2. Il y aura une séquence récursive qui apparaît.
3. Il y a des "blocs" répétés qui sont ajoutés.
4. Séparer les nombres en une partie Fibonacci et une partie non-Fibonacci.
5. Répéter la fonction pour la partie non-Fibonacci jusqu'à atteindre une décomposition Fibonacci.

### Observations
- L'algorithme glouton fonctionne en choisissant toujours le plus grand nombre de Fibonacci possible.
- Il y a des motifs répétitifs dans les représentations qui permettent d'optimiser le calcul.
- On peut utiliser la programmation dynamique pour calculer efficacement la somme.

### Complexité
- **Temporelle** : O(log n) pour chaque nombre, où n est la valeur maximale.
- **Spatiale** : O(log n) pour stocker les nombres de Fibonacci.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

long long fibonacci(int n) {
    long long v1 = 1, v2 = 1, v3 = 0;
    string bits = "";
    int temp = n;
    while (temp > 0) {
        bits = to_string(temp % 2) + bits;
        temp /= 2;
    }
    
    for (size_t i = 1; i < bits.length(); i++) {
        long long calc = v2 * v2;
        v1 = v1 * v1 + calc;
        v2 = (v1 - calc + v3) * v2;
        v3 = calc + v3 * v3;
        
        if (bits[i] == '1') {
            long long temp_v1 = v1 + v2;
            long long temp_v2 = v1;
            long long temp_v3 = v2;
            v1 = temp_v1;
            v2 = temp_v2;
            v3 = temp_v3;
        }
    }
    return v2;
}

vector<long long> fib_till(long long x) {
    vector<long long> fibnumbers;
    int n = 1;
    while (fibonacci(n) <= x) {
        fibnumbers.push_back(fibonacci(n));
        n++;
    }
    return fibnumbers;
}

int ZeckendorfRepresentation(long long x) {
    vector<long long> fibs = fib_till(x);
    reverse(fibs.begin(), fibs.end());
    
    long long number = x;
    int count = 0;
    int terms = 0;
    
    while (number != 0) {
        if (number - fibs[count] >= 0) {
            number -= fibs[count];
            terms++;
        }
        count++;
    }
    return terms;
}

long long compute(long long limit) {
    long long total = 0;
    for (long long n = 1; n < limit; n++) {
        total += ZeckendorfRepresentation(n);
    }
    return total;
}

int main() {
    cout << compute(100000000000000000LL) << endl;
    return 0;
}
```
