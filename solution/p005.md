# Plus Petit Multiple

## Description du Problème
2520 est le plus petit nombre qui peut être divisé par chacun des nombres de 1 à 10 sans aucun reste.

**Objectif** : Quel est le plus petit nombre positif qui est divisible (sans reste) par tous les nombres de 1 à 20 ?

---

## Exemples
### Exemple 1
**Entrée** : Nombres de 1 à 10
**Sortie** : `2520`
**Explication** : 2520 est divisible par tous les nombres de 1 à 10 sans reste.

---

## Approche de Résolution

### Stratégie
Le problème demande de trouver le Plus Petit Commun Multiple (PPCM) de tous les nombres de 1 à 20.

Le PPCM de plusieurs nombres peut être calculé de manière itérative :
- PPCM(a, b, c, ..., z) = PPCM(...PPCM(PPCM(a, b), c)..., z)
- PPCM(x, y) = (x × y) / PGCD(x, y)

**Méthode :**
1. Commencer avec PPCM = 1.
2. Pour chaque nombre i de 1 à 20 :
   - Calculer PPCM = PPCM(PPCM actuel, i).
3. Le résultat final est la réponse.

### Observations
- Pour calculer PPCM(x, y), utiliser l'algorithme d'Euclide pour trouver le PGCD.
- PPCM = (x × y) / PGCD(x, y).
- On divise d'abord x par le PGCD pour éviter les débordements, puis on multiplie par y.
- Le PPCM est commutatif, associatif et idempotent.

### Complexité
- **Temporelle** : O(n × log(min(x, y))), où `n` = 20 et log(min(x, y)) est la complexité de l'algorithme d'Euclide.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>

long long gcd(long long a, long long b) {
    while (b != 0) {
        long long temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

long long lcm(long long a, long long b) {
    return (a / gcd(a, b)) * b;
}

int main() {
    long long allLcm = 1;
    
    for (int i = 1; i <= 20; ++i) {
        allLcm = lcm(allLcm, i);
    }
    
    std::cout << allLcm << std::endl;
    return 0;
}
```
