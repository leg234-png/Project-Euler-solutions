# Nombres somme de chiffres

## Description du Problème
Un nombre DS est un nombre positif qui peut être écrit comme la somme de deux nombres positifs ayant la même somme de chiffres que le nombre lui-même.

Par exemple, 2022 est un nombre DS car 2022 = 2002 + 20 et les trois nombres ont la même somme de chiffres (6).

Soit S(n) la somme de tous les nombres DS de n chiffres.

On vous donne S(3) = 63270 et S(7) = 85499991450.

**Objectif** : Trouver S(2020). Donner votre réponse modulo 10¹⁶.

---

## Exemples
### Exemple 1
**Entrée** : `n = 3`
**Sortie** : `63270`
**Explication** : S(3) = 63270.

---

## Approche de Résolution

### Stratégie
1. Un nombre DS est un nombre où il existe un chiffre d tel que sum_of_digits(x) = 2i où i est la position de d.
2. Trouver toutes les partitions de la somme de chiffres.
3. Pour chaque partition, compter le nombre de façons de permuter les chiffres.
4. Pour un nombre de n chiffres avec une partition donnée, le nombre de façons est n!/(d₀!d₁!...d₉!).
5. Si un chiffre est fixé en position k, cela contribue i × 10^k × (n-1)!/(d₀!d₁!...d₉!).
6. Utiliser les partitions pour calculer efficacement S(n).

### Observations
- On peut utiliser la programmation dynamique pour générer les partitions.
- Les factorielles peuvent être précalculées.
- On doit gérer les grands nombres avec l'arithmétique modulaire.

### Complexité
- **Temporelle** : O(P × n) où P est le nombre de partitions.
- **Spatiale** : O(n) pour les factorielles.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
#include <cmath>

using namespace std;

long long mod = 10000000000000000LL; // 10^16

vector<vector<vector<int>>> generate_partitions(int goal) {
    vector<vector<vector<int>>> part(goal + 1);
    part[0] = {{}};
    
    for (int options = 1; options <= goal; options++) {
        for (int i = 0; i <= goal - options; i++) {
            for (auto y : part[i]) {
                vector<int> new_part = {options};
                new_part.insert(new_part.end(), y.begin(), y.end());
                part[i + options].push_back(new_part);
            }
        }
    }
    return part;
}

long long factorial_mod(int n, long long mod) {
    long long result = 1;
    for (int i = 2; i <= n; i++) {
        result = (result * i) % mod;
    }
    return result;
}

long long compute_S(int n, long long mod) {
    long long total = 0;
    vector<vector<vector<int>>> partitions = generate_partitions(2 * n);
    
    for (int sum = 2; sum <= 2 * n; sum += 2) {
        for (auto part : partitions[sum]) {
            if (part.size() > n) continue;
            
            vector<int> digits(10, 0);
            for (int p : part) {
                if (p < 10) digits[p]++;
            }
            digits[0] = n - part.size();
            
            long long ways = factorial_mod(n - 1, mod);
            long long denom = 1;
            for (int d : digits) {
                denom = (denom * factorial_mod(d, mod)) % mod;
            }
            ways = (ways * mod_inverse(denom, mod)) % mod;
            
            long long sum_digits = 0;
            for (size_t i = 0; i < part.size(); i++) {
                sum_digits += part[i];
            }
            
            total = (total + ways * sum_digits) % mod;
        }
    }
    
    return total;
}

int main() {
    cout << compute_S(2020, mod) << endl;
    return 0;
}
```
