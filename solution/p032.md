# Produits Pandigitaux

## Description du Problème
On dira qu'un nombre à n chiffres est pandigital s'il utilise tous les chiffres de 1 à n exactement une fois ; par exemple, le nombre à 5 chiffres, 15234, est pandigital de 1 à 5.

Le produit 7254 est inhabituel, car l'identité, 39 × 186 = 7254, contenant le multiplicande, le multiplicateur et le produit, est pandigital de 1 à 9.

**Objectif** : Trouver la somme de tous les produits dont l'identité multiplicande/multiplicateur/produit peut être écrite comme pandigital de 1 à 9.

**INDICE** : Certains produits peuvent être obtenus de plus d'une façon, alors assurez-vous de ne l'inclure qu'une seule fois dans votre somme.

---

## Exemples
### Exemple 1
**Entrée** : Produits pandigitaux
**Sortie** : `45228`
**Explication** : La somme de tous les produits uniques qui forment une identité pandigital 1-9.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre i de 1 à 9999 :
   - Trouver tous les facteurs de i.
   - Pour chaque facteur j, vérifier si la concaténation de i, j et i/j est pandigital 1-9.
   - Si oui, ajouter i à la somme (une seule fois).
2. Retourner la somme.

### Observations
- Si un produit z ≥ 10000, alors x×y consomme au moins 5 chiffres. Avec les 4 chiffres restants, même x=99 et y=99 produisent x×y < 10000, donc z < 10000.
- On peut optimiser en ne testant que les produits < 10000.
- Pour vérifier si une chaîne est pandigital 1-9, trier les caractères et comparer avec "123456789".

### Complexité
- **Temporelle** : O(n × √n), où `n` = 10000.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>
#include <set>

bool isPandigital(const std::string& s) {
    if (s.length() != 9) return false;
    std::string sorted = s;
    std::sort(sorted.begin(), sorted.end());
    return sorted == "123456789";
}

bool hasPandigitalProduct(int n) {
    for (int i = 1; i * i <= n; ++i) {
        if (n % i == 0) {
            std::string concat = std::to_string(n) + std::to_string(i) + std::to_string(n / i);
            if (isPandigital(concat)) {
                return true;
            }
        }
    }
    return false;
}

int main() {
    std::set<int> products;
    
    for (int i = 1; i < 10000; ++i) {
        if (hasPandigitalProduct(i)) {
            products.insert(i);
        }
    }
    
    int sum = 0;
    for (int p : products) {
        sum += p;
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
