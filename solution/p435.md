# Polynômes de nombres de Fibonacci

## Description du Problème
Définissons les polynômes {Fₙ, n≥0} comme Fₙ(x) = ∑ᵢ₌₀ⁿ fᵢxᵢ.

Par exemple, F₇(x) = x + x² + 2x³ + 3x⁴ + 5x⁵ + 8x⁶ + 13x⁷, et F₇(11) = 268357683.

**Objectif** : Soit n = 10¹⁵. Trouver la somme ∑ₓ₌₀¹⁰⁰ Fₙ(x) et donner votre réponse modulo 1307674368000 (=15!).

---

## Exemples
### Exemple 1
**Entrée** : `n = 7, x = 11`
**Sortie** : `268357683`
**Explication** : F₇(11) = 268357683.

---

## Approche de Résolution

### Stratégie
1. Fₙ(x) = (f_{n+1}x^{n+1} + fₙx^{n+2} - x)/(x² + x - 1).
2. Modifier la fonction Fibonacci pour inclure des mods pour trouver f_{10^15} rapidement.
3. Noter que (x² + x - 1) divise (f_{n+1}x^{n+1} + fₙx^{n+2} - x) car les solutions de Fₙ(x) sont des entiers.
4. Utiliser la propriété : si a mod b = 0, alors a/b mod m = (a mod m×b)/b.
5. Calculer Fₙ(x) pour chaque x de 0 à 100 et sommer.

### Observations
- On peut utiliser l'exponentiation matricielle rapide pour calculer les nombres de Fibonacci modulo.
- La formule permet d'éviter de calculer explicitement tous les termes du polynôme.
- On doit gérer la division modulaire correctement.

### Complexité
- **Temporelle** : O(log n) pour chaque calcul de Fₙ(x), où n = 10¹⁵.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long fibonacci_mod(long long n, long long mod) {
    long long v1 = 1, v2 = 1, v3 = 0;
    string bits = "";
    long long temp = n;
    while (temp > 0) {
        bits = to_string(temp % 2) + bits;
        temp /= 2;
    }
    
    for (size_t i = 1; i < bits.length(); i++) {
        long long calc = (v2 * v2) % mod;
        v1 = (v1 * v1 + calc) % mod;
        v2 = ((v1 - calc + v3) * v2) % mod;
        v3 = (calc + v3 * v3) % mod;
        
        if (bits[i] == '1') {
            long long temp_v1 = (v1 + v2) % mod;
            long long temp_v2 = v1;
            long long temp_v3 = v2;
            v1 = temp_v1;
            v2 = temp_v2;
            v3 = temp_v3;
        }
    }
    return v2;
}

long long F(long long n, long long x, long long mod) {
    long long den = (x * x + x - 1);
    long long mod_den = mod * den;
    
    long long fn = fibonacci_mod(n, mod_den);
    long long fn1 = fibonacci_mod(n + 1, mod_den);
    
    long long x_pow_n1 = 1, x_pow_n2 = 1;
    for (long long i = 0; i < n + 1; i++) {
        x_pow_n1 = (x_pow_n1 * x) % mod_den;
    }
    for (long long i = 0; i < n + 2; i++) {
        x_pow_n2 = (x_pow_n2 * x) % mod_den;
    }
    
    long long num = (fn * x_pow_n2 + fn1 * x_pow_n1 - x + mod_den) % mod_den;
    return num / den;
}

long long compute(long long n, int limit, long long mod) {
    long long total = 0;
    for (int x = 0; x <= limit; x++) {
        total = (total + F(n, x, mod)) % mod;
    }
    return total;
}

int main() {
    long long mod = 1307674368000LL; // 15!
    cout << compute(1000000000000000LL, 100, mod) << endl;
    return 0;
}
```
