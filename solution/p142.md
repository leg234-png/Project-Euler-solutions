# Perfect Square Collection

## Description du Problème
Trouver le plus petit x + y + z avec des entiers x > y > z > 0 tel que x + y, x - y, x + z, x - z, y + z, y - z sont tous des carrés parfaits.

---

## Exemples
### Exemple 1
**Entrée** : `x = 10, y = 6, z = 3`
**Sortie** : `false`
**Explication** : Tous les 6 nombres ne sont pas des carrés parfaits.

---

## Approche de Résolution

### Stratégie
1. Supposons x + y = a² et x - y = b², donc x = (a² + b²)/2 et y = (a² - b²)/2.
2. Pour chaque paire (a, b) avec a > b > 0 et a, b de même parité, calculer x et y.
3. Pour chaque z < y, vérifier si x + z, x - z, y + z, y - z sont tous des carrés parfaits.
4. Trouver le plus petit x + y + z.

### Observations
- On peut utiliser un tableau booléen pour vérifier rapidement si un nombre est un carré.
- On doit s'assurer que x, y, z sont des entiers positifs.

### Complexité
- **Temporelle** : O(n²), où n est la limite pour a et b.
- **Spatiale** : O(n) pour stocker le tableau des carrés.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int findSum(int limit) {
    std::vector<bool> isSquare(limit, false);
    for (int i = 0; i * i < limit; i++) {
        isSquare[i * i] = true;
    }
    
    for (int a = 1; a * a < limit; a++) {
        for (int b = a - 1; b > 0; b--) {
            if ((a + b) % 2 != 0) continue;
            
            int x = (a * a + b * b) / 2;
            int y = (a * a - b * b) / 2;
            if (x + y + 1 >= limit) continue;
            
            int zlimit = std::min(y, limit - x - y);
            for (int c = (int)sqrt(y) + 1; c * c - y < zlimit; c++) {
                int z = c * c - y;
                if (isSquare[x + z] && isSquare[x - z] && isSquare[y - z]) {
                    return x + y + z;
                }
            }
        }
    }
    return -1;
}

int main() {
    int sumLimit = 10;
    while (findSum(sumLimit) == -1) {
        sumLimit *= 10;
    }
    
    while (true) {
        int sum = findSum(sumLimit);
        if (sum == -1) break;
        sumLimit = sum;
    }
    
    std::cout << sumLimit << std::endl;
    return 0;
}
```
