# Arranged Probability

## Description du Problème
Si une boîte contient vingt-et-un disques colorés, composés de quinze disques bleus et six disques rouges, et que deux disques étaient pris au hasard, on peut voir que la probabilité de prendre deux disques bleus, P(BB) = (15/21) × (14/20) = 1/2.

Le prochain tel arrangement, pour lequel il y a exactement 50% de chance de prendre deux disques bleus au hasard, est une boîte contenant quatre-vingt-cinq disques bleus et trente-cinq disques rouges.

**Objectif** : En trouvant le premier arrangement pour contenir plus de 10^12 = 1 000 000 000 000 disques au total, déterminer le nombre de disques bleus que la boîte contiendrait.

---

## Exemples
### Exemple 1
**Entrée** : `total ≤ 100`
**Sortie** : `85`
**Explication** : 85 bleus + 35 rouges = 120 total, P(BB) = (85/120) × (84/119) = 1/2.

---

## Approche de Résolution

### Stratégie
1. Soit b le nombre de disques bleus et r le nombre de disques rouges.
2. La probabilité de prendre 2 disques bleus est : [b(b-1)] / [(b+r)(b+r-1)] = 1/2
3. Réarranger l'équation pour obtenir : b = r + [sqrt(8r² + 1) + 1]/2
4. Pour que b soit un entier, on a besoin que sqrt(8r² + 1) soit impair et que 8r² + 1 soit un carré parfait.
5. Cela forme l'équation de Pell : x² - 8y² = 1, où x = sqrt(8r² + 1) et y = r.
6. Utiliser la solution fondamentale (3, 1) et générer les solutions suivantes.
7. Trouver la première solution où b + r > 10^12.

### Observations
- L'équation se réduit à une équation de Pell.
- La solution fondamentale est (3, 1).
- On peut générer toutes les solutions en multipliant par la solution fondamentale.
- On doit vérifier que sqrt(8r² + 1) est impair pour que b soit un entier.

### Complexité
- **Temporelle** : O(log n), où n=10^12, car on génère les solutions de Pell jusqu'à atteindre la limite.
- **Spatiale** : O(1), car on génère une solution à la fois.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <gmpxx.h>  // Utiliser GMP pour les grands entiers

int main() {
    // Solution fondamentale de l'équation de Pell x² - 8y² = 1
    mpz_class x0("3");
    mpz_class y0("1");
    
    // Solution actuelle
    mpz_class x = x0;
    mpz_class y = y0;  // Alias pour le nombre de disques rouges
    
    mpz_class limit;
    mpz_ui_pow_ui(limit.get_mpz_t(), 10, 12);  // 10^12
    
    while (true) {
        // Vérifier si cette solution est acceptable
        mpz_class sqrt_val = y * y * 8 + 1;
        mpz_class sqrt_root;
        if (mpz_perfect_square_p(sqrt_val.get_mpz_t())) {
            mpz_sqrt(sqrt_root.get_mpz_t(), sqrt_val.get_mpz_t());
            if (mpz_odd_p(sqrt_root.get_mpz_t())) {
                mpz_class blue = (sqrt_root + 1) / 2 + y;
                if (blue + y > limit) {
                    std::cout << blue << std::endl;
                    return 0;
                }
            }
        }
        
        // Créer la prochaine solution plus grande
        mpz_class nextx = x * x0 + y * y0 * 8;
        mpz_class nexty = x * y0 + y * x0;
        x = nextx;
        y = nexty;
    }
    
    return 0;
}
```
