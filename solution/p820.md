# N-ième chiffre des inverses

## Description du Problème
Soit d_n(x) le n-ième chiffre décimal de la partie fractionnaire de x, ou 0 si la partie fractionnaire a moins de n chiffres.

Par exemple, d_7(1/17) = 1 car 1/17 = 0.0588235294117647... et le 7-ième chiffre est 1.

Soit S(n) = ∑_{k=1}^n d_n(1/k).

On vous donne : S(7) = 0+0+3+0+0+6+1 = 10.

**Objectif** : Trouver S(10⁷).

---

## Exemples
### Exemple 1
**Entrée** : `n = 7`
**Sortie** : `10`
**Explication** : S(7) = 10.

---

## Approche de Résolution

### Stratégie
1. Soit 1/x = a.bcde...n de sorte que le n-ième chiffre décimal soit n.
2. Alors 10^n/x = abcde...n.op...
3. On a alors que n = ⌊10^n/x⌋ (mod 10) = ⌊10×(10^(n-1))/x⌋ (mod 10) = ⌊10 × (10^(n-1) (mod x)) / x⌋.
4. Utiliser l'exponentiation modulaire pour calculer 10^(n-1) mod x.
5. Calculer d_n(1/k) pour chaque k de 1 à n.
6. Sommer tous les résultats.

### Observations
- On peut utiliser l'exponentiation modulaire pour éviter les grands nombres.
- Le calcul peut être fait efficacement pour chaque k.
- On doit gérer le cas où 1/k a moins de n chiffres décimaux.

### Complexité
- **Temporelle** : O(n × log n) pour calculer tous les d_n(1/k).
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long d(long long n, long long k) {
    if (k == 0) return 0;
    long long power = mod_pow(10, n - 1, k);
    return (10 * power) / k;
}

long long S(long long n) {
    long long total = 0;
    for (long long k = 1; k <= n; k++) {
        total += d(n, k);
    }
    return total;
}

int main() {
    cout << S(10000000) << endl;
    return 0;
}
```
