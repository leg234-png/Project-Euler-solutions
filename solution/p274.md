# Multiplicateurs de divisibilité

## Description du Problème
Pour un nombre premier p qui est copremier avec 10, nous pouvons toujours trouver un multiplicateur de divisibilité m < p qui vérifie la divisibilité de p en vérifiant si un nombre n est divisible par p.

Par exemple, le multiplicateur de divisibilité pour 113 est 34.

f(76275) = 7627 + 5×34 = 7797 : 76275 et 7797 sont tous deux divisibles par 113.
f(12345) = 1234 + 5×34 = 1404 : 12345 et 1404 ne sont pas tous deux divisibles par 113.

La somme des multiplicateurs de divisibilité pour les nombres premiers qui sont copremiers avec 10 et inférieurs à 1000 est 39517.

**Objectif** : Quelle est la somme des multiplicateurs de divisibilité pour les nombres premiers qui sont copremiers avec 10 et inférieurs à 10⁷ ?

---

## Exemples
### Exemple 1
**Entrée** : `p = 113`
**Sortie** : `34`
**Explication** : Le multiplicateur de divisibilité pour 113 est 34.

---

## Approche de Résolution

### Stratégie
1. Décomposer un nombre en 10a + b = 0 mod p.
2. Si on a un n tel que 10n = 1 mod p, alors n(10a + b) = a + nb = 0 mod p.
3. Donc n = 10^(-1) mod p.
4. Pour chaque nombre premier p, calculer l'inverse modulaire de 10 modulo p.
5. Sommer tous les multiplicateurs.

### Observations
- Le multiplicateur de divisibilité est l'inverse modulaire de 10 modulo p.
- On peut utiliser l'algorithme d'Euclide étendu pour trouver l'inverse modulaire.
- Pour p = 3 et p = 7, il y a des cas spéciaux à traiter.

### Complexité
- **Temporelle** : O(P × log p) où P est le nombre de nombres premiers, pour calculer les inverses modulaires.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<int> list_primes(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i] && i != 2 && i != 5) {
            primes.push_back(i);
        }
    }
    return primes;
}

long long mod_inverse(long long a, long long m) {
    // Algorithme d'Euclide étendu
    long long m0 = m, t, q;
    long long x0 = 0, x1 = 1;
    
    if (m == 1) return 0;
    
    while (a > 1) {
        q = a / m;
        t = m;
        m = a % m;
        a = t;
        t = x0;
        x0 = x1 - q * x0;
        x1 = t;
    }
    
    if (x1 < 0) x1 += m0;
    return x1;
}

long long find_divisibility_multiplier(int p) {
    if (p == 3) return 1;
    if (p == 7) return 5;
    return mod_inverse(10, p);
}

long long compute(int limit) {
    vector<int> primes = list_primes(limit);
    long long total = 0;
    
    for (int p : primes) {
        total += find_divisibility_multiplier(p);
    }
    
    return total;
}

int main() {
    cout << compute(10000000) << endl;
    return 0;
}
```
