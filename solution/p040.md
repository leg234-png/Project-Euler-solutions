# Constante de Champernowne

## Description du Problème
Une fraction décimale irrationnelle est créée en concaténant les entiers positifs :

0.123456789101112131415161718192021⋯

On peut voir que le 12ème chiffre de la partie fractionnaire est 1.

Si dₙ représente le nème chiffre de la partie fractionnaire, trouver la valeur de l'expression suivante :

d₁ × d₁₀ × d₁₀₀ × d₁₀₀₀ × d₁₀₀₀₀ × d₁₀₀₀₀₀ × d₁₀₀₀₀₀₀

---

## Exemples
### Exemple 1
**Entrée** : Positions = [1, 10, 100]
**Sortie** : `1`
**Explication** : d₁ = 1, d₁₀ = 1, d₁₀₀ = 5, donc 1×1×5 = 5.

---

## Approche de Résolution

### Stratégie
1. Construire la chaîne en concaténant les nombres de 1 jusqu'à ce qu'elle soit assez longue.
2. Extraire les chiffres aux positions 1, 10, 100, 1000, 10000, 100000, 1000000.
3. Multiplier ces chiffres.
4. Retourner le produit.

### Observations
- On n'a pas besoin de construire toute la chaîne jusqu'à 1 000 000, seulement jusqu'à ce qu'on ait au moins 1 000 000 chiffres.
- On peut optimiser en calculant directement dans quelle partie de la chaîne se trouve chaque position.
- Les positions sont 1-indexées, donc on soustrait 1 pour l'indexation 0-based.

### Complexité
- **Temporelle** : O(n), où `n` est le nombre nécessaire pour atteindre 1 000 000 chiffres.
- **Spatiale** : O(n), pour stocker la chaîne.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>

int main() {
    std::string constant;
    for (int i = 1; constant.length() < 1000000; ++i) {
        constant += std::to_string(i);
    }
    
    int prod = 1;
    int positions[] = {1, 10, 100, 1000, 10000, 100000, 1000000};
    
    for (int pos : positions) {
        prod *= (constant[pos - 1] - '0');
    }
    
    std::cout << prod << std::endl;
    return 0;
}
```
