# Le rêve d'un ingénieur devient réalité

## Description du Problème
Nous pouvons occasionnellement trouver un triple-paire, ce qui signifie trois paires de nombres premiers sexy consécutives, telles que le second membre de chaque paire est le premier membre de la paire suivante.

Nous appellerons un nombre n tel que :
- (n−9,n−3), (n−3,n+3), (n+3,n+9) forment un triple-paire, et
- les nombres n−8, n−4, n, n+4 et n+8 sont tous pratiques.

**Objectif** : Trouver la somme des 4 premiers nombres n avec cette propriété.

---

## Exemples
### Exemple 1
**Entrée** : `n tel que les conditions sont satisfaites`
**Sortie** : `Somme des 4 premiers`
**Explication** : La somme des 4 premiers nombres n qui satisfont toutes les conditions.

---

## Approche de Résolution

### Stratégie
1. Vérifier que (n−9,n−3), (n−3,n+3), (n+3,n+9) sont des paires de nombres premiers sexy.
2. Vérifier que n−8, n−4, n, n+4, n+8 sont tous des nombres pratiques.
3. Un nombre pratique (panarithmique) est un nombre tel que tous les entiers positifs plus petits peuvent être écrits comme somme de diviseurs distincts.
4. Utiliser un crible pour trouver les nombres premiers et vérifier les paires sexy.
5. Pour chaque candidat n, vérifier si les 5 nombres sont pratiques.

### Observations
- Les nombres pratiques ont des propriétés spécifiques liées à leurs facteurs premiers.
- Un nombre est pratique si et seulement si ses facteurs premiers satisfont certaines conditions.
- On peut optimiser en vérifiant d'abord les paires de nombres premiers sexy.

### Complexité
- **Temporelle** : O(N × P) où N est la limite et P est le coût de vérifier si un nombre est pratique.
- **Spatiale** : O(N) pour le crible de nombres premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<bool> prime_sieve(long long n) {
    vector<bool> result(n + 1, true);
    result[0] = result[1] = false;
    for (long long i = 2; i * i <= n; i++) {
        if (result[i]) {
            for (long long j = 2 * i; j <= n; j += i) {
                result[j] = false;
            }
        }
    }
    return result;
}

vector<pair<long long, int>> prime_factors_with_exponent(long long n) {
    vector<pair<long long, int>> factors;
    long long d = 2;
    while (n > 1) {
        int count = 0;
        while (n % d == 0) {
            count++;
            n /= d;
        }
        if (count > 0) {
            factors.push_back({d, count});
        }
        d++;
        if (d * d > n) {
            if (n > 1) {
                factors.push_back({n, 1});
            }
            break;
        }
    }
    return factors;
}

bool is_practical(long long x) {
    vector<pair<long long, int>> pf = prime_factors_with_exponent(x);
    if (pf.empty() || pf[0].first != 2) {
        return false;
    }
    if (pf.size() == 1) {
        return true; // x = 2^e
    }
    
    long long total = 1;
    for (size_t i = 1; i < pf.size(); i++) {
        long long p = pf[i].first;
        if (total + 1 < p) {
            return false;
        }
        total *= (pow(p, pf[i].second + 1) - 1) / (p - 1);
    }
    return true;
}

bool is_triple_pair(long long n, const vector<bool>& is_prime) {
    if (n < 9) return false;
    return is_prime[n - 9] && is_prime[n - 3] &&
           is_prime[n - 3] && is_prime[n + 3] &&
           is_prime[n + 3] && is_prime[n + 9];
}

long long compute(long long limit) {
    vector<bool> is_prime = prime_sieve(limit + 20);
    long long sum = 0;
    int count = 0;
    
    for (long long n = 9; n <= limit && count < 4; n++) {
        if (is_triple_pair(n, is_prime)) {
            if (is_practical(n - 8) && is_practical(n - 4) && 
                is_practical(n) && is_practical(n + 4) && is_practical(n + 8)) {
                sum += n;
                count++;
            }
        }
    }
    
    return sum;
}

int main() {
    cout << compute(1000000000) << endl;
    return 0;
}
```
