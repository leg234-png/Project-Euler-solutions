# Division de nombres

## Description du Problème
On appelle un nombre n un S-number si on peut diviser sa représentation décimale en deux ou plusieurs nombres qui, additionnés, donnent la racine carrée de n.

Par exemple, 8281 est un S-number : √8281 = 8+2+81 = 82+8+1.

9801 est un S-number : √9801 = 98+0+1.

De plus, définissons T(N) comme la somme de tous les nombres S n ≤ N. On vous donne T(10⁴) = 41333.

**Objectif** : Trouver T(10¹²).

---

## Exemples
### Exemple 1
**Entrée** : `n = 8281`
**Sortie** : `8281` (S-number)
**Explication** : √8281 = 8+2+81 = 82+8+1.

---

## Approche de Résolution

### Stratégie
1. Pour chaque carré parfait n = k², vérifier si on peut diviser la représentation décimale de n en parties qui somment à k.
2. Générer toutes les façons possibles de diviser la représentation décimale.
3. Pour chaque division, vérifier si la somme des parties égale k.
4. Si oui, ajouter n à la somme totale.
5. Utiliser la récursion ou la programmation dynamique pour générer toutes les divisions possibles.

### Observations
- On n'a besoin de vérifier que les carrés parfaits.
- On peut générer les divisions de manière systématique.
- On peut utiliser la mémorisation pour éviter les recalculs.

### Complexité
- **Temporelle** : O(√N × 2^d) où d est le nombre de chiffres.
- **Spatiale** : O(d) pour la récursion.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <cmath>

using namespace std;

bool can_split(long long n, long long target) {
    string s = to_string(n);
    int len = s.length();
    
    // Générer toutes les façons de diviser la chaîne
    function<bool(int, long long)> dfs = [&](int pos, long long sum) -> bool {
        if (pos == len) {
            return sum == target;
        }
        
        for (int i = pos + 1; i <= len; i++) {
            long long num = stoll(s.substr(pos, i - pos));
            if (dfs(i, sum + num)) {
                return true;
            }
        }
        
        return false;
    };
    
    return dfs(0, 0);
}

long long compute(long long limit) {
    long long total = 0;
    long long sqrt_limit = sqrt(limit);
    
    for (long long k = 2; k <= sqrt_limit; k++) {
        long long n = k * k;
        if (can_split(n, k)) {
            total += n;
        }
    }
    
    return total;
}

int main() {
    long long limit = 1;
    for (int i = 0; i < 12; i++) {
        limit *= 10;
    }
    cout << compute(limit) << endl;
    return 0;
}
```
