# Diviseurs du produit binomial

## Description du Problème
Soit B(n) = (n!)^(n+1) / ∏_{i=1}^n (i!)^2.

Soit D(n) la somme de tous les diviseurs de B(n).

Par exemple, D(5) = 1 + 2 + 4 + 5 + 10 + 20 + 25 + 50 + 100 + 125 + 250 + 500 + 625 + 1250 + 2500 = 5467.

Soit S(n) = ∑_{k=1}^n D(k).

On vous donne S(5) = 5736, S(10) = 141740594713218418 et S(100) mod 1000000007 = 332792866.

**Objectif** : Trouver S(20000) mod 1000000007.

---

## Exemples
### Exemple 1
**Entrée** : `n = 5`
**Sortie** : `5467`
**Explication** : D(5) = 5467.

---

## Approche de Résolution

### Stratégie
1. B(n) = B(n-1) × n^n / n!, donc on peut calculer B(n) récursivement.
2. Utiliser la fonction diviseur : si n = p₁^e₁ × p₂^e₂ × ..., alors d(n) = ∏ (e_i + 1).
3. Calculer les facteurs premiers de B(n) en ajoutant les facteurs de n^n et en soustrayant les facteurs de n!.
4. Utiliser des tableaux précalculés pour les facteurs premiers.
5. Calculer D(n) en utilisant la formule des diviseurs.
6. Sommer tous les D(k) pour obtenir S(n).

### Observations
- On peut maintenir les facteurs premiers de B(n) de manière incrémentale.
- La fonction diviseur peut être calculée à partir des facteurs premiers.
- On doit gérer les grands nombres avec l'arithmétique modulaire.

### Complexité
- **Temporelle** : O(n² × log n) pour calculer tous les B(n) et leurs diviseurs.
- **Spatiale** : O(n) pour stocker les facteurs premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <map>
#include <cmath>

using namespace std;

map<int, int> prime_factors(int n) {
    map<int, int> factors;
    for (int d = 2; d * d <= n; d++) {
        while (n % d == 0) {
            factors[d]++;
            n /= d;
        }
    }
    if (n > 1) factors[n]++;
    return factors;
}

long long divisor_sum(const map<int, int>& factors, long long mod) {
    long long total = 1;
    for (auto [p, exp] : factors) {
        long long term = 1;
        long long power = 1;
        for (int i = 0; i <= exp; i++) {
            term = (term + power) % mod;
            power = (power * p) % mod;
        }
        total = (total * term) % mod;
    }
    return total;
}

long long compute(int n, long long mod) {
    map<int, int> B_factors;
    long long total = 0;
    
    for (int k = 1; k <= n; k++) {
        // Ajouter les facteurs de k^k
        map<int, int> k_factors = prime_factors(k);
        for (auto [p, exp] : k_factors) {
            B_factors[p] += exp * k;
        }
        
        // Soustraire les facteurs de k!
        for (int i = 2; i <= k; i++) {
            map<int, int> i_factors = prime_factors(i);
            for (auto [p, exp] : i_factors) {
                B_factors[p] -= 2 * exp;
            }
        }
        
        long long D_k = divisor_sum(B_factors, mod);
        total = (total + D_k) % mod;
    }
    
    return total;
}

int main() {
    long long mod = 1000000007;
    cout << compute(20000, mod) << endl;
    return 0;
}
```
