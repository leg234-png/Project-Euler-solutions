# Un million de membres

## Description du Problème
Soit t(n) le nombre de twopals dont les éléments somment à n.

On peut voir que t(6) = 4 : (1,1,2,1,1), (1,2,2,1), (2,1,1,2), (2,2,2).

De même, t(20) = 824.

En cherchant la réponse à la question ultime de la vie, l'univers et tout, on peut vérifier que t(42) = 1999923, qui se trouve être la première valeur de t(n) qui dépasse un million.

**Objectif** : Cependant, votre défi pour la question "ultimatest" de la vie, l'univers et tout est de trouver la plus petite valeur de n > 42 telle que t(n) est divisible par un million.

---

## Exemples
### Exemple 1
**Entrée** : `n = 6`
**Sortie** : `4`
**Explication** : t(6) = 4.

---

## Approche de Résolution

### Stratégie
1. Définir c1(n) et c2(n) comme des fonctions récursives.
2. c1(n) = somme de c1(n-i) pour i de 1 à n, moins c1(n-2).
3. c2(n) = somme de c2(n-i) pour i de 1 à n, plus c1(n-2).
4. t(n) dépend de c1 et c2 selon la parité de n.
5. Utiliser la programmation dynamique pour calculer t(n).
6. Chercher le plus petit n > 42 tel que t(n) ≡ 0 (mod 1000000).

### Observations
- Il y a des relations de récurrence claires pour c1 et c2.
- On peut calculer t(n) de manière incrémentale.
- On doit chercher jusqu'à trouver la solution.

### Complexité
- **Temporelle** : O(N²) où N est la valeur recherchée.
- **Spatiale** : O(N) pour les tableaux DP.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

long long mod = 1000000;

long long compute() {
    vector<long long> c1(1000000, 0);
    vector<long long> c2(1000000, 0);
    vector<long long> sc1(1000000, 0);
    vector<long long> sc2(1000000, 0);
    vector<long long> t(1000000, 0);
    
    c1[0] = c1[1] = 1;
    sc1[0] = 1;
    sc1[1] = 2;
    
    for (int n = 2; n < 1000000; n++) {
        c2[n] = (sc2[n-1] + c1[n-2]) % mod;
        c1[n] = (sc1[n-1] - c1[n-2] + mod) % mod;
        sc2[n] = (sc2[n-1] + c2[n]) % mod;
        sc1[n] = (sc1[n-1] + c1[n]) % mod;
        
        if (n % 2 == 0) {
            t[n] = (sc2[n/2] + c1[n/2 - 1]) % mod;
        } else {
            t[n] = sc2[n/2] % mod;
        }
        
        if (n > 42 && t[n] == 0) {
            return n;
        }
    }
    
    return -1;
}

int main() {
    cout << compute() << endl;
    return 0;
}
```
