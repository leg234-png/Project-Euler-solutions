# Primonacci

## Description du Problème
Soit `F(n)` le `n`-ième nombre de Fibonacci, où `F(1) = 1`, `F(2) = 1`, et `F(n) = F(n-1) + F(n-2)` pour `n > 2`.

**Objectif** : Trouver `Σ F(p)` pour les 100 000 premiers nombres premiers `p` supérieurs à `10^14`. Donner la réponse modulo `1234567891011`.

---

## Exemples
### Exemple 1
**Entrée** : `p = 2`
**Sortie** : `F(2) = 1`
**Explication** : Le premier nombre premier est 2, donc on calcule F(2).

---

## Approche de Résolution

### Stratégie
1. Utiliser un crible d'Ératosthène pour trouver les nombres premiers à partir de `BASE = 10^14`.
2. Pour chaque nombre premier `p`, calculer `F(p) mod MODULUS`.
3. Utiliser l'exponentiation rapide pour calculer les nombres de Fibonacci modulo `MODULUS` :
   - Utiliser la formule matricielle : `[F(n+1), F(n)] = [F(n), F(n-1)] × [[1,1],[1,0]]`.
   - Ou utiliser la formule : `F(2n) = F(n) × (2F(n+1) - F(n))`, `F(2n+1) = F(n+1)² + F(n)²`.
4. Additionner tous les `F(p)` modulo `MODULUS`.

### Observations
- On doit calculer `F(p)` pour de très grands `p` (jusqu'à `10^14 + 10^7`).
- L'exponentiation rapide permet de calculer `F(n)` en `O(log n)` opérations.
- On utilise un crible segmenté pour trouver les nombres premiers dans une grande plage.

### Complexité
- **Temporelle : O(√(BASE+RANGE) + 100000 × log p)`.
- **Spatiale : O(RANGE)`.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

const long long BASE = 100000000000000LL;
const int SEARCH_RANGE = 10000000;
const long long MODULUS = 1234567891011LL;

vector<int> listPrimes(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i)
                isPrime[j] = false;
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i])
            primes.push_back(i);
    }
    return primes;
}

long long fibonacciMod(long long n) {
    cpp_int a = 0, b = 1;
    for (int i = 63; i >= 0; i--) {
        cpp_int d = a * (b * 2 - a) % MODULUS;
        cpp_int e = (a * a + b * b) % MODULUS;
        a = d;
        b = e;
        if ((n >> i) & 1) {
            cpp_int c = (a + b) % MODULUS;
            a = b;
            b = c;
        }
    }
    return a.convert_to<long long>();
}

int main() {
    vector<int> primes = listPrimes(sqrt(BASE + SEARCH_RANGE));
    vector<bool> isComposite(SEARCH_RANGE, false);
    
    for (int p : primes) {
        long long start = ((BASE + p - 1) / p) * p;
        for (long long i = start - BASE; i < SEARCH_RANGE; i += p)
            isComposite[i] = true;
    }
    
    long long sum = 0;
    int count = 0;
    for (int i = 0; count < 100000; i++) {
        if (!isComposite[i]) {
            sum = (sum + fibonacciMod(BASE + i)) % MODULUS;
            count++;
        }
    }
    
    cout << sum << endl;
    return 0;
}
```
