# Probabilités pythagoriciennes

## Description du Problème
Par exemple, si k=6, a=0.2 et b=0.85, alors (k·a+1)²+(k·b+1)²=42.05.

La racine carrée de 42.05 est 6.484⋯ et lorsqu'elle est arrondie à l'entier le plus proche, elle devient 6.

Ceci est égal à k, donc il marque 6 points.

On peut montrer que s'il joue 10 tours avec k=1,k=2,…,k=10, la valeur attendue de son score total, arrondie à cinq décimales, est 10.20914.

**Objectif** : S'il joue 10⁵ tours avec k=1,k=2,k=3,…,k=10⁵, quelle est la valeur attendue de son score total, arrondie à cinq décimales ?

---

## Exemples
### Exemple 1
**Entrée** : `k=6, a=0.2, b=0.85`
**Sortie** : `6`
**Explication** : Le score est 6 car la racine carrée arrondie est égale à k.

---

## Approche de Résolution

### Stratégie
1. La probabilité de gagner peut être mappée à l'intérieur d'un tore.
2. (√((k - 0.5)² - (k×x + 1)²) - 1)/k ≤ y ≤ (√((k + 0.5)² - (k×x + 1)²) - 1)/k.
3. Pour chaque k, calculer l'aire de la région où le score est k.
4. Utiliser l'intégration numérique pour calculer l'aire.
5. Multiplier par k et sommer pour obtenir la valeur attendue.

### Observations
- La région de victoire est définie par des cercles concentriques.
- On peut utiliser l'intégration numérique (comme la méthode de Simpson) pour calculer l'aire.
- Pour chaque k, on calcule l'aire entre deux cercles.

### Complexité
- **Temporelle** : O(limit × I) où I est le nombre de points d'intégration.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <cmath>

using namespace std;

double integrate_upper(int k, double x) {
    double val = (k + 0.5) * (k + 0.5) - (k * x + 1) * (k * x + 1);
    if (val < 0) return 0;
    return (sqrt(val) - 1) / k;
}

double integrate_lower(int k, double x) {
    double val = (k - 0.5) * (k - 0.5) - (k * x + 1) * (k * x + 1);
    if (val < 0) return 0;
    return (sqrt(val) - 1) / k;
}

double compute(int limit) {
    double total = 0;
    
    // Cas k=1
    auto f_upper1 = [](double x) {
        return sqrt(1.5 * 1.5 - (x + 1) * (x + 1)) - 1;
    };
    
    double x_max = f_upper1(0);
    double area1 = 0;
    int steps = 10000;
    double dx = x_max / steps;
    for (int i = 0; i < steps; i++) {
        double x = i * dx;
        area1 += f_upper1(x) * dx;
    }
    total += area1;
    
    // Cas k >= 2
    for (int k = 2; k <= limit; k++) {
        double area_upper = 0, area_lower = 0;
        double x_max_upper = integrate_upper(k, 0);
        double x_max_lower = integrate_lower(k, 0);
        
        if (x_max_upper > 0) {
            double dx = x_max_upper / steps;
            for (int i = 0; i < steps; i++) {
                double x = i * dx;
                area_upper += integrate_upper(k, x) * dx;
            }
        }
        
        if (x_max_lower > 0) {
            double dx = x_max_lower / steps;
            for (int i = 0; i < steps; i++) {
                double x = i * dx;
                area_lower += integrate_lower(k, x) * dx;
            }
        }
        
        total += k * (area_upper - area_lower);
    }
    
    return round(total * 100000) / 100000.0;
}

int main() {
    cout << fixed << setprecision(5) << compute(100000) << endl;
    return 0;
}
```
