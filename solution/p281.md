# Garnitures de pizza

## Description du Problème
Ainsi, par exemple, f(2,1)=1, f(2,2)=f(3,1)=2 et f(3,2)=16. f(3,2) est montré ci-dessous.

**Objectif** : Trouver la somme de tous les f(m,n) tels que f(m,n) ≤ 10¹⁵.

---

## Exemples
### Exemple 1
**Entrée** : `m=2, n=1`
**Sortie** : `1`
**Explication** : f(2,1) = 1.

### Exemple 2
**Entrée** : `m=3, n=2`
**Sortie** : `16`
**Explication** : f(3,2) = 16.

---

## Approche de Résolution

### Stratégie
1. Utiliser le lemme de Cauchy-Frobenius / théorème d'énumération de Pólya.
2. Nous avons |S\G| = 1/(m×n) × ∑_{d|m×n} φ(d) × x_d^(m×n/d).
3. Remplacer x_d par a^i + b^i + ... + c^i qui représentent les couleurs.
4. Rechercher le coefficient a^n×b^n×...×c^n.
5. Calculer f(m,n) pour tous les m,n tels que f(m,n) ≤ 10¹⁵.

### Observations
- Le théorème de Pólya permet de compter les colorations distinctes sous l'action d'un groupe.
- On utilise la fonction d'Euler φ pour compter les éléments du groupe.
- On doit calculer les coefficients dans le développement polynomial.

### Complexité
- **Temporelle** : O(m×n×D) où D est le nombre de diviseurs de m×n.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long phi(long long n) {
    if (n == 1) return 1;
    long long result = 1;
    long long d = 2;
    while (n > 1) {
        int count = 0;
        while (n % d == 0) {
            count++;
            n /= d;
        }
        if (count > 0) {
            result *= pow(d, count - 1) * (d - 1);
        }
        d++;
        if (d * d > n) {
            if (n > 1) {
                result *= (n - 1);
            }
            break;
        }
    }
    return result;
}

long long factorial(long long n) {
    long long result = 1;
    for (long long i = 2; i <= n; i++) {
        result *= i;
    }
    return result;
}

long long f(long long m, long long n) {
    long long total = 0;
    long long mn = m * n;
    for (long long d = 1; d <= n; d++) {
        if (n % d == 0) {
            total += phi(n / d) * factorial(m * d) / pow(factorial(d), m);
        }
    }
    return total / (m * n);
}

long long compute(long long limit) {
    long long total = 0;
    for (long long m = 2; m < 100; m++) {
        for (long long n = 1; n < 100; n++) {
            long long t = f(m, n);
            if (t <= limit) {
                total += t;
            } else {
                break;
            }
        }
    }
    return total;
}

int main() {
    cout << compute(1000000000000000LL) << endl;
    return 0;
}
```
