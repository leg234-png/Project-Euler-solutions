# Multiples de 3 ou 5

## Description du Problème
Si on liste tous les nombres naturels inférieurs à 10 qui sont des multiples de 3 ou 5, on obtient 3, 5, 6 et 9. La somme de ces multiples est 23.

**Objectif** : Trouver la somme de tous les multiples de 3 ou 5 inférieurs à 1000.

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : `23`
**Explication** : Les multiples de 3 ou 5 inférieurs à 10 sont 3, 5, 6 et 9. Leur somme est 23.

---

## Approche de Résolution

### Stratégie
1. Parcourir tous les nombres de 0 à 999.
2. Pour chaque nombre, vérifier s'il est divisible par 3 **ou** par 5.
3. Si c'est le cas, l'ajouter à la somme totale.
4. Retourner la somme totale.

### Observations
- L'opérateur modulo (`%`) permet de vérifier la divisibilité :
  - `n % 3 == 0` → divisible par 3.
  - `n % 5 == 0` → divisible par 5.
- Utiliser l'opérateur logique **OU** (`||`) pour vérifier les deux conditions.

### Complexité
- **Temporelle** : O(n), où `n` est la limite supérieure (ici, 1000).
- **Spatiale** : O(1), car on utilise seulement une variable pour accumuler la somme.

---

## Solution en C++

```cpp
#include <iostream>

int main() {
    int sum = 0;
    for (int i = 0; i < 1000; ++i) {
        if (i % 3 == 0 || i % 5 == 0) {
            sum += i;
        }
    }
    std::cout << sum << std::endl;
    return 0;
}
