# Déchiffrement XOR

## Description du Problème
Chaque caractère d'un ordinateur est assigné un code unique et le standard préféré est ASCII (American Standard Code for Information Interchange). Par exemple, le caractère majuscule A = 65, astérisque (*) = 42, et le caractère minuscule k = 107.

Une méthode moderne de cryptage consiste à prendre un fichier texte, convertir les octets en ASCII, puis XOR chaque octet avec une valeur donnée, prise d'une clé secrète. L'avantage de la fonction XOR est qu'en utilisant la même clé de chiffrement, on peut récupérer le texte original. Par exemple, 65 XOR 42 = 107, puis 107 XOR 42 = 65.

Pour un chiffrement non cassable, la clé est de la même longueur que le message en texte clair, la clé est faite de caractères aléatoires, et la clé n'est jamais réutilisée. Cependant, la clé utilisée ici est courte et sera répétée cycliquement tout au long du message. Le message de chiffrement est dans le fichier cipher.txt.

**Objectif** : Décrypter le message et trouver la somme des valeurs ASCII dans le texte original.

---

## Exemples
### Exemple 1
**Entrée** : Texte chiffré avec clé courte
**Sortie** : Somme des valeurs ASCII du texte déchiffré
**Explication** : On essaie toutes les clés possibles de 3 lettres et on choisit celle qui produit le texte le plus lisible.

---

## Approche de Résolution

### Stratégie
1. Lire le texte chiffré depuis le fichier.
2. Pour chaque clé possible de 3 lettres (a-z) :
   - Déchiffrer le texte en XORant avec la clé répétée.
   - Calculer un score de "lisibilité" (plus de lettres = meilleur score).
   - Garder la clé avec le meilleur score.
3. Calculer la somme des valeurs ASCII du texte déchiffré.
4. Retourner la somme.

### Observations
- La clé est de 3 lettres et se répète cycliquement.
- On utilise une fonction heuristique pour évaluer la lisibilité (lettres = bon, caractères de contrôle = mauvais).
- On teste toutes les combinaisons possibles de 3 lettres.

### Complexité
- **Temporelle** : O(26³ × n), où n est la longueur du texte chiffré.
- **Spatiale** : O(n), pour stocker le texte déchiffré.

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <fstream>
#include <sstream>
#include <string>
#include <algorithm>

double score(const std::vector<int>& text) {
    double sum = 0;
    for (int c : text) {
        if ('A' <= c && c <= 'Z') {
            sum += 1;  // Uppercase letters are good
        } else if ('a' <= c && c <= 'z') {
            sum += 2;  // Lowercase letters are excellent
        } else if (c < 0x20 || c == 0x7F) {
            sum -= 10;  // Control characters are very bad
        }
    }
    return sum;
}

std::vector<int> decrypt(const std::vector<int>& ciphertext, const std::vector<int>& key) {
    std::vector<int> plaintext;
    for (size_t i = 0; i < ciphertext.size(); ++i) {
        plaintext.push_back(ciphertext[i] ^ key[i % key.size()]);
    }
    return plaintext;
}

int main() {
    std::ifstream file("cipher.txt");
    std::string line;
    std::vector<int> ciphertext;
    
    while (std::getline(file, line, ',')) {
        ciphertext.push_back(std::stoi(line));
    }
    
    std::vector<int> bestKey;
    std::vector<int> bestDecrypted;
    double bestScore = -1e9;
    
    for (int x = 'a'; x <= 'z'; ++x) {
        for (int y = 'a'; y <= 'z'; ++y) {
            for (int z = 'a'; z <= 'z'; ++z) {
                std::vector<int> key = {x, y, z};
                std::vector<int> decrypted = decrypt(ciphertext, key);
                double s = score(decrypted);
                
                if (s > bestScore) {
                    bestKey = key;
                    bestDecrypted = decrypted;
                    bestScore = s;
                }
            }
        }
    }
    
    int sum = 0;
    for (int c : bestDecrypted) {
        sum += c;
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
