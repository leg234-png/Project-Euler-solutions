# Sommes de totients de puissances

## Description du Problème
Soit f(n) = ∑ᵢ₌₁ⁿ φ(nⁱ) mod (n+1).

On vous donne que g(100) = 2007.

**Objectif** : Trouver g(5×10⁸).

---

## Exemples
### Exemple 1
**Entrée** : `n = 100`
**Sortie** : `2007`
**Explication** : g(100) = 2007.

---

## Approche de Résolution

### Stratégie
1. Observer que f(n) = 0 quand n est pair et est égal à φ(n) quand n est impair.
2. φ(nᵏ) = n^(k-1) × φ(n).
3. φ(nᵏ) = (-1)^(k-1) φ(n) mod (n+1).
4. Donc f(n) = ∑ᵢ₌₁ⁿ (-1)^(i-1) φ(n) mod (n+1).
5. Quand n est pair, la somme alterne et donne 0.
6. Quand n est impair, f(n) = φ(n).

### Observations
- On peut simplifier considérablement le calcul.
- On n'a besoin de calculer φ(n) que pour les n impairs.
- On peut utiliser un crible de totients pour calculer efficacement.

### Complexité
- **Temporelle** : O(n log log n) pour le crible de totients.
- **Spatiale** : O(n) pour stocker les valeurs de φ.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

long long totient_sieve(long long n) {
    vector<long long> phi(n + 1);
    for (long long i = 0; i <= n; i++) {
        phi[i] = i;
    }
    
    for (long long p = 3; p <= n; p += 2) {
        if (phi[p] == p) {
            phi[p] -= 1;
            for (long long i = 3 * p; i <= n; i += 2 * p) {
                phi[i] -= (phi[i] / p);
            }
        }
    }
    
    long long total = 0;
    for (long long x = 1; x <= n; x += 2) {
        total += phi[x];
    }
    return total;
}

int main() {
    cout << totient_sieve(500000000) << endl;
    return 0;
}
```
