# Nombre avec 2^500500 diviseurs

## Description du Problème
Le plus petit nombre avec exactement 2 diviseurs est 2.
Le plus petit nombre avec exactement 4 diviseurs est 6.
Le plus petit nombre avec exactement 8 diviseurs est 24.

**Objectif** : Trouver le plus petit nombre avec exactement 2^500500 diviseurs. Donner la réponse modulo 500500507.

---

## Exemples
### Exemple 1
**Entrée** : `k = 2`
**Sortie** : `2`
**Explication** : 2 a exactement 2 diviseurs (1, 2).

### Exemple 2
**Entrée** : `k = 4`
**Sortie** : `6`
**Explication** : 6 a exactement 4 diviseurs (1, 2, 3, 6).

---

## Approche de Résolution

### Stratégie
1. Si n = p1^k1 × p2^k2 × ... × p_m^k_m, alors n a exactement (k1+1) × (k2+1) × ... × (k_m+1) diviseurs.
2. Pour avoir 2^500500 diviseurs, chaque exposant plus un doit être une puissance de 2.
3. Utiliser un algorithme glouton avec une file de priorité :
   - Commencer avec le produit = 1.
   - À chaque étape, choisir le plus petit élément parmi :
     - Les prochaines puissances des nombres premiers déjà utilisés
     - Le prochain nombre premier non utilisé
   - Multiplier le produit par cet élément.
   - Mettre le carré de l'élément dans la file de priorité.
4. Répéter 500500 fois.

### Observations
- On peut multiplier un nombre par un nombre premier non utilisé pour doubler le nombre de diviseurs.
- On peut aussi augmenter l'exposant d'un nombre premier existant pour doubler le nombre de diviseurs.
- L'algorithme glouton choisit toujours l'option la moins coûteuse.

### Complexité
- **Temporelle** : O(T × log P), où T = 500500 et P est le nombre de nombres premiers utilisés.
- **Spatiale** : O(P), pour la file de priorité.

---

## Solution en C++

```cpp
#include <iostream>
#include <queue>
#include <vector>
#include <cmath>

using namespace std;

const int TARGET = 500500;
const long long MODULUS = 500500507LL;

bool isPrime(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    
    int sqrtN = (int)sqrt(n);
    for (int i = 3; i <= sqrtN; i += 2) {
        if (n % i == 0) {
            return false;
        }
    }
    return true;
}

int main() {
    priority_queue<long long, vector<long long>, greater<long long>> queue;
    int nextPrime = 2;
    queue.push(nextPrime);
    
    long long product = 1;
    for (int i = 0; i < TARGET; i++) {
        long long item = queue.top();
        queue.pop();
        product = product * (item % MODULUS) % MODULUS;
        queue.push(item * item);
        
        if (item == nextPrime) {
            do {
                nextPrime++;
            } while (!isPrime(nextPrime));
            queue.push(nextPrime);
        }
    }
    
    cout << product << endl;
    return 0;
}
```
