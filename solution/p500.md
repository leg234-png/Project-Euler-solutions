# Nombre avec 2^500500 diviseurs

## Description du Problème
Le nombre de diviseurs de 120 est 16.

En fait, 120 est le plus petit nombre ayant 16 diviseurs.

**Objectif** : Trouver le plus petit nombre avec 2^500500 diviseurs. Donner votre réponse modulo 500500507.

---

## Exemples
### Exemple 1
**Entrée** : `16 diviseurs`
**Sortie** : `120`
**Explication** : 120 est le plus petit nombre avec 16 diviseurs.

---

## Approche de Résolution

### Stratégie
1. Si n = p₁^a × p₂^b × p₃^c × ..., alors d(n) = (a+1)(b+1)(c+1)...
2. On veut d(n) = 2^500500 = (a+1)(b+1)(c+1)...
3. On construit n = p₁^a × p₂^b × ... de manière à minimiser n.
4. Observer que d(2×3×5×...×7376507) = 2^500500 (produit de tous les nombres premiers).
5. Algorithme : choisir d'augmenter une des puissances à la puissance de 2 suivante ou d'ajouter un nouveau nombre premier.
6. Toujours choisir l'option qui minimise n.

### Observations
- On peut construire les nombres de manière gloutonne.
- À chaque étape, on choisit entre augmenter une puissance existante ou ajouter un nouveau premier.
- On choisit l'option qui donne le plus petit n.

### Complexité
- **Temporelle** : O(500500 × log P) où P est le nombre de nombres premiers.
- **Spatiale** : O(500500) pour stocker les puissances.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <cmath>

using namespace std;

vector<int> primes_up_to(int limit) {
    vector<bool> is_prime(limit + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= limit; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= limit; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= limit; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

long long compute(long long mod) {
    vector<int> primes = primes_up_to(10000000);
    priority_queue<pair<long long, pair<int, int>>, 
                   vector<pair<long long, pair<int, int>>>, 
                   greater<pair<long long, pair<int, int>>>> pq;
    
    for (int p : primes) {
        pq.push({p, {p, 1}}); // {valeur, {premier, exposant}}
    }
    
    long long result = 1;
    int divisors_count = 1;
    int target = 500500;
    
    while (divisors_count < target) {
        auto [value, info] = pq.top();
        pq.pop();
        int p = info.first;
        int exp = info.second;
        
        result = (result * value) % mod;
        divisors_count *= 2;
        
        // Ajouter la prochaine option pour ce premier
        long long next_value = (long long)pow(p, exp + 1);
        pq.push({next_value, {p, exp + 1}});
    }
    
    return result;
}

int main() {
    long long mod = 500500507;
    cout << compute(mod) << endl;
    return 0;
}
```
