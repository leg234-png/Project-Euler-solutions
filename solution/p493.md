# Sous l'arc-en-ciel

## Description du Problème
70 boules colorées sont placées dans une urne, 10 pour chacune des sept couleurs de l'arc-en-ciel.

**Objectif** : Quel est le nombre attendu de couleurs distinctes dans 20 boules choisies au hasard ?

Donner votre réponse avec neuf chiffres après la virgule (a.bcdefghij).

---

## Exemples
### Exemple 1
**Entrée** : `20 boules choisies`
**Sortie** : `Nombre attendu`
**Explication** : Le nombre attendu de couleurs distinctes.

---

## Approche de Résolution

### Stratégie
1. Approche mathématique : trouver la probabilité qu'une couleur ne fasse pas partie des 20 boules.
2. C'est-à-dire, les 20 boules sont choisies parmi 60 (excluant cette couleur).
3. Donc on a C(60,20) / C(70,20) comme probabilité qu'une couleur ne soit pas choisie.
4. Donc la probabilité qu'une couleur soit choisie est 1 - C(60,20) / C(70,20).
5. Faire cela pour chaque couleur : 7(1 - C(60,20) / C(70,20)).

### Observations
- Par linéarité de l'espérance, E[nombre de couleurs] = ∑P(couleur i est choisie).
- Toutes les couleurs sont équivalentes, donc on multiplie par 7.
- On peut calculer les combinaisons efficacement.

### Complexité
- **Temporelle** : O(1) pour le calcul direct.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <cmath>

using namespace std;

double n_choose_r(int n, int r) {
    if (r > n) return 0;
    double result = 1;
    for (int i = 0; i < r; i++) {
        result = result * (n - i) / (i + 1);
    }
    return result;
}

double compute() {
    double prob_not_chosen = n_choose_r(60, 20) / n_choose_r(70, 20);
    double prob_chosen = 1.0 - prob_not_chosen;
    return 7.0 * prob_chosen;
}

int main() {
    cout << fixed << setprecision(9) << compute() << endl;
    return 0;
}
```
