# Nombres Triangulaires Codés

## Description du Problème
Le nème terme de la séquence des nombres triangulaires est donné par, tₙ = ½n(n+1) ; donc les dix premiers nombres triangulaires sont :

1, 3, 6, 10, 15, 21, 28, 36, 45, 55, ...

En convertissant chaque lettre d'un mot en un nombre correspondant à sa position dans l'alphabet et en additionnant ces valeurs, on forme une valeur de mot. Par exemple, la valeur de mot de SKY est 19 + 11 + 25 = 55 = t₁₀. Si la valeur de mot est un nombre triangulaire, on appellera le mot un mot triangulaire.

**Objectif** : En utilisant words.txt, combien de mots sont triangulaires ?

---

## Exemples
### Exemple 1
**Entrée** : Liste de mots
**Sortie** : `162`
**Explication** : Le nombre de mots triangulaires dans la liste.

---

## Approche de Résolution

### Stratégie
1. Lire tous les mots du fichier.
2. Pour chaque mot :
   - Calculer sa valeur de mot (somme des positions des lettres).
   - Vérifier si cette valeur est un nombre triangulaire.
   - Si oui, compter le mot.
3. Retourner le compte.

### Observations
- Un nombre x est triangulaire s'il existe n tel que x = n(n+1)/2.
- On peut résoudre n² + n - 2x = 0 et vérifier si la solution est un entier positif.
- Ou générer les nombres triangulaires jusqu'à une limite et vérifier l'appartenance.

### Complexité
- **Temporelle** : O(n × m), où `n` est le nombre de mots et `m` est la longueur moyenne d'un mot.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <cmath>

bool isTriangularNumber(int x) {
    // Résoudre n² + n - 2x = 0
    // n = (-1 + sqrt(1 + 8x)) / 2
    int discriminant = 1 + 8 * x;
    int sqrtD = sqrt(discriminant);
    
    if (sqrtD * sqrtD == discriminant) {
        return (sqrtD - 1) % 2 == 0;
    }
    return false;
}

int wordValue(const std::string& word) {
    int sum = 0;
    for (char c : word) {
        if (c >= 'A' && c <= 'Z') {
            sum += c - 'A' + 1;
        }
    }
    return sum;
}

int main() {
    // Note: Les mots devraient être lus depuis le fichier words.txt
    // Pour cet exemple, on suppose qu'ils sont dans un vecteur
    std::vector<std::string> words = {
        "A", "ABILITY", "ABLE", "ABOUT", "ABOVE"
        // ... (tous les mots du fichier)
    };
    
    int count = 0;
    for (const std::string& word : words) {
        if (isTriangularNumber(wordValue(word))) {
            count++;
        }
    }
    
    std::cout << count << std::endl;
    return 0;
}
```
