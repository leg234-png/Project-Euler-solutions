# Inverse modulaire égal à lui-même

## Description du Problème
Soit m un entier positif. Un nombre n est appelé un inverse modulaire de m modulo m si n × n ≡ 1 (mod m).

Par exemple, 3 est un inverse modulaire de 5 modulo 5 car 3 × 3 = 9 ≡ 1 (mod 5).

Soit I(n) le plus grand nombre m < n tel que m est un inverse modulaire de n modulo n. Par exemple, I(5) = 3.

**Objectif** : Trouver ΣI(n) pour 3 <= n <= 2×10⁷.

---

## Exemples
### Exemple 1
**Entrée** : `n = 5`
**Sortie** : `3`
**Explication** : I(5) = 3 car 3² = 9 ≡ 1 (mod 5) et 3 est le plus grand nombre < 5 avec cette propriété.

---

## Approche de Résolution

### Stratégie
1. Si m est un inverse modulaire de n modulo n, alors m² = 1 mod n.
2. Si n est premier, alors m² = 1 mod n a exactement deux solutions : m = 1, n-1.
3. Si n n'est pas premier, utiliser la factorisation en nombres premiers et le théorème des restes chinois.
4. Soit p le plus petit facteur premier de n. Si p = n, alors les solutions sont {1, n-1}.
5. Sinon, utiliser les solutions pour n/p (déjà calculées) et les étendre en vérifiant m' + 0k, m' + 1k, ..., m' + (p-1)k modulo n.
6. Garder la deuxième plus grande solution pour chaque n.

### Observations
- On construit un tableau des plus petits facteurs premiers pour factoriser rapidement.
- On stocke les solutions pour chaque n/p et on les étend pour n.
- On utilise le théorème des restes chinois pour combiner les solutions.

### Complexité
- **Temporelle** : O(N × P), où N = 2×10⁷ et P est le nombre moyen de solutions.
- **Spatiale** : O(N), pour stocker les solutions.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

const int LIMIT = 20000000;

vector<int> listSmallestPrimeFactors(int n) {
    vector<int> smallestPrimeFactor(n + 1, 0);
    for (int i = 2; i <= n; i++) {
        if (smallestPrimeFactor[i] == 0) {
            smallestPrimeFactor[i] = i;
            for (int j = i * 2; j <= n; j += i) {
                if (smallestPrimeFactor[j] == 0) {
                    smallestPrimeFactor[j] = i;
                }
            }
        }
    }
    return smallestPrimeFactor;
}

// Retourne toutes les solutions (en ordre croissant) telles que
// pour chaque k, 1 <= k < n et k² = 1 mod n.
vector<int> getSolutions(int n, const vector<int>& smallestPrimeFactor, 
                         const vector<vector<int>>& solutions) {
    if (smallestPrimeFactor[n] == n) {  // n est premier
        return {1, n - 1};
    } else {
        vector<int> temp;
        int p = smallestPrimeFactor[n];
        vector<int> sols = solutions[n / p];
        int inc = n / p;
        
        for (int i = 0; i < n; i += inc) {
            for (int j : sols) {
                int k = i + j;
                if ((long long)k * k % n == 1) {
                    temp.push_back(k);
                }
            }
        }
        
        sort(temp.begin(), temp.end());
        return temp;
    }
}

int main() {
    vector<int> smallestPrimeFactor = listSmallestPrimeFactors(LIMIT);
    vector<vector<int>> solutions(LIMIT / 2 + 1);
    
    solutions[0] = {};
    solutions[1] = {};
    solutions[2] = {1};
    
    long long sum = 0;
    for (int i = 3; i <= LIMIT; i++) {
        vector<int> sols = getSolutions(i, smallestPrimeFactor, solutions);
        if (i <= LIMIT / 2) {
            solutions[i] = sols;
        }
        if (sols.size() >= 2) {
            sum += sols[sols.size() - 2];  // Deuxième plus grande solution
        }
    }
    
    cout << sum << endl;
    return 0;
}
```
