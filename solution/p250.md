# 250250

## Description du Problème
Trouver le nombre de sous-ensembles non vides de `{1^1, 2^2, 3^3, ..., 250250^250250}` dont la somme est divisible par 250. Donner les 16 derniers chiffres de la réponse.

---

## Exemples
### Exemple 1
**Entrée** : `n = 3`
**Sortie** : Nombre de sous-ensembles dont la somme est divisible par 250
**Explication** : On compte les sous-ensembles de {1, 4, 27} dont la somme est divisible par 250.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique modulo 250.
2. `numSubsets[i]` : nombre de sous-ensembles dont la somme ≡ `i (mod 250)`.
3. Pour chaque nombre `k` de 1 à 250250 :
   - Calculer `val = k^k mod 250`.
   - Pour chaque résidu `j` modulo 250 :
     - `newArray[(j + val) % 250] += numSubsets[j] + numSubsets[(j + val) % 250]`.
4. La réponse est `numSubsets[0] - 1` (soustraire 1 pour exclure l'ensemble vide).

### Observations
- On travaille modulo 250 pour réduire l'espace de recherche.
- On peut calculer `k^k mod 250` efficacement avec l'exponentiation modulaire.

### Complexité
- **Temporelle : O(n × 250)**, où n=250250.
- **Spatiale : O(250)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

const long long MODULUS = 10000000000000000LL;

long long powMod(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp % 2 == 1)
            result = (result * base) % mod;
        base = (base * base) % mod;
        exp /= 2;
    }
    return result;
}

int main() {
    vector<long long> numSubsets(250, 0);
    numSubsets[0] = 1;
    
    for (int i = 1; i <= 250250; i++) {
        int temp = powMod(i, i, 250);
        vector<long long> newArray(250, 0);
        for (int j = 0; j < 250; j++) {
            newArray[(j + temp) % 250] = (numSubsets[j] + numSubsets[(j + temp) % 250]) % MODULUS;
        }
        numSubsets = newArray;
    }
    
    long long result = (numSubsets[0] - 1 + MODULUS) % MODULUS;
    cout << result << endl;
    return 0;
}
```
