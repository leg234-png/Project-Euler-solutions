# Special Isosceles Triangles

## Description du Problème
Considérons le triangle isocèle avec la longueur de base, b = 16, et les jambes, L = 17.

En utilisant le théorème de Pythagore, on peut voir que la hauteur du triangle, h = √(17² - 8²) = 15, qui est un de moins que la longueur de base.

Avec b = 272 et L = 305, on obtient h = 273, qui est un de plus que la longueur de base, et c'est le deuxième plus petit triangle isocèle avec la propriété que h = b ± 1.

**Objectif** : Trouver ∑L pour les douze plus petits triangles isocèles pour lesquels h = b ± 1 et b, L sont des entiers positifs.

---

## Exemples
### Exemple 1
**Entrée** : `b = 16, L = 17`
**Sortie** : `17`
**Explication** : h = 15 = b - 1.

---

## Approche de Résolution

### Stratégie
1. Pour un triangle isocèle avec base b et jambes L, h = √(L² - (b/2)²).
2. Si h = b ± 1, alors L² = (b/2)² + (b ± 1)².
3. Cela donne L² = (5b² ± 4b + 4) / 4.
4. Pour que L soit entier, 5b² ± 4b + 4 doit être un carré parfait.
5. Utiliser les triplets de Pythagore ou une recherche pour trouver les solutions.

### Observations
- Les solutions suivent une séquence récurrente.
- On peut utiliser les approximations de Pell ou une recherche directe.

### Complexité
- **Temporelle** : O(n), où n est le nombre de solutions à trouver.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

int main() {
    // Les triangles suivent une séquence récurrente
    // L(n) = 18 * L(n-1) - L(n-2) - 8 pour h = b - 1
    // L(n) = 18 * L(n-1) - L(n-2) + 8 pour h = b + 1
    
    std::vector<long long> L;
    L.push_back(17);  // Premier triangle
    L.push_back(305); // Deuxième triangle
    
    // Générer les 12 plus petits
    for (int i = 2; i < 12; i++) {
        L.push_back(18 * L[i-1] - L[i-2] - 8);
    }
    
    long long sum = 0;
    for (long long l : L) {
        sum += l;
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
