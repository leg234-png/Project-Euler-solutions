# Totient Chains

## Description du Problème
Soit `φ` la fonction indicatrice d'Euler (totient). Une chaîne de totient de longueur `n` est une séquence `1 = a₁, a₂, ..., aₙ` où `aᵢ = φ(aᵢ₋₁)` pour `2 ≤ i ≤ n`.

Par exemple, en commençant par 5, on obtient la chaîne : 5 → 4 → 2 → 1, donc une chaîne de longueur 4.

**Objectif** : Trouver la somme de tous les nombres premiers inférieurs à 40 000 000 qui forment une chaîne de totient de longueur exactement 25.

---

## Exemples
### Exemple 1
**Entrée** : `n = 5`
**Sortie** : `4`
**Explication** : 5 → φ(5)=4 → φ(4)=2 → φ(2)=1, donc longueur 4.

---

## Approche de Résolution

### Stratégie
1. Pré-calculer `φ(n)` pour tous les `n` jusqu'à la limite en utilisant un crible.
2. Calculer la longueur de chaîne pour chaque nombre :
   - `chainLength[1] = 1`.
   - `chainLength[n] = chainLength[φ(n)] + 1`.
3. Pour chaque nombre premier `p` (vérifié par `φ(p) = p-1`) :
   - Si `chainLength[p] == 25`, ajouter `p` à la somme.
4. Remplir le tableau `chainLength` en ordre croissant car les chaînes sont strictement décroissantes.

### Observations
- Les chaînes de totient sont strictement décroissantes (sauf pour 1).
- On peut calculer les longueurs de chaîne en ordre croissant.
- Un nombre est premier si et seulement si `φ(n) = n-1`.

### Complexité
- **Temporelle : O(n log log n)** pour le crible.
- **Spatiale : O(n)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

const int LIMIT = 40000000;

vector<int> listTotients(int n) {
    vector<int> totient(n + 1);
    for (int i = 0; i <= n; i++)
        totient[i] = i;
    
    for (int i = 2; i <= n; i++) {
        if (totient[i] == i) {  // i is prime
            for (int j = i; j <= n; j += i)
                totient[j] -= totient[j] / i;
        }
    }
    return totient;
}

int main() {
    vector<int> totient = listTotients(LIMIT - 1);
    vector<int> totientChainLength(totient.size());
    totientChainLength[0] = 0;
    
    long long sum = 0;
    for (int i = 1; i < totient.size(); i++) {
        int chainlen = totientChainLength[totient[i]] + 1;
        totientChainLength[i] = chainlen;
        if (chainlen == 25 && totient[i] == i - 1)  // i is prime
            sum += i;
    }
    
    cout << sum << endl;
    return 0;
}
```
