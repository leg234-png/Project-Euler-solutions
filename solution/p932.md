# 2025

## Description du Problème
Un nombre 2025 est un nombre positif qui peut être écrit comme la concaténation de deux nombres positifs a et b tels que a² - b² = 2025.

Par exemple, 9801 est un nombre 2025 car 9801 = 98 || 01 et 98² - 1² = 9604 - 1 = 9603 ≠ 2025. Attendez, vérifions : 9801 peut être divisé comme 98 || 01, mais 98² - 1² = 9604 - 1 = 9603.

Notez que 9801 n'est pas un nombre 2025 car la concaténation de 98 et 1 est 981.

Soit T(n) la somme de tous les nombres 2025 avec n chiffres ou moins. On vous donne T(4) = 5131.

**Objectif** : Trouver T(16).

---

## Exemples
### Exemple 1
**Entrée** : `n = 4`
**Sortie** : `5131`
**Explication** : T(4) = 5131.

---

## Approche de Résolution

### Stratégie
1. Un nombre 2025 peut être écrit comme a || b où a² - b² = 2025.
2. a² - b² = (a-b)(a+b) = 2025.
3. Pour chaque factorisation de 2025, trouver a et b.
4. Pour chaque b de 1 à 10^(n/2), calculer si on peut former un nombre 2025.
5. v = 4b(1 - 10^n) + 10^(2n), si v > 0 et est un carré parfait, alors on a une solution.
6. a = (10^n - 2b ± √v) / 2.
7. Compter tous les nombres 2025 valides et les sommer.

### Observations
- On peut utiliser la factorisation de 2025 pour trouver les solutions.
- Il faut vérifier que la concaténation donne bien le nombre attendu.
- On peut optimiser en limitant la plage de recherche pour b.

### Complexité
- **Temporelle** : O(10^(n/2) × log n) pour tester tous les b.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>
#include <string>

using namespace std;

long long compute(int N) {
    long long total = -1; // Pour exclure (a, b) = (0, 1)
    
    for (int b = 1; b < pow(10, N / 2); b++) {
        int n = to_string(b).length();
        long long v = 4 * b * (1 - (long long)pow(10, n)) + (long long)pow(10, 2 * n);
        
        if (v > 0) {
            long long sqrt_v = sqrt(v);
            if (sqrt_v * sqrt_v == v) {
                long long a1 = ((long long)pow(10, n) - 2 * b + sqrt_v) / 2;
                total += a1 * (long long)pow(10, n) + b;
                
                long long a2 = ((long long)pow(10, n) - 2 * b - sqrt_v) / 2;
                if (a2 > 0) {
                    total += a2 * (long long)pow(10, n) + b;
                }
            }
        }
    }
    
    return total;
}

int main() {
    cout << compute(16) << endl;
    return 0;
}
```
