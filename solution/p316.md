# Nombres dans les expansions décimales

## Description du Problème
Soit p un nombre réel positif avec une expansion décimale qui commence par les chiffres de n : p = 0.n...

Soit k le plus petit entier positif tel que les k premiers chiffres décimaux de p coïncident avec les k chiffres décimaux de n. Aussi, soit g(n) la valeur attendue de k ; on peut prouver que g(n) est toujours finie et, intéressamment, toujours un nombre entier.

Par exemple, si n=535, alors pour p=31415926535897⋯, on obtient k=9, pour p=355287143650049560000490848764084685354⋯, on obtient k=36, etc. et on trouve que g(535)=1008.

**Objectif** : Étant donné que ∑ₙ₌₂⁹⁹⁹ g(⌊10⁶n⌋) = 27280188, trouver ∑ₙ₌₂⁹⁹⁹⁹⁹⁹ g(⌊10¹⁶n⌋).

---

## Exemples
### Exemple 1
**Entrée** : `n = 535`
**Sortie** : `1008`
**Explication** : g(535) = 1008.

---

## Approche de Résolution

### Stratégie
1. Utiliser les chaînes de Markov pour modéliser le problème.
2. Définir les états : état 0 (base), état 1 (si on obtient le premier chiffre), état 2 (si on obtient les deux premiers chiffres), etc.
3. Construire la matrice de transition de Markov.
4. Utiliser la formule pour les chaînes de Markov absorbantes : N = (I - P)⁻¹, puis t = N × [1,1,1]ᵀ.
5. t[0] donne la réponse pour g(n).

### Observations
- C'est un problème de chaîne de Markov absorbante.
- La probabilité de passer d'un état à un autre dépend des chiffres de n.
- On peut résoudre le système d'équations linéaires pour obtenir g(n).

### Complexité
- **Temporelle** : O(L³) où L est la longueur de n, pour inverser la matrice.
- **Spatiale** : O(L²) pour stocker la matrice.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <iomanip>

using namespace std;

double solve_markov_chain(const string& n) {
    int len = n.length();
    vector<vector<double>> P(len + 1, vector<double>(len + 1, 0.0));
    
    // Construire la matrice de transition
    for (int i = 0; i < len; i++) {
        string prefix = n.substr(0, i);
        for (int digit = 0; digit < 10; digit++) {
            string new_prefix = prefix + to_string(digit);
            int next_state = 0;
            for (int j = 0; j <= new_prefix.length(); j++) {
                if (n.substr(0, j) == new_prefix.substr(new_prefix.length() - j)) {
                    next_state = j;
                }
            }
            P[i][next_state] += 0.1;
        }
    }
    P[len][len] = 1.0; // État absorbant
    
    // Résoudre (I - P) × t = [1,1,...,1]ᵀ
    // ... logique de résolution de système linéaire ...
    
    return 0; // Placeholder
}

long long compute(int limit) {
    long long total = 0;
    for (int n = 2; n <= limit; n++) {
        string n_str = to_string((long long)pow(10, 16) * n);
        total += (long long)solve_markov_chain(n_str);
    }
    return total;
}

int main() {
    cout << compute(999999) << endl;
    return 0;
}
```
