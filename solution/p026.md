# Cycle Décimal Réciproque

## Description du Problème
Une fraction unitaire contient 1 au numérateur. La représentation décimale des fractions unitaires avec des dénominateurs de 2 à 10 sont données :

1/2 = 0.5
1/3 = 0.(3)
1/4 = 0.25
1/5 = 0.2
1/6 = 0.1(6)
1/7 = 0.(142857)
1/8 = 0.125
1/9 = 0.(1)
1/10 = 0.1

Où 0.1(6) signifie 0.166666..., et a un cycle de 1 chiffre. On peut voir que 1/7 a un cycle de 6 chiffres.

**Objectif** : Trouver la valeur de d < 1000 pour laquelle 1/d contient le cycle récurrent le plus long dans sa partie décimale.

---

## Exemples
### Exemple 1
**Entrée** : Limite = 10
**Sortie** : `7`
**Explication** : 1/7 a le cycle récurrent le plus long (6 chiffres) parmi les dénominateurs < 10.

---

## Approche de Résolution

### Stratégie
1. Pour chaque dénominateur d de 1 à 999, calculer la longueur du cycle récurrent de 1/d.
2. Garder trace du dénominateur avec le cycle le plus long.
3. Retourner ce dénominateur.

### Observations
- Pour calculer la longueur du cycle, utiliser la division longue.
- On garde trace des restes rencontrés lors de la division.
- Quand on rencontre un reste déjà vu, on a trouvé le cycle.
- La longueur du cycle est la différence entre les positions où le même reste apparaît.

### Complexité
- **Temporelle** : O(n × d), où `n` = 1000 et `d` est la longueur moyenne du cycle.
- **Spatiale** : O(d), pour stocker les restes rencontrés.

---

## Solution en C++

```cpp
#include <iostream>
#include <unordered_map>

int getCycleLength(int n) {
    std::unordered_map<int, int> remainderPos;
    int remainder = 1;
    int position = 0;
    
    while (remainder != 0) {
        if (remainderPos.find(remainder) != remainderPos.end()) {
            return position - remainderPos[remainder];
        }
        
        remainderPos[remainder] = position;
        remainder = (remainder * 10) % n;
        position++;
    }
    
    return 0;  // Pas de cycle (division exacte)
}

int main() {
    int bestNumber = 0;
    int bestLength = 0;
    
    for (int i = 1; i < 1000; ++i) {
        int len = getCycleLength(i);
        if (len > bestLength) {
            bestNumber = i;
            bestLength = len;
        }
    }
    
    std::cout << bestNumber << std::endl;
    return 0;
}
```
