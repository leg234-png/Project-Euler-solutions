# Counting Capacitor Circuits

## Description du Problème
En utilisant jusqu'à 18 condensateurs identiques de 60 µF chacun, on peut construire des circuits en série et en parallèle.

**Objectif** : Combien de valeurs de capacité distinctes peuvent être obtenues en utilisant exactement 18 condensateurs de 60 µF ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 1`
**Sortie** : `1`
**Explication** : Avec un seul condensateur, on obtient uniquement 60 µF.

### Exemple 2
**Entrée** : `n = 2`
**Sortie** : `2`
**Explication** : En série : 1/(1/60 + 1/60) = 30 µF. En parallèle : 60 + 60 = 120 µF.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique pour construire tous les circuits possibles.
2. `possible[i]` : ensemble de toutes les valeurs de capacité possibles avec exactement `i` condensateurs.
3. Pour `i = 1` : `possible[1] = {60}`.
4. Pour `i >= 2` :
   - Pour chaque partition `j` et `i-j` (où `1 <= j <= i-j`) :
     - Pour chaque valeur `a` dans `possible[j]` et `b` dans `possible[i-j]` :
       - Ajouter `a + b` (parallèle).
       - Ajouter `1/(1/a + 1/b)` (série).
5. Utiliser une classe `FastFraction` pour gérer les fractions avec précision.
6. Compter toutes les valeurs distinctes dans l'union de tous les `possible[i]`.

### Observations
- Les condensateurs peuvent être connectés en série ou en parallèle.
- La formule pour les condensateurs en série : `C_eq = 1/(1/C1 + 1/C2)`.
- La formule pour les condensateurs en parallèle : `C_eq = C1 + C2`.
- On doit utiliser des fractions pour éviter les erreurs d'arrondi.

### Complexité
- **Temporelle** : O(2^n × n²) dans le pire cas, où n est le nombre de condensateurs.
- **Spatiale** : O(2^n) pour stocker toutes les valeurs possibles.

---

## Solution en C++

```cpp
#include <iostream>
#include <set>
#include <vector>
#include <algorithm>
#include <numeric>
#include <cmath>

using namespace std;

struct Fraction {
    long long num, den;
    
    Fraction(long long n = 0, long long d = 1) {
        long long g = __gcd(n, d);
        num = n / g;
        den = d / g;
        if (den < 0) {
            num = -num;
            den = -den;
        }
    }
    
    bool operator<(const Fraction& other) const {
        return num * other.den < other.num * den;
    }
    
    bool operator==(const Fraction& other) const {
        return num == other.num && den == other.den;
    }
    
    Fraction operator+(const Fraction& other) const {
        return Fraction(num * other.den + other.num * den, den * other.den);
    }
    
    Fraction reciprocalAdd(const Fraction& other) const {
        // Returns 1/(1/this + 1/other) = (this * other) / (this + other)
        return Fraction(num * other.num, num * other.den + other.num * den);
    }
};

const int SIZE = 18;

int main() {
    vector<set<Fraction>> possible(SIZE + 1);
    set<Fraction> all;
    
    possible[0] = set<Fraction>();
    possible[1].insert(Fraction(60, 1));
    all.insert(Fraction(60, 1));
    
    for (int i = 2; i <= SIZE; i++) {
        set<Fraction> poss;
        for (int j = 1; j <= i - j; j++) {
            for (const Fraction& a : possible[j]) {
                for (const Fraction& b : possible[i - j]) {
                    poss.insert(a + b);  // Parallel
                    poss.insert(a.reciprocalAdd(b));  // Series
                }
            }
        }
        possible[i] = poss;
        all.insert(poss.begin(), poss.end());
    }
    
    cout << all.size() << endl;
    return 0;
}
```
