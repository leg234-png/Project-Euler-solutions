# Hautes puissances de nombres irrationnels

## Description du Problème
Soit f(a, n) = ⌊(⌈√a⌉ + √a)^n⌋.

Par exemple, f(5, 10) = 14226.

Soit G(n) = ∑_{a=1}^n f(a, a²).

G(1000) mod 999999937 = 163861845.

**Objectif** : Trouver G(5000000). Donner votre réponse modulo 999999937.

---

## Exemples
### Exemple 1
**Entrée** : `a = 5, n = 10`
**Sortie** : `14226`
**Explication** : f(5, 10) = 14226.

---

## Approche de Résolution

### Stratégie
1. Soit x = ⌈√a⌉, alors (x + √a)^n = a(n) + b(n)√a.
2. (x + √a)^0 = 1 => a(0) = 1, b(0) = 0.
3. (x + √a)^1 = x + √a => a(1) = x, b(1) = 1.
4. (x + √a)^(n+1) = (a(n) + b(n)√a)(x + √a) = (xa(n) + ab(n)) + (a(n) + xb(n))√a.
5. => a(n+1) = xa(n) + ab(n), b(n+1) = a(n) + xb(n).
6. f(a, n) = a(n) - 1 (car ⌊(x + √a)^n⌋ = a(n) - 1 quand n est pair, ou similaire).
7. Utiliser l'exponentiation binaire pour calculer efficacement a(n) et b(n).

### Observations
- On peut calculer (x + √a)^n en utilisant la récurrence.
- L'exponentiation binaire permet de calculer rapidement.
- On doit gérer l'arithmétique modulaire correctement.

### Complexité
- **Temporelle** : O(n × log a²) pour calculer tous les f(a, a²).
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long mod = 999999937;

pair<long long, long long> bin_exp(long long x, long long a, long long n, long long mod) {
    long long a_n = 1, b_n = 0;
    long long a_sq = x, b_sq = 1;
    
    string bits = "";
    long long temp = n;
    while (temp > 0) {
        bits = to_string(temp % 2) + bits;
        temp /= 2;
    }
    
    for (size_t i = 1; i < bits.length(); i++) {
        long long new_a_sq = (a_sq * a_sq + a * b_sq * b_sq) % mod;
        long long new_b_sq = (2 * a_sq * b_sq) % mod;
        a_sq = new_a_sq;
        b_sq = new_b_sq;
        
        if (bits[i] == '1') {
            long long new_a_n = (a_n * a_sq + a * b_n * b_sq) % mod;
            long long new_b_n = (a_n * b_sq + b_n * a_sq) % mod;
            a_n = new_a_n;
            b_n = new_b_n;
        }
    }
    
    return {a_n, b_n};
}

long long f(long long a, long long n, long long mod) {
    long long x = (long long)ceil(sqrt(a));
    auto [a_n, b_n] = bin_exp(x, a, n, mod);
    return (a_n - 1 + mod) % mod;
}

long long compute(long long n, long long mod) {
    long long total = 0;
    for (long long a = 1; a <= n; a++) {
        total = (total + f(a, a * a, mod)) % mod;
    }
    return total;
}

int main() {
    cout << compute(5000000, mod) << endl;
    return 0;
}
```
