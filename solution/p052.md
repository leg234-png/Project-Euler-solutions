# Multiples avec les Mêmes Chiffres

## Description du Problème
On peut voir que le nombre 125874 et son double 251748 contiennent exactement les mêmes chiffres, mais dans un ordre différent.

**Objectif** : Trouver le plus petit entier positif x tel que 2x, 3x, 4x, 5x et 6x contiennent les mêmes chiffres.

---

## Exemples
### Exemple 1
**Entrée** : x = 125874
**Sortie** : `125874`
**Explication** : 125874 et 251748 (2×125874) contiennent les mêmes chiffres.

---

## Approche de Résolution

### Stratégie
1. Parcourir les entiers positifs à partir de 1.
2. Pour chaque nombre x :
   - Calculer les multiples 2x, 3x, 4x, 5x, 6x.
   - Convertir chaque nombre en chaîne de caractères triée.
   - Vérifier si toutes les chaînes triées sont identiques.
   - Si oui, retourner x.
3. Retourner le résultat.

### Observations
- On peut comparer les chiffres en triant les chaînes de caractères.
- Si x a n chiffres, alors 6x doit avoir au plus n+1 chiffres.
- On peut arrêter la recherche si x dépasse Integer.MAX_VALUE / 6.

### Complexité
- **Temporelle** : O(n × log n), où n est le nombre à tester (tri des chiffres).
- **Spatiale** : O(log n), pour stocker les chaînes de caractères.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>

std::string toSortedDigits(int x) {
    std::string s = std::to_string(x);
    std::sort(s.begin(), s.end());
    return s;
}

bool multiplesHaveSameDigits(int x) {
    std::string base = toSortedDigits(x);
    for (int i = 2; i <= 6; ++i) {
        if (toSortedDigits(i * x) != base) {
            return false;
        }
    }
    return true;
}

int main() {
    for (int i = 1; ; ++i) {
        if (i > INT_MAX / 6) {
            std::cerr << "Overflow" << std::endl;
            return 1;
        }
        if (multiplesHaveSameDigits(i)) {
            std::cout << i << std::endl;
            return 0;
        }
    }
}
```
