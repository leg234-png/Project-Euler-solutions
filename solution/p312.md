# Chemins cycliques sur les graphes de Sierpiński

## Description du Problème
On peut aussi vérifier que :
C(1)=C(2)=1
C(5)=71328803586048
C(10000) mod 10⁸ = 37652224
C(10000) mod 13⁸ = 617720485

**Objectif** : Trouver C(C(C(10000))) mod 13⁸.

---

## Exemples
### Exemple 1
**Entrée** : `n = 5`
**Sortie** : `71328803586048`
**Explication** : C(5) = 71328803586048.

---

## Approche de Résolution

### Stratégie
1. Utiliser la formule explicite : C(n) = 8 × 12^((3^(n-2) - 3)/2) pour n > 4.
2. Observer les cycles :
   - C(n) mod 13 = C(n + 2) mod 13
   - C(n) mod 13² = C(n + 6) mod 13²
   - C(n) mod 13³ = C(n + 6×13) mod 13³
   - C(n) mod 13⁴ = C(n + 6×13²) mod 13⁴
   - etc.
3. Utiliser ces cycles pour réduire les calculs :
   - C(C(C(10000))) mod 13⁸ = C(C(C(10000) mod 6×13⁶)) mod 13⁸
   - C(C(10000)) mod 6×13⁶ = C(C(10000) mod 6×13⁴) mod 6×13⁶
   - C(10000) mod 6×13⁴ = C(10000 mod 6×13²) mod 6×13⁴
4. Calculer récursivement en utilisant les cycles.

### Observations
- Il y a une périodicité dans les valeurs modulo les puissances de 13.
- On peut utiliser l'exponentiation modulaire rapide pour calculer 12^exposant mod mod.
- Les cycles permettent de réduire considérablement les calculs.

### Complexité
- **Temporelle** : O(log n) pour chaque calcul de C(n) mod m.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long C(long long n, long long mod) {
    if (n == 1 || n == 2) {
        return 1;
    }
    if (n == 3) {
        return 8;
    }
    if (n == 4) {
        return 8 * 12 % mod;
    }
    
    long long exp = (mod_pow(3, n - 2, mod * 2) - 3) / 2;
    return 8 * mod_pow(12, exp, mod) % mod;
}

long long compute() {
    long long mod = 1;
    for (int i = 0; i < 8; i++) {
        mod *= 13;
    }
    
    // Calculer C(10000) mod 6*13^2
    long long mod1 = 6 * 13 * 13;
    long long c1 = C(10000, mod1);
    
    // Calculer C(c1) mod 6*13^4
    long long mod2 = 6 * 13 * 13 * 13 * 13;
    long long c2 = C(c1, mod2);
    
    // Calculer C(c2) mod 13^8
    long long c3 = C(c2, mod);
    
    return c3;
}

int main() {
    cout << compute() << endl;
    return 0;
}
```
