# Coureur et nageur

## Description du Problème
Un coureur et un nageur jouent dans une piscine en forme d'hexagone régulier. Le nageur commence au centre de la piscine, tandis que le coureur commence au milieu d'un des bords de la piscine.

En supposant une stratégie optimale des deux joueurs, on peut montrer que le nageur peut toujours gagner en s'échappant de la piscine à un certain point au bord avant que le coureur n'y arrive, si v est inférieur à la vitesse critique V_Hexagone, et ne peut jamais gagner si v > V_Hexagone.

**Objectif** : Trouver V_Hexagone. Donner votre réponse arrondie à 8 places après la virgule décimale.

---

## Exemples
### Exemple 1
**Entrée** : `Piscine hexagonale`
**Sortie** : `V_Hexagone`
**Explication** : La vitesse critique pour une piscine hexagonale.

---

## Approche de Résolution

### Stratégie
1. Pour une piscine hexagonale de côté 1, le nageur est au centre et le coureur au milieu d'un bord.
2. Utiliser la géométrie pour calculer la distance minimale que le nageur doit parcourir pour atteindre le bord.
3. Calculer la distance que le coureur doit parcourir pour atteindre le même point.
4. La vitesse critique est le rapport de ces distances.
5. Utiliser le calcul pour trouver l'angle optimal.

### Observations
- Le problème nécessite des calculs géométriques précis.
- On peut utiliser le calcul pour trouver l'optimum.
- La solution implique des fonctions trigonométriques.

### Complexité
- **Temporelle** : O(1) pour le calcul analytique.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <cmath>

using namespace std;

double compute(int n) {
    double K = 0;
    double theta = M_PI / n;
    
    auto f = [&](double k) {
        return sin(k * theta) - (k + n) * tan(theta) * cos(k * theta);
    };
    
    while (f(K) < 0) {
        K += 1;
    }
    K -= 1;
    
    double a = (K * theta + acos(2 * sin(K * theta) / ((K + n) * tan(theta)) - cos(K * theta))) / 2;
    return round(1.0 / cos(a) * 1e8) / 1e8;
}

int main() {
    cout << fixed << setprecision(8) << compute(6) << endl;
    return 0;
}
```
