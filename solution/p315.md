# Digital root clocks

## Description du Problème
Sam et Max construisent deux types d'horloges numériques différentes. Les deux affichent un nombre en utilisant des segments de type affichage à 7 segments.

Sam construit un type d'horloge qui affiche simplement le nombre, puis calcule la somme des chiffres, puis affiche ce résultat, et ainsi de suite jusqu'à ce qu'un nombre à un chiffre soit atteint.

Max construit un type d'horloge qui affiche le nombre, puis calcule la somme des chiffres, mais n'affiche que les segments qui changent entre les deux nombres.

**Objectif** : Pour tous les nombres premiers entre 10^7 et 2×10^7, trouver la différence entre le nombre total de transitions de Sam et le nombre total de transitions de Max.

---

## Exemples
### Exemple 1
**Entrée** : `n = 137`
**Sortie** : Différence entre les transitions de Sam et Max
**Explication** : Sam affiche 137 → 11 → 2. Max n'affiche que les segments qui changent.

---

## Approche de Résolution

### Stratégie
1. Convertir chaque nombre en représentation de segments (7 bits par chiffre).
2. Pour Sam : compter tous les segments allumés à chaque étape (2 × nombre de segments allumés).
3. Pour Max : compter seulement les segments qui changent entre deux nombres consécutifs (XOR des représentations).
4. Calculer la différence pour chaque nombre premier et additionner.

### Observations
- Chaque chiffre décimal correspond à un pattern de 7 segments.
- La somme des chiffres (digital root) est calculée itérativement jusqu'à obtenir un nombre à un chiffre.
- On utilise un XOR pour trouver les segments qui changent.

### Complexité
- **Temporelle : O(π(2×10^7) × log n)`.
- **Spatiale : O(1)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

const int DECIMAL_DIGIT_TO_SEGMENT[] = {0x77, 0x12, 0x5D, 0x5B, 0x3A, 0x6B, 0x6F, 0x72, 0x7F, 0x7B};

long long numberToSegments(int n) {
    long long result = 0;
    int i = 0;
    do {
        result |= (long long)DECIMAL_DIGIT_TO_SEGMENT[n % 10] << (i * 7);
        n /= 10;
        i++;
    } while (n != 0);
    return result;
}

int digitSum(int n) {
    int result = 0;
    while (n != 0) {
        result += n % 10;
        n /= 10;
    }
    return result;
}

int samTransitionsMinusMaxTransitions(int n, const vector<bool>& isPrime) {
    int samTrans = 0;
    int maxTrans = 0;
    long long segmentState = 0;
    
    while (true) {
        long long newState = numberToSegments(n);
        if (newState == segmentState)
            break;
        maxTrans += __builtin_popcountll(newState ^ segmentState);
        segmentState = newState;
        samTrans += 2 * __builtin_popcountll(newState);
        n = digitSum(n);
    }
    maxTrans += __builtin_popcountll(segmentState);
    return samTrans - maxTrans;
}

int main() {
    vector<bool> isPrime = listPrimality(20000000);
    int sum = 0;
    for (int i = 10000000; i < 20000000; i++) {
        if (isPrime[i])
            sum += samTransitionsMinusMaxTransitions(i, isPrime);
    }
    cout << sum << endl;
    return 0;
}
```
