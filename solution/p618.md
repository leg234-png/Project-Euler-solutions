# Nombres avec une somme de facteurs premiers donnée

## Description du Problème
Soit S(k) la somme de tous les nombres n où la somme des facteurs premiers (avec multiplicité) de n est k.

Par exemple, S(8) = 15+16+18 = 49.

Autres exemples : S(1) = 0, S(2) = 2, S(3) = 3, S(5) = 5+6 = 11.

La séquence de Fibonacci est F₁ = 1, F₂ = 1, F₃ = 2, F₄ = 3, F₅ = 5, ....

**Objectif** : Trouver ∑_{k=2}^{24} S(Fₖ). Donner votre réponse modulo 1000000000.

---

## Exemples
### Exemple 1
**Entrée** : `k = 8`
**Sortie** : `49`
**Explication** : S(8) = 15+16+18 = 49.

---

## Approche de Résolution

### Stratégie
1. Soit d(k, p) = somme des nombres tels que la somme de leur factorisation en nombres premiers est k et il n'y a pas de facteur premier plus grand que p.
2. Observer que d(k, 2) = 2 × d(k-2, 2), d(k, 3) = 3 × d(k-3, 3).
3. Relation générale : d(k, p) = p × d(k-p, p) quand p est premier.
4. Utiliser la programmation dynamique pour calculer d(k, p).
5. S(k) = somme de tous les d(k, p) pour tous les nombres premiers p.

### Observations
- On n'a besoin de traiter que les nombres premiers p.
- La relation de récurrence permet de calculer efficacement d(k, p).
- On doit générer tous les nombres premiers jusqu'à une certaine limite.

### Complexité
- **Temporelle** : O(K × P) où K est la valeur maximale de k et P est le nombre de nombres premiers.
- **Spatiale** : O(K × P) pour la table DP.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<int> prime_sieve(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

long long S(int k, const vector<int>& primes, long long mod) {
    vector<long long> d(k + 1, 0);
    d[0] = 1;
    
    for (int p : primes) {
        if (p > k) break;
        for (int i = k; i >= p; i--) {
            d[i] = (d[i] + p * d[i - p]) % mod;
        }
    }
    
    return d[k];
}

long long fibonacci(int n) {
    if (n <= 2) return 1;
    long long a = 1, b = 1;
    for (int i = 3; i <= n; i++) {
        long long temp = a + b;
        a = b;
        b = temp;
    }
    return b;
}

long long compute(long long mod) {
    vector<int> primes = prime_sieve(46368); // 24ème nombre de Fibonacci
    long long total = 0;
    
    for (int k = 2; k <= 24; k++) {
        long long F_k = fibonacci(k);
        total = (total + S(F_k, primes, mod)) % mod;
    }
    
    return total;
}

int main() {
    long long mod = 1000000000;
    cout << compute(mod) << endl;
    return 0;
}
```
