# Somme de chemin : deux directions

## Description du Problème
Dans la matrice 5 par 5 ci-dessous, la somme minimale de chemin du coin supérieur gauche au coin inférieur droit, en se déplaçant uniquement vers la droite et vers le bas, est indiquée en gras rouge et est égale à 2427.

```
131  673  234  103  18
201  96   342  965  150
630  803  746  422  111
537  699  497  121  956
805  732  524  37   331
```

**Objectif** : Trouver la somme minimale de chemin du coin supérieur gauche au coin inférieur droit en se déplaçant uniquement vers la droite et vers le bas dans matrix.txt, un fichier texte de 31K contenant une matrice de 80 par 80.

---

## Exemples
### Exemple 1
**Entrée** : Matrice 5×5
**Sortie** : `2427`
**Explication** : Le chemin optimal est 131 → 201 → 96 → 342 → 746 → 422 → 121 → 37 → 331 = 2427.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique en partant du coin inférieur droit.
2. Pour chaque cellule, calculer la somme minimale en ajoutant la valeur de la cellule au minimum de ses deux voisins (droite et bas).
3. Modifier la matrice en place de bas en haut et de droite à gauche.
4. La valeur à la position (0,0) sera la somme minimale.

### Observations
- On peut se déplacer uniquement vers la droite ou vers le bas.
- Pour chaque cellule, on choisit le chemin avec la somme minimale parmi les deux options possibles.
- En calculant de bas en haut et de droite à gauche, les valeurs dépendantes sont toujours calculées avant d'être utilisées.

### Complexité
- **Temporelle** : O(n²), où n est la taille de la matrice (80).
- **Spatiale** : O(n²), pour stocker la matrice.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <fstream>
#include <sstream>
#include <algorithm>

using namespace std;

int main() {
    // Lire la matrice depuis le fichier
    vector<vector<int>> grid;
    ifstream file("matrix.txt");
    string line;
    
    while (getline(file, line)) {
        vector<int> row;
        istringstream iss(line);
        string num;
        while (getline(iss, num, ',')) {
            row.push_back(stoi(num));
        }
        grid.push_back(row);
    }
    file.close();
    
    // Programmation dynamique
    for (int i = grid.size() - 1; i >= 0; i--) {
        for (int j = grid[i].size() - 1; j >= 0; j--) {
            int temp;
            if (i + 1 < (int)grid.size() && j + 1 < (int)grid[i].size())
                temp = min(grid[i + 1][j], grid[i][j + 1]);
            else if (i + 1 < (int)grid.size())
                temp = grid[i + 1][j];
            else if (j + 1 < (int)grid[i].size())
                temp = grid[i][j + 1];
            else
                temp = 0;
            grid[i][j] += temp;
        }
    }
    
    cout << grid[0][0] << endl;
    return 0;
}
```
