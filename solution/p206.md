# Concealed Square

## Description du Problème
Trouver le seul entier positif dont le carré a la forme `1_2_3_4_5_6_7_8_9_0`, où chaque `_` est un chiffre unique (peut être le même).

---

## Exemples
### Exemple 1
**Entrée** : Pattern `1_2_3_4_5_6_7_8_9_0`
**Sortie** : Le nombre dont le carré correspond au pattern
**Explication** : Le carré doit avoir 19 chiffres et correspondre au pattern donné.

---

## Approche de Résolution

### Stratégie
1. Le pattern est `1_2_3_4_5_6_7_8_9_0`, donc le carré se termine par 0, ce qui implique que le nombre se termine par 0.
2. Le carré a 19 chiffres, donc le nombre est d'environ 10^9.
3. Optimisation : faire l'arithmétique en base 10 directement pour éviter les divisions/modulo coûteux.
4. Commencer à n = 10^9 (car le pattern est > 10^18).
5. Incrémenter n par 10 (car n doit se terminer par 0).
6. Pour chaque n, calculer n² et vérifier si les chiffres aux positions appropriées correspondent au pattern.
7. Utiliser des tableaux de chiffres pour représenter n et n² en base 10.

### Observations
- Le carré se termine par 0, donc n se termine par 0.
- On peut optimiser en calculant (n+10)² = n² + 20n + 100 directement sur les tableaux de chiffres.
- Les positions à vérifier sont : 18, 16, 14, 12, 10, 8, 6, 4, 2, 0.

### Complexité
- **Temporelle : O(√N)**, où N est la limite supérieure.
- **Spatiale : O(1)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

bool isConcealedSquare(const vector<int>& n) {
    // Check pattern: 1_2_3_4_5_6_7_8_9_0
    // Positions: 18, 16, 14, 12, 10, 8, 6, 4, 2, 0
    for (int i = 1; i <= 9; i++) {
        if (n[20 - i * 2] != i)
            return false;
    }
    return n[0] == 0;  // Last digit must be 0
}

void add10Pow(vector<int>& n, int i) {
    while (n[i] == 9) {
        n[i] = 0;
        i++;
    }
    n[i]++;
}

void add20n(const vector<int>& n, vector<int>& n2) {
    int carry = 0;
    size_t i;
    for (i = 0; i < n.size(); i++) {
        int sum = n[i] * 2 + n2[i + 1] + carry;
        n2[i + 1] = sum % 10;
        carry = sum / 10;
    }
    for (i++; carry > 0; i++) {
        int sum = n2[i] + carry;
        n2[i] = sum % 10;
        carry = sum / 10;
    }
}

int main() {
    long long n = 1000000000;
    vector<int> ndigits(10, 0);
    vector<int> n2digits(19, 0);
    
    long long temp = n;
    for (int i = 0; i < 10; i++, temp /= 10)
        ndigits[i] = temp % 10;
    
    temp = n * n;
    for (int i = 0; i < 19; i++, temp /= 10)
        n2digits[i] = temp % 10;
    
    while (!isConcealedSquare(n2digits)) {
        add20n(ndigits, n2digits);
        add10Pow(n2digits, 2);
        
        n += 10;
        add10Pow(ndigits, 1);
    }
    
    cout << n << endl;
    return 0;
}
```
