# Somme des Chiffres d'une Puissance

## Description du Problème
2¹⁵ = 32768 et la somme de ses chiffres est 3 + 2 + 7 + 6 + 8 = 26.

**Objectif** : Quelle est la somme des chiffres du nombre 2¹⁰⁰⁰ ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 15`
**Sortie** : `26`
**Explication** : 2¹⁵ = 32768, la somme des chiffres est 3+2+7+6+8 = 26.

---

## Approche de Résolution

### Stratégie
1. Calculer 2¹⁰⁰⁰ (un nombre très grand).
2. Convertir le résultat en chaîne de caractères.
3. Parcourir chaque chiffre et les additionner.
4. Retourner la somme.

### Observations
- 2¹⁰⁰⁰ est un nombre très grand qui ne tient pas dans les types entiers standards.
- On doit utiliser une bibliothèque pour les grands nombres (comme `BigInteger` en Java ou `std::string` en C++).
- En C++, on peut utiliser des chaînes de caractères pour représenter les grands nombres et implémenter la multiplication manuellement.

### Complexité
- **Temporelle** : O(n × m), où `n` = 1000 (exposant) et `m` est le nombre de chiffres du résultat.
- **Spatiale** : O(m), pour stocker le nombre.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>

std::string multiplyBy2(const std::string& num) {
    std::string result;
    int carry = 0;
    
    for (int i = num.length() - 1; i >= 0; --i) {
        int digit = (num[i] - '0') * 2 + carry;
        result.push_back((digit % 10) + '0');
        carry = digit / 10;
    }
    
    if (carry > 0) {
        result.push_back(carry + '0');
    }
    
    std::reverse(result.begin(), result.end());
    return result;
}

int main() {
    std::string power = "1";
    
    for (int i = 0; i < 1000; ++i) {
        power = multiplyBy2(power);
    }
    
    int sum = 0;
    for (char c : power) {
        sum += c - '0';
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
