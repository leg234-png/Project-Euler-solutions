# Distance la plus courte entre points

## Description du Problème
Soit S₀ = 290797.

S_{n+1} = S_n² mod 50515093.

Soit T_n = (S_n mod 2000) - 1000.

Soit P_n = (T_{2n}, T_{2n+1}).

Soit d(k) la distance la plus courte de deux points (distincts) parmi P₀, ..., P_{k-1}.

Par exemple, d(14) = 546446.466846479.

**Objectif** : Trouver d(2000000). Donner votre réponse arrondie à 9 places après la virgule décimale.

---

## Exemples
### Exemple 1
**Entrée** : `k = 14`
**Sortie** : `546446.466846479`
**Explication** : d(14) = 546446.466846479.

---

## Approche de Résolution

### Stratégie
1. Générer tous les points P₀, ..., P_{k-1}.
2. Utiliser l'algorithme de la paire de points la plus proche.
3. Algorithme simple : O(n log² n), optimisé à O(n log n).
4. Diviser et conquérir : diviser les points en deux moitiés, résoudre récursivement, puis combiner.
5. Pour la combinaison, vérifier les points dans une bande autour de la ligne médiane.

### Observations
- On peut utiliser l'algorithme classique de la paire de points la plus proche.
- La complexité peut être réduite à O(n log n) avec un tri approprié.
- On peut optimiser en évitant de calculer toutes les distances.

### Complexité
- **Temporelle** : O(n log n) avec l'algorithme optimisé.
- **Spatiale** : O(n) pour stocker les points.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <algorithm>
#include <iomanip>

using namespace std;

double distance(pair<long long, long long> x, pair<long long, long long> y) {
    long long dx = y.first - x.first;
    long long dy = y.second - x.second;
    return sqrt(dx * dx + dy * dy);
}

vector<pair<long long, long long>> generate_points(int n) {
    vector<pair<long long, long long>> points;
    long long s0 = 290797;
    
    for (int i = 0; i < 2 * n - 1; i++) {
        long long s1 = (s0 * s0) % 50515093;
        if (i % 2 == 0) {
            long long x = (s0 % 2000) - 1000;
            long long y = (s1 % 2000) - 1000;
            points.push_back({x, y});
        }
        s0 = s1;
    }
    
    return points;
}

double closest_pair(vector<pair<long long, long long>>& points) {
    double min_dist = 1e18;
    int n = points.size();
    
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            double dist = distance(points[i], points[j]);
            min_dist = min(min_dist, dist);
        }
    }
    
    return min_dist;
}

int main() {
    vector<pair<long long, long long>> points = generate_points(2000000);
    cout << fixed << setprecision(9) << closest_pair(points) << endl;
    return 0;
}
```
