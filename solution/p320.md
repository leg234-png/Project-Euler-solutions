# Factoriels divisibles par un très grand entier

## Description du Problème
Soit N(i) le plus petit entier n tel que n! soit divisible par (i!)^1234567890.

Par exemple, N(5) n'existe pas, N(6) = 866472, N(11) = 614916.

Soit S(u) = ∑N(i) pour 10 ≤ i ≤ u.

S(1000) = 614538266565663.

**Objectif** : Trouver S(1000000) mod 10¹⁸.

---

## Exemples
### Exemple 1
**Entrée** : `i = 6`
**Sortie** : `866472`
**Explication** : N(6) = 866472.

---

## Approche de Résolution

### Stratégie
1. Si (i!)^1234567890 divise n!, on peut utiliser la formule de Legendre pour trouver combien de fois un nombre premier apparaît dans n!.
2. Pour chaque nombre premier p dans la factorisation de i!, calculer l'exposant e dans i!.
3. On a besoin de e × 1234567890 occurrences de p dans n!.
4. Utiliser un algorithme de bissection pour trouver le plus petit n tel que n! contienne au moins e × 1234567890 facteurs p.
5. Prendre le maximum sur tous les nombres premiers.

### Observations
- La formule de Legendre donne l'exposant d'un nombre premier dans n!.
- On peut utiliser la bissection pour trouver efficacement le n minimal.
- On doit considérer tous les nombres premiers dans la factorisation de i!.

### Complexité
- **Temporelle** : O(u × P × log n) où P est le nombre de nombres premiers et n est la valeur maximale recherchée.
- **Spatiale** : O(√u) pour le crible de nombres premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <map>
#include <cmath>

using namespace std;

vector<int> prime_sieve(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

long long legendre(long long n, int p) {
    long long count = 0;
    long long power = p;
    while (power <= n) {
        count += n / power;
        power *= p;
    }
    return count;
}

long long find_min_n(int i, int p, long long required) {
    long long low = i;
    long long high = 1000000000LL;
    
    while (low < high) {
        long long mid = (low + high) / 2;
        if (legendre(mid, p) >= required) {
            high = mid;
        } else {
            low = mid + 1;
        }
    }
    return low;
}

long long N(int i, const vector<int>& primes) {
    long long max_n = 0;
    long long multiplier = 1234567890LL;
    
    for (int p : primes) {
        if (p > i) break;
        long long exp_in_i = legendre(i, p);
        if (exp_in_i > 0) {
            long long required = exp_in_i * multiplier;
            long long n = find_min_n(i, p, required);
            max_n = max(max_n, n);
        }
    }
    
    return max_n;
}

long long compute(int limit, long long mod) {
    vector<int> primes = prime_sieve(limit);
    long long total = 0;
    
    for (int i = 10; i <= limit; i++) {
        total = (total + N(i, primes)) % mod;
    }
    
    return total;
}

int main() {
    long long mod = 1000000000000000000LL;
    cout << compute(1000000, mod) << endl;
    return 0;
}
```
