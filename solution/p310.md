# Nim carré

## Description du Problème
Le nombre de pierres dans les trois tas est représenté par le triplet ordonné (a,b,c).

Si 0≤a≤b≤c≤29, alors le nombre de positions perdantes pour le prochain joueur est 1160.

**Objectif** : Trouver le nombre de positions perdantes pour le prochain joueur si 0≤a≤b≤c≤100000.

---

## Exemples
### Exemple 1
**Entrée** : `0≤a≤b≤c≤29`
**Sortie** : `1160`
**Explication** : Il y a 1160 positions perdantes.

---

## Approche de Résolution

### Stratégie
1. Utiliser le théorème de Sprague-Grundy pour calculer les valeurs Grundy.
2. Calculer la valeur Sprague-Grundy pour un seul tas.
3. XOR les 3 tas : si la valeur est 0, alors la position est perdante pour le prochain joueur.
4. On veut s_a ^ s_b ^ s_c = 0 <=> s_a = s_b ^ s_c.
5. Pour chaque valeur Grundy possible, compter les triplets (a,b,c) qui satisfont cette condition.

### Observations
- Un joueur peut retirer i² pierres où i est un entier positif.
- La valeur Grundy pour un tas de x pierres est le mex des valeurs Grundy des positions accessibles.
- On peut utiliser la mémorisation pour accélérer le calcul.

### Complexité
- **Temporelle** : O(limit^(3/2)) pour calculer toutes les valeurs Grundy, puis O(G²) où G est le nombre de valeurs Grundy distinctes.
- **Spatiale** : O(limit) pour stocker les valeurs Grundy.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <unordered_map>
#include <cmath>

using namespace std;

int mex(const vector<int>& vec) {
    for (int i = 0; i < vec.size(); i++) {
        if (vec[i] == 0) {
            return i;
        }
    }
    return vec.size();
}

vector<int> compute_sG(int limit) {
    vector<int> sG_cache(limit + 1, -1);
    
    function<int(int)> sG = [&](int x) -> int {
        if (x == 0) {
            sG_cache[x] = 0;
            return 0;
        }
        
        if (sG_cache[x] != -1) {
            return sG_cache[x];
        }
        
        vector<int> vec(100, 0);
        int sqrt_x = sqrt(x);
        for (int i = sqrt_x; i >= 1; i--) {
            int v = sG(x - i * i);
            if (v >= vec.size()) {
                vec.resize(v + 10, 0);
            }
            vec[v] = 1;
        }
        
        sG_cache[x] = mex(vec);
        return sG_cache[x];
    };
    
    for (int i = 0; i <= limit; i++) {
        sG(i);
    }
    
    return sG_cache;
}

long long compute(int limit) {
    vector<int> sG_values = compute_sG(limit);
    
    unordered_map<int, long long> count;
    for (int i = 0; i <= limit; i++) {
        count[sG_values[i]]++;
    }
    
    long long total = 0;
    for (auto& [g_a, cnt_a] : count) {
        for (auto& [g_b, cnt_b] : count) {
            int g_c = g_a ^ g_b;
            if (count.find(g_c) != count.end() && g_c >= g_b) {
                if (g_a == g_b && g_b == g_c) {
                    // Cas où a = b = c
                    total += cnt_a * (cnt_a + 1) * (cnt_a + 2) / 6;
                } else if (g_a == g_b) {
                    total += cnt_a * (cnt_a + 1) / 2 * count[g_c];
                } else if (g_b == g_c) {
                    total += cnt_a * cnt_b * (cnt_b + 1) / 2;
                } else {
                    total += cnt_a * cnt_b * count[g_c];
                }
            }
        }
    }
    
    return total;
}

int main() {
    cout << compute(100000) << endl;
    return 0;
}
```
