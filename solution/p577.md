# Comptage d'hexagones

## Description du Problème
Soit H(n) le nombre d'hexagones réguliers de longueur de côté entière qui peuvent être formés dans un triangle équilatéral de côté n.

Par exemple, H(3) = 1, H(6) = 12 et H(20) = 966.

**Objectif** : Trouver ∑_{n=3}^{12345} H(n).

---

## Exemples
### Exemple 1
**Entrée** : `n = 3`
**Sortie** : `1`
**Explication** : H(3) = 1.

### Exemple 2
**Entrée** : `n = 6`
**Sortie** : `12`
**Explication** : H(6) = 12.

---

## Approche de Résolution

### Stratégie
1. Observer les patterns : H(3) = T(1), H(4) = T(2), H(5) = T(3), H(6) = T(4) + 2T(1), H(7) = T(5) + 2T(2).
2. H(20) = T(18) + 2T(15) + 3T(12) + 4T(9) + 5T(6) + 6T(3).
3. En général, chaque fois que H(3n), on ajoute n nouveaux types d'hexagones.
4. H(n) = ∑_{y=1}^{n/3} y × T(n - 2 - 3(y-1)), où T(k) = k(k+1)/2 est le k-ième nombre triangulaire.

### Observations
- Il y a un pattern clair avec les nombres triangulaires.
- On peut calculer H(n) directement avec la formule.
- La somme peut être calculée efficacement.

### Complexité
- **Temporelle** : O(n²) pour calculer la somme sur tous les n.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

long long T(long long n) {
    return n * (n + 1) / 2;
}

long long H(int n) {
    long long total = 0;
    for (int y = 1; y <= n / 3; y++) {
        int arg = n - 2 - 3 * (y - 1);
        if (arg > 0) {
            total += (long long)y * T(arg);
        }
    }
    return total;
}

long long compute(int limit) {
    long long total = 0;
    for (int n = 3; n <= limit; n++) {
        total += H(n);
    }
    return total;
}

int main() {
    cout << compute(12345) << endl;
    return 0;
}
```
