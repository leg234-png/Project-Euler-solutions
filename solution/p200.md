# Squbes prime-proof contenant "200"

## Description du Problème
Nous définissons un sqube comme un nombre de la forme p²q³, où p et q sont des nombres premiers distincts. Par exemple, 200 = 5² × 2³ ou 120072949 = 23² × 61³.

Les cinq premiers squbes sont 72, 108, 200, 392 et 500.

Intéressamment, 200 est aussi le premier nombre pour lequel on ne peut changer aucun chiffre pour faire un nombre premier ; nous appellerons de tels nombres prime-proof. Le prochain sqube prime-proof qui contient la sous-chaîne contiguë "200" est 1992008.

**Objectif** : Trouver le 200ème sqube prime-proof contenant la sous-chaîne contiguë "200".

---

## Exemples
### Exemple 1
**Entrée** : `n = 1`
**Sortie** : `200`
**Explication** : 200 est le premier sqube prime-proof contenant "200".

### Exemple 2
**Entrée** : `n = 2`
**Sortie** : `1992008`
**Explication** : 1992008 est le deuxième sqube prime-proof contenant "200".

---

## Approche de Résolution

### Stratégie
1. Générer tous les squbes de la forme p²q³ où p et q sont des nombres premiers distincts.
2. Filtrer ceux qui contiennent la sous-chaîne "200".
3. Pour chaque candidat, vérifier s'il est prime-proof en testant tous les changements possibles d'un chiffre.
4. Utiliser un test de primalité rapide (comme Fermat) pour vérifier si un nombre modifié est premier.
5. Compter jusqu'au 200ème.

### Observations
- Un nombre est prime-proof si aucun changement d'un seul chiffre ne produit un nombre premier.
- On doit tester tous les chiffres à toutes les positions.
- Un test de primalité probabiliste peut être utilisé pour accélérer.

### Complexité
- **Temporelle** : O(N × d × log n) où N est le nombre de squbes, d est le nombre de chiffres, et log n est le coût du test de primalité.
- **Spatiale** : O(1) si on traite les nombres un par un.

#include <iostream>
#include <vector>
#include <string>
#include <cmath>
#include <algorithm>

using namespace std;

vector<bool> list_primality(long long n) {
    vector<bool> result(n + 1, true);
    result[0] = result[1] = false;
    for (long long i = 2; i * i <= n; i++) {
        if (result[i]) {
            for (long long j = 2 * i; j <= n; j += i) {
                result[j] = false;
            }
        }
    }
    return result;
}

bool fermat_primality_test(long long n) {
    if (n < 2) return false;
    if (pow(4, n - 1) % n == 1 && pow(6, n - 1) % n == 1) {
        return true;
    }
    return false;
}

bool prime_proof_checker(long long x) {

                return false;
            }
        }
        number[pos] = original;
    }
