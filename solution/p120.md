# Square Remainders

## Description du Problème
Soit r le reste lorsque (a-1)ⁿ + (a+1)ⁿ est divisé par a².

Par exemple, si a=7 et n=3, alors r=42 : 6³ + 8³ = 728 ≡ 42 mod 49. Et à mesure que n varie, r variera aussi, mais pour a=7, il s'avère que r_max=42.

**Objectif** : Pour 3 ≤ a ≤ 1000, trouver ∑r_max.

---

## Exemples
### Exemple 1
**Entrée** : `a = 7`
**Sortie** : `42`
**Explication** : r_max = 42 pour a = 7.

---

## Approche de Résolution

### Stratégie
1. Pour chaque a de 3 à 1000, trouver r_max.
2. Utiliser le théorème binomial pour simplifier (a+1)ⁿ + (a-1)ⁿ mod a².
3. Si n est pair : (a+1)ⁿ + (a-1)ⁿ ≡ 2 mod a².
4. Si n est impair : (a+1)ⁿ + (a-1)ⁿ ≡ 2an mod a².
5. Maximiser 2an mod a² équivaut à maximiser 2n mod a.
6. Si a est pair : maximum = a-2 avec n = a/2 - 1.
7. Si a est impair : maximum = a-1 avec n = (a-1)/2.

### Observations
- On peut dériver une formule directe sans tester tous les n.
- r_max = a(a-2) si a est pair, a(a-1) si a est impair.

### Complexité
- **Temporelle** : O(a), où a ≤ 1000.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>

int main() {
    int sum = 0;
    for (int a = 3; a <= 1000; a++) {
        if (a % 2 == 0) {
            sum += a * (a - 2);
        } else {
            sum += a * (a - 1);
        }
    }
    std::cout << sum << std::endl;
    return 0;
}
```
