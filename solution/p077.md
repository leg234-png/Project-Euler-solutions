# Sommations de nombres premiers

## Description du Problème
Il est possible d'écrire dix comme la somme de nombres premiers d'exactement cinq façons différentes :
7 + 3
5 + 5
5 + 3 + 2
3 + 3 + 2 + 2
2 + 2 + 2 + 2 + 2

**Objectif** : Quel est le premier nombre qui peut être écrit comme la somme de nombres premiers de plus de cinq mille façons différentes ?

---

## Exemples
### Exemple 1
**Entrée** : n = 10
**Sortie** : `5`
**Explication** : Il y a 5 façons d'écrire 10 comme somme de nombres premiers.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique pour calculer le nombre de façons de partitionner un nombre en sommes de nombres premiers.
2. Soit P(i, n) le nombre de façons que n peut être écrit comme une somme non ordonnée de nombres premiers où aucun nombre premier n'est plus grand que i.
3. P(i, 0) = 1 pour tout i.
4. Si i est premier et i ≤ n, alors P(i, n) = P(i-1, n) + P(i, n-i).
5. Sinon, P(i, n) = P(i-1, n).
6. Rechercher le plus petit n tel que P(n, n) > 5000.

### Observations
- On peut optimiser en utilisant seulement une dimension pour la programmation dynamique.
- On traite les nombres premiers dans l'ordre croissant.
- On utilise une recherche binaire ou une recherche linéaire avec des limites croissantes.

### Complexité
- **Temporelle** : O(n²), où n est le nombre recherché.
- **Spatiale** : O(n), pour le tableau de partitions.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

bool isPrime(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    for (int i = 3; i * i <= n; i += 2) {
        if (n % i == 0) return false;
    }
    return true;
}

int search(int limit, int target) {
    vector<int> partitions(limit, 0);
    partitions[0] = 1;
    
    for (int i = 0; i < limit; i++) {
        if (!isPrime(i))
            continue;
        for (int j = i; j < limit; j++)
            partitions[j] += partitions[j - i];
    }
    
    for (int i = 0; i < limit; i++) {
        if (partitions[i] > target)
            return i;
    }
    return -1;
}

int main() {
    const int TARGET = 5000;
    for (int limit = 1; ; limit *= 2) {
        int result = search(limit, TARGET);
        if (result != -1) {
            cout << result << endl;
            return 0;
        }
    }
    return 1;
}
```
