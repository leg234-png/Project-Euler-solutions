# Ensembles de Nombres Premiers Concaténés

## Description du Problème
Les nombres premiers 3, 7, 109 et 673 sont assez remarquables. En prenant deux nombres premiers quelconques et en les concaténant dans n'importe quel ordre, le résultat sera toujours premier. Par exemple, en prenant 7 et 109, les deux nombres 7109 et 1097 sont premiers. La somme de ces quatre nombres premiers, 792, représente la plus petite somme pour un ensemble de quatre nombres premiers ayant cette propriété.

**Objectif** : Trouver la plus petite somme pour un ensemble de cinq nombres premiers pour lesquels deux nombres premiers quelconques concaténés produisent un autre nombre premier.

---

## Exemples
### Exemple 1
**Entrée** : Ensemble de 4 nombres premiers
**Sortie** : `792`
**Explication** : {3, 7, 109, 673} est un tel ensemble.

---

## Approche de Résolution

### Stratégie
1. Générer tous les nombres premiers jusqu'à une limite.
2. Utiliser une recherche récursive avec mémorisation :
   - Pour chaque ensemble partiel de nombres premiers :
     - Essayer d'ajouter un nouveau nombre premier.
     - Vérifier que toutes les concaténations sont premières.
     - Si l'ensemble a 5 éléments, retourner la somme.
3. Retourner la plus petite somme trouvée.

### Observations
- On utilise la mémorisation pour éviter de recalculer les concaténations.
- On peut arrêter tôt si la somme dépasse une limite.
- On doit vérifier les concaténations dans les deux sens (ab et ba).

### Complexité
- **Temporelle** : O(P^5 × log m), où P est le nombre de nombres premiers et m est la taille maximale des concaténations.
- **Spatiale** : O(P²), pour la mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <unordered_map>
#include <string>

std::vector<bool> sieve(int n) {
    std::vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    int sqrtN = sqrt(n);
    for (int i = 2; i <= sqrtN; ++i) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    return isPrime;
}

bool isPrime(long long n, const std::vector<int>& primes) {
    if (n < 2) return false;
    int sqrtN = sqrt(n);
    for (int p : primes) {
        if (p > sqrtN) break;
        if (n % p == 0) return false;
    }
    return true;
}

bool isConcatPrime(int a, int b, const std::vector<int>& primes) {
    long long mult = 1;
    int temp = b;
    while (temp > 0) {
        mult *= 10;
        temp /= 10;
    }
    return isPrime((long long)a * mult + b, primes);
}

int findSetSum(const std::vector<int>& prefix, int targetSize, int sumLimit,
               const std::vector<int>& primes, 
               std::unordered_map<std::string, bool>& memo) {
    if (prefix.size() == targetSize) {
        int sum = 0;
        for (int p : prefix) sum += p;
        return sum;
    }
    
    int start = prefix.empty() ? 0 : (std::find(primes.begin(), primes.end(), prefix.back()) - primes.begin() + 1);
    
    for (int i = start; i < primes.size() && primes[i] <= sumLimit; ++i) {
        bool valid = true;
        for (int p : prefix) {
            std::string key1 = std::to_string(i) + "," + std::to_string(p);
            std::string key2 = std::to_string(p) + "," + std::to_string(i);
            
            if (memo.find(key1) == memo.end()) {
                memo[key1] = isConcatPrime(primes[i], p, primes);
            }
            if (memo.find(key2) == memo.end()) {
                memo[key2] = isConcatPrime(p, primes[i], primes);
            }
            
            if (!memo[key1] || !memo[key2]) {
                valid = false;
                break;
            }
        }
        
        if (valid) {
            std::vector<int> appended = prefix;
            appended.push_back(primes[i]);
            int result = findSetSum(appended, targetSize, sumLimit - primes[i], primes, memo);
            if (result != -1) {
                return result;
            }
        }
    }
    
    return -1;
}

int main() {
    const int limit = 100000;
    std::vector<bool> isPrimeVec = sieve(limit);
    std::vector<int> primes;
    for (int i = 2; i <= limit; ++i) {
        if (isPrimeVec[i]) {
            primes.push_back(i);
        }
    }
    
    std::unordered_map<std::string, bool> memo;
    int sumLimit = limit;
    
    while (true) {
        int sum = findSetSum({}, 5, sumLimit - 1, primes, memo);
        if (sum == -1) {
            std::cout << sumLimit << std::endl;
            return 0;
        }
        sumLimit = sum;
    }
}
```
