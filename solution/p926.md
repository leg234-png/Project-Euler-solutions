# Rondeur totale

## Description du Problème
Un nombre rond est un nombre qui se termine par un ou plusieurs zéros dans une base donnée.

Par exemple, 20 a une rondeur de 2 en base 2, car la représentation en base 2 de 20 est 10100, qui se termine par 2 zéros.

Définissons aussi R(n), la rondeur totale d'un nombre n, comme la somme de la rondeur de n en base b pour tous les b > 1.

Par exemple, 20 a une rondeur de 2 en base 2 et une rondeur de 1 en base 4, 5, 10, 20, donc on obtient R(20) = 6.

On vous donne aussi R(10!) = 312.

**Objectif** : Trouver R(10¹⁶!).

---

## Exemples
### Exemple 1
**Entrée** : `n = 20`
**Sortie** : `6`
**Explication** : R(20) = 6.

---

## Approche de Résolution

### Stratégie
1. Pour un nombre n et une base b, calculer la rondeur de n en base b.
2. La rondeur est le nombre de zéros à la fin de la représentation de n en base b.
3. Pour chaque base b > 1, calculer la rondeur et sommer.
4. Pour n = 10¹⁶!, on doit utiliser des techniques spéciales car 10¹⁶! est énorme.
5. Utiliser la formule de Legendre pour calculer les exposants des facteurs premiers.

### Observations
- On peut utiliser la formule de Legendre pour calculer les zéros finaux.
- Pour de très grandes factorielles, on doit utiliser des approximations ou des formules analytiques.
- La rondeur en base b est liée aux facteurs premiers de b.

### Complexité
- **Temporelle** : O(√n × log n) pour calculer R(n).
- **Spatiale** : O(1).

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

int roundness_in_base(long long n, int base) {
    int count = 0;
    while (n % base == 0) {
        count++;
        n /= base;
    }
    return count;
}

long long R(long long n) {
    long long total = 0;
    
    // Pour chaque base b > 1
    for (int b = 2; b <= n; b++) {
        total += roundness_in_base(n, b);
    }
    
    return total;
}
```
