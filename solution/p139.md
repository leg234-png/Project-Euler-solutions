# Pythagorean Tiles

## Description du Problème
Soit (a, b, c) représenter les trois côtés d'un triangle rectangle avec des longueurs de côtés intégrales. Il est possible de placer quatre de ces triangles ensemble pour former un carré de longueur c.

Par exemple, les triangles (3, 4, 5) peuvent être placés ensemble pour former un carré de 5 par 5 avec un trou de 1 par 1 au milieu et on peut voir que le carré de 5 par 5 peut être carrelé avec vingt-cinq carrés de 1 par 1.

Cependant, si les triangles (5, 12, 13) étaient utilisés, alors le trou mesurerait 7 par 7 et ceux-ci ne pourraient pas être utilisés pour carreler le carré de 13 par 13.

**Objectif** : Étant donné que le périmètre du triangle rectangle est inférieur à cent millions, combien de triangles de Pythagore permettraient un tel carrelage ?

---

## Exemples
### Exemple 1
**Entrée** : `(a, b, c) = (3, 4, 5)`
**Sortie** : `true`
**Explication** : Le trou est de 1×1, ce qui permet le carrelage.

---

## Approche de Résolution

### Stratégie
1. Pour un triplet de Pythagore primitif (a, b, c), le trou a une taille de (b - a) × (b - a).
2. Pour que le carrelage soit possible, c doit être divisible par (b - a).
3. Générer tous les triplets de Pythagore primitifs avec périmètre < 10⁸.
4. Pour chaque triplet, vérifier si c est divisible par (b - a).
5. Compter combien de triplets satisfont cette condition.

### Observations
- On utilise le théorème des triplets de Pythagore : a = st, b = (s²-t²)/2, c = (s²+t²)/2.
- On doit vérifier que s et t sont copremiers et impairs.

### Complexité
- **Temporelle** : O(n²), où n est la limite pour s et t.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>

long long gcd(long long a, long long b) {
    while (b != 0) {
        long long temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

int main() {
    const long long LIMIT = 100000000;
    int count = 0;
    
    for (long long s = 3; s * s / 2 < LIMIT; s += 2) {
        for (long long t = 1; t < s; t += 2) {
            long long a = s * t;
            long long b = (s * s - t * t) / 2;
            long long c = (s * s + t * t) / 2;
            long long p = a + b + c;
            
            if (p >= LIMIT) break;
            if (c % (a - b) == 0 && gcd(s, t) == 1) {
                count += (LIMIT - 1) / p;
            }
        }
    }
    
    std::cout << count << std::endl;
    return 0;
}
```
