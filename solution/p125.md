# Palindromic Sums

## Description du Problème
Le nombre palindromique 595 est intéressant car il peut être écrit comme la somme de carrés consécutifs : 6² + 7² + 8² + 9² + 10² + 11² + 12².

Il y a exactement onze palindromes en dessous de mille qui peuvent être écrits comme des sommes de carrés consécutifs, et la somme de ces palindromes est 4164. Notez que 1 = 0² + 1² n'a pas été inclus car ce problème concerne les carrés d'entiers positifs.

**Objectif** : Trouver la somme de tous les nombres inférieurs à 10⁸ qui sont à la fois palindromiques et peuvent être écrits comme la somme de carrés consécutifs.

---

## Exemples
### Exemple 1
**Entrée** : `n = 595`
**Sortie** : `true`
**Explication** : 595 = 6² + 7² + 8² + 9² + 10² + 11² + 12² et est palindromique.

---

## Approche de Résolution

### Stratégie
1. Pour chaque point de départ i, calculer les sommes de carrés consécutifs.
2. Pour chaque somme, vérifier si elle est palindromique et < 10⁸.
3. Utiliser un HashSet pour éviter de compter les mêmes nombres plusieurs fois.
4. Sommer tous les nombres palindromiques uniques trouvés.

### Observations
- On commence à i=1 et on ajoute des carrés consécutifs jusqu'à ce que la somme dépasse 10⁸.
- On vérifie la propriété de palindrome en comparant les chiffres.
- On utilise un set pour éviter les doublons.

### Complexité
- **Temporelle** : O(n²), où n ≈ 10⁴ est le nombre de points de départ à considérer.
- **Spatiale** : O(n) pour stocker les nombres palindromiques uniques.

---

## Solution en C++

```cpp
#include <iostream>
#include <set>
#include <string>
#include <algorithm>

bool isPalindrome(int n) {
    std::string s = std::to_string(n);
    std::string rev = s;
    std::reverse(rev.begin(), rev.end());
    return s == rev;
}

int main() {
    const int LIMIT = 100000000;
    std::set<int> nums;
    
    for (int i = 1; i <= 10000; i++) {
        int sum = i * i;
        for (int j = i + 1; ; j++) {
            sum += j * j;
            if (sum >= LIMIT) break;
            if (isPalindrome(sum)) {
                nums.insert(sum);
            }
        }
    }
    
    long long total = 0;
    for (int x : nums) {
        total += x;
    }
    std::cout << total << std::endl;
    return 0;
}
```
