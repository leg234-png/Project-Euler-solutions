# Triangles avec côtés entiers et un angle entier

## Description du Problème
Combien y a-t-il de triangles avec des côtés entiers, au moins un angle entier (mesuré en degrés), et un périmètre qui ne dépasse pas 10⁸ ?

---

## Exemples
### Exemple 1
**Entrée** : `périmètre ≤ 100`
**Sortie** : `Nombre de triangles`
**Explication** : Le nombre de triangles satisfaisant les conditions.

---

## Approche de Résolution

### Stratégie
1. Utiliser la loi des cosinus : c² = a² + b² - 2ab×cos(C).
2. Si a, b, c sont rationnels, alors cos(C) sera rationnel, ce qui implique que C = 60°, 90°, ou 120° (cos = 1/2, 0, -1/2 respectivement).
3. Pour 60° :
   - a = m² + n² - mn
   - b = 2mn - n²
   - c = m² - n²
4. Pour 90° : utiliser les triplets pythagoriciens.
5. Pour 120° :
   - a = m² + mn + n²
   - b = 2mn + n²
   - c = m² - n²
6. Générer tous les triangles pour chaque angle et compter ceux avec périmètre ≤ 10⁸.

### Observations
- Les seuls angles entiers possibles sont 60°, 90°, et 120°.
- Chaque type d'angle a des formules paramétriques spécifiques.
- On doit éviter de compter les mêmes triangles plusieurs fois.

### Complexité
- **Temporelle** : O(√limit) pour chaque type d'angle.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>
#include <set>

using namespace std;

long long DegreeIntegerTriangles90(long long limit) {
    long long total = 0;
    long long sqrt_limit = sqrt(limit);
    
    for (long long m = 2; m <= sqrt_limit; m++) {
        for (long long n = 1; n < m; n++) {
            if ((m + n) % 2 == 1 && __gcd(m, n) == 1) {
                long long a = m * m + n * n;
                long long b = m * m - n * n;
                long long c = 2 * m * n;
                long long p = a + b + c;
                if (p <= limit) {
                    total += limit / p;
                }
            }
        }
    }
    return total;
}

long long DegreeIntegerTriangles60(long long limit) {
    long long total = 0;
    long long sqrt_limit = sqrt(limit);
    
    for (long long m = 2; m <= sqrt_limit; m++) {
        for (long long n = 1; n < m; n++) {
            if (__gcd(m, n) == 1) {
                long long a = m * m + n * n - m * n;
                long long b = 2 * m * n - n * n;
                long long c = m * m - n * n;
                if (a > 0 && b > 0 && c > 0) {
                    long long p = a + b + c;
                    if (p <= limit) {
                        total += limit / p;
                    }
                }
            }
        }
    }
    return total;
}

long long DegreeIntegerTriangles120(long long limit) {
    long long total = 0;
    long long sqrt_limit = sqrt(limit);
    
    for (long long m = 2; m <= sqrt_limit; m++) {
        for (long long n = 1; n < m; n++) {
            if (__gcd(m, n) == 1 && (m - n) % 3 != 0) {
                long long a = m * m + m * n + n * n;
                long long b = 2 * m * n + n * n;
                long long c = m * m - n * n;
                if (a > 0 && b > 0 && c > 0) {
                    long long p = a + b + c;
                    if (p <= limit) {
                        total += limit / p;
                    }
                }
            }
        }
    }
    return total;
}

long long compute(long long limit) {
    return DegreeIntegerTriangles90(limit) + 
           DegreeIntegerTriangles60(limit) + 
           DegreeIntegerTriangles120(limit);
}

int main() {
    cout << compute(100000000) << endl;
    return 0;
}
```
