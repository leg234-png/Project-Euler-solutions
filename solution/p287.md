# Quadtree encoding (a simple compression algorithm)

## Description du Problème
On dessine un cercle (disque rempli) de rayon `R = 2^(N-1)` centré à `(2^(N-1), 2^(N-1))` sur une image carrée de `2^N × 2^N` pixels.

Un pixel est noir si `(x - 2^(N-1))² + (y - 2^(N-1))² ≤ R²`, sinon il est blanc.

L'image est encodée avec un quadtree : si une région est entièrement noire ou blanche, elle est encodée avec 2 bits. Sinon, elle est divisée en 4 sous-régions.

**Objectif** : Pour `N = 24`, trouver le nombre minimum de bits nécessaires pour encoder l'image.

---

## Exemples
### Exemple 1
**Entrée** : `N = 2`
**Sortie** : Nombre de bits nécessaires
**Explication** : On encode l'image avec un quadtree.

---

## Approche de Résolution

### Stratégie
1. Utiliser une approche récursive pour calculer la longueur de compression.
2. Pour une région `[xStart, xEnd) × [yStart, yEnd)` :
   - Calculer `minR² = minX² + minY²` et `maxR² = maxX² + maxY²`.
   - Si `maxR² ≤ R²` : toute la région est noire → 2 bits.
   - Si `minR² > R²` : toute la région est blanche → 2 bits.
   - Sinon : diviser en 4 sous-régions et récursivement calculer.
3. Translater le système de coordonnées pour simplifier : `x' = x - 2^(N-1)`, `y' = y - 2^(N-1)`.

### Observations
- On peut déterminer si une région est entièrement noire/blanche en calculant les bornes de `x² + y²`.
- La récursion s'arrête quand une région est uniforme.

### Complexité
- **Temporelle : O(4^N)` dans le pire cas.
- **Spatiale : O(N)` pour la pile de récursion.

---

## Solution en C++

```cpp
#include <iostream>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

const int N = 24;
const long long RADIUS_SQUARED = 1LL << (2 * N - 2);

cpp_int compressedLength(int n, int xStart, int xEnd, int yStart, int yEnd) {
    int minAbsX = (xStart <= 0 && 0 < xEnd) ? 0 : min(abs(xStart), abs(xEnd - 1));
    int minAbsY = (yStart <= 0 && 0 < yEnd) ? 0 : min(abs(yStart), abs(yEnd - 1));
    int maxAbsX = max(abs(xStart), abs(xEnd - 1));
    int maxAbsY = max(abs(yStart), abs(yEnd - 1));
    
    long long minRadius = (long long)minAbsX * minAbsX + (long long)minAbsY * minAbsY;
    long long maxRadius = (long long)maxAbsX * maxAbsX + (long long)maxAbsY * maxAbsY;
    
    if (maxRadius <= RADIUS_SQUARED || minRadius > RADIUS_SQUARED)
        return 2;
    else {
        int xMid = (xStart + xEnd) / 2;
        int yMid = (yStart + yEnd) / 2;
        return 1
            + compressedLength(n - 1, xStart, xMid, yMid, yEnd)
            + compressedLength(n - 1, xMid, xEnd, yMid, yEnd)
            + compressedLength(n - 1, xStart, xMid, yStart, yMid)
            + compressedLength(n - 1, xMid, xEnd, yStart, yMid);
    }
}

int main() {
    int temp = 1 << (N - 1);
    cout << compressedLength(N, -temp, temp, -temp, temp) << endl;
    return 0;
}
```
