# Nombres repunit forts

## Description du Problème
Un nombre repunit est un nombre qui, dans une base b >= 2, est composé uniquement de chiffres 1. Par exemple, R(3) = 111 en base 10.

Un nombre est appelé un "strong repunit" s'il est un repunit dans au moins deux bases différentes.

**Objectif** : Trouver la somme de tous les nombres strong repunit inférieurs à 10¹².

---

## Exemples
### Exemple 1
**Entrée** : `n = 1`
**Sortie** : `1` (inclus)
**Explication** : 1 est un strong repunit car dans toute base b >= 2, sa représentation est "1".

### Exemple 2
**Entrée** : `n = 2`
**Sortie** : `0` (non inclus)
**Explication** : 2 n'est pas un strong repunit car en base 2 il est "10", et dans toute autre base b >= 3 il est "2".

---

## Approche de Résolution

### Stratégie
1. 1 est un cas spécial (toujours un strong repunit).
2. Pour tout nombre n >= 3, il est trivialement un repunit en base n-1 (représentation "11").
3. Pour qu'il soit un strong repunit, il doit être un repunit dans au moins une autre base.
4. Générer tous les repunits de longueur 3, 4, 5, etc. dans différentes bases.
5. Évaluer leur valeur et ajouter à un ensemble pour éliminer les doublons.
6. Sommer tous les nombres générés.

### Observations
- Un nombre n >= 3 est un repunit en base n-1 (représentation "11").
- Pour être un strong repunit, il doit avoir une représentation "111", "1111", etc. dans une autre base.
- La longueur maximale à tester est au plus le nombre de bits de la limite (40 bits pour 10¹²).

### Complexité
- **Temporelle** : O(L × B), où L est la longueur maximale et B est le nombre de bases à tester.
- **Spatiale** : O(N), où N est le nombre de strong repunits trouvés.

---

## Solution en C++

```cpp
#include <iostream>
#include <unordered_set>
#include <cmath>
#include <climits>

using namespace std;

const long long LIMIT = 1000000000000LL;

int bitLength(long long n) {
    int count = 0;
    while (n > 0) {
        count++;
        n >>= 1;
    }
    return count;
}

int main() {
    unordered_set<long long> strongRepunits;
    strongRepunits.insert(1LL);  // Cas spécial
    
    int maxLength = bitLength(LIMIT);
    
    // Pour chaque longueur possible de repunit (en ignorant la longueur triviale de 2)
    for (int length = 3; length <= maxLength; length++) {
        
        // Pour chaque base, jusqu'à ce que la valeur dépasse la limite
        for (int base = 2; ; base++) {
            
            // Évaluer valeur = base^(length-1) + base^(length-2) + ... + base^1 + base^0
            long long value = 0;
            bool overflow = false;
            
            for (int i = 0; i < length; i++) {
                // Vérifier le dépassement arithmétique
                if (value > LLONG_MAX / base) {
                    overflow = true;
                    break;
                }
                value *= base;
                if (value > LLONG_MAX - 1) {
                    overflow = true;
                    break;
                }
                value++;
            }
            
            if (overflow || value >= LIMIT) {
                break;
            }
            
            strongRepunits.insert(value);
        }
    }
    
    // Sommer tous les nombres générés
    long long sum = 0;
    for (long long x : strongRepunits) {
        sum += x;
    }
    
    cout << sum << endl;
    return 0;
}
```
