# Nombre double pandigital divisible par 11

## Description du Problème
Nous appelons un entier positif double pandigital s'il utilise tous les chiffres 0 à 9 exactement deux fois (sans zéro en tête). Par exemple, 40561817703823564929 est un tel nombre.

**Objectif** : Combien de nombres doubles pandigitaux sont divisibles par 11 ?

---

## Exemples
### Exemple 1
**Entrée** : `Nombre double pandigital`
**Sortie** : `Nombre divisible par 11`
**Explication** : Le nombre de nombres doubles pandigitaux divisibles par 11.

---

## Approche de Résolution

### Stratégie
1. Un nombre est divisible par 11 si la somme alternée de ses chiffres est divisible par 11.
2. Pour un nombre double pandigital, on a 20 chiffres au total.
3. Si on a la somme d'un ensemble de 10 chiffres, disons S, alors l'autre ensemble a la somme 90 - S.
4. Si 11|n, alors 11|(S - (90-S)) = 11|(2S - 90).
5. Pour chaque combinaison de 10 chiffres, vérifier si 2S - 90 est divisible par 11.
6. Pour chaque combinaison valide, réarranger les chiffres restants en fonction du nombre de zéros.

### Observations
- On doit choisir 10 chiffres parmi "00112233445566778899".
- Le nombre de façons de réarranger dépend du nombre de zéros et des doublons.
- On peut utiliser des factorielles pour compter les arrangements.

### Complexité
- **Temporelle** : O(C(20,10)) pour générer toutes les combinaisons.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
#include <cmath>

using namespace std;

long long factorial(int n) {
    long long result = 1;
    for (int i = 2; i <= n; i++) {
        result *= i;
    }
    return result;
}

int number_of_doubles(const string& pand) {
    int doubles = 0;
    for (char c = '0'; c <= '9'; c++) {
        if (count(pand.begin(), pand.end(), c) == 2) {
            doubles++;
        }
    }
    return doubles;
}

long long compute() {
    string digits = "00112233445566778899";
    long long total = 0;
    long long fac_9 = factorial(9);
    long long fac_10 = factorial(10);
    
    // Générer toutes les combinaisons de 10 chiffres
    vector<bool> mask(20, false);
    fill(mask.begin(), mask.begin() + 10, true);
    
    do {
        string selected = "";
        for (int i = 0; i < 20; i++) {
            if (mask[i]) {
                selected += digits[i];
            }
        }
        
        int zero_count = count(selected.begin(), selected.end(), '0');
        int sum = 0;
        for (char c : selected) {
            sum += (c - '0');
        }
        
        if ((2 * sum - 90) % 11 == 0) {
            int num_doubles = number_of_doubles(selected);
            long long arrangements = 1;
            
            if (zero_count == 0) {
                arrangements = fac_10 / pow(2, num_doubles);
            } else {
                arrangements = fac_9 / pow(2, num_doubles - (zero_count == 2 ? 1 : 0));
            }
            
            total += arrangements * arrangements; // Pour les deux ensembles
        }
    } while (prev_permutation(mask.begin(), mask.end()));
    
    return total;
}

int main() {
    cout << compute() << endl;
    return 0;
}
```
