# Triangle concave

## Description du Problème
Un diagramme montre un carré de côté 1 avec un quart de cercle de rayon 1 centré dans le coin supérieur gauche. Une ligne diagonale de pente 1/n (pour un n positif) est tracée depuis le coin inférieur gauche.

L'aire de la section L bleue est 1 - π/4.

L'aire du triangle concave orange est formée par la ligne diagonale et le cercle.

**Objectif** : Trouver le plus petit n tel que l'aire du triangle concave orange est inférieure à 0,1% de l'aire de la section L bleue.

---

## Exemples
### Exemple 1
**Entrée** : `n = 1`
**Sortie** : `Aire du triangle concave`
**Explication** : Pour n = 1, la pente est 1, et on calcule l'aire du triangle concave.

---

## Approche de Résolution

### Stratégie
1. Définir le système de coordonnées : origine (0, 0), rayon du cercle = 1.
2. Le cercle le plus à gauche est centré en (1, 1), équation : (x-1)² + (y-1)² = 1.
3. La ligne diagonale a la pente s = 1/n, équation : y = s × x.
4. Trouver l'abscisse x où la ligne diagonale intersecte le premier cercle :
   - Substituer y = s×x dans l'équation du cercle.
   - Résoudre l'équation quadratique pour obtenir X.
5. Le triangle concave orange peut être divisé en deux parties :
   - Partie gauche : triangle propre, aire = X² × s / 2.
   - Partie droite : région entre le cercle et la ligne de base, calculée par intégration.
6. L'intégrale de f(x) = 1 - sqrt(2x - x²) est F(x) = (x-1) - [sqrt(2x-x²)×(x-1) + asin(x-1)] / 2.
7. L'aire de la partie droite est F(1) - F(X).
8. Rechercher n = 1, 2, 3, ... jusqu'à ce que le ratio soit < 0,001.

### Observations
- On utilise des calculs en virgule flottante pour les opérations arithmétiques et les fonctions irrationnelles.
- L'aire du triangle concave diminue strictement quand n augmente.
- On pourrait utiliser une recherche binaire pour optimiser.

### Complexité
- **Temporelle** : O(n), où n est la réponse (environ 2240).
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

// L'intégrale indéfinie de (1 - sqrt(2x - x²)) dx.
double integral(double x) {
    double t = x - 1;
    return t - (sqrt(x * (2 - x)) * t + asin(t)) / 2;
}

int main() {
    double lSectionArea = 1 - M_PI / 4;
    
    for (int i = 1; ; i++) {
        double slope = 1.0 / i;
        double a = slope * slope + 1;
        double b = -2 * (slope + 1);
        double c = 1;
        double discriminant = b * b - 4 * a * c;
        double x = (2 * c) / (-b + sqrt(discriminant));
        
        double concaveTriangleArea = (x * x * slope / 2) + (integral(1) - integral(x));
        
        if (concaveTriangleArea / lSectionArea < 0.001) {
            cout << i << endl;
            return 0;
        }
        
        if (i == INT_MAX) {
            cerr << "Dépassement" << endl;
            return 1;
        }
    }
    
    return 0;
}
```
