# Counting the number of "hollow" square laminae that can form one, two, three, ... distinct arrangements

## Description du Problème
On définit `N(t)` comme le nombre de façons de former une lamelle carrée creuse en utilisant exactement `t` carreaux.

Par exemple, `N(15) = 1` car il n'y a qu'une seule façon de former une lamelle avec 15 carreaux.

**Objectif** : Combien de valeurs de `t` ≤ 1 000 000 ont `1 ≤ N(t) ≤ 10` ?

---

## Exemples
### Exemple 1
**Entrée** : `t = 15`
**Sortie** : `N(15) = 1`
**Explication** : Il n'y a qu'une seule façon de former une lamelle avec 15 carreaux.

---

## Approche de Résolution

### Stratégie
1. Générer toutes les lamelles possibles avec au plus `SIZE_LIMIT` carreaux.
2. Pour chaque taille de carré extérieur `n` (à partir de 3) :
   - Pour chaque taille de trou intérieur `m` (de `n-2` à 1, en décrémentant de 2) :
     - Nombre de carreaux = `n² - m²`.
     - Si ce nombre dépasse `SIZE_LIMIT`, arrêter.
     - Sinon, incrémenter `type[tiles]`.
3. Compter les valeurs de `t` pour lesquelles `1 ≤ type[t] ≤ TYPE_LIMIT`.

### Observations
- Similaire au problème 173, mais on compte le nombre de façons pour chaque nombre de carreaux.
- On stocke dans un tableau `type[t]` le nombre de façons de former une lamelle avec `t` carreaux.

### Complexité
- **Temporelle : O(√SIZE_LIMIT × √SIZE_LIMIT) = O(SIZE_LIMIT)**.
- **Spatiale : O(SIZE_LIMIT)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

const int SIZE_LIMIT = 1000000;
const int TYPE_LIMIT = 10;

int main() {
    vector<int> type(SIZE_LIMIT + 1, 0);
    
    for (int n = 3; (n - 1) * 4 <= SIZE_LIMIT; n++) {  // Outer square size
        for (int m = n - 2; m >= 1; m -= 2) {  // Inner square hole size
            int tiles = n * n - m * m;
            if (tiles > SIZE_LIMIT)
                break;
            type[tiles]++;
        }
    }
    
    int count = 0;
    for (int t : type) {
        if (1 <= t && t <= TYPE_LIMIT)
            count++;
    }
    
    cout << count << endl;
    return 0;
}
```
