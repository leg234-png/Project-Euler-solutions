# Probabilités de score

## Description du Problème
Barbara est une mathématicienne et une joueuse de basket-ball. Elle a trouvé que la probabilité de marquer un point en tirant depuis une distance x est exactement (1−x/q), où q est une constante réelle supérieure à 50.

Lors de chaque séance d'entraînement, elle prend des tirs depuis les distances x=1,x=2,…,x=50 et, selon ses records, elle a précisément une chance de 2% de marquer un total d'exactement 20 points.

**Objectif** : Trouver q et donner votre réponse arrondie à 10 décimales.

---

## Exemples
### Exemple 1
**Entrée** : `50 tirs, 20 points nécessaires`
**Sortie** : `q arrondi à 10 décimales`
**Explication** : La valeur de q qui donne une probabilité de 2% de marquer exactement 20 points.

---

## Approche de Résolution

### Stratégie
1. Mettre en place le jeu avec mémorisation.
2. Utiliser l'algorithme de bissection pour trouver le q correct.
3. Pour chaque tir, la probabilité de réussir est (1 - shots/q) et la probabilité d'échouer est shots/q.
4. Utiliser la programmation dynamique récursive avec mémorisation pour calculer la probabilité.
5. Utiliser la bissection pour trouver q tel que la probabilité soit exactement 0.02.

### Observations
- La probabilité de marquer diminue avec la distance (shots).
- On doit calculer la probabilité d'obtenir exactement le score nécessaire après 50 tirs.
- La bissection converge rapidement vers la solution.

### Complexité
- **Temporelle** : O(log(high-low) × shots × score) pour la bissection et le calcul récursif.
- **Spatiale** : O(shots × score) pour la table de mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <unordered_map>
#include <cmath>
#include <string>

using namespace std;

unordered_map<string, double> memo;

double game(double q, int score, int shots, int needed_score) {
    if (score > needed_score) {
        return 0;
    }
    if (shots > 50) {
        return (score == needed_score) ? 1.0 : 0.0;
    }
    
    string key = to_string(q) + "," + to_string(score) + "," + to_string(shots);
    if (memo.find(key) != memo.end()) {
        return memo[key];
    }
    
    double lose_chance = (double)shots / q;
    double win_chance = 1.0 - lose_chance;
    
    double result = win_chance * game(q, score + 1, shots + 1, needed_score) +
                    lose_chance * game(q, score, shots + 1, needed_score);
    
    memo[key] = result;
    return result;
}

double compute(int needed_score) {
    double goal = 0.02;
    double accuracy = pow(10, -12);
    double low = 50.0;
    double high = 1000.0;
    
    while (high - low > accuracy) {
        memo.clear();
        double mid = (high + low) / 2.0;
        double prob = game(mid, 0, 1, needed_score);
        
        if (prob < goal) {
            high = mid;
        } else {
            low = mid;
        }
    }
    
    return round(high * 10000000000.0) / 10000000000.0;
}

int main() {
    cout << fixed << setprecision(10) << compute(20) << endl;
    return 0;
}
```
