# Divisor Square Sum

## Description du Problème
Pour un entier positif `n`, soit `σ₂(n)` la somme des carrés de ses diviseurs. Par exemple, `σ₂(6) = 1² + 2² + 3² + 6² = 50`.

**Objectif** : Trouver la somme de tous les `n`, `1 < n < 64 000 000`, pour lesquels `σ₂(n)` est un carré parfait.

---

## Exemples
### Exemple 1
**Entrée** : `n = 6`
**Sortie** : `σ₂(6) = 50` (pas un carré)
**Explication** : `σ₂(6) = 1 + 4 + 9 + 36 = 50`, qui n'est pas un carré parfait.

---

## Approche de Résolution

### Stratégie
1. Utiliser un crible modifié pour calculer `σ₂(n)` pour tous les `n` jusqu'à la limite.
2. Pour chaque nombre premier `p` :
   - Pour chaque multiple `j` de `p` :
     - Calculer la plus grande puissance `p^k` qui divise `j`.
     - Multiplier `σ₂(j)` par `(1 + p² + p⁴ + ... + p^(2k))`.
3. Pour chaque `n`, vérifier si `σ₂(n)` est un carré parfait.
4. Utiliser un test de carré parfait optimisé avec résidus modulaires.

### Observations
- `σ₂(n)` est une fonction multiplicative : `σ₂(p^k) = 1 + p² + p⁴ + ... + p^(2k)`.
- On peut utiliser un test de résidu modulaire pour exclure rapidement les non-carrés.
- Un test complet de carré utilise une recherche binaire sur les bits.

### Complexité
- **Temporelle : O(n log n)** pour le crible.
- **Spatiale : O(n)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

const int LIMIT = 64000000;

long long powerSquareSum(int n, int k) {
    long long result = 1;
    long long k2 = (long long)k * k;
    while (n % k == 0) {
        n /= k;
        result = result * k2 + 1;
    }
    return result;
}

class SquareTester {
private:
    vector<bool> isResidue;
    int modulus;
    
public:
    SquareTester(int mod) : modulus(mod) {
        isResidue.resize(mod, false);
        for (int i = 0; i < mod; i++)
            isResidue[(long long)i * i % mod] = true;
    }
    
    bool isPerfectSquare(long long x) {
        if (!isResidue[x % modulus])
            return false;
        
        long long y = 0;
        for (long long i = 1LL << 31; i != 0; i >>= 1) {
            y |= i;
            if (y > 3037000499LL || y * y > x)
                y ^= i;
        }
        return y * y == x;
    }
};

int main() {
    vector<long long> sigma2(LIMIT, 1);
    
    for (int i = 2; i < LIMIT; i++) {
        if (sigma2[i] == 1) {  // i is prime
            for (int j = i; j < LIMIT; j += i)
                sigma2[j] *= powerSquareSum(j, i);
        }
    }
    
    SquareTester sqt(3 * 5 * 7 * 11 * 13 * 17);
    long long sum = 0;
    for (int i = 1; i < LIMIT; i++) {
        if (sqt.isPerfectSquare(sigma2[i]))
            sum += i;
    }
    
    cout << sum << endl;
    return 0;
}
```
