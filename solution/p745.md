# Somme de carrés II

## Description du Problème
Soit g(n) le plus grand carré parfait qui divise n.

Par exemple, g(18) = 9, g(19) = 1.

Soit S(N) = ∑_{n=1}^N g(n).

Par exemple, S(10) = 24 et S(100) = 767.

**Objectif** : Trouver S(10¹⁴). Donner votre réponse modulo 1000000007.

---

## Exemples
### Exemple 1
**Entrée** : `N = 10`
**Sortie** : `24`
**Explication** : S(10) = 24.

---

## Approche de Résolution

### Stratégie
1. S(N) = ∑_{n=1}^N g(n) où g(n) est le plus grand carré parfait qui divise n.
2. Pour chaque carré parfait k², compter combien de nombres ≤ N sont divisibles par k² mais pas par (k+1)².
3. Il y a ⌊N/k²⌋ - ⌊N/(k+1)²⌋ tels nombres.
4. Donc S(N) = ∑_{k=1}^{√N} k² × (⌊N/k²⌋ - ⌊N/(k+1)²⌋).
5. Simplifier : S(N) = ∑_{k=1}^{√N} k² × ⌊N/k²⌋ - ∑_{k=2}^{√N+1} (k-1)² × ⌊N/k²⌋.
6. Utiliser une méthode de sommation efficace.

### Observations
- On peut calculer directement sans itérer sur tous les n.
- La formule permet de calculer en O(√N).
- On peut utiliser des techniques de sommation de Dirichlet pour optimiser davantage.

### Complexité
- **Temporelle** : O(√N) pour la méthode directe.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long mod = 1000000007;

long long compute(long long N) {
    long long sqrtN = sqrt(N);
    long long total = 0;
    
    for (long long k = 1; k <= sqrtN; k++) {
        long long k_sq = k * k;
        long long count = N / k_sq;
        total = (total + k_sq % mod * (count % mod)) % mod;
    }
    
    return total;
}

int main() {
    long long N = 1;
    for (int i = 0; i < 14; i++) {
        N *= 10;
    }
    cout << compute(N) << endl;
    return 0;
}
```
