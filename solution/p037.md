# Nombres Premiers Tronquables

## Description du Problème
Le nombre 3797 a une propriété intéressante. Étant premier lui-même, il est possible de retirer continuellement des chiffres de gauche à droite, et rester premier à chaque étape : 3797, 797, 97, et 7. De même, on peut travailler de droite à gauche : 3797, 379, 37, et 3.

**Objectif** : Trouver la somme des onze seuls nombres premiers qui sont à la fois tronquables de gauche à droite et de droite à gauche.

**NOTE** : 2, 3, 5, et 7 ne sont pas considérés comme des nombres premiers tronquables.

---

## Exemples
### Exemple 1
**Entrée** : Nombres premiers tronquables
**Sortie** : `748317`
**Explication** : La somme des 11 nombres premiers tronquables.

---

## Approche de Résolution

### Stratégie
1. Tester chaque nombre premier à partir de 10 (les nombres à un chiffre sont exclus).
2. Pour chaque nombre premier :
   - Tester la troncature de gauche à droite : retirer des chiffres de gauche.
   - Tester la troncature de droite à gauche : retirer des chiffres de droite.
   - Si les deux conditions sont satisfaites, ajouter à la somme.
3. Arrêter quand on a trouvé 11 nombres.
4. Retourner la somme.

### Observations
- Pour la troncature de gauche à droite, on utilise le modulo pour obtenir les suffixes.
- Pour la troncature de droite à gauche, on divise par 10 pour obtenir les préfixes.
- On peut optimiser en utilisant un test de primalité efficace.

### Complexité
- **Temporelle** : O(n × d × √n), où `n` est le nombre premier recherché et `d` est le nombre de chiffres.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

bool isPrime(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    
    int sqrtN = sqrt(n);
    for (int i = 3; i <= sqrtN; i += 2) {
        if (n % i == 0) {
            return false;
        }
    }
    return true;
}

bool isTruncatablePrime(int n) {
    // Test troncature de gauche à droite
    for (long long i = 10; i <= n; i *= 10) {
        if (!isPrime(n % i)) {
            return false;
        }
    }
    
    // Test troncature de droite à gauche
    int temp = n;
    while (temp > 0) {
        if (!isPrime(temp)) {
            return false;
        }
        temp /= 10;
    }
    
    return true;
}

int main() {
    long long sum = 0;
    int count = 0;
    
    for (int n = 10; count < 11; ++n) {
        if (isTruncatablePrime(n)) {
            sum += n;
            count++;
        }
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
