# Fractions à Chiffres Annulés

## Description du Problème
La fraction 49/98 est une fraction curieuse, car un mathématicien inexpérimenté en tentant de la simplifier peut croire incorrectement que 49/98 = 4/8, ce qui est correct, est obtenu en annulant les 9.

On considérera les fractions comme 30/50 = 3/5 comme des exemples triviaux.

Il y a exactement quatre exemples non triviaux de ce type de fraction, inférieurs à un en valeur, et contenant deux chiffres au numérateur et au dénominateur.

**Objectif** : Si le produit de ces quatre fractions est donné dans ses termes les plus bas communs, trouver la valeur du dénominateur.

---

## Exemples
### Exemple 1
**Entrée** : Fractions à deux chiffres
**Sortie** : `100`
**Explication** : Le dénominateur du produit simplifié des quatre fractions curieuses.

---

## Approche de Résolution

### Stratégie
1. Pour chaque fraction n/d où 10 ≤ n < d < 100 :
   - Extraire les chiffres : n = 10×n₁ + n₀, d = 10×d₁ + d₀
   - Vérifier les cas non triviaux :
     - Si n₁ = d₀, vérifier si n₀/d₁ = n/d
     - Si n₀ = d₁, vérifier si n₁/d₀ = n/d
   - Si une condition est satisfaite, multiplier le numérateur et le dénominateur.
2. Simplifier le produit final en divisant par le PGCD.
3. Retourner le dénominateur.

### Observations
- On ne peut pas avoir n₀ = d₀ car cela impliquerait n = d.
- On ne peut pas avoir n₁ = d₁ pour la même raison.
- On doit seulement considérer les cas où n₀ = d₁ ou n₁ = d₀.

### Complexité
- **Temporelle** : O(n²), où `n` = 90 (nombres à deux chiffres).
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>

long long gcd(long long a, long long b) {
    while (b != 0) {
        long long temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

int main() {
    long long numer = 1;
    long long denom = 1;
    
    for (int d = 10; d < 100; ++d) {
        for (int n = 10; n < d; ++n) {
            int n0 = n % 10, n1 = n / 10;
            int d0 = d % 10, d1 = d / 10;
            
            if ((n1 == d0 && n0 * d == n * d1) || 
                (n0 == d1 && n1 * d == n * d0)) {
                numer *= n;
                denom *= d;
            }
        }
    }
    
    long long g = gcd(numer, denom);
    std::cout << (denom / g) << std::endl;
    return 0;
}
```
