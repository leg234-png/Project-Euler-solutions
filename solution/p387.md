# Nombres Harshad forts

## Description du Problème
Un nombre Harshad est un nombre divisible par la somme de ses chiffres.
201 est un nombre Harshad car 201 est divisible par 3 (somme de ses chiffres).

Un nombre Harshad fort est un nombre Harshad dont la somme de ses chiffres est un nombre premier.
201 est un nombre Harshad fort car 3 est premier.

Un nombre Harshad tronquable à droite est un nombre Harshad qui, lorsqu'on supprime des chiffres à droite, reste un nombre Harshad.
201 est un nombre Harshad tronquable à droite car 201, 20 et 2 sont tous des nombres Harshad.

**Objectif** : Trouver la somme de tous les nombres Harshad forts tronquables à droite qui sont aussi des nombres premiers, inférieurs à 10¹⁴.

---

## Exemples
### Exemple 1
**Entrée** : `n = 201`
**Sortie** : `201` (si c'est un Harshad fort tronquable à droite et premier)
**Explication** : 201 est Harshad (201 % 3 = 0), fort (3 est premier), tronquable à droite (20 et 2 sont Harshad).

---

## Approche de Résolution

### Stratégie
1. Utiliser une approche récursive pour construire les nombres Harshad tronquables à droite.
2. Commencer avec tous les nombres à un chiffre (1-9), qui sont trivialement des nombres Harshad.
3. Pour chaque nombre Harshad tronquable à droite n :
   - Calculer la somme de ses chiffres.
   - Vérifier si n est un Harshad fort (somme des chiffres est premier) et si n est premier.
   - Si oui, ajouter n à la somme.
   - Étendre n en ajoutant un chiffre à droite (0-9) :
     - Si le nouveau nombre est Harshad, continuer récursivement.
4. Continuer jusqu'à ce que les nombres dépassent la limite.

### Observations
- On construit les nombres de manière récursive en partant des nombres à un chiffre.
- On vérifie à chaque étape si le nombre est Harshad avant de continuer.
- On vérifie si un nombre est un Harshad fort et premier avant de l'ajouter à la somme.

### Complexité
- **Temporelle** : O(10^d), où d est le nombre de chiffres (environ 14).
- **Spatiale** : O(d), pour la récursion.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>
#include <string>

using namespace std;

const long long LIMIT = 100000000000000LL;

long long sum = 0;

bool isPrime(long long x) {
    if (x < 2) {
        return false;
    }
    if (x == 2) {
        return true;
    }
    if (x % 2 == 0) {
        return false;
    }
    
    long long sqrtX = (long long)sqrt(x);
    for (long long i = 3; i <= sqrtX; i += 2) {
        if (x % i == 0) {
            return false;
        }
    }
    return true;
}

// Note: n doit être un nombre Harshad tronquable à droite,
// et les autres arguments sont des propriétés du nombre n.
void findHarshadPrimes(long long n, int digitSum, bool isStrong) {
    // Décaler d'un chiffre vers la gauche, et essayer les 10 possibilités pour le chiffre de droite
    long long m = n * 10;
    int s = digitSum;
    
    for (int i = 0; i < 10 && m < LIMIT; i++, m++, s++) {
        if (isStrong && isPrime(m)) {
            sum += m;
        }
        if (m % s == 0) {
            findHarshadPrimes(m, s, isPrime(m / s));
        }
    }
}

int main() {
    // Tous les nombres à un chiffre sont trivialement des nombres Harshad
    for (int i = 1; i <= 9; i++) {
        findHarshadPrimes(i, i, false);
    }
    
    cout << sum << endl;
    return 0;
}
```
