# Cuboid Layers

## Description du Problème
Le nombre minimum de cubes pour couvrir chaque face visible sur un cuboïde mesurant 3 × 2 × 1 est vingt-deux.

Si on ajoute ensuite une deuxième couche à ce solide, il faudrait quarante-six cubes pour couvrir chaque face visible, la troisième couche nécessiterait soixante-dix-huit cubes, et la quatrième couche nécessiterait cent dix-huit cubes pour couvrir chaque face visible.

Cependant, la première couche sur un cuboïde mesurant 5 × 1 × 1 nécessite également vingt-deux cubes ; de même, la première couche sur des cuboïdes mesurant 5 × 3 × 1, 7 × 2 × 1 et 11 × 1 × 1 contiennent tous quarante-six cubes.

Nous définirons C(n) pour représenter le nombre de cuboïdes qui contiennent n cubes dans une de ses couches. Donc C(22)=2, C(46)=4, C(78)=5 et C(118)=8.

Il s'avère que 154 est la plus petite valeur de n pour laquelle C(n)=10.

**Objectif** : Trouver la plus petite valeur de n pour laquelle C(n)=1000.

---

## Exemples
### Exemple 1
**Entrée** : `n = 22`
**Sortie** : `2`
**Explication** : C(22) = 2, car deux cuboïdes différents ont une couche de 22 cubes.

---

## Approche de Résolution

### Stratégie
1. Pour chaque cuboïde (a, b, c) avec a ≤ b ≤ c, calculer le nombre de cubes dans chaque couche.
2. La formule pour le nombre de cubes dans la k-ième couche est :
   - 2(ab + ac + bc) + 4(a + b + c)(k-1) + 8(k-1)(k-2)/2
3. Compter combien de cuboïdes produisent chaque valeur de n.
4. Trouver le plus petit n tel que C(n) = 1000.

### Observations
- On peut limiter la recherche en fixant une borne supérieure pour a, b, c.
- La formule peut être simplifiée pour chaque couche.

### Complexité
- **Temporelle** : O(n³ × k), où n est la limite pour les dimensions et k le nombre de couches.
- **Spatiale** : O(n) pour compter C(n).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int countCubes(int a, int b, int c, int layer) {
    // Formule pour le nombre de cubes dans la k-ième couche
    return 2 * (a * b + a * c + b * c) + 
           4 * (a + b + c) * (layer - 1) + 
           8 * (layer - 1) * (layer - 2) / 2;
}

int main() {
    const int TARGET = 1000;
    const int LIMIT = 20000;  // Limite pour n
    
    std::vector<int> C(LIMIT + 1, 0);
    
    for (int a = 1; a <= 200; a++) {
        for (int b = a; b <= 200; b++) {
            for (int c = b; c <= 200; c++) {
                for (int layer = 1; ; layer++) {
                    int cubes = countCubes(a, b, c, layer);
                    if (cubes > LIMIT) break;
                    C[cubes]++;
                }
            }
        }
    }
    
    for (int n = 1; n <= LIMIT; n++) {
        if (C[n] == TARGET) {
            std::cout << n << std::endl;
            return 0;
        }
    }
    return 0;
}
```
