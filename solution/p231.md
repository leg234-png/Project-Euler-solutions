# The prime factorisation of binomial coefficients

## Description du Problème
Le coefficient binomial C(20000000, 15000000) est un très grand nombre.

**Objectif** : Trouver la somme de tous les facteurs premiers distincts de C(20000000, 15000000).

Par exemple, C(10, 3) = 120 = 2³ × 3 × 5, donc la somme est 2 + 3 + 5 = 10.

---

## Exemples
### Exemple 1
**Entrée** : `C(10, 3)`
**Sortie** : `10`
**Explication** : `C(10, 3) = 120 = 2³ × 3 × 5`, somme = 2 + 3 + 5 = 10.

---

## Approche de Résolution

### Stratégie
1. Utiliser la formule : `C(n, k) = n! / (k! × (n-k)!)`.
2. La somme des facteurs premiers de `C(n, k)` = somme des facteurs premiers de `n!` - somme des facteurs premiers de `k!` - somme des facteurs premiers de `(n-k)!`.
3. Pour calculer la somme des facteurs premiers de `n!` :
   - Pour chaque nombre `i` de 1 à `n` :
     - Factoriser `i` en utilisant le plus petit facteur premier.
     - Ajouter chaque facteur premier à la somme.
4. Utiliser un crible pour trouver le plus petit facteur premier de chaque nombre.

### Observations
- On peut factoriser efficacement en utilisant le plus petit facteur premier.
- La somme des facteurs premiers (avec multiplicité) de `n!` peut être calculée en sommant les facteurs premiers de chaque nombre de 1 à `n`.

### Complexité
- **Temporelle : O(n log log n)` pour le crible + O(n log n)` pour la factorisation.
- **Spatiale : O(n)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

const int N = 20000000;
const int K = 15000000;

vector<int> listSmallestPrimeFactors(int n) {
    vector<int> smallest(n + 1);
    for (int i = 2; i <= n; i++) {
        if (smallest[i] == 0) {
            smallest[i] = i;
            for (int j = i * 2; j <= n; j += i) {
                if (smallest[j] == 0)
                    smallest[j] = i;
            }
        }
    }
    return smallest;
}

long long factorialPrimeFactorSum(int n, const vector<int>& smallestPrimeFactor) {
    long long sum = 0;
    for (int i = 1; i <= n; i++) {
        int j = i;
        while (j > 1) {
            int p = smallestPrimeFactor[j];
            sum += p;
            j /= p;
        }
    }
    return sum;
}

int main() {
    vector<int> smallestPrimeFactor = listSmallestPrimeFactors(N);
    long long result = factorialPrimeFactorSum(N, smallestPrimeFactor)
                     - factorialPrimeFactorSum(K, smallestPrimeFactor)
                     - factorialPrimeFactorSum(N - K, smallestPrimeFactor);
    cout << result << endl;
    return 0;
}
```
