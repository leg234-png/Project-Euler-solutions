# Somme maximale de chemin II

## Description du Problème
En commençant en haut du triangle ci-dessous et en se déplaçant vers les nombres adjacents de la ligne en dessous, la somme maximale du haut vers le bas est 23.

```
   3
  7 4
 2 4 6
8 5 9 3
```

C'est-à-dire, 3 + 7 + 4 + 9 = 23.

**Objectif** : Trouver la somme maximale du haut vers le bas dans triangle.txt, un fichier texte de 15K contenant un triangle avec cent lignes.

**NOTE** : C'est une version beaucoup plus difficile du Problème 18. Il n'est pas possible d'essayer chaque route pour résoudre ce problème, car il y en a 2⁹⁹ au total ! Si vous pouviez vérifier un billion (10¹²) de routes chaque seconde, il faudrait plus de vingt milliards d'années pour toutes les vérifier. Il existe un algorithme efficace pour le résoudre. ;o)

---

## Exemples
### Exemple 1
**Entrée** : Triangle de 4 lignes
```
   3
  7 4
 2 4 6
8 5 9 3
```
**Sortie** : `23`
**Explication** : Le chemin optimal est 3 → 7 → 4 → 9 = 23.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique en partant du bas du triangle.
2. Pour chaque cellule, calculer la somme maximale en ajoutant la valeur de la cellule au maximum de ses deux enfants en dessous.
3. Modifier le triangle en place de bas en haut.
4. La valeur à la racine sera la somme maximale.

### Observations
- Si on commence à une cellule particulière, quelle est la somme maximale du chemin ?
- Si la cellule est en bas du triangle, c'est simplement la valeur de la cellule.
- Sinon, la réponse est la valeur de la cellule plus soit {la somme maximale de la cellule en bas à gauche}, soit {la somme maximale de la cellule en bas à droite}, selon le plus grand.
- En calculant les valeurs du triangle vierge de bas en haut, les valeurs dépendantes sont toujours calculées avant d'être utilisées.

### Complexité
- **Temporelle** : O(n²), où n est le nombre de lignes (100).
- **Spatiale** : O(n²), pour stocker le triangle.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <fstream>
#include <sstream>

using namespace std;

int main() {
    // Lire le triangle depuis le fichier
    vector<vector<int>> triangle;
    ifstream file("triangle.txt");
    string line;
    
    while (getline(file, line)) {
        vector<int> row;
        istringstream iss(line);
        int num;
        while (iss >> num) {
            row.push_back(num);
        }
        triangle.push_back(row);
    }
    file.close();
    
    // Programmation dynamique de bas en haut
    for (int i = triangle.size() - 2; i >= 0; i--) {
        for (int j = 0; j < (int)triangle[i].size(); j++) {
            triangle[i][j] += max(triangle[i + 1][j], triangle[i + 1][j + 1]);
        }
    }
    
    cout << triangle[0][0] << endl;
    return 0;
}
```
