# Fourmi pythagoricienne

## Description du Problème
Dave a découpé un triangle avec des côtés de 30cm, 40cm et 50cm. Une rafale de vent fait tomber le triangle dans le jardin.

Une autre rafale fait tomber une petite fourmi directement sur ce triangle. La pauvre fourmi est complètement désorientée et commence à ramper tout droit dans une direction aléatoire pour retourner dans l'herbe.

**Objectif** : En supposant que toutes les positions possibles de la fourmi dans le triangle et toutes les directions de mouvement possibles sont équiprobables, quelle est la probabilité que la fourmi quitte le triangle le long de son côté le plus long ? Donner votre réponse arrondie à 10 chiffres après la virgule décimale.

---

## Exemples
### Exemple 1
**Entrée** : `Triangle 30-40-50`
**Sortie** : `Probabilité`
**Explication** : La probabilité que la fourmi quitte par le côté le plus long.

---

## Approche de Résolution

### Stratégie
1. Le triangle a des côtés 30, 40, 50 (triangle rectangle).
2. Le côté le plus long est 50 (hypoténuse).
3. Pour chaque position (x, y) dans le triangle, calculer l'angle γ tel que la fourmi quitte par le côté le plus long.
4. γ = π/2 + arctan(x/(4a-y)) + arctan(y/(3b-x)), où a et b sont des facteurs d'échelle.
5. La probabilité est la moyenne de γ/(2π) sur toutes les positions dans le triangle.
6. Utiliser l'intégration numérique ou une méthode de Monte Carlo.

### Observations
- Le problème nécessite une intégration sur la région du triangle.
- On peut utiliser une méthode de quadrature numérique.
- La probabilité dépend de la position et de l'angle de sortie.

### Complexité
- **Temporelle** : O(N²) où N est la résolution de la grille.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <cmath>

using namespace std;

double compute(int a, int b) {
    double total = 0;
    int count = 0;
    
    for (int y = 0; y < 4 * a; y++) {
        for (int x = 0; x < 3 * b; x++) {
            if (y >= (-4.0 * x) / 3.0 + 4 * a) {
                break;
            }
            
            double gamma = M_PI / 2.0 + atan((double)x / (4.0 * a - y)) + 
                          atan((double)y / (3.0 * b - x));
            total += gamma / (2.0 * M_PI);
            count++;
        }
    }
    
    return total / count;
}

int main() {
    cout << fixed << setprecision(10) << compute(1000, 1000) << endl;
    return 0;
}
```
