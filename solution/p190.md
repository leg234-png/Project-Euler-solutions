# Maximisation d'un produit pondéré

## Description du Problème
Soit S_m = (x₁, x₂, ..., x_m) un m-uplet de nombres réels positifs satisfaisant x₁ + x₂ + ... + x_m = m et x₁ × 1 + x₂ × 2 + ... + x_m × m = m(m+1)/2.

Soit P_m = x₁¹ × x₂² × ... × x_m^m le produit pondéré.

Par exemple, on peut vérifier que ⌊P₁₀⌋ = 4112 (⌊⌋ est la fonction partie entière).

**Objectif** : Trouver ∑ₘ₌₂¹⁵ ⌊P_m⌋.

---

## Exemples
### Exemple 1
**Entrée** : `m = 10`
**Sortie** : `4112`
**Explication** : ⌊P₁₀⌋ = 4112.

---

## Approche de Résolution

### Stratégie
1. Utiliser les multiplicateurs de Lagrange pour trouver le maximum.
2. Pour m donné, la solution optimale est x_i = 2i/(m+1).
3. Calculer P_m = ∏ᵢ₌₁ᵐ (2i/(m+1))^i.
4. Prendre la partie entière et sommer pour m de 2 à 15.

### Observations
- Le problème peut être résolu analytiquement en utilisant les multiplicateurs de Lagrange.
- La solution optimale donne x_i = 2i/(m+1) pour chaque i.
- On peut calculer directement P_m sans résoudre numériquement le système d'équations.

### Complexité
- **Temporelle** : O(m²) pour calculer P_m pour chaque m.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long P(int m) {
    double total = 1.0;
    for (int i = 1; i <= m; i++) {
        double x = (2.0 * i) / (m + 1);
        total *= pow(x, i);
    }
    return (long long)floor(total);
}

long long compute(int limit) {
    long long total = 0;
    for (int m = 2; m <= limit; m++) {
        total += P(m);
    }
    return total;
}

int main() {
    cout << compute(15) << endl;
    return 0;
}
```
