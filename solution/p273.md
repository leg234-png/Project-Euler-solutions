# Somme de carrés

## Description du Problème
Considérons les équations de la forme : a² + b² = N, 0 ≤ a ≤ b, a, b et N entiers.

Pour N=65, il y a deux solutions : a=1, b=8 et a=4, b=7.

Nous appelons S(N) la somme des valeurs de a de toutes les solutions de a² + b² = N, 0 ≤ a ≤ b, a, b et N entiers.

Ainsi S(65) = 1 + 4 = 5.

**Objectif** : Trouver ∑S(N), pour tous les N squarefree uniquement divisibles par des nombres premiers de la forme 4k+1 avec 4k+1 < 150.

---

## Exemples
### Exemple 1
**Entrée** : `N = 65`
**Sortie** : `5`
**Explication** : S(65) = 1 + 4 = 5.

---

## Approche de Résolution

### Stratégie
1. Utiliser le théorème de Fermat sur les sommes de deux carrés : p = 4k+1 si et seulement si p = x² + y² pour certains x et y.
2. N doit être squarefree et divisible uniquement par des nombres premiers de la forme 4k+1.
3. Pour un nombre premier p avec solution (a, b), et q avec solution (c, d), utiliser l'identité : (a² + b²)(c² + d²) = (ac + bd)² + (ad - bc)².
4. Générer toutes les combinaisons de nombres premiers et calculer les solutions pour chaque produit.
5. Sommer les valeurs de a pour toutes les solutions.

### Observations
- Il y a seulement 65534 nombres à vérifier, mais ils peuvent devenir très grands.
- L'identité permet de construire récursivement les solutions pour les produits de nombres premiers.
- Pour chaque produit, on peut avoir plusieurs représentations comme somme de deux carrés.

### Complexité
- **Temporelle** : O(2^P × R) où P est le nombre de nombres premiers et R est le nombre de représentations.
- **Spatiale** : O(2^P) pour stocker les combinaisons.
