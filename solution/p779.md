# Facteur premier et exposant

## Description du Problème
Soit p(n) le plus petit facteur premier de n.

Soit α(n) l'exposant de p(n) dans la factorisation en nombres premiers de n.

Soit f_K(n) = (α(n) - 1) / p(n)^K.

Définissons aussi f̄_K par : f̄_K = lim_{N→∞} 1/N ∑_{n=2}^N f_K(n).

On peut vérifier que f̄₁ ≈ 0.282419756159.

**Objectif** : Trouver f̄_15. Donner votre réponse arrondie à 12 places après la virgule décimale.

---

## Exemples
### Exemple 1
**Entrée** : `K = 1`
**Sortie** : `0.282419756159`
**Explication** : f̄₁ ≈ 0.282419756159.

---

## Approche de Résolution

### Stratégie
1. Dans une plage de n nombres, il y a n/2 nombres dont le plus petit facteur premier est 2.
2. Il y a n×(1-1/2)×(1/3) nombres dont le plus petit facteur premier est 3.
3. En général, il y a n×(1-1/2)×(1-1/3)×...×(1-1/(p_i-1))×(1/p_i) nombres dont le plus petit facteur premier est p_i.
4. Pour un p_i donné, il y a (p_i - 1)/p_i^k valeurs qui ont un ordre p-adique k.
5. f̄_K = ∑_{p_i premier} (1-1/2)×(1-1/3)×...×(1-1/(p_i-1)) × (1/(p_i^(K+1)(p_i-1))).

### Observations
- On peut calculer f̄_K directement avec une formule.
- La série converge rapidement.
- On n'a besoin que d'un nombre fini de nombres premiers.

### Complexité
- **Temporelle** : O(P) où P est le nombre de nombres premiers à considérer.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <vector>
#include <cmath>

using namespace std;

vector<int> prime_sieve(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

double compute(int K) {
    vector<int> primes = prime_sieve(1000000);
    double total = 0.0;
    double product = 1.0;
    
    for (int p : primes) {
        double term = product * (1.0 / (pow(p, K + 1) * (p - 1)));
        total += term;
        product *= (1.0 - 1.0 / p);
    }
    
    return total;
}

int main() {
    cout << fixed << setprecision(12) << compute(15) << endl;
    return 0;
}
```
