# Logique circulaire

## Description du Problème
Nous voulons trouver toutes les fonctions T telles que T(a, b, c, d, e, f) & T(b, c, d, e, f, a ^ (b & c)) = 0.

En général, pour n variables, nous avons 2^n entrées possibles, et pour chaque entrée T(...) = 0 ou 1, donc au total nous avons 2^(2^n) fonctions T.

Mais dans notre cas, notez que si T(a, b, c, d, e, f) = 1, il doit être que T(b, c, d, e, f, a ^ (b & c)) = 0 et puisque nous pouvons déterminer l'entrée de la seconde moitié à partir de la première, nous créerons des cycles avec des sorties restreintes limitant nos T possibles.

**Objectif** : Trouver le nombre total de fonctions T qui satisfont la condition.

---

## Exemples
### Exemple 1
**Entrée** : `n = 6 variables`
**Sortie** : `Nombre de fonctions T`
**Explication** : Le nombre de fonctions T qui satisfont la contrainte.

---

## Approche de Résolution

### Stratégie
1. Parcourir tous les cycles possibles dans l'espace des 2^6 = 64 états.
2. Pour chaque cycle, déterminer quelles valeurs peuvent être assignées à T.
3. Si T(a, b, c, d, e, f) = 0, alors T(b, c, d, e, f, a ^ (b & c)) peut être 0 ou 1.
4. Si T(a, b, c, d, e, f) = 1, alors T(b, c, d, e, f, a ^ (b & c)) = 0.
5. Calculer le nombre total en utilisant les nombres de Lucas (qui sont liés aux nombres de Fibonacci).

### Observations
- Les cycles créent des contraintes sur les valeurs possibles de T.
- Le calcul de petites valeurs donne les nombres de Lucas.
- L(n) = F(n+1) + F(n-1) où F sont les nombres de Fibonacci.
- On peut calculer cela très rapidement en utilisant l'exponentiation matricielle.

### Complexité
- **Temporelle** : O(64 × log L) où L est la longueur du cycle, pour trouver tous les cycles.
- **Spatiale** : O(64) pour stocker les cycles.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

using namespace std;

int find_next(int curr) {
    string t = "";
    int temp = curr;
    for (int i = 0; i < 6; i++) {
        t += to_string(temp % 2);
        temp /= 2;
    }
    
    vector<int> b(6);
    for (int i = 0; i < 6; i++) {
        b[i] = t[i] - '0';
    }
    
    vector<int> nb(6);
    for (int i = 0; i < 5; i++) {
        nb[i] = b[i + 1];
    }
    nb[5] = b[0] ^ (b[1] & b[2]);
    
    int result = 0;
    for (int i = 0; i < 6; i++) {
        result += nb[i] * (1 << i);
    }
    return result;
}

long long lucas(long long n) {
    // Calcul du nème nombre de Lucas en utilisant l'exponentiation matricielle
    long long f2 = 1, f1 = 1, f0 = 0;
    string bits = "";
    long long temp = n;
    while (temp > 0) {
        bits = to_string(temp % 2) + bits;
        temp /= 2;
    }
    
    for (size_t i = 1; i < bits.length(); i++) {
        long long v = f1 * f1;
        long long new_f2 = f2 * f2 + v;
        long long new_f1 = (f2 + f0) * f1;
        long long new_f0 = v + f0 * f0;
        f2 = new_f2;
        f1 = new_f1;
        f0 = new_f0;
        
        if (bits[i] == '1') {
            long long temp_f2 = f2 + f1;
            long long temp_f1 = f2;
            long long temp_f0 = f1;
            f2 = temp_f2;
            f1 = temp_f1;
            f0 = temp_f0;
        }
    }
    return f2 + f0;
}

long long compute() {
    vector<int> pointer(64, -1);
    long long total = 1;
    
    for (int start = 0; start < 64; start++) {
        if (pointer[start] != -1) continue;
        
        int curr = start;
        int cycle_length = 0;
        while (pointer[curr] == -1) {
            pointer[curr] = start;
            curr = find_next(curr);
            cycle_length++;
        }
        
        if (pointer[curr] == start) {
            total *= lucas(cycle_length);
        }
    }
    
    return total;
}

int main() {
    cout << compute() << endl;
    return 0;
}
```
