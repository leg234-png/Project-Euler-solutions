# Nombres Amicaux

## Description du Problème
Soit d(n) défini comme la somme des diviseurs propres de n (nombres inférieurs à n qui divisent n de manière égale).

Si d(a) = b et d(b) = a, où a ≠ b, alors a et b forment une paire amicable et chacun de a et b est appelé un nombre amicable.

Par exemple, les diviseurs propres de 220 sont 1, 2, 4, 5, 10, 11, 20, 22, 44, 55 et 110 ; donc d(220) = 284. Les diviseurs propres de 284 sont 1, 2, 4, 71 et 142 ; donc d(284) = 220.

**Objectif** : Évaluer la somme de tous les nombres amicaux inférieurs à 10000.

---

## Exemples
### Exemple 1
**Entrée** : Limite = 300
**Sortie** : `504`
**Explication** : Les nombres amicaux inférieurs à 300 sont 220 et 284. Leur somme est 504.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre i de 1 à 9999 :
   - Calculer d(i) (somme des diviseurs propres de i).
   - Vérifier si d(d(i)) = i et d(i) ≠ i.
   - Si c'est le cas, i est un nombre amicable, l'ajouter à la somme.
2. Retourner la somme.

### Observations
- Pour calculer la somme des diviseurs propres, parcourir de 1 à √n.
- Si i divise n, alors n/i divise aussi n.
- Compter chaque paire (i, n/i) une seule fois.
- Si n est un carré parfait, compter √n une seule fois.

### Complexité
- **Temporelle** : O(n × √n), où `n` = 10000.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

int divisorSum(int n) {
    int sum = 0;
    int sqrtN = sqrt(n);
    
    for (int i = 1; i < sqrtN; ++i) {
        if (n % i == 0) {
            sum += i + (n / i);
        }
    }
    
    if (sqrtN * sqrtN == n) {
        sum += sqrtN;
    }
    
    return sum - n;  // Diviseurs propres (exclure n)
}

bool isAmicable(int n) {
    int m = divisorSum(n);
    return m != n && divisorSum(m) == n;
}

int main() {
    int sum = 0;
    for (int i = 1; i < 10000; ++i) {
        if (isAmicable(i)) {
            sum += i;
        }
    }
    std::cout << sum << std::endl;
    return 0;
}
```
