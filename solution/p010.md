# Somme des Nombres Premiers

## Description du Problème
La somme des nombres premiers inférieurs à 10 est 2 + 3 + 5 + 7 = 17.

**Objectif** : Trouver la somme de tous les nombres premiers inférieurs à deux millions.

---

## Exemples
### Exemple 1
**Entrée** : Limite = 10
**Sortie** : `17`
**Explication** : Les nombres premiers inférieurs à 10 sont 2, 3, 5, 7. Leur somme est 17.

---

## Approche de Résolution

### Stratégie
1. Utiliser le crible d'Ératosthène pour trouver tous les nombres premiers inférieurs à 2 000 000.
2. Additionner tous les nombres premiers trouvés.
3. Retourner la somme.

### Observations
- Le crible d'Ératosthène est efficace pour trouver tous les nombres premiers jusqu'à une limite donnée.
- Algorithme du crible :
  1. Créer un tableau de booléens de taille n, initialisé à `true`.
  2. Marquer 0 et 1 comme non premiers.
  3. Pour chaque nombre i de 2 à √n, si i est premier, marquer tous ses multiples comme non premiers.
  4. Les nombres restants marqués `true` sont premiers.
- La somme peut être très grande, donc utiliser `long long`.

### Complexité
- **Temporelle** : O(n log log n), où `n` = 2 000 000.
- **Spatiale** : O(n), pour le tableau du crible.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

std::vector<bool> sieve(int n) {
    std::vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    int sqrtN = sqrt(n);
    for (int i = 2; i <= sqrtN; ++i) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    return isPrime;
}

int main() {
    const int limit = 2000000;
    std::vector<bool> isPrime = sieve(limit);
    
    long long sum = 0;
    for (int i = 2; i < limit; ++i) {
        if (isPrime[i]) {
            sum += i;
        }
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
