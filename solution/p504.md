# Carré à l'intérieur

## Description du Problème
Soit ABCD un quadrilatère dont les sommets sont des points de grille situés sur les axes de coordonnées comme suit :

A(a, 0), B(0, b), C(−c, 0), D(0, −d), où 1 ≤ a, b, c, d ≤ m et a, b, c, d, m sont des entiers.

On peut montrer que pour m = 4, il y a exactement 256 façons valides de construire ABCD. De ces 256 quadrilatères, 42 d'entre eux contiennent strictement un nombre carré de points de grille.

**Objectif** : Combien de quadrilatères ABCD contiennent strictement un nombre carré de points de grille pour m = 100 ?

---

## Exemples
### Exemple 1
**Entrée** : `m = 4`
**Sortie** : `42`
**Explication** : 42 quadrilatères contiennent strictement un nombre carré de points de grille.

---

## Approche de Résolution

### Stratégie
1. Utiliser le théorème de Pick : A = i + b/2 - 1, où A est l'aire, i est le nombre de points intérieurs, b est le nombre de points de frontière.
2. L'aire du quadrilatère est A = 1/2(ab + bc + ad + cd) = 1/2(a + c)(b + d).
3. Le segment entre A et B a gcd(a, b) + 1 points de frontière entiers.
4. Calculer le nombre total de points de frontière pour les 4 segments.
5. Calculer i = A - b/2 + 1.
6. Vérifier si i est un carré parfait.

### Observations
- Le théorème de Pick permet de calculer le nombre de points intérieurs.
- Le nombre de points de frontière sur un segment de (0,0) à (a,b) est gcd(a,b) + 1.
- On doit soustraire les sommets qui sont comptés plusieurs fois.

### Complexité
- **Temporelle** : O(m⁴) pour itérer sur tous les quadruplets (a,b,c,d).
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long boundary_points(int a, int b, int c, int d) {
    long long b1 = __gcd(a, b) + 1;
    long long b2 = __gcd(b, c) + 1;
    long long b3 = __gcd(c, d) + 1;
    long long b4 = __gcd(d, a) + 1;
    return b1 + b2 + b3 + b4 - 4; // Soustraire les 4 sommets comptés deux fois
}

bool is_perfect_square(long long n) {
    long long root = sqrt(n);
    return root * root == n;
}

long long compute(int m) {
    long long count = 0;
    
    for (int a = 1; a <= m; a++) {
        for (int b = 1; b <= m; b++) {
            for (int c = 1; c <= m; c++) {
                for (int d = 1; d <= m; d++) {
                    long long area = (long long)(a + c) * (b + d) / 2;
                    long long boundary = boundary_points(a, b, c, d);
                    long long interior = area - boundary / 2 + 1;
                    
                    if (interior > 0 && is_perfect_square(interior)) {
                        count++;
                    }
                }
            }
        }
    }
    
    return count;
}

int main() {
    cout << compute(100) << endl;
    return 0;
}
```
