# Comptage de rectangles

## Description du Problème
En comptant soigneusement, on peut voir qu'une grille rectangulaire mesurant 3 par 2 contient dix-huit rectangles.

**Objectif** : Bien qu'il n'existe pas de grille rectangulaire qui contienne exactement deux millions de rectangles, trouver l'aire de la grille avec la solution la plus proche.

---

## Exemples
### Exemple 1
**Entrée** : Grille 3×2
**Sortie** : `18`
**Explication** : Il y a 18 rectangles dans une grille 3×2.

---

## Approche de Résolution

### Stratégie
1. Le nombre de rectangles dans une grille m×n est donné par : (m+1)×m×(n+1)×n / 4.
2. Pour chaque largeur w de 1 à √(2 000 000), calculer la hauteur h qui donne le nombre de rectangles le plus proche de 2 000 000.
3. Garder la combinaison (w, h) qui donne la différence minimale avec 2 000 000.
4. Retourner l'aire w×h.

### Observations
- Le nombre de rectangles est : Σ(i=1 à m) Σ(j=1 à n) (m-i+1)×(n-j+1) = (m+1)×m×(n+1)×n / 4.
- On peut limiter la recherche à w ≤ √(2 000 000) car le nombre de rectangles croît rapidement.

### Complexité
- **Temporelle** : O(√T), où T = 2 000 000.
- **Spatiale** : O(1), utilisation d'espace constant.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>
#include <climits>

using namespace std;

int numberOfRectangles(int m, int n) {
    return (m + 1) * m * (n + 1) * n / 4;
}

int main() {
    const int TARGET = 2000000;
    int bestDiff = INT_MAX;
    int bestArea = -1;
    int sqrt_target = (int)sqrt(TARGET);
    
    for (int w = 1; w <= sqrt_target; w++) {
        for (int h = 1; h <= sqrt_target; h++) {
            int diff = abs(numberOfRectangles(w, h) - TARGET);
            if (diff < bestDiff) {
                bestDiff = diff;
                bestArea = w * h;
            }
        }
    }
    
    cout << bestArea << endl;
    return 0;
}
```
