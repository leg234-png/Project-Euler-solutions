# Somme d'une formule modulaire

## Description du Problème
Pour un nombre premier impair p, soit f(p) = ⌊2^(2^p) / p⌋ (mod 2^p).

Définissons aussi g(p) = f(p) (mod p). On vous donne g(31) = 17.

Maintenant, définissons G(N) comme la somme de g(p) pour tous les nombres premiers impairs inférieurs à N.

On vous donne G(100) = 474 et G(10⁴) = 2819236.

**Objectif** : Trouver G(10⁷).

---

## Exemples
### Exemple 1
**Entrée** : `p = 31`
**Sortie** : `17`
**Explication** : g(31) = 17.

---

## Approche de Résolution

### Stratégie
1. Soit 2^(2^p) = qp + r, 0 ≤ r < p, alors q = ⌊2^(2^p) / p⌋ = (2^(2^p) - r) / p.
2. r = 2^(2^p) (mod p).
3. Noter que 2^(2^p) = 2^p × 2^(2^p - p) => 2^(2^p) = 0 (mod 2^p).
4. Donc f(p) = (2^2^p - r) / p (mod 2^p) = -r / p (mod 2^p) = -r × p^(-1) (mod 2^p).
5. Pour calculer r : utiliser le petit théorème de Fermat, 2^(p-1) = 1 (mod p).
6. Donc 2^(2^p) = 2^(2^p (mod p-1)) (mod p).
7. Pour calculer p^(-1) : p^(-1) = ((p-1)×2^(p-1) + 1) / p (mod 2^p).
8. Calculer f(p) puis g(p) = f(p) (mod p).

### Observations
- On peut utiliser l'exponentiation modulaire rapide pour calculer 2^(2^p) mod p.
- L'inverse modulaire peut être calculé efficacement.
- On doit gérer les grands nombres avec l'arithmétique modulaire.

### Complexité
- **Temporelle** : O(P × log p) où P est le nombre de nombres premiers.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long g(long long p) {
    // Calculer r = 2^(2^p) mod p
    long long exp_mod = mod_pow(2, p, p - 1);
    long long r = mod_pow(2, exp_mod, p);
    
    // Calculer p^(-1) mod 2^p
    long long two_p = 1LL << p;
    long long p_inv = ((p - 1) * mod_pow(2, p - 1, two_p) + 1) / p;
    p_inv = p_inv % two_p;
    
    // Calculer f(p) = -r * p^(-1) mod 2^p
    long long f_p = (-r * p_inv % two_p + two_p) % two_p;
    
    // g(p) = f(p) mod p
    return f_p % p;
}

long long compute(int limit) {
    vector<bool> is_prime(limit + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= limit; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= limit; j += i) {
                is_prime[j] = false;
            }
        }
    }
    
    long long total = 0;
    for (int p = 3; p < limit; p += 2) {
        if (is_prime[p]) {
            total += g(p);
        }
    }
    
    return total;
}

int main() {
    cout << compute(10000000) << endl;
    return 0;
}
```
