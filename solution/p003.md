# Plus Grand Facteur Premier

## Description du Problème
Les facteurs premiers de 13195 sont 5, 7, 13 et 29.

**Objectif** : Quel est le plus grand facteur premier du nombre 600851475143 ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 13195`
**Sortie** : `29`
**Explication** : Les facteurs premiers de 13195 sont 5, 7, 13 et 29. Le plus grand est 29.

---

## Approche de Résolution

### Stratégie
Selon le théorème fondamental de l'arithmétique, tout entier n > 1 a une factorisation unique en produit de nombres premiers. On peut trouver le plus grand facteur premier en divisant récursivement le nombre par son plus petit facteur premier.

**Méthode :**
1. Trouver le plus petit facteur premier de n.
2. Diviser n par ce facteur.
3. Répéter jusqu'à ce que n soit lui-même premier (le plus petit facteur est n).
4. Le dernier facteur obtenu est le plus grand facteur premier.

### Observations
- Pour trouver le plus petit facteur, tester tous les nombres de 2 à √n.
- Le premier diviseur trouvé est nécessairement premier.
- Si aucun diviseur n'est trouvé, n est premier.
- On n'a pas besoin de tester au-delà de √n car si n a un facteur > √n, il a aussi un facteur < √n.

### Complexité
- **Temporelle** : O(√n), où `n` est le nombre à factoriser.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

long long smallestFactor(long long n) {
    if (n <= 1) return -1;
    long long end = sqrt(n);
    for (long long i = 2; i <= end; ++i) {
        if (n % i == 0) {
            return i;
        }
    }
    return n;  // n est premier
}

int main() {
    long long n = 600851475143LL;
    
    while (true) {
        long long p = smallestFactor(n);
        if (p < n) {
            n /= p;
        } else {
            std::cout << n << std::endl;
            break;
        }
    }
    
    return 0;
}
```
