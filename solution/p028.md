# Somme des Diagonales d'une Spirale de Nombres

## Description du Problème
En commençant par le nombre 1 et en se déplaçant vers la droite dans le sens des aiguilles d'une montre, une spirale 5×5 est formée comme suit :

```
21 22 23 24 25
20  7  8  9 10
19  6  1  2 11
18  5  4  3 12
17 16 15 14 13
```

On peut vérifier que la somme des nombres sur les diagonales est 101.

**Objectif** : Quelle est la somme des nombres sur les diagonales dans une spirale 1001×1001 formée de la même manière ?

---

## Exemples
### Exemple 1
**Entrée** : Taille = 5
**Sortie** : `101`
**Explication** : 1 + 3 + 5 + 7 + 9 + 13 + 17 + 21 + 25 = 101.

---

## Approche de Résolution

### Stratégie
Pour une spirale n×n (où n est impair), les quatre coins d'un anneau de taille k sont :
- Coin supérieur droit : k²
- Coin supérieur gauche : k² - (k - 1)
- Coin inférieur gauche : k² - 2(k - 1)
- Coin inférieur droit : k² - 3(k - 1)

La contribution d'un anneau de taille k est : 4k² - 6(k - 1)

1. Commencer avec la somme = 1 (pour le centre).
2. Pour chaque taille d'anneau k de 3 à n (par pas de 2) :
   - Ajouter 4k² - 6(k - 1) à la somme.
3. Retourner la somme.

### Observations
- La formule fermée pour la somme totale est : (4m³ + 3m² + 8m - 9) / 6, où m = n.
- On peut calculer directement sans générer toute la spirale.

### Complexité
- **Temporelle** : O(n), où `n` est la taille de la spirale.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>

int main() {
    const int size = 1001;
    long long sum = 1;  // Centre
    
    for (int n = 3; n <= size; n += 2) {
        sum += 4 * n * n - 6 * (n - 1);
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
