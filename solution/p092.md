# Square Digit Chains

## Description du Problème
Une chaîne de nombres est créée en ajoutant continuellement le carré des chiffres d'un nombre pour former un nouveau nombre jusqu'à ce qu'il ait été vu auparavant.

Par exemple,
- 44 → 32 → 13 → 10 → 1 → 1
- 85 → 89 → 145 → 42 → 20 → 4 → 16 → 37 → 58 → 89

Par conséquent, toute chaîne qui arrive à 1 ou 89 restera bloquée dans une boucle sans fin. Ce qui est le plus étonnant, c'est que TOUT nombre de départ finira par arriver à 1 ou 89.

**Objectif** : Combien de nombres de départ inférieurs à dix millions arriveront à 89 ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 44`
**Sortie** : `1` (arrive à 1)
**Explication** : 44 → 32 → 13 → 10 → 1 → 1

### Exemple 2
**Entrée** : `n = 85`
**Sortie** : `89` (arrive à 89)
**Explication** : 85 → 89 → 145 → 42 → 20 → 4 → 16 → 37 → 58 → 89

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre de 1 à 10⁷-1, suivre la chaîne jusqu'à arriver à 1 ou 89.
2. Calculer le prochain nombre en sommant les carrés des chiffres.
3. Compter combien de nombres arrivent à 89.

### Observations
- Tous les nombres finissent par arriver à 1 ou 89.
- On peut optimiser en mémorisant les résultats, mais pour 10⁷ nombres, une approche directe est suffisante.
- La fonction `nextNumber` calcule la somme des carrés des chiffres.

### Complexité
- **Temporelle** : O(n × log n), où n=10⁷, car pour chaque nombre on fait quelques itérations (généralement moins de 10).
- **Spatiale** : O(1), car on utilise seulement des variables pour suivre la chaîne.

---

## Solution en C++

```cpp
#include <iostream>

const int LIMIT = 10000000;

int nextNumber(int x) {
    int sum = 0;
    while (x != 0) {
        sum += (x % 10) * (x % 10);
        x /= 10;
    }
    return sum;
}

bool isClass89(int x) {
    while (true) {
        if (x == 1) return false;
        if (x == 89) return true;
        x = nextNumber(x);
    }
}

int main() {
    int count = 0;
    for (int i = 1; i < LIMIT; i++) {
        if (isClass89(i)) {
            count++;
        }
    }
    std::cout << count << std::endl;
    return 0;
}
```
