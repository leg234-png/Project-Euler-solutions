# Somme maximale de la matrice

## Description du Problème
Nous définissons la somme maximale de la matrice comme la somme maximale obtenue en sélectionnant un élément de chaque ligne de la matrice, où chaque colonne est utilisée au plus une fois.

**Objectif** : Trouver la somme maximale de la matrice donnée.

---

## Exemples
### Exemple 1
**Entrée** : Matrice 15×15
**Sortie** : `13938`
**Explication** : La somme maximale est obtenue en sélectionnant un élément de chaque ligne sans réutiliser de colonne.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique avec mémorisation (méthode Held-Karp).
2. Pour chaque ligne, essayer toutes les colonnes disponibles.
3. Utiliser un masque de bits pour représenter les colonnes déjà utilisées.
4. Mémoriser les résultats pour éviter les recalculs.

### Observations
- Au lieu de tester toutes les O(n!) permutations, on utilise la mémorisation pour obtenir O(n² × 2ⁿ).
- C'est le même principe que l'algorithme Held-Karp pour le problème du voyageur de commerce.
- On utilise un tableau de mémorisation `maxSum[row][mask]` où `mask` représente les colonnes disponibles.

### Complexité
- **Temporelle** : O(n² × 2ⁿ), où n est le nombre de colonnes (15).
- **Spatiale** : O(n × 2ⁿ), pour le tableau de mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <climits>

using namespace std;

const int ROWS = 15;
const int COLUMNS = 15;

int MATRIX[ROWS][COLUMNS] = {
    {  7,  53, 183, 439, 863, 497, 383, 563,  79, 973, 287,  63, 343, 169, 583},
    {627, 343, 773, 959, 943, 767, 473, 103, 699, 303, 957, 703, 583, 639, 913},
    {447, 283, 463,  29,  23, 487, 463, 993, 119, 883, 327, 493, 423, 159, 743},
    {217, 623,   3, 399, 853, 407, 103, 983,  89, 463, 290, 516, 212, 462, 350},
    {960, 376, 682, 962, 300, 780, 486, 502, 912, 800, 250, 346, 172, 812, 350},
    {870, 456, 192, 162, 593, 473, 915,  45, 989, 873, 823, 965, 425, 329, 803},
    {973, 965, 905, 919, 133, 673, 665, 235, 509, 613, 673, 815, 165, 992, 326},
    {322, 148, 972, 962, 286, 255, 941, 541, 265, 323, 925, 281, 601,  95, 973},
    {445, 721,  11, 525, 473,  65, 511, 164, 138, 672,  18, 428, 154, 448, 848},
    {414, 456, 310, 312, 798, 104, 566, 520, 302, 248, 694, 976, 430, 392, 198},
    {184, 829, 373, 181, 631, 101, 969, 613, 840, 740, 778, 458, 284, 760, 390},
    {821, 461, 843, 513,  17, 901, 711, 993, 293, 157, 274,  94, 192, 156, 574},
    { 34, 124,   4, 878, 450, 476, 712, 914, 838, 669, 875, 299, 823, 329, 699},
    {815, 559, 813, 459, 522, 788, 168, 586, 966, 232, 308, 833, 251, 631, 107},
    {813, 883, 451, 509, 615,  77, 281, 613, 459, 205, 380, 274, 302,  35, 805},
};

vector<vector<int>> maxSum;

// Retourne la somme maximale en considérant la sous-matrice de la ligne 'startRow' jusqu'en bas,
// avec le masque de bits 'setOfCols' indiquant quelles colonnes sont encore disponibles.
int findMaximumSum(int startRow, int setOfCols) {
    if (startRow == ROWS) {
        return 0;
    }
    
    if (maxSum[startRow][setOfCols] == 0) {
        int max = 0;
        for (int col = 0; col < COLUMNS; col++) {
            int bit = 1 << col;
            if ((setOfCols & bit) != 0) {
                max = std::max(MATRIX[startRow][col] + findMaximumSum(startRow + 1, setOfCols ^ bit), max);
            }
        }
        maxSum[startRow][setOfCols] = max;
    }
    return maxSum[startRow][setOfCols];
}

int main() {
    maxSum.assign(ROWS, vector<int>(1 << COLUMNS, 0));
    cout << findMaximumSum(0, (1 << COLUMNS) - 1) << endl;
    return 0;
}
```
