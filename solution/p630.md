# Lignes croisées

## Description du Problème
Soit S₀ = 290797.

S_{n+1} = Sₙ² mod 50515093.

Tₙ = (Sₙ mod 2000) - 1000.

Par exemple, les trois premiers points sont : (527, 144), (-488, 732), (-454, -947).

Étant donné les n premiers points générés de cette manière, soit Lₙ l'ensemble des lignes uniques qui peuvent être formées en joignant chaque point avec chaque autre point.

Soit M(n) le nombre de lignes dans Lₙ.

**Objectif** : Trouver M(2500). Donner votre réponse sous la forme (M, S) où M est le nombre de lignes et S est la somme des pentes de toutes les lignes uniques.

---

## Exemples
### Exemple 1
**Entrée** : `n = 3`
**Sortie** : `Nombre de lignes`
**Explication** : Le nombre de lignes uniques formées par 3 points.

---

## Approche de Résolution

### Stratégie
1. Générer tous les points en utilisant la récurrence donnée.
2. Pour chaque paire de points (x₁, y₁) et (x₂, y₂), créer la ligne ax + by + c = 0.
3. a = y₁ - y₂, b = x₂ - x₁, c = y₂x₁ - x₂y₁.
4. Normaliser la ligne en divisant par gcd(a, b) pour obtenir une représentation unique.
5. Assurer que a > 0, ou si a = 0 alors b > 0.
6. Stocker les lignes uniques dans un set.
7. Calculer M et S (somme des pentes).

### Observations
- On doit normaliser les lignes pour éviter les doublons.
- La pente d'une ligne verticale est infinie, donc on doit gérer ce cas spécialement.
- On peut utiliser un set pour stocker les lignes uniques.

### Complexité
- **Temporelle** : O(n² × log n) pour générer toutes les paires et stocker dans un set.
- **Spatiale** : O(n²) pour stocker les lignes uniques.

---

## Solution en C++

```cpp
#include <iostream>
#include <set>
#include <tuple>
#include <cmath>

using namespace std;

pair<long long, long long> normalize_line(long long a, long long b, long long c) {
    long long g = __gcd(abs(a), abs(b));
    if (g > 0) {
        a /= g;
        b /= g;
        c /= g;
    }
    
    if (a < 0 || (a == 0 && b < 0)) {
        a = -a;
        b = -b;
        c = -c;
    }
    
    return {a, b};
}

pair<long long, long long> compute(int n) {
    vector<long long> S(n);
    S[0] = 290797;
    for (int i = 1; i < n; i++) {
        S[i] = (S[i-1] * S[i-1]) % 50515093;
    }
    
    vector<pair<long long, long long>> points;
    for (int i = 0; i < n; i++) {
        long long x = (S[i] % 2000) - 1000;
        long long y = (S[(i+1) % n] % 2000) - 1000;
        points.push_back({x, y});
    }
    
    set<tuple<long long, long long, long long>> lines;
    long long slope_sum = 0;
    
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            long long x1 = points[i].first, y1 = points[i].second;
            long long x2 = points[j].first, y2 = points[j].second;
            
            long long a = y1 - y2;
            long long b = x2 - x1;
            long long c = y2 * x1 - x2 * y1;
            
            auto [norm_a, norm_b] = normalize_line(a, b, c);
            tuple<long long, long long, long long> line = {norm_a, norm_b, c};
            
            if (lines.find(line) == lines.end()) {
                lines.insert(line);
                if (norm_b != 0) {
                    slope_sum += (double)norm_a / norm_b;
                }
            }
        }
    }
    
    return {lines.size(), slope_sum};
}

int main() {
    auto [M, S] = compute(2500);
    cout << "(" << M << ", " << S << ")" << endl;
    return 0;
}
```
