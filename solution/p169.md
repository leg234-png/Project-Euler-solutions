# Exploring the number of different ways a number can be expressed as a sum of powers of 2

## Description du Problème
Définissons `f(0) = 1` et `f(n)` comme le nombre de façons d'exprimer `n` comme une somme de puissances de 2 où chaque puissance est utilisée au plus deux fois.

Par exemple, `f(10) = 5` car il y a 5 façons d'exprimer 10 :
- `8 + 2`
- `8 + 1 + 1`
- `4 + 4 + 2`
- `4 + 2 + 2 + 1 + 1`
- `4 + 4 + 1 + 1`

**Objectif** : Trouver `f(10^25)`.

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : `5`
**Explication** : Il y a 5 façons d'exprimer 10 comme somme de puissances de 2 (chaque puissance utilisée au plus 2 fois).

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique avec mémoïsation.
2. `countWays(number, exponent, repetitions)` : nombre de façons d'exprimer `number` comme somme de puissances de 2 où :
   - La plus grande puissance possible est `2^exponent`.
   - Le terme `2^exponent` est utilisé entre 0 et `repetitions` fois.
   - Toutes les puissances inférieures sont utilisées au plus 2 fois.
3. Cas de base :
   - Si `exponent < 0` : retourner 1 si `number == 0`, sinon 0.
4. Récursion :
   - Ne pas utiliser `2^exponent` : `countWays(number, exponent-1, 2)`.
   - Utiliser `2^exponent` une fois (si possible) : `countWays(number - 2^exponent, exponent, repetitions-1)`.
5. Utiliser `BigInteger` pour gérer les grands nombres.

### Observations
- Chaque puissance de 2 peut être utilisée 0, 1 ou 2 fois.
- On traite les puissances de la plus grande à la plus petite.
- La mémoïsation est cruciale pour éviter les recalculs.

### Complexité
- **Temporelle** : O(log n × log n) dans le pire cas.
- **Spatiale** : O(log n × log n) pour la mémoïsation.

---

## Solution en C++

```cpp
#include <iostream>
#include <map>
#include <vector>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

map<vector<cpp_int>, cpp_int> ways;

cpp_int countWays(cpp_int number, int exponent, int repetitions) {
    vector<cpp_int> key = {number, cpp_int(exponent), cpp_int(repetitions)};
    
    if (ways.count(key))
        return ways[key];
    
    cpp_int result;
    if (exponent < 0) {
        result = (number == 0) ? 1 : 0;
    } else {
        result = countWays(number, exponent - 1, 2);
        cpp_int pow = cpp_int(1) << exponent;
        cpp_int upper = pow * (repetitions + 2);
        if (repetitions > 0 && pow <= number && number < upper) {
            result += countWays(number - pow, exponent, repetitions - 1);
        }
    }
    
    ways[key] = result;
    return result;
}

int main() {
    cpp_int NUMBER = pow(cpp_int(10), 25);
    int bitLength = 0;
    cpp_int temp = NUMBER;
    while (temp > 0) {
        bitLength++;
        temp >>= 1;
    }
    
    cout << countWays(NUMBER, bitLength - 1, 2) << endl;
    return 0;
}
```
