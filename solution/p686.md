# Puissances de 2

## Description du Problème
2⁷ = 128 est la première puissance de deux dont les chiffres de tête sont "12".

La prochaine puissance de deux dont les chiffres de tête sont "12" est 2⁸⁰.

Soit p(L, n) la n-ième plus petite valeur de j telle que la représentation en base 10 de 2^j commence par les chiffres de L.

On vous donne aussi que p(123, 45) = 12710.

**Objectif** : Trouver p(123, 678910).

---

## Exemples
### Exemple 1
**Entrée** : `L = 12, n = 1`
**Sortie** : `7`
**Explication** : p(12, 1) = 7.

### Exemple 2
**Entrée** : `L = 12, n = 2`
**Sortie** : `80`
**Explication** : p(12, 2) = 80.

---

## Approche de Résolution

### Stratégie
1. On cherche 2^j qui commence par L.
2. 2^j = L... × 10^x => log₁₀(2^j) = log₁₀(L...) + x.
3. j × log₁₀(2) = log₁₀(L...) + x, où x est un entier.
4. La partie fractionnaire de j × log₁₀(2) doit être dans [log₁₀(L), log₁₀(L+1)).
5. Parcourir j et vérifier si la partie fractionnaire est dans cet intervalle.
6. Compter jusqu'à trouver le n-ième.

### Observations
- On peut utiliser les logarithmes pour éviter de calculer 2^j explicitement.
- La partie fractionnaire de j × log₁₀(2) suit une distribution uniforme.
- On peut prédire approximativement où se trouve le prochain j.

### Complexité
- **Temporelle** : O(n × log L) où n est la position recherchée.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long compute(int L, int n) {
    double lower = log10(L);
    double upper = log10(L + 1);
    double cons = log10(2.0);
    
    int count = 0;
    long long j = 0;
    
    while (count < n) {
        j++;
        double temp = j * cons;
        double fractional = temp - floor(temp);
        
        if (lower < fractional && fractional < upper) {
            count++;
        }
    }
    
    return j;
}

int main() {
    cout << compute(123, 678910) << endl;
    return 0;
}
```
