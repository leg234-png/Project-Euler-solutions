# Facteurs de 2 dans les coefficients binomiaux

## Description du Problème
Soit g(n, m) la plus grande puissance de 2 qui divise C(n, m).

Soit F(n) = max{g(n, m) | 0 ≤ m ≤ n}.

Soit S(N) = ∑_{n=1}^N F(n).

On vous donne que S(100) = 389 et S(10⁷) = 203222840.

**Objectif** : Trouver S(10¹⁶).

---

## Exemples
### Exemple 1
**Entrée** : `N = 100`
**Sortie** : `389`
**Explication** : S(100) = 389.

---

## Approche de Résolution

### Stratégie
1. Utiliser le théorème de Kummer : la plus grande puissance de 2 dans C(n, m) = S(m) + S(n-m) - S(n), où S(x) est la somme des chiffres de x en base 2.
2. F(n) = ⌊log₂(n+1)⌋ - v₂(n+1), où v₂(n+1) est la valuation 2-adique de n+1.
3. S(N) = ∑_{n=1}^N ⌊log₂(n+1)⌋ - ∑_{n=1}^N v₂(n+1).
4. ∑_{n=1}^N ⌊log₂(n+1)⌋ = (N+2)×⌊log₂(N+1)⌋ - 2×(2^⌊log₂(N+1)⌋ - 1).
5. ∑_{n=1}^N v₂(n+1) = N+1 - bit_count(N+1).

### Observations
- Il y a des formules directes pour calculer S(N).
- On peut utiliser les formules de somme partielle.
- Le calcul peut être fait en O(log N).

### Complexité
- **Temporelle** : O(log N) pour calculer S(N).
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

int bit_count(long long n) {
    int count = 0;
    while (n > 0) {
        count += n & 1;
        n >>= 1;
    }
    return count;
}

long long compute(long long N) {
    long long log2_N_plus_1 = (long long)log2(N + 1);
    long long two_power = 1LL << log2_N_plus_1;
    
    long long sum_log = (N + 2) * log2_N_plus_1 - 2 * (two_power - 1);
    long long sum_val = N + 1 - bit_count(N + 1);
    
    return sum_log - sum_val;
}

int main() {
    long long N = 1;
    for (int i = 0; i < 16; i++) {
        N *= 10;
    }
    cout << compute(N) << endl;
    return 0;
}
```
