# Somme de chiffres inverse

## Description du Problème
Soit s(n) le plus petit nombre tel que la somme de ses chiffres en base 10 est n.

Par exemple, s(10) = 19 car 1+9 = 10 et 19 est le plus petit nombre avec cette propriété.

Soit S(k) = ∑_{n=1}^k s(n).

Soit f_i la séquence de Fibonacci définie par f₀ = 0, f₁ = 1 et f_i = f_{i-2} + f_{i-1} pour tout i ≥ 2.

**Objectif** : Trouver ∑_{i=2}^{90} S(f_i). Donner votre réponse modulo 1000000007.

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : `19`
**Explication** : s(10) = 19.

---

## Approche de Résolution

### Stratégie
1. s(n) = ((n mod 9 + 1) × 10^(n/9)) - 1 si n mod 9 ≠ 0, sinon s(n) = 10^(n/9) - 1.
2. S(k) peut être calculé en utilisant une formule pour la somme des s(n).
3. S(k) = ∑_{n=1}^k s(n) = (somme des termes de la série géométrique) mod 1000000007.
4. Calculer S(f_i) pour chaque i de 2 à 90.
5. Sommer tous les résultats.

### Observations
- Il y a une formule directe pour s(n).
- On peut calculer S(k) efficacement sans itérer sur tous les n.
- Les nombres de Fibonacci peuvent être calculés rapidement.

### Complexité
- **Temporelle** : O(90 × log f_i) pour calculer tous les S(f_i).
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long mod = 1000000007;

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long s(long long n, long long mod) {
    long long q = n / 9;
    long long r = n % 9;
    if (r == 0) {
        return (mod_pow(10, q, mod) - 1 + mod) % mod;
    } else {
        return ((r + 1) * mod_pow(10, q, mod) - 1 + mod) % mod;
    }
}

long long S(long long k, long long mod) {
    long long q = k / 9;
    long long r = k % 9;
    
    long long sum1 = (6 * (mod_pow(10, q, mod) - 1) - 9 * q + mod) % mod;
    long long sum2 = 0;
    for (int i = 2; i <= r + 1; i++) {
        sum2 = (sum2 + i * mod_pow(10, q, mod) - 1 + mod) % mod;
    }
    
    return (sum1 + sum2) % mod;
}

long long fibonacci(int n) {
    if (n == 0) return 0;
    if (n == 1) return 1;
    long long a = 0, b = 1;
    for (int i = 2; i <= n; i++) {
        long long temp = a + b;
        a = b;
        b = temp;
    }
    return b;
}

long long compute() {
    long long total = 0;
    for (int i = 2; i <= 90; i++) {
        long long f_i = fibonacci(i);
        total = (total + S(f_i, mod)) % mod;
    }
    return total;
}

int main() {
    cout << compute() << endl;
    return 0;
}
```
