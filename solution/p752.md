# Puissances de 1 + √7

## Description du Problème
Soit (1 + √7)^n = a(n) + b(n)√7.

Soit g(x) le plus petit entier positif n tel que a(n) ≡ 1 (mod x) et b(n) ≡ 0 (mod x), ou 0 s'il n'existe pas une telle valeur de n.

Par exemple, g(3) = 0, g(5) = 12.

Définissons aussi G(N) = ∑_{x=2}^N g(x).

On vous donne G(10²) = 28891 et G(10³) = 13131583.

**Objectif** : Trouver G(10⁶).

---

## Exemples
### Exemple 1
**Entrée** : `x = 5`
**Sortie** : `12`
**Explication** : g(5) = 12.

---

## Approche de Résolution

### Stratégie
1. (1 + √7)^n = ∑_{k=0}^n C(n,k) √7^(n-k).
2. (1 + √7)^2 = 8 + 2√7.
3. a(n) = ∑_{n-k est pair} C(n,k) 7^((n-k)/2).
4. b(n) = ∑_{n-k est impair} C(n,k) 7^((n-k-1)/2).
5. (1 + √7)^(n+1) = (a(n) + b(n)√7)(1 + √7) = (a(n) + 7b(n)) + (a(n) + b(n))√7.
6. => a(n+1) = a(n) + 7b(n), b(n+1) = a(n) + b(n).
7. On veut n tel que a(n) ≡ 1 (mod x) et b(n) ≡ 0 (mod x).
8. Utiliser la récurrence pour calculer a(n) et b(n) modulo x.
9. Chercher le plus petit n qui satisfait les conditions.

### Observations
- Seuls les nombres de la forme 6k ± 1 peuvent avoir une solution.
- On peut utiliser l'exponentiation rapide pour calculer efficacement.
- Il y a des patterns dans les valeurs de g(x).

### Complexité
- **Temporelle** : O(N × x_max) où x_max est la valeur maximale à tester.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long g(long long x) {
    // Vérifier si x = 6k ± 1
    if (x % 6 != 1 && x % 6 != 5) {
        return 0;
    }
    
    long long a_prev = 1, b_prev = 0;
    long long a_curr = 1, b_curr = 1;
    
    for (long long n = 1; n <= x * x; n++) {
        if (a_curr % x == 1 && b_curr % x == 0) {
            return n;
        }
        
        long long a_next = (a_curr + 7 * b_curr) % x;
        long long b_next = (a_curr + b_curr) % x;
        a_prev = a_curr;
        b_prev = b_curr;
        a_curr = a_next;
        b_curr = b_next;
    }
    
    return 0;
}

long long compute(long long N) {
    long long total = 0;
    for (long long x = 2; x <= N; x++) {
        total += g(x);
    }
    return total;
}

int main() {
    cout << compute(1000000) << endl;
    return 0;
}
```
