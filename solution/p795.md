# Somme GCD alternée

## Description du Problème
Soit g(n) = ∑_{i=1}^n (-1)^i gcd(n, i²).

On vous donne aussi g(1234) = 1233.

Soit G(N) = ∑_{n=1}^N g(n). On vous donne G(1234) = 2194708.

**Objectif** : Trouver G(12345678).

---

## Exemples
### Exemple 1
**Entrée** : `n = 1234`
**Sortie** : `1233`
**Explication** : g(1234) = 1233.

---

## Approche de Résolution

### Stratégie
1. g(n) = ∑_{i=1}^n (-1)^i gcd(n, i²).
2. Utiliser la propriété que gcd(n, i²) = gcd(n, i)² / gcd(n, i) pour certains cas.
3. Grouper par diviseurs de n.
4. Pour chaque diviseur d de n, compter combien de i ont gcd(n, i) = d.
5. Utiliser la fonction de Möbius pour compter efficacement.
6. Calculer g(n) pour chaque n et sommer.

### Observations
- On peut utiliser un crible pour trouver tous les diviseurs.
- La fonction de Möbius peut être utilisée pour compter.
- On peut optimiser en utilisant des techniques de sommation.

### Complexité
- **Temporelle** : O(N × log N) pour calculer tous les g(n).
- **Spatiale** : O(N) pour le crible.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<int> spf_sieve(int N) {
    vector<int> spf(N + 1);
    for (int i = 0; i <= N; i++) {
        spf[i] = i;
    }
    
    for (int i = 2; i * i <= N; i++) {
        if (spf[i] == i) {
            for (int j = i * i; j <= N; j += i) {
                if (spf[j] == j) {
                    spf[j] = i;
                }
            }
        }
    }
    
    return spf;
}

long long g(int n, const vector<int>& spf) {
    long long total = 0;
    for (int i = 1; i <= n; i++) {
        int sign = (i % 2 == 0) ? -1 : 1;
        int gcd_val = __gcd(n, i * i);
        total += sign * gcd_val;
    }
    return total;
}

long long compute(int N) {
    vector<int> spf = spf_sieve(N);
    long long total = 0;
    
    for (int n = 1; n <= N; n++) {
        total += g(n, spf);
    }
    
    return total;
}

int main() {
    cout << compute(12345678) << endl;
    return 0;
}
```
