# Une cuillère de blancmange

## Description du Problème
Soit C le cercle de centre (1/4, 1/2) et de rayon 1/4, montré en noir dans le diagramme.

**Objectif** : Quelle aire sous la courbe de blancmange est enfermée par C ? Donner votre réponse arrondie à huit décimales sous la forme 0.abcdefgh.

---

## Exemples
### Exemple 1
**Entrée** : `Cercle de centre (1/4, 1/2) et rayon 1/4`
**Sortie** : `0.11316017`
**Explication** : L'aire sous la courbe de blancmange enfermée par le cercle.

---

## Approche de Résolution

### Stratégie
1. La courbe de blancmange satisfait : blanc(x) = blanc(2x)/2 + s(x).
2. L'intégrale I(x) = ∫₀ˣ blanc(y) dy satisfait :
   - I(x) = I(2x)/4 + x²/2 si 0 < x ≤ 1/2
   - I(x) = 1/2 - I(1 - x) si 1/2 < x ≤ 1
   - I(x) = n/2 + I(x - n) si n ≤ x ≤ n + 1
3. Le cercle est (x - 1/4)² + (y - 1/2)² = 1/16.
4. La moitié inférieure est y = (1 - √(2x - 4x²))/2.
5. Trouver l'intersection et calculer l'aire entre la courbe et le cercle.

### Observations
- L'intersection se trouve à environ 0.0789 et 0.5.
- On peut utiliser une intégration numérique pour calculer l'aire sous le cercle.
- La fonction I(x) peut être calculée récursivement.

### Complexité
- **Temporelle** : O(log n) pour le calcul récursif de I(x), où n est la précision.
- **Spatiale** : O(log n) pour la pile de récursion.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>
#include <iomanip>

using namespace std;

double I(double x) {
    if (x == 1.0) {
        return 0.5;
    }
    
    if (0 < x && x <= 0.5) {
        return I(2*x) / 4.0 + (x*x) / 2.0;
    }
    
    if (0.5 < x && x <= 1.0) {
        return 0.5 - I(1.0 - x);
    }
    
    long long n = (long long)x;
    return n / 2.0 + I(x - n);
}

double f(double x) {
    return (1.0 - sqrt(2*x - 4*x*x)) / 2.0;
}

double compute() {
    double area_under_blancmange = I(0.5) - I(0.0789);
    
    // Intégration numérique de f(x) de 0.0789 à 0.5
    double area_under_circle = 0.0;
    int steps = 10000;
    double dx = (0.5 - 0.0789) / steps;
    for (int i = 0; i < steps; i++) {
        double x = 0.0789 + i * dx;
        area_under_circle += f(x) * dx;
    }
    
    return round((area_under_blancmange - area_under_circle) * 100000000) / 100000000.0;
}

int main() {
    cout << fixed << setprecision(8) << compute() << endl;
    return 0;
}
```
