# Dice Game

## Description du Problème
Peter a neuf dés pyramidaux à 4 faces (numérotés 1, 2, 3, 4).
Colin a six dés cubiques à 6 faces (numérotés 1, 2, 3, 4, 5, 6).

Peter et Colin lancent leurs dés et comparent les totaux : le plus grand total gagne. En cas d'égalité, c'est un match nul.

**Objectif** : Quelle est la probabilité que Peter gagne ? Donner la réponse arrondie à 7 décimales.

---

## Exemples
### Exemple 1
**Entrée** : 1 dé pyramidal vs 1 dé cubique
**Sortie** : Probabilité que le dé pyramidal gagne
**Explication** : On calcule la distribution de probabilité pour chaque combinaison.

---

## Approche de Résolution

### Stratégie
1. Calculer la distribution de probabilité (PDF) pour la somme de 9 dés pyramidaux :
   - PDF d'un dé pyramidal : [0, 1, 1, 1, 1] (valeurs 1-4, chacune avec probabilité 1/4).
   - Convoluer 9 fois pour obtenir la PDF de la somme.
2. Calculer la PDF pour la somme de 6 dés cubiques :
   - PDF d'un dé cubique : [0, 1, 1, 1, 1, 1, 1] (valeurs 1-6, chacune avec probabilité 1/6).
   - Convoluer 6 fois pour obtenir la PDF de la somme.
3. Pour chaque somme possible `i` de Peter :
   - Probabilité que Peter obtienne `i` × Probabilité que Colin obtienne < `i`.
4. Additionner toutes ces probabilités.

### Observations
- La convolution de PDFs donne la PDF de la somme de variables aléatoires indépendantes.
- On calcule la probabilité que Peter gagne en sommant P(Peter=i) × P(Colin<i) pour tous les i.

### Complexité
- **Temporelle : O(S1² + S2²)**, où S1 et S2 sont les sommes maximales possibles.
- **Spatiale : O(S1 + S2)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <iomanip>

using namespace std;

vector<int> convolve(const vector<int>& a, const vector<int>& b) {
    vector<int> c(a.size() + b.size() - 1, 0);
    for (size_t i = 0; i < a.size(); i++) {
        for (size_t j = 0; j < b.size(); j++)
            c[i + j] += a[i] * b[j];
    }
    return c;
}

int sum(const vector<int>& array, int start, int end) {
    int s = 0;
    for (int i = start; i < end; i++)
        s += array[i];
    return s;
}

int main() {
    vector<int> pyramidalDie = {0, 1, 1, 1, 1};
    vector<int> cubicDie = {0, 1, 1, 1, 1, 1, 1};
    
    vector<int> ninePyramidalPdf = {1};
    for (int i = 0; i < 9; i++)
        ninePyramidalPdf = convolve(ninePyramidalPdf, pyramidalDie);
    
    vector<int> sixCubicPdf = {1};
    for (int i = 0; i < 6; i++)
        sixCubicPdf = convolve(sixCubicPdf, cubicDie);
    
    long long numer = 0;
    for (size_t i = 0; i < ninePyramidalPdf.size(); i++)
        numer += (long long)ninePyramidalPdf[i] * sum(sixCubicPdf, 0, i);
    
    long long denom = (long long)sum(ninePyramidalPdf, 0, ninePyramidalPdf.size()) * 
                      sum(sixCubicPdf, 0, sixCubicPdf.size());
    
    cout << fixed << setprecision(7) << (double)numer / denom << endl;
    return 0;
}
```
