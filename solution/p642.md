# Somme des plus grands facteurs premiers

## Description du Problème
Soit F(n) la somme des plus grands facteurs premiers de tous les nombres positifs ≤ n.

Par exemple, F(10) = 32, F(100) = 1915 et F(10000) = 10118280.

**Objectif** : Trouver F(201820182018). Donner votre réponse modulo 10⁹.

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : `32`
**Explication** : F(10) = 32.

---

## Approche de Résolution

### Stratégie
1. Utiliser un crible pour trouver le plus grand facteur premier de chaque nombre.
2. Pour chaque nombre premier p, tous les multiples de p qui n'ont pas de facteur premier plus grand contribuent p.
3. Utiliser une approche de type "sieve" pour calculer efficacement.
4. Pour de très grandes valeurs, utiliser une méthode de sommation de Dirichlet.
5. Utiliser la formule : F(n) = ∑_{p premier} p × (nombre de multiples de p sans facteur premier > p).

### Observations
- On peut utiliser un crible modifié pour trouver les plus grands facteurs premiers.
- Pour de très grandes valeurs, on doit utiliser des techniques avancées.
- La méthode de sommation de Dirichlet permet de calculer efficacement.

### Complexité
- **Temporelle** : O(n log log n) avec un crible standard, ou O(n^(2/3)) avec des techniques avancées.
- **Spatiale** : O(n) pour le crible.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long compute(long long n, long long mod) {
    long long sqrt_n = sqrt(n);
    vector<long long> largest_prime(n + 1, 0);
    
    for (long long i = 2; i <= n; i++) {
        if (largest_prime[i] == 0) {
            for (long long j = i; j <= n; j += i) {
                largest_prime[j] = i;
            }
        }
    }
    
    long long total = 0;
    for (long long i = 2; i <= n; i++) {
        total = (total + largest_prime[i]) % mod;
    }
    
    return total;
}

int main() {
    long long mod = 1000000000;
    cout << compute(201820182018LL, mod) << endl;
    return 0;
}
```
