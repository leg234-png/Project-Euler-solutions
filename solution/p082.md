# Somme de chemin : trois directions

## Description du Problème
La somme minimale de chemin dans la matrice 5 par 5 ci-dessous, en commençant dans n'importe quelle cellule de la colonne de gauche et en finissant dans n'importe quelle cellule de la colonne de droite, et en se déplaçant uniquement vers le haut, le bas et la droite, est indiquée en rouge et en gras ; la somme est égale à 994.

**Objectif** : Trouver la somme minimale de chemin de la colonne de gauche à la colonne de droite dans matrix.txt, un fichier texte de 31K contenant une matrice de 80 par 80.

**NOTE** : Ce problème est une version plus difficile du Problème 81.

---

## Exemples
### Exemple 1
**Entrée** : Matrice 5×5
**Sortie** : `994`
**Explication** : Le chemin optimal commence dans la colonne de gauche et se termine dans la colonne de droite.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique en traitant les colonnes de gauche à droite.
2. Pour chaque colonne, calculer la somme minimale pour chaque cellule en considérant :
   - Le chemin depuis la colonne précédente (gauche)
   - Le chemin depuis la cellule du dessus (haut)
   - Le chemin depuis la cellule du dessous (bas)
3. Faire deux passes : une de haut en bas, puis une de bas en haut pour gérer les mouvements verticaux.
4. Trouver le minimum dans la colonne de droite.

### Observations
- On peut se déplacer vers la gauche, la droite, le haut et le bas.
- On traite les colonnes séquentiellement de gauche à droite.
- Pour chaque colonne, on fait deux passes pour gérer les mouvements verticaux dans les deux directions.

### Complexité
- **Temporelle** : O(n²), où n est la taille de la matrice (80).
- **Spatiale** : O(n²), pour stocker les distances.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <fstream>
#include <sstream>
#include <climits>
#include <algorithm>

using namespace std;

const int INFINITY = INT_MAX / 2;

int main() {
    // Lire la matrice depuis le fichier
    vector<vector<int>> GRID;
    ifstream file("matrix.txt");
    string line;
    
    while (getline(file, line)) {
        vector<int> row;
        istringstream iss(line);
        string num;
        while (getline(iss, num, ',')) {
            row.push_back(stoi(num));
        }
        GRID.push_back(row);
    }
    file.close();
    
    int h = GRID.size();
    int w = GRID[0].size();
    vector<vector<int>> distance(h, vector<int>(w));
    
    // Programmation dynamique
    for (int x = 0; x < w; x++) {
        for (int y = 0; y < h; y++) {
            int left = (x > 0) ? distance[y][x - 1] : 0;
            int up = (y > 0) ? distance[y - 1][x] : INFINITY;
            distance[y][x] = GRID[y][x] + min(left, up);
        }
        for (int y = h - 1; y >= 0; y--) {
            int down = (y + 1 < h) ? distance[y + 1][x] : INFINITY;
            distance[y][x] = min(GRID[y][x] + down, distance[y][x]);
        }
    }
    
    int min = INFINITY;
    for (int y = 0; y < h; y++)
        min = std::min(distance[y][w - 1], min);
    
    cout << min << endl;
    return 0;
}
```
