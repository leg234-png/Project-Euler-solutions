# Hexagonal Tile Differences

## Description du Problème
Une tuile hexagonale avec le numéro 1 est entourée d'un anneau de six tuiles hexagonales, commençant à "12 heures" et numérotant les tuiles 2 à 7 dans le sens antihoraire.

De nouveaux anneaux sont ajoutés de la même manière, les anneaux suivants étant numérotés 8 à 19, 20 à 37, 38 à 61, et ainsi de suite.

En trouvant la différence entre la tuile n et chacune de ses six voisines, nous définirons PD(n) comme étant le nombre de ces différences qui sont premières.

Par exemple, en travaillant dans le sens horaire autour de la tuile 8, les différences sont 12, 29, 11, 6, 1 et 13. Donc PD(8)=3.

De la même manière, les différences autour de la tuile 17 sont 1, 17, 16, 1, 11 et 10, donc PD(17)=2.

On peut montrer que la valeur maximale de PD(n) est 3.

Si toutes les tuiles pour lesquelles PD(n)=3 sont listées par ordre croissant pour former une séquence, la 10ème tuile serait 271.

**Objectif** : Trouver la 2000ème tuile dans cette séquence.

---

## Exemples
### Exemple 1
**Entrée** : `n = 8`
**Sortie** : `3`
**Explication** : PD(8) = 3, car 3 des 6 différences sont premières.

---

## Approche de Résolution

### Stratégie
1. Analyser mathématiquement la structure de la spirale hexagonale.
2. Seules certaines tuiles peuvent avoir PD(n) = 3 :
   - Les tuiles de coin dans la direction vers le haut
   - Les tuiles de bord immédiatement à droite de la colonne verticale vers le haut
3. Pour les tuiles de coin : vérifier si {6k-1, 6k+1, 12k+5} sont premiers.
4. Pour les tuiles de bord : vérifier si {6k-1, 6k+5, 12k-7} sont premiers.
5. Compter jusqu'à trouver la 2000ème tuile.

### Observations
- La plupart des tuiles ont PD(n) ≤ 2.
- Seules les tuiles spéciales mentionnées peuvent avoir PD(n) = 3.
- On peut calculer directement les numéros de tuiles à partir du numéro d'anneau.

### Complexité
- **Temporelle** : O(k), où k est le numéro d'anneau nécessaire pour trouver 2000 tuiles.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

bool isPrime(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    for (int i = 3; i * i <= n; i += 2) {
        if (n % i == 0) return false;
    }
    return true;
}

int main() {
    const int TARGET = 2000;
    int count = 2;  // n = 1 et 2 satisfont PD(n) = 3
    
    for (int ring = 2; ; ring++) {
        // Test pour les tuiles de coin
        if (isPrime(ring * 6 - 1) && 
            isPrime(ring * 6 + 1) && 
            isPrime(ring * 12 + 5)) {
            count++;
            if (count == TARGET) {
                long long n = (long long)ring * (ring - 1) * 3 + 2;
                std::cout << n << std::endl;
                return 0;
            }
        }
        
        // Test pour les tuiles de bord
        if (isPrime(ring * 6 - 1) && 
            isPrime(ring * 6 + 5) && 
            isPrime(ring * 12 - 7)) {
            count++;
            if (count == TARGET) {
                long long n = (long long)ring * (ring + 1) * 3 + 1;
                std::cout << n << std::endl;
                return 0;
            }
        }
    }
    return 0;
}
```
