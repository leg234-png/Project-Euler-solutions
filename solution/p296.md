# Bissectrice angulaire et tangente

## Description du Problème
Combien de triangles ABC avec un périmètre ne dépassant pas 100000 existent tels que BE a une longueur entière ?

---

## Exemples
### Exemple 1
**Entrée** : `périmètre ≤ 100000`
**Sortie** : `Nombre de triangles`
**Explication** : Le nombre de triangles où BE a une longueur entière.

---

## Approche de Résolution

### Stratégie
1. Les triangles BCE et ACD sont similaires, et DBE est un triangle isocèle, donc BE = BD.
2. Utilisant le fait que k est une bissectrice angulaire de ACB, on a AD/BD = AC/BC => BD = AD × BC / AC.
3. AB = AD + BD, donc BD × AC = AB × BC - BD × BC.
4. BD × (AC + BC) = AB × BC => BE = BD = AB × BC / (AC + BC).
5. Cette valeur doit être un entier, donc on a besoin de tous les triangles tels que :
   - BC ≤ AC ≤ AB, BC + AC > AB
   - BC + AC + AB ≤ 100000
   - AC + BC divise AB × BC
6. Optimiser en utilisant gcd(AC, BC) = d.

### Observations
- Soit d = gcd(AC, BC) => AC = d × AC', BC = d × BC' où gcd(AC', BC') = 1.
- Alors BE = AB × BC' / (AC' + BC').
- Clairement AC' + BC' ne peut pas diviser BC', donc AC' + BC' | AB.
- On peut utiliser cette propriété pour optimiser la recherche.

### Complexité
- **Temporelle** : O(limit²) pour la recherche brute, mais optimisable avec la propriété gcd.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long compute_optimized(long long limit) {
    long long count = 0;
    
    for (long long BC = 1; BC <= limit / 3; BC++) {
        for (long long AC = BC; AC <= (limit - BC) / 2; AC++) {
            long long d = __gcd(BC, AC);
            long long BC_prime = BC / d;
            long long AC_prime = AC / d;
            long long divisor = AC_prime + BC_prime;
            
            // AC' + BC' doit diviser AB, et AB = AC + BC - reste
            // On cherche AB tel que divisor | AB et BC ≤ AC ≤ AB
            long long min_AB = AC;
            long long max_AB = min(BC + AC - 1, limit - BC - AC);
            
            // Trouver le premier AB >= min_AB qui est divisible par divisor
            long long start_AB = ((min_AB + divisor - 1) / divisor) * divisor;
            
            for (long long AB = start_AB; AB <= max_AB; AB += divisor) {
                if (AB * BC % (AC + BC) == 0) {
                    count++;
                }
            }
        }
    }
    
    return count;
}

int main() {
    cout << compute_optimized(100000) << endl;
    return 0;
}
```
