# Somme des carrés des diviseurs unitaires

## Description du Problème
Un diviseur unitaire d'un nombre n est un diviseur d de n tel que gcd(d, n/d) = 1.

Par exemple, les diviseurs unitaires de 4! = 24 sont 1, 3, 8 et 24.

Soit S(n) la somme des carrés des diviseurs unitaires de n. Par exemple, S(4!) = 1² + 3² + 8² + 24² = 650.

**Objectif** : Trouver S(10⁸!) modulo 1000000009.

---

## Exemples
### Exemple 1
**Entrée** : `n = 4`
**Sortie** : `650`
**Explication** : S(4!) = 1² + 3² + 8² + 24² = 1 + 9 + 64 + 576 = 650.

---

## Approche de Résolution

### Stratégie
1. Si n est factorisé comme p1^k1 × p2^k2 × ... × p_m^k_m, alors les diviseurs unitaires de n sont exactement tous les sous-ensembles de puissances maximales de nombres premiers.
2. Pour n!, sa factorisation en nombres premiers utilise tous les nombres premiers de 1 à n.
3. Pour chaque nombre premier p, n! a exactement floor(n/p) + floor(n/p²) + floor(n/p³) + ... facteurs de p.
4. Utiliser la programmation dynamique : si on a trouvé tous les diviseurs unitaires qui sont des produits de puissances maximales de nombres premiers moins que p_i^k_i, alors quand on inclut p_i^k_i, on double la taille de l'ensemble.
5. Pour chaque nombre premier p avec exposant k dans n!, la contribution est 1 + p^(2k) mod MODULUS.

### Observations
- Les diviseurs unitaires correspondent aux sous-ensembles de puissances maximales de nombres premiers.
- On peut calculer directement le produit (1 + p^(2k)) pour tous les nombres premiers p.

### Complexité
- **Temporelle** : O(P × log n), où P est le nombre de nombres premiers jusqu'à n.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

const int LIMIT = 100000000;
const long long MODULUS = 1000000009LL;

vector<int> listPrimes(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

// Retourne le nombre de facteurs de p (premier) dans factorielle(n)
int countFactors(int n, int p) {
    if (n == 0) {
        return 0;
    } else {
        return n / p + countFactors(n / p, p);
    }
}

long long powMod(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        base = (base * base) % mod;
        exp /= 2;
    }
    return result;
}

int main() {
    vector<int> primes = listPrimes(LIMIT);
    long long sum = 1;  // Dans ce calcul, c'est en fait un produit
    
    for (int p : primes) {
        int power = countFactors(LIMIT, p);
        sum = sum * (1 + powMod(p, power * 2, MODULUS)) % MODULUS;
    }
    
    cout << sum << endl;
    return 0;
}
```
