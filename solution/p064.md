# Périodes impaires des racines carrées

## Description du Problème
Toutes les racines carrées sont périodiques lorsqu'écrites sous forme de fractions continues et peuvent être écrites sous la forme :

√N = a₀ + 1/(a₁ + 1/(a₂ + 1/(a₃ + ...)))

Par exemple, pour √23 :
√23 = [4;(1,3,1,8)], où le bloc (1,3,1,8) se répète indéfiniment.

Les dix premières représentations en fractions continues de racines carrées (irrationnelles) sont :
- √2 = [1;(2)], période = 1
- √3 = [1;(1,2)], période = 2
- √5 = [2;(4)], période = 1
- √6 = [2;(2,4)], période = 2
- √7 = [2;(1,1,1,4)], période = 4
- √8 = [2;(1,4)], période = 2
- √10 = [3;(6)], période = 1
- √11 = [3;(3,6)], période = 2
- √12 = [3;(2,6)], période = 2
- √13 = [3;(1,1,1,1,6)], période = 5

Exactement quatre fractions continues, pour N ≤ 13, ont une période impaire.

**Objectif** : Combien de fractions continues pour N ≤ 10000 ont une période impaire ?

---

## Exemples
### Exemple 1
**Entrée** : N ≤ 13
**Sortie** : `4`
**Explication** : √2, √5, √10, et √13 ont des périodes impaires (1, 1, 1, 5 respectivement).

---

## Approche de Résolution

### Stratégie
1. Pour chaque N de 1 à 10000, si N n'est pas un carré parfait, calculer la période de la fraction continue de √N.
2. Utiliser une classe QuadraticSurd pour représenter les nombres de la forme (a + b√d)/c.
3. Calculer les termes de la fraction continue jusqu'à détecter une répétition.
4. Compter combien ont une période impaire.

### Observations
- On utilise une classe QuadraticSurd pour manipuler les nombres quadratiques.
- La période est détectée lorsqu'on retrouve une valeur QuadraticSurd déjà vue.
- On simplifie les fractions à chaque étape pour éviter les nombres trop grands.

### Complexité
- **Temporelle** : O(N × P), où N est la limite (10000) et P est la période moyenne.
- **Spatiale** : O(P), pour stocker les valeurs vues dans la période.

---

## Solution en C++

```cpp
#include <iostream>
#include <map>
#include <vector>
#include <gmpxx.h>
#include <cmath>

using namespace std;

class QuadraticSurd {
public:
    mpz_class a, b, c, d;
    
    QuadraticSurd(mpz_class a, mpz_class b, mpz_class c, mpz_class d) {
        if (c == 0)
            throw invalid_argument("c cannot be zero");
        
        // Simplifier
        if (c < 0) {
            a = -a;
            b = -b;
            c = -c;
        }
        mpz_class gcd_val = gcd(gcd(a, b), c);
        if (gcd_val != 1) {
            a /= gcd_val;
            b /= gcd_val;
            c /= gcd_val;
        }
        
        this->a = a;
        this->b = b;
        this->c = c;
        this->d = d;
    }
    
    QuadraticSurd subtract(QuadraticSurd other) {
        if (d != other.d)
            throw invalid_argument("d must match");
        return QuadraticSurd(
            a * other.c - other.a * c,
            b * other.c - other.b * c,
            c * other.c,
            d
        );
    }
    
    QuadraticSurd reciprocal() {
        return QuadraticSurd(
            -a * c,
            b * c,
            b * b * d - a * a,
            d
        );
    }
    
    mpz_class floor() {
        mpz_class temp = sqrt(b * b * d);
        if (b < 0)
            temp = -(temp + 1);
        temp += a;
        if (temp < 0)
            temp -= (c - 1);
        return temp / c;
    }
    
    bool operator<(const QuadraticSurd& other) const {
        if (a != other.a) return a < other.a;
        if (b != other.b) return b < other.b;
        if (c != other.c) return c < other.c;
        return d < other.d;
    }
    
private:
    mpz_class gcd(mpz_class x, mpz_class y) {
        while (y != 0) {
            mpz_class temp = y;
            y = x % y;
            x = temp;
        }
        return x;
    }
};

int getSqrtContinuedFractionPeriod(int n) {
    map<QuadraticSurd, int> seen;
    QuadraticSurd val(0, 1, 1, n);
    do {
        seen[val] = seen.size();
        mpz_class flr = val.floor();
        val = val.subtract(QuadraticSurd(flr, 0, 1, val.d)).reciprocal();
    } while (seen.find(val) == seen.end());
    return seen.size() - seen[val];
}

int main() {
    int count = 0;
    for (int i = 1; i <= 10000; i++) {
        int sqrt_i = (int)sqrt(i);
        if (sqrt_i * sqrt_i != i) {
            if (getSqrtContinuedFractionPeriod(i) % 2 == 1)
                count++;
        }
    }
    cout << count << endl;
    return 0;
}
```
