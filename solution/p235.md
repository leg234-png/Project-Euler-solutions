# Une suite arithmético-géométrique

## Description du Problème
Étant donnée la suite arithmético-géométrique u(k) = (900-3k)r^(k-1).

Soit s(n) = Σₖ₌₁ⁿ u(k).

**Objectif** : Trouver la valeur de r pour laquelle s(5000) = -600,000,000,000.

Donner votre réponse arrondie à 12 décimales après la virgule.

---

## Exemples
### Exemple 1
**Entrée** : `n = 5000, goal = -600000000000`
**Sortie** : `1.002322108633`
**Explication** : La valeur de r qui satisfait s(5000) = -600,000,000,000.

---

## Approche de Résolution

### Stratégie
1. Plus r est grand, plus s(5000) devient négatif.
2. Si on atteint un r qui donne un s(5000) plus petit (plus négatif), on a besoin d'un r plus petit.
3. Utiliser une recherche binaire ou une recherche par chiffres pour trouver r.
4. Construire r chiffre par chiffre en testant chaque chiffre de 0 à 9.

### Observations
- La fonction s(n, r) est décroissante en r (pour r > 1).
- On peut utiliser une recherche binaire pour trouver r efficacement.
- Alternativement, on peut construire r chiffre par chiffre.

### Complexité
- **Temporelle** : O(n × d) où n = 5000 et d est le nombre de décimales, pour la recherche par chiffres.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <cmath>
#include <string>

using namespace std;

double u(int k, double r) {
    return (900 - 3*k) * pow(r, k - 1);
}

double s(int n, double r) {
    double total = 0;
    for (int k = 1; k <= n; k++) {
        total += u(k, r);
    }
    return total;
}

double compute() {
    string r_str = "1.";
    double goal = -600000000000.0;
    
    while (r_str.length() < 16) {
        bool found = false;
        for (int x = 0; x <= 9; x++) {
            double r_test = stod(r_str + to_string(x));
            if (s(5000, r_test) < goal) {
                if (x > 0) {
                    r_str += to_string(x - 1);
                } else {
                    r_str += "0";
                }
                found = true;
                break;
            }
        }
        if (!found) {
            r_str += "9";
        }
    }
    
    return round(stod(r_str) * 1000000000000.0) / 1000000000000.0;
}

int main() {
    cout << fixed << setprecision(12) << compute() << endl;
    return 0;
}
```
