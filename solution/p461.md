# Presque Pi

## Description du Problème
Soit fₙ(k) = e^(k/n) - 1 pour tous les entiers k ≥ 0.

Soit g(n) = a² + b² + c² + d² pour a, b, c, d qui minimisent l'erreur : |fₙ(a) + fₙ(b) + fₙ(c) + fₙ(d) - π|.

On vous donne que g(200) = 6² + 75² + 89² + 226² = 64658.

**Objectif** : Trouver g(10000).

---

## Exemples
### Exemple 1
**Entrée** : `n = 200`
**Sortie** : `64658`
**Explication** : g(200) = 6² + 75² + 89² + 226² = 64658.

---

## Approche de Résolution

### Stratégie
1. Utiliser l'algorithme "meet in the middle".
2. Générer tous les fₙ(k) = e^(k/n) - 1 possibles, k va jusqu'à environ 14211.
3. Trouver toutes les paires fₙ(a) + fₙ(b) où a < b.
4. Pour chaque paire fₙ(a) + fₙ(b), rechercher binaire (π - fₙ(a) - fₙ(b)) dans les paires.
5. Supposer que fₙ(a) + fₙ(b) + fₙ(c) + fₙ(d) < π (sinon on devrait augmenter les valeurs).

### Observations
- L'algorithme "meet in the middle" réduit la complexité de O(n⁴) à O(n² log n).
- On peut trier les paires pour permettre la recherche binaire.
- On cherche à minimiser |somme - π|.

### Complexité
- **Temporelle** : O(n² log n) pour générer les paires et faire les recherches binaires.
- **Spatiale** : O(n²) pour stocker les paires.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>
#include <iomanip>

using namespace std;

double f(int n, int k) {
    return exp((double)k / n) - 1.0;
}

long long compute(int n) {
    vector<double> values;
    int k = 1;
    double pi = M_PI;
    
    while (f(n, k) <= pi) {
        values.push_back(f(n, k));
        k++;
    }
    
    vector<pair<double, pair<int, int>>> pairs;
    for (int a = 0; a < values.size(); a++) {
        for (int b = a + 1; b < values.size(); b++) {
            pairs.push_back({values[a] + values[b], {a + 1, b + 1}});
        }
    }
    
    sort(pairs.begin(), pairs.end());
    
    double min_error = 1e9;
    long long best_sum = 0;
    
    for (auto& p1 : pairs) {
        double target = pi - p1.first;
        auto it = lower_bound(pairs.begin(), pairs.end(), 
                             make_pair(target, make_pair(0, 0)));
        
        if (it != pairs.end()) {
            double error1 = abs(p1.first + it->first - pi);
            if (error1 < min_error) {
                min_error = error1;
                int a = p1.second.first, b = p1.second.second;
                int c = it->second.first, d = it->second.second;
                best_sum = a*a + b*b + c*c + d*d;
            }
        }
        
        if (it != pairs.begin()) {
            it--;
            double error2 = abs(p1.first + it->first - pi);
            if (error2 < min_error) {
                min_error = error2;
                int a = p1.second.first, b = p1.second.second;
                int c = it->second.first, d = it->second.second;
                best_sum = a*a + b*b + c*c + d*d;
            }
        }
    }
    
    return best_sum;
}

int main() {
    cout << compute(10000) << endl;
    return 0;
}
```
