# Carrés premiers réversibles

## Description du Problème
169 et 961 sont tous les deux le carré d'un nombre premier. 169 est l'inverse de 961.

On appelle un nombre un carré premier réversible si :
1. Il n'est pas un palindrome, et
2. Il est le carré d'un nombre premier, et
3. Son inverse est aussi le carré d'un nombre premier.

169 et 961 ne sont pas des palindromes, donc tous les deux sont des carrés premiers réversibles.

**Objectif** : Trouver la somme des 50 premiers carrés premiers réversibles.

---

## Exemples
### Exemple 1
**Entrée** : `169, 961`
**Sortie** : `1130`
**Explication** : 169 + 961 = 1130 (les deux premiers carrés premiers réversibles).

---

## Approche de Résolution

### Stratégie
1. Générer tous les nombres premiers jusqu'à une certaine limite.
2. Pour chaque nombre premier p, calculer p².
3. Inverser p² pour obtenir rev(p²).
4. Vérifier si rev(p²) est un carré parfait.
5. Si oui, vérifier si la racine carrée de rev(p²) est un nombre premier.
6. Vérifier que p² n'est pas un palindrome.
7. Collecter les 50 premiers et les sommer.

### Observations
- On peut optimiser en vérifiant d'abord si le nombre inversé est un carré parfait.
- On peut éviter les doublons en ne considérant que p² < rev(p²) ou vice versa.
- Il y a relativement peu de tels nombres.

### Complexité
- **Temporelle** : O(P × log p) où P est le nombre de nombres premiers à tester.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <string>
#include <algorithm>
#include <set>

using namespace std;

vector<int> list_primes(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

bool is_palindrome(long long n) {
    string s = to_string(n);
    string rev = s;
    reverse(rev.begin(), rev.end());
    return s == rev;
}

bool is_square(long long n) {
    long long root = sqrt(n);
    return root * root == n;
}

long long reverse_number(long long n) {
    long long rev = 0;
    while (n > 0) {
        rev = rev * 10 + n % 10;
        n /= 10;
    }
    return rev;
}

long long compute(int count) {
    vector<int> primes = list_primes(1000000);
    set<long long> reversible_squares;
    
    for (int p : primes) {
        long long square = (long long)p * p;
        if (is_palindrome(square)) continue;
        
        long long rev_square = reverse_number(square);
        if (is_square(rev_square)) {
            long long rev_root = sqrt(rev_square);
            // Vérifier si rev_root est premier
            bool is_prime_rev = true;
            for (int d = 2; d * d <= rev_root; d++) {
                if (rev_root % d == 0) {
                    is_prime_rev = false;
                    break;
                }
            }
            if (is_prime_rev && rev_root > 1) {
                reversible_squares.insert(square);
                reversible_squares.insert(rev_square);
            }
        }
    }
    
    long long total = 0;
    int collected = 0;
    for (long long num : reversible_squares) {
        if (collected >= count) break;
        total += num;
        collected++;
    }
    
    return total;
}

int main() {
    cout << compute(50) << endl;
    return 0;
}
```
