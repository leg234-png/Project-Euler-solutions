# Singleton Difference

## Description du Problème
Les entiers positifs, x, y et z, sont des termes consécutifs d'une progression arithmétique. Étant donné que n est un entier positif, l'équation, x² - y² - z² = n, a exactement une solution lorsque n = 20 :
13² - 10² - 7² = 20.

En fait, il y a vingt-cinq valeurs de n en dessous de cent pour lesquelles l'équation a une solution unique.

**Objectif** : Combien de valeurs de n inférieures à cinquante millions ont exactement une solution ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 20`
**Sortie** : `1`
**Explication** : Il y a exactement 1 solution : (13,10,7).

---

## Approche de Résolution

### Stratégie
1. Similaire au problème 135, utiliser la formule : n = (m - k)(5k - m).
2. Pour chaque m et k valides, calculer n et compter les solutions.
3. Compter combien de n ont exactement 1 solution.
4. La limite est 50 millions au lieu de 1 million.

### Observations
- Même approche que le problème 135, mais avec une limite plus élevée.
- On doit ajuster la recherche pour m < 2 × 50 × 10⁶.

### Complexité
- **Temporelle** : O(n²), où n ≈ 100 × 10⁶.
- **Spatiale** : O(n) pour stocker les compteurs.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

const int LIMIT = 50000000;

int main() {
    std::vector<int> solutions(LIMIT, 0);
    
    for (int m = 1; m < LIMIT * 2; m++) {
        for (int k = m / 5 + 1; k * 2 < m; k++) {
            long long temp = (long long)(m - k) * (k * 5 - m);
            if (temp >= LIMIT) break;
            solutions[temp]++;
        }
    }
    
    int count = 0;
    for (int x : solutions) {
        if (x == 1) {
            count++;
        }
    }
    
    std::cout << count << std::endl;
    return 0;
}
```
