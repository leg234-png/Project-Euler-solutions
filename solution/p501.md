# Huit diviseurs

## Description du Problème
Les huit diviseurs de 24 sont 1, 2, 3, 4, 6, 8, 12 et 24.

Les dix nombres ne dépassant pas 100 ayant exactement huit diviseurs sont 24, 30, 40, 42, 54, 56, 66, 70, 78 et 88.

Soit f(n) le nombre de nombres ne dépassant pas n avec exactement huit diviseurs.

On vous donne f(100)=10, f(1000)=180 et f(10⁶)=224427.

**Objectif** : Trouver f(10¹²).

---

## Exemples
### Exemple 1
**Entrée** : `n = 100`
**Sortie** : `10`
**Explication** : f(100) = 10.

---

## Approche de Résolution

### Stratégie
1. De la fonction diviseur, on sait que si n = p₁^e₁..., alors le nombre de diviseurs de n est (e₁+1)(e₂+1)...
2. Si le nombre de diviseurs est 8, il n'y a que 3 scénarios :
   - n = p⁷ => d(n) = (7+1) = 8
   - n = p₁³p₂ => d(n) = (3+1)(1+1) = 8
   - n = p₁p₂p₃ => d(n) = (1+1)(1+1)(1+1) = 8
3. Pour chaque cas, générer tous les nombres possibles et compter.
4. Pour le cas 3, comme p₁p₂p₃ < p₃³, on n'a besoin que des nombres premiers jusqu'à la racine cubique de N.

### Observations
- Il y a seulement 3 façons d'obtenir 8 diviseurs.
- On peut traiter chaque cas séparément.
- Pour le cas avec 3 nombres premiers, on doit éviter les doublons.

### Complexité
- **Temporelle** : O(π(N^(1/7)) + O(π(N^(1/3))²) + O(π(N^(1/3))³).
- **Spatiale** : O(√N) pour le crible de nombres premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<int> list_primes(long long n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (long long i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (long long j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (long long i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

long long compute(long long limit) {
    vector<int> primes = list_primes(cbrt(limit));
    long long count = 0;
    
    // Cas 1: n = p^7
    for (int p : primes) {
        long long n = pow(p, 7);
        if (n <= limit) {
            count++;
        }
    }
    
    // Cas 2: n = p₁³p₂
    for (size_t i = 0; i < primes.size(); i++) {
        long long p1 = primes[i];
        long long p1_cubed = p1 * p1 * p1;
        if (p1_cubed > limit) break;
        
        for (size_t j = 0; j < primes.size(); j++) {
            if (i == j) continue;
            long long p2 = primes[j];
            long long n = p1_cubed * p2;
            if (n <= limit) {
                count++;
            } else {
                break;
            }
        }
    }
    
    // Cas 3: n = p₁p₂p₃
    for (size_t i = 0; i < primes.size(); i++) {
        long long p1 = primes[i];
        for (size_t j = i + 1; j < primes.size(); j++) {
            long long p2 = primes[j];
            if (p1 * p2 > limit) break;
            
            for (size_t k = j + 1; k < primes.size(); k++) {
                long long p3 = primes[k];
                long long n = p1 * p2 * p3;
                if (n <= limit) {
                    count++;
                } else {
                    break;
                }
            }
        }
    }
    
    return count;
}

int main() {
    cout << compute(1000000000000LL) << endl;
    return 0;
}
```
