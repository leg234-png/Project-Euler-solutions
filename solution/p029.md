# Puissances Distinctes

## Description du Problème
Considérons toutes les combinaisons entières de aᵇ pour 2 ≤ a ≤ 5 et 2 ≤ b ≤ 5 :

2²=4, 2³=8, 2⁴=16, 2⁵=32
3²=9, 3³=27, 3⁴=81, 3⁵=243
4²=16, 4³=64, 4⁴=256, 4⁵=1024
5²=25, 5³=125, 5⁴=625, 5⁵=3125

Si on les place dans l'ordre numérique, en supprimant les répétitions, on obtient la séquence suivante de 15 termes distincts :

4, 8, 9, 16, 25, 27, 32, 64, 81, 125, 243, 256, 625, 1024, 3125

**Objectif** : Combien de termes distincts sont dans la séquence générée par aᵇ pour 2 ≤ a ≤ 100 et 2 ≤ b ≤ 100 ?

---

## Exemples
### Exemple 1
**Entrée** : 2 ≤ a ≤ 5, 2 ≤ b ≤ 5
**Sortie** : `15`
**Explication** : Il y a 15 termes distincts dans la séquence.

---

## Approche de Résolution

### Stratégie
1. Pour chaque paire (a, b) où 2 ≤ a ≤ 100 et 2 ≤ b ≤ 100 :
   - Calculer aᵇ.
   - Ajouter à un ensemble (pour éliminer les doublons).
2. Retourner la taille de l'ensemble.

### Observations
- On peut utiliser un `std::set` ou `std::unordered_set` pour stocker les valeurs uniques.
- Comme les nombres peuvent être très grands, on doit utiliser `BigInteger` ou des chaînes de caractères.
- En C++, on peut utiliser des chaînes pour représenter les grands nombres et implémenter la multiplication manuellement.

### Complexité
- **Temporelle** : O(a × b × m), où `a` = 99, `b` = 99, et `m` est la complexité de calcul de aᵇ.
- **Spatiale** : O(a × b), pour stocker l'ensemble des valeurs distinctes.

---

## Solution en C++

```cpp
#include <iostream>
#include <set>
#include <string>
#include <algorithm>

std::string multiply(const std::string& num, int factor) {
    std::string result;
    int carry = 0;
    
    for (int i = num.length() - 1; i >= 0; --i) {
        int digit = (num[i] - '0') * factor + carry;
        result.push_back((digit % 10) + '0');
        carry = digit / 10;
    }
    
    while (carry > 0) {
        result.push_back((carry % 10) + '0');
        carry /= 10;
    }
    
    std::reverse(result.begin(), result.end());
    return result;
}

std::string power(int base, int exp) {
    std::string result = "1";
    for (int i = 0; i < exp; ++i) {
        result = multiply(result, base);
    }
    return result;
}

int main() {
    std::set<std::string> distinct;
    
    for (int a = 2; a <= 100; ++a) {
        for (int b = 2; b <= 100; ++b) {
            distinct.insert(power(a, b));
        }
    }
    
    std::cout << distinct.size() << std::endl;
    return 0;
}
```
