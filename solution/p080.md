# Expansion décimale des racines carrées

## Description du Problème
Il est bien connu que si la racine carrée d'un nombre naturel n'est pas un entier, alors elle est irrationnelle. L'expansion décimale de telles racines carrées est infinie sans aucun motif répétitif.

La racine carrée de deux est 1.41421356237309504880..., et la somme des chiffres des cent premiers chiffres décimaux est 475.

**Objectif** : Pour les cent premiers nombres naturels, trouver la somme totale des cent premiers chiffres décimaux pour toutes les racines carrées irrationnelles.

---

## Exemples
### Exemple 1
**Entrée** : √2, 100 premiers chiffres décimaux
**Sortie** : `475`
**Explication** : La somme des 100 premiers chiffres décimaux de √2 est 475.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre i de 1 à 100, calculer √i avec une précision de 100 chiffres décimaux.
2. Si i n'est pas un carré parfait, calculer √i en utilisant un algorithme de division longue.
3. Multiplier i par 10^(200) pour obtenir 100 chiffres après la virgule.
4. Calculer la racine carrée en utilisant un algorithme de division longue.
5. Extraire les 100 premiers chiffres et les additionner.

### Observations
- On utilise BigInteger pour gérer les grands nombres.
- L'algorithme de racine carrée par division longue est similaire à la division longue manuelle.
- On commence par la position la plus à gauche et on extrait les chiffres de gauche à droite.

### Complexité
- **Temporelle** : O(100 × D²), où D est le nombre de chiffres (100).
- **Spatiale** : O(D), pour stocker les nombres intermédiaires.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <gmpxx.h>
#include <cmath>

using namespace std;

mpz_class sqrt_bigint(mpz_class x) {
    // Trouver la position la plus à gauche
    int i = 0;
    mpz_class ten_power;
    mpz_pow_ui(ten_power.get_mpz_t(), mpz_class(10).get_mpz_t(), i * 2);
    while (ten_power <= x) {
        i++;
        mpz_pow_ui(ten_power.get_mpz_t(), mpz_class(10).get_mpz_t(), i * 2);
    }
    
    // Extraire la racine carrée de gauche à droite
    mpz_class y = 0;
    for (; i >= 0; i--) {
        int j;
        mpz_class delta;
        for (j = 9; j >= 0; j--) {
            mpz_class temp = j;
            mpz_class power;
            mpz_pow_ui(power.get_mpz_t(), mpz_class(10).get_mpz_t(), i);
            temp *= power;
            delta = (y * 2 + temp) * temp;
            if (delta <= x)
                break;
        }
        if (j < 0)
            throw runtime_error("Error in sqrt calculation");
        x -= delta;
        mpz_class power;
        mpz_pow_ui(power.get_mpz_t(), mpz_class(10).get_mpz_t(), i);
        y += j * power;
    }
    return y;
}

int main() {
    int sum = 0;
    for (int i = 1; i <= 100; i++) {
        int sqrt_i = (int)sqrt(i);
        if (sqrt_i * sqrt_i == i)
            continue;  // Ignorer les carrés parfaits
        
        mpz_class x = i;
        mpz_class power;
        mpz_pow_ui(power.get_mpz_t(), mpz_class(10).get_mpz_t(), 200);
        x *= power;
        
        mpz_class y = sqrt_bigint(x);
        string s = y.get_str();
        if ((int)s.length() > 100)
            s = s.substr(0, 100);
        
        for (char c : s)
            sum += c - '0';
    }
    cout << sum << endl;
    return 0;
}
```
