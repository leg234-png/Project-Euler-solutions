# Reversible Numbers

## Description du Problème
Certains entiers positifs n ont la propriété que la somme [n + reverse(n)] consiste entièrement en chiffres impairs (décimaux). Par exemple, 36 + 63 = 99 et 409 + 904 = 1313. Nous appellerons de tels nombres réversibles ; donc 36, 63, 409 et 904 sont réversibles. Les zéros de tête ne sont pas autorisés dans n ou reverse(n).

Il y a 120 nombres réversibles en dessous de mille.

**Objectif** : Combien de nombres réversibles y a-t-il en dessous d'un milliard (10⁹) ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 36`
**Sortie** : `true`
**Explication** : 36 + 63 = 99, tous les chiffres sont impairs.

### Exemple 2
**Entrée** : `n = 409`
**Sortie** : `true`
**Explication** : 409 + 904 = 1313, tous les chiffres sont impairs.

---

## Approche de Résolution

### Stratégie
1. Analyser les propriétés des nombres réversibles par nombre de chiffres.
2. Pour n pair : 20 × 30^(n/2 - 1) nombres réversibles.
3. Pour n = 3 mod 4 : 100 × 500^((n-3)/4) nombres réversibles.
4. Pour n = 1 mod 4 : 0 nombres réversibles.
5. Parcourir les nombres de chiffres de 1 à 9 et sommer.

### Observations
- Les nombres à 1 chiffre n'ont pas de solutions (le dernier chiffre est toujours pair).
- Pour les nombres pairs de chiffres, il n'y a pas de retenues dans l'addition.
- Pour les nombres impairs de chiffres, il y a un pattern de retenues alternées.

### Complexité
- **Temporelle** : O(1) car on calcule directement avec des formules.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

long long pow(long long base, int exp) {
    long long result = 1;
    for (int i = 0; i < exp; i++) {
        result *= base;
    }
    return result;
}

int main() {
    long long sum = 0;
    for (int digits = 1; digits <= 9; digits++) {
        if (digits % 2 == 0) {
            sum += 20 * pow(30, digits / 2 - 1);
        } else if (digits % 4 == 3) {
            sum += 100 * pow(500, (digits - 3) / 4);
        }
        // digits % 4 == 1 donne 0 solutions
    }
    std::cout << sum << std::endl;
    return 0;
}
```
