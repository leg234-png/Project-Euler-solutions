# Somme de Puissances Cinquièmes de Chiffres

## Description du Problème
Étonnamment, il n'y a que trois nombres qui peuvent être écrits comme la somme de puissances quatrièmes de leurs chiffres :

1634 = 1⁴ + 6⁴ + 3⁴ + 4⁴
8208 = 8⁴ + 2⁴ + 0⁴ + 8⁴
9474 = 9⁴ + 4⁴ + 7⁴ + 4⁴

Comme 1 = 1⁴ n'est pas une somme, elle n'est pas incluse.

La somme de ces nombres est 1634 + 8208 + 9474 = 19316.

**Objectif** : Trouver la somme de tous les nombres qui peuvent être écrits comme la somme de puissances cinquièmes de leurs chiffres.

---

## Exemples
### Exemple 1
**Entrée** : Puissance = 4
**Sortie** : `19316`
**Explication** : 1634 + 8208 + 9474 = 19316.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre i de 2 à une limite supérieure :
   - Calculer la somme des puissances cinquièmes de ses chiffres.
   - Si cette somme est égale à i, ajouter i à la somme totale.
2. Retourner la somme totale.

### Observations
- Pour déterminer la limite supérieure :
  - Si un nombre a n chiffres, la somme maximale des puissances cinquièmes est n × 9⁵.
  - Pour n ≥ 7, n × 9⁵ < 10ⁿ⁻¹, donc on peut limiter la recherche à 6 chiffres.
  - Une limite sûre est 6 × 9⁵ = 354294.
- On peut précalculer les puissances cinquièmes de 0 à 9 pour optimiser.

### Complexité
- **Temporelle** : O(n × d), où `n` est la limite (environ 354294) et `d` est le nombre de chiffres.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>

int fifthPowerSum(int n) {
    int sum = 0;
    while (n > 0) {
        int digit = n % 10;
        sum += digit * digit * digit * digit * digit;
        n /= 10;
    }
    return sum;
}

int main() {
    int sum = 0;
    // Limite : 6 chiffres × 9⁵ = 354294
    for (int i = 2; i < 1000000; ++i) {
        if (i == fifthPowerSum(i)) {
            sum += i;
        }
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
