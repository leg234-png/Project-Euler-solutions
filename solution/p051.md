# Remplacement de Chiffres Premiers

## Description du Problème
En remplaçant le premier chiffre du nombre à 2 chiffres *3, on découvre que six des neuf valeurs possibles sont premières : 13, 23, 43, 53, 73 et 83.

En remplaçant le troisième et le quatrième chiffre de 56**3 par le même chiffre, ce nombre à 5 chiffres est le premier exemple ayant sept nombres premiers parmi les dix valeurs générées, donnant la famille : 56003, 56113, 56333, 56443, 56663, 56773, et 56993. Par conséquent, 56003, étant le premier membre de cette famille, est le plus petit nombre premier avec cette propriété.

**Objectif** : Trouver le plus petit nombre premier qui, en remplaçant une partie du nombre (pas nécessairement des chiffres adjacents) par le même chiffre, fait partie d'une famille de huit nombres premiers.

---

## Exemples
### Exemple 1
**Entrée** : Famille de 7 nombres premiers
**Sortie** : `56003`
**Explication** : 56003 est le premier membre de la famille de 7 nombres premiers obtenus en remplaçant les 3ème et 4ème chiffres.

---

## Approche de Résolution

### Stratégie
1. Générer tous les nombres premiers jusqu'à 1 000 000.
2. Pour chaque nombre premier :
   - Convertir en tableau de chiffres.
   - Pour chaque masque binaire (représentant quels chiffres remplacer) :
     - Remplacer les chiffres marqués par 0-9.
     - Compter combien de résultats sont premiers.
     - Si on obtient exactement 8 nombres premiers, retourner le plus petit.
3. Retourner le résultat.

### Observations
- On utilise un masque binaire pour identifier quels chiffres remplacer.
- Le premier chiffre ne peut pas être 0 (sinon le nombre n'aurait pas le bon nombre de chiffres).
- On teste toutes les combinaisons possibles de remplacement.

### Complexité
- **Temporelle** : O(P × 2^D × 10), où P est le nombre de nombres premiers et D est le nombre de chiffres.
- **Spatiale** : O(P), pour stocker le crible de primalité.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>

std::vector<bool> sieve(int n) {
    std::vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    int sqrtN = sqrt(n);
    for (int i = 2; i <= sqrtN; ++i) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    return isPrime;
}

std::vector<int> toDigits(int n) {
    std::vector<int> digits;
    while (n > 0) {
        digits.push_back(n % 10);
        n /= 10;
    }
    std::reverse(digits.begin(), digits.end());
    return digits;
}

int toNumber(const std::vector<int>& digits) {
    int result = 0;
    for (int d : digits) {
        result = result * 10 + d;
    }
    return result;
}

std::vector<int> applyMask(const std::vector<int>& digits, int mask) {
    std::vector<int> result = digits;
    for (size_t i = 0; i < digits.size(); ++i) {
        if (!(mask & (1 << i))) {
            result[i] = 0;
        }
    }
    return result;
}

std::vector<int> addMask(const std::vector<int>& digits, int mask) {
    std::vector<int> result = digits;
    for (size_t i = 0; i < digits.size(); ++i) {
        if (mask & (1 << i)) {
            result[i]++;
        }
    }
    return result;
}

int main() {
    const int limit = 1000000;
    std::vector<bool> isPrime = sieve(limit);
    
    for (int i = 0; i < limit; ++i) {
        if (!isPrime[i]) continue;
        
        std::vector<int> n = toDigits(i);
        for (int mask = 0; mask < (1 << n.size()); ++mask) {
            std::vector<int> digits = applyMask(n, mask);
            int count = 0;
            
            for (int j = 0; j < 10; ++j) {
                if (digits[0] != 0 && isPrime[toNumber(digits)]) {
                    count++;
                }
                digits = addMask(digits, mask);
            }
            
            if (count == 8) {
                digits = applyMask(n, mask);
                for (int j = 0; j < 10; ++j) {
                    if (digits[0] != 0 && isPrime[toNumber(digits)]) {
                        std::cout << toNumber(digits) << std::endl;
                        return 0;
                    }
                    digits = addMask(digits, mask);
                }
            }
        }
    }
    
    return 0;
}
```
