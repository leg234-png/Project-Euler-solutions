# Factorial Trailing Digits

## Description du Problème
Pour tout `n`, soit `f(n)` les 5 derniers chiffres non nuls de `n!`, c'est-à-dire les 5 derniers chiffres avant les zéros finaux.

Par exemple :
- `9! = 362880`, donc `f(9) = 36288`.
- `10! = 3628800`, donc `f(10) = 36288`.
- `20! = 2432902008176640000`, donc `f(20) = 17664`.

**Objectif** : Trouver `f(1 000 000 000 000)`.

---

## Exemples
### Exemple 1
**Entrée** : `n = 9`
**Sortie** : `36288`
**Explication** : `9! = 362880`, les 5 derniers chiffres non nuls sont `36288`.

### Exemple 2
**Entrée** : `n = 20`
**Sortie** : `17664`
**Explication** : `20! = 2432902008176640000`, les 5 derniers chiffres non nuls sont `17664`.

---

## Approche de Résolution

### Stratégie
1. Le problème consiste à calculer les 5 derniers chiffres non nuls de `n!`.
2. Étapes :
   - Compter le nombre de facteurs 2 et 5 dans `n!`.
   - Calculer `n!` sans les facteurs 2 et 5, modulo 10^5.
   - Multiplier par les facteurs 2 restants (après avoir soustrait les facteurs 5), modulo 10^5.
3. Utiliser des fonctions récursives :
   - `countFactors(n, p)` : compte le nombre de facteurs `p` dans `n!`.
   - `factorialish(n)` : calcule `n!` sans les facteurs 2 et 5, modulo 10^5.
   - `evenFactorialish(n)` : calcule le produit des nombres pairs sans 2 et 5.
   - `oddFactorialish(n)` : calcule le produit des nombres impairs sans 2 et 5.
   - `factorialCoprime(n)` : calcule le produit des nombres copremiers avec 10.
4. Utiliser la propriété cyclique : `2^5 = 2^2505 = 32 mod 100000`.

### Observations
- Les zéros finaux proviennent des facteurs 2 et 5.
- On peut réduire les facteurs 2 en utilisant la cyclicité modulo 100000.
- `factorialCoprime(10^5) = 1`, donc on peut prendre `n mod 10^5`.

### Complexité
- **Temporelle** : O(log n) pour la récursion.
- **Spatiale** : O(log n) pour la pile de récursion.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

const long long MOD = 100000;

long long countFactors(long long n, long long p) {
    if (n == 0) return 0;
    return n / p + countFactors(n / p, p);
}

long long factorialCoprime(long long n) {
    n %= MOD;
    long long product = 1;
    for (long long i = 1; i <= n; i++) {
        if (i % 2 != 0 && i % 5 != 0)
            product = (product * i) % MOD;
    }
    return product;
}

long long oddFactorialish(long long n) {
    if (n == 0) return 1;
    return (oddFactorialish(n / 5) * factorialCoprime(n)) % MOD;
}

long long evenFactorialish(long long n) {
    if (n == 0) return 1;
    return factorialish(n / 2);
}

long long factorialish(long long n) {
    return (evenFactorialish(n) * oddFactorialish(n)) % MOD;
}

long long powMod(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp % 2 == 1)
            result = (result * base) % mod;
        base = (base * base) % mod;
        exp /= 2;
    }
    return result;
}

long long factorialSuffix(long long n) {
    long long twos = countFactors(n, 2) - countFactors(n, 5);
    
    // Reduce twos using cycle: 2^5 = 2^2505 = 32 mod 100000
    if (twos >= 2505)
        twos = (twos - 5) % 2500 + 5;
    
    return (factorialish(n) * powMod(2, twos, MOD)) % MOD;
}

int main() {
    cout << factorialSuffix(1000000000000LL) << endl;
    return 0;
}
```
