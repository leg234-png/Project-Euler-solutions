# Fractions ordonnées

## Description du Problème
Considérons la fraction n/d, où n et d sont des entiers positifs. Si n < d et HCF(n,d) = 1, elle est appelée une fraction propre réduite.

Si on liste l'ensemble des fractions propres réduites pour d ≤ 8 par ordre croissant de taille, on obtient :
1/8, 1/7, 1/6, 1/5, 1/4, 2/7, 1/3, 3/8, 2/5, 3/7, 1/2, 4/7, 3/5, 5/8, 2/3, 5/7, 3/4, 4/5, 5/6, 6/7, 7/8

On peut voir que 2/5 est la fraction immédiatement à gauche de 3/7.

**Objectif** : En listant l'ensemble des fractions propres réduites pour d ≤ 1 000 000 par ordre croissant de taille, trouver le numérateur de la fraction immédiatement à gauche de 3/7.

---

## Exemples
### Exemple 1
**Entrée** : d ≤ 8
**Sortie** : `2`
**Explication** : La fraction 2/5 est immédiatement à gauche de 3/7 dans la liste ordonnée.

---

## Approche de Résolution

### Stratégie
1. Pour chaque dénominateur d de 1 à 1 000 000, calculer le plus grand numérateur n tel que n/d < 3/7.
2. Si d est un multiple de 7, alors n = (d/7) × 3 - 1.
3. Sinon, n = floor(d × 3 / 7).
4. Garder la fraction n/d qui est la plus grande parmi toutes celles inférieures à 3/7.
5. Retourner le numérateur de cette fraction.

### Observations
- On évite de simplifier explicitement les fractions en traitant les dénominateurs par ordre croissant.
- Si (n, d) n'est pas en termes réduits, sa forme réduite (n', d') aurait été sauvegardée quand le plus petit dénominateur d' a été traité.
- Comme n/d n'est pas plus grand que n'/d' (ils sont égaux), la valeur sauvegardée ne serait pas écrasée.

### Complexité
- **Temporelle** : O(LIMIT), où LIMIT = 1 000 000.
- **Spatiale** : O(1), car on ne stocke que la meilleure fraction trouvée.

---

## Solution en C++

```cpp
#include <iostream>

using namespace std;

int main() {
    const int LIMIT = 1000000;
    int maxN = 0;
    int maxD = 1;
    
    for (int d = 1; d <= LIMIT; d++) {
        int n = d * 3 / 7;
        if (d % 7 == 0)
            n--;
        if ((long long)n * maxD > (long long)maxN * d) {  // n/d > maxN/maxD
            maxN = n;
            maxD = d;
        }
    }
    
    cout << maxN << endl;
    return 0;
}
```
