# Une longue rangée de dés

## Description du Problème
On commence avec une rangée de n dés montrant tous 1.

Ensuite, pour chaque i de 1 à n, on augmente chaque dé divisible par i de 1.

Si un dé montre 6 et est augmenté, il est remis à 1.

Soit f(n) le nombre de dés qui montrent 1 quand le processus se termine. On vous donne f(100) = 2 et f(10⁸) = 69.

**Objectif** : Trouver f(10³⁶).

---

## Exemples
### Exemple 1
**Entrée** : `n = 100`
**Sortie** : `2`
**Explication** : f(100) = 2.

---

## Approche de Résolution

### Stratégie
1. Le n-ième dé sera augmenté pour chaque i qui divise n.
2. Le n-ième dé montrera donc σ₀(n) (nombre de diviseurs de n) sans soustraire 6.
3. Le problème demande combien de k ≤ n sont tels que σ₀(k) ≡ 1 (mod 6).
4. σ₀(k) = ∏_{i=1}^r (e_i + 1) où k = p₁^e₁...p_r^e_r.
5. Pour que σ₀(k) soit impair, tous les e_i doivent être pairs, donc k doit être un carré parfait.
6. Pour que σ₀(k) ≡ 1 (mod 6), on a des conditions supplémentaires sur les exposants.
7. Utiliser un crible pour calculer σ₀(k) pour tous les carrés parfaits.

### Observations
- Seuls les carrés parfaits peuvent avoir σ₀(k) impair.
- On peut utiliser un crible pour calculer efficacement σ₀(k).
- Il y a des patterns dans les nombres qui satisfont σ₀(k) ≡ 1 (mod 6).

### Complexité
- **Temporelle** : O(√n × log log n) avec un crible optimisé.
- **Spatiale** : O(√n) pour le crible.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long compute(long long n) {
    long long sqrt_n = sqrt(n);
    vector<int> divisors(sqrt_n + 1, 1);
    
    // Calculer σ₀(k) pour tous les carrés parfaits
    for (long long i = 2; i <= sqrt_n; i++) {
        if (divisors[i] == 1) {
            for (long long j = i; j <= sqrt_n; j += i) {
                int count = 0;
                long long temp = j;
                while (temp % i == 0) {
                    count++;
                    temp /= i;
                }
                divisors[j] *= (2 * count + 1);
            }
        }
    }
    
    long long count = 0;
    for (long long k = 1; k <= sqrt_n; k++) {
        if (divisors[k] % 6 == 1) {
            count++;
        }
    }
    
    return count;
}

int main() {
    long long n = 1;
    for (int i = 0; i < 36; i++) {
        n *= 10;
    }
    cout << compute(n) << endl;
    return 0;
}
```
