# Permutations de Nombres Premiers

## Description du Problème
La suite arithmétique 1487, 4817, 8147, dans laquelle chacun des termes augmente de 3330, est inhabituelle de deux façons : (i) chacun des trois termes est premier, et (ii) chacun des termes à 4 chiffres est une permutation de l'autre.

Il n'existe pas de suite arithmétique composée de trois nombres à 1 ou 2 chiffres qui présente cette propriété, mais il existe une autre suite à 4 chiffres croissante.

**Objectif** : Quelle suite arithmétique à 12 chiffres forme en concaténant les trois termes ?

---

## Exemples
### Exemple 1
**Entrée** : Nombres premiers à 4 chiffres
**Sortie** : `296962999629`
**Explication** : La concaténation des trois nombres premiers en progression arithmétique qui sont des permutations l'un de l'autre.

---

## Approche de Résolution

### Stratégie
1. Trouver tous les nombres premiers à 4 chiffres.
2. Pour chaque nombre premier de base :
   - Pour chaque pas possible :
     - Calculer base + step et base + 2×step.
     - Vérifier si les trois sont premiers.
     - Vérifier si les trois sont des permutations l'un de l'autre.
     - Si oui et si ce n'est pas 1487, 4817, 8147, c'est la réponse.
3. Retourner la concaténation des trois nombres.

### Observations
- Les trois nombres doivent être à 4 chiffres, donc base < 10000, base + step < 10000, base + 2×step < 10000.
- Pour vérifier si deux nombres sont des permutations, trier leurs chiffres et comparer.
- On peut utiliser le crible d'Ératosthène pour trouver tous les nombres premiers.

### Complexité
- **Temporelle** : O(n²), où `n` est le nombre de nombres premiers à 4 chiffres.
- **Spatiale** : O(n), pour stocker les nombres premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
#include <cmath>

std::vector<bool> sieve(int n) {
    std::vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    int sqrtN = sqrt(n);
    for (int i = 2; i <= sqrtN; ++i) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    return isPrime;
}

bool hasSameDigits(int x, int y) {
    std::string xStr = std::to_string(x);
    std::string yStr = std::to_string(y);
    
    std::sort(xStr.begin(), xStr.end());
    std::sort(yStr.begin(), yStr.end());
    
    return xStr == yStr;
}

int main() {
    const int limit = 10000;
    std::vector<bool> isPrime = sieve(limit - 1);
    
    for (int base = 1000; base < limit; ++base) {
        if (isPrime[base]) {
            for (int step = 1; step < limit; ++step) {
                int a = base + step;
                int b = a + step;
                
                if (a < limit && isPrime[a] && hasSameDigits(a, base) &&
                    b < limit && isPrime[b] && hasSameDigits(b, base) &&
                    (base != 1487 || a != 4817)) {
                    std::cout << base << a << b << std::endl;
                    return 0;
                }
            }
        }
    }
    
    return 0;
}
```
