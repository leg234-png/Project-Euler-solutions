# Diviseurs cube-full

## Description du Problème
Un nombre n est cube-full si pour chaque nombre premier p qui divise n, p³ divise aussi n.

=> n = p₁^(3e₁) × p₂^(3e₂) × ...

Soit s(n) le plus grand diviseur cube-full de n.

Soit S(n) la fonction sommatoire de s(n), c'est-à-dire S(n) = ∑_{i=1}^n s(i).

On vous donne S(16) = 19, S(100) = 126 et S(10000) = 13344.

**Objectif** : Trouver S(10¹⁸).

---

## Exemples
### Exemple 1
**Entrée** : `n = 16`
**Sortie** : `19`
**Explication** : S(16) = 19.

---

## Approche de Résolution

### Stratégie
1. S(n) = ∑_{1 ≤ cube-full i ≤ n} ⌊n/i⌋.
2. lim_{n→∞} S(n)/n = lim_{n→∞} 1/n ∑_{1 ≤ cube-full i ≤ n} ⌊n/i⌋.
3. En intervertissant la sommation et la limite, on obtient lim_{n→∞} S(n)/n = ∑_{1 ≤ cube-full i} 1/i.
4. = ∏_{p premier} (1 + 1/p³ + 1/p⁴ + ...) = ∏_{p premier} (1 + 1/(p²(p-1))).
5. Cette série converge.
6. Pour calculer S(10¹⁸), utiliser cette convergence et calculer la somme partielle.

### Observations
- Il y a une limite claire pour S(n)/n.
- On peut utiliser la convergence pour approximer S(10¹⁸).
- Le produit infini converge rapidement.

### Complexité
- **Temporelle** : O(P) où P est le nombre de nombres premiers à considérer.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <iomanip>

using namespace std;

vector<int> prime_sieve(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

long double compute_limit() {
    vector<int> primes = prime_sieve(1000000);
    long double product = 1.0;
    
    for (int p : primes) {
        long double term = 1.0 + 1.0 / ((long double)p * p * (p - 1));
        product *= term;
    }
    
    return product;
}

long long compute_S(long long n) {
    long double limit = compute_limit();
    return (long long)(n * limit);
}

int main() {
    long long n = 1;
    for (int i = 0; i < 18; i++) {
        n *= 10;
    }
    cout << compute_S(n) << endl;
    return 0;
}
```
