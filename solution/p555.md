# Fonction McCarthy 91

## Description du Problème
La fonction McCarthy 91 est définie comme :
M₉₁(n) = n - 10 si n > 100, sinon M₉₁(M₉₁(n + 11)).

On peut généraliser cela à M_{m,k,s}(n) = n - s si n > m, sinon M_{m,k,s}(M_{m,k,s}(n + k)).

Soit F_{m,k,s} l'ensemble des points fixes de M_{m,k,s}, c'est-à-dire F_{m,k,s} = {n ∈ ℕ | M_{m,k,s}(n) = n}.

Par exemple, le seul point fixe de M₉₁ est n = 91. En d'autres termes, F_{100,11,10} = {91}.

Soit SF(m, k, s) = ∑_{n ∈ F_{m,k,s}} n.

**Objectif** : Trouver ∑_{1 ≤ m < p ≤ 1000000} SF(m, k, s) où p est premier et k = ⌊p/2⌋, s = ⌈p/2⌉.

---

## Exemples
### Exemple 1
**Entrée** : `m = 100, k = 11, s = 10`
**Sortie** : `91`
**Explication** : F_{100,11,10} = {91}.

---

## Approche de Résolution

### Stratégie
1. Utiliser le théorème de généralisation : si (c-1)b < d, alors f(x) = x - b si x > a, sinon f(x + d - (c-1)b).
2. Dans notre cas, a = m, b = s, c = 2, d = k.
3. Si s < k, alors M_{m,k,s}(n) = n - s si n > m, sinon m + k - 2s - (m - n (mod k - s)).
4. Un point fixe doit être entre 0 et m.
5. Si t = m + k - 2s, alors M(t, m, k, s) = t si et seulement si (2s - k) (mod k - s) = 0.
6. Cela se réduit à k - s | s.

### Observations
- Il y a une formule directe pour les points fixes.
- On peut calculer SF(m, k, s) efficacement sans itérer sur tous les nombres.
- La condition k - s | s détermine l'existence de points fixes.

### Complexité
- **Temporelle** : O(P × log p) où P est le nombre de nombres premiers.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<int> prime_sieve(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

long long SF(int m, int k, int s) {
    if ((k - s) == 0 || s % (k - s) != 0) {
        return 0;
    }
    
    int t = m + k - 2 * s;
    if (t < 0 || t > m) {
        return 0;
    }
    
    // Tous les nombres de t - (k - s) + 1 à t sont des points fixes
    int count = k - s;
    long long sum = (long long)count * (2 * t - count + 1) / 2;
    return sum;
}

long long compute(int limit) {
    vector<int> primes = prime_sieve(limit);
    long long total = 0;
    
    for (int p : primes) {
        if (p > limit) break;
        int m = p;
        int k = p / 2;
        int s = (p + 1) / 2;
        total += SF(m, k, s);
    }
    
    return total;
}

int main() {
    cout << compute(1000000) << endl;
    return 0;
}
```
