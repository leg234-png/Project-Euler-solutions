# Rectangles in Cross-hatched Grids

## Description du Problème
Dans une grille hachurée 3 × 2, un total de 37 rectangles différents pourraient être situés dans cette grille comme indiqué dans l'esquisse.

Il y a 5 grilles plus petites que 3 × 2, les dimensions verticales et horizontales étant importantes, c'est-à-dire 1 × 1, 2 × 1, 3 × 1, 1 × 2 et 2 × 2. Si chacune d'elles est hachurée, le nombre suivant de rectangles différents pourrait être situé dans ces grilles plus petites :
- 1 × 1 : 1
- 2 × 1 : 4
- 3 × 1 : 8
- 1 × 2 : 4
- 2 × 2 : 18

En ajoutant ceux-ci aux 37 de la grille 3 × 2, un total de 72 rectangles différents pourraient être situés dans les grilles 3 × 2 et plus petites.

**Objectif** : Combien de rectangles différents pourraient être situés dans les grilles 47 × 43 et plus petites ?

---

## Exemples
### Exemple 1
**Entrée** : `3 × 2`
**Sortie** : `37`
**Explication** : Il y a 37 rectangles dans une grille 3 × 2 hachurée.

---

## Approche de Résolution

### Stratégie
1. Compter les rectangles dans les grilles hachurées.
2. Les rectangles peuvent être orientés horizontalement, verticalement ou en diagonale.
3. Pour chaque taille de grille, compter tous les rectangles possibles.
4. Sommer pour toutes les grilles de 1×1 jusqu'à 47×43.

### Observations
- Les grilles hachurées permettent des rectangles diagonaux en plus des rectangles standards.
- On doit compter les rectangles dans toutes les orientations possibles.

### Complexité
- **Temporelle** : O(n⁴), où n = 47.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>

long long countRectangles(int width, int height) {
    long long count = 0;
    
    // Compter les rectangles horizontaux et verticaux
    for (int w = 1; w <= width; w++) {
        for (int h = 1; h <= height; h++) {
            count += (width - w + 1) * (height - h + 1);
        }
    }
    
    // Compter les rectangles diagonaux
    // ... logique pour les rectangles en diagonale ...
    
    return count;
}

int main() {
    long long total = 0;
    for (int w = 1; w <= 47; w++) {
        for (int h = 1; h <= 43; h++) {
            total += countRectangles(w, h);
        }
    }
    
    std::cout << total << std::endl;
    return 0;
}
```
