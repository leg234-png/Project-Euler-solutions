# Hexadecimal Numbers

## Description du Problème
Dans un nombre hexadécimal de `n` chiffres, le premier chiffre peut être de 0 à F (15), mais les chiffres suivants peuvent être de 0 à F.

**Objectif** : Trouver la somme de tous les nombres hexadécimaux de 1 à 16 chiffres qui contiennent au moins un chiffre 0, au moins un chiffre 1, et au moins un chiffre A. Exprimer la réponse en hexadécimal (en majuscules).

---

## Exemples
### Exemple 1
**Entrée** : `n = 1`
**Sortie** : `0`
**Explication** : Un nombre à 1 chiffre ne peut pas contenir à la fois 0, 1 et A.

---

## Approche de Résolution

### Stratégie
1. Utiliser le principe d'inclusion-exclusion pour compter les nombres hexadécimaux qui contiennent au moins un 0, un 1 et un A.
2. Pour chaque longueur `n` de 1 à 16 :
   - Total de nombres : `15 × 16^(n-1)`.
   - Nombres sans 0 : `15^n`.
   - Nombres sans 1 : `14 × 15^(n-1)`.
   - Nombres sans A : `14 × 15^(n-1)`.
   - Nombres sans 0 et 1 : `14^n`.
   - Nombres sans 0 et A : `14^n`.
   - Nombres sans 1 et A : `13 × 14^(n-1)`.
   - Nombres sans 0, 1 et A : `13^n`.
3. Utiliser la formule d'inclusion-exclusion :
   - Nombres avec 0, 1 et A = Total - (sans 0) - (sans 1) - (sans A) + (sans 0 et 1) + (sans 0 et A) + (sans 1 et A) - (sans 0, 1 et A)
   - = `15×16^(n-1) - 43×15^(n-1) + 41×14^(n-1) - 13^n`
4. Additionner pour tous les `n` de 1 à 16.
5. Convertir le résultat en hexadécimal.

### Observations
- Le principe d'inclusion-exclusion permet de compter les nombres qui satisfont plusieurs conditions.
- On doit gérer les grands nombres avec `BigInteger`.

### Complexité
- **Temporelle** : O(n × log n), où n=16.
- **Spatiale** : O(log n) pour stocker les grands nombres.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <algorithm>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

cpp_int bi(int n) {
    return cpp_int(n);
}

string toHex(cpp_int n) {
    string hex = "";
    if (n == 0) return "0";
    while (n > 0) {
        int digit = (n % 16).convert_to<int>();
        if (digit < 10)
            hex += char('0' + digit);
        else
            hex += char('A' + digit - 10);
        n /= 16;
    }
    reverse(hex.begin(), hex.end());
    return hex;
}

int main() {
    cpp_int sum = 0;
    for (int n = 1; n <= 16; n++) {
        cpp_int term = bi(15) * pow(bi(16), n - 1)
                     - bi(43) * pow(bi(15), n - 1)
                     + bi(41) * pow(bi(14), n - 1)
                     - pow(bi(13), n);
        sum += term;
    }
    cout << toHex(sum) << endl;
    return 0;
}
```
