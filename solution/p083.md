# Somme de chemin : quatre directions

## Description du Problème
Dans la matrice 5 par 5 ci-dessous, la somme minimale de chemin du coin supérieur gauche au coin inférieur droit, en se déplaçant à gauche, à droite, en haut et en bas, est indiquée en gras rouge et est égale à 2297.

**Objectif** : Trouver la somme minimale de chemin du coin supérieur gauche au coin inférieur droit en se déplaçant à gauche, à droite, en haut et en bas dans matrix.txt, un fichier texte de 31K contenant une matrice de 80 par 80.

**NOTE** : Ce problème est une version significativement plus difficile du Problème 81.

---

## Exemples
### Exemple 1
**Entrée** : Matrice 5×5
**Sortie** : `2297`
**Explication** : Le chemin optimal peut maintenant se déplacer dans toutes les directions.

---

## Approche de Résolution

### Stratégie
1. Utiliser l'algorithme de Bellman-Ford pour trouver le plus court chemin dans un graphe.
2. Initialiser la distance du coin supérieur gauche à sa valeur.
3. Pour chaque itération, mettre à jour les distances de toutes les cellules en considérant leurs voisins (haut, bas, gauche, droite).
4. Répéter jusqu'à convergence (n² itérations suffisent pour une grille).

### Observations
- On peut se déplacer dans toutes les quatre directions.
- C'est essentiellement un problème de plus court chemin dans un graphe.
- L'algorithme de Bellman-Ford fonctionne même avec des poids positifs.
- On peut aussi utiliser Dijkstra pour une meilleure performance.

### Complexité
- **Temporelle** : O(n⁴), où n est la taille de la matrice (80), car on fait n² itérations et on examine n² cellules à chaque itération.
- **Spatiale** : O(n²), pour stocker les distances.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <fstream>
#include <sstream>
#include <climits>
#include <algorithm>

using namespace std;

const int INFINITY = INT_MAX / 2;

int main() {
    // Lire la matrice depuis le fichier
    vector<vector<int>> GRID;
    ifstream file("matrix.txt");
    string line;
    
    while (getline(file, line)) {
        vector<int> row;
        istringstream iss(line);
        string num;
        while (getline(iss, num, ',')) {
            row.push_back(stoi(num));
        }
        GRID.push_back(row);
    }
    file.close();
    
    int h = GRID.size();
    int w = GRID[0].size();
    vector<vector<int>> distance(h, vector<int>(w, INFINITY));
    distance[0][0] = GRID[0][0];
    
    // Algorithme de Bellman-Ford
    for (int i = 0; i < w * h; i++) {
        for (int y = 0; y < h; y++) {
            for (int x = 0; x < w; x++) {
                int temp = INFINITY;
                if (x > 0) temp = min(distance[y][x - 1], temp);
                if (x + 1 < w) temp = min(distance[y][x + 1], temp);
                if (y > 0) temp = min(distance[y - 1][x], temp);
                if (y + 1 < h) temp = min(distance[y + 1][x], temp);
                distance[y][x] = min(GRID[y][x] + temp, distance[y][x]);
            }
        }
    }
    
    cout << distance[h - 1][w - 1] << endl;
    return 0;
}
```
