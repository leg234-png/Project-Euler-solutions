# Conjecture de Goldbach

## Description du Problème
Christian Goldbach a proposé que tout nombre composé impair peut être écrit comme la somme d'un nombre premier et du double d'un carré.

9 = 7 + 2×1²
15 = 7 + 2×2²
21 = 3 + 2×3²
25 = 7 + 2×3²
27 = 19 + 2×2²
33 = 31 + 2×1²

Il s'avère que la conjecture était fausse.

**Objectif** : Quel est le plus petit nombre composé impair qui ne peut pas être écrit comme la somme d'un nombre premier et du double d'un carré ?

---

## Exemples
### Exemple 1
**Entrée** : Nombres composés impairs
**Sortie** : `5777`
**Explication** : Le plus petit nombre composé impair qui viole la conjecture.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre composé impair i à partir de 9 :
   - Vérifier s'il peut être écrit comme p + 2×k² où p est premier et k est un entier.
   - Tester tous les k tels que 2×k² < i.
   - Pour chaque k, vérifier si i - 2×k² est premier.
   - Si aucune combinaison ne fonctionne, c'est la réponse.
2. Retourner le premier nombre qui ne satisfait pas la conjecture.

### Observations
- Un nombre composé impair est un nombre impair qui n'est pas premier.
- On peut tester k de 1 jusqu'à √(i/2).
- On peut utiliser un test de primalité efficace.

### Complexité
- **Temporelle** : O(n × √n × √n), où `n` est le nombre recherché.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

bool isPrime(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    
    int sqrtN = sqrt(n);
    for (int i = 3; i <= sqrtN; i += 2) {
        if (n % i == 0) {
            return false;
        }
    }
    return true;
}

bool satisfiesConjecture(int n) {
    if (n % 2 == 0 || isPrime(n)) {
        return true;
    }
    
    // n est un nombre composé impair
    for (int k = 1; 2 * k * k < n; ++k) {
        int remainder = n - 2 * k * k;
        if (isPrime(remainder)) {
            return true;
        }
    }
    
    return false;
}

int main() {
    for (int i = 9; ; i += 2) {
        if (!satisfiesConjecture(i)) {
            std::cout << i << std::endl;
            break;
        }
    }
    
    return 0;
}
```
