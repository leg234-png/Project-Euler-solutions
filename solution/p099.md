# Largest Exponential

## Description du Problème
Comparer deux nombres écrits sous forme d'exposant comme 2^11 et 3^7 n'est pas difficile, car n'importe quelle calculatrice confirmerait que 2^11 = 2048 < 3^7 = 2187.

Cependant, confirmer que 632382^518061 > 519432^525806 serait beaucoup plus difficile, car les deux nombres contiennent plus de trois millions de chiffres.

En utilisant base_exp.txt, un fichier texte de 22K contenant mille lignes avec une paire base/exposant sur chaque ligne, déterminer quel numéro de ligne a la plus grande valeur numérique.

**Objectif** : Trouver le numéro de ligne (1-indexé) avec la plus grande valeur numérique.

---

## Exemples
### Exemple 1
**Entrée** : 
```
519432,525806
632382,518061
```
**Sortie** : `2`
**Explication** : La ligne 2 (632382^518061) a la plus grande valeur.

---

## Approche de Résolution

### Stratégie
1. Lire toutes les paires (base, exposant) depuis le fichier.
2. Pour comparer deux nombres a^b et c^d sans les calculer entièrement, utiliser le logarithme :
   - a^b > c^d si et seulement si b × log(a) > d × log(c)
3. Parcourir toutes les lignes et trouver celle avec la plus grande valeur de b × log(a).
4. Retourner le numéro de ligne (1-indexé).

### Observations
- Calculer directement les nombres serait impossible car ils ont des millions de chiffres.
- On utilise les logarithmes pour comparer : log(a^b) = b × log(a).
- On peut utiliser une arithmétique d'intervalles pour des comparaisons approximatives, puis une comparaison de précision complète si nécessaire.

### Complexité
- **Temporelle** : O(n), où n=1000 est le nombre de lignes.
- **Spatiale** : O(1), car on compare une ligne à la fois.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <fstream>
#include <sstream>

int comparePowers(int xBase, int xExp, int yBase, int yExp) {
    // Utiliser les logarithmes pour comparer
    // xBase^xExp > yBase^yExp si xExp * log(xBase) > yExp * log(yBase)
    double xLog = xExp * std::log(static_cast<double>(xBase));
    double yLog = yExp * std::log(static_cast<double>(yBase));
    
    if (xLog > yLog) return 1;
    if (xLog < yLog) return -1;
    return 0;
}

int main() {
    std::vector<std::pair<int, int>> data;
    
    // Lire depuis le fichier (simplifié)
    // std::ifstream file("base_exp.txt");
    // std::string line;
    // while (std::getline(file, line)) {
    //     std::stringstream ss(line);
    //     int base, exp;
    //     char comma;
    //     ss >> base >> comma >> exp;
    //     data.push_back({base, exp});
    // }
    
    std::pair<int, int> maxVal = data[0];
    int maxIndex = 0;
    
    for (size_t i = 1; i < data.size(); i++) {
        if (comparePowers(data[i].first, data[i].second, 
                         maxVal.first, maxVal.second) > 0) {
            maxVal = data[i];
            maxIndex = i;
        }
    }
    
    std::cout << (maxIndex + 1) << std::endl;
    return 0;
}
```
