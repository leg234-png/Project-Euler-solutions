# Partitions de pièces

## Description du Problème
Soit p(n) le nombre de façons différentes dont n pièces peuvent être séparées en piles. Par exemple, cinq pièces peuvent être séparées en piles d'exactement sept façons différentes, donc p(5) = 7 :

OOO OO
OOO O O
OO OO O
OO O O O
O O O O O

**Objectif** : Trouver la plus petite valeur de n pour laquelle p(n) est divisible par un million.

---

## Exemples
### Exemple 1
**Entrée** : n = 5
**Sortie** : `7`
**Explication** : Il y a 7 façons de séparer 5 pièces en piles.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique pour calculer les nombres de partitions modulo 1 000 000.
2. La fonction de partition p(n) peut être calculée récursivement.
3. On utilise la formule de récurrence : p(n) = Σ(-1)^(k+1) [p(n - k(3k-1)/2) + p(n - k(3k+1)/2)] pour k ≥ 1.
4. Cependant, pour ce problème, on peut utiliser une approche plus simple : p(n) = Σ p(n-k) pour k de 1 à n.
5. Calculer modulo 1 000 000 et chercher le premier n tel que p(n) ≡ 0 (mod 1 000 000).

### Observations
- On calcule les partitions modulo 1 000 000 pour éviter les grands nombres.
- On utilise une recherche avec des limites croissantes jusqu'à trouver la solution.
- La formule simple p(n) = Σ p(n-k) fonctionne mais est moins efficace que la formule pentagonale.

### Complexité
- **Temporelle** : O(n²), où n est la valeur recherchée.
- **Spatiale** : O(n), pour stocker les partitions.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

int search(int limit) {
    const int MODULUS = 1000000;
    vector<int> partitions(limit, 0);
    partitions[0] = 1;
    
    for (int i = 1; i < limit; i++) {
        for (int j = i; j < limit; j++)
            partitions[j] = (partitions[j] + partitions[j - i]) % MODULUS;
    }
    
    for (int i = 0; i < limit; i++) {
        if (partitions[i] == 0)
            return i;
    }
    return -1;
}

int main() {
    for (int limit = 1; ; limit *= 2) {
        int result = search(limit);
        if (result != -1) {
            cout << result << endl;
            return 0;
        }
    }
    return 1;
}
```
