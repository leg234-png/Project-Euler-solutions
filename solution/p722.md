# Série à convergence lente

## Description du Problème
Soit E_k(q) = ∑_{n=1}^∞ σ_k(n) q^n, où σ_k(n) est la somme des k-ièmes puissances des diviseurs de n.

Par exemple, E₁(1-1/2⁴) = 3.872155809243e2, E₃(1-1/2⁸) = 2.767385314772e10, E₇(1-1/2¹⁵) = 6.725803486744e39.

Toutes les valeurs ci-dessus sont données en notation scientifique arrondie à douze chiffres après la virgule décimale.

**Objectif** : Trouver la valeur de E₁₅(1-1/2²⁵). Donner la réponse en notation scientifique arrondie à douze chiffres après la virgule décimale.

---

## Exemples
### Exemple 1
**Entrée** : `k = 1, q = 1-1/2⁴`
**Sortie** : `3.872155809243e2`
**Explication** : E₁(1-1/2⁴) = 3.872155809243e2.

---

## Approche de Résolution

### Stratégie
1. Utiliser la relation : ∑_{n=1}^∞ q^n σ_k(n) = ζ(k+1) × ∑_{n=1}^∞ n^k q^n.
2. Calculer ζ(16) (fonction zêta de Riemann).
3. Calculer ∑_{n=1}^∞ n¹⁵ q^n où q = 1-1/2²⁵.
4. Multiplier les deux résultats.
5. Utiliser des approximations numériques pour les séries infinies.

### Observations
- On peut utiliser la fonction zêta de Riemann pour simplifier.
- La série ∑ n^k q^n peut être calculée analytiquement.
- On peut utiliser des bibliothèques mathématiques pour ζ(16).

### Complexité
- **Temporelle** : O(1) pour le calcul analytique.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <cmath>

using namespace std;

double zeta(int k) {
    // Approximation de ζ(k) pour k pair
    // ζ(16) peut être calculé analytiquement
    return 1.000015282259408651871732571487636722023237388990471531; // ζ(16)
}

double geometric_series_sum(int k, double q) {
    // ∑_{n=1}^∞ n^k q^n
    // Pour k=15, on peut utiliser des formules analytiques
    // Approximation numérique
    double total = 0.0;
    for (int n = 1; n < 1000000; n++) {
        total += pow(n, k) * pow(q, n);
        if (pow(q, n) < 1e-20) break;
    }
    return total;
}

double compute() {
    double q = 1.0 - 1.0 / pow(2, 25);
    double zeta_16 = zeta(16);
    double series_sum = geometric_series_sum(15, q);
    return zeta_16 * series_sum;
}

int main() {
    double result = compute();
    cout << scientific << setprecision(12) << result << endl;
    return 0;
}
```
