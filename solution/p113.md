# Non-bouncy Numbers

## Description du Problème
En travaillant de gauche à droite, si aucun chiffre n'est dépassé par le chiffre à sa gauche, on l'appelle un nombre croissant ; par exemple, 134468.

De même, si aucun chiffre n'est dépassé par le chiffre à sa droite, on l'appelle un nombre décroissant ; par exemple, 66420.

Nous appellerons un entier positif qui n'est ni croissant ni décroissant un nombre "rebondissant" ; par exemple, 155349.

À mesure que n augmente, la proportion de nombres rebondissants en dessous de n augmente de telle sorte qu'il n'y a que 12951 nombres en dessous d'un million qui ne sont pas rebondissants et seulement 277032 nombres non rebondissants en dessous de 10¹⁰.

**Objectif** : Combien de nombres en dessous d'un googol (10¹⁰⁰) ne sont pas rebondissants ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 10^6`
**Sortie** : `12951`
**Explication** : Il y a 12951 nombres non rebondissants en dessous d'un million.

---

## Approche de Résolution

### Stratégie
1. Utiliser la combinatoire pour compter les nombres croissants et décroissants.
2. Pour les nombres croissants : C(n+9, 9) - 1 (soustraire 1 pour exclure 0).
3. Pour les nombres décroissants : C(n+10, 10) - (n+1) (soustraire les façons de former 0).
4. Soustraire les nombres "plats" (tous les chiffres identiques) qui sont comptés deux fois.
5. Il y a 9n nombres plats (1, 2, ..., 9; 11, 22, ..., 99; etc.).

### Observations
- Les nombres croissants peuvent être modélisés comme des arrangements de n emplacements et 9 incréments.
- Les nombres décroissants peuvent être modélisés comme des arrangements de n emplacements et 10 opérations (1 incrément vers 9, 9 décréments).
- On utilise les coefficients binomiaux pour compter.

### Complexité
- **Temporelle** : O(1) pour calculer les coefficients binomiaux.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <gmpxx.h>

mpz_class binomial(int n, int k) {
    if (k > n - k) k = n - k;
    mpz_class result = 1;
    for (int i = 0; i < k; i++) {
        result = result * (n - i) / (i + 1);
    }
    return result;
}

int main() {
    const int DIGITS = 100;
    mpz_class increasing = binomial(DIGITS + 9, 9) - 1;
    mpz_class decreasing = binomial(DIGITS + 10, 10) - (DIGITS + 1);
    mpz_class flat = DIGITS * 9;
    
    mpz_class result = increasing + decreasing - flat;
    std::cout << result << std::endl;
    return 0;
}
```
