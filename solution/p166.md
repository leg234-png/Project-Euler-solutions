# Criss Cross

## Description du Problème
Un carré magique 4×4 est un carré où chaque ligne, colonne et les deux diagonales ont la même somme.

**Objectif** : Combien de carrés magiques 4×4 distincts peuvent être formés en utilisant les chiffres de 0 à 9 ?

---

## Exemples
### Exemple 1
**Entrée** : Grille 4×4
**Sortie** : Nombre de carrés magiques distincts
**Explication** : Toutes les lignes, colonnes et diagonales doivent avoir la même somme.

---

## Approche de Résolution

### Stratégie
1. Utiliser l'algèbre linéaire pour réduire le nombre de variables libres.
2. Avec 16 variables et 10 contraintes (4 lignes, 4 colonnes, 2 diagonales), il y a 8 variables libres.
3. Fixer 8 variables : `{a, b, c, d, e, g, i, k}`.
4. Calculer les autres variables en fonction de ces 8 :
   - `m = b + c + d - e - i`
   - `o = a + b + d - g - k`
   - `j = a + b + c - g - m`
   - `l = a + b + c + d - i - j - k`
   - `f = b + c + 2d - e - i - k`
   - `h = a + b + c + d - e - f - g`
   - `n = a + c + d - f - j`
   - `p = a + b + c - h - l`
5. Vérifier que toutes les variables sont entre 0 et 9.
6. Compter tous les carrés valides.

### Observations
- On peut réduire le problème à 8 variables libres en utilisant les équations linéaires.
- Chaque variable doit être un chiffre (0-9).

### Complexité
- **Temporelle** : O(10^8) pour itérer sur toutes les combinaisons de 8 variables.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>

using namespace std;

bool increment(int* num, int len) {
    int i = 0;
    while (i < len && num[i] == 9) {
        num[i] = 0;
        i++;
    }
    if (i == len) return false;
    num[i]++;
    return true;
}

int main() {
    int count = 0;
    int num[8] = {0};  // Counter in base 10, little-endian
    
    do {
        int a = num[0], b = num[1], c = num[2], d = num[3];
        int e = num[4], g = num[5], i = num[6], k = num[7];
        
        int m = b + c + d - e - i;
        if (m < 0 || m > 9) continue;
        
        int o = a + b + d - g - k;
        if (o < 0 || o > 9) continue;
        
        int j = a + b + c - g - m;
        if (j < 0 || j > 9) continue;
        
        int l = a + b + c + d - i - j - k;
        if (l < 0 || l > 9) continue;
        
        int f = b + c + 2*d - e - i - k;
        if (f < 0 || f > 9) continue;
        
        int h = a + b + c + d - e - f - g;
        if (h < 0 || h > 9) continue;
        
        int n = a + c + d - f - j;
        if (n < 0 || n > 9) continue;
        
        int p = a + b + c - h - l;
        if (p < 0 || p > 9) continue;
        
        count++;
    } while (increment(num, 8));
    
    cout << count << endl;
    return 0;
}
```
