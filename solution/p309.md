# Échelles entières

## Description du Problème
Ici, nous ne nous préoccupons que des instances où les quatre variables sont des entiers positifs.

Par exemple, si x=70, y=119 et h=30, on peut calculer que w=56.

En fait, pour des valeurs entières x, y, h et 0<x<y<200, il n'y a que cinq triplets (x,y,h) produisant des solutions entières pour w :
(70,119,30), (74,182,21), (87,105,35), (100,116,35) et (119,175,40).

**Objectif** : Pour des valeurs entières x,y,h et 0<x<y<10⁶, combien de triplets (x,y,h) produisent des solutions entières pour w ?

---

## Exemples
### Exemple 1
**Entrée** : `x=70, y=119, h=30`
**Sortie** : `w=56`
**Explication** : w = 56 est la solution entière.

---

## Approche de Résolution

### Stratégie
1. Utiliser le problème des échelles croisées.
2. Soit A² + w² = y² et B² + w² = x².
3. Utiliser les triplets pythagoriciens pour trouver des triplets qui ont un w correspondant.
4. Tester si h est un entier : h = AB/(A + B).
5. Générer tous les triplets pythagoriciens et trouver ceux qui partagent le même w.
6. Pour chaque paire (A, B) avec le même w, vérifier si h = AB/(A+B) est un entier.

### Observations
- Le triangle BDA est une version agrandie de BEF, donc h/w₁ = B/w.
- De même, ACD est agrandi de AEF => h/w₂ = A/w.
- Donc w = w₁ + w₂ = hw/B + hw/A => h = AB/(A + B).
- On doit trouver des triplets pythagoriciens qui partagent le même w.

### Complexité
- **Temporelle** : O(limit²) pour générer tous les triplets pythagoriciens.
- **Spatiale** : O(limit) pour stocker les triplets par w.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <map>
#include <set>
#include <cmath>

using namespace std;

vector<vector<pair<int, int>>> ppt(int limit) {
    vector<vector<pair<int, int>>> triples(limit + 1);
    
    for (int m = 2; m * m <= limit; m++) {
        for (int n = 1; n < m; n++) {
            if ((m + n) % 2 == 1 && __gcd(m, n) == 1) {
                int a = m * m + n * n;
                int b = m * m - n * n;
                int c = 2 * m * n;
                
                for (int k = 1; k * a <= limit; k++) {
                    if (k * b <= limit) {
                        triples[k * b].push_back({k * c, k * a});
                        triples[k * c].push_back({k * b, k * a});
                    }
                }
            }
        }
    }
    return triples;
}

long long compute(int limit) {
    vector<vector<pair<int, int>>> triangles = ppt(limit);
    set<vector<int>> valid_triples;
    
    for (int w = 1; w <= limit; w++) {
        if (triangles[w].size() < 2) continue;
        
        for (size_t i = 0; i < triangles[w].size(); i++) {
            int A = triangles[w][i].first;
            int y = triangles[w][i].second;
            
            for (size_t j = i + 1; j < triangles[w].size(); j++) {
                int B = triangles[w][j].first;
                int x = triangles[w][j].second;
                
                if (x < y && (long long)A * B % (A + B) == 0) {
                    int h = (A * B) / (A + B);
                    if (h > 0 && x < limit && y < limit) {
                        vector<int> triple = {x, y, h};
                        valid_triples.insert(triple);
                    }
                }
            }
        }
    }
    
    return valid_triples.size();
}

int main() {
    cout << compute(1000000) << endl;
    return 0;
}
```
