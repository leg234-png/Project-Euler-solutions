# Noms Scores

## Description du Problème
En utilisant names.txt (un fichier texte de 46K contenant plus de cinq mille prénoms), commencez par le trier par ordre alphabétique. Ensuite, calculez la valeur alphabétique de chaque nom, multipliez cette valeur par sa position dans la liste triée, et obtenez la somme de ces produits.

Par exemple, lorsque la liste est triée par ordre alphabétique, COLIN, qui vaut 3 + 15 + 12 + 9 + 14 = 53, est le 938ème nom dans la liste. Donc, COLIN obtiendrait un score de 938 × 53 = 49714.

**Objectif** : Quel est le total de tous les scores de noms dans le fichier ?

---

## Exemples
### Exemple 1
**Entrée** : Liste de noms triée
**Sortie** : `871198282`
**Explication** : La somme de tous les scores de noms.

---

## Approche de Résolution

### Stratégie
1. Lire tous les noms du fichier.
2. Trier les noms par ordre alphabétique.
3. Pour chaque nom à la position i (1-indexée) :
   - Calculer la valeur alphabétique (somme des positions des lettres : A=1, B=2, ..., Z=26).
   - Multiplier par la position dans la liste.
   - Ajouter au total.
4. Retourner le total.

### Observations
- La valeur alphabétique d'une lettre est sa position dans l'alphabet (A=1, B=2, ..., Z=26).
- On peut calculer la valeur d'une lettre en soustrayant 'A' et en ajoutant 1.
- Le score d'un nom est : position × valeur_alphabétique.

### Complexité
- **Temporelle** : O(n log n + n × m), où `n` est le nombre de noms et `m` la longueur moyenne d'un nom.
- **Spatiale** : O(n × m), pour stocker tous les noms.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
#include <fstream>
#include <sstream>

int nameValue(const std::string& name) {
    int value = 0;
    for (char c : name) {
        value += c - 'A' + 1;
    }
    return value;
}

int main() {
    // Note: Le fichier names.txt devrait être lu ici
    // Pour cet exemple, on suppose que les noms sont dans un vecteur
    std::vector<std::string> names = {
        "MARY", "PATRICIA", "LINDA", "BARBARA", "ELIZABETH", "JENNIFER", "MARIA", "SUSAN", "MARGARET", "DOROTHY"
        // ... (tous les noms du fichier)
    };
    
    std::sort(names.begin(), names.end());
    
    long long sum = 0;
    for (size_t i = 0; i < names.size(); ++i) {
        sum += (i + 1) * nameValue(names[i]);
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
