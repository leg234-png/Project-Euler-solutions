# Somme de séquence récursive

## Description du Problème
Soit a₁ = 1 et pour n ≥ 1 :
- a_{2n} = 2a_n
- a_{2n+1} = a_n - 3a_{n+1}

Les dix premiers termes sont 1, 2, -5, 4, 17, -10, -17, 8, -47, 34.

Définissons S(N) = ∑_{n=1}^N a_n. On vous donne S(10) = -13.

**Objectif** : Trouver S(10¹²).

---

## Exemples
### Exemple 1
**Entrée** : `N = 10`
**Sortie** : `-13`
**Explication** : S(10) = -13.

---

## Approche de Résolution

### Stratégie
1. Utiliser la récurrence pour calculer a_n.
2. Expansion simple de S(N) donne :
   - S(N) = 4 - a_{N//2} si N est pair
   - S(N) = 4 - 3a_{N//2+1} si N est impair
3. Utiliser la mémorisation pour calculer efficacement a_n.
4. Calculer récursivement a_n en utilisant les relations de récurrence.

### Observations
- La formule pour S(N) permet de calculer directement sans sommer tous les termes.
- On peut utiliser la mémorisation pour éviter les recalculs.
- La complexité est O(log N) pour calculer a_n.

### Complexité
- **Temporelle** : O(log N) pour calculer S(N) avec mémorisation.
- **Spatiale** : O(log N) pour la table de mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <unordered_map>

using namespace std;

unordered_map<long long, long long> memo;

long long a(long long n) {
    if (n == 1) return 1;
    if (memo.find(n) != memo.end()) {
        return memo[n];
    }
    
    long long result;
    if (n % 2 == 0) {
        result = 2 * a(n / 2);
    } else {
        result = a(n / 2) - 3 * a(n / 2 + 1);
    }
    
    memo[n] = result;
    return result;
}

long long S(long long N) {
    if (N % 2 == 0) {
        return 4 - a(N / 2);
    } else {
        return 4 - 3 * a(N / 2 + 1);
    }
}

int main() {
    long long N = 1;
    for (int i = 0; i < 12; i++) {
        N *= 10;
    }
    cout << S(N) << endl;
    return 0;
}
```
