# Différence entre Somme des Carrés et Carré de la Somme

## Description du Problème
La somme des carrés des dix premiers nombres naturels est :

1² + 2² + ... + 10² = 385

Le carré de la somme des dix premiers nombres naturels est :

(1 + 2 + ... + 10)² = 55² = 3025

La différence entre la somme des carrés des dix premiers nombres naturels et le carré de la somme est 3025 − 385 = 2640.

**Objectif** : Trouver la différence entre la somme des carrés des cent premiers nombres naturels et le carré de la somme.

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : `2640`
**Explication** : (1+2+...+10)² - (1²+2²+...+10²) = 55² - 385 = 3025 - 385 = 2640.

---

## Approche de Résolution

### Stratégie
1. Calculer la somme des carrés : 1² + 2² + ... + n²
2. Calculer le carré de la somme : (1 + 2 + ... + n)²
3. Retourner la différence entre les deux

### Observations
- On peut utiliser des formules fermées pour optimiser :
  - Somme : S = n(n + 1) / 2
  - Somme des carrés : S₂ = n(n + 1)(2n + 1) / 6
  - Différence : S² - S₂ = (n⁴/4) + (n³/6) - (n²/4) - (n/6)
- Pour n = 100, on peut aussi calculer directement avec une boucle.

### Complexité
- **Temporelle** : O(n), où `n` est le nombre de termes (100).
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>

int main() {
    const int n = 100;
    int sum = 0;
    int sum2 = 0;
    
    for (int i = 1; i <= n; ++i) {
        sum += i;
        sum2 += i * i;
    }
    
    std::cout << (sum * sum - sum2) << std::endl;
    return 0;
}
```
