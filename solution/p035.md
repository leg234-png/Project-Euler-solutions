# Nombres Premiers Circulaires

## Description du Problème
Le nombre 197 est appelé un nombre premier circulaire car toutes les rotations de ses chiffres : 197, 971, et 719, sont elles-mêmes premières.

Il y a treize de ces nombres premiers en dessous de 100 : 2, 3, 5, 7, 11, 13, 17, 31, 37, 71, 73, 79, et 97.

**Objectif** : Combien y a-t-il de nombres premiers circulaires en dessous d'un million ?

---

## Exemples
### Exemple 1
**Entrée** : Limite = 100
**Sortie** : `13`
**Explication** : Il y a 13 nombres premiers circulaires en dessous de 100.

---

## Approche de Résolution

### Stratégie
1. Utiliser le crible d'Ératosthène pour trouver tous les nombres premiers jusqu'à 1 000 000.
2. Pour chaque nombre premier :
   - Générer toutes les rotations de ses chiffres.
   - Vérifier si toutes les rotations sont premières.
   - Si oui, compter ce nombre.
3. Retourner le compte.

### Observations
- Pour générer les rotations d'un nombre, on peut décaler les chiffres circulairement.
- Par exemple, pour 197 : 197 → 971 → 719 → 197.
- On peut optimiser en évitant de tester les nombres qui contiennent des chiffres pairs (sauf 2) ou 5, car leurs rotations ne seront pas toutes premières.

### Complexité
- **Temporelle** : O(n log log n + n × d), où `n` = 1 000 000 et `d` est le nombre de chiffres.
- **Spatiale** : O(n), pour le crible.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <cmath>

std::vector<bool> sieve(int n) {
    std::vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    int sqrtN = sqrt(n);
    for (int i = 2; i <= sqrtN; ++i) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    return isPrime;
}

bool isCircularPrime(int n, const std::vector<bool>& isPrime) {
    std::string s = std::to_string(n);
    int len = s.length();
    
    for (int i = 0; i < len; ++i) {
        int rotated = std::stoi(s.substr(i) + s.substr(0, i));
        if (!isPrime[rotated]) {
            return false;
        }
    }
    
    return true;
}

int main() {
    const int limit = 1000000;
    std::vector<bool> isPrime = sieve(limit - 1);
    
    int count = 0;
    for (int i = 0; i < limit; ++i) {
        if (isPrime[i] && isCircularPrime(i, isPrime)) {
            count++;
        }
    }
    
    std::cout << count << std::endl;
    return 0;
}
```
