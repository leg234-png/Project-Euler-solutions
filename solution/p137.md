# Fibonacci Golden Nuggets

## Description du Problème
Considérons la série polynomiale infinie A_F(x) = x F₁ + x² F₂ + x³ F₃ + ⋯, où Fₖ est le k-ième terme de la séquence de Fibonacci : 1, 1, 2, 3, 5, 8, … ; c'est-à-dire, Fₖ = F_{k-1} + F_{k-2}, F₁ = 1 et F₂ = 1.

Pour ce problème, nous nous intéresserons aux valeurs de x pour lesquelles A_F(x) est un entier positif.

Étonnamment, A_F(1/2) = (1/2) × 1 + (1/2)² × 1 + (1/2)³ × 2 + (1/2)⁴ × 3 + (1/2)⁵ × 5 + ⋯ = 1/2 + 1/4 + 2/8 + 3/16 + 5/32 + ⋯ = 2.

Les valeurs correspondantes de x pour les cinq premiers nombres naturels sont montrées ci-dessous.

Nous appellerons A_F(x) un "golden nugget" si x est rationnel, car ils deviennent de plus en plus rares ; par exemple, le 10ème golden nugget est 74049690.

**Objectif** : Trouver le 15ème golden nugget.

---

## Exemples
### Exemple 1
**Entrée** : `x = 1/2`
**Sortie** : `2`
**Explication** : A_F(1/2) = 2.

---

## Approche de Résolution

### Stratégie
1. La série A_F(x) peut être résolue analytiquement.
2. Si A_F(x) = n, alors x satisfait une équation quadratique.
3. Pour que x soit rationnel, le discriminant doit être un carré parfait.
4. Cela conduit à une équation de Pell ou similaire.
5. Les golden nuggets suivent une séquence récurrente basée sur les nombres de Fibonacci.

### Observations
- Les golden nuggets sont liés aux nombres de Fibonacci.
- Le n-ième golden nugget peut être calculé à partir des nombres de Fibonacci.

### Complexité
- **Temporelle** : O(n) pour calculer le n-ième golden nugget.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <gmpxx.h>

int main() {
    // Les golden nuggets suivent une séquence basée sur les nombres de Fibonacci
    // Le n-ième golden nugget est F(2n) * F(2n+1)
    
    mpz_class f1 = 1, f2 = 1;
    for (int i = 2; i <= 30; i++) {
        mpz_class temp = f2;
        f2 = f1 + f2;
        f1 = temp;
    }
    
    // Le 15ème golden nugget
    mpz_class result = f1 * f2;
    std::cout << result << std::endl;
    return 0;
}
```
