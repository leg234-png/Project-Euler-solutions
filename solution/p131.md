# Prime Cube Partnership

## Description du Problème
Il existe certaines valeurs premières, p, pour lesquelles il existe un entier positif, n, tel que l'expression n³ + n²p est un cube parfait.

Par exemple, lorsque p = 19, 8³ + 8² × 19 = 12³.

Ce qui est peut-être le plus surprenant, c'est que pour chaque nombre premier avec cette propriété, la valeur de n est unique, et il n'y a que quatre tels nombres premiers en dessous de cent.

**Objectif** : Combien de nombres premiers en dessous d'un million ont cette propriété remarquable ?

---

## Exemples
### Exemple 1
**Entrée** : `p = 19`
**Sortie** : `true`
**Explication** : 8³ + 8² × 19 = 512 + 1216 = 1728 = 12³.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre premier p, chercher n tel que n³ + n²p = k³ pour un certain k.
2. Réécrire : n³ + n²p = k³ implique n²(n + p) = k³.
3. Pour que cela soit un cube, n et n+p doivent être des cubes ou des multiples de cubes.
4. Soit n = a³ et n+p = b³, donc p = b³ - a³ = (b-a)(b²+ab+a²).
5. Comme p est premier, b-a = 1 et p = 3a² + 3a + 1.
6. Parcourir les valeurs de a et vérifier si 3a² + 3a + 1 est premier et < 10⁶.

### Observations
- On peut dériver une formule directe : p = 3a² + 3a + 1.
- On doit vérifier que p est premier et < 10⁶.

### Complexité
- **Temporelle** : O(√n), où n = 10⁶.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

bool isPrime(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    for (int i = 3; i * i <= n; i += 2) {
        if (n % i == 0) return false;
    }
    return true;
}

int main() {
    int count = 0;
    for (int a = 1; ; a++) {
        int p = 3 * a * a + 3 * a + 1;
        if (p >= 1000000) break;
        if (isPrime(p)) {
            count++;
        }
    }
    std::cout << count << std::endl;
    return 0;
}
```
