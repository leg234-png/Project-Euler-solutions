# Produit de factorielles de Gauss

## Description du Problème
Soit g(n) le produit de tous les nombres positifs ≤ n qui sont premiers avec n.

Par exemple, g(6) = 1 × 5 = 5.

Définissons aussi G(n) = ∏_{i=1}^n g(i).

On vous donne G(10) = 23044331520000.

**Objectif** : Trouver G(10⁸). Donner votre réponse modulo 1000000007.

---

## Exemples
### Exemple 1
**Entrée** : `n = 6`
**Sortie** : `5`
**Explication** : g(6) = 1 × 5 = 5.

---

## Approche de Résolution

### Stratégie
1. g(n) = n^φ(n) × ∏_{d|n} (d!/d^d)^μ(n/d) où μ est la fonction de Möbius.
2. Utiliser un crible pour calculer φ(n) et μ(n) pour tous les n ≤ 10⁸.
3. Précalculer les factorielles modulo 1000000007.
4. Calculer g(n) pour chaque n en utilisant la formule.
5. Multiplier tous les g(n) pour obtenir G(n).

### Observations
- On peut utiliser un crible pour calculer efficacement φ et μ.
- Les factorielles peuvent être précalculées.
- On doit gérer l'arithmétique modulaire correctement.

### Complexité
- **Temporelle** : O(n log log n) avec un crible optimisé.
- **Spatiale** : O(n) pour les tableaux.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long mod = 1000000007;

pair<vector<int>, vector<int>> totient_mobius_sieve(int n) {
    vector<int> phi(n + 1);
    vector<int> mob(n + 1, 1);
    
    for (int i = 0; i <= n; i++) {
        phi[i] = i;
    }
    mob[0] = 0;
    
    for (int p = 2; p <= n; p++) {
        if (phi[p] == p) {
            for (int i = p; i <= n; i += p) {
                phi[i] -= phi[i] / p;
                mob[i] *= -1;
            }
            long long sq = (long long)p * p;
            if (sq <= n) {
                for (int j = sq; j <= n; j += sq) {
                    mob[j] = 0;
                }
            }
        }
    }
    
    return {phi, mob};
}

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long compute(int n, long long mod) {
    auto [phi, mob] = totient_mobius_sieve(n);
    
    vector<long long> factorial(n + 1, 1);
    for (int i = 2; i <= n; i++) {
        factorial[i] = (factorial[i - 1] * i) % mod;
    }
    
    long long G = 1;
    for (int i = 1; i <= n; i++) {
        long long g_i = mod_pow(i, phi[i], mod);
        
        // Calculer ∏_{d|i} (d!/d^d)^μ(i/d)
        for (int d = 1; d <= i; d++) {
            if (i % d == 0) {
                long long term = (factorial[d] * mod_pow(mod_pow(d, d, mod), mod - 2, mod)) % mod;
                term = mod_pow(term, mob[i / d] + mod, mod);
                g_i = (g_i * term) % mod;
            }
        }
        
        G = (G * g_i) % mod;
    }
    
    return G;
}

int main() {
    cout << compute(100000000, mod) << endl;
    return 0;
}
```
