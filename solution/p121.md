# Disc Game Prize Fund

## Description du Problème
Un sac contient un disque rouge et un disque bleu. Dans un jeu de hasard, un joueur prend un disque au hasard et sa couleur est notée. Après chaque tour, le disque est retourné dans le sac, un disque rouge supplémentaire est ajouté, et un autre disque est pris au hasard.

Le joueur paie £1 pour jouer et gagne s'il a pris plus de disques bleus que de disques rouges à la fin du jeu.

Si le jeu est joué pendant quatre tours, la probabilité qu'un joueur gagne est exactement 11/120, et donc le fonds de prix maximum que le banquier devrait allouer pour gagner dans ce jeu serait £10 avant qu'il ne s'attende à subir une perte. Notez que tout paiement sera un nombre entier de livres et inclut également la £1 originale payée pour jouer au jeu, donc dans l'exemple donné, le joueur gagne en fait £9.

**Objectif** : Trouver le fonds de prix maximum qui devrait être alloué à un seul jeu dans lequel quinze tours sont joués.

---

## Exemples
### Exemple 1
**Entrée** : `tours = 4`
**Sortie** : `10`
**Explication** : La probabilité de gagner est 11/120, donc le fonds maximum est floor(120/11) = 10.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique pour compter les façons d'accumuler exactement b disques bleus après k tours.
2. f(k, b) = nombre de façons d'accumuler b disques bleus après k tours.
3. Relations de récurrence :
   - f(0, 0) = 1
   - f(k, 0) = k × f(k-1, 0)
   - f(k, b) = f(k-1, b-1) + k × f(k-1, b)
4. Calculer la probabilité de gagner = somme des façons où b > n-b, divisée par (n+1)!.
5. Le fonds maximum = floor(1 / probabilité).

### Observations
- Au début du tour k, il y a k+2 disques au total.
- Le nombre total de résultats possibles est (n+1)!.
- On doit avoir plus de disques bleus que rouges pour gagner.

### Complexité
- **Temporelle** : O(n²), où n=15 est le nombre de tours.
- **Spatiale** : O(n²).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <gmpxx.h>

const int TURNS = 15;

mpz_class factorial(int n) {
    mpz_class result = 1;
    for (int i = 2; i <= n; i++) {
        result *= i;
    }
    return result;
}

int main() {
    // Programmation dynamique
    std::vector<std::vector<mpz_class>> ways(TURNS + 1);
    ways[0] = {1};
    
    for (int i = 1; i <= TURNS; i++) {
        ways[i].resize(i + 1);
        for (int j = 0; j <= i; j++) {
            mpz_class temp = 0;
            if (j < i) {
                temp = ways[i - 1][j] * i;
            }
            if (j > 0) {
                temp += ways[i - 1][j - 1];
            }
            ways[i][j] = temp;
        }
    }
    
    mpz_class numer = 0;
    for (int i = TURNS / 2 + 1; i <= TURNS; i++) {
        numer += ways[TURNS][i];
    }
    
    mpz_class denom = factorial(TURNS + 1);
    std::cout << (denom / numer) << std::endl;
    return 0;
}
```
