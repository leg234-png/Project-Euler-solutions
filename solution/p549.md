# Plus petit factoriel divisible

## Description du Problème
Le plus petit nombre m tel que m! est divisible par n est noté s(n).

Par exemple, s(10) = 5 car 5! = 120 est divisible par 10, mais 4! = 24 ne l'est pas.

**Objectif** : Trouver Σs(n) pour 2 <= n <= 10⁸.

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : `5`
**Explication** : s(10) = 5 car 5! = 120 est divisible par 10.

---

## Approche de Résolution

### Stratégie
1. Factoriser n comme un produit de puissances de nombres premiers uniques : n = p0^k0 × p1^k1 × ...
2. Pour chaque puissance de nombre premier p_i^k_i, trouver le plus petit j tel que j! est divisible par p_i^k_i.
3. Le maximum parmi tous ces j est s(n).
4. Utiliser une modification du crible d'Ératosthène :
   - Pour chaque nombre premier p, itérer j = p, 2p, 3p, ...
   - Pour chaque j, calculer le nombre de facteurs de p dans j!.
   - Mettre à jour tous les multiples de p^k avec j.
5. Continuer jusqu'à ce que p^k dépasse la limite.

### Observations
- j doit être un multiple de p pour que j! contienne des facteurs de p.
- On peut calculer le nombre de facteurs de p dans j! en comptant combien de fois p apparaît dans la factorisation de j.
- On utilise un seul tableau pour cribler les nombres premiers et calculer s(n) simultanément.

### Complexité
- **Temporelle** : O(N log log N), similaire au crible d'Ératosthène.
- **Spatiale** : O(N), pour le tableau s(n).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

const int LIMIT = 100000000;

int main() {
    // Modification du crible d'Ératosthène
    vector<int> smallestDivisibleFactorials(LIMIT + 1, 0);
    
    for (int i = 2; i < smallestDivisibleFactorials.size(); i++) {
        if (smallestDivisibleFactorials[i] == 0) {
            // Maintenant on sait que i est premier
            
            long long power = 1;
            for (int j = i; ; j += i) {
                // On sait que j contient au moins un facteur de i
                power *= i;
                if (power > LIMIT) {
                    break;
                }
                
                // Mettre à jour la réponse pour tous les multiples de 'power'
                for (int k = power; k < smallestDivisibleFactorials.size(); k += power) {
                    smallestDivisibleFactorials[k] = max(j, smallestDivisibleFactorials[k]);
                }
                
                // Mettre à jour power pour inclure les facteurs restants de i dans j
                int temp = j / i;
                while (temp % i == 0) {
                    power *= i;
                    if (power > LIMIT) {
                        goto next_prime;
                    }
                    temp /= i;
                }
                // Maintenant à ce point, power = nombre de facteurs de i dans la factorisation de j!
            }
            next_prime:;
        }
    }
    
    // Ne dépassera pas car 0 <= sum <= LIMIT^2 ~= 2^53
    long long sum = 0;
    for (int x : smallestDivisibleFactorials) {
        sum += x;
    }
    
    cout << sum << endl;
    return 0;
}
```
