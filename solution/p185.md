# Number Mind

## Description du Problème
Un jeu de devinettes où on doit trouver un code secret à partir d'indices indiquant combien de chiffres sont corrects.

Par exemple :
- 90342 ; 2 corrects
- 70794 ; 0 corrects
- 39458 ; 2 corrects
- 34109 ; 1 correct
- 51545 ; 2 corrects
- 12531 ; 1 correct

**Objectif** : Trouver le code secret qui satisfait tous les indices donnés.

---

## Exemples
### Exemple 1
**Entrée** : Indices avec 6 tentatives
**Sortie** : `39542`
**Explication** : Le code qui satisfait tous les indices de l'exemple.

---

## Approche de Résolution

### Stratégie
1. Modéliser le problème comme un système d'équations linéaires.
2. Créer une matrice où a_ij = 1 si le chiffre i apparaît à la position j.
3. Pour chaque position j, la somme des a_ij doit être 1 (un seul chiffre par position).
4. Pour chaque indice, ajouter une contrainte sur la somme des chiffres corrects.
5. Résoudre le système d'équations linéaires avec des variables entières (0 ou 1).

### Observations
- C'est un problème de programmation linéaire en nombres entiers (ILP).
- On peut utiliser des solveurs ILP ou une recherche avec backtracking.
- Les contraintes garantissent qu'exactement un chiffre apparaît à chaque position.

### Complexité
- **Temporelle** : O(10n × m) où n est la longueur du code et m le nombre d'indices, pour un solveur ILP.
- **Spatiale** : O(10n) pour la matrice de contraintes.
