# Nombres semi-divisibles

## Description du Problème
Appelons un entier n≥4 semi-divisible si l'un de lps(n) et ups(n) divise n, mais pas les deux.

La somme des nombres semi-divisibles ne dépassant pas 15 est 30, les nombres sont 8, 10 et 12. 15 n'est pas semi-divisible car il est un multiple de lps(15)=3 et ups(15)=5.

Comme autre exemple, la somme des 92 nombres semi-divisibles jusqu'à 1000 est 34825.

**Objectif** : Quelle est la somme de tous les nombres semi-divisibles ne dépassant pas 999966663333 ?

---

## Exemples
### Exemple 1
**Entrée** : `limit = 15`
**Sortie** : `30`
**Explication** : Les nombres semi-divisibles sont 8, 10, 12, leur somme est 30.

---

## Approche de Résolution

### Stratégie
1. Trouver tous les facteurs premiers inférieurs à √N et le facteur premier suivant.
2. Vérifier chaque paire de nombres premiers consécutifs.
3. Pour chaque paire (lps, ups), tous les nombres entre lps² et ups² auront lps et ups comme facteurs premiers inférieurs et supérieurs.
4. Calculer la somme des nombres divisibles par lps mais pas par ups, et vice versa.
5. Soustraire les nombres divisibles par les deux (lps × ups).

### Observations
- Pour chaque paire de nombres premiers consécutifs, on peut calculer efficacement la somme.
- On utilise des formules de somme arithmétique pour calculer rapidement.
- On doit éviter de compter deux fois les nombres divisibles par les deux.

### Complexité
- **Temporelle** : O(√N / log √N) pour trouver les nombres premiers, plus O(N / p) pour chaque paire.
- **Spatiale** : O(√N) pour stocker les nombres premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<bool> list_primality(long long n) {
    vector<bool> result(n + 1, true);
    result[0] = result[1] = false;
    for (long long i = 2; i * i <= n; i++) {
        if (result[i]) {
            for (long long j = 2 * i; j <= n; j += i) {
                result[j] = false;
            }
        }
    }
    return result;
}

vector<long long> list_primes(long long n) {
    vector<bool> is_prime = list_primality(n);
    vector<long long> primes;
    for (long long i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

long long sumseq(long long ups, long long lps, long long limit) {
    long long total = 0;
    
    // Nombres divisibles par lps dans [lps², ups²]
    for (long long x = lps*lps + lps; x <= min(ups*ups, limit); x += lps) {
        if (x % ups != 0) {
            total += x;
        }
    }
    
    // Nombres divisibles par ups dans [lps², ups²]
    for (long long x = ups*ups - ups; x > lps*lps && x <= limit; x -= ups) {
        if (x % lps != 0) {
            total += x;
        }
    }
    
    return total;
}

long long compute(long long limit) {
    long long sqrt_limit = sqrt(limit);
    vector<long long> primes = list_primes(sqrt_limit);
    
    long long total = 0;
    for (size_t i = 0; i < primes.size() - 1; i++) {
        long long lps = primes[i];
        long long ups = primes[i + 1];
        total += sumseq(ups, lps, limit);
    }
    
    return total;
}

int main() {
    cout << compute(999966663333LL) << endl;
    return 0;
}
```
