# Eulercoin

## Description du Problème
Leonhard Euler est né le 15 avril 1707.

Considérons la séquence 1504170715041707n mod 4503599627370517.

Un élément de cette séquence est défini comme un Eulercoin s'il est strictement plus petit que tous les Eulercoins précédemment trouvés.

Par exemple, le premier terme est 1504170715041707 qui est le premier Eulercoin. Le deuxième terme est 3008341430083414 qui est plus grand que 1504170715041707 donc n'est pas un Eulercoin. Cependant, le troisième terme est 8912517754604 qui est assez petit pour être un nouvel Eulercoin.

La somme des 2 premiers Eulercoins est donc 1513083232796311.

**Objectif** : Trouver la somme de tous les Eulercoins.

---

## Exemples
### Exemple 1
**Entrée** : `Séquence`
**Sortie** : `1513083232796311`
**Explication** : La somme des 2 premiers Eulercoins.

---

## Approche de Résolution

### Stratégie
1. Générer la séquence 1504170715041707n mod 4503599627370517.
2. Garder une trace du plus petit Eulercoin trouvé jusqu'à présent.
3. Quand on trouve un nouveau Eulercoin, l'ajouter à la liste.
4. Continuer jusqu'à ce qu'on atteigne un point fixe (quand le plus petit Eulercoin est 1).
5. Utiliser l'inverse modulaire pour trouver efficacement les prochains Eulercoins.
6. Sommer tous les Eulercoins trouvés.

### Observations
- On peut utiliser l'inverse modulaire pour sauter rapidement aux prochains candidats.
- Il y a un nombre fini d'Eulercoins.
- On peut optimiser en utilisant les propriétés de la séquence.

### Complexité
- **Temporelle** : O(E) où E est le nombre d'Eulercoins.
- **Spatiale** : O(E) pour stocker les Eulercoins.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long mod_inverse(long long a, long long mod) {
    return mod_pow(a, mod - 2, mod);
}

long long compute() {
    long long a = 1504170715041707LL;
    long long m = 4503599627370517LL;
    
    vector<long long> eulercoins;
    eulercoins.push_back(a);
    long long current_min = a;
    long long n = 2;
    
    while (current_min > 1) {
        long long value = (a * n) % m;
        if (value < current_min) {
            current_min = value;
            eulercoins.push_back(value);
        }
        n++;
    }
    
    // Continuer avec l'inverse modulaire pour trouver les autres
    long long inv = mod_inverse(a, m);
    long long target = 1;
    
    while (target < current_min) {
        long long n_val = (target * inv) % m;
        if (n_val > 0 && (a * n_val) % m == target) {
            eulercoins.push_back(target);
            current_min = target;
        }
        target++;
    }
    
    long long total = 0;
    for (long long coin : eulercoins) {
        total += coin;
    }
    
    return total;
}

int main() {
    cout << compute() << endl;
    return 0;
}
```
