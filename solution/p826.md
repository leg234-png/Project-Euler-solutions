# Oiseaux sur un fil

## Description du Problème
n oiseaux sont placés aléatoirement sur un fil de longueur 1. Chaque oiseau choisit une position uniformément au hasard sur le fil, indépendamment des autres oiseaux.

Un oiseau peint le fil depuis sa position jusqu'au plus proche voisin (qui peut être à l'une ou l'autre extrémité). Si deux oiseaux sont à la même distance, l'oiseau peint vers le voisin de gauche.

Définissons F(n) comme la longueur attendue du fil qui est peint. On vous donne F(3) = 0.5.

**Objectif** : Trouver la moyenne de F(n) où n parcourt tous les nombres premiers impairs inférieurs à un million. Donner votre réponse arrondie à 10 places après la virgule décimale.

---

## Exemples
### Exemple 1
**Entrée** : `n = 3`
**Sortie** : `0.5`
**Explication** : F(3) = 0.5.

---

## Approche de Résolution

### Stratégie
1. Utiliser une approximation avec un grand nombre d'essais (Monte Carlo).
2. Pour chaque n, simuler de nombreuses fois le placement aléatoire des oiseaux.
3. Calculer la longueur moyenne du fil peint.
4. Faire la moyenne sur tous les nombres premiers impairs < 10⁶.
5. Utiliser des fractions pour approximer les valeurs décimales.

### Observations
- Pour de grandes valeurs de n, F(n) converge vers 7/18.
- On peut utiliser une formule analytique pour certains cas.
- La simulation Monte Carlo peut être utilisée pour vérifier.

### Complexité
- **Temporelle** : O(P × T × n) où P est le nombre de nombres premiers et T le nombre d'essais.
- **Spatiale** : O(n) pour stocker les positions.

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <vector>
#include <random>
#include <algorithm>
#include <cmath>

using namespace std;

double F(int n, int trials) {
    random_device rd;
    mt19937 gen(rd());
    uniform_real_distribution<double> dis(0.0, 1.0);
    
    double total_painted = 0.0;
    
    for (int t = 0; t < trials; t++) {
        vector<double> positions;
        for (int i = 0; i < n; i++) {
            positions.push_back(dis(gen));
        }
        sort(positions.begin(), positions.end());
        
        double painted = 0.0;
        for (int i = 0; i < n; i++) {
            double left_dist = (i == 0) ? positions[i] : positions[i] - positions[i-1];
            double right_dist = (i == n-1) ? 1.0 - positions[i] : positions[i+1] - positions[i];
            painted += min(left_dist, right_dist);
        }
        
        total_painted += painted;
    }
    
    return total_painted / trials;
}

double compute() {
    vector<int> primes;
    // Générer tous les nombres premiers impairs < 10^6
    vector<bool> is_prime(1000000, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i < 1000000; i++) {
        if (is_prime[i] && i % 2 == 1 && i > 2) {
            primes.push_back(i);
            for (int j = 2 * i; j < 1000000; j += i) {
                is_prime[j] = false;
            }
        }
    }
    
    double total = 0.0;
    for (int p : primes) {
        total += F(p, 1000); // Utiliser 1000 essais par nombre premier
    }
    
    return total / primes.size();
}

int main() {
    cout << fixed << setprecision(10) << compute() << endl;
    return 0;
}
```
