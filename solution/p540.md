# Comptage de triplets pythagoriciens primitifs

## Description du Problème
Un triplet pythagoricien primitif est un ensemble de trois entiers positifs a < b < c tels que a² + b² = c² et gcd(a, b, c) = 1.

Soit P(n) le nombre de triplets pythagoriciens primitifs avec c ≤ n.

Par exemple, P(20) = 3, puisqu'il y a trois triplets : (3,4,5), (5,12,13) et (8,15,17).

On vous donne que P(10⁶) = 159139.

**Objectif** : Trouver P(3141592653589793).

---

## Exemples
### Exemple 1
**Entrée** : `n = 20`
**Sortie** : `3`
**Explication** : P(20) = 3.

---

## Approche de Résolution

### Stratégie
1. Utiliser la formule paramétrique pour les triplets pythagoriciens primitifs : a = m² - n², b = 2mn, c = m² + n², où m > n, gcd(m, n) = 1, et m et n ont des parités opposées.
2. On a c = m² + n² ≤ limit, donc m ≤ √limit.
3. Pour chaque m, n avec les conditions ci-dessus, générer le triplet et compter.
4. Utiliser la fonction de Möbius pour compter efficacement.
5. P(n) ~ n/(2π) selon Lehmer.

### Observations
- On peut utiliser un crible de Möbius pour compter efficacement.
- Il existe des algorithmes optimisés pour compter les triplets primitifs.
- Pour de très grandes valeurs, on peut utiliser des approximations.

### Complexité
- **Temporelle** : O(√n × log log n) avec un crible optimisé.
- **Spatiale** : O(√n) pour le crible.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<int> mobius_sieve(long long limit) {
    vector<bool> is_prime(limit + 1, true);
    vector<int> mobius(limit + 1, 1);
    is_prime[0] = is_prime[1] = false;
    
    for (long long p = 2; p <= limit; p++) {
        if (is_prime[p]) {
            mobius[p] = -1;
            for (long long j = 2 * p; j <= limit; j += p) {
                is_prime[j] = false;
                mobius[j] *= -1;
            }
            long long sq = p * p;
            if (sq <= limit) {
                for (long long j = sq; j <= limit; j += sq) {
                    mobius[j] = 0;
                }
            }
        }
    }
    return mobius;
}

long long compute(long long limit) {
    long long sqrt_limit = sqrt(limit);
    vector<int> mobius = mobius_sieve(sqrt_limit);
    long long count = 0;
    
    for (long long m = 2; m * m <= limit; m++) {
        for (long long n = 1; n < m; n++) {
            if ((m + n) % 2 == 1 && __gcd(m, n) == 1) {
                long long c = m * m + n * n;
                if (c <= limit) {
                    count += limit / c;
                }
            }
        }
    }
    
    return count;
}

int main() {
    cout << compute(3141592653589793LL) << endl;
    return 0;
}
```
