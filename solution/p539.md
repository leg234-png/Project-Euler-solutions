# Élimination impaire

## Description du Problème
On commence avec une liste contenant les nombres 1, 2, 3, ..., n dans l'ordre croissant. On répète ensuite l'opération suivante jusqu'à ce qu'il ne reste qu'un seul nombre :

- Supprimer tous les nombres aux positions impaires de la liste.

Soit P(n) le dernier nombre restant en commençant avec une liste de longueur n.

Soit S(n) = ∑ₖ₌₁ⁿ P(k).

On vous donne P(1)=1, P(9)=6, P(1000)=510, S(1000)=268271.

**Objectif** : Trouver S(10¹⁸). Donner votre réponse modulo 987654321.

---

## Exemples
### Exemple 1
**Entrée** : `n = 9`
**Sortie** : `6`
**Explication** : P(9) = 6.

---

## Approche de Résolution

### Stratégie
1. Observer que P(2k) = P(2k+1) pour tout k.
2. Soit v = n//2 = floor(n/2).
3. Alors P(n) = 2v + 2 - 2P(v).
4. Pour n pair, S(n) = S(n-1) + P(n).
5. Pour n impair, on peut utiliser une formule récursive.
6. Utiliser la mémorisation pour calculer efficacement S(10¹⁸).

### Observations
- Il y a une relation de récurrence claire pour P(n).
- On peut calculer S(n) récursivement sans itérer sur tous les nombres.
- La mémorisation est essentielle pour les grandes valeurs.

### Complexité
- **Temporelle** : O(log n) pour le calcul récursif avec mémorisation.
- **Spatiale** : O(log n) pour la table de mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <unordered_map>

using namespace std;

unordered_map<long long, long long> memo_P;
unordered_map<long long, long long> memo_S;
long long mod = 987654321;

long long P(long long n) {
    if (n == 1) return 1;
    if (memo_P.find(n) != memo_P.end()) {
        return memo_P[n];
    }
    
    long long v = n / 2;
    long long result = (2 * v + 2 - 2 * P(v) + mod) % mod;
    memo_P[n] = result;
    return result;
}

long long S(long long n) {
    if (n == 0) return 0;
    if (n == 1) return 1;
    if (memo_S.find(n) != memo_S.end()) {
        return memo_S[n];
    }
    
    long long result;
    if (n % 2 == 0) {
        result = (S(n - 1) + P(n)) % mod;
    } else {
        long long m = (n - 1) / 2;
        result = (1 + 2 * m * (m + 1) + 4 * m - 4 * S(m) + mod) % mod;
    }
    
    memo_S[n] = result;
    return result;
}

int main() {
    cout << S(1000000000000000000LL) << endl;
    return 0;
}
```
