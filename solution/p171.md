# Finding numbers for which the sum of the squares of the digits is a square

## Description du Problème
Pour un nombre positif `n`, soit `f(n)` la somme des carrés des chiffres de `n` (en base 10).

Par exemple, `f(3) = 3^2 = 9`, `f(25) = 2^2 + 5^2 = 4 + 25 = 29`, `f(442) = 16 + 16 + 4 = 36`.

**Objectif** : Trouver la somme de tous les nombres `n` (1 ≤ n < 10^20) pour lesquels `f(n)` est un carré parfait. Donner les 9 derniers chiffres de la réponse.

---

## Exemples
### Exemple 1
**Entrée** : `n = 25`
**Sortie** : `f(25) = 29` (pas un carré)
**Explication** : `f(25) = 2^2 + 5^2 = 29`, qui n'est pas un carré parfait.

### Exemple 2
**Entrée** : `n = 442`
**Sortie** : `f(442) = 36` (c'est un carré)
**Explication** : `f(442) = 4^2 + 4^2 + 2^2 = 36 = 6^2`.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique pour construire les nombres un chiffre à la fois.
2. `sum[n][s]` : somme de tous les nombres de `n` chiffres (avec zéros en tête possibles) dont la somme des carrés des chiffres est `s`, modulo 10^9.
3. `count[n][s]` : nombre de nombres de `n` chiffres dont la somme des carrés des chiffres est `s`, modulo 10^9.
4. Cas de base : `count[0][0] = 1`.
5. Pour `i = 1` à `LENGTH` :
   - Pour chaque chiffre `j` (0-9) :
     - Pour chaque somme `k` :
       - `sum[i][k + j*j] += sum[i-1][k] + 10^(i-1) * j * count[i-1][k]`
       - `count[i][k + j*j] += count[i-1][k]`
6. Additionner `sum[LENGTH][s]` pour tous les `s` qui sont des carrés parfaits.

### Observations
- On construit les nombres de gauche à droite.
- On doit suivre à la fois la somme et le nombre de façons.
- La contribution d'un chiffre à la somme totale dépend de sa position.

### Complexité
- **Temporelle** : O(LENGTH × BASE × MAX_SQR_DIGIT_SUM), où LENGTH=20, BASE=10, MAX_SQR_DIGIT_SUM=20×81=1620.
- **Spatiale** : O(LENGTH × MAX_SQR_DIGIT_SUM).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

const int LENGTH = 20;
const int BASE = 10;
const int MODULUS = 1000000000;

long long powMod(long long base, int exp, int mod) {
    long long result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp % 2 == 1)
            result = (result * base) % mod;
        base = (base * base) % mod;
        exp /= 2;
    }
    return result;
}

int main() {
    int MAX_SQR_DIGIT_SUM = (BASE - 1) * (BASE - 1) * LENGTH;
    
    vector<vector<long long>> sum(LENGTH + 1, vector<long long>(MAX_SQR_DIGIT_SUM + 1, 0));
    vector<vector<long long>> count(LENGTH + 1, vector<long long>(MAX_SQR_DIGIT_SUM + 1, 0));
    
    count[0][0] = 1;
    
    for (int i = 1; i <= LENGTH; i++) {
        for (int j = 0; j < BASE; j++) {
            for (int k = 0; k + j * j <= MAX_SQR_DIGIT_SUM; k++) {
                long long power = powMod(BASE, i - 1, MODULUS);
                sum[i][k + j * j] = (sum[i][k + j * j] + sum[i - 1][k] + power * j % MODULUS * count[i - 1][k]) % MODULUS;
                count[i][k + j * j] = (count[i][k + j * j] + count[i - 1][k]) % MODULUS;
            }
        }
    }
    
    long long s = 0;
    for (int i = 1; i * i <= MAX_SQR_DIGIT_SUM; i++) {
        s = (s + sum[LENGTH][i * i]) % MODULUS;
    }
    
    cout << s << endl;
    return 0;
}
```
