# Exploration de la pyramide de Pascal

## Description du Problème
Un chemin commence au sommet et progresse vers le bas vers l'une des trois sphères directement en dessous de la position actuelle.

Par conséquent, le nombre de chemins pour atteindre une certaine position est la somme des nombres immédiatement au-dessus (selon la position, il y a jusqu'à trois nombres au-dessus).

Le résultat est la pyramide de Pascal et les nombres à chaque niveau n sont les coefficients de l'expansion trinomiale (x+y+z)ⁿ.

**Objectif** : Combien de coefficients dans l'expansion de (x+y+z)²⁰⁰⁰⁰⁰ sont des multiples de 10¹² ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 3`
**Sortie** : `Nombre de coefficients`
**Explication** : Pour (x+y+z)³, on a les coefficients correspondants à la pyramide de Pascal.

---

## Approche de Résolution

### Stratégie
1. Utiliser la relation C(n, i, j) = C(i, j) × C(n, i).
2. Construire récursivement la factorisation en nombres premiers des coefficients binomiaux.
3. Ne s'intéresser qu'aux facteurs 2 et 5 pour vérifier si 10¹² = 2¹² × 5¹² divise le coefficient.
4. Compter tous les coefficients qui satisfont cette condition.

### Observations
- La pyramide de Pascal est une généralisation 3D du triangle de Pascal.
- On peut utiliser la formule de relation avec le triangle de Pascal pour calculer les coefficients.
- Pour vérifier si 10¹² divise un coefficient, il suffit de vérifier les puissances de 2 et 5.

### Complexité
- **Temporelle** : O(n³) pour parcourir tous les coefficients (i, j, k) avec i+j+k = n.
- **Spatiale** : O(n) pour stocker les factorisations partielles.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int fac(long long n, int d) {
    int c = 0;
    while (n % d == 0) {
        n /= d;
        c++;
    }
    return c;
}

long long compute(int N, int a, int ae, int b, int be) {
    vector<int> pfa(N + 1, 0);
    vector<int> pfb(N + 1, 0);
    
    for (int n = a; n <= N; n += a) {
        pfa[n] = fac(n, a);
    }
    
    for (int n = b; n <= N; n += b) {
        pfb[n] = fac(n, b);
    }
    
    for (int n = 1; n <= N; n++) {
        pfa[n] += pfa[n - 1];
        pfb[n] += pfb[n - 1];
    }
    
    long long total = 0;
    for (int i = 0; i <= N/3; i++) {
        for (int j = i; j <= (N - i)/2; j++) {
            int k = N - i - j;
            if (pfb[N] - pfb[i] - pfb[j] - pfb[k] >= be) {
                if (pfa[N] - pfa[i] - pfa[j] - pfa[k] >= ae) {
                    if (i == j && j == k) {
                        total += 1;
                    } else if (i == j || j == k) {
                        total += 3;
                    } else {
                        total += 6;
                    }
                }
            }
        }
    }
    return total;
}

int main() {
    cout << compute(200000, 2, 12, 5, 12) << endl;
    return 0;
}
```
