# Comptage de fractions dans une plage

## Description du Problème
Considérons la fraction n/d, où n et d sont des entiers positifs. Si n < d et HCF(n,d) = 1, elle est appelée une fraction propre réduite.

Si on liste l'ensemble des fractions propres réduites pour d ≤ 8 par ordre croissant de taille, on obtient :
1/8, 1/7, 1/6, 1/5, 1/4, 2/7, 1/3, 3/8, 2/5, 3/7, 1/2, 4/7, 3/5, 5/8, 2/3, 5/7, 3/4, 4/5, 5/6, 6/7, 7/8

On peut voir qu'il y a 3 fractions entre 1/3 et 1/2 dans cette liste.

**Objectif** : Combien de fractions se trouvent entre 1/3 et 1/2 dans l'ensemble ordonné de fractions propres réduites pour d ≤ 12 000 ?

---

## Exemples
### Exemple 1
**Entrée** : d ≤ 8, plage entre 1/3 et 1/2
**Sortie** : `3`
**Explication** : Les fractions 3/8, 2/5, et 3/7 sont entre 1/3 et 1/2.

---

## Approche de Résolution

### Stratégie
1. Utiliser l'arbre de Stern-Brocot pour compter efficacement les fractions dans la plage.
2. L'arbre de Stern-Brocot est un arbre de recherche binaire infini de tous les nombres rationnels positifs.
3. Utiliser une approche récursive pour parcourir l'arbre et compter les fractions avec d ≤ 12 000.
4. Entre deux fractions adjacentes leftN/leftD et rightN/rightD, insérer (leftN+rightN)/(leftD+rightD).
5. Si le dénominateur dépasse 12 000, arrêter la récursion.

### Observations
- L'arbre de Stern-Brocot génère toutes les fractions en termes réduits.
- Chaque fraction apparaît exactement une fois.
- On peut compter récursivement en utilisant la propriété que les fractions entre leftN/leftD et rightN/rightD sont générées par (leftN+rightN)/(leftD+rightD).

### Complexité
- **Temporelle** : O(N), où N est le nombre de fractions dans la plage avec d ≤ 12 000.
- **Spatiale** : O(log N), pour la profondeur de récursion.

---

## Solution en C++

```cpp
#include <iostream>

using namespace std;

// Compte le nombre de fractions réduites n/d telles que leftN/leftD < n/d < rightN/rightD et d <= 12000.
// leftN/leftD et rightN/rightD doivent être adjacents dans l'arbre de Stern-Brocot.
int sternBrocotCount(int leftN, int leftD, int rightN, int rightD) {
    int n = leftN + rightN;
    int d = leftD + rightD;
    if (d > 12000)
        return 0;
    else
        return 1 + sternBrocotCount(leftN, leftD, n, d) + sternBrocotCount(n, d, rightN, rightD);
}

int main() {
    cout << sternBrocotCount(1, 3, 1, 2) << endl;
    return 0;
}
```
