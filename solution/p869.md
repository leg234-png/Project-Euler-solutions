# Devinette de nombres premiers

## Description du Problème
Un joueur pense à un nombre premier p entre 2 et N (inclus). Le joueur répond "oui" ou "non" à chaque question.

Le joueur gagne des points égaux au nombre de questions posées avant que le nombre premier ne soit identifié.

Soit E(N) le nombre de points attendu en supposant que le joueur devine toujours pour maximiser son score. Par exemple, E(10) = 2, réalisable en devinant toujours "1". On vous donne aussi E(30) = 2.9.

**Objectif** : Trouver E(10⁸). Donner votre réponse arrondie à huit chiffres après la virgule décimale.

---

## Exemples
### Exemple 1
**Entrée** : `N = 10`
**Sortie** : `2.0`
**Explication** : E(10) = 2.

---

## Approche de Résolution

### Stratégie
1. Représenter les nombres premiers en binaire dans un arbre.
2. 2 = 10, 3 = 11, 5 = 101, 7 = 111.
3. Utiliser un arbre pour représenter les nombres premiers.
4. Stratégie : Choisir 0 ou 1 selon si le sous-arbre gauche ou droit est plus grand.
5. Si la devinette est correcte, continuer dans ce sous-arbre, sinon changer de sous-arbre.
6. Calculer le score attendu en utilisant la stratégie optimale.

### Observations
- On peut utiliser un arbre binaire pour représenter les nombres premiers.
- La stratégie optimale consiste à toujours choisir le sous-arbre le plus grand.
- On peut calculer récursivement le score attendu.

### Complexité
- **Temporelle** : O(P × log P) où P est le nombre de nombres premiers.
- **Spatiale** : O(P) pour l'arbre.

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <vector>
#include <cmath>
#include <string>

using namespace std;

struct Node {
    Node* left = nullptr;
    Node* right = nullptr;
    int key = 0;
    int count = 0;
};

class Tree {
public:
    Node* root;
    
    Tree() : root(new Node()) {}
    
    void insert(int prime) {
        string binary = "";
        int temp = prime;
        while (temp > 0) {
            binary = to_string(temp % 2) + binary;
            temp /= 2;
        }
        
        Node* curr = root;
        for (char bit : binary) {
            if (bit == '0') {
                if (!curr->left) curr->left = new Node();
                curr = curr->left;
            } else {
                if (!curr->right) curr->right = new Node();
                curr = curr->right;
            }
            curr->count++;
        }
        curr->key = prime;
    }
    
    double expected_score(Node* node) {
        if (!node) return 0;
        if (node->key > 0) return 1; // Feuille
        
        int left_count = node->left ? node->left->count : 0;
        int right_count = node->right ? node->right->count : 0;
        
        if (left_count == 0 && right_count == 0) return 0;
        
        if (left_count >= right_count) {
            return 1 + (left_count * expected_score(node->left) + right_count * (1 + expected_score(node->right))) / (left_count + right_count);
        } else {
            return 1 + (right_count * expected_score(node->right) + left_count * (1 + expected_score(node->left))) / (left_count + right_count);
        }
    }
};

double compute(int N) {
    vector<bool> is_prime(N + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= N; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= N; j += i) {
                is_prime[j] = false;
            }
        }
    }
    
    Tree tree;
    for (int i = 2; i <= N; i++) {
        if (is_prime[i]) {
            tree.insert(i);
        }
    }
    
    return tree.expected_score(tree.root);
}

int main() {
    cout << fixed << setprecision(8) << compute(100000000) << endl;
    return 0;
}
```
