# Somme d'un carré et d'un cube

## Description du Problème
De nombreux nombres peuvent être exprimés comme la somme d'un carré parfait et d'un cube parfait. Par exemple, 9 = 1² + 2³.

**Objectif** : Trouver la somme des 5 plus petits nombres palindromiques qui peuvent être exprimés comme la somme d'un carré > 1 et d'un cube > 1 de exactement 4 façons différentes.

---

## Exemples
### Exemple 1
**Entrée** : `n = 9`
**Sortie** : `1` (façon)
**Explication** : 9 = 1² + 2³ (mais 1 n'est pas > 1, donc ne compte pas).

---

## Approche de Résolution

### Stratégie
1. Pour une plage d'entiers [0, n), calculer en masse combien de façons chaque entier peut être exprimé comme somme d'un carré et d'un cube.
2. Initialiser un tableau `ways` de longueur n avec des zéros.
3. Pour chaque cube i³ et chaque carré j² tels que i³ + j² < n :
   - Incrémenter `ways[i³ + j²]`.
4. Parcourir le tableau et sélectionner les nombres qui sont :
   - Des palindromes en base 10
   - Peuvent être exprimés en exactement 4 façons
5. Si on ne trouve pas 5 nombres, augmenter la plage et réessayer.

### Observations
- La boucle extérieure exécute O(n^(1/3)) itérations.
- La boucle intérieure exécute O(n^(1/2)) itérations par boucle extérieure.
- Le corps de la boucle intérieure s'exécute donc O(n^(5/6)) fois.
- On peut limiter le comptage à TARGET_WAYS+1 pour éviter les valeurs trop grandes.

### Complexité
- **Temporelle** : O(n^(5/6)), où n est la limite de recherche.
- **Spatiale** : O(n), pour le tableau `ways`.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

using namespace std;

const int TARGET_WAYS = 4;
const int TARGET_COUNT = 5;

bool isPalindrome(int n) {
    string s = to_string(n);
    string rev = s;
    reverse(rev.begin(), rev.end());
    return s == rev;
}

int sqrt(int n) {
    return (int)std::sqrt(n);
}

int cbrt(int x) {
    if (x < 0) {
        return -1;
    }
    int y = 0;
    for (int i = 1 << 10; i != 0; i >>= 1) {
        y |= i;
        if (y > 1290 || (long long)y * y * y > x) {
            y ^= i;
        }
    }
    return y;
}

long long trySearch(int limit) {
    // ways[i] est exactement le nombre de façons que i peut être exprimé comme
    // somme d'un carré > 1 et d'un cube > 1. Le comptage est saturé à TARGET_WAYS+1.
    vector<int> ways(limit, 0);
    
    // Compter les deux boucles vers le bas, pour que la valeur de fin soit constante
    for (int i = cbrt(limit - 1); i > 1; i--) {
        int cube = i * i * i;
        for (int j = sqrt(limit - 1 - cube); j > 1; j--) {
            int index = cube + j * j;
            if (index < limit) {
                ways[index] = min(ways[index] + 1, TARGET_WAYS + 1);
            }
        }
    }
    
    // Examiner quels nombres sont des palindromes et ont le nombre cible de façons
    long long result = 0;
    int count = 0;
    for (int i = 0; i < ways.size(); i++) {
        if (ways[i] == TARGET_WAYS && isPalindrome(i)) {
            result += i;
            count++;
            if (count == TARGET_COUNT) {
                return result;
            }
        }
    }
    return -1;  // Non trouvé
}

int main() {
    for (long long limit = 1; ; limit *= 10) {
        if (limit > INT_MAX) {
            cerr << "Dépassement" << endl;
            return 1;
        }
        long long answer = trySearch((int)limit);
        if (answer != -1) {
            cout << answer << endl;
            return 0;
        }
    }
    return 0;
}
```
