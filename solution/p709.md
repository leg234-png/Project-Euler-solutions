# Even Stevens

## Description du Problème
Chaque jour pendant les n derniers jours, Even Stevens rapporte ses courses dans un sac en plastique. Il range ces sacs en plastique dans un placard.

Il soit met le sac en plastique dans le placard avec le reste, soit il prend un nombre pair de sacs existants (qui peuvent être vides ou précédemment remplis avec d'autres sacs eux-mêmes) et les place dans le sac actuel.

Après 4 jours, il y a 5 arrangements possibles et si les sacs sont numérotés 1 (le plus ancien), 2, 3, 4, ils sont :
- Quatre sacs vides,
- 1 et 2 à l'intérieur de 3, 4 vide,
- 1 et 3 à l'intérieur de 4, 2 vide,
- 1 et 2 à l'intérieur de 4, 3 vide,
- 2 et 3 à l'intérieur de 4, 1 vide.

Soit f(n) le nombre d'arrangements possibles de n sacs. Donc f(4) = 5. On vous donne aussi f(8) = 1385.

**Objectif** : Trouver f(24680) en donnant votre réponse modulo 1020202009.

---

## Exemples
### Exemple 1
**Entrée** : `n = 4`
**Sortie** : `5`
**Explication** : f(4) = 5.

---

## Approche de Résolution

### Stratégie
1. f(0) = 1, f(1) = 1, f(2) = 1, f(3) = 2, f(4) = 5.
2. Recherche dans OEIS : A000111 (nombres zigzag d'Euler).
3. Utiliser les nombres d'Entringer E(n, k).
4. f(n) = E(n, n) où E(n, k) suit une récurrence spécifique.
5. Utiliser la programmation dynamique pour calculer E(n, k).

### Observations
- Les nombres zigzag d'Euler apparaissent naturellement.
- On peut utiliser les nombres d'Entringer pour calculer efficacement.
- La récurrence permet de calculer f(n) sans générer tous les arrangements.

### Complexité
- **Temporelle** : O(n²) pour calculer E(n, k) pour tous les n, k.
- **Spatiale** : O(n²) pour la table DP.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

long long mod = 1020202009;

long long compute(int n) {
    vector<vector<long long>> E(n + 1, vector<long long>(n + 1, 0));
    E[0][0] = 1;
    
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= i; j++) {
            if (j == 1) {
                E[i][j] = E[i-1][i-1];
            } else {
                E[i][j] = (E[i][j-1] + E[i-1][i-j]) % mod;
            }
        }
    }
    
    return E[n][n];
}

int main() {
    cout << compute(24680) << endl;
    return 0;
}
```
