# Binary Circles

## Description du Problème
Pour un entier positif `n`, un arrangement circulaire binaire de `2^n` bits est un nombre binaire de `2^n` bits où chaque fenêtre cyclique de `n` bits est unique.

Par exemple, pour `n = 3`, un arrangement valide est `00011101` (en binaire), qui contient toutes les fenêtres de 3 bits : 000, 001, 011, 111, 110, 101, 010, 100.

**Objectif** : Pour `n = 5`, trouver la somme de tous les nombres binaires de 32 bits qui sont des arrangements circulaires valides et qui commencent par `000001`.

---

## Exemples
### Exemple 1
**Entrée** : `n = 3`
**Sortie** : Nombre d'arrangements valides
**Explication** : Il y a plusieurs arrangements circulaires pour n=3.

---

## Approche de Résolution

### Stratégie
1. Les arrangements valides doivent commencer par `000001` (n zéros suivis d'un 1).
2. Ils doivent aussi se terminer par `1` pour que la fenêtre cyclique fonctionne.
3. Donc la forme est `000001xxxxxxxxxxxxxxxxxxxxxxxxx1` (32 bits).
4. Pour chaque combinaison des 25 bits du milieu :
   - Vérifier que toutes les 32 fenêtres cycliques de 5 bits sont uniques.
5. Utiliser un masque de bits pour vérifier l'unicité des fenêtres.

### Observations
- Il y a exactement `2^5 = 32` fenêtres possibles et `2^5 = 32` valeurs possibles, donc chaque valeur doit apparaître exactement une fois.
- On peut vérifier l'unicité en utilisant un masque de bits.

### Complexité
- **Temporelle : O(2^25 × 32)`.
- **Spatiale : O(1)**.

---

## Solution en C++

```cpp
#include <iostream>

using namespace std;

const int N = 5;
const int TWO_POW_N = 1 << N;
const int MASK = TWO_POW_N - 1;
const int FULL_SET = (1LL << TWO_POW_N) - 1;

bool checkArrangement(int digits) {
    int seen = 0;
    long long temp = ((long long)digits & 0xFFFFFFFFLL) | (((long long)digits & 0xFFFFFFFFLL) << TWO_POW_N);
    for (int i = 0; i < TWO_POW_N; i++)
        seen |= 1 << ((temp >> i) & MASK);
    return seen == FULL_SET;
}

int main() {
    long long sum = 0;
    int start = (1 << (TWO_POW_N - N - 1)) + 1;
    int end = 1 << (TWO_POW_N - N);
    for (int i = start; i < end; i += 2) {
        if (checkArrangement(i))
            sum += i;
    }
    cout << sum << endl;
    return 0;
}
```
