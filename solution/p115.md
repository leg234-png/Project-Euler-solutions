# Counting Block Combinations II

## Description du Problème
NOTE: Ceci est une version plus difficile du Problème 114.

Une rangée mesurant n unités de longueur a des blocs rouges d'une longueur minimale de m unités placés dessus, de telle sorte que deux blocs rouges quelconques (qui peuvent être de longueurs différentes) sont séparés par au moins un carré noir.

Soit la fonction de comptage de remplissage, F(m,n), représente le nombre de façons dont une rangée peut être remplie.

Par exemple, F(3,29)=673135 et F(3,30)=1089155.

C'est-à-dire, pour m=3, on peut voir que n=30 est la plus petite valeur pour laquelle la fonction de comptage de remplissage dépasse d'abord un million.

De la même manière, pour m=10, on peut vérifier que F(10,56)=880711 et F(10,57)=1148904, donc n=57 est la plus petite valeur pour laquelle la fonction de comptage de remplissage dépasse un million.

**Objectif** : Pour m=50, trouver la plus petite valeur de n pour laquelle la fonction de comptage de remplissage dépasse d'abord un million.

---

## Exemples
### Exemple 1
**Entrée** : `m = 3`
**Sortie** : `30`
**Explication** : F(3,30) = 1089155 > 1000000.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique similaire au problème 114.
2. ways[n] = nombre de façons de remplir une rangée de n unités.
3. Cas de base : ways[0] = 1.
4. Pour n ≥ 1 :
   - Si on place un carré noir à gauche : ways[n-1]
   - Si on place un bloc rouge de longueur k (m ≤ k ≤ n) : 1 (si k=n) ou ways[n-k-1] (si k<n)

### Observations
- Similaire au problème 114, mais avec une longueur minimale m variable.
- On cherche le plus petit n tel que ways[n] > 1000000.

### Complexité
- **Temporelle** : O(n²) dans le pire cas.
- **Spatiale** : O(n).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

const int M = 50;

int main() {
    std::vector<long long> ways;
    ways.push_back(1);
    
    for (int n = 1; ; n++) {
        long long sum = ways[n - 1];
        for (int k = M; k < n; k++) {
            sum += ways[n - k - 1];
        }
        if (n >= M) {
            sum++;  // Bloc rouge de longueur n
        }
        ways.push_back(sum);
        
        if (sum > 1000000) {
            std::cout << n << std::endl;
            return 0;
        }
    }
    return 0;
}
```
