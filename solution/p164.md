# Numbers for which no three consecutive digits have a sum greater than a given value

## Description du Problème
Combien de nombres de 20 chiffres (sans zéros en tête) n'ont pas trois chiffres consécutifs dont la somme dépasse 9 ?

---

## Exemples
### Exemple 1
**Entrée** : `digits = 3, maxSum = 9`
**Sortie** : Nombre de nombres de 3 chiffres sans trois chiffres consécutifs dont la somme > 9
**Explication** : Par exemple, 100 est valide (1+0+0=1 ≤ 9), mais 991 n'est pas valide (9+9+1=19 > 9).

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique.
2. `ways[d][p]` : nombre de façons de former un nombre de `d` chiffres (avec zéros en tête possibles) dont les 3 premiers chiffres forment le préfixe `p` (sur 3 chiffres), et où aucune somme de 3 chiffres consécutifs ne dépasse `maxSum`.
3. Cas de base : `ways[0][0] = 1`, `ways[0][p] = 0` pour `p > 0`.
4. Pour `d >= 1` :
   - Pour chaque préfixe `p` :
     - Si la somme des chiffres de `p` ≤ `maxSum` :
       - Pour chaque chiffre suivant `nextDigit` (0-9) :
         - Nouveau préfixe = `(p % 100) * 10 + nextDigit`.
         - `ways[d][p] += ways[d-1][nouveau_préfixe]`.
5. Le résultat final = `ways[20+3][0] - ways[20+3-1][0]` (pour exclure les zéros en tête).

### Observations
- On utilise un préfixe de 3 chiffres pour vérifier la contrainte sur les sommes consécutives.
- On doit gérer les grands nombres avec `BigInteger`.

### Complexité
- **Temporelle** : O(d × 10^3 × 10), où d=23.
- **Spatiale** : O(d × 10^3).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

const int BASE = 10;
const int DIGITS = 20;
const int CONSECUTIVE = 3;
const int MAX_SUM = 9;

int digitSum(int n) {
    int sum = 0;
    while (n != 0) {
        sum += n % 10;
        n /= 10;
    }
    return sum;
}

int pow(int base, int exp) {
    int result = 1;
    for (int i = 0; i < exp; i++)
        result *= base;
    return result;
}

int main() {
    int prefixSize = pow(BASE, CONSECUTIVE);
    vector<vector<cpp_int>> ways(DIGITS + CONSECUTIVE + 1, vector<cpp_int>(prefixSize, 0));
    
    ways[0][0] = 1;
    for (int prefix = 1; prefix < prefixSize; prefix++)
        ways[0][prefix] = 0;
    
    for (int digits = 1; digits < ways.size(); digits++) {
        for (int prefix = 0; prefix < prefixSize; prefix++) {
            cpp_int sum = 0;
            if (digitSum(prefix) <= MAX_SUM) {
                for (int nextDigit = 0; nextDigit < BASE; nextDigit++) {
                    int newPrefix = (prefix % pow(BASE, CONSECUTIVE - 1)) * BASE + nextDigit;
                    sum += ways[digits - 1][newPrefix];
                }
            }
            ways[digits][prefix] = sum;
        }
    }
    
    cout << (ways[DIGITS + CONSECUTIVE][0] - ways[DIGITS + CONSECUTIVE - 1][0]) << endl;
    return 0;
}
```
