# Bouncy Numbers

## Description du Problème
En travaillant de gauche à droite, si aucun chiffre n'est dépassé par le chiffre à sa gauche, on l'appelle un nombre croissant ; par exemple, 134468.

De même, si aucun chiffre n'est dépassé par le chiffre à sa droite, on l'appelle un nombre décroissant ; par exemple, 66420.

Nous appellerons un entier positif qui n'est ni croissant ni décroissant un nombre "rebondissant" ; par exemple, 155349.

Il est clair qu'il ne peut y avoir de nombres rebondissants en dessous de cent, mais un peu plus de la moitié des nombres en dessous de mille (525) sont rebondissants. En fait, le plus petit nombre pour lequel la proportion de nombres rebondissants atteint d'abord 50% est 538.

Étonnamment, les nombres rebondissants deviennent de plus en plus communs et au moment où nous atteignons 21780, la proportion de nombres rebondissants est égale à 90%.

**Objectif** : Trouver le plus petit nombre pour lequel la proportion de nombres rebondissants est exactement 99%.

---

## Exemples
### Exemple 1
**Entrée** : `n = 538`
**Sortie** : `50%` de nombres rebondissants
**Explication** : À 538, 50% des nombres sont rebondissants.

---

## Approche de Résolution

### Stratégie
1. Parcourir les nombres de 1 en 1.
2. Pour chaque nombre, vérifier s'il est rebondissant.
3. Compter le nombre total de nombres rebondissants.
4. Arrêter quand la proportion atteint exactement 99%.

### Observations
- Un nombre est rebondissant s'il n'est ni croissant ni décroissant.
- Un nombre est croissant si chaque chiffre est >= au chiffre précédent.
- Un nombre est décroissant si chaque chiffre est <= au chiffre précédent.

### Complexité
- **Temporelle** : O(n log n), où n est le nombre recherché.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>

bool isBouncy(int x) {
    if (x < 100) return false;
    
    bool nonincreasing = true;
    bool nondecreasing = true;
    int lastDigit = x % 10;
    x /= 10;
    
    while (x != 0) {
        int digit = x % 10;
        if (digit > lastDigit) nondecreasing = false;
        else if (digit < lastDigit) nonincreasing = false;
        lastDigit = digit;
        x /= 10;
    }
    
    return !nonincreasing && !nondecreasing;
}

int main() {
    int bouncy = 0;
    for (int i = 1; ; i++) {
        if (isBouncy(i)) {
            bouncy++;
        }
        if (bouncy * 100 == i * 99) {
            std::cout << i << std::endl;
            return 0;
        }
    }
    return 0;
}
```
