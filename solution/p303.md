# Multiples with small digits

## Description du Problème
Pour un entier positif `n`, soit `f(n)` le plus petit multiple positif de `n` qui, écrit en base 10, utilise uniquement les chiffres 0, 1 et 2.

Par exemple, `f(2) = 2`, `f(3) = 12`, `f(7) = 1002`, `f(42) = 12012`.

**Objectif** : Trouver `Σ f(n)/n` pour `1 ≤ n ≤ 10000`.

---

## Exemples
### Exemple 1
**Entrée** : `n = 2`
**Sortie** : `f(2) = 2`, donc `f(2)/2 = 1`

### Exemple 2
**Entrée** : `n = 3`
**Sortie** : `f(3) = 12`, donc `f(3)/3 = 4`

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique pour trouver le plus petit multiple.
2. `feasible[i][j]` : indique si un nombre de `i` chiffres (utilisant uniquement 0,1,2) peut avoir un reste `j` modulo `n`.
3. Construire récursivement :
   - Commencer avec 0 chiffres : seul le reste 0 est possible.
   - Pour chaque longueur `i`, ajouter un chiffre (0, 1, ou 2) à gauche.
   - Mettre à jour les restes possibles.
4. Une fois qu'un reste 0 est atteint (avec un nombre non nul), construire le nombre minimal :
   - Choisir le plus petit chiffre possible à chaque position.
   - Vérifier qu'il existe toujours une solution pour les positions restantes.

### Observations
- On ne stocke que les restes possibles, pas tous les nombres.
- La construction du nombre minimal se fait de gauche à droite (chiffre le plus significatif en premier).

### Complexité
- **Temporelle : O(k × n)`, où k est le nombre de chiffres nécessaires.
- **Spatiale : O(k × n)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

BigInteger findMinimumMultiple(int n) {
    vector<vector<int>> feasible;  // feasible[i][j] = 0 (impossible), 1 (only zero), 2 (non-zero possible)
    feasible.push_back(vector<int>(n, 0));
    feasible[0][0] = 1;
    
    int i = 0;
    while (feasible[i][0] != 2) {
        vector<int> cur(n, 0);
        int digitMod = 1;
        for (int pow = 0; pow < i; pow++)
            digitMod = (digitMod * 10) % n;
        
        for (int j = 0; j < n; j++) {
            if (feasible[i][j] > 0) {
                cur[(j + digitMod * 0) % n] = feasible[i][j];
                cur[(j + digitMod * 1) % n] = 2;
                cur[(j + digitMod * 2) % n] = 2;
            }
        }
        feasible.push_back(cur);
        i++;
    }
    
    BigInteger result = 0;
    int remainder = 0;
    for (int pos = feasible.size() - 2; pos >= 0; pos--) {
        int digitMod = 1;
        for (int pow = 0; pow < pos; pow++)
            digitMod = (digitMod * 10) % n;
        
        int start = (pos == feasible.size() - 2) ? 1 : 0;
        for (int j = start; j <= 2; j++) {
            int newRem = (remainder - digitMod * j % n + n) % n;
            if (feasible[pos][newRem] > 0) {
                result = result * 10 + j;
                remainder = newRem;
                break;
            }
        }
    }
    return result;
}

int main() {
    BigInteger sum = 0;
    for (int n = 1; n <= 10000; n++)
        sum += findMinimumMultiple(n) / n;
    cout << sum << endl;
    return 0;
}
```
