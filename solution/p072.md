# Comptage de fractions

## Description du Problème
Considérons la fraction n/d, où n et d sont des entiers positifs. Si n < d et HCF(n,d) = 1, elle est appelée une fraction propre réduite.

Si on liste l'ensemble des fractions propres réduites pour d ≤ 8 par ordre croissant de taille, on obtient :
1/8, 1/7, 1/6, 1/5, 1/4, 2/7, 1/3, 3/8, 2/5, 3/7, 1/2, 4/7, 3/5, 5/8, 2/3, 5/7, 3/4, 4/5, 5/6, 6/7, 7/8

On peut voir qu'il y a 21 éléments dans cet ensemble.

**Objectif** : Combien d'éléments seraient contenus dans l'ensemble des fractions propres réduites pour d ≤ 1 000 000 ?

---

## Exemples
### Exemple 1
**Entrée** : d ≤ 8
**Sortie** : `21`
**Explication** : Il y a 21 fractions propres réduites pour d ≤ 8.

---

## Approche de Résolution

### Stratégie
1. Le nombre de fractions propres réduites pour d ≤ LIMIT est égal à la somme de φ(i) pour i de 2 à LIMIT, où φ est la fonction totient d'Euler.
2. Pré-calculer tous les totients jusqu'à 1 000 000 en utilisant un crible.
3. Sommer les totients de 2 à 1 000 000.
4. Retourner la somme.

### Observations
- Pour chaque dénominateur d, il y a exactement φ(d) fractions propres réduites avec ce dénominateur.
- La fonction totient φ(n) compte le nombre d'entiers de 1 à n qui sont relativement premiers à n.
- On utilise un crible modifié pour calculer efficacement tous les totients.

### Complexité
- **Temporelle** : O(n log log n), pour le crible d'Eratosthène modifié.
- **Spatiale** : O(n), pour stocker les totients.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

vector<int> listTotients(int n) {
    vector<int> totients(n + 1);
    for (int i = 0; i <= n; i++)
        totients[i] = i;
    
    for (int i = 2; i <= n; i++) {
        if (totients[i] == i) {  // i est premier
            for (int j = i; j <= n; j += i)
                totients[j] -= totients[j] / i;
        }
    }
    return totients;
}

int main() {
    const int LIMIT = 1000000;
    vector<int> totients = listTotients(LIMIT);
    
    long long sum = 0;
    for (int i = 2; i < (int)totients.size(); i++)
        sum += totients[i];
    
    cout << sum << endl;
    return 0;
}
```
