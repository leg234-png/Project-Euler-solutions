# Convergents de Racine Carrée de 2

## Description du Problème
Il est possible de montrer que la racine carrée de deux peut être exprimée comme une fraction continue infinie.

√2 = 1 + 1/(2 + 1/(2 + 1/(2 + ... ))) = 1.414213...

En calculant les premières convergences, on obtient :

1 + 1/2 = 3/2 = 1.5
1 + 1/(2 + 1/2) = 7/5 = 1.4
1 + 1/(2 + 1/(2 + 1/2)) = 17/12 = 1.41666...
1 + 1/(2 + 1/(2 + 1/(2 + 1/2))) = 41/29 = 1.41379...

Les trois prochaines convergences sont 99/70, 239/169, et 577/408, mais la huitième convergence, 1393/985, est le premier exemple où le nombre de chiffres dans le numérateur dépasse le nombre de chiffres dans le dénominateur.

**Objectif** : Dans les mille premières convergences, combien de fractions ont un numérateur avec plus de chiffres que le dénominateur ?

---

## Exemples
### Exemple 1
**Entrée** : Convergence 8
**Sortie** : `1`
**Explication** : 1393/985 est la première où le numérateur a plus de chiffres.

---

## Approche de Résolution

### Stratégie
1. Calculer les convergences de √2 - 1 en utilisant la récurrence :
   - n' = d
   - d' = 2d + n
2. Pour chaque convergence :
   - Vérifier si le nombre de chiffres de (n + d) > nombre de chiffres de d.
   - Si oui, incrémenter le compteur.
3. Retourner le compteur.

### Observations
- On calcule les convergences de √2 - 1, puis on ajoute 1 pour obtenir √2.
- On utilise BigInteger car les nombres deviennent très grands.
- La récurrence est : n' = d, d' = 2d + n.

### Complexité
- **Temporelle** : O(n × log m), où n = 1000 et m est la taille maximale des nombres.
- **Spatiale** : O(log m), pour stocker les BigInteger.

---

## Solution en C++

```cpp
#include <iostream>
#include <gmpxx.h>

int main() {
    mpz_class n = 0;
    mpz_class d = 1;
    int count = 0;
    
    for (int i = 0; i < 1000; ++i) {
        mpz_class temp = d * 2 + n;
        n = d;
        d = temp;
        
        // Check if (n + d) has more digits than d
        mpz_class num = n + d;
        if (num.get_str().length() > d.get_str().length()) {
            count++;
        }
    }
    
    std::cout << count << std::endl;
    return 0;
}
```
