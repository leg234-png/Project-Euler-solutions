# Minimal Network

## Description du Problème
Le réseau non orienté suivant se compose de sept sommets et de douze arêtes avec un poids total de 243.

Le même réseau peut être représenté par la matrice ci-dessous.

Cependant, il est possible d'optimiser le réseau en supprimant certaines arêtes tout en s'assurant que tous les points du réseau restent connectés. Le réseau qui atteint l'économie maximale est montré ci-dessous. Il a un poids de 93, représentant une économie de 243 − 93 = 150 par rapport au réseau original.

En utilisant network.txt, un fichier texte de 6K contenant un réseau avec quarante sommets, et donné sous forme de matrice, trouver l'économie maximale qui peut être réalisée en supprimant les arêtes redondantes tout en s'assurant que le réseau reste connecté.

**Objectif** : Trouver l'économie maximale (poids total original - poids de l'arbre couvrant minimal).

---

## Exemples
### Exemple 1
**Entrée** : Réseau avec 7 sommets, poids total 243
**Sortie** : `150`
**Explication** : L'arbre couvrant minimal a un poids de 93, donc l'économie est 243 - 93 = 150.

---

## Approche de Résolution

### Stratégie
1. Lire la matrice d'adjacence depuis le fichier.
2. Calculer le poids total de toutes les arêtes.
3. Trouver l'arbre couvrant minimal (MST) en utilisant l'algorithme de Prim ou Kruskal.
4. Calculer l'économie = poids total - poids du MST.

### Observations
- On utilise l'algorithme de Prim : commencer par un sommet, ajouter l'arête de poids minimal qui connecte un sommet dans l'arbre à un sommet hors de l'arbre.
- On peut aussi utiliser l'algorithme de Kruskal : trier les arêtes par poids et ajouter celles qui ne créent pas de cycle.

### Complexité
- **Temporelle** : O(V²) avec Prim naïf, où V=40 est le nombre de sommets.
- **Spatiale** : O(V²) pour stocker la matrice d'adjacence.

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <climits>

int main() {
    // Lire la matrice depuis le fichier
    // std::vector<std::vector<int>> weights;
    // ... lire network.txt ...
    
    int vertices = weights.size();
    
    // Calculer le poids total original
    int oldWeight = 0;
    for (int i = 0; i < vertices; i++) {
        for (int j = i + 1; j < vertices; j++) {
            if (weights[i][j] != -1) {
                oldWeight += weights[i][j];
            }
        }
    }
    
    // Algorithme de Prim pour trouver le MST
    std::vector<bool> reachable(vertices, false);
    reachable[0] = true;
    int newWeight = 0;
    
    for (int i = 1; i < vertices; i++) {
        int lowestWeight = -1;
        int target = -1;
        
        for (int j = 0; j < vertices; j++) {
            if (reachable[j]) {
                for (int k = 0; k < vertices; k++) {
                    if (!reachable[k] && weights[j][k] != -1) {
                        if (lowestWeight == -1 || weights[j][k] < lowestWeight) {
                            lowestWeight = weights[j][k];
                            target = k;
                        }
                    }
                }
            }
        }
        
        if (lowestWeight == -1) {
            throw std::runtime_error("No spanning tree exists");
        }
        
        reachable[target] = true;
        newWeight += lowestWeight;
    }
    
    std::cout << (oldWeight - newWeight) << std::endl;
    return 0;
}
```
