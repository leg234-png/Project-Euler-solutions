# abc-hits

## Description du Problème
Le radical de n, rad(n), est le produit des facteurs premiers distincts de n. Par exemple, 504 = 2³ × 3² × 7, donc rad(504) = 2 × 3 × 7 = 42.

Nous définirons le triplet d'entiers positifs (a, b, c) comme étant un abc-hit si :
1. gcd(a, b) = gcd(a, c) = gcd(b, c) = 1
2. a < b
3. a + b = c
4. rad(abc) < c

Par exemple, (5, 27, 32) est un abc-hit, car :
- gcd(5, 27) = gcd(5, 32) = gcd(27, 32) = 1
- 5 < 27
- 5 + 27 = 32
- rad(4320) = 30 < 32

Il s'avère que les abc-hits sont assez rares et il n'y a que trente-et-un abc-hits pour c < 1000, avec ∑c = 12523.

**Objectif** : Trouver ∑c pour c < 120000.

---

## Exemples
### Exemple 1
**Entrée** : `c < 1000`
**Sortie** : `12523`
**Explication** : La somme de tous les c pour les abc-hits avec c < 1000 est 12523.

---

## Approche de Résolution

### Stratégie
1. Calculer rad(n) pour tous les n jusqu'à 120000 en utilisant un crible modifié.
2. Pour chaque c de 2 à 120000, parcourir a de 1 à c/2.
3. Pour chaque paire (a, c), calculer b = c - a.
4. Vérifier les conditions :
   - gcd(a, b) = 1 (implique aussi gcd(a, c) = 1 et gcd(b, c) = 1)
   - rad(a) × rad(b) × rad(c) < c
5. Si toutes les conditions sont satisfaites, ajouter c à la somme.

### Observations
- Si rad(c) = c (c'est-à-dire c est premier ou sans facteurs carrés), alors rad(abc) ≥ c, donc on peut ignorer ces cas.
- On peut optimiser en vérifiant d'abord si rad(c) = c.

### Complexité
- **Temporelle** : O(n²) pour parcourir tous les a et c.
- **Spatiale** : O(n) pour stocker les radicaux.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

long long gcd(long long a, long long b) {
    while (b != 0) {
        long long temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

int main() {
    const int LIMIT = 120000;
    
    // Calculer rad(n) pour tous les n
    std::vector<int> rads(LIMIT, 1);
    for (int i = 2; i < LIMIT; i++) {
        if (rads[i] == 1) {
            for (int j = i; j < LIMIT; j += i) {
                rads[j] *= i;
            }
        }
    }
    
    long long sum = 0;
    for (int c = 2; c < LIMIT; c++) {
        if (rads[c] == c) continue;  // Optimisation
        
        for (int a = 1, end = (c - 1) / 2; a <= end; a++) {
            int b = c - a;
            if (gcd(a, b) == 1) {
                if ((long long)rads[a] * rads[b] * rads[c] < c) {
                    sum += c;
                }
            }
        }
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
