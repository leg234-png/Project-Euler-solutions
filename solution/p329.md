# Prime Frog

## Description du Problème
Une grenouille saute sur les cases numérotées de 1 à 500. Elle commence sur une case choisie uniformément au hasard et coasse. Ensuite, elle saute 14 fois (gauche ou droite avec probabilité égale) et coasse après chaque saut.

La probabilité que la grenouille coasse "P" (Prime) ou "N" (Not Prime) dépend de si la case est première :
- Si la case est première : P avec probabilité 2/3, N avec probabilité 1/3.
- Si la case n'est pas première : P avec probabilité 1/3, N avec probabilité 2/3.

**Objectif** : Étant donné la séquence de coassements observée "PPPPNNPPPNPPNPN", quelle est la probabilité que cette séquence soit produite ? Donner la réponse sous forme de fraction irréductible.

---

## Exemples
### Exemple 1
**Entrée** : Séquence de coassements
**Sortie** : Probabilité sous forme de fraction
**Explication** : On simule tous les chemins possibles et on calcule la probabilité.

---

## Approche de Résolution

### Stratégie
1. Pour chaque case de départ (1 à 500) :
   - Pour chaque séquence de 14 sauts (2^14 possibilités) :
     - Simuler le chemin de la grenouille.
     - Calculer la probabilité que la séquence de coassements observée soit produite.
     - Ajouter cette probabilité au numérateur global.
2. Le dénominateur est `500 × 2^14 × 3^15`.
3. Simplifier la fraction résultante.

### Observations
- Il y a `500 × 2^14 = 8 192 000` chemins possibles.
- Pour chaque chemin, la probabilité est `2^(nombre_de_matchs) / 3^15`.
- On peut éviter les calculs en virgule flottante en utilisant des entiers.

### Complexité
- **Temporelle : O(500 × 2^14 × 15)`.
- **Spatiale : O(1)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

const int START_NUM = 1;
const int END_NUM = 500;
const string CROAK_SEQ = "PPPPNNPPPNPPNPN";
const int NUM_JUMPS = CROAK_SEQ.length() - 1;
const int NUM_TRIALS = 1 << NUM_JUMPS;

struct Fraction {
    cpp_int num, den;
    
    Fraction(cpp_int n, cpp_int d) : num(n), den(d) {
        cpp_int g = gcd(n, d);
        num /= g;
        den /= g;
    }
    
    string toString() const {
        return num.str() + "/" + den.str();
    }
};

int main() {
    vector<bool> isPrime = listPrimality(END_NUM);
    cpp_int globalNumerator = 0;
    
    for (int i = START_NUM; i <= END_NUM; i++) {
        for (int j = 0; j < NUM_TRIALS; j++) {
            int pos = i;
            cpp_int trialNumerator = 1;
            
            if (isPrime[pos] == (CROAK_SEQ[0] == 'P'))
                trialNumerator *= 2;
            
            for (int k = 0; k < NUM_JUMPS; k++) {
                if (pos <= START_NUM)
                    pos++;
                else if (pos >= END_NUM)
                    pos--;
                else if (((j >> k) & 1) == 0)
                    pos++;
                else
                    pos--;
                
                if (isPrime[pos] == (CROAK_SEQ[k + 1] == 'P'))
                    trialNumerator *= 2;
            }
            globalNumerator += trialNumerator;
        }
    }
    
    cpp_int globalDenominator = cpp_int(END_NUM + 1 - START_NUM) * (cpp_int(1) << NUM_JUMPS) * pow(cpp_int(3), CROAK_SEQ.length());
    Fraction result(globalNumerator, globalDenominator);
    cout << result.toString() << endl;
    return 0;
}
```
