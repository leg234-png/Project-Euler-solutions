# Totients 5-smooth

## Description du Problème
Les nombres 5-smooth sont les nombres dont le plus grand facteur premier ne dépasse pas 5. Les nombres 5-smooth sont aussi appelés nombres de Hamming.

Soit S(L) la somme des nombres n ne dépassant pas L tels que la fonction totient d'Euler φ(n) est un nombre de Hamming.

S(100) = 3728.

**Objectif** : Trouver S(10¹²). Donner votre réponse modulo 2³².

---

## Exemples
### Exemple 1
**Entrée** : `L = 100`
**Sortie** : `3728`
**Explication** : S(100) = 3728.

---

## Approche de Résolution

### Stratégie
1. Pour un nombre n = p₁^e₁ × p₂^e₂ × ... × pₖ^eₖ, φ(n) = p₁^(e₁-1) × (p₁-1) × p₂^(e₂-1) × (p₂-1) × ... × pₖ^(eₖ-1) × (pₖ-1).
2. Un nombre de Hamming est de la forme 2ᵃ × 3ᵇ × 5ᶜ.
3. On cherche tous les n de 1 à 10¹² tels que φ(n) = 2ᵃ × 3ᵇ × 5ᶜ.
4. Si pₖ > 5, alors φ(n) contiendra pₖ^(eₖ-1) × (pₖ-1), qui ne peut être un nombre de Hamming que si eₖ = 1 et pₖ - 1 est un nombre de Hamming.
5. Donc p = 2ᵃ × 3ᵇ × 5ᶜ + 1 doit être premier.
6. Générer tous les nombres de Hamming, ajouter 1, et vérifier s'ils sont premiers.

### Observations
- Seuls les nombres premiers de la forme nombre de Hamming + 1 peuvent apparaître.
- On peut générer tous les nombres de Hamming jusqu'à une certaine limite.
- On doit ensuite construire tous les n possibles avec ces nombres premiers.

### Complexité
- **Temporelle** : O(H × P) où H est le nombre de nombres de Hamming et P est le nombre de nombres premiers générés.
- **Spatiale** : O(H + P).

#include <iostream>
#include <vector>
#include <set>
#include <cmath>

using namespace std;

bool is_prime(long long x) {
    if (x <= 1) return false;
    if (x <= 3) return true;
    if (x % 2 == 0) return false;
    for (long long i = 3; i * i <= x; i += 2) {
        if (x % i == 0) return false;
    }
    return true;
}

vector<long long> generate_hamming(long long limit) {
    set<long long> hamming;
    hamming.insert(1);
    
    for (long long h : hamming) {
        if (h * 2 <= limit) hamming.insert(h * 2);
        if (h * 3 <= limit) hamming.insert(h * 3);
        if (h * 5 <= limit) hamming.insert(h * 5);
    }
    
    return vector<long long>(hamming.begin(), hamming.end());
}
