# Traversée de marais

## Description du Problème
Frodo et Sam doivent traverser un marais pour aller du point A au point B. Le marais consiste en 5 régions distinctes, chacune de 10 lieues de large. La vitesse de traversée diminue progressivement : 9, 8, 7, 6, et enfin 5 lieues par jour pour la dernière région, avant que le terrain redevienne plus facile avec une vitesse de 10 lieues par jour.

Si Frodo et Sam allaient directement vers l'est pour le point B, ils voyageraient exactement 100 lieues, et le voyage prendrait environ 13.4738 jours. Cependant, ce temps peut être raccourci s'ils dévient de la route directe.

**Objectif** : Si Frodo et Sam choisissent leur trajectoire de manière optimale, quel est le temps de voyage le plus court possible ? Donner votre réponse en jours, arrondi à 10 décimales après la virgule.

---

## Exemples
### Exemple 1
**Entrée** : `Traversée directe`
**Sortie** : `13.4738 jours`
**Explication** : Le temps pour une traversée directe.

---

## Approche de Résolution

### Stratégie
1. Utiliser la loi de Snell avec un angle de départ variable.
2. Calculer la distance et trouver le chemin de trajet minimum.
3. Pour chaque région, utiliser la loi de Snell : sin(θ₁)/v₁ = sin(θ₂)/v₂.
4. Calculer les angles successifs en fonction de l'angle initial.
5. Trouver l'angle initial qui minimise le temps total.
6. Utiliser une recherche binaire ou une méthode d'optimisation.

### Observations
- La loi de Snell s'applique à la réfraction de la lumière, mais peut être adaptée ici.
- On peut utiliser le calcul pour trouver l'optimum.
- Le problème nécessite une optimisation numérique.

### Complexité
- **Temporelle** : O(I) où I est le nombre d'itérations pour la convergence.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <cmath>

using namespace std;

double compute() {
    double s = (100.0 / sqrt(2.0) - 50.0) / 2.0;
    double min_time = 1e9;
    
    // Recherche de l'angle optimal
    for (int x = 1; x <= 900000; x++) {
        double theta1 = (x / 10000.0) * M_PI / 180.0;
        double d1 = s / cos(theta1);
        double h1 = s * tan(theta1);
        
        double theta2 = asin((9.0 * sin(theta1)) / 10.0);
        double d2 = 10.0 / cos(theta2);
        double h2 = 10.0 * tan(theta2);
        
        double theta3 = asin((8.0 * sin(theta2)) / 9.0);
        double d3 = 10.0 / cos(theta3);
        double h3 = 10.0 * tan(theta3);
        
        double theta4 = asin((7.0 * sin(theta3)) / 8.0);
        double d4 = 10.0 / cos(theta4);
        double h4 = 10.0 * tan(theta4);
        
        double theta5 = asin((6.0 * sin(theta4)) / 7.0);
        double d5 = 10.0 / cos(theta5);
        double h5 = 10.0 * tan(theta5);
        
        double total_h = h1 + h2 + h3 + h4 + h5;
        if (abs(total_h - 50.0) < 0.01) {
            double time = d1/9.0 + d2/8.0 + d3/7.0 + d4/6.0 + d5/5.0 + (100 - s - 50)/10.0;
            min_time = min(min_time, time);
        }
    }
    
    return min_time;
}

int main() {
    cout << fixed << setprecision(10) << compute() << endl;
    return 0;
}
```
