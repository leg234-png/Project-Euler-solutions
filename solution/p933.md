# Découpage de papier

## Description du Problème
Deux joueurs jouent à un jeu de découpage de papier. Le premier joueur découpe le papier en deux morceaux rectangulaires le long d'une ligne horizontale ou verticale. Le second joueur choisit l'un des morceaux et le découpe de la même manière. Les joueurs alternent jusqu'à ce qu'un joueur ne puisse plus faire de coup gagnant.

Soit C(w, h) le nombre de coups gagnants pour le premier joueur, lorsque le papier original a une taille w × h. Par exemple, C(5, 3) = 4, avec les quatre coups gagnants montrés ci-dessous.

Écrivons aussi D(W, H) = ∑_{w=2}^W ∑_{h=2}^H C(w, h). On vous donne que D(12, 123) = 327398.

**Objectif** : Trouver D(123, 1234567).

---

## Exemples
### Exemple 1
**Entrée** : `w = 5, h = 3`
**Sortie** : `4`
**Explication** : C(5, 3) = 4.

---

## Approche de Résolution

### Stratégie
1. C'est un jeu impartial, on peut utiliser la théorie des jeux combinatoires.
2. Utiliser la fonction de Grundy sG(w, h) pour calculer la valeur de Grundy.
3. Pour chaque coup possible (découper en (i, j), (i, h-j), (w-i, j), (w-i, h-j)), calculer le XOR des valeurs de Grundy.
4. C(w, h) = nombre de coups où le XOR est 0 (position perdante pour l'adversaire).
5. Utiliser la mémorisation pour calculer efficacement sG(w, h).
6. Observer les patterns : pour w fixe, C(w, h) suit une séquence avec des runs.

### Observations
- On peut utiliser la mémorisation pour éviter les recalculs.
- Il y a des patterns dans les valeurs de C(w, h) qui permettent d'optimiser.
- Pour de grandes valeurs, on peut utiliser des patterns observés.

### Complexité
- **Temporelle** : O(W × H × W × H) pour calculer tous les C(w, h).
- **Spatiale** : O(W × H) pour la mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <unordered_map>
#include <vector>
#include <algorithm>

using namespace std;

unordered_map<string, int> sG_memo;

int mex(vector<int>& vec) {
    for (int i = 0; i < vec.size(); i++) {
        if (vec[i] == 0) {
            return i;
        }
    }
    return vec.size();
}

int sG(int w, int h) {
    if (h > w) return sG(h, w);
    if (h == 1) return 0;
    
    string key = to_string(w) + "," + to_string(h);
    if (sG_memo.find(key) != sG_memo.end()) {
        return sG_memo[key];
    }
    
    vector<int> vec(1000, 0);
    
    for (int i = 1; i < w; i++) {
        for (int j = 1; j < h; j++) {
            int TL = sG(i, j);
            int TR = sG(i, h - j);
            int BL = sG(w - i, j);
            int BR = sG(w - i, h - j);
            
            int win = TL ^ TR ^ BL ^ BR;
            if (win >= vec.size()) {
                vec.resize(win + 100, 0);
            }
            vec[win] = 1;
        }
    }
    
    int result = mex(vec);
    sG_memo[key] = result;
    return result;
}

int C(int w, int h) {
    if (h > w) return C(h, w);
    if (h == 1) return 0;
    
    int count = 0;
    for (int i = 1; i < w; i++) {
        for (int j = 1; j < h; j++) {
            int TL = sG(i, j);
            int TR = sG(i, h - j);
            int BL = sG(w - i, j);
            int BR = sG(w - i, h - j);
            
            int win = TL ^ TR ^ BL ^ BR;
            if (win == 0) {
                count++;
            }
        }
    }
    return count;
}

long long D(int W, int H) {
    long long total = 0;
    
    for (int w = 2; w <= W; w++) {
        int prev = -1;
        int run = 0;
        long long r = 0;
        
        for (int h = 2; h <= H; h++) {
            int curr = C(w, h);
            if (prev != -1) {
                if (curr - prev == w - 1) {
                    run++;
                } else {
                    run = 0;
                }
            }
            prev = curr;
            
            if (run < 150) {
                r += curr;
            } else {
                r += (long long)(w - 1) * ((H - h) * (H - h + 1)) / 2 + (H - h + 1) * curr;
                break;
            }
        }
        total += r;
    }
    
    return total;
}

int main() {
    cout << D(123, 1234567) << endl;
    return 0;
}
```
