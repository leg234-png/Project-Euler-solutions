# Nombres tau

## Description du Problème
Un nombre tau est un entier positif x tel que τ(x) divise x, où τ(x) est le nombre de diviseurs de x.

Par exemple, 8 est un nombre tau car τ(8) = 4 divise 8.

Soit m(k) le plus petit nombre tau x tel que τ(x) = k. Par exemple, m(8) = 24, m(12) = 60 et m(16) = 384.

Définissons aussi M(n) comme la somme de tous les m(k) dont les valeurs ne dépassent pas 10^n. On vous donne M(3) = 3189.

**Objectif** : Trouver M(16).

---

## Exemples
### Exemple 1
**Entrée** : `k = 8`
**Sortie** : `24`
**Explication** : m(8) = 24.

---

## Approche de Résolution

### Stratégie
1. Pour chaque k, trouver le plus petit nombre x tel que τ(x) = k et k | x.
2. τ(x) = k signifie que x a exactement k diviseurs.
3. Si x = p₁^e₁ × p₂^e₂ × ... × p_r^e_r, alors τ(x) = (e₁+1)(e₂+1)...(e_r+1) = k.
4. Pour chaque factorisation de k, trouver le plus petit x correspondant.
5. Vérifier que k | x.
6. Collecter tous les m(k) ≤ 10^n et les sommer.

### Observations
- On peut générer toutes les factorisations de k.
- Pour chaque factorisation, on peut construire le plus petit x.
- Il faut vérifier que k divise x.

### Complexité
- **Temporelle** : O(K × log k) où K est le nombre de valeurs de k à considérer.
- **Spatiale** : O(K) pour stocker les m(k).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <map>
#include <cmath>
#include <climits>

using namespace std;

int tau(long long x) {
    int count = 0;
    for (long long i = 1; i * i <= x; i++) {
        if (x % i == 0) {
            count++;
            if (i * i != x) {
                count++;
            }
        }
    }
    return count;
}

long long m(int k) {
    long long limit = 1;
    for (int i = 0; i < 16; i++) {
        limit *= 10;
    }
    
    for (long long x = 1; x <= limit; x++) {
        if (tau(x) == k && x % k == 0) {
            return x;
        }
    }
    
    return -1; // Non trouvé
}

long long compute(int n) {
    long long limit = 1;
    for (int i = 0; i < n; i++) {
        limit *= 10;
    }
    
    long long total = 0;
    map<int, long long> m_values;
    
    for (int k = 1; k <= 1000; k++) { // Ajuster selon les besoins
        long long m_k = m(k);
        if (m_k > 0 && m_k <= limit) {
            m_values[k] = m_k;
            total += m_k;
        }
    }
    
    return total;
}

int main() {
    cout << compute(16) << endl;
    return 0;
}
```
