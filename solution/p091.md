# Right Triangles with Integer Coordinates

## Description du Problème
Les points P(x1,y1) et Q(x2,y2) sont tracés aux coordonnées entières et sont joints à l'origine, O(0,0), pour former △OPQ.

Il y a exactement quatorze triangles contenant un angle droit qui peuvent être formés lorsque chaque coordonnée est comprise entre 0 et 2 inclus ; c'est-à-dire, 0≤x1,y1,x2,y2≤2.

**Objectif** : Étant donné que 0≤x1,y1,x2,y2≤50, combien de triangles rectangles peuvent être formés ?

---

## Exemples
### Exemple 1
**Entrée** : `LIMIT = 2`
**Sortie** : `14`
**Explication** : Il y a exactement 14 triangles rectangles qui peuvent être formés avec des coordonnées entre 0 et 2.

---

## Approche de Résolution

### Stratégie
1. Parcourir toutes les combinaisons possibles de points P(x1,y1) et Q(x2,y2) dans la grille 0≤x,y≤50.
2. Pour chaque paire de points, vérifier si le triangle formé avec l'origine (0,0) est rectangle.
3. Utiliser le théorème de Pythagore pour vérifier si l'un des trois angles est droit.
4. Compter uniquement les triangles uniques en s'assurant que (x1,y1) a un angle plus grand que (x2,y2).

### Observations
- Un triangle est rectangle si la somme des carrés de deux côtés égale le carré du troisième côté.
- Pour les trois points (0,0), (x1,y1), (x2,y2), on calcule :
  - a² = x1² + y1²
  - b² = x2² + y2²
  - c² = (x2-x1)² + (y2-y1)²
- Le triangle est rectangle si a² + b² = c², ou b² + c² = a², ou c² + a² = b².

### Complexité
- **Temporelle** : O(n⁴), où n=50, car on parcourt toutes les combinaisons de deux points.
- **Spatiale** : O(1), car on utilise seulement des variables pour compter.

---

## Solution en C++

```cpp
#include <iostream>

const int LIMIT = 50;

bool isRightTriangle(int x1, int y1, int x2, int y2) {
    int a = x1 * x1 + y1 * y1;
    int b = x2 * x2 + y2 * y2;
    int c = (x2 - x1) * (x2 - x1) + (y2 - y1) * (y2 - y1);
    return a + b == c || b + c == a || c + a == b;
}

int main() {
    int count = 0;
    for (int x1 = 0; x1 <= LIMIT; x1++) {
        for (int y1 = 0; y1 <= LIMIT; y1++) {
            for (int x2 = 0; x2 <= LIMIT; x2++) {
                for (int y2 = 0; y2 <= LIMIT; y2++) {
                    // Pour l'unicité, s'assurer que (x1,y1) a un angle plus grand que (x2,y2)
                    if (y2 * x1 < y1 * x2 && isRightTriangle(x1, y1, x2, y2)) {
                        count++;
                    }
                }
            }
        }
    }
    std::cout << count << std::endl;
    return 0;
}
```
