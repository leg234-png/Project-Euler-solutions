# Resilience

## Description du Problème
Une fraction `n/d` est dite "résiliente" si `n` et `d` sont premiers entre eux.

La résilience d'un dénominateur `d`, notée `R(d)`, est le rapport entre le nombre de fractions résilientes et le nombre total de fractions propres : `R(d) = φ(d) / (d - 1)`, où `φ` est la fonction indicatrice d'Euler.

**Objectif** : Trouver le plus petit dénominateur `d` tel que `R(d) < 15499/94744`.

---

## Exemples
### Exemple 1
**Entrée** : `d = 12`
**Sortie** : `R(12) = φ(12)/(11) = 4/11`
**Explication** : Les fractions résilientes pour d=12 sont 1/12, 5/12, 7/12, 11/12.

---

## Approche de Résolution

### Stratégie
1. Utiliser la pseudo-résilience `R'(d) = φ(d)/d` qui ne dépend que des facteurs premiers distincts de `d`.
2. Calculer `R'(primorial(n))` pour trouver une borne inférieure sur le nombre de facteurs premiers nécessaires.
3. Une fois qu'on a trouvé `n` tel que `R'(primorial(n)) < target` :
   - Essayer des multiples `i × primorial(n)` pour `i = 1, 2, ..., p_{n+1}-1`.
   - Trouver le plus petit `i` tel que `R(i × primorial(n)) < target`.
4. Si aucun `i` ne fonctionne, augmenter `n` et réessayer.

### Observations
- `R'(d) = (p₁-1)/p₁ × (p₂-1)/p₂ × ...` ne dépend que des facteurs premiers distincts.
- On peut utiliser les primorielles pour trouver une borne inférieure.
- Il est avantageux d'utiliser les plus petits nombres premiers en premier.

### Complexité
- **Temporelle : O(P × log P)`, où P est le nombre de nombres premiers nécessaires.
- **Spatiale : O(1)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <boost/multiprecision/cpp_int.hpp>
#include <vector>

using namespace boost::multiprecision;
using namespace std;

struct Fraction {
    cpp_int num, den;
    
    Fraction(cpp_int n, cpp_int d) : num(n), den(d) {
        cpp_int g = gcd(n, d);
        num /= g;
        den /= g;
    }
    
    bool operator<(const Fraction& other) const {
        return num * other.den < other.num * den;
    }
};

bool isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}

int main() {
    Fraction target(15499, 94744);
    cpp_int totient = 1;
    cpp_int denominator = 1;
    
    for (int p = 2; ; ) {
        totient *= (p - 1);
        denominator *= p;
        
        // Advance to next prime
        do p++;
        while (!isPrime(p));
        
        Fraction pseudoResilience(totient, denominator);
        if (pseudoResilience < target) {
            // Try to find the lowest factor i such that R(i*d) < TARGET
            for (int i = 1; i < p; i++) {
                cpp_int numer = cpp_int(i) * totient;
                cpp_int denom = cpp_int(i) * denominator;
                Fraction resilience(numer, denom - 1);
                if (resilience < target) {
                    cout << denom << endl;
                    return 0;
                }
            }
        }
    }
}
```
