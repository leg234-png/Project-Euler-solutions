# Somme des carrés des diviseurs

## Description du Problème
Les diviseurs de 6 sont 1, 2, 3 et 6. La somme des carrés de ces diviseurs est 1² + 2² + 3² + 6² = 50.

Soit sigma2(n) la somme des carrés des diviseurs de n. Ainsi sigma2(6) = 50.

**Objectif** : Trouver Σsigma2(i) pour 1 <= i <= 10¹⁵. Donner la réponse modulo 10⁹.

---

## Exemples
### Exemple 1
**Entrée** : `n = 6`
**Sortie** : `50`
**Explication** : sigma2(6) = 1² + 2² + 3² + 6² = 1 + 4 + 9 + 36 = 50.

---

## Approche de Résolution

### Stratégie
1. Au lieu de considérer chaque entier et sommer ses diviseurs au carré, considérer chaque diviseur k de 1 à n et calculer sa contribution.
2. Pour un diviseur k, il apparaît exactement floor(n/k) fois dans l'union multiset de tous les ensembles de diviseurs.
3. La contribution de k est floor(n/k) × k².
4. Pour k > √n, floor(n/k) ne change pas souvent, donc on peut regrouper les calculs.
5. Pour un compte m = floor(n/k), les valeurs de k qui donnent ce compte sont dans l'intervalle (floor(n/(m+1)), floor(n/m)].

### Observations
- On peut diviser le calcul en deux parties : k <= splitAt (somme directe) et k > splitAt (regroupés par compte).
- Formule utile : Σk² pour k=1 à n = n(n+1)(2n+1)/6.
- On optimise en mettant plus de poids sur les sommes directes au lieu des sommes BigInteger lentes.

### Complexité
- **Temporelle** : O(√n), où n = 10¹⁵.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <gmpxx.h>

using namespace std;

const long long LIMIT = 1000000000000000LL;
const long long MODULUS = 1000000009LL;

long long sqrt(long long n) {
    return (long long)std::sqrt(n);
}

// Retourne 1² + 2² + 3² + ... + n² mod MODULUS.
long long sumSquaresMod(long long n) {
    mpz_class x(n);
    mpz_class y = x * (x + 1) * (2 * x + 1) / 6;
    mpz_class mod(MODULUS);
    y = y % mod;
    return y.get_si();
}

int main() {
    // Peut être n'importe quel nombre de 1 à LIMIT, mais quelque part près de sqrt(LIMIT) est préféré
    int splitCount = (int)sqrt(LIMIT);
    // Optimisation : Mettre plus de poids sur les sommes directes au lieu des sommes BigInteger lentes
    splitCount = max(splitCount / 3, 1);
    // Considérer les diviseurs individuellement jusqu'à et y compris ce nombre
    long long splitAt = LIMIT / (splitCount + 1);
    
    // Sommer les diviseurs individuels
    long long sum = 0;
    for (long long i = 1; i <= splitAt; i++) {
        long long count = LIMIT / i % MODULUS;
        long long term = (i % MODULUS) * (i % MODULUS) % MODULUS;
        term = term * count % MODULUS;
        sum = (sum + term) % MODULUS;
    }
    
    // Sommer les diviseurs regroupés par compte
    for (int i = splitCount; i >= 1; i--) {
        // Trouver tous les diviseurs avec le compte de i
        long long start = LIMIT / (i + 1);  // Exclusif
        long long end = LIMIT / i;  // Inclusif
        long long sumSquares = (sumSquaresMod(end) - sumSquaresMod(start) + MODULUS) % MODULUS;
        sum = (sum + i % MODULUS * sumSquares % MODULUS) % MODULUS;
    }
    
    cout << sum << endl;
    return 0;
}
```
