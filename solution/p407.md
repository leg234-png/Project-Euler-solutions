# Inverse idempotent

## Description du Problème
Si nous calculons a² mod 6 pour 0 <= a < 6, nous obtenons : 0,1,4,3,4,1.

Le plus grand a tel que a² ≡ a mod 6 est 4. Définissons M(n) comme le plus grand a < n tel que a² ≡ a (mod n). Donc M(6) = 4.

**Objectif** : Trouver ΣM(n) pour 1 <= n <= 10⁷.

---

## Exemples
### Exemple 1
**Entrée** : `n = 6`
**Sortie** : `4`
**Explication** : M(6) = 4 car 4² = 16 ≡ 4 (mod 6).

---

## Approche de Résolution

### Stratégie
1. Si a² = a mod n, alors c'est aussi vrai pour tout m qui divise n.
2. Se concentrer sur les moduli qui sont des puissances de nombres premiers, p^k.
3. Les seules solutions de a² = a mod p^k sont a = 0, 1 mod p^k.
4. Si on factorise n comme un produit de puissances de nombres premiers, on a un système de congruences.
5. Utiliser le théorème des restes chinois pour résoudre ces congruences.
6. La plus grande solution parmi ces 2^N solutions distinctes (où N est le nombre de facteurs premiers distincts de n) est M(n).

### Observations
- Pour chaque n, on factorise n en puissances de nombres premiers copremiers.
- On utilise le théorème des restes chinois pour combiner les solutions.
- On garde seulement la plus grande solution pour chaque n.

### Complexité
- **Temporelle** : O(N × 2^F), où N = 10⁷ et F est le nombre maximum de facteurs premiers distincts.
- **Spatiale** : O(N), pour le tableau des plus petits facteurs premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

const int LIMIT = 10000000;

vector<int> listSmallestPrimeFactors(int n) {
    vector<int> smallestPrimeFactor(n + 1, 0);
    for (int i = 2; i <= n; i++) {
        if (smallestPrimeFactor[i] == 0) {
            smallestPrimeFactor[i] = i;
            for (int j = i * 2; j <= n; j += i) {
                if (smallestPrimeFactor[j] == 0) {
                    smallestPrimeFactor[j] = i;
                }
            }
        }
    }
    return smallestPrimeFactor;
}

// Calcule l'inverse modulaire de a modulo m
long long modInverse(long long a, long long m) {
    long long m0 = m;
    long long y = 0, x = 1;
    
    if (m == 1) {
        return 0;
    }
    
    while (a > 1) {
        long long q = a / m;
        long long t = m;
        
        m = a % m;
        a = t;
        t = y;
        
        y = x - q * y;
        x = t;
    }
    
    if (x < 0) {
        x += m0;
    }
    
    return x;
}

int main() {
    vector<int> smallestPrimeFactor = listSmallestPrimeFactors(LIMIT);
    
    // Taille maximale de l'ensemble de facteurs premiers où le produit de l'ensemble <= LIMIT
    int maxNumPrimeFactors = 0;
    for (int i = 2, prod = 1; i < smallestPrimeFactor.size(); i++) {
        if (smallestPrimeFactor[i] == i) {  // i est premier
            if (LIMIT / prod < i) {
                break;
            }
            prod *= i;
            maxNumPrimeFactors++;
        }
    }
    
    long long sum = 0;
    // Tableaux temporaires
    vector<int> solns(1 << maxNumPrimeFactors);
    vector<int> newsolns(1 << maxNumPrimeFactors);
    
    for (int i = 1; i <= LIMIT; i++) {
        // Calculer la factorisation comme puissances de nombres premiers copremiers
        vector<int> factorization;
        for (int j = i; j != 1; ) {
            int p = smallestPrimeFactor[j];
            int q = 1;
            do {
                j /= p;
                q *= p;
            } while (j % p == 0);
            factorization.push_back(q);
        }
        
        solns[0] = 0;
        int solnslen = 1;
        int modulus = 1;
        
        for (int q : factorization) {
            // Utiliser le théorème des restes chinois
            int recip = modInverse(q % modulus, modulus);
            int newmod = q * modulus;
            
            int newsolnslen = 0;
            for (int j = 0; j < solnslen; j++) {
                newsolns[newsolnslen++] = ((0 + (long long)(solns[j] - 0 + modulus) * recip % modulus) * q) % newmod;
                newsolns[newsolnslen++] = ((1 + (long long)(solns[j] - 1 + modulus) * recip % modulus) * q) % newmod;
            }
            
            solnslen = newsolnslen;
            modulus = newmod;
            
            // Échanger les tampons
            swap(solns, newsolns);
        }
        
        int max = 0;
        for (int j = 0; j < solnslen; j++) {
            max = std::max(solns[j], max);
        }
        sum += max;
    }
    
    cout << sum << endl;
    return 0;
}
```
