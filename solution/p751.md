# Coïncidence de concaténation

## Description du Problème
Soit θ un nombre réel positif. On génère une séquence a₁, a₂, a₃, ... comme suit :
- a₁ = ⌊θ⌋
- Pour n ≥ 1, a_{n+1} = ⌊a_n × (θ - ⌊θ⌋ + 1)⌋

Par exemple, θ = 2.956938891377988... génère la séquence de Fibonacci : 2, 3, 5, 8, 13, 21, 34, 55, 89, ...

La concaténation d'une séquence d'entiers positifs a_n est une valeur réelle notée τ construite en concaténant les éléments de la séquence après la virgule décimale, en commençant à a₁ : a₁.a₂a₃a₄...

Par exemple, la séquence de Fibonacci construite à partir de θ = 2.956938891377988... donne la concaténation τ = 2.3581321345589... Clairement, τ ≠ θ pour cette valeur de θ.

**Objectif** : Trouver la seule valeur de θ pour laquelle la séquence générée commence à a₁ = 2 et la concaténation de la séquence générée est égale à la valeur originale : τ = θ. Donner votre réponse arrondie à 24 places après la virgule décimale.

---

## Exemples
### Exemple 1
**Entrée** : `θ générant la séquence de Fibonacci`
**Sortie** : `2.956938891377988...`
**Explication** : θ génère la séquence de Fibonacci mais τ ≠ θ.

---

## Approche de Résolution

### Stratégie
1. θ = 2.956938891377988...
2. a₁ = 2 => 2 < θ < 3.
3. Construire θ de manière itérative en choisissant chaque chiffre décimal.
4. Pour chaque position, essayer chaque chiffre de 0 à 9.
5. Vérifier si la séquence générée correspond à la concaténation.
6. Continuer jusqu'à obtenir 24 décimales.

### Observations
- On peut construire θ chiffre par chiffre.
- Chaque chiffre peut être déterminé en testant les possibilités.
- La construction est unique.

### Complexité
- **Temporelle** : O(d × 10) où d est le nombre de décimales.
- **Spatiale** : O(d) pour stocker θ.

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <string>
#include <cmath>

using namespace std;

string sequence(double theta, int length) {
    string tau = to_string((int)theta) + ".";
    double b1 = theta;
    
    while (tau.length() < length) {
        int floor_b1 = (int)b1;
        double bn = floor_b1 * (b1 - floor_b1 + 1);
        tau += to_string((int)bn);
        b1 = bn;
    }
    
    return tau.substr(0, length);
}

string compute(int length) {
    string curr = "2.";
    
    while (curr.length() <= length) {
        bool found = false;
        for (int x = 0; x <= 9; x++) {
            string temp_curr = curr + to_string(x);
            double theta = stod(temp_curr);
            string generated = sequence(theta, temp_curr.length());
            
            if (temp_curr == generated) {
                curr += to_string(x);
                found = true;
                break;
            }
        }
        if (!found) break;
    }
    
    return curr.substr(0, length);
}

int main() {
    cout << fixed << setprecision(24) << stod(compute(26)) << endl;
    return 0;
}
```
