# Chiffres romains

## Description du Problème
Pour qu'un nombre écrit en chiffres romains soit considéré comme valide, il y a des règles de base qui doivent être suivies. Même si les règles permettent à certains nombres d'être exprimés de plus d'une façon, il y a toujours une "meilleure" façon d'écrire un nombre particulier.

Par exemple, il semblerait qu'il y ait au moins six façons d'écrire le nombre seize :
- IIIIIIIIIIIIIIII
- VIIIIIIIIIII
- VVIIIIII
- XIIIIII
- VVVI
- XVI

Cependant, selon les règles, seuls XIIIIII et XVI sont valides, et le dernier exemple est considéré comme le plus efficace, car il utilise le moins de chiffres.

**Objectif** : Le fichier texte de 11K, roman.txt, contient mille nombres écrits en chiffres romains valides, mais pas nécessairement minimaux. Trouver le nombre de caractères économisés en écrivant chacun d'eux dans leur forme minimale.

**Note** : Vous pouvez supposer que tous les chiffres romains dans le fichier contiennent au plus quatre unités identiques consécutives.

---

## Exemples
### Exemple 1
**Entrée** : "XIIIIII"
**Sortie** : `4`
**Explication** : "XIIIIII" (8 caractères) peut être écrit comme "XVI" (3 caractères), économisant 5 caractères.

---

## Approche de Résolution

### Stratégie
1. Parser chaque nombre romain du fichier en nombre entier.
2. Convertir le nombre entier en sa représentation romaine minimale.
3. Calculer la différence de longueur entre l'original et la version minimale.
4. Sommer toutes les différences.

### Observations
- Les préfixes romains sont : M=1000, CM=900, D=500, CD=400, C=100, XC=90, L=50, XL=40, X=10, IX=9, V=5, IV=4, I=1.
- Pour parser, on cherche les préfixes de la plus grande valeur à la plus petite.
- Pour convertir en romain minimal, on utilise les longueurs de chiffres pré-calculées.

### Complexité
- **Temporelle** : O(N × L), où N est le nombre de nombres (1000) et L est la longueur moyenne.
- **Spatiale** : O(1), pour stocker les préfixes et longueurs.

---

## Solution en C++

```cpp
#include <iostream>
#include <fstream>
#include <string>
#include <vector>

using namespace std;

vector<pair<string, int>> PREFIXES = {
    {"M", 1000}, {"CM", 900}, {"D", 500}, {"CD", 400},
    {"C", 100}, {"XC", 90}, {"L", 50}, {"XL", 40},
    {"X", 10}, {"IX", 9}, {"V", 5}, {"IV", 4}, {"I", 1}
};

int parseRomanNumeral(const string& s) {
    int result = 0;
    string remaining = s;
    while (!remaining.empty()) {
        bool found = false;
        for (const auto& prefix : PREFIXES) {
            if (remaining.find(prefix.first) == 0) {
                result += prefix.second;
                remaining = remaining.substr(prefix.first.length());
                found = true;
                break;
            }
        }
        if (!found)
            throw invalid_argument("Cannot parse Roman numeral");
    }
    return result;
}

int DIGIT_LENGTHS[] = {0, 1, 2, 3, 2, 1, 2, 3, 4, 2};

int romanNumeralLength(int n) {
    if (n <= 0 || n >= 5000)
        throw invalid_argument("Number out of range");
    
    int count = 0;
    if (n >= 4000)
        count += 2;  // Compenser pour MMMM
    while (n != 0) {
        count += DIGIT_LENGTHS[n % 10];
        n /= 10;
    }
    return count;
}

int main() {
    ifstream file("roman.txt");
    string line;
    int sum = 0;
    
    while (getline(file, line)) {
        int num = parseRomanNumeral(line);
        int minimalLength = romanNumeralLength(num);
        sum += line.length() - minimalLength;
    }
    file.close();
    
    cout << sum << endl;
    return 0;
}
```
