# Nombres triangulaires 47-smooth

## Description du Problème
Un nombre est p-smooth s'il n'a pas de facteurs premiers plus grands que p.

Soit T la séquence des nombres triangulaires, c'est-à-dire T(n) = n(n+1)/2.

**Objectif** : Trouver la somme de tous les indices n tels que T(n) est 47-smooth.

---

## Exemples
### Exemple 1
**Entrée** : `n tel que T(n) est 47-smooth`
**Sortie** : `Somme des indices`
**Explication** : La somme de tous les n tels que T(n) est 47-smooth.

---

## Approche de Résolution

### Stratégie
1. Le théorème de Størmer nous dit qu'il y a une borne supérieure.
2. Le plus grand nombre 47-smooth est 1109496723126 (trouvé dans OEIS A117581).
3. Générer tous les nombres 47-smooth sous cette limite.
4. Pour chaque nombre 47-smooth, vérifier s'il est de la forme n(n+1)/2.
5. Résoudre n(n+1)/2 = k pour n : n = (-1 + √(1+8k))/2.
6. Si n est un entier, alors k est un nombre triangulaire.
7. Sommer tous les n valides.

### Observations
- On peut générer efficacement tous les nombres 47-smooth.
- On peut vérifier si un nombre est triangulaire en résolvant une équation quadratique.
- Il y a une borne supérieure claire pour n.

### Complexité
- **Temporelle** : O(H) où H est le nombre de nombres 47-smooth.
- **Spatiale** : O(H) pour stocker les nombres 47-smooth.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <cmath>

using namespace std;

vector<int> primes_up_to(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

set<long long> generate_smooth_numbers(const vector<int>& primes, long long limit) {
    set<long long> smooth;
    smooth.insert(1);
    
    for (int p : primes) {
        set<long long> new_smooth;
        for (long long h : smooth) {
            long long power = p;
            while (h * power <= limit) {
                new_smooth.insert(h * power);
                power *= p;
            }
        }
        smooth.insert(new_smooth.begin(), new_smooth.end());
    }
    
    return smooth;
}

bool is_triangular(long long k) {
    long long discriminant = 1 + 8 * k;
    long long root = sqrt(discriminant);
    if (root * root == discriminant) {
        long long n = (-1 + root) / 2;
        return n * (n + 1) / 2 == k;
    }
    return false;
}

long long compute() {
    vector<int> primes = primes_up_to(47);
    long long limit = 1109496723126LL;
    set<long long> smooth = generate_smooth_numbers(primes, limit);
    
    long long total = 0;
    for (long long k : smooth) {
        if (is_triangular(k)) {
            long long n = (-1 + sqrt(1 + 8 * k)) / 2;
            total += n;
        }
    }
    
    return total;
}

int main() {
    cout << compute() << endl;
    return 0;
}
```
