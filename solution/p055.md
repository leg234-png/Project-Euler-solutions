# Nombres de Lychrel

## Description du Problème
Si on prend 47, on l'inverse et on l'ajoute, 47 + 74 = 121, qui est un palindrome.

Tous les nombres ne produisent pas de palindrome aussi rapidement. Par exemple,

349 + 943 = 1292,
1292 + 2921 = 4213
4213 + 3124 = 7337

C'est-à-dire que 349 a pris trois itérations pour arriver à un palindrome.

Bien qu'aucun nombre n'ait été prouvé être un nombre de Lychrel, beaucoup de nombres sont supposés l'être. En fait, 10677 est le premier nombre pour lequel il a fallu plus de 50 itérations avant d'arriver à un palindrome. Bien qu'il n'ait pas été prouvé mathématiquement, on pense que 10677 est un nombre de Lychrel.

**Objectif** : Combien de nombres de Lychrel y a-t-il en dessous de dix mille ?

---

## Exemples
### Exemple 1
**Entrée** : n = 47
**Sortie** : `121` (après 1 itération)
**Explication** : 47 + 74 = 121 est un palindrome.

### Exemple 2
**Entrée** : n = 349
**Sortie** : `7337` (après 3 itérations)
**Explication** : 349 → 1292 → 4213 → 7337.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre de 0 à 9999 :
   - Effectuer jusqu'à 49 itérations :
     - Inverser le nombre et l'ajouter.
     - Si le résultat est un palindrome, ce n'est pas un nombre de Lychrel.
   - Si après 49 itérations aucun palindrome n'est trouvé, compter comme nombre de Lychrel.
2. Retourner le compteur.

### Observations
- On doit utiliser BigInteger car les nombres peuvent devenir très grands.
- On limite à 49 itérations comme spécifié dans le problème.
- Un nombre de Lychrel est un nombre qui ne produit jamais de palindrome.

### Complexité
- **Temporelle** : O(n × 49 × log m), où n = 10000 et m est la taille maximale des nombres.
- **Spatiale** : O(log m), pour stocker les BigInteger.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>
#include <gmpxx.h>

bool isPalindrome(const std::string& s) {
    for (size_t i = 0; i < s.length() / 2; ++i) {
        if (s[i] != s[s.length() - 1 - i]) {
            return false;
        }
    }
    return true;
}

std::string reverse(const std::string& s) {
    return std::string(s.rbegin(), s.rend());
}

bool isLychrel(int n) {
    mpz_class temp = n;
    for (int i = 0; i < 49; ++i) {
        std::string s = temp.get_str();
        std::string rev = reverse(s);
        temp += mpz_class(rev);
        
        if (isPalindrome(temp.get_str())) {
            return false;
        }
    }
    return true;
}

int main() {
    int count = 0;
    for (int i = 0; i < 10000; ++i) {
        if (isLychrel(i)) {
            count++;
        }
    }
    std::cout << count << std::endl;
    return 0;
}
```
