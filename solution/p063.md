# Comptage de puissances à n chiffres

## Description du Problème
Le nombre à 5 chiffres, 16807 = 7⁵, est aussi une cinquième puissance. De même, le nombre à 9 chiffres, 134217728 = 8⁹, est une neuvième puissance.

**Objectif** : Combien de nombres entiers positifs à n chiffres existent qui sont aussi une puissance n-ième ?

---

## Exemples
### Exemple 1
**Entrée** : Recherche de nombres à n chiffres qui sont des puissances n-ièmes
**Sortie** : `49`
**Explication** : Par exemple, 7⁵ = 16807 (5 chiffres), 8⁹ = 134217728 (9 chiffres).

---

## Approche de Résolution

### Stratégie
1. Observer que seules les bases à un chiffre (1-9) peuvent produire des puissances n-ièmes à n chiffres.
2. Pour n ≥ 10, nⁿ a toujours plus de n chiffres.
3. Pour chaque base de 1 à 9, tester les exposants de 1 à 21 (car 9²² a 21 chiffres).
4. Compter combien de ces puissances ont exactement n chiffres.

### Observations
- Quand n = 10, nⁿ a exactement n+1 chiffres, donc ces cas sont exclus.
- Par extension, quand n > 10, nⁿ a au moins n+1 chiffres.
- Quand n = 9 et k = 22, alors 9²² a 21 chiffres, ce qui est insuffisant.
- On ne considère donc que 1 ≤ n ≤ 9 et 1 ≤ k ≤ 21.

### Complexité
- **Temporelle** : O(9 × 21) = O(1), nombre constant d'opérations.
- **Spatiale** : O(1), utilisation d'espace constant.

---

## Solution en C++

```cpp
#include <iostream>
#include <gmpxx.h>
#include <string>

using namespace std;

int main() {
    int count = 0;
    for (int n = 1; n <= 9; n++) {
        for (int k = 1; k <= 21; k++) {
            mpz_class base(n);
            mpz_pow_ui(base.get_mpz_t(), base.get_mpz_t(), k);
            string s = base.get_str();
            if ((int)s.length() == k)
                count++;
        }
    }
    cout << count << endl;
    return 0;
}
```
