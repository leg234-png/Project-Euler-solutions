# Investigation des séquences d'Ulam

## Description du Problème
Par exemple, la séquence U(1,2) commence par 1, 2, 3=1+2, 4=1+3, 6=2+4, 8=2+6, 11=3+8.

5 n'appartient pas à la séquence car 5=1+4=2+3 a deux représentations comme somme de deux membres précédents, de même 7=1+6=3+4.

**Objectif** : Trouver ∑ₙ₌₂¹⁰ U(2,2n+1)ₖ, où k = 10¹¹.

---

## Exemples
### Exemple 1
**Entrée** : `v = 1, t = 2, k = 10`
**Sortie** : `U(1,2)₁₀`
**Explication** : Le 10ème terme de la séquence U(1,2).

---

## Approche de Résolution

### Stratégie
1. Utiliser les propriétés périodiques des séquences d'Ulam.
2. Construire la séquence jusqu'à détecter la période.
3. Utiliser la période pour calculer le kème terme sans générer toute la séquence.
4. Itérer sur n de 2 à 10 et sommer les résultats.

### Observations
- Les séquences d'Ulam ont des propriétés périodiques décrites dans la littérature mathématique.
- On peut détecter la période en observant les différences entre termes consécutifs.
- Une fois la période détectée, on peut calculer n'importe quel terme efficacement.

### Complexité
- **Temporelle** : O(P) où P est la longueur de la période, pour chaque séquence.
- **Spatiale** : O(P) pour stocker la séquence jusqu'à la détection de la période.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int delta(int x) {
    return (x == 0) ? 1 : 0;
}

long long Ulam(int v, long long t) {
    int a = 2;
    int e2 = 2*v + 2;
    vector<int> u = {a};
    for (int x = v; x < e2; x += 2) {
        u.push_back(x);
    }
    u.push_back(e2);
    
    vector<int> b = {0};
    for (int x = 3; x < e2; x += 2) {
        b.push_back(find(u.begin(), u.end(), x) != u.end() ? 1 : 0);
    }
    
    int n = (e2 - 1) / 2 + 1;
    int cs = -1, ce = -1;
    
    while (true) {
        int curr = 2*n + 1;
        int bn = delta(b[n - 1] - 1) + delta(b[n - v - 1] - 1);
        
        if (bn == 1) {
            u.push_back(curr);
            if (u.size() >= 2 && u[u.size()-1] - u[u.size()-2] == e2) {
                if (cs == -1) {
                    cs = u.size() - 2;
                } else {
                    ce = u.size() - 2;
                    break;
                }
            }
        }
        b.push_back(bn);
        n++;
    }
    
    if (t < ce) {
        return u[t];
    }
    
    int p = ce - cs;
    long long diff = u[ce] - u[cs];
    long long cycles = (t - cs) / p;
    int remainder = (t - cs) % p;
    
    return u[cs + remainder] + cycles * diff;
}

int main() {
    long long total = 0;
    long long k = 100000000000LL;
    for (int n = 2; n <= 10; n++) {
        total += Ulam(2, k);
    }
    cout << total << endl;
    return 0;
}
```
