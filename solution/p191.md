# Prize Strings

## Description du Problème
Une école offre une récompense aux élèves qui ont une présence parfaite et qui ne sont jamais en retard pendant 30 jours consécutifs.

Un élève peut être :
- À l'heure (O)
- En retard (L) - au plus une fois au total
- Absent (A) - au plus deux fois consécutives

**Objectif** : Combien de "chaînes de récompense" (prize strings) de 30 jours existent ?

---

## Exemples
### Exemple 1
**Entrée** : `days = 3`
**Sortie** : Nombre de chaînes valides
**Explication** : Exemples valides : OOO, OOA, OOL, OAO, etc. (sous les contraintes).

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique.
2. `numPrizeStrings[i][j][k]` : nombre de chaînes de longueur `i` avec :
   - Exactement `j` absences consécutives à la fin.
   - Exactement `k` retards au total.
3. Cas de base : `numPrizeStrings[0][0][0] = 1`.
4. Transitions :
   - Si `j == 0` (pas d'absence à la fin) :
     - Ajouter O : `sum += numPrizeStrings[i-1][l][k]` pour tous les `l`.
     - Ajouter L (si `k > 0`) : `sum += numPrizeStrings[i-1][l][k-1]` pour tous les `l`.
   - Si `j > 0` (absences à la fin) :
     - Ajouter A : `sum = numPrizeStrings[i-1][j-1][k]`.
5. Additionner tous les `numPrizeStrings[NUM_DAYS][j][k]`.

### Observations
- On suit le nombre d'absences consécutives à la fin pour vérifier la contrainte "au plus 2 absences consécutives".
- On suit le nombre total de retards pour vérifier "au plus 1 retard".

### Complexité
- **Temporelle : O(DAYS × MAX_ABSENT × MAX_LATE × MAX_ABSENT)**.
- **Spatiale : O(DAYS × MAX_ABSENT × MAX_LATE)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

const int NUM_DAYS = 30;
const int MAX_ABSENT = 2;
const int MAX_LATE = 1;

int main() {
    vector<vector<vector<long long>>> numPrizeStrings(
        NUM_DAYS + 1,
        vector<vector<long long>>(
            MAX_ABSENT + 1,
            vector<long long>(MAX_LATE + 1, 0)
        )
    );
    
    numPrizeStrings[0][0][0] = 1;
    
    for (int i = 1; i <= NUM_DAYS; i++) {
        for (int j = 0; j <= MAX_ABSENT; j++) {
            for (int k = 0; k <= MAX_LATE; k++) {
                long long sum = 0;
                if (j == 0) {
                    for (int l = 0; l <= MAX_ABSENT; l++)
                        sum += numPrizeStrings[i - 1][l][k];  // On time
                    if (k > 0) {
                        for (int l = 0; l <= MAX_ABSENT; l++)
                            sum += numPrizeStrings[i - 1][l][k - 1];  // Late
                    }
                } else {
                    sum = numPrizeStrings[i - 1][j - 1][k];  // Absent
                }
                numPrizeStrings[i][j][k] = sum;
            }
        }
    }
    
    long long sum = 0;
    for (int j = 0; j <= MAX_ABSENT; j++) {
        for (int k = 0; k <= MAX_LATE; k++)
            sum += numPrizeStrings[NUM_DAYS][j][k];
    }
    
    cout << sum << endl;
    return 0;
}
```
