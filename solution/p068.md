# Anneau magique 5-gon

## Description du Problème
Considérons l'anneau "magique" 3-gon suivant, rempli avec les nombres 1 à 6, et chaque ligne additionnant neuf.

En travaillant dans le sens horaire, et en commençant par le groupe de trois avec le nœud externe numériquement le plus bas (4,3,2 dans cet exemple), chaque solution peut être décrite de manière unique. Par exemple, la solution ci-dessus peut être décrite par l'ensemble : 4,3,2; 6,2,1; 5,1,3.

Il est possible de compléter l'anneau avec quatre totaux différents : 9, 10, 11, et 12. Il y a huit solutions au total.

En concaténant chaque groupe, il est possible de former des chaînes de 9 chiffres ; la chaîne maximale pour un anneau 3-gon est 432621513.

**Objectif** : En utilisant les nombres 1 à 10, et selon les arrangements, il est possible de former des chaînes de 16 et 17 chiffres. Quelle est la chaîne maximale de 16 chiffres pour un anneau "magique" 5-gon ?

---

## Exemples
### Exemple 1
**Entrée** : Anneau 3-gon avec nombres 1-6
**Sortie** : `432621513`
**Explication** : La chaîne maximale pour un anneau 3-gon est 432621513.

---

## Approche de Résolution

### Stratégie
1. Générer toutes les permutations des nombres 1 à 10.
2. Pour chaque permutation, vérifier si elle forme un anneau magique 5-gon valide.
3. Un anneau est valide si toutes les lignes ont la même somme.
4. Construire la chaîne de 16 chiffres en commençant par le nœud externe le plus bas.
5. Garder la chaîne lexicographiquement la plus grande.

### Observations
- L'anneau 5-gon a 5 nœuds externes et 5 nœuds internes.
- Chaque ligne consiste en un nœud externe et deux nœuds internes adjacents.
- La chaîne doit commencer par le nœud externe le plus bas pour être unique.
- On ne garde que les chaînes de 16 chiffres (pas 17).

### Complexité
- **Temporelle** : O(10! × 5) = O(10! × 5), pour toutes les permutations et vérifications.
- **Spatiale** : O(1), pour stocker la permutation courante.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>
#include <climits>

using namespace std;

bool nextPermutation(int* arr, int n) {
    int i = n - 2;
    while (i >= 0 && arr[i] >= arr[i + 1]) i--;
    if (i < 0) return false;
    
    int j = n - 1;
    while (arr[j] <= arr[i]) j--;
    swap(arr[i], arr[j]);
    reverse(arr + i + 1, arr + n);
    return true;
}

int main() {
    int state[10];
    for (int i = 0; i < 10; i++)
        state[i] = i + 1;
    
    string max = "";
    
    do {
        int sum = state[0] + state[5] + state[6];
        if (state[1] + state[6] + state[7] != sum ||
            state[2] + state[7] + state[8] != sum ||
            state[3] + state[8] + state[9] != sum ||
            state[4] + state[9] + state[5] != sum)
            continue;
        
        int minOuterIndex = -1;
        int minOuter = INT_MAX;
        for (int i = 0; i < 5; i++) {
            if (state[i] < minOuter) {
                minOuterIndex = i;
                minOuter = state[i];
            }
        }
        
        string s = "";
        for (int i = 0; i < 5; i++) {
            int idx = (minOuterIndex + i) % 5;
            s += to_string(state[idx]);
            s += to_string(state[idx + 5]);
            s += to_string(state[(idx + 1) % 5 + 5]);
        }
        
        if (s.length() == 16 && (max == "" || s > max))
            max = s;
    } while (nextPermutation(state, 10));
    
    cout << max << endl;
    return 0;
}
```
