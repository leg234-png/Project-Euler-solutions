# Sphere Packing

## Description du Problème
On veut placer 21 sphères de rayons 30000, 31000, 32000, ..., 50000 micromètres dans un tube cylindrique de rayon 50000 micromètres.

Les sphères doivent être placées de manière à minimiser la longueur totale du tube nécessaire.

**Objectif** : Quelle est la longueur minimale (en micromètres) nécessaire pour placer toutes les sphères ?

---

## Exemples
### Exemple 1
**Entrée** : 2 sphères de rayons r1 et r2
**Sortie** : Longueur minimale
**Explication** : La distance entre les centres est `√((r1 + r2 - R) × 2R)` où R est le rayon du tube.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique avec mémoïsation.
2. `minLength[i][mask]` : longueur minimale pour placer les sphères représentées par `mask`, avec la sphère `i` à l'extrémité gauche.
3. Pour chaque état :
   - Si une seule sphère reste : retourner son rayon.
   - Sinon : pour chaque sphère suivante `j` :
     - Calculer la distance entre les centres : `√((r_i + r_j - R) × 2R)`.
     - Récursivement calculer la longueur minimale pour les sphères restantes.
4. Utiliser un masque de bits pour représenter l'ensemble de sphères restantes.

### Observations
- La distance entre deux sphères de rayons `r1` et `r2` dans un tube de rayon `R` est `√((r1 + r2 - R) × 2R)`.
- On doit minimiser la longueur totale en choisissant l'ordre optimal des sphères.

### Complexité
- **Temporelle : O(n × 2^n)**, où n=21.
- **Spatiale : O(n × 2^n)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <climits>
#include <algorithm>

using namespace std;

const double TUBE_RADIUS = 50000.0;
vector<double> sphereRadii(21);
vector<vector<double>> minLength;

double findMinimumLength(int currentSphereIndex, int setOfSpheres) {
    if ((setOfSpheres & (1 << currentSphereIndex)) == 0)
        return 0;
    
    if (minLength[currentSphereIndex][setOfSpheres] >= 0)
        return minLength[currentSphereIndex][setOfSpheres];
    
    double result;
    if (__builtin_popcount(setOfSpheres) == 1) {
        result = sphereRadii[currentSphereIndex];
    } else {
        result = 1e18;
        int newSet = setOfSpheres ^ (1 << currentSphereIndex);
        for (int i = 0; i < 21; i++) {
            if ((newSet & (1 << i)) == 0)
                continue;
            double dist = sqrt((sphereRadii[i] + sphereRadii[currentSphereIndex] - TUBE_RADIUS) * 200000.0);
            result = min(result, dist + findMinimumLength(i, newSet));
        }
    }
    
    minLength[currentSphereIndex][setOfSpheres] = result;
    return result;
}

int main() {
    for (int i = 0; i < 21; i++)
        sphereRadii[i] = (i + 30) * 1000.0;
    
    minLength.assign(21, vector<double>(1 << 21, -1));
    
    double minTotal = 1e18;
    for (int i = 0; i < 21; i++)
        minTotal = min(minTotal, findMinimumLength(i, (1 << 21) - 1) + sphereRadii[i]);
    
    cout << (long long)round(minTotal) << endl;
    return 0;
}
```
