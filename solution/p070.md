# Permutation de la fonction totient

## Description du Problème
La fonction totient d'Euler, φ(n) [parfois appelée la fonction phi], est utilisée pour déterminer le nombre de nombres positifs inférieurs ou égaux à n qui sont relativement premiers à n. Par exemple, comme 1, 2, 4, 5, 7, et 8, sont tous inférieurs à neuf et relativement premiers à neuf, φ(9) = 6. Le nombre 1 est considéré comme relativement premier à tout nombre positif, donc φ(1) = 1.

Intéressant, φ(87109) = 79180, et on peut voir que 87109 est une permutation de 79180.

**Objectif** : Trouver la valeur de n, 1 < n < 10⁷, pour laquelle φ(n) est une permutation de n et le ratio n/φ(n) produit un minimum.

---

## Exemples
### Exemple 1
**Entrée** : Recherche dans la plage
**Sortie** : `8319823` (exemple, la vraie réponse peut être différente)
**Explication** : On cherche n tel que les chiffres de n et φ(n) soient les mêmes, et n/φ(n) soit minimisé.

---

## Approche de Résolution

### Stratégie
1. Pré-calculer tous les totients jusqu'à 10⁷ - 1 en utilisant un crible.
2. Pour chaque n de 2 à 10⁷ - 1 :
   - Vérifier si n et φ(n) ont les mêmes chiffres (permutation).
   - Si oui, calculer le ratio n/φ(n) et garder le minimum.
3. Retourner la valeur de n qui donne le ratio minimum.

### Observations
- Pour que n/φ(n) soit minimisé, on veut que φ(n) soit aussi grand que possible par rapport à n.
- Cela signifie que n devrait avoir peu de facteurs premiers, et ces facteurs devraient être grands.
- Cependant, n et φ(n) doivent être des permutations l'un de l'autre, ce qui ajoute une contrainte.
- On vérifie les permutations en triant les chiffres des deux nombres.

### Complexité
- **Temporelle** : O(n log log n + n log n), pour le crible et les vérifications de permutation.
- **Spatiale** : O(n), pour stocker les totients.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

using namespace std;

vector<int> listTotients(int n) {
    vector<int> totients(n + 1);
    for (int i = 0; i <= n; i++)
        totients[i] = i;
    
    for (int i = 2; i <= n; i++) {
        if (totients[i] == i) {  // i est premier
            for (int j = i; j <= n; j += i)
                totients[j] -= totients[j] / i;
        }
    }
    return totients;
}

bool hasSameDigits(int x, int y) {
    string xdigits = to_string(x);
    string ydigits = to_string(y);
    sort(xdigits.begin(), xdigits.end());
    sort(ydigits.begin(), ydigits.end());
    return xdigits == ydigits;
}

int main() {
    const int LIMIT = 10000000;
    vector<int> totients = listTotients(LIMIT - 1);
    
    int minNumer = 1;
    int minDenom = 0;
    
    for (int n = 2; n < (int)totients.size(); n++) {
        int tot = totients[n];
        if ((long long)n * minDenom < (long long)minNumer * tot && hasSameDigits(n, tot)) {
            minNumer = n;
            minDenom = tot;
        }
    }
    
    if (minDenom == 0) {
        cerr << "Not found" << endl;
        return 1;
    }
    
    cout << minNumer << endl;
    return 0;
}
```
