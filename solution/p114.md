# Counting Block Combinations I

## Description du Problème
Une rangée mesurant sept unités de longueur a des blocs rouges d'une longueur minimale de trois unités placés dessus, de telle sorte que deux blocs rouges quelconques (qui peuvent être de longueurs différentes) sont séparés par au moins un carré gris. Il y a exactement dix-sept façons de faire cela.

**Objectif** : Combien de façons peut-on remplir une rangée mesurant cinquante unités de longueur ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 7`
**Sortie** : `17`
**Explication** : Il y a 17 façons de remplir une rangée de 7 unités.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique.
2. ways[n] = nombre de façons de remplir une rangée de n unités.
3. Cas de base : ways[0] = 1, ways[1] = 1, ways[2] = 1.
4. Pour n ≥ 3 :
   - Si on place un carré gris à gauche : ways[n-1]
   - Si on place un bloc rouge de longueur k (3 ≤ k ≤ n) : 1 (si k=n) ou ways[n-k-1] (si k<n)

### Observations
- On doit séparer les blocs rouges par au moins un carré gris.
- On peut mélanger différentes longueurs de blocs rouges.

### Complexité
- **Temporelle** : O(n²), où n=50.
- **Spatiale** : O(n).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

const int LENGTH = 50;

int main() {
    std::vector<long long> ways(LENGTH + 1);
    ways[0] = 1;
    ways[1] = 1;
    ways[2] = 1;
    
    for (int n = 3; n <= LENGTH; n++) {
        long long sum = ways[n - 1] + 1;  // Carré gris ou bloc rouge de longueur n
        for (int k = 3; k < n; k++) {
            sum += ways[n - k - 1];  // Bloc rouge de longueur k suivi d'un carré gris
        }
        ways[n] = sum;
    }
    
    std::cout << ways[LENGTH] << std::endl;
    return 0;
}
```
