# The hyperexponentiation of a number

## Description du Problème
L'hyperexponentiation (ou tétration) de `a` à la puissance `b`, notée `a↑↑b` ou `^ba`, est définie récursivement par :
- `a↑↑1 = a`
- `a↑↑(k+1) = a^(a↑↑k)`

Par exemple, `3↑↑2 = 3^3 = 27`, `3↑↑3 = 3^27 = 7625597484987`, et `3↑↑4` est environ `10^(3.6383346400240996*10^12)`.

**Objectif** : Trouver les 8 derniers chiffres de `1777↑↑1855`.

---

## Exemples
### Exemple 1
**Entrée** : `a = 3, b = 2`
**Sortie** : `27`
**Explication** : `3↑↑2 = 3^3 = 27`.

### Exemple 2
**Entrée** : `a = 3, b = 3`
**Sortie** : `7625597484987`
**Explication** : `3↑↑3 = 3^27 = 7625597484987`.

---

## Approche de Résolution

### Stratégie
1. Utiliser le théorème d'Euler : Si `gcd(a, m) = 1`, alors `a^φ(m) ≡ 1 (mod m)`.
2. Pour calculer `a↑↑b mod m` :
   - Si `b = 1` : retourner `a mod m`.
   - Sinon : `a↑↑b mod m = a^(a↑↑(b-1) mod φ(m)) mod m`.
3. Calculer récursivement en utilisant la fonction indicatrice d'Euler `φ(m)`.
4. Pour `m = 10^8`, on a `φ(10^8) = 4 × 10^7`.

### Observations
- La tétration croît extrêmement rapidement.
- On utilise le théorème d'Euler pour réduire l'exposant modulo `φ(m)`.
- La récursion s'arrête quand on atteint `b = 1`.

### Complexité
- **Temporelle : O(b × log m)**, où b est le niveau de tétration.
- **Spatiale : O(b)** pour la pile de récursion.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long totient(long long n) {
    long long result = n;
    for (long long i = 2; i * i <= n; i++) {
        if (n % i == 0) {
            while (n % i == 0)
                n /= i;
            result -= result / i;
        }
    }
    if (n > 1)
        result -= result / n;
    return result;
}

long long powMod(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp % 2 == 1)
            result = (result * base) % mod;
        base = (base * base) % mod;
        exp /= 2;
    }
    return result;
}

long long tetrationMod(long long x, long long y, long long m) {
    if (y == 1)
        return x % m;
    else
        return powMod(x, tetrationMod(x, y - 1, totient(m)), m);
}

int main() {
    long long result = tetrationMod(1777, 1855, 100000000);
    cout << result << endl;
    return 0;
}
```
