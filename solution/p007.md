# 10 001ème Nombre Premier

## Description du Problème
En listant les six premiers nombres premiers : 2, 3, 5, 7, 11 et 13, on peut voir que le 6ème nombre premier est 13.

**Objectif** : Quel est le 10 001ème nombre premier ?

---

## Exemples
### Exemple 1
**Entrée** : Position = 6
**Sortie** : `13`
**Explication** : Les six premiers nombres premiers sont 2, 3, 5, 7, 11, 13. Le 6ème est 13.

---

## Approche de Résolution

### Stratégie
1. Tester chaque nombre à partir de 2 pour vérifier s'il est premier.
2. Compter les nombres premiers trouvés.
3. Retourner le 10 001ème nombre premier.

### Observations
- On peut utiliser un test de primalité simple pour chaque nombre.
- On peut optimiser en utilisant le crible d'Ératosthène, mais pour 10 001 nombres premiers, un test simple suffit.
- Pour tester si un nombre n est premier, on vérifie s'il est divisible par un nombre entre 2 et √n.

### Complexité
- **Temporelle** : O(n × √n), où `n` est le nombre premier recherché (environ 104743).
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

bool isPrime(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    
    int sqrtN = sqrt(n);
    for (int i = 3; i <= sqrtN; i += 2) {
        if (n % i == 0) {
            return false;
        }
    }
    return true;
}

int main() {
    int count = 0;
    int num = 2;
    
    while (count < 10001) {
        if (isPrime(num)) {
            ++count;
            if (count == 10001) {
                std::cout << num << std::endl;
                break;
            }
        }
        ++num;
    }
    
    return 0;
}
```
