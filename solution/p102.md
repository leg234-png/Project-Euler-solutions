# Triangle Containment

## Description du Problème
Trois points distincts sont tracés au hasard sur un plan cartésien, pour lesquels -1000 ≤ x, y ≤ 1000, de telle sorte qu'un triangle soit formé.

Considérons les deux triangles suivants :
- A(-340,495), B(-153,-910), C(835,-947)
- X(-175,41), Y(-421,-714), Z(574,-645)

On peut vérifier que le triangle ABC contient l'origine, alors que le triangle XYZ ne le contient pas.

En utilisant triangles.txt, un fichier texte de 27K contenant les coordonnées de mille triangles "aléatoires", trouver le nombre de triangles dont l'intérieur contient l'origine.

**Objectif** : Compter combien de triangles contiennent l'origine (0,0).

---

## Exemples
### Exemple 1
**Entrée** : Triangle ABC avec A(-340,495), B(-153,-910), C(835,-947)
**Sortie** : `true` (contient l'origine)
**Explication** : Le point (0,0) est à l'intérieur du triangle ABC.

### Exemple 2
**Entrée** : Triangle XYZ avec X(-175,41), Y(-421,-714), Z(574,-645)
**Sortie** : `false` (ne contient pas l'origine)
**Explication** : Le point (0,0) n'est pas à l'intérieur du triangle XYZ.

---

## Approche de Résolution

### Stratégie
1. Pour chaque triangle, vérifier si l'origine (0,0) est à l'intérieur.
2. Utiliser le produit vectoriel pour déterminer de quel côté de chaque arête se trouve l'origine.
3. Si l'origine est du même côté de toutes les arêtes (tous les produits vectoriels ont le même signe), alors elle est à l'intérieur.

### Observations
- Pour un triangle avec sommets A, B, C, l'origine est à l'intérieur si elle est du même côté de toutes les arêtes.
- Le produit vectoriel (B-A) × (O-A) donne le signe de l'orientation.
- Si les trois produits vectoriels ont le même signe (ou zéro), l'origine est à l'intérieur.

### Complexité
- **Temporelle** : O(n), où n=1000 est le nombre de triangles.
- **Spatiale** : O(1) pour chaque triangle.

#include <iostream>
#include <vector>

bool isInTriangle(const std::vector<int>& t, int x, int y) {
    // Calculer les produits vectoriels pour chaque arête
    int a = ((t[0] - t[2]) * (y - t[1]) - (t[1] - t[3]) * (x - t[0]));
    int b = ((t[2] - t[4]) * (y - t[3]) - (t[3] - t[5]) * (x - t[2]));
    int c = ((t[4] - t[0]) * (y - t[5]) - (t[5] - t[1]) * (x - t[4]));
    
    // Vérifier si tous les signes sont identiques (ou zéro)
    return (a == 0 || b == 0 || c == 0) || 
           ((a > 0 && b > 0 && c > 0) || (a < 0 && b < 0 && c < 0));
}
