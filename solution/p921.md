# Récurrence dorée

## Description du Problème
Soit p_n et q_n définis par :
- p₀ = 1, q₀ = 2
- p_{n+1} = p_n + q_n
- q_{n+1} = p_n

Soit s(n) = p_n⁵ + q_n⁵. Donc s(0) = 1⁵ + 2⁵ = 33.

La séquence de Fibonacci est définie comme : F₁ = 1, F₂ = 1, F_n = F_{n-1} + F_{n-2} pour n > 2.

Définissons S(m) = ∑_{i=2}^m s(F_i).

**Objectif** : Trouver S(10⁶). Donner votre réponse modulo 10⁹+7.

---

## Exemples
### Exemple 1
**Entrée** : `n = 0`
**Sortie** : `33`
**Explication** : s(0) = 1⁵ + 2⁵ = 33.

---

## Approche de Résolution

### Stratégie
1. Calculer p_n et q_n en utilisant les relations de récurrence.
2. Calculer s(n) = p_n⁵ + q_n⁵.
3. Calculer les nombres de Fibonacci F_i pour i de 2 à m.
4. Calculer s(F_i) pour chaque F_i.
5. Sommer tous les résultats.

### Observations
- On peut calculer p_n et q_n efficacement avec l'exponentiation matricielle.
- Les nombres de Fibonacci peuvent être calculés rapidement.
- On doit gérer l'arithmétique modulaire pour les grandes valeurs.

### Complexité
- **Temporelle** : O(m × log F_m) pour calculer tous les s(F_i).
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

long long mod = 1000000007;

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

pair<long long, long long> compute_pq(long long n, long long mod) {
    if (n == 0) return {1, 2};
    
    long long p = 1, q = 2;
    for (long long i = 1; i <= n; i++) {
        long long new_p = (p + q) % mod;
        long long new_q = p;
        p = new_p;
        q = new_q;
    }
    
    return {p, q};
}

long long s(long long n, long long mod) {
    auto [p, q] = compute_pq(n, mod);
    long long p5 = mod_pow(p, 5, mod);
    long long q5 = mod_pow(q, 5, mod);
    return (p5 + q5) % mod;
}

long long fibonacci(int n) {
    if (n <= 2) return 1;
    long long a = 1, b = 1;
    for (int i = 3; i <= n; i++) {
        long long temp = a + b;
        a = b;
        b = temp;
    }
    return b;
}

long long compute(int m, long long mod) {
    long long total = 0;
    for (int i = 2; i <= m; i++) {
        long long F_i = fibonacci(i);
        total = (total + s(F_i, mod)) % mod;
    }
    return total;
}

int main() {
    cout << compute(1000000, mod) << endl;
    return 0;
}
```
