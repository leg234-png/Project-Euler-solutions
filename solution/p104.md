# Pandigital Fibonacci Ends

## Description du Problème
La séquence de Fibonacci est définie par la relation de récurrence :
F_n = F_{n-1} + F_{n-2}, où F₁ = 1 et F₂ = 1.

Il s'avère que F₅₄₁, qui contient 113 chiffres, est le premier nombre de Fibonacci pour lequel les neuf derniers chiffres sont 1-9 pandigitaux (contiennent tous les chiffres 1 à 9, mais pas nécessairement dans l'ordre). Et F₂₇₄₉, qui contient 575 chiffres, est le premier nombre de Fibonacci pour lequel les neuf premiers chiffres sont 1-9 pandigitaux.

**Objectif** : Étant donné que F_k est le premier nombre de Fibonacci pour lequel les neuf premiers chiffres ET les neuf derniers chiffres sont 1-9 pandigitaux, trouver k.

---

## Exemples
### Exemple 1
**Entrée** : `F₅₄₁`
**Sortie** : `541` (pour les 9 derniers chiffres pandigitaux)
**Explication** : F₅₄₁ a 113 chiffres et ses 9 derniers chiffres sont pandigitaux.

### Exemple 2
**Entrée** : `F₂₇₄₉`
**Sortie** : `2749` (pour les 9 premiers chiffres pandigitaux)
**Explication** : F₂₇₄₉ a 575 chiffres et ses 9 premiers chiffres sont pandigitaux.

---

## Approche de Résolution

### Stratégie
1. Calculer les nombres de Fibonacci en utilisant l'arithmétique modulaire pour les 9 derniers chiffres.
2. Pour les 9 premiers chiffres, utiliser une approximation logarithmique pour éviter de calculer le nombre entier.
3. Vérifier si les 9 premiers et les 9 derniers chiffres sont pandigitaux.
4. Continuer jusqu'à trouver le premier nombre qui satisfait les deux conditions.

### Observations
- On peut calculer F_n mod 10⁹ pour obtenir les 9 derniers chiffres.
- Pour les 9 premiers chiffres, on utilise log₁₀(F_n) pour estimer le nombre de chiffres, puis on extrait les 9 premiers.
- Un nombre est pandigital 1-9 s'il contient exactement les chiffres 1 à 9 une fois chacun.

### Complexité
- **Temporelle** : O(k log k), où k est l'index recherché, car on calcule F_k.
- **Spatiale** : O(1), car on travaille avec des nombres de taille fixe.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>
#include <cmath>
#include <gmpxx.h>

bool isPandigital(const std::string& s) {
    if (s.length() != 9) return false;
    std::string sorted = s;
    std::sort(sorted.begin(), sorted.end());
    return sorted == "123456789";
}

std::string leading9Digits(const mpz_class& x) {
    // Utiliser le logarithme pour estimer le nombre de chiffres
    int log10 = (mpz_sizeinbase(x.get_mpz_t(), 2) - 1) * 3 / 10;
    
    // Couper les chiffres de droite
    mpz_class temp = x;
    if (log10 + 1 > 9) {
        temp = temp / mpz_class(10).pow(log10 + 1 - 9);
    }
    
    std::string result = temp.get_str();
    return result.substr(0, 9);
}

std::pair<mpz_class, mpz_class> fibonacci(int n) {
    if (n == 0) return {0, 1};
    if (n == 1) return {1, 1};
    
    auto [a, b] = fibonacci(n / 2);
    mpz_class c = a * (2 * b - a);
    mpz_class d = a * a + b * b;
    
    if (n % 2 == 0) {
        return {c, d};
    } else {
        return {d, c + d};
    }
}

int main() {
    int a = 0, b = 1;
    int i = 0;
    
    while (true) {
        int c = (a + b) % 1000000000;
        a = b;
        b = c;
        i++;
        
        std::string last9 = std::to_string(a);
        if (last9.length() >= 9) {
            last9 = last9.substr(last9.length() - 9);
            if (isPandigital(last9)) {
                auto [fib, _] = fibonacci(i);
                if (fib % 1000000000 == a) {
                    std::string first9 = leading9Digits(fib);
                    if (isPandigital(first9)) {
                        std::cout << i << std::endl;
                        return 0;
                    }
                }
            }
        }
    }
    
    return 0;
}
```
