# Searching a triangular array for a sub-triangle having minimum-sum

## Description du Problème
Un triangle de nombres est généré en utilisant un générateur pseudo-aléatoire (Linear Congruential Generator). Chaque ligne `i` contient `i+1` nombres.

**Objectif** : Trouver la somme minimale d'un sous-triangle (triangle dont le sommet est à la position `(i, j)` et qui s'étend vers le bas).

---

## Exemples
### Exemple 1
**Entrée** : Triangle de 3 lignes
**Sortie** : Somme minimale d'un sous-triangle
**Explication** : Un sous-triangle commence à un sommet et s'étend vers le bas.

---

## Approche de Résolution

### Stratégie
1. Générer le triangle en utilisant le générateur LCG.
2. Calculer les sommes cumulatives pour chaque ligne (préfixes).
3. Pour chaque position `(i, j)` comme sommet :
   - Pour chaque ligne `k` de `i` à la fin :
   - Calculer la somme du sous-triangle de `(i, j)` à la ligne `k` en utilisant les sommes cumulatives.
   - Mettre à jour la somme minimale.
4. Le générateur LCG : `state = (615949 * state + 797807) & ((1 << 20) - 1)`, puis `return state - (1 << 19)`.

### Observations
- Les sommes cumulatives permettent de calculer rapidement les sommes de sous-triangles.
- Un sous-triangle de sommet `(i, j)` et de base à la ligne `k` contient les éléments de `(i, j)` à `(k, j + k - i)`.

### Complexité
- **Temporelle : O(ROWS³)**, où ROWS=1000.
- **Spatiale : O(ROWS²)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <climits>
#include <algorithm>

using namespace std;

const int ROWS = 1000;

class LcgRandom {
private:
    int state;
    
public:
    LcgRandom() : state(0) {}
    
    int next() {
        state = (615949 * state + 797807) & ((1 << 20) - 1);
        return state - (1 << 19);
    }
};

int main() {
    LcgRandom rand;
    vector<vector<int>> triangle(ROWS);
    
    for (int i = 0; i < ROWS; i++) {
        triangle[i].resize(i + 1);
        for (int j = 0; j <= i; j++)
            triangle[i][j] = rand.next();
    }
    
    // Calculate cumulative sums for each row
    vector<vector<int>> rowSums(ROWS);
    for (int i = 0; i < ROWS; i++) {
        rowSums[i].resize(i + 2);
        rowSums[i][0] = 0;
        for (int j = 0; j <= i; j++)
            rowSums[i][j + 1] = rowSums[i][j] + triangle[i][j];
    }
    
    // Calculate minimum subtriangle sum
    long long minSum = 0;
    for (int i = 0; i < ROWS; i++) {
        for (int j = 0; j < triangle[i].size(); j++) {
            long long curSum = 0;
            for (int k = i; k < ROWS; k++) {
                curSum += rowSums[k][k - i + 1 + j] - rowSums[k][j];
                minSum = min(curSum, minSum);
            }
        }
    }
    
    cout << minSum << endl;
    return 0;
}
```
