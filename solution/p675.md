# 2^ω(n)

## Description du Problème
Soit ω(n) le nombre de facteurs premiers distincts de n.

Soit S(n) = ∑_{r|n} 2^ω(r).

Par exemple, S(6) = 2^ω(1) + 2^ω(2) + 2^ω(3) + 2^ω(6) = 2^0 + 2^1 + 2^1 + 2^2 = 9.

Soit F(n) = ∑_{i=2}^n S(i!).

F(10) = 4821.

**Objectif** : Trouver F(10⁸) mod 1000000007.

---

## Exemples
### Exemple 1
**Entrée** : `n = 6`
**Sortie** : `9`
**Explication** : S(6) = 9.

---

## Approche de Résolution

### Stratégie
1. On a ∑_{r|n} 2^ω(r) = d(n²) où d(n) est le nombre de diviseurs de n.
2. S(n) = d(n²).
3. On peut calculer S((i+1)!) à partir de S(i!).
4. Si v_p(i) = x et v_p(i+1) = y, alors S((i+1)!) = S(i!) × y/x pour tous les p < i+1.
5. Utiliser un crible pour trouver tous les facteurs premiers et la division modulaire.

### Observations
- On peut calculer S(i!) de manière incrémentale.
- La fonction diviseur peut être calculée à partir des facteurs premiers.
- On doit gérer l'arithmétique modulaire correctement.

### Complexité
- **Temporelle** : O(n × log log n) avec un crible.
- **Spatiale** : O(n) pour le crible.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <map>
#include <cmath>

using namespace std;

long long mod = 1000000007;

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long mod_inverse(long long a, long long mod) {
    return mod_pow(a, mod - 2, mod);
}

long long compute(int n) {
    vector<map<int, int>> factors(n + 1);
    
    // Calculer les facteurs premiers de chaque nombre
    for (int i = 2; i <= n; i++) {
        if (factors[i].empty()) {
            for (int j = i; j <= n; j += i) {
                int count = 0;
                int temp = j;
                while (temp % i == 0) {
                    count++;
                    temp /= i;
                }
                factors[j][i] = count;
            }
        }
    }
    
    long long S_factorial = 1; // S(1!) = 1
    long long total = 0;
    
    for (int i = 2; i <= n; i++) {
        // Calculer S(i!) à partir de S((i-1)!)
        for (auto [p, exp] : factors[i]) {
            long long old_exp = factors[i-1].count(p) ? factors[i-1][p] : 0;
            if (old_exp > 0) {
                S_factorial = (S_factorial * mod_inverse(2 * old_exp + 1, mod)) % mod;
            }
            S_factorial = (S_factorial * (2 * exp + 1)) % mod;
        }
        total = (total + S_factorial) % mod;
    }
    
    return total;
}

int main() {
    cout << compute(100000000) << endl;
    return 0;
}
```
