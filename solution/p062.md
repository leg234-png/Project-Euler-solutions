# Permutations cubiques

## Description du Problème
Le cube, 41063625 (345³), peut être permuté pour produire deux autres cubes : 56623104 (384³) et 66430125 (405³). En fait, 41063625 est le plus petit cube qui a exactement trois permutations de ses chiffres qui sont aussi des cubes.

**Objectif** : Trouver le plus petit cube pour lequel exactement cinq permutations de ses chiffres sont des cubes.

---

## Exemples
### Exemple 1
**Entrée** : Recherche de cubes avec 3 permutations
**Sortie** : `41063625`
**Explication** : 41063625 = 345³, et ses permutations 56623104 = 384³ et 66430125 = 405³ sont aussi des cubes.

---

## Approche de Résolution

### Stratégie
1. Générer des cubes successifs et les convertir en "classe de nombre" (chiffres triés).
2. Pour chaque classe de nombre, maintenir le plus petit cube et le nombre de cubes trouvés.
3. Lorsqu'on passe à un nombre de chiffres supérieur, vérifier si une classe a exactement 5 cubes.
4. Retourner le plus petit cube de cette classe.

### Observations
- Les cubes avec le même ensemble de chiffres (permutations) appartiennent à la même "classe".
- On groupe les cubes par leur représentation en chiffres triés.
- On traite les cubes par nombre de chiffres pour optimiser la recherche.

### Complexité
- **Temporelle** : O(N log N), où N est le nombre de cubes à générer jusqu'à trouver la solution.
- **Spatiale** : O(N), pour stocker les classes de nombres et leurs compteurs.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <map>
#include <algorithm>
#include <gmpxx.h>

using namespace std;

string getCubeNumberClass(mpz_class x) {
    string s = x.get_str();
    sort(s.begin(), s.end());
    return s;
}

mpz_class cube(mpz_class x) {
    return x * x * x;
}

int main() {
    int numDigits = 0;
    map<string, int> lowest;
    map<string, int> counts;
    
    for (mpz_class i = 0; ; i++) {
        mpz_class c = cube(i);
        string numClass = getCubeNumberClass(c);
        
        if ((int)numClass.length() > numDigits) {
            // Traiter et vider les données pour un nombre de chiffres plus petit
            mpz_class min = -1;
            for (auto& p : counts) {
                if (p.second == 5) {
                    if (min == -1 || lowest[p.first] < min)
                        min = lowest[p.first];
                }
            }
            if (min != -1) {
                cout << cube(min) << endl;
                return 0;
            }
            
            lowest.clear();
            counts.clear();
            numDigits = numClass.length();
        }
        
        if (lowest.find(numClass) == lowest.end()) {
            lowest[numClass] = i.get_si();
            counts[numClass] = 0;
        }
        counts[numClass]++;
    }
    
    return 1;
}
```
