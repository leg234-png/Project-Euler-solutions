# Nombres Non Abondants

## Description du Problème
Un nombre parfait est un nombre pour lequel la somme de ses diviseurs propres est égale au nombre lui-même. Par exemple, la somme des diviseurs propres de 28 serait 1 + 2 + 4 + 7 + 14 = 28, ce qui signifie que 28 est un nombre parfait.

Un nombre n est appelé déficient si la somme de ses diviseurs propres est inférieure à n et il est appelé abondant si cette somme dépasse n.

Comme 12 est le plus petit nombre abondant, 1 + 2 + 3 + 4 + 6 = 16, le plus petit nombre qui peut être écrit comme la somme de deux nombres abondants est 24. Par analyse mathématique, on peut montrer que tous les entiers supérieurs à 28123 peuvent être écrits comme la somme de deux nombres abondants. Cependant, cette limite supérieure ne peut pas être réduite par l'analyse bien qu'on sache que le plus grand nombre qui ne peut pas être exprimé comme la somme de deux nombres abondants est inférieur à cette limite.

**Objectif** : Trouver la somme de tous les entiers positifs qui ne peuvent pas être écrits comme la somme de deux nombres abondants.

---

## Exemples
### Exemple 1
**Entrée** : Limite = 28123
**Sortie** : `4179871`
**Explication** : La somme de tous les nombres qui ne peuvent pas être écrits comme la somme de deux nombres abondants.

---

## Approche de Résolution

### Stratégie
1. Créer un tableau pour marquer quels nombres sont abondants (jusqu'à 28123).
2. Pour chaque nombre de 1 à 28123, vérifier s'il peut être écrit comme la somme de deux nombres abondants.
3. Si non, l'ajouter à la somme.
4. Retourner la somme.

### Observations
- Un nombre est abondant si la somme de ses diviseurs propres > n.
- Pour vérifier si un nombre n peut être écrit comme la somme de deux nombres abondants, tester toutes les paires (i, n-i) où i et n-i sont abondants.
- On peut optimiser en créant d'abord une liste de tous les nombres abondants.

### Complexité
- **Temporelle** : O(n²), où `n` = 28123.
- **Spatiale** : O(n), pour stocker le tableau des nombres abondants.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

int divisorSum(int n) {
    int sum = 1;
    int sqrtN = sqrt(n);
    
    for (int i = 2; i <= sqrtN; ++i) {
        if (n % i == 0) {
            sum += i;
            if (i != n / i) {
                sum += n / i;
            }
        }
    }
    
    return sum;
}

bool isAbundant(int n) {
    return divisorSum(n) > n;
}

bool isSumOf2Abundants(int n, const std::vector<bool>& isAbundant) {
    for (int i = 0; i <= n; ++i) {
        if (isAbundant[i] && isAbundant[n - i]) {
            return true;
        }
    }
    return false;
}

int main() {
    const int limit = 28123;
    std::vector<bool> abundant(limit + 1, false);
    
    for (int i = 1; i <= limit; ++i) {
        abundant[i] = isAbundant(i);
    }
    
    int sum = 0;
    for (int i = 1; i <= limit; ++i) {
        if (!isSumOf2Abundants(i, abundant)) {
            sum += i;
        }
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
