# Plus grande permutation de chiffres II

## Description du Problème
Soit B(n) le plus petit nombre plus grand que n qui peut être formé en réarrangeant les chiffres de n, ou 0 si un tel nombre n'existe pas.

Définissons a₀ = 0 et a_n = a_{n-1}² + 2 pour n > 0.

Soit U(N) = ∑_{n=1}^N B(a_n). On vous donne U(10) ≡ 543870437 (mod 10⁹+7).

**Objectif** : Trouver U(10¹⁶). Donner votre réponse modulo 10⁹+7.

---

## Exemples
### Exemple 1
**Entrée** : `N = 10`
**Sortie** : `543870437`
**Explication** : U(10) ≡ 543870437 (mod 10⁹+7).

---

## Approche de Résolution

### Stratégie
1. Calculer a_n = a_{n-1}² + 2 pour n de 1 à N.
2. Pour chaque a_n, trouver B(a_n) : le plus petit nombre > a_n formé en réarrangeant les chiffres de a_n.
3. Si tous les chiffres sont en ordre décroissant, B(a_n) = 0.
4. Sinon, trouver la prochaine permutation lexicographique.
5. Sommer tous les B(a_n).

### Observations
- On peut utiliser l'algorithme de prochaine permutation.
- Pour de très grandes valeurs de a_n, on doit gérer les grands nombres.
- On peut optimiser en détectant les patterns dans la séquence a_n.

### Complexité
- **Temporelle** : O(N × d) où d est le nombre de chiffres.
- **Spatiale** : O(d) pour stocker les chiffres.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>
#include <vector>

using namespace std;

long long mod = 1000000007;

long long B(long long n) {
    string s = to_string(n);
    
    // Vérifier si on peut former un nombre plus grand
    string sorted = s;
    sort(sorted.rbegin(), sorted.rend());
    if (s == sorted) {
        return 0; // Pas de permutation plus grande
    }
    
    // Trouver la prochaine permutation
    string next = s;
    next_permutation(next.begin(), next.end());
    
    return stoll(next);
}

long long compute(long long N, long long mod) {
    long long a = 0;
    long long total = 0;
    
    for (long long n = 1; n <= N; n++) {
        a = (a * a + 2) % mod;
        long long b_val = B(a);
        total = (total + b_val) % mod;
    }
    
    return total;
}

int main() {
    long long N = 1;
    for (int i = 0; i < 16; i++) {
        N *= 10;
    }
    cout << compute(N, mod) << endl;
    return 0;
}
```
