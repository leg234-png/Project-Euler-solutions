# Spirale Première

## Description du Problème
En partant de 1 et en se déplaçant dans le sens des aiguilles d'une montre, une spirale carrée de 5 par 5 est formée comme suit :

21 22 23 24 25
20  7  8  9 10
19  6  1  2 11
18  5  4  3 12
17 16 15 14 13

On peut vérifier que la somme des nombres sur les diagonales est 101.

**Objectif** : Si on continue ce processus, quelle est la longueur du côté de la spirale carrée pour laquelle le ratio de nombres premiers le long des deux diagonales tombe en dessous de 10% pour la première fois ?

---

## Exemples
### Exemple 1
**Entrée** : Spirale 5×5
**Sortie** : `101` (somme des diagonales)
**Explication** : Les nombres sur les diagonales sont 1, 3, 5, 7, 9, 13, 17, 21, 25.

---

## Approche de Résolution

### Stratégie
1. Pour chaque côté n (commençant à 1, incrémentant de 2) :
   - Calculer les 4 coins de la spirale n×n :
     - Coin inférieur droit : n²
     - Coin inférieur gauche : n² - (n - 1)
     - Coin supérieur gauche : n² - 2(n - 1)
     - Coin supérieur droit : n² - 3(n - 1)
   - Vérifier si chaque coin est premier.
   - Compter le nombre total de nombres premiers sur les diagonales.
   - Calculer le ratio : nombres premiers / (2n - 1).
   - Si le ratio < 0.1, retourner n.
2. Retourner le résultat.

### Observations
- Pour une spirale n×n, il y a 2n - 1 nombres sur les diagonales.
- On peut calculer les coins directement sans construire la spirale.
- On ignore le centre (1) car il n'est pas premier.

### Complexité
- **Temporelle** : O(n × √m), où n est la taille de la spirale et m est le plus grand nombre à tester.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>
#include <vector>

bool isPrime(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    
    int sqrtN = sqrt(n);
    for (int i = 3; i <= sqrtN; i += 2) {
        if (n % i == 0) {
            return false;
        }
    }
    return true;
}

int main() {
    int numPrimes = 0;
    for (int n = 1; ; n += 2) {
        for (int i = 0; i < 4; ++i) {
            int corner = n * n - i * (n - 1);
            if (isPrime(corner)) {
                numPrimes++;
            }
        }
        
        if (n > 1) {
            int total = 2 * n - 1;
            if (numPrimes * 10 < total) {
                std::cout << n << std::endl;
                return 0;
            }
        }
    }
}
```
