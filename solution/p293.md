# Nombres pseudo-Fortunés

## Description du Problème
Un entier positif pair N sera appelé admissible s'il est une puissance de 2 ou si ses facteurs premiers distincts sont des nombres premiers consécutifs.

Les douze premiers nombres admissibles sont 2,4,6,8,12,16,18,24,30,32,36,48.

Si N est admissible, le plus petit entier M > 1 tel que N+M est premier sera appelé le nombre pseudo-Fortuné pour N.

Par exemple, N=630 est admissible puisque c'est un nombre pair et ses facteurs premiers distincts sont les nombres premiers consécutifs 2,3,5 et 7.

Le prochain nombre premier après 631 est 641 ; donc, le nombre pseudo-Fortuné pour 630 est M=11.

On peut aussi voir que le nombre pseudo-Fortuné pour 16 est 3.

**Objectif** : Trouver la somme de tous les nombres pseudo-Fortunés distincts pour les nombres admissibles N inférieurs à 10⁹.

---

## Exemples
### Exemple 1
**Entrée** : `N = 630`
**Sortie** : `11`
**Explication** : Le nombre pseudo-Fortuné pour 630 est 11.

### Exemple 2
**Entrée** : `N = 16`
**Sortie** : `3`
**Explication** : Le nombre pseudo-Fortuné pour 16 est 3.

---

## Approche de Résolution

### Stratégie
1. Si N est admissible et n'est pas une puissance de 2, alors il doit être un produit d'au moins 2 nombres premiers consécutifs.
2. On n'a besoin que des nombres premiers jusqu'à la racine carrée de la limite.
3. Générer tous les nombres admissibles N < 10⁹.
4. Pour chaque N admissible, trouver le plus petit M > 1 tel que N+M est premier.
5. Collecter tous les nombres pseudo-Fortunés distincts et les sommer.

### Observations
- Les nombres admissibles sont soit des puissances de 2, soit des produits de nombres premiers consécutifs.
- On peut générer efficacement les nombres admissibles en multipliant des nombres premiers consécutifs.
- On doit trouver le prochain nombre premier après N+1.

### Complexité
- **Temporelle** : O(A × P) où A est le nombre de nombres admissibles et P est le coût de trouver le prochain premier.
- **Spatiale** : O(A) pour stocker les nombres pseudo-Fortunés distincts.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <cmath>

using namespace std;

vector<int> list_primes(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

bool is_prime(long long x) {
    if (x <= 1) return false;
    if (x <= 3) return true;
    if (x % 2 == 0) return false;
    for (long long i = 3; i * i <= x; i += 2) {
        if (x % i == 0) return false;
    }
    return true;
}

long long next_prime(long long n) {
    long long candidate = n + 1;
    while (!is_prime(candidate)) {
        candidate++;
    }
    return candidate;
}

long long compute(long long limit) {
    vector<int> primes = list_primes(sqrt(limit));
    set<long long> pseudo_fortunate;
    
    // Puissances de 2
    for (long long p2 = 2; p2 < limit; p2 *= 2) {
        long long next = next_prime(p2 + 1);
        pseudo_fortunate.insert(next - p2);
    }
    
    // Produits de nombres premiers consécutifs
    for (size_t i = 0; i < primes.size(); i++) {
        long long product = 1;
        for (size_t j = i; j < primes.size(); j++) {
            if (product > limit / primes[j]) break;
            product *= primes[j];
            if (product >= limit) break;
            
            long long next = next_prime(product + 1);
            pseudo_fortunate.insert(next - product);
        }
    }
    
    long long total = 0;
    for (long long pf : pseudo_fortunate) {
        total += pf;
    }
    
    return total;
}

int main() {
    cout << compute(1000000000) << endl;
    return 0;
}
```
