# Nombres dominants

## Description du Problème
Un nombre dominant est un nombre positif qui contient plus de la moitié de ses chiffres comme le même chiffre.

Par exemple, 2022 est un nombre dominant car il contient trois 2 (plus de la moitié de ses 4 chiffres).

Soit D(N) le nombre de nombres dominants inférieurs à 10^N.

Par exemple, D(4) = 603 et D(10) = 21893256.

**Objectif** : Trouver D(2022). Donner votre réponse modulo 1000000007.

---

## Exemples
### Exemple 1
**Entrée** : `N = 4`
**Sortie** : `603`
**Explication** : D(4) = 603.

---

## Approche de Résolution

### Stratégie
1. D(N) = ∑_{n=1}^N ∑_{m=n/2+1}^n c(n, m) où c(n, m) = 9^(n-m+1) × C(n, m).
2. Pour un nombre de n chiffres, choisir m positions pour le chiffre dominant.
3. Il y a 9 choix pour le chiffre dominant (1-9, pas 0).
4. Pour les n-m positions restantes, il y a 9^(n-m) choix (pas le chiffre dominant).
5. Mais on doit exclure les nombres commençant par 0, donc on ajuste.
6. Utiliser la programmation dynamique ou des formules combinatoires.

### Observations
- On peut calculer c(n, m) directement avec des coefficients binomiaux.
- Les factorielles peuvent être précalculées.
- On doit gérer l'arithmétique modulaire.

### Complexité
- **Temporelle** : O(N²) pour calculer tous les c(n, m).
- **Spatiale** : O(N) pour les factorielles.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

using namespace std;

long long mod = 1000000007;

vector<long long> factorial_mod(int n, long long mod) {
    vector<long long> fac(n + 1, 1);
    for (int i = 2; i <= n; i++) {
        fac[i] = (fac[i - 1] * i) % mod;
    }
    return fac;
}

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long mod_inverse(long long a, long long mod) {
    return mod_pow(a, mod - 2, mod);
}

long long D(int N, long long mod) {
    vector<long long> fac = factorial_mod(N, mod);
    
    auto c = [&](int n, int m) -> long long {
        long long binom = (fac[n] * mod_inverse(fac[n - m], mod) % mod) * mod_inverse(fac[m], mod) % mod;
        return mod_pow(9, n - m + 1, mod) * binom % mod;
    };
    
    long long total = 0;
    for (int n = 1; n <= N; n++) {
        for (int m = n / 2 + 1; m <= n; m++) {
            total = (total + c(n, m)) % mod;
        }
    }
    
    return total;
}

int main() {
    cout << D(2022, mod) << endl;
    return 0;
}
```
