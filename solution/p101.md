# Optimum Polynomial

## Description du Problème
Si on nous présente les k premiers termes d'une séquence, il est impossible de dire avec certitude la valeur du terme suivant, car il existe une infinité de fonctions polynomiales qui peuvent modéliser la séquence.

Comme exemple, considérons la séquence des nombres cubes. Celle-ci est définie par la fonction génératrice u_n = n³ : 1, 8, 27, 64, 125, 216, …

Supposons qu'on ne nous donne que les deux premiers termes de cette séquence. En travaillant sur le principe que "simple est meilleur", nous devrions supposer une relation linéaire et prédire le terme suivant comme étant 15 (différence commune 7). Même si on nous présentait les trois premiers termes, par le même principe de simplicité, une relation quadratique devrait être supposée.

Nous définirons OP(k,n) comme étant le n-ième terme du polynôme optimal générateur pour les k premiers termes d'une séquence. Il devrait être clair que OP(k,n) générera avec précision les termes de la séquence pour n≤k, et potentiellement le premier terme incorrect (FIT) sera OP(k,k+1) ; dans ce cas, nous l'appellerons un mauvais OP (BOP).

Comme base, si on ne nous donnait que le premier terme de la séquence, il serait le plus sensé d'assumer la constance ; c'est-à-dire, pour n≥2, OP(1,n)=u₁.

D'où nous obtenons les OP suivants pour la séquence cubique :
- OP(1,n)=1 : 1, 1, 1, 1, …
- OP(2,n)=7n−6 : 1, 8, 15, …
- OP(3,n)=6n²−11n+6 : 1, 8, 27, 58, …
- OP(4,n)=n³ : 1, 8, 27, 64, 125, …

Clairement, aucun BOP n'existe pour k≥4.

En considérant la somme des FITs générés par les BOPs (indiqués en rouge ci-dessus), nous obtenons 1+15+58=74.

Considérons la fonction génératrice polynomiale de degré dix suivante :
u_n = 1 − n + n² − n³ + n⁴ − n⁵ + n⁶ − n⁷ + n⁸ − n⁹ + n¹⁰.

**Objectif** : Trouver la somme des FITs pour les BOPs.

---

## Exemples
### Exemple 1
**Entrée** : Séquence cubique
**Sortie** : `74`
**Explication** : Les FITs sont 1, 15, 58, leur somme est 74.

---

## Approche de Résolution

### Stratégie
1. Pour chaque k de 1 à 10, construire le polynôme optimal OP(k,n) en utilisant l'interpolation de Lagrange.
2. Évaluer OP(k,k+1) pour trouver le premier terme incorrect (FIT).
3. Vérifier que ce terme diffère de la valeur réelle u(k+1).
4. Sommer tous les FITs trouvés.

### Observations
- L'interpolation de Lagrange permet de construire un polynôme de degré k-1 qui passe exactement par k points.
- Pour k > 10, OP(k,n) = u(n) exactement, donc aucun BOP n'existe.
- On utilise des fractions pour maintenir la précision exacte.

### Complexité
- **Temporelle** : O(k³) pour chaque k, où k≤10, donc O(10³) = O(1000).
- **Spatiale** : O(k) pour stocker les points d'interpolation.

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <gmpxx.h>  // Utiliser GMP pour les grands entiers et fractions

struct Fraction {
    mpz_class numerator;
    mpz_class denominator;
    
    Fraction(mpz_class num = 0, mpz_class den = 1) : numerator(num), denominator(den) {
        if (denominator < 0) {
            numerator = -numerator;
            denominator = -denominator;
        }
    }
    
    Fraction add(const Fraction& other) const {
        mpz_class num = numerator * other.denominator + other.numerator * denominator;
        mpz_class den = denominator * other.denominator;
        return Fraction(num, den);
    }
    
    Fraction multiply(const Fraction& other) const {
        return Fraction(numerator * other.numerator, denominator * other.denominator);
    }
    
    bool equals(const Fraction& other) const {
        return numerator * other.denominator == other.numerator * denominator;
    }
    
    std::string toString() const {
        if (denominator == 1) {
            return numerator.get_str();
        }
        return numerator.get_str() + "/" + denominator.get_str();
    }
};

const int DEGREE = 10;

mpz_class generatingFunction(int n) {
    mpz_class sum = 0;
    mpz_class negN = -n;
    for (int i = 0; i <= DEGREE; i++) {
        sum += mpz_class(negN).pow(i);
    }
    return sum;
}

Fraction optimumPolynomial(int k, int n) {
    // Interpolation de Lagrange
    Fraction sum(0, 1);
    for (int i = 1; i <= k; i++) {
        Fraction product(generatingFunction(i), 1);
        for (int j = 1; j <= k; j++) {
            if (j != i) {
                Fraction term(n - j, i - j);
                product = product.multiply(term);
            }
        }
        sum = sum.add(product);
    }
    return sum;
}

int main() {
    Fraction sum(0, 1);
    for (int k = 1; k <= DEGREE; k++) {
        for (int n = k + 1; ; n++) {
            if (n == DEGREE + 2) {
                throw std::runtime_error("No FIT found");
            }
            Fraction reference(generatingFunction(n), 1);
            Fraction term = optimumPolynomial(k, n);
            if (!term.equals(reference)) {
                sum = sum.add(term);
                break;
            }
        }
    }
    std::cout << sum.toString() << std::endl;
    return 0;
}
```
