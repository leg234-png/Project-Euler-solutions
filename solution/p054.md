# Mains de Poker

## Description du Problème
Dans le fichier poker.txt, il y a mille mains de poker aléatoires. Chaque ligne contient dix cartes (séparées par un espace) : les cinq premières cartes sont la main du Joueur 1 et les cinq dernières sont la main du Joueur 2. Vous pouvez supposer que toutes les mains sont valides (pas de caractères invalides ou de cartes répétées), et que la main de chaque joueur est dans un ordre quelconque.

**Objectif** : Combien de mains le Joueur 1 gagne-t-il ?

---

## Exemples
### Exemple 1
**Entrée** : Deux mains de poker
**Sortie** : Nombre de victoires du Joueur 1
**Explication** : On compare les mains selon les règles du poker standard.

---

## Approche de Résolution

### Stratégie
1. Lire toutes les mains du fichier.
2. Pour chaque main :
   - Parser les cartes et les diviser entre les deux joueurs.
   - Calculer un score pour chaque main (en utilisant un encodage qui permet la comparaison).
   - Comparer les scores et compter les victoires du Joueur 1.
3. Retourner le nombre de victoires.

### Observations
- On encode le type de main dans les bits de poids fort (straight flush = 8, four of a kind = 7, etc.).
- On encode ensuite les 5 cartes les plus fréquentes dans les bits de poids faible.
- Les cartes sont représentées par leur rang (2-9, T, J, Q, K, A) et leur couleur (S, H, C, D).

### Complexité
- **Temporelle** : O(n), où n est le nombre de mains (1000).
- **Spatiale** : O(1), pour chaque main.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
#include <fstream>
#include <sstream>

struct Card {
    int rank;
    int suit;
    
    Card(const std::string& str) {
        const std::string ranks = "23456789TJQKA";
        const std::string suits = "SHCD";
        rank = ranks.find(str[0]);
        suit = suits.find(str[1]);
    }
};

int getScore(const std::vector<Card>& hand) {
    int rankCounts[13] = {0};
    int flushSuit = hand[0].suit;
    
    for (const Card& card : hand) {
        rankCounts[card.rank]++;
        if (card.suit != flushSuit) {
            flushSuit = -1;
        }
    }
    
    int rankCountHist[6] = {0};
    for (int count : rankCounts) {
        rankCountHist[count]++;
    }
    
    // Get best cards (most frequent first, then highest)
    int bestCards = 0;
    for (int i = 5; i >= 0; --i) {
        for (int j = 12; j >= 0; --j) {
            if (rankCounts[j] == i) {
                for (int k = 0; k < i && bestCards < 5; ++k) {
                    bestCards = (bestCards << 4) | j;
                }
            }
        }
    }
    
    // Check for straight
    int straightHigh = -1;
    for (int i = 12; i >= 3; --i) {
        bool isStraight = true;
        for (int j = 0; j < 5; ++j) {
            if (rankCounts[(i - j + 13) % 13] == 0) {
                isStraight = false;
                break;
            }
        }
        if (isStraight) {
            straightHigh = i;
            break;
        }
    }
    
    // Determine hand type
    if (straightHigh != -1 && flushSuit != -1) {
        return (8 << 20) | straightHigh;  // Straight flush
    } else if (rankCountHist[4] == 1) {
        return (7 << 20) | bestCards;  // Four of a kind
    } else if (rankCountHist[3] == 1 && rankCountHist[2] == 1) {
        return (6 << 20) | bestCards;  // Full house
    } else if (flushSuit != -1) {
        return (5 << 20) | bestCards;  // Flush
    } else if (straightHigh != -1) {
        return (4 << 20) | straightHigh;  // Straight
    } else if (rankCountHist[3] == 1) {
        return (3 << 20) | bestCards;  // Three of a kind
    } else if (rankCountHist[2] == 2) {
        return (2 << 20) | bestCards;  // Two pairs
    } else if (rankCountHist[2] == 1) {
        return (1 << 20) | bestCards;  // One pair
    } else {
        return (0 << 20) | bestCards;  // High card
    }
}

int main() {
    std::ifstream file("poker.txt");
    std::string line;
    int count = 0;
    
    while (std::getline(file, line)) {
        std::istringstream iss(line);
        std::vector<Card> player1, player2;
        
        for (int i = 0; i < 5; ++i) {
            std::string card;
            iss >> card;
            player1.push_back(Card(card));
        }
        
        for (int i = 0; i < 5; ++i) {
            std::string card;
            iss >> card;
            player2.push_back(Card(card));
        }
        
        if (getScore(player1) > getScore(player2)) {
            count++;
        }
    }
    
    std::cout << count << std::endl;
    return 0;
}
```
