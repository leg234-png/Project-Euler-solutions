# 2011 neufs

## Description du Problème
En fait, on peut prouver que la partie fractionnaire de (√2+√3)^(2n) approche 1 pour de grands n.

Considérons tous les nombres réels de la forme √p+√q avec p et q entiers positifs et p<q, tels que la partie fractionnaire de (√p+√q)^(2n) approche 1 pour de grands n.

**Objectif** : Soit N(p,q) la valeur minimale de n telle que C(p,q,n) ≥ 2011. Trouver ∑N(p,q) pour p+q ≤ 2011.

---

## Exemples
### Exemple 1
**Entrée** : `p=2, q=3`
**Sortie** : `N(2,3)`
**Explication** : La valeur minimale de n pour laquelle C(2,3,n) ≥ 2011.

---

## Approche de Résolution

### Stratégie
1. Pour que la partie fractionnaire de (√p+√q)^(2n) approche 1, on a besoin que 0 < p + q - 2√(pq) < 1.
2. Calculer N(p,q) = ⌈-2011/log₁₀(p + q - 2√(pq))⌉ si 0 < p + q - 2√(pq) < 1.
3. Itérer sur tous les p, q avec p < q et p + q ≤ 2011.
4. Vérifier la condition et sommer les valeurs de N(p,q).

### Observations
- La condition p + q - 2√(pq) < 1 est équivalente à (√p - √q)² < 1.
- On peut simplifier en vérifiant que p + q - 2√(pq) est dans (0, 1).
- La formule utilise le logarithme en base 10.

### Complexité
- **Temporelle** : O(limit²) pour itérer sur tous les p, q.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long compute(int limit) {
    long long total = 0;
    
    for (int p = 1; p < limit; p++) {
        for (int q = p + 1; q <= limit - p; q++) {
            double t = p + q - 2 * sqrt(p * q);
            if (t > 0 && t < 1) {
                total += (long long)ceil(-limit / log10(t));
            }
        }
    }
    
    return total;
}

int main() {
    cout << compute(2011) << endl;
    return 0;
}
```
