# Dérivation du code d'accès

## Description du Problème
Une méthode de sécurité courante pour les comptes bancaires en ligne est de demander à l'utilisateur trois caractères d'un code d'accès. Par exemple, si le code d'accès était 531278, ils pourraient demander le 2ème, 3ème et 5ème caractères ; la réponse attendue serait : 317.

Le fichier texte, keylog.txt, contient cinquante tentatives de connexion réussies qui fournissent des informations sur des caractères du code d'accès dans l'ordre, mais pas nécessairement de manière consécutive.

Étant donné que chaque fichier contient trois caractères dans l'ordre, analyser le fichier pour déterminer le code d'accès le plus court possible d'une longueur inconnue.

**Objectif** : Déterminer le code d'accès le plus court possible.

---

## Exemples
### Exemple 1
**Entrée** : Sous-séquences : "319", "680", "180"
**Sortie** : `73162890` (exemple)
**Explication** : Le code d'accès doit contenir ces sous-séquences dans l'ordre.

---

## Approche de Résolution

### Stratégie
1. Essayer toutes les combinaisons possibles de codes d'accès par longueur croissante.
2. Pour chaque candidat, vérifier si toutes les sous-séquences données sont présentes dans l'ordre.
3. Retourner le premier code d'accès valide trouvé.

### Observations
- On peut optimiser en utilisant un graphe de dépendances entre les chiffres.
- Cependant, une approche de force brute fonctionne pour ce problème car le code d'accès est court.
- On teste les longueurs de 3 à 10 (ou plus si nécessaire).

### Complexité
- **Temporelle** : O(10^L × S), où L est la longueur du code et S est le nombre de sous-séquences.
- **Spatiale** : O(L), pour stocker le code candidat.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

using namespace std;

vector<string> SUBSEQS = {
    "319", "680", "180", "690", "129", "620", "762", "689", "762", "318",
    "368", "710", "720", "710", "629", "168", "160", "689", "716", "731",
    "736", "729", "316", "729", "729", "710", "769", "290", "719", "680",
    "318", "389", "162", "289", "162", "718", "729", "319", "790", "680",
    "890", "362", "319", "760", "316", "729", "380", "319", "728", "716"
};

bool isConsistent(const string& guess) {
    for (const string& subseq : SUBSEQS) {
        int j = 0;  // Index dans subseq
        for (char c : guess) {
            if (j < (int)subseq.length() && c == subseq[j])
                j++;
        }
        if (j < (int)subseq.length())
            return false;
    }
    return true;
}

string toChars(int n, int len) {
    string result(len, '0');
    int i = 0;
    while (n > 0 && i < len) {
        result[i++] = '0' + (n % 10);
        n /= 10;
    }
    if (n != 0)
        return "";
    return result;
}

int main() {
    for (int len = 3; len <= 10; len++) {
        int end = 1;
        for (int i = 0; i < len; i++)
            end *= 10;
        
        for (int guess = 0; guess < end; guess++) {
            string guessStr = toChars(guess, len);
            if (guessStr.empty()) continue;
            if (isConsistent(guessStr)) {
                cout << guessStr << endl;
                return 0;
            }
        }
    }
    cerr << "Not found" << endl;
    return 1;
}
```
