# Permutation III

## Description du Problème
Soit B(n) le plus petit nombre plus grand que n qui peut être formé en réarrangeant les chiffres de n, ou 0 si un tel nombre n'existe pas.

Définissons T(N) = ∑_{n=1}^N B(n²). On vous donne T(10) = 270 et T(100) = 335316.

**Objectif** : Trouver T(10¹⁶). Donner votre réponse modulo 10⁹+7.

---

## Exemples
### Exemple 1
**Entrée** : `N = 10`
**Sortie** : `270`
**Explication** : T(10) = 270.

---

## Approche de Résolution

### Stratégie
1. Pour chaque n de 1 à N, calculer n².
2. Pour chaque n², trouver B(n²) : le plus petit nombre > n² formé en réarrangeant les chiffres de n².
3. Si tous les chiffres sont en ordre décroissant, B(n²) = 0.
4. Sinon, trouver la prochaine permutation lexicographique.
5. Sommer tous les B(n²).

### Observations
- On peut utiliser l'algorithme de prochaine permutation.
- Pour de très grandes valeurs de N, on doit optimiser considérablement.
- On peut utiliser des techniques de comptage pour éviter de traiter chaque nombre individuellement.

### Complexité
- **Temporelle** : O(N × d) où d est le nombre de chiffres de n².
- **Spatiale** : O(d) pour stocker les chiffres.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>

using namespace std;

long long mod = 1000000007;

long long B(long long n) {
    string s = to_string(n);
    
    // Vérifier si on peut former un nombre plus grand
    string sorted = s;
    sort(sorted.rbegin(), sorted.rend());
    if (s == sorted) {
        return 0; // Pas de permutation plus grande
    }
    
    // Trouver la prochaine permutation
    string next = s;
    next_permutation(next.begin(), next.end());
    
    return stoll(next);
}

long long compute(long long N, long long mod) {
    long long total = 0;
    
    for (long long n = 1; n <= N; n++) {
        long long n_sq = n * n;
        long long b_val = B(n_sq);
        total = (total + b_val) % mod;
    }
    
    return total;
}

int main() {
    long long N = 1;
    for (int i = 0; i < 16; i++) {
        N *= 10;
    }
    cout << compute(N, mod) << endl;
    return 0;
}
```
