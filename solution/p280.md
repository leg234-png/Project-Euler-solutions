# Ant and seeds

## Description du Problème
Une fourmi se déplace sur une grille 5×5. Il y a 5 graines dans la rangée du bas et la fourmi doit les transporter vers la rangée du haut.

À chaque étape, la fourmi se déplace aléatoirement vers une case adjacente (haut, bas, gauche, droite). Si la fourmi est dans la rangée du bas et qu'une case contient une graine, elle la ramasse. Si la fourmi est dans la rangée du haut et qu'elle porte une graine, elle la dépose.

**Objectif** : Trouver le nombre moyen d'étapes nécessaires pour que toutes les graines soient transportées vers la rangée du haut.

---

## Exemples
### Exemple 1
**Entrée** : Grille 5×5 avec 5 graines en bas
**Sortie** : Nombre moyen d'étapes
**Explication** : On simule le processus stochastique.

---

## Approche de Résolution

### Stratégie
1. Modéliser le problème comme un processus de Markov.
2. Un état est défini par :
   - Position de la fourmi (x, y).
   - Quelles cases/la fourmi contiennent des graines (11 bits : 5 pour le haut, 5 pour le bas, 1 pour la fourmi).
3. Utiliser la programmation dynamique :
   - `probs[state]` : probabilité d'être dans cet état.
   - Pour chaque état, calculer les états successeurs possibles.
   - Mettre à jour les probabilités à chaque étape.
4. Calculer l'espérance : `sum += probabilité_terminaison × numéro_étape`.

### Observations
- Il y a au maximum 5×5×2^11 = 102400 états possibles (plus l'état terminal).
- Le processus est markovien : l'état suivant ne dépend que de l'état actuel.
- On peut arrêter quand la probabilité de terminaison devient négligeable.

### Complexité
- **Temporelle : O(steps × S × 4)`, où S est le nombre d'états.
- **Spatiale : O(S)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <iomanip>
#include <vector>
#include <set>
#include <map>

using namespace std;

struct State {
    bool isDone;
    int antX, antY;
    vector<bool> hasSeed;  // 11 elements
    
    bool operator<(const State& other) const {
        if (isDone != other.isDone) return isDone < other.isDone;
        if (antX != other.antX) return antX < other.antX;
        if (antY != other.antY) return antY < other.antY;
        return hasSeed < other.hasSeed;
    }
    
    set<State> getSuccessors() const {
        set<State> result;
        if (isDone) return result;
        
        int dx[] = {-1, 1, 0, 0};
        int dy[] = {0, 0, -1, 1};
        
        for (int i = 0; i < 4; i++) {
            int x = antX + dx[i];
            int y = antY + dy[i];
            if (x < 0 || x >= 5 || y < 0 || y >= 5)
                continue;
            
            vector<bool> seed = hasSeed;
            bool done = false;
            
            if (!seed[10] && y == 4 && seed[5 + x]) {
                seed[5 + x] = false;
                seed[10] = true;
            } else if (seed[10] && y == 0 && !seed[x]) {
                seed[10] = false;
                seed[x] = true;
                done = seed[0] && seed[1] && seed[2] && seed[3] && seed[4];
            }
            
            result.insert({done, x, y, seed});
        }
        return result;
    }
};

int main() {
    // Build state space and transition matrix
    map<State, int> stateToId;
    vector<State> idToState;
    vector<vector<int>> successors;
    
    // ... (implementation of state enumeration and successor calculation)
    
    // Run Markov chain simulation
    double sum = 0;
    vector<double> probs(idToState.size(), 0);
    probs[0] = 1.0;  // Start state
    
    for (int i = 1; ; i++) {
        vector<double> nextProbs(idToState.size(), 0);
        for (size_t j = 0; j < probs.size(); j++) {
            if (probs[j] > 0) {
                for (int k : successors[j]) {
                    nextProbs[k] += probs[j] / successors[j].size();
                }
            }
        }
        
        double doneNow = nextProbs[stateToId[State{true, 0, 0, {}}]];
        if (i > 44 && doneNow < 1e-20)
            break;
        sum += doneNow * i;
        probs = nextProbs;
    }
    
    cout << fixed << setprecision(6) << sum << endl;
    return 0;
}
```
