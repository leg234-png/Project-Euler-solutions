# Pandigital Prime Sets

## Description du Problème
En utilisant tous les chiffres de 1 à 9 et en les concaténant librement pour former des entiers décimaux, différents ensembles peuvent être formés. Intéressant, avec l'ensemble {2,5,47,89,631}, tous les éléments qui lui appartiennent sont premiers.

**Objectif** : Combien d'ensembles distincts contenant chacun des chiffres un à neuf exactement une fois contiennent uniquement des éléments premiers ?

---

## Exemples
### Exemple 1
**Entrée** : `{2,5,47,89,631}`
**Sortie** : `true`
**Explication** : Tous les éléments sont premiers et utilisent les chiffres 1-9 exactement une fois.

---

## Approche de Résolution

### Stratégie
1. Générer toutes les permutations des chiffres 1-9.
2. Pour chaque permutation, essayer toutes les façons de diviser en nombres premiers.
3. Utiliser un backtracking récursif pour trouver toutes les divisions possibles.
4. Compter les ensembles uniques qui satisfont les conditions.

### Observations
- On doit diviser la permutation en segments qui sont tous premiers.
- Les nombres doivent être en ordre croissant pour éviter les doublons.
- On utilise un crible de primalité pour vérifier efficacement.

### Complexité
- **Temporelle** : O(9! × nombre_de_divisions_possibles).
- **Spatiale** : O(9!) pour stocker les permutations.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

std::vector<bool> isPrime;

bool isPrimeNumber(int n) {
    if (n < isPrime.size()) return isPrime[n];
    // Vérification de primalité pour les grands nombres
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}

int toInteger(const std::vector<int>& digits, int start, int end) {
    int result = 0;
    for (int i = start; i < end; i++) {
        result = result * 10 + digits[i];
    }
    return result;
}

int count = 0;

void countPrimeSets(std::vector<int>& digits, int startIndex, int prevNum) {
    if (startIndex == digits.size()) {
        count++;
        return;
    }
    
    for (int split = startIndex + 1; split <= digits.size(); split++) {
        int num = toInteger(digits, startIndex, split);
        if (num > prevNum && isPrimeNumber(num)) {
            countPrimeSets(digits, split, num);
        }
    }
}

int main() {
    // Générer le crible de primalité
    isPrime.resize(10000, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i * i < isPrime.size(); i++) {
        if (isPrime[i]) {
            for (int j = i * i; j < isPrime.size(); j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    std::vector<int> digits = {1, 2, 3, 4, 5, 6, 7, 8, 9};
    do {
        countPrimeSets(digits, 0, 0);
    } while (std::next_permutation(digits.begin(), digits.end()));
    
    std::cout << count << std::endl;
    return 0;
}
```
