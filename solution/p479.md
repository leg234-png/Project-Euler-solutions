# Racines en hausse

## Description du Problème
Soit aₖ, bₖ, cₖ les 3 racines (réelles ou complexes) de x³ - kx² + x/k - k² = 0.

Par exemple, pour k=5, on voit que {a₅,b₅,c₅} est approximativement {5.727244,−0.363622+2.057397i,−0.363622−2.057397i}.

**Objectif** : Soit S(n) = ∑ₚ₌₁ⁿ ∑ₖ₌₁ⁿ (aₖ+bₖ)^p(bₖ+cₖ)^p(cₖ+aₖ)^p. Intéressamment, S(n) est toujours un entier. Par exemple, S(4) = 51160. Trouver S(10⁶) modulo 1000000007.

---

## Exemples
### Exemple 1
**Entrée** : `n = 4`
**Sortie** : `51160`
**Explication** : S(4) = 51160.

---

## Approche de Résolution

### Stratégie
1. Utiliser les relations de Viète pour les racines d'un polynôme cubique.
2. Pour x³ - kx² + x/k - k² = 0, on a :
   - a + b + c = k
   - ab + ac + bc = 1/k
   - abc = k²
3. Calculer (a+b)^p(b+c)^p(c+a)^p = (k - c)^p(k - a)^p(k - b)^p.
4. Utiliser les identités : (a+b)(b+c)(c+a) = (a+b+c)(ab+ac+bc) - abc = k × 1/k - k² = 1 - k².
5. Donc S(n) = ∑ₚ₌₁ⁿ ∑ₖ₌₁ⁿ (1 - k²)^p.
6. Utiliser la formule de la série géométrique : ∑ₚ₌₁ⁿ (1-k²)^p = ((1-k²) - (1-k²)^(n+1))/k².

### Observations
- Les relations de Viète simplifient considérablement le problème.
- On peut calculer directement sans trouver les racines explicitement.
- La division modulaire est nécessaire pour les grands nombres.

### Complexité
- **Temporelle** : O(n × log n) pour calculer les puissances modulaires.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long mod_inverse(long long a, long long mod) {
    return mod_pow(a, mod - 2, mod);
}

long long geometric_sum(long long r, long long n, long long mod) {
    if (r == 0) return 0;
    if (r == 1) return n % mod;
    
    long long numerator = (mod_pow(r, n + 1, mod) - r + mod) % mod;
    long long denominator = (r - 1) % mod;
    long long inv = mod_inverse(denominator, mod);
    return (numerator * inv) % mod;
}

long long compute(long long n, long long mod) {
    long long total = 0;
    
    for (long long k = 1; k <= n; k++) {
        long long r = (1 - k * k + mod) % mod;
        long long sum_p = geometric_sum(r, n, mod);
        total = (total + sum_p) % mod;
    }
    
    return (total + mod) % mod;
}

int main() {
    long long mod = 1000000007;
    cout << compute(1000000, mod) << endl;
    return 0;
}
```
