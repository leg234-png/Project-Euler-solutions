# Plus Longue Chaîne de Collatz

## Description du Problème
La suite suivante est définie pour l'ensemble des entiers positifs :

n → n/2 (si n est pair)
n → 3n + 1 (si n est impair)

En utilisant la règle ci-dessus et en commençant par 13, on génère la suite suivante :

13 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1

On peut voir que cette suite (en commençant à 13 et en finissant à 1) contient 10 termes. Bien qu'elle n'ait pas encore été prouvée (problème de Collatz), on pense que tous les nombres de départ finissent à 1.

**Objectif** : Quel nombre de départ, inférieur à un million, produit la plus longue chaîne ?

---

## Exemples
### Exemple 1
**Entrée** : Limite = 13
**Sortie** : `9`
**Explication** : Le nombre 9 produit une chaîne de 20 termes, la plus longue pour les nombres inférieurs à 13.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre de 1 à 1 000 000, calculer la longueur de la chaîne de Collatz.
2. Garder trace du nombre qui produit la chaîne la plus longue.
3. Utiliser la mémorisation (cache) pour éviter de recalculer les chaînes déjà calculées.

### Observations
- La règle de Collatz :
  - Si n est pair : n → n/2
  - Si n est impair : n → 3n + 1
- On peut utiliser la mémorisation pour optimiser : si on a déjà calculé la longueur pour un nombre, on la réutilise.
- Les nombres peuvent devenir très grands (dépasser 1 000 000) pendant le calcul, donc utiliser `long long`.
- On peut arrêter quand on atteint 1.

### Complexité
- **Temporelle** : O(n × log n), où `n` = 1 000 000, avec mémorisation.
- **Spatiale** : O(n), pour le cache de mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <unordered_map>

long long collatzChainLength(long long n, std::unordered_map<long long, int>& cache) {
    if (n == 1) {
        return 1;
    }
    
    if (cache.find(n) != cache.end()) {
        return cache[n];
    }
    
    long long next;
    if (n % 2 == 0) {
        next = n / 2;
    } else {
        next = 3 * n + 1;
    }
    
    int length = 1 + collatzChainLength(next, cache);
    cache[n] = length;
    return length;
}

int main() {
    const int limit = 1000000;
    std::unordered_map<long long, int> cache;
    
    int maxArg = -1;
    int maxChain = 0;
    
    for (int i = 1; i < limit; ++i) {
        int chainLen = collatzChainLength(i, cache);
        if (chainLen > maxChain) {
            maxArg = i;
            maxChain = chainLen;
        }
    }
    
    std::cout << maxArg << std::endl;
    return 0;
}
```
