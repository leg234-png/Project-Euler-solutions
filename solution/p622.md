# Mélanges par faro

## Description du Problème
Soit s(n) le nombre minimum de mélanges par faro consécutifs nécessaires pour restaurer un jeu de taille n à sa configuration originale, où n est un entier positif pair.

Étonnamment, un jeu standard de 52 cartes retournera à sa configuration originale après seulement 8 mélanges parfaits, donc s(52) = 8. On peut vérifier qu'un jeu de 86 cartes retournera aussi à sa configuration originale après exactement 8 mélanges, et la somme de toutes les valeurs de n qui satisfont s(n) = 8 est 412.

**Objectif** : Trouver la somme de toutes les valeurs de n qui satisfont s(n) = 60.

---

## Exemples
### Exemple 1
**Entrée** : `s(n) = 8`
**Sortie** : `412`
**Explication** : La somme de tous les n tels que s(n) = 8 est 412.

---

## Approche de Résolution

### Stratégie
1. En général, k mélanges parfaits restaureront l'ordre d'un jeu de n cartes si 2ᵏ = 1 (mod n-1).
2. Par exemple, 8 mélanges consécutifs restaureront l'ordre si 2⁸ = 1 (mod n-1).
3. Donc la somme de tous les n tels que s(n) = 8 est tous les n tels que 2⁸ = 1 (mod n-1) et aucun diviseur de n-1 ne le fait aussi.
4. 2⁸ - 1 = (n-1)l => 255 = (n-1)l.
5. Trouver les diviseurs de 255, ce sont nos nombres candidats.
6. Pour chaque candidat x, vérifier si 2⁸ (mod x) = 1, et si c'est bien le plus petit.

### Observations
- Le problème se réduit à trouver les diviseurs de 2ᵏ - 1.
- On doit vérifier que c'est bien l'ordre minimal.
- On peut utiliser l'exponentiation modulaire rapide.

### Complexité
- **Temporelle** : O(D × log k) où D est le nombre de diviseurs de 2ᵏ - 1.
- **Spatiale** : O(D) pour stocker les diviseurs.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <cmath>

using namespace std;

vector<long long> divisors(long long x) {
    set<long long> divs;
    divs.insert(x);
    for (long long i = 2; i * i <= x; i++) {
        if (x % i == 0) {
            divs.insert(i);
            divs.insert(x / i);
        }
    }
    return vector<long long>(divs.begin(), divs.end());
}

long long mod_pow(long long base, long long exp, long long mod) {
    long long result = 1;
    base = base % mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    return result;
}

long long compute(int k) {
    long long two_power_k_minus_1 = (1LL << k) - 1;
    vector<long long> divs = divisors(two_power_k_minus_1);
    vector<long long> divs_k = divisors(k);
    
    long long total = 0;
    for (long long x : divs) {
        if (mod_pow(2, k, x) == 1) {
            bool is_minimal = true;
            for (long long d : divs_k) {
                if (d < k && mod_pow(2, d, x) == 1) {
                    is_minimal = false;
                    break;
                }
            }
            if (is_minimal) {
                total += x + 1; // n = x + 1
            }
        }
    }
    return total;
}

int main() {
    cout << compute(60) << endl;
    return 0;
}
```
