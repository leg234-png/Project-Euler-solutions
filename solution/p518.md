# Triplets premiers et séquences géométriques

## Description du Problème
Soit S(n) = Σ a+b+c sur tous les triplets (a,b,c) tels que :
- a, b, et c sont des nombres premiers.
- a < b < c < n.
- a+1, b+1, et c+1 forment une séquence géométrique.

Par exemple, S(100) = 1035 avec les triplets suivants :
(2,5,11), (2,11,47), (5,11,23), (5,17,53), (7,11,17), (7,23,71), (11,23,47), (17,23,31), (17,41,97), (31,47,71), (71,83,97)

**Objectif** : Trouver S(10⁸).

---

## Exemples
### Exemple 1
**Entrée** : `n = 100`
**Sortie** : `1035`
**Explication** : S(100) = 1035.

---

## Approche de Résolution

### Stratégie
1. On cherche des nombres premiers a < b < c tels que (a+1), (b+1), (c+1) forment une séquence géométrique.
2. (c+1)/(b+1) = (b+1)/(a+1) = r, où r > 1.
3. Soit r = z/y, où z > y et gcd(z,y) = 1.
4. Soit x = (a+1)/y², alors (a+1) = x × y², (b+1) = z × x × y, (c+1) = z² × x.
5. On peut déterminer tous les triplets (a+1, b+1, c+1) en utilisant seulement z, y tels que z > y et gcd(z,y) = 1.
6. Pour chaque triplet, vérifier si a, b, c sont premiers.

### Observations
- On peut paramétrer les solutions en utilisant des triplets (x, y, z).
- On doit générer tous les (z, y) copremiers avec z > y.
- On peut optimiser en limitant la plage de recherche.

### Complexité
- **Temporelle** : O(√n × log n) pour générer les triplets et vérifier les nombres premiers.
- **Spatiale** : O(√n) pour le crible de nombres premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

bool is_prime(long long x) {
    if (x <= 1) return false;
    if (x <= 3) return true;
    if (x % 2 == 0) return false;
    for (long long i = 3; i * i <= x; i += 2) {
        if (x % i == 0) return false;
    }
    return true;
}

long long compute(long long limit) {
    long long total = 0;
    
    for (long long z = 2; z * z <= limit; z++) {
        for (long long y = 1; y < z; y++) {
            if (__gcd(z, y) != 1) continue;
            
            for (long long x = 1; x * z * z <= limit; x++) {
                long long a_plus_1 = x * y * y;
                long long b_plus_1 = x * y * z;
                long long c_plus_1 = x * z * z;
                
                if (a_plus_1 > 1 && b_plus_1 > a_plus_1 && c_plus_1 > b_plus_1) {
                    long long a = a_plus_1 - 1;
                    long long b = b_plus_1 - 1;
                    long long c = c_plus_1 - 1;
                    
                    if (a < limit && b < limit && c < limit &&
                        is_prime(a) && is_prime(b) && is_prime(c)) {
                        total += a + b + c;
                    }
                }
            }
        }
    }
    
    return total;
}

int main() {
    cout << compute(100000000) << endl;
    return 0;
}
```
