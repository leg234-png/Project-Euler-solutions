# Triplets premiers en progression géométrique

## Description du Problème
Soit S(n) la somme de a + b + c sur tous les triplets de nombres premiers (a, b, c) avec a < b < c < n tels que a+1, b+1, c+1 forment une progression géométrique.

Par exemple, S(100) = 1035 avec les triplets (2, 5, 11), (2, 11, 47), (5, 11, 23), (5, 17, 53), (7, 23, 71), (11, 23, 47), (17, 41, 97), (31, 47, 71), (71, 83, 97).

**Objectif** : Trouver S(10⁸).

---

## Exemples
### Exemple 1
**Entrée** : `n = 100`
**Sortie** : `1035`
**Explication** : La somme de a + b + c pour tous les triplets valides.

---

## Approche de Résolution

### Stratégie
1. Si (a+1, b+1, c+1) forment une progression géométrique, alors (c+1)/(b+1) = (b+1)/(a+1) = r pour un ratio r.
2. Exprimer r = z/y en termes les plus bas (y et z sont copremiers).
3. Définir x = (a+1) / y². Alors :
   - a + 1 = x × y × y
   - b + 1 = x × y × z
   - c + 1 = x × z × z
4. Rechercher tous les triplets (x, y, z) tels que :
   - x >= 1, y >= 1, z > y
   - gcd(y, z) = 1
   - a = x × y² - 1, b = x × y × z - 1, c = x × z² - 1 sont tous premiers
5. Vérifier que a, b, c < n et a < b < c.

### Observations
- On peut rechercher x, y, z dans un ordre croissant.
- On s'arrête chaque boucle quand aucun candidat n'est possible car ils dépassent tous la limite.
- On utilise un crible pour vérifier rapidement si un nombre est premier.

### Complexité
- **Temporelle** : O(N × log N), où N = 10⁸, pour le crible et la recherche.
- **Spatiale** : O(N), pour le tableau de primalité.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

const int LIMIT = 100000000;

vector<bool> listPrimality(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    return isPrime;
}

int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

int main() {
    vector<bool> isPrime = listPrimality(LIMIT - 1);
    long long sum = 0;
    
    // Rechercher tous les x possibles
    for (int x = 1; x < isPrime.size(); x++) {
        
        // Rechercher tous les y possibles
        for (int y = 1; ; y++) {
            long long a = (long long)x * y * y - 1;
            if (a >= isPrime.size()) {
                break;
            }
            if (!isPrime[a]) {
                continue;
            }
            
            // Rechercher tous les z valides
            for (int z = y + 1; ; z++) {
                if (gcd(y, z) != 1) {
                    continue;
                }
                long long b = (long long)x * y * z - 1;
                long long c = (long long)x * z * z - 1;
                if (c >= isPrime.size()) {
                    break;
                }
                
                // Vérifier si (a, b, c) est une solution
                if (isPrime[b] && isPrime[c]) {
                    sum += a + b + c;
                }
            }
        }
    }
    
    cout << sum << endl;
    return 0;
}
```
