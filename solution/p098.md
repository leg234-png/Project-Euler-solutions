# Anagramic Squares

## Description du Problème
En remplaçant chacune des lettres du mot CARE par 1, 2, 9 et 6 respectivement, nous formons un nombre carré : 1296 = 36². Ce qui est remarquable, c'est qu'en utilisant les mêmes substitutions numériques, l'anagramme RACE forme également un nombre carré : 9216 = 96². Nous appellerons CARE (et RACE) une paire de mots anagrammes carrés et spécifierons en outre que les zéros de tête ne sont pas autorisés, et qu'une lettre différente ne peut pas avoir la même valeur numérique qu'une autre lettre.

En utilisant words.txt, un fichier texte de 16K contenant près de deux mille mots anglais communs, trouver toutes les paires de mots anagrammes carrés (un mot palindromique n'est PAS considéré comme un anagramme de lui-même).

**Objectif** : Quel est le plus grand nombre carré formé par n'importe quel membre d'une telle paire ?

---

## Exemples
### Exemple 1
**Entrée** : `CARE` et `RACE`
**Sortie** : `9216`
**Explication** : CARE → 1296 = 36², RACE → 9216 = 96². Le plus grand est 9216.

---

## Approche de Résolution

### Stratégie
1. Lire tous les mots depuis le fichier.
2. Grouper les mots par leurs anagrammes (même ensemble de lettres triées).
3. Pour chaque paire d'anagrammes, essayer toutes les assignations de chiffres aux lettres.
4. Vérifier si les deux mots forment des nombres carrés avec la même assignation.
5. Retourner le plus grand nombre carré trouvé.

### Observations
- On utilise un backtracking pour assigner des chiffres aux lettres.
- On doit s'assurer que le premier chiffre de chaque mot n'est pas zéro.
- On vérifie que les deux nombres formés sont des carrés parfaits.
- On utilise une fonction pour canoniser les mots (trier les lettres).

### Complexité
- **Temporelle** : O(n² × m × 10^m), où n est le nombre de mots, m est la longueur moyenne des mots.
- **Spatiale** : O(n) pour stocker les mots et leurs anagrammes.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <map>
#include <algorithm>
#include <cmath>
#include <set>

std::string canonicalize(const std::string& s) {
    std::string result = s;
    std::sort(result.begin(), result.end());
    return result;
}

bool isSquare(long long n) {
    long long root = (long long)std::sqrt(n);
    return root * root == n;
}

int maxSquarePair(const std::string& a, const std::string& b, 
                  int index, std::map<char, int>& assignments, 
                  std::vector<bool>& isDigitUsed) {
    if (index == a.length()) {
        if (assignments[a[0]] == 0 || assignments[b[0]] == 0) {
            return 0;
        }
        
        long long aNum = 0, bNum = 0;
        for (char c : a) {
            aNum = aNum * 10 + assignments[c];
        }
        for (char c : b) {
            bNum = bNum * 10 + assignments[c];
        }
        
        if (isSquare(aNum) && isSquare(bNum)) {
            return std::max(aNum, bNum);
        }
        return 0;
    }
    
    if (assignments.find(a[index]) != assignments.end()) {
        return maxSquarePair(a, b, index + 1, assignments, isDigitUsed);
    }
    
    int max = 0;
    for (int i = 0; i < 10; i++) {
        if (!isDigitUsed[i]) {
            isDigitUsed[i] = true;
            assignments[a[index]] = i;
            max = std::max(maxSquarePair(a, b, index + 1, assignments, isDigitUsed), max);
            isDigitUsed[i] = false;
            assignments.erase(a[index]);
        }
    }
    return max;
}

int main() {
    // Lire les mots depuis le fichier
    std::vector<std::string> words;
    // ... lire words.txt ...
    
    // Grouper par anagrammes
    std::map<std::string, std::vector<std::string>> anagrams;
    for (const std::string& word : words) {
        std::string key = canonicalize(word);
        anagrams[key].push_back(word);
    }
    
    int max = 0;
    for (const auto& pair : anagrams) {
        const std::vector<std::string>& words_list = pair.second;
        for (size_t i = 0; i < words_list.size(); i++) {
            for (size_t j = i + 1; j < words_list.size(); j++) {
                std::map<char, int> assignments;
                std::vector<bool> isDigitUsed(10, false);
                max = std::max(maxSquarePair(words_list[i], words_list[j], 0, 
                                            assignments, isDigitUsed), max);
            }
        }
    }
    
    std::cout << max << std::endl;
    return 0;
}
```
