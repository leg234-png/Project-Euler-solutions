# Diophantine Reciprocals I

## Description du Problème
Dans l'équation suivante, x, y et n sont des entiers positifs.

1/x + 1/y = 1/n

Pour n = 4, il y a exactement trois solutions distinctes :
- 1/5 + 1/20 = 1/4
- 1/6 + 1/12 = 1/4
- 1/8 + 1/8 = 1/4

**Objectif** : Quelle est la plus petite valeur de n pour laquelle le nombre de solutions distinctes dépasse mille ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 4`
**Sortie** : `3`
**Explication** : Il y a 3 solutions distinctes pour n = 4.

---

## Approche de Résolution

### Stratégie
1. Réécrire l'équation : 1/x + 1/y = 1/n implique que n² = (x-n)(y-n).
2. Le nombre de solutions est égal au nombre de diviseurs de n² dans la plage [1, n].
3. Si n² a m diviseurs, alors le nombre de solutions est (m+1)/2.
4. Parcourir les valeurs de n jusqu'à trouver celle qui a plus de 1000 solutions.

### Observations
- On peut calculer le nombre de diviseurs de n² en factorisant n.
- Si n = p₁^a₁ × p₂^a₂ × ... × pₖ^aₖ, alors n² a (2a₁+1)(2a₂+1)...(2aₖ+1) diviseurs.
- On cherche le plus petit n tel que (nombre de diviseurs de n² + 1) / 2 > 1000.

### Complexité
- **Temporelle** : O(n log n) pour tester chaque n jusqu'à trouver la solution.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

int countDivisorsSquared(int n) {
    int count = 1;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) {
            int j = 0;
            do {
                n /= i;
                j++;
            } while (n % i == 0);
            count *= j * 2 + 1;
        }
    }
    if (n != 1) {
        count *= 3;  // Facteur premier restant
    }
    return count;
}

int main() {
    for (int n = 1; ; n++) {
        if ((countDivisorsSquared(n) + 1) / 2 > 1000) {
            std::cout << n << std::endl;
            return 0;
        }
    }
    return 0;
}
```
