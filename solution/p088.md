# Nombres produit-somme

## Description du Problème
Un nombre naturel, N, qui peut être écrit comme la somme et le produit d'un ensemble donné d'au moins deux nombres naturels, {a₁, a₂, ..., aₖ} est appelé un nombre produit-somme : N = a₁ + a₂ + ... + aₖ = a₁ × a₂ × ... × aₖ.

Par exemple, 6 = 1 + 2 + 3 = 1 × 2 × 3.

Pour un ensemble de taille k, on appellera le plus petit N avec cette propriété un nombre produit-somme minimal. Les nombres produit-somme minimaux pour les ensembles de taille k = 2, 3, 4, 5, et 6 sont :
- k=2: 4 = 2×2 = 2+2
- k=3: 6 = 1×2×3 = 1+2+3
- k=4: 8 = 1×1×2×4 = 1+1+2+4
- k=5: 8 = 1×1×2×2×2 = 1+1+2+2+2
- k=6: 12 = 1×1×1×1×2×6 = 1+1+1+1+2+6

Par conséquent, pour 2≤k≤6, la somme de tous les nombres produit-somme minimaux est 4+6+8+12 = 30 ; notez que 8 n'est compté qu'une fois dans la somme.

En fait, comme l'ensemble complet des nombres produit-somme minimaux pour 2≤k≤12 est {4,6,8,12,15,16}, la somme est 61.

**Objectif** : Quelle est la somme de tous les nombres produit-somme minimaux pour 2≤k≤12000 ?

---

## Exemples
### Exemple 1
**Entrée** : k ≤ 12
**Sortie** : `61`
**Explication** : L'ensemble est {4,6,8,12,15,16}, la somme est 61.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre n de 2 à 2×LIMIT, factoriser n de toutes les façons possibles.
2. Pour chaque factorisation, calculer le nombre de termes k = (somme des facteurs) + (n - produit des facteurs).
3. Si k ≤ LIMIT et n < minSumProduct[k], mettre à jour minSumProduct[k] = n.
4. Éliminer les doublons et calculer la somme.

### Observations
- minSumProduct[k] > k car la somme minimale de k entiers positifs est k, mais le produit serait 1.
- minSumProduct[k] ≤ 2k car 1+...+1+2+k (k termes) = 2k et le produit est aussi 2k.
- On factorise chaque nombre et on calcule le nombre de termes nécessaires.

### Complexité
- **Temporelle** : O(N × F), où N = 2×LIMIT et F est le nombre moyen de factorisations.
- **Spatiale** : O(LIMIT), pour stocker minSumProduct.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <unordered_set>
#include <climits>
#include <algorithm>

using namespace std;

const int LIMIT = 12000;
vector<int> minSumProduct;

void factorize(int n, int remain, int maxFactor, int sum, int terms) {
    if (remain == 1) {
        terms += n - sum;
        if (terms <= LIMIT && n < minSumProduct[terms])
            minSumProduct[terms] = n;
    } else {
        for (int i = 2; i <= maxFactor && i <= remain; i++) {
            if (remain % i == 0) {
                factorize(n, remain / i, min(i, maxFactor), sum + i, terms + 1);
            }
        }
    }
}

int main() {
    minSumProduct.assign(LIMIT + 1, INT_MAX);
    
    for (int i = 2; i <= LIMIT * 2; i++)
        factorize(i, i, i, 0, 0);
    
    unordered_set<int> items;
    for (int i = 2; i < (int)minSumProduct.size(); i++)
        items.insert(minSumProduct[i]);
    
    int sum = 0;
    for (int n : items)
        sum += n;
    
    cout << sum << endl;
    return 0;
}
```
