# Défi des nombres

## Description du Problème
Le fichier number-challenges.txt contient 200 problèmes, un par ligne au format :
v:n₁,n₂,...,nₖ

où v est la valeur cible et n₁, n₂, ..., nₖ sont les nombres disponibles.

**Objectif** : Pour chaque problème, trouver une expression utilisant les nombres disponibles et les opérations +, -, ×, ÷ qui évalue à v. Sommer les valeurs cibles pour tous les problèmes résolus.

---

## Exemples
### Exemple 1
**Entrée** : `v:n₁,n₂,...,nₖ`
**Sortie** : `Expression évaluant à v`
**Explication** : Trouver une expression qui évalue à la valeur cible.

---

## Approche de Résolution

### Stratégie
1. Utiliser une approche récursive pour générer toutes les expressions possibles.
2. Pour chaque sous-ensemble de nombres, générer toutes les façons de les combiner.
3. Utiliser la mémorisation pour éviter les recalculs.
4. Vérifier si une expression évalue à la valeur cible.
5. Collecter toutes les valeurs cibles pour les problèmes résolus.

### Observations
- On peut utiliser la programmation dynamique pour éviter les recalculs.
- Il faut gérer les divisions pour s'assurer que le résultat est un entier.
- On peut optimiser en évitant de générer des expressions redondantes.

### Complexité
- **Temporelle** : O(2^n × n!) pour générer toutes les expressions.
- **Spatiale** : O(2^n) pour la mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <string>
#include <fstream>
#include <sstream>
#include <unordered_map>

using namespace std;

struct Expression {
    int value;
    string expr;
};

set<int> solve(vector<int>& numbers) {
    if (numbers.size() == 1) {
        return {numbers[0]};
    }
    
    set<int> results;
    
    for (size_t i = 0; i < numbers.size(); i++) {
        for (size_t j = i + 1; j < numbers.size(); j++) {
            int a = numbers[i];
            int b = numbers[j];
            
            vector<int> remaining;
            for (size_t k = 0; k < numbers.size(); k++) {
                if (k != i && k != j) {
                    remaining.push_back(numbers[k]);
                }
            }
            
            vector<int> operations = {a + b, a - b, b - a, a * b};
            if (b != 0 && a % b == 0) operations.push_back(a / b);
            if (a != 0 && b % a == 0) operations.push_back(b / a);
            
            for (int op : operations) {
                vector<int> new_numbers = remaining;
                new_numbers.push_back(op);
                set<int> sub_results = solve(new_numbers);
                results.insert(sub_results.begin(), sub_results.end());
            }
        }
    }
    
    return results;
}

int main() {
    ifstream file("number-challenges.txt");
    string line;
    int total = 0;
    
    while (getline(file, line)) {
        size_t colon = line.find(':');
        int target = stoi(line.substr(0, colon));
        string nums_str = line.substr(colon + 1);
        
        vector<int> numbers;
        stringstream ss(nums_str);
        string num;
        while (getline(ss, num, ',')) {
            numbers.push_back(stoi(num));
        }
        
        set<int> results = solve(numbers);
        if (results.find(target) != results.end()) {
            total += target;
        }
    }
    
    cout << total << endl;
    return 0;
}
```
