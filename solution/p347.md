# Plus grand entier avec deux facteurs premiers distincts

## Description du Problème
Le plus grand entier M(p,q,N) qui a exactement deux facteurs premiers distincts p et q, et qui est inférieur ou égal à N, est défini comme le plus grand entier de la forme p^a × q^b (où a, b >= 1) qui est <= N.

**Objectif** : Trouver la somme de tous les M(p,q,10⁷) pour toutes les paires de nombres premiers distincts (p,q) où p < q.

---

## Exemples
### Exemple 1
**Entrée** : `p = 2, q = 3, N = 20`
**Sortie** : `18`
**Explication** : Les nombres de la forme 2^a × 3^b <= 20 sont : 6, 12, 18. Le plus grand est 18.

---

## Approche de Résolution

### Stratégie
1. Générer tous les nombres premiers jusqu'à N/2.
2. Pour chaque paire de nombres premiers distincts (p, q) où p < q :
   - Calculer le PPCM de p et q : lcm = p × q.
   - Si lcm > N, passer à la suivante.
   - Trouver le multiplicateur maximum tel que multiplier × lcm <= N.
   - Le multiplicateur doit être de la forme p^a × q^b.
   - Trouver la combinaison optimale de puissances de p et q.
3. Ajouter M(p,q,N) à un ensemble pour éviter les doublons.
4. Sommer tous les nombres dans l'ensemble.

### Observations
- On itère seulement sur les paires où p <= √N car p × q <= N implique p <= √N.
- Pour chaque paire (p, q), on cherche le multiplicateur maximum de la forme p^a × q^b.
- On utilise une approche gloutonne pour trouver le multiplicateur optimal.

### Complexité
- **Temporelle** : O(P² × log N), où P est le nombre de nombres premiers jusqu'à N/2.
- **Spatiale** : O(M), où M est le nombre de valeurs M(p,q,N) distinctes.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <unordered_set>
#include <cmath>
#include <algorithm>

using namespace std;

const int LIMIT = 10000000;

vector<int> listPrimes(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
    
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

int sqrt(int n) {
    return (int)std::sqrt(n);
}

int main() {
    unordered_set<int> possible;
    vector<int> primes = listPrimes(LIMIT / 2);
    int end = sqrt(LIMIT);
    
    for (int i = 0; i < primes.size(); i++) {
        int p = primes[i];
        if (p > end) {
            break;
        }
        
        for (int j = i + 1; j < primes.size(); j++) {
            int q = primes[j];
            int lcm = p * q;
            if (lcm > LIMIT) {
                break;
            }
            
            int multLimit = LIMIT / lcm;
            
            // Trouver le multiplicateur maximum de la forme p^a × q^b
            int multiplier = 1;
            while (multiplier * p <= multLimit) {
                multiplier *= p;
            }
            int maxMult = multiplier;
            
            // Essayer différentes combinaisons de puissances de p et q
            while (multiplier % p == 0) {
                multiplier /= p;
                while (multiplier * q <= multLimit) {
                    multiplier *= q;
                }
                maxMult = max(multiplier, maxMult);
            }
            
            possible.insert(maxMult * lcm);
        }
    }
    
    long long sum = 0;
    for (int x : possible) {
        sum += x;
    }
    
    cout << sum << endl;
    return 0;
}
```
