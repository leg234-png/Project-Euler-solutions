# Nombres Triangulaires, Pentagonaux et Hexagonaux

## Description du Problème
Les nombres triangulaires, pentagonaux et hexagonaux sont générés par les formules suivantes :

Triangulaire : Tₙ = n(n+1)/2
Pentagonal : Pₙ = n(3n−1)/2
Hexagonal : Hₙ = n(2n−1)

On peut vérifier que T₂₈₅ = P₁₆₅ = H₁₄₃ = 40755.

**Objectif** : Trouver le prochain nombre triangulaire qui est aussi pentagonal et hexagonal.

---

## Exemples
### Exemple 1
**Entrée** : Nombres après 40755
**Sortie** : `1533776805`
**Explication** : Le prochain nombre qui est à la fois triangulaire, pentagonal et hexagonal.

---

## Approche de Résolution

### Stratégie
1. Commencer avec les indices initiaux : i = 286 (triangulaire), j = 166 (pentagonal), k = 144 (hexagonal).
2. Calculer Tᵢ, Pⱼ, Hₖ.
3. Trouver le minimum des trois.
4. Si les trois sont égaux, c'est la réponse.
5. Sinon, incrémenter l'indice correspondant au minimum.
6. Répéter jusqu'à trouver la correspondance.

### Observations
- Tous les nombres hexagonaux sont aussi triangulaires (Hₙ = T₂ₙ₋₁).
- On peut donc chercher seulement les nombres qui sont à la fois pentagonaux et hexagonaux.
- On peut utiliser des formules inverses pour vérifier si un nombre est pentagonal ou hexagonal.

### Complexité
- **Temporelle** : O(n), où `n` est l'indice recherché.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

long long triangular(int n) {
    return (long long)n * (n + 1) / 2;
}

long long pentagonal(int n) {
    return (long long)n * (3 * n - 1) / 2;
}

long long hexagonal(int n) {
    return (long long)n * (2 * n - 1);
}

bool isPentagonal(long long x) {
    long long temp = 24 * x + 1;
    long long sqrtTemp = sqrt(temp);
    if (sqrtTemp * sqrtTemp == temp) {
        return (sqrtTemp + 1) % 6 == 0;
    }
    return false;
}

int main() {
    int i = 286;
    int j = 166;
    int k = 144;
    
    while (true) {
        long long tri = triangular(i);
        long long pent = pentagonal(j);
        long long hex = hexagonal(k);
        
        long long min = std::min(std::min(tri, pent), hex);
        
        if (tri == pent && pent == hex) {
            std::cout << tri << std::endl;
            break;
        }
        
        if (min == tri) i++;
        if (min == pent) j++;
        if (min == hex) k++;
    }
    
    return 0;
}
```
