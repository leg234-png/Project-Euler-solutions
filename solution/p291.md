# Nombres premiers de Panaitopol

## Description du Problème
Un nombre premier p est appelé un nombre premier de Panaitopol si p = (x⁴ - y⁴)/(x³ + y³) pour certains entiers positifs x et y.

**Objectif** : Trouver combien de nombres premiers de Panaitopol sont inférieurs à 5×10¹⁵.

---

## Exemples
### Exemple 1
**Entrée** : `x, y tels que p = (x⁴ - y⁴)/(x³ + y³)`
**Sortie** : `Nombre de nombres premiers`
**Explication** : Le nombre de nombres premiers de Panaitopol.

---

## Approche de Résolution

### Stratégie
1. Simplifier : p = (x⁴ - y⁴)/(x³ + y³) = (x - y)(x² + y²)/(x² - xy + y²).
2. La force brute mène à la séquence OEIS A027862 : nombres premiers de la forme n² + (n+1)² = 2n² + 2n + 1.
3. Vérifier que 2n² + 2n + 1 est premier pour n de 1 à √((limit-1)/2).
4. Utiliser un test de primalité rapide (comme Fermat) pour les grands nombres.

### Observations
- Les nombres premiers de Panaitopol peuvent être exprimés sous une forme simple.
- La séquence correspond aux nombres premiers de la forme 2n² + 2n + 1.
- On peut utiliser un test de primalité probabiliste pour accélérer.

### Complexité
- **Temporelle** : O(√limit × log limit) pour tester la primalité de chaque candidat.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

using namespace std;

bool is_prime_small(long long x) {
    if (x <= 1) return false;
    if (x <= 3) return true;
    if (x % 2 == 0) return false;
    for (long long i = 3; i * i <= x; i += 2) {
        if (x % i == 0) return false;
    }
    return true;
}

bool fermat_primality_test(long long n, int tests) {
    if (n < 100000) {
        return is_prime_small(n);
    }
    for (int x = 0; x < tests; x++) {
        long long base = 2 * (x + 2);
        long long result = 1;
        long long exp = n - 1;
        long long mod = n;
        
        // Exponentiation modulaire rapide
        base = base % mod;
        while (exp > 0) {
            if (exp % 2 == 1) {
                result = (result * base) % mod;
            }
            exp = exp >> 1;
            base = (base * base) % mod;
        }
        
        if (result != 1) {
            return false;
        }
    }
    return true;
}

long long compute(long long limit) {
    long long count = 0;
    long long max_n = sqrt((limit - 1) / 2);
    
    for (long long n = 1; n <= max_n; n++) {
        long long p = 2 * n * n + 2 * n + 1;
        if (p >= limit) break;
        if (fermat_primality_test(p, 5)) {
            count++;
        }
    }
    
    return count;
}

int main() {
    cout << compute(5000000000000000LL) << endl;
    return 0;
}
```
