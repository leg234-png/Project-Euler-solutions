# Produit de Coefficients d'un Polynôme Quadratique

## Description du Problème
Euler a découvert la remarquable formule quadratique :

n² + n + 41

Il s'avère que la formule produit 40 nombres premiers pour les valeurs consécutives n = 0 à 39. Cependant, lorsque n = 40, 40² + 40 + 41 = 40(40 + 1) + 41 est divisible par 41, et certainement lorsque n = 41, 41² + 41 + 41 est clairement divisible par 41.

La formule incroyable n² − 79n + 1601 a été découverte, qui produit 80 nombres premiers pour les valeurs consécutives n = 0 à 79. Le produit des coefficients, −79 et 1601, est −126479.

En considérant les formules quadratiques de la forme :

n² + an + b, où |a| < 1000 et |b| ≤ 1000

où |n| est le module/valeur absolue de n, par exemple |11| = 11 et |−4| = 4.

**Objectif** : Trouver le produit des coefficients, a et b, pour l'expression quadratique qui produit le nombre maximum de nombres premiers pour des valeurs consécutives de n, en commençant par n = 0.

---

## Exemples
### Exemple 1
**Entrée** : |a| < 10, |b| ≤ 10
**Sortie** : `-21`
**Explication** : Pour a = 1 et b = 41, on obtient 40 nombres premiers consécutifs.

---

## Approche de Résolution

### Stratégie
1. Pour chaque paire (a, b) où |a| < 1000 et |b| ≤ 1000 :
   - Calculer le nombre de nombres premiers consécutifs générés par n² + an + b en commençant par n = 0.
   - Garder trace de la paire qui produit le maximum.
2. Retourner le produit a × b de la meilleure paire.

### Observations
- Pour chaque paire (a, b), tester n = 0, 1, 2, ... jusqu'à ce qu'on obtienne un nombre non premier.
- On peut optimiser en remarquant que pour n = 0, on doit avoir b premier.
- Pour n = 1, on doit avoir 1 + a + b premier, donc a + b + 1 premier.

### Complexité
- **Temporelle** : O(a × b × n × √max), où `a` = 2000, `b` = 2001, `n` est le nombre moyen de tests, et `max` est la valeur maximale testée.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>

bool isPrime(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    
    int sqrtN = sqrt(n);
    for (int i = 3; i <= sqrtN; i += 2) {
        if (n % i == 0) {
            return false;
        }
    }
    return true;
}

int consecutivePrimes(int a, int b) {
    int n = 0;
    while (true) {
        int value = n * n + a * n + b;
        if (value < 0 || !isPrime(value)) {
            return n;
        }
        n++;
    }
}

int main() {
    int bestNum = 0;
    int bestA = 0;
    int bestB = 0;
    
    for (int a = -999; a < 1000; ++a) {
        for (int b = -1000; b <= 1000; ++b) {
            int num = consecutivePrimes(a, b);
            if (num > bestNum) {
                bestNum = num;
                bestA = a;
                bestB = b;
            }
        }
    }
    
    std::cout << (bestA * bestB) << std::endl;
    return 0;
}
```
