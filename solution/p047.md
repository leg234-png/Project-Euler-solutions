# Nombres Consécutifs avec Facteurs Premiers Distincts

## Description du Problème
Les deux premiers nombres consécutifs à avoir deux facteurs premiers distincts chacun sont :

14 = 2 × 7
15 = 3 × 5

Les trois premiers nombres consécutifs à avoir trois facteurs premiers distincts chacun sont :

644 = 2² × 7 × 23
645 = 3 × 5 × 43
646 = 2 × 17 × 19

**Objectif** : Trouver les quatre premiers entiers consécutifs à avoir chacun quatre facteurs premiers distincts. Quel est le premier de ces nombres ?

---

## Exemples
### Exemple 1
**Entrée** : k = 2 facteurs distincts
**Sortie** : `14`
**Explication** : 14 et 15 sont les premiers nombres consécutifs avec 2 facteurs premiers distincts chacun.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre i à partir de 2 :
   - Vérifier si i, i+1, i+2, i+3 ont chacun exactement 4 facteurs premiers distincts.
   - Si oui, retourner i.
2. Continuer jusqu'à trouver la réponse.

### Observations
- Pour compter les facteurs premiers distincts, factoriser le nombre et compter les facteurs uniques.
- On peut optimiser en utilisant un crible pour précalculer les facteurs premiers.
- On doit compter seulement les facteurs premiers distincts, pas leur multiplicité.

### Complexité
- **Temporelle** : O(n × √n), où `n` est le nombre recherché.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <cmath>
#include <set>

int countDistinctPrimeFactors(int n) {
    std::set<int> factors;
    int sqrtN = sqrt(n);
    
    for (int i = 2; i <= sqrtN; ++i) {
        if (n % i == 0) {
            factors.insert(i);
            while (n % i == 0) {
                n /= i;
            }
            sqrtN = sqrt(n);
        }
    }
    
    if (n > 1) {
        factors.insert(n);
    }
    
    return factors.size();
}

bool has4PrimeFactors(int n) {
    return countDistinctPrimeFactors(n) == 4;
}

int main() {
    for (int i = 2; ; ++i) {
        if (has4PrimeFactors(i) &&
            has4PrimeFactors(i + 1) &&
            has4PrimeFactors(i + 2) &&
            has4PrimeFactors(i + 3)) {
            std::cout << i << std::endl;
            break;
        }
    }
    
    return 0;
}
```
