# Composites with Prime Repunit Property

## Description du Problème
Un nombre composé entièrement de uns est appelé un repunit. Nous définirons R(k) comme étant un repunit de longueur k ; par exemple, R(6) = 111111.

Étant donné que n est un entier positif et gcd(n, 10) = 1, on peut montrer qu'il existe toujours une valeur, k, pour laquelle R(k) est divisible par n, et soit A(n) la plus petite telle valeur de k ; par exemple, A(7) = 6 et A(41) = 5.

On vous donne que pour tous les nombres premiers, p > 5, que p - 1 est divisible par A(p). Par exemple, lorsque p = 41, A(41) = 5, et 40 est divisible par 5.

Cependant, il existe des valeurs composites rares pour lesquelles cela est également vrai ; les cinq premiers exemples étant 91, 259, 451, 481 et 703.

**Objectif** : Trouver la somme des vingt-cinq premières valeurs composites de n pour lesquelles gcd(n, 10) = 1 et n - 1 est divisible par A(n).

---

## Exemples
### Exemple 1
**Entrée** : `n = 91`
**Sortie** : `true`
**Explication** : 91 est composite, gcd(91, 10) = 1, et 90 est divisible par A(91).

---

## Approche de Résolution

### Stratégie
1. Parcourir les nombres impairs à partir de 7.
2. Ignorer les multiples de 5.
3. Pour chaque nombre, vérifier s'il est composite (non premier).
4. Calculer A(n) en trouvant le plus petit k tel que R(k) ≡ 0 mod n.
5. Vérifier si (n - 1) est divisible par A(n).
6. Compter jusqu'à trouver 25 valeurs et les sommer.

### Observations
- On peut réutiliser la fonction findLeastDivisibleRepunit du problème 129.
- On doit vérifier que n est composite (pas premier).
- On ignore les multiples de 2 et 5.

### Complexité
- **Temporelle** : O(n × A(n)), où n est la valeur maximale à tester.
- **Spatiale** : O(1).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

bool isPrime(int n) {
    if (n < 2) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    for (int i = 3; i * i <= n; i += 2) {
        if (n % i == 0) return false;
    }
    return true;
}

int findLeastDivisibleRepunit(int n) {
    if (n % 2 == 0 || n % 5 == 0) {
        return 0;
    }
    
    int sum = 1;
    int pow = 1;
    int k = 1;
    
    while (sum % n != 0) {
        k++;
        pow = (pow * 10) % n;
        sum = (sum + pow) % n;
    }
    
    return k;
}

int main() {
    int sum = 0;
    int found = 0;
    
    for (int i = 7; found < 25; i += 2) {
        if (i % 5 != 0 && !isPrime(i)) {
            int a = findLeastDivisibleRepunit(i);
            if (a > 0 && (i - 1) % a == 0) {
                sum += i;
                found++;
            }
        }
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
