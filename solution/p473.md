# Base phigitale

## Description du Problème
Le nombre d'or φ = (1+√5)/2 satisfait de nombreuses propriétés intéressantes.

Pour représenter cette somme de puissances de φ, on utilise une chaîne de 0 et de 1 avec un point pour indiquer où commencent les exposants négatifs.

Nous appelons cela la représentation dans la base phigitale.

Ainsi 1=1_φ, 2=10.01_φ, 3=100.01_φ et 14=100100.001001_φ.

Les chaînes représentant 1, 2 et 14 dans la base phigitale sont palindromiques, tandis que la chaîne représentant 3 ne l'est pas.

**Objectif** : Trouver la somme de tous les entiers positifs n ≤ 10¹⁰ dont la représentation dans la base phigitale est palindromique.

---

## Exemples
### Exemple 1
**Entrée** : `n = 1`
**Sortie** : `1` (palindromique)
**Explication** : 1 = 1_φ est palindromique.

### Exemple 2
**Entrée** : `n = 2`
**Sortie** : `2` (palindromique)
**Explication** : 2 = 10.01_φ est palindromique.

---

## Approche de Résolution

### Stratégie
1. Utiliser les propriétés de φ : φ² = φ + 1, donc φ^n = f(n)φ + f(n-1).
2. φ^(-n) = (-1)^(n+1)(f(n)φ - f(n+1)).
3. Construire les représentations phigitales pour chaque nombre.
4. Vérifier si la représentation est palindromique.
5. Sommer tous les nombres palindromiques ≤ 10¹⁰.

### Observations
- Les nombres de Fibonacci apparaissent naturellement dans les représentations.
- On peut construire les représentations en utilisant les relations de φ.
- Il y a des patterns dans les nombres palindromiques.

### Complexité
- **Temporelle** : O(N × L) où N = 10¹⁰ et L est la longueur moyenne de la représentation.
- **Spatiale** : O(L) pour stocker la représentation.

bool is_palindrome(const string& s) {
    string cleaned = s;
    size_t dot_pos = cleaned.find('.');
    if (dot_pos != string::npos) {
        cleaned.erase(dot_pos, 1);
    }
    string reversed = cleaned;
