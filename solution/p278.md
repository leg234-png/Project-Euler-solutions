# Combinaisons linéaires de semi-premiers

## Description du Problème
Étant donné des entiers positifs a₁, a₂, ..., aₖ avec gcd(a₁, a₂, ..., aₖ) = 1, nous appelons le plus grand entier positif qui ne peut pas être exprimé comme une combinaison linéaire non négative de ces entiers f(a₁, a₂, ..., aₖ).

Par exemple, si a₁=5 et a₂=7, il n'y a pas de q₁≥0 et q₂≥0 tels que b pourrait être 1, 2, 3, 4, 6, 8, 9, 11, 13, 16, 18 ou 23.

En fait, 23 est la plus grande valeur impossible de b pour a₁=5 et a₂=7. Nous appelons donc f(5,7)=23. De même, on peut montrer que f(6,10,15)=29 et f(14,22,77)=195.

**Objectif** : Trouver ∑f(pᵢ,pⱼ,pₖ) pour tous les triplets de nombres premiers (pᵢ,pⱼ,pₖ) avec pᵢ < pⱼ < pₖ ≤ 1000.

---

## Exemples
### Exemple 1
**Entrée** : `a₁=5, a₂=7`
**Sortie** : `23`
**Explication** : f(5,7) = 23.

---

## Approche de Résolution

### Stratégie
1. Utiliser la formule de Johnson pour calculer f(a₁, a₂, a₃).
2. Si gcd(a₁, a₂, a₃) = 1 et gcd(a₁, a₂) = d et aᵢ = d×aᵢ' pour i = 2, 3, alors :
   f(a₁, a₂, a₃) = d×f(a₁, a₂', a₃') + a₁×(d - 1).
3. Pour trois nombres premiers distincts p, q, r, on a :
   f(p, q, r) = p×q×r×2 - p×q - p×r - q×r.
4. Générer tous les triplets de nombres premiers et sommer les résultats.

### Observations
- La formule de Johnson permet de calculer efficacement f pour trois nombres.
- Pour trois nombres premiers, la formule se simplifie considérablement.
- On peut itérer sur tous les triplets de nombres premiers.

### Complexité
- **Temporelle** : O(P³) où P est le nombre de nombres premiers ≤ 1000.
- **Spatiale** : O(P) pour stocker les nombres premiers.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<int> list_primes(int n) {
    vector<bool> is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (is_prime[i]) {
            for (int j = 2 * i; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
    }
    return primes;
}

long long compute(int limit) {
    vector<int> primes = list_primes(limit);
    long long total = 0;
    
    for (size_t a = 0; a < primes.size(); a++) {
        int p = primes[a];
        for (size_t b = a + 1; b < primes.size(); b++) {
            int q = primes[b];
            for (size_t c = b + 1; c < primes.size(); c++) {
                int r = primes[c];
                total += (long long)p * q * r * 2 - (long long)p * q - (long long)p * r - (long long)q * r;
            }
        }
    }
    
    return total;
}

int main() {
    cout << compute(1000) << endl;
    return 0;
}
```
