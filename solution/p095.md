# Amicable Chains

## Description du Problème
Les diviseurs propres d'un nombre sont tous les diviseurs à l'exclusion du nombre lui-même. Par exemple, les diviseurs propres de 28 sont 1, 2, 4, 7 et 14. Comme la somme de ces diviseurs est égale à 28, nous l'appelons un nombre parfait.

Intéressant, la somme des diviseurs propres de 220 est 284 et la somme des diviseurs propres de 284 est 220, formant une chaîne de deux nombres. Pour cette raison, 220 et 284 sont appelés une paire amicale.

Peut-être moins connues sont les chaînes plus longues. Par exemple, en commençant par 12496, nous formons une chaîne de cinq nombres :
12496 → 14288 → 15472 → 14536 → 14264 (→ 12496 → ⋯)

Comme cette chaîne revient à son point de départ, elle est appelée une chaîne amicale.

**Objectif** : Trouver le plus petit membre de la plus longue chaîne amicale sans élément dépassant un million.

---

## Exemples
### Exemple 1
**Entrée** : `220`
**Sortie** : `220` (chaîne de 2)
**Explication** : 220 → 284 → 220

### Exemple 2
**Entrée** : `12496`
**Sortie** : `12496` (chaîne de 5)
**Explication** : 12496 → 14288 → 15472 → 14536 → 14264 → 12496

---

## Approche de Résolution

### Stratégie
1. Calculer la somme des diviseurs propres pour tous les nombres jusqu'à 10⁶.
2. Pour chaque nombre, suivre la chaîne en calculant récursivement la somme des diviseurs.
3. Détecter les cycles en vérifiant si on revient au nombre de départ.
4. Trouver la chaîne la plus longue et retourner son plus petit membre.

### Observations
- On peut calculer efficacement la somme des diviseurs en utilisant un crible.
- Une chaîne amicale forme un cycle, donc on doit détecter quand on revient au point de départ.
- Si un nombre dépasse la limite ou forme une chaîne qui ne revient pas au départ (forme rho), on l'ignore.

### Complexité
- **Temporelle** : O(n log n) pour calculer les sommes de diviseurs, puis O(n × k) où k est la longueur moyenne des chaînes.
- **Spatiale** : O(n) pour stocker les sommes de diviseurs.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <set>

const int LIMIT = 1000000;

int main() {
    // divisorSum[n] est la somme de tous les diviseurs propres de n
    std::vector<int> divisorSum(LIMIT + 1, 0);
    for (int i = 1; i <= LIMIT; i++) {
        for (int j = i * 2; j <= LIMIT; j += i) {
            divisorSum[j] += i;
        }
    }
    
    // Analyser la longueur de la chaîne amicale pour chaque nombre
    int maxChainLen = 0;
    int minChainElem = -1;
    for (int i = 0; i <= LIMIT; i++) {
        std::set<int> visited;
        int count = 1;
        int cur = i;
        visited.insert(cur);
        
        while (true) {
            int next = divisorSum[cur];
            if (next == i) {
                if (count > maxChainLen) {
                    minChainElem = i;
                    maxChainLen = count;
                }
                break;
            } else if (next > LIMIT || visited.find(next) != visited.end()) {
                break;
            } else {
                visited.insert(next);
                cur = next;
                count++;
            }
        }
    }
    
    std::cout << minChainElem << std::endl;
    return 0;
}
```
