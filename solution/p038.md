# Multiples Pandigitaux

## Description du Problème
Prenez le nombre 192 et multipliez-le par chacun de 1, 2, et 3 :

192 × 1 = 192
192 × 2 = 384
192 × 3 = 576

En concaténant chaque produit, on obtient le pandigital 1 à 9, 192384576. On appellera 192384576 le produit concaténé de 192 et (1,2,3).

La même chose peut être réalisée en commençant par 9 et en multipliant par 1, 2, 3, 4, et 5, donnant le pandigital, 918273645, qui est le produit concaténé de 9 et (1,2,3,4,5).

**Objectif** : Quel est le plus grand nombre pandigital 1 à 9 à 9 chiffres qui peut être formé comme le produit concaténé d'un entier avec (1,2,…,n) où n > 1 ?

---

## Exemples
### Exemple 1
**Entrée** : Produits concaténés
**Sortie** : `932718654`
**Explication** : Le plus grand nombre pandigital 1-9 formé comme produit concaténé.

---

## Approche de Résolution

### Stratégie
1. Pour chaque n de 2 à 9 (nombre de multiplicateurs) :
   - Pour chaque entier i de 1 à une limite :
     - Concaténer les produits i×1, i×2, ..., i×n.
     - Vérifier si le résultat est pandigital 1-9.
     - Garder trace du maximum.
2. Retourner le maximum.

### Observations
- Pour n = 2, i doit être < 10⁴ (4 chiffres max).
- Pour n = 3, i doit être < 10³ (3 chiffres max).
- En général, pour n multiplicateurs, i doit être < 10^(9/n).
- On peut arrêter quand la concaténation dépasse 9 chiffres.

### Complexité
- **Temporelle** : O(n × m), où `n` = 9 et `m` est la limite pour i.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>

bool isPandigital(const std::string& s) {
    if (s.length() != 9) return false;
    std::string sorted = s;
    std::sort(sorted.begin(), sorted.end());
    return sorted == "123456789";
}

int main() {
    int max = -1;
    
    for (int n = 2; n <= 9; ++n) {
        int limit = 1;
        for (int i = 0; i < 9 / n; ++i) {
            limit *= 10;
        }
        
        for (int i = 1; i < limit; ++i) {
            std::string concat;
            for (int j = 1; j <= n; ++j) {
                concat += std::to_string(i * j);
            }
            
            if (isPandigital(concat)) {
                int num = std::stoi(concat);
                if (num > max) {
                    max = num;
                }
            }
        }
    }
    
    std::cout << max << std::endl;
    return 0;
}
```
