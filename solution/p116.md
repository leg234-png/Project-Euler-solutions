# Red, Green or Blue Tiles

## Description du Problème
Une rangée de cinq carrés gris doit avoir un certain nombre de ses carreaux remplacés par des carreaux oblongs colorés choisis parmi rouge (longueur deux), vert (longueur trois) ou bleu (longueur quatre).

Si des carreaux rouges sont choisis, il y a exactement sept façons de le faire.

Si des carreaux verts sont choisis, il y a trois façons.

Et si des carreaux bleus sont choisis, il y a deux façons.

En supposant que les couleurs ne peuvent pas être mélangées, il y a 7+3+2=12 façons de remplacer les carreaux gris dans une rangée mesurant cinq unités de longueur.

**Objectif** : Combien de façons différentes peut-on remplacer les carreaux gris dans une rangée mesurant cinquante unités de longueur si les couleurs ne peuvent pas être mélangées et qu'au moins un carreau coloré doit être utilisé ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 5`
**Sortie** : `12`
**Explication** : 7 (rouge) + 3 (vert) + 2 (bleu) = 12.

---

## Approche de Résolution

### Stratégie
1. Pour chaque couleur (rouge=2, vert=3, bleu=4), calculer le nombre de façons séparément.
2. Utiliser la programmation dynamique : ways[n] = ways[n-1] + ways[n-m] où m est la longueur du carreau.
3. Soustraire 1 pour exclure le cas où tous les carreaux sont gris.
4. Sommer les résultats pour les trois couleurs.

### Observations
- Chaque couleur est traitée indépendamment.
- ways[n] = nombre de façons avec des carreaux gris et des carreaux de longueur m.
- On exclut le cas où aucun carreau coloré n'est utilisé.

### Complexité
- **Temporelle** : O(n) pour chaque couleur, donc O(3n) = O(n).
- **Spatiale** : O(n).

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>

const int LENGTH = 50;

long long countWays(int length, int m) {
    std::vector<long long> ways(length + 1);
    ways[0] = 1;
    
    for (int n = 1; n <= length; n++) {
        ways[n] += ways[n - 1];  // Carré gris
        if (n >= m) {
            ways[n] += ways[n - m];  // Carreau coloré de longueur m
        }
    }
    
    return ways[length] - 1;  // Exclure le cas tout gris
}

int main() {
    long long result = countWays(LENGTH, 2) +  // Rouge
                       countWays(LENGTH, 3) +  // Vert
                       countWays(LENGTH, 4);    // Bleu
    std::cout << result << std::endl;
    return 0;
}
```
