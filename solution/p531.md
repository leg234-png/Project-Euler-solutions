# Restes chinois

## Description du Problème
Soit g(a,n,b,m) la plus petite solution non négative x au système :
- x = a mod n
- x = b mod m

si une telle solution existe, sinon 0.

Par exemple, g(2,4,4,6) = 10, mais g(3,4,4,6) = 0.

Soit φ(n) la fonction totient d'Euler.

Soit f(n,m) = g(φ(n), n, φ(m), m).

**Objectif** : Trouver ∑ f(n,m) pour 1000000 ≤ n < m < 1005000.

---

## Exemples
### Exemple 1
**Entrée** : `a=2, n=4, b=4, m=6`
**Sortie** : `10`
**Explication** : g(2,4,4,6) = 10.

---

## Approche de Résolution

### Stratégie
1. Utiliser l'algorithme d'Euclide étendu pour trouver le pgcd et les coefficients de Bézout.
2. Utiliser le théorème des restes chinois généralisé pour résoudre le système.
3. Si gcd(n, m) ne divise pas (a - b), alors il n'y a pas de solution.
4. Sinon, utiliser la formule du théorème des restes chinois.
5. Calculer φ(n) pour tous les n dans la plage.
6. Calculer f(n,m) pour toutes les paires et sommer.

### Observations
- On peut utiliser un crible pour calculer efficacement φ(n).
- Le théorème des restes chinois généralisé gère le cas où n et m ne sont pas copremiers.
- On doit vérifier l'existence d'une solution avant de la calculer.

### Complexité
- **Temporelle** : O((m-n)² × log(max(n,m))) pour toutes les paires.
- **Spatiale** : O(m-n) pour stocker les valeurs de φ.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

pair<long long, pair<long long, long long>> extended_gcd(long long a, long long b) {
    if (b == 0) {
        return {a, {1, 0}};
    }
    auto [g, coef] = extended_gcd(b, a % b);
    long long x1 = coef.first;
    long long y1 = coef.second;
    return {g, {y1, x1 - (a / b) * y1}};
}

long long chinese_remainder(long long a, long long n, long long b, long long m) {
    auto [g, coef] = extended_gcd(n, m);
    if ((a - b) % g != 0) {
        return 0;
    }
    
    long long lcm = (n / g) * m;
    long long x = a + n * ((b - a) / g) * coef.first;
    return ((x % lcm) + lcm) % lcm;
}

vector<long long> totient_sieve(long long n) {
    vector<long long> phi(n + 1);
    for (long long i = 0; i <= n; i++) {
        phi[i] = i;
    }
    for (long long i = 2; i <= n; i++) {
        if (phi[i] == i) {
            for (long long j = i; j <= n; j += i) {
                phi[j] -= phi[j] / i;
            }
        }
    }
    return phi;
}

long long compute(long long start, long long end) {
    vector<long long> phi = totient_sieve(end);
    long long total = 0;
    
    for (long long n = start; n < end; n++) {
        for (long long m = n + 1; m < end; m++) {
            total += chinese_remainder(phi[n], n, phi[m], m);
        }
    }
    
    return total;
}

int main() {
    cout << compute(1000000, 1005000) << endl;
    return 0;
}
```
