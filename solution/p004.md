# Plus Grand Produit Palindrome

## Description du Problème
Un nombre palindrome se lit de la même manière dans les deux sens. Le plus grand palindrome formé par le produit de deux nombres à 2 chiffres est 9009 = 91 × 99.

**Objectif** : Trouver le plus grand palindrome formé par le produit de deux nombres à 3 chiffres.

---

## Exemples
### Exemple 1
**Entrée** : Produits de nombres à 2 chiffres
**Sortie** : `9009`
**Explication** : 9009 = 91 × 99 est le plus grand palindrome formé par le produit de deux nombres à 2 chiffres.

---

## Approche de Résolution

### Stratégie
1. Parcourir tous les produits possibles de deux nombres à 3 chiffres.
2. Pour chaque produit, vérifier s'il est un palindrome.
3. Garder trace du plus grand palindrome trouvé.
4. Retourner ce maximum.

### Observations
- On peut commencer par les plus grands nombres et descendre pour optimiser.
- On peut éviter les produits déjà testés (si on teste i × j, pas besoin de tester j × i).
- Pour vérifier si un nombre est palindrome :
  - Convertir le nombre en chaîne de caractères.
  - Comparer la chaîne avec son inverse.
  - Ou inverser le nombre mathématiquement et comparer.

### Complexité
- **Temporelle** : O(n²), où `n` est le nombre de nombres à 3 chiffres (900).
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>

bool isPalindrome(int n) {
    std::string s = std::to_string(n);
    std::string rev = s;
    std::reverse(rev.begin(), rev.end());
    return s == rev;
}

int main() {
    int maxPalin = -1;
    
    for (int i = 100; i < 1000; ++i) {
        for (int j = 100; j < 1000; ++j) {
            int prod = i * j;
            if (isPalindrome(prod) && prod > maxPalin) {
                maxPalin = prod;
            }
        }
    }
    
    std::cout << maxPalin << std::endl;
    return 0;
}
```
