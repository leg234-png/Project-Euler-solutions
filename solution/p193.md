# Nombres sans facteur carré

## Description du Problème
Un entier positif n est appelé squarefree (sans facteur carré) si aucun carré de nombre premier ne divise n. Ainsi, 1, 2, 3, 5, 6, 7, 10, 11 sont squarefree, mais pas 4, 8, 9, 12.

**Objectif** : Combien de nombres squarefree y a-t-il en dessous de 2⁵⁰ ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 12`
**Sortie** : `false` (non squarefree)
**Explication** : 12 = 2² × 3, donc 12 n'est pas squarefree car 2² divise 12.

---

## Approche de Résolution

### Stratégie
1. Utiliser la fonction de Möbius généralisée μₖ(n) pour les nombres k-free.
2. Pour k=2 (squarefree), utiliser μ₂(n) qui vaut 1 si n est squarefree avec un nombre pair de facteurs premiers, -1 si impair, et 0 si n a un facteur carré.
3. Compter S(n) = ∑ₐ₌₁ⁿ |μ₂(a)| = ∑ₐ₌₁^⌊√n⌋ μ₂(a) × ⌊n/a²⌋.
4. Utiliser un crible pour calculer efficacement la fonction de Möbius.

### Observations
- La fonction de Möbius généralisée permet de compter efficacement les nombres k-free.
- On peut utiliser un crible modifié pour calculer μₖ(n).
- La formule utilise la sommation sur les diviseurs carrés.

### Complexité
- **Temporelle** : O(√n) pour le crible et la sommation.
- **Spatiale** : O(√n) pour stocker la fonction de Möbius.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

vector<int> mobius_k_sieve(long long n, int k) {
    vector<bool> isprime(n + 1, true);
    isprime[0] = isprime[1] = false;
    vector<int> mob(n + 1, 1);
    
    for (int p = 2; p <= n; p++) {
        if (isprime[p]) {
            mob[p] *= -1;
            for (int i = 2*p; i <= n; i += p) {
                isprime[i] = false;
                mob[i] *= -1;
            }
            long long sq = pow(p, k);
            if (sq <= n) {
                for (long long j = sq; j <= n; j += sq) {
                    mob[j] = 0;
                }
            }
        }
    }
    return mob;
}

long long count_kfree(long long n, int k) {
    long long sq = pow(n, 1.0/k);
    vector<int> mobius_k = mobius_k_sieve(sq, k);
    
    long long total = 0;
    for (int i = 1; i <= sq; i++) {
        total += mobius_k[i] * (n / pow(i, k));
    }
    return total;
}

int main() {
    long long n = pow(2, 50);
    cout << count_kfree(n, 2) << endl;
    return 0;
}
```
