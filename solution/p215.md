# Crack-free Walls

## Description du Problème
On veut construire un mur de 32 unités de largeur et 10 unités de hauteur en utilisant des briques de longueur 2 ou 3 unités. Les briques doivent être placées horizontalement et ne peuvent pas se chevaucher.

Une "fissure" se produit lorsque les extrémités de deux briques dans des rangées adjacentes s'alignent verticalement.

**Objectif** : Combien de façons de construire un tel mur sans fissures ?

---

## Exemples
### Exemple 1
**Entrée** : `width = 4, height = 2`
**Sortie** : Nombre de façons sans fissures
**Explication** : On doit éviter que les joints verticaux s'alignent entre les rangées.

---

## Approche de Résolution

### Stratégie
1. Générer toutes les positions de fissures possibles pour une rangée :
   - Utiliser une récursion pour placer des briques de longueur 2 ou 3.
   - Les positions de fissures sont les positions où les briques se terminent (sauf la dernière).
2. Pour chaque position de fissures, vérifier quelles autres positions sont compatibles (disjointes).
3. Utiliser la programmation dynamique :
   - `ways[i][j]` : nombre de façons de construire `i` rangées avec la position de fissures `j`.
   - `ways[i][j] = sum(ways[i-1][k])` pour tous les `k` compatibles avec `j`.
4. La réponse est la somme de `ways[HEIGHT][j]` pour tous les `j`.

### Observations
- Deux positions de fissures sont compatibles si elles n'ont aucune position en commun.
- On peut générer toutes les positions de fissures récursivement.
- La programmation dynamique permet de compter efficacement les façons.

### Complexité
- **Temporelle : O(height × P²)**, où P est le nombre de positions de fissures possibles.
- **Spatiale : O(height × P)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <stack>
#include <algorithm>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

const int WIDTH = 32;
const int HEIGHT = 10;

void getCrackPositions(vector<int>& cracks, int position, vector<vector<int>>& result) {
    if (position < WIDTH) {
        for (int i = 2; i <= 3; i++) {
            cracks.push_back(position + i);
            getCrackPositions(cracks, position + i, result);
            cracks.pop_back();
        }
    } else if (position == WIDTH) {
        vector<int> temp(cracks.begin(), cracks.end() - 1);
        result.push_back(temp);
    }
}

bool areDisjointSorted(const vector<int>& a, const vector<int>& b) {
    int i = 0, j = 0;
    while (i < a.size() && j < b.size()) {
        if (a[i] == b[j])
            return false;
        else if (a[i] < b[j])
            i++;
        else
            j++;
    }
    return true;
}

int main() {
    vector<vector<int>> crackPositions;
    vector<int> cracks;
    getCrackPositions(cracks, 0, crackPositions);
    
    vector<cpp_int> ways(crackPositions.size(), 1);
    
    for (int i = 1; i < HEIGHT; i++) {
        vector<cpp_int> newWays(ways.size(), 0);
        for (size_t j = 0; j < newWays.size(); j++) {
            for (size_t k = 0; k < ways.size(); k++) {
                if (areDisjointSorted(crackPositions[j], crackPositions[k]))
                    newWays[j] += ways[k];
            }
        }
        ways = newWays;
    }
    
    cpp_int sum = 0;
    for (const cpp_int& x : ways)
        sum += x;
    
    cout << sum << endl;
    return 0;
}
```
