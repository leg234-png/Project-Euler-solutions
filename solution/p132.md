# Large Repunit Factors

## Description du Problème
Un nombre composé entièrement de uns est appelé un repunit. Nous définirons R(k) comme étant un repunit de longueur k.

Par exemple, R(10) = 1111111111 = 11 × 41 × 271 × 9091, et la somme de ces facteurs premiers est 9414.

**Objectif** : Trouver la somme des quarante premiers facteurs premiers de R(10⁹).

---

## Exemples
### Exemple 1
**Entrée** : `k = 10`
**Sortie** : `9414`
**Explication** : Les facteurs premiers de R(10) sont 11, 41, 271, 9091, leur somme est 9414.

---

## Approche de Résolution

### Stratégie
1. R(k) = (10ᵏ - 1) / 9.
2. Pour qu'un nombre premier p divise R(k), on doit avoir (10ᵏ - 1) / 9 ≡ 0 mod p.
3. Cela équivaut à 10ᵏ ≡ 1 mod (9p).
4. Parcourir les nombres premiers et vérifier si 10¹⁰⁹ ≡ 1 mod (9p).
5. Compter les 40 premiers et les sommer.

### Observations
- On utilise l'exponentiation modulaire pour calculer 10¹⁰⁹ mod (9p).
- On peut utiliser powMod(10, 10⁹, 9p) pour calculer efficacement.

### Complexité
- **Temporelle** : O(n log k), où n est le nombre de nombres premiers à tester.
- **Spatiale** : O(1).

long long powMod(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        base = (base * base) % mod;
        exp /= 2;
    }
    return result;
}
