# Probabilités au Monopoly

## Description du Problème
Dans le jeu Monopoly, le plateau standard est configuré de manière spécifique. Un joueur commence sur la case GO et ajoute les scores de deux dés à 6 faces pour déterminer le nombre de cases dont il avance dans le sens horaire. Sans règles supplémentaires, on s'attendrait à visiter chaque case avec une probabilité égale : 2.5%. Cependant, atterrir sur G2J (Go To Jail), CC (community chest), et CH (chance) change cette distribution.

En plus de G2J, et d'une carte de chaque CC et CH, qui ordonne au joueur d'aller directement en prison, si un joueur fait trois doubles consécutifs, il n'avance pas le résultat de son 3ème lancer. Au lieu de cela, il va directement en prison.

Au début du jeu, les cartes CC et CH sont mélangées. Quand un joueur atterrit sur CC ou CH, il prend une carte du haut de la pile respective et, après avoir suivi les instructions, elle est retournée au bas de la pile.

**Objectif** : Si, au lieu d'utiliser deux dés à 6 faces, deux dés à 4 faces sont utilisés, trouver la chaîne de six chiffres modale.

---

## Exemples
### Exemple 1
**Entrée** : Deux dés à 6 faces
**Sortie** : `102400`
**Explication** : Les trois cases les plus populaires sont JAIL (10), E3 (24), et GO (00).

---

## Approche de Résolution

### Stratégie
1. Simuler le jeu pour un grand nombre de lancers (100 millions).
2. Utiliser des dés tétraédriques (4 faces) au lieu de dés à 6 faces.
3. Gérer les règles spéciales :
   - Trois doubles consécutifs → aller en prison
   - Cartes Chance et Community Chest
   - G2J → aller en prison
4. Compter les visites de chaque case.
5. Trouver les trois cases les plus visitées et construire la chaîne de 6 chiffres.

### Observations
- C'est un algorithme d'approximation statistique par échantillonnage.
- Un algorithme exact impliquerait de calculer le vecteur propre de la plus grande valeur propre de la matrice de transition.
- On utilise un mélange Fisher-Yates pour les cartes.

### Complexité
- **Temporelle** : O(N), où N est le nombre de simulations (100 000 000).
- **Spatiale** : O(1), pour stocker les compteurs de visites.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <random>
#include <iomanip>

using namespace std;

class CardDeck {
private:
    vector<int> cards;
    int index;
    mt19937 rng;
    
public:
    CardDeck(int size) : index(size), rng(random_device{}()) {
        cards.resize(size);
        for (int i = 0; i < size; i++)
            cards[i] = i;
    }
    
    int nextCard() {
        if (index == (int)cards.size()) {
            // Mélange Fisher-Yates
            for (int i = cards.size() - 1; i >= 0; i--) {
                uniform_int_distribution<int> dist(0, i);
                int j = dist(rng);
                swap(cards[i], cards[j]);
            }
            index = 0;
        }
        return cards[index++];
    }
};

int main() {
    const int SIMULATIONS = 100000000;
    vector<int> visitCounts(40, 0);
    
    CardDeck chance(16);
    CardDeck communityChest(16);
    int consecutiveDoubles = 0;
    int location = 0;
    mt19937 rng(random_device{}());
    uniform_int_distribution<int> die(1, 4);
    
    for (int i = 0; i < SIMULATIONS; i++) {
        int die0 = die(rng);
        int die1 = die(rng);
        
        if (die0 == die1)
            consecutiveDoubles++;
        else
            consecutiveDoubles = 0;
        
        if (consecutiveDoubles < 3)
            location = (location + die0 + die1) % 40;
        else {
            location = 30;  // Aller en prison
            consecutiveDoubles = 0;
        }
        
        // Traiter les actions pour certaines cases
        if (location == 7 || location == 22 || location == 36) {  // Chance
            switch (chance.nextCard()) {
                case 0: location = 0; break;
                case 1: location = 10; break;
                case 2: location = 11; break;
                case 3: location = 24; break;
                case 4: location = 39; break;
                case 5: location = 5; break;
                case 6: case 7:  // Prochaine gare
                    location = (location + 5) / 10 % 4 * 10 + 5;
                    break;
                case 8:  // Prochaine compagnie
                    location = (location > 12 && location < 28) ? 28 : 12;
                    break;
                case 9:
                    location -= 3;
                    break;
            }
        }
        
        if (location == 30)  // Aller en prison
            location = 10;
        
        if (location == 2 || location == 17 || location == 33) {  // Community Chest
            switch (communityChest.nextCard()) {
                case 0: location = 0; break;
                case 1: location = 10; break;
            }
        }
        
        visitCounts[location]++;
    }
    
    // Trouver les 3 cases les plus visitées
    vector<pair<int, int>> counts;
    for (int i = 0; i < 40; i++)
        counts.push_back({visitCounts[i], i});
    
    sort(counts.rbegin(), counts.rend());
    
    cout << setfill('0') << setw(2) << counts[0].second
         << setw(2) << counts[1].second
         << setw(2) << counts[2].second << endl;
    return 0;
}
```
