# Une relation de récurrence étrange

## Description du Problème
f(1) = 1
f(3) = 3
f(2n) = f(n)
f(4n + 1) = 2f(2n + 1) - f(n)
f(4n + 3) = 3f(2n + 1) - 2f(n)

La fonction S(n) est définie comme ∑ᵢ₌₁ⁿ f(i).

S(8) = 22 et S(100) = 3604.

**Objectif** : Trouver S(3³⁷). Donner les 9 derniers chiffres de votre réponse.

---

## Exemples
### Exemple 1
**Entrée** : `n = 8`
**Sortie** : `22`
**Explication** : S(8) = 22.

---

## Approche de Résolution

### Stratégie
1. Analyser les relations de récurrence pour trouver des patterns.
2. Observer que f(4n+3) - f(4n+2) = 2f(2n+1) - 2f(n).
3. Observer que f(4n+2) - f(4n+1) = -f(2n+1) + f(n).
4. Observer que f(4n+1) - f(4n) = 2f(2n+1) - 2f(n).
5. Donc f(4n+3) + f(4n+2) + f(4n+1) + f(4n) = 6f(2n+1) - 2f(n) = 6(f(2n+1) + f(2n)) - 8f(n).
6. Finalement, S(4n+3) = 6S(2n+1) - 8S(n).
7. De même, S(4n+2) = S(2n+1) - 3S(n).

### Observations
- On peut calculer S(n) récursivement en utilisant ces relations.
- Les relations permettent de réduire rapidement la taille du problème.
- On peut utiliser la mémorisation pour éviter les recalculs.

### Complexité
- **Temporelle** : O(log n) pour le calcul récursif.
- **Spatiale** : O(log n) pour la table de mémorisation.

---

## Solution en C++

```cpp
#include <iostream>
#include <unordered_map>

using namespace std;

unordered_map<long long, long long> memo;

long long S(long long n) {
    if (n == 0) return 0;
    if (n == 1) return 1;
    if (n == 2) return 2;
    if (n == 3) return 5;
    
    if (memo.find(n) != memo.end()) {
        return memo[n];
    }
    
    long long result;
    if (n % 4 == 0) {
        result = S(n / 2);
    } else if (n % 4 == 1) {
        long long k = (n - 1) / 4;
        result = 2 * S(2 * k + 1) - S(k);
    } else if (n % 4 == 2) {
        long long k = (n - 2) / 4;
        result = S(2 * k + 1) - 3 * S(k);
    } else { // n % 4 == 3
        long long k = (n - 3) / 4;
        result = 6 * S(2 * k + 1) - 8 * S(k);
    }
    
    memo[n] = result;
    return result;
}

int main() {
    long long n = 1;
    for (int i = 0; i < 37; i++) {
        n *= 3;
    }
    long long mod = 1000000000;
    cout << S(n) % mod << endl;
    return 0;
}
```
