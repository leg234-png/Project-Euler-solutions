# Palindromes Double Base

## Description du Problème
Le nombre décimal, 585 = 1001001001₂ (binaire), est palindromique dans les deux bases.

**Objectif** : Trouver la somme de tous les nombres, inférieurs à un million, qui sont palindromiques en base 10 et en base 2.

**NOTE** : Veuillez noter que le nombre palindromique, dans l'une ou l'autre base, ne peut pas inclure de zéros en tête.

---

## Exemples
### Exemple 1
**Entrée** : Limite = 1000
**Sortie** : `1772`
**Explication** : La somme des nombres palindromiques en base 10 et base 2 inférieurs à 1000.

---

## Approche de Résolution

### Stratégie
1. Pour chaque nombre i de 1 à 999 999 :
   - Vérifier s'il est palindrome en base 10.
   - Vérifier s'il est palindrome en base 2.
   - Si les deux conditions sont satisfaites, ajouter i à la somme.
2. Retourner la somme.

### Observations
- Pour vérifier si un nombre est palindrome en base 10, convertir en chaîne et comparer avec son inverse.
- Pour vérifier si un nombre est palindrome en base 2, convertir en binaire et comparer avec son inverse.
- On peut optimiser en générant seulement les palindromes en base 10, puis vérifier s'ils sont palindromes en base 2.

### Complexité
- **Temporelle** : O(n × log n), où `n` = 1 000 000.
- **Spatiale** : O(log n), pour stocker les représentations en chaînes.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>
#include <bitset>

bool isPalindrome(const std::string& s) {
    std::string rev = s;
    std::reverse(rev.begin(), rev.end());
    return s == rev;
}

std::string toBinary(int n) {
    std::string binary;
    while (n > 0) {
        binary = char('0' + (n % 2)) + binary;
        n /= 2;
    }
    return binary;
}

int main() {
    long long sum = 0;
    
    for (int i = 1; i < 1000000; ++i) {
        std::string decimal = std::to_string(i);
        std::string binary = toBinary(i);
        
        if (isPalindrome(decimal) && isPalindrome(binary)) {
            sum += i;
        }
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
