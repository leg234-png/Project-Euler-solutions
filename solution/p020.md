# Somme des Chiffres d'une Factorielle

## Description du Problème
n! signifie n × (n − 1) × ⋯ × 3 × 2 × 1.

Par exemple, 10! = 10 × 9 × ⋯ × 3 × 2 × 1 = 3628800, et la somme des chiffres du nombre 10! est 3 + 6 + 2 + 8 + 8 + 0 + 0 = 27.

**Objectif** : Trouver la somme des chiffres du nombre 100!.

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : `27`
**Explication** : 10! = 3628800, la somme des chiffres est 3+6+2+8+8+0+0 = 27.

---

## Approche de Résolution

### Stratégie
1. Calculer 100! (un nombre très grand).
2. Convertir le résultat en chaîne de caractères.
3. Parcourir chaque chiffre et les additionner.
4. Retourner la somme.

### Observations
- 100! est un nombre très grand qui ne tient pas dans les types entiers standards.
- On doit utiliser une bibliothèque pour les grands nombres ou implémenter la multiplication manuellement avec des chaînes.
- En C++, on peut utiliser des chaînes de caractères pour représenter les grands nombres.

### Complexité
- **Temporelle** : O(n × m), où `n` = 100 et `m` est le nombre de chiffres du résultat.
- **Spatiale** : O(m), pour stocker le nombre.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>
#include <algorithm>

std::string multiply(const std::string& num, int factor) {
    std::string result;
    int carry = 0;
    
    for (int i = num.length() - 1; i >= 0; --i) {
        int digit = (num[i] - '0') * factor + carry;
        result.push_back((digit % 10) + '0');
        carry = digit / 10;
    }
    
    while (carry > 0) {
        result.push_back((carry % 10) + '0');
        carry /= 10;
    }
    
    std::reverse(result.begin(), result.end());
    return result;
}

int main() {
    std::string factorial = "1";
    
    for (int i = 2; i <= 100; ++i) {
        factorial = multiply(factorial, i);
    }
    
    int sum = 0;
    for (char c : factorial) {
        sum += c - '0';
    }
    
    std::cout << sum << std::endl;
    return 0;
}
```
