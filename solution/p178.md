# Step Numbers

## Description du Problème
Un nombre à "pas" (step number) est un nombre où chaque chiffre consécutif diffère d'exactement 1. Par exemple, 123434 est un nombre à pas.

Un nombre "pandigital step" est un nombre à pas qui contient tous les chiffres de 0 à 9 au moins une fois.

**Objectif** : Combien de nombres pandigital step de 40 chiffres ou moins existent ?

---

## Exemples
### Exemple 1
**Entrée** : `n = 10`
**Sortie** : Nombre de nombres pandigital step
**Explication** : Un nombre pandigital step doit contenir tous les chiffres 0-9 et être un nombre à pas.

---

## Approche de Résolution

### Stratégie
1. Utiliser la programmation dynamique.
2. `ways[digits][head][low][high]` : nombre de nombres à pas de `digits` chiffres qui :
   - Commencent par le chiffre `head` (peut être 0).
   - Contiennent tous les chiffres de `low` à `high` (inclus).
3. Cas de base : `ways[0 ou 1][head][low][high] = 1` si `low == head == high`, sinon 0.
4. Récursion :
   - Si `head - 1 >= low` : ajouter `ways[digits-1][head-1][low][high]`.
   - Si `head == high` : ajouter `ways[digits-1][head-1][low][high-1]`.
   - Si `head + 1 <= high` : ajouter `ways[digits-1][head+1][low][high]`.
   - Si `head == low` : ajouter `ways[digits-1][head+1][low+1][high]`.
5. Additionner tous les `ways[digits][head][0][9]` où `head > 0` et `digits <= 40`.

### Observations
- Un nombre à pas doit avoir des chiffres consécutifs qui diffèrent de 1.
- On suit l'intervalle `[low, high]` des chiffres utilisés pour vérifier la condition pandigital.

### Complexité
- **Temporelle : O(LIMIT × 10 × 10 × 10 × 2) = O(LIMIT × 2000)**.
- **Spatiale : O(LIMIT × 10 × 10 × 10)**.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <boost/multiprecision/cpp_int.hpp>

using namespace boost::multiprecision;
using namespace std;

const int LIMIT = 40;

int main() {
    vector<vector<vector<vector<cpp_int>>>> ways(
        LIMIT + 1,
        vector<vector<vector<cpp_int>>>(
            10,
            vector<vector<cpp_int>>(
                10,
                vector<cpp_int>(10, 0)
            )
        )
    );
    
    cpp_int sum = 0;
    
    for (int digits = 0; digits <= LIMIT; digits++) {
        for (int head = 0; head < 10; head++) {
            for (int low = 0; low < 10; low++) {
                for (int high = 0; high < 10; high++) {
                    cpp_int val;
                    if (digits <= 1) {
                        val = (low == head && head == high) ? 1 : 0;
                    } else {
                        val = 0;
                        if (head - 1 >= low) {
                            val += ways[digits - 1][head - 1][low][high];
                            if (head == high)
                                val += ways[digits - 1][head - 1][low][high - 1];
                        }
                        if (head + 1 <= high) {
                            val += ways[digits - 1][head + 1][low][high];
                            if (head == low)
                                val += ways[digits - 1][head + 1][low + 1][high];
                        }
                    }
                    ways[digits][head][low][high] = val;
                    if (head > 0 && low == 0 && high == 9)
                        sum += val;
                }
            }
        }
    }
    
    cout << sum << endl;
    return 0;
}
```
