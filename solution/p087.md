# Triplets de puissances de nombres premiers

## Description du Problème
Le plus petit nombre exprimable comme la somme d'un carré de nombre premier, d'un cube de nombre premier, et d'une puissance quatrième de nombre premier est 28. En fait, il y a exactement quatre nombres inférieurs à cinquante qui peuvent être exprimés de cette façon :

28 = 2² + 2³ + 2⁴
33 = 3² + 2³ + 2⁴
49 = 5² + 2³ + 2⁴
47 = 2² + 3³ + 2⁴

**Objectif** : Combien de nombres inférieurs à cinquante millions peuvent être exprimés comme la somme d'un carré de nombre premier, d'un cube de nombre premier, et d'une puissance quatrième de nombre premier ?

---

## Exemples
### Exemple 1
**Entrée** : Limite = 50
**Sortie** : `4`
**Explication** : Les 4 nombres sont 28, 33, 49, et 47.

---

## Approche de Résolution

### Stratégie
1. Générer tous les nombres premiers jusqu'à √(50 000 000).
2. Pour chaque puissance (carré, cube, quatrième), générer toutes les sommes possibles.
3. Utiliser un HashSet pour stocker les sommes uniques.
4. Compter le nombre d'éléments dans le HashSet.

### Observations
- On calcule d'abord les carrés de nombres premiers, puis on ajoute les cubes, puis les puissances quatrièmes.
- On utilise un HashSet pour éviter les doublons.
- On s'arrête quand la somme dépasse la limite.

### Complexité
- **Temporelle** : O(P² × P³ × P⁴), où P est le nombre de nombres premiers, mais optimisé avec des limites.
- **Spatiale** : O(N), où N est le nombre de sommes uniques.

---

## Solution en C++

```cpp
#include <iostream>
#include <vector>
#include <unordered_set>
#include <cmath>

using namespace std;

vector<int> listPrimes(int n) {
    vector<bool> isPrime(n + 1, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i)
                isPrime[j] = false;
        }
    }
    vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i])
            primes.push_back(i);
    }
    return primes;
}

int main() {
    const int LIMIT = 50000000;
    vector<int> primes = listPrimes((int)sqrt(LIMIT));
    
    unordered_set<int> sums;
    sums.insert(0);
    
    for (int i = 2; i <= 4; i++) {
        unordered_set<int> newSums;
        for (int p : primes) {
            long long q = 1;
            for (int j = 0; j < i; j++)
                q *= p;
            if (q > LIMIT)
                break;
            
            int r = (int)q;
            for (int x : sums) {
                if (x + r <= LIMIT)
                    newSums.insert(x + r);
            }
        }
        sums = newSums;
    }
    
    cout << sums.size() << endl;
    return 0;
}
```
