# Nombre de Lettres

## Description du Problème
Si les nombres 1 à 5 sont écrits en mots : one, two, three, four, five, alors il y a 3 + 3 + 5 + 4 + 4 = 19 lettres utilisées au total.

**Objectif** : Si tous les nombres de 1 à 1000 (un thousand) inclus étaient écrits en mots, combien de lettres seraient utilisées ?

**NOTE** : Ne pas compter les espaces ou les tirets. Par exemple, 342 (three hundred and forty-two) contient 23 lettres et 115 (one hundred and fifteen) contient 20 lettres. L'utilisation de "and" lors de l'écriture des nombres est conforme à l'usage britannique.

---

## Exemples
### Exemple 1
**Entrée** : Nombres de 1 à 5
**Sortie** : `19`
**Explication** : one(3) + two(3) + three(5) + four(4) + five(4) = 19 lettres.

---

## Approche de Résolution

### Stratégie
1. Créer une fonction pour convertir un nombre en mots.
2. Pour chaque nombre de 1 à 1000, convertir en mots et compter les lettres.
3. Additionner toutes les lettres.

### Observations
- Règles pour convertir un nombre en mots :
  - 0-19 : mots spécifiques (zero, one, two, ..., nineteen)
  - 20-99 : dizaines (twenty, thirty, ..., ninety) + unités si nécessaire
  - 100-999 : centaines (one hundred, two hundred, ...) + "and" + reste si nécessaire
  - 1000 : "one thousand"
- Ne pas compter les espaces et les tirets.

### Complexité
- **Temporelle** : O(n), où `n` = 1000.
- **Spatiale** : O(1), car on utilise seulement quelques variables.

---

## Solution en C++

```cpp
#include <iostream>
#include <string>

std::string ones[] = {
    "zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine",
    "ten", "eleven", "twelve", "thirteen", "fourteen", "fifteen", "sixteen", "seventeen", "eighteen", "nineteen"
};

std::string tens[] = {
    "", "", "twenty", "thirty", "forty", "fifty", "sixty", "seventy", "eighty", "ninety"
};

std::string toEnglish(int n) {
    if (n >= 0 && n < 20) {
        return ones[n];
    } else if (n >= 20 && n < 100) {
        return tens[n / 10] + (n % 10 != 0 ? ones[n % 10] : "");
    } else if (n >= 100 && n < 1000) {
        return ones[n / 100] + "hundred" + (n % 100 != 0 ? "and" + toEnglish(n % 100) : "");
    } else if (n == 1000) {
        return "onethousand";
    }
    return "";
}

int main() {
    int sum = 0;
    for (int i = 1; i <= 1000; ++i) {
        sum += toEnglish(i).length();
    }
    std::cout << sum << std::endl;
    return 0;
}
```
